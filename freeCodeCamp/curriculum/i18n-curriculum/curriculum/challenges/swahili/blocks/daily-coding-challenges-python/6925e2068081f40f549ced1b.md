# --description--
Ako vam je dat višelinijski niz karaktera koji koristi karaktere novog reda (`\n`) za predstavljanje prekida linija, vratite novi niz gde je svaka linija jednako odražena i spakovana na kraju originalne linije.

- ASCII linijom menjanjem cele sekvence svojih karaktera, uključujući razmake.

Na primer, ako dobijete `"* \n *\n* "`, što se ispisuje u konzoli kao:

```sh
* 
 *
* 
```

Vrati `"*  *\n ** \n*  *"`, što se prikazuje u konzoli kao:

```sh
*  *
 ** 
*  *
```

Pazite posebno na prazninu između sekvence koja je data i one koja je vraćena. Uverite se da ne preskačete nijednu prazninu.

```py
def generate_snowflake(crystals):
    lines = crystals.split("\n")
    mirrored = []

    for line in lines:
        reversed_line = line[::-1]
        mirrored.append(line + reversed_line)

    return "\n".join(mirrored)
```
