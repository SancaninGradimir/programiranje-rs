# --interactive--

CSS, što je skraćenica od Cascading Style Sheets, predstavlja veoma važan deo savremenog veb razvoja. To je jezik za stilizovanje koji se koristi za određivanje izgleda HTML-a. Jednostavno rečeno, ako HTML predstavlja strukturu veb stranice, CSS je ono što joj daje lep izgled.

Evo primera za dekorisanje navbar-a.

**NAPOMENA**: Za sada ne brini ako ne razumeš CSS kod. Naučićeš kako sve funkcioniše u narednim lekcijama i radionicama. Ovo služi samo da stekneš predstavu o tome šta možeš da uradiš uz malo CSS-a.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<nav class="navbar">
    <ul class="nav-links">
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Services</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>
```

```css
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

body {
  font-family: 'Segoe UI', sans-serif;
  background-color: #f4f4f4;
}

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  background-color: #333;
  padding: 1rem 2rem;
  color: white;
}

.navbar .logo {
  font-size: 1.5rem;
  font-weight: bold;
}

.nav-links {
  list-style: none;
  display: flex;
  gap: 1.5rem;
}

.nav-links a {
  color: white;
  text-decoration: none;
  font-weight: 500;
  transition: color 0.3s ease;
}

.nav-links a:hover {
  color: #ff9800;
}
```

:::

Glavna uloga CSS-a je razdvajanje sadržaja veb stranice od njenog vizuelnog izgleda. Ovo razdvajanje omogućava veb programerima da prave fleksibilne i lako održive veb sajtove.

Pomoću CSS-a možeš kontrolisati raspored, boje, fontove i celokupan izgled veb stranica bez menjanja HTML strukture.

Zamisli veb sajt kao kuću. HTML predstavlja temelje i konstrukciju, dok CSS predstavlja boju, tapete i dekoraciju koji čine kuću lepom i jedinstvenom.

CSS funkcioniše tako što bira HTML elemente i primenjuje stilove na njih.

Evo primera koji prikazuje nestilizovano i stilizovano dugme. Da bi koristio ovaj primer, potrebno je da omogućiš interaktivni editor.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<button class="regular-btn">Unstyled Button</button>
<button class="round-btn">Round Button</button>
```

```css

.round-btn {
  font-size: 1rem;
  font-family: 'Segoe UI', sans-serif;
  border: none;
  cursor: pointer;
  transition: background-color 0.3s ease, transform 0.2s ease;
  background-color: #2ecc71;
  color: white;
  border-radius: 50px;
  padding: 0.6rem 1.6rem;
}

.round-btn:hover {
  background-color: #27ae60;
  transform: translateY(-2px);
}
```

:::

Ovi stilovi mogu obuhvatati osobine kao što su boja, veličina fonta i mnoge druge. Menjanjem tih osobina možeš značajno promeniti izgled veb stranice bez izmene njenog sadržaja.

Jedna od najmoćnijih mogućnosti CSS-a je sposobnost kreiranja prilagodljivih rasporeda.

To znači da, za CSS, možete učiniti da vaš sajt izgleda dobro na bilo kom uređaju, bilo da je to stolni računar, tablet ili mobilni telefon.

CSS omogućava vam da prilagodite podešavanja, veličinu fonta i druge vidljive elemente u skladu sa veličinom ekrana uređaja koji pregleda veb stranicu.

Pokušajte da omogućite interaktivni urednik i prilagodite veličinu prozora za prikaz kako biste videli kako se raspored menja u zavisnosti od veličine ekrana.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />

<header class="header">
    <h1>Welcome to My Responsive Site</h1>
    <p>This layout adapts to your screen size!</p>
</header>

<main class="content">
    <section class="box">Box 1</section>
    <section class="box">Box 2</section>
    <section class="box">Box 3</section>
</main>
```

```css
body {
  margin: 0;
  font-family: 'Segoe UI', sans-serif;
  background-color: #f9f9f9;
  color: #333;
}

.header {
  background-color: #4CAF50;
  color: white;
  padding: 2rem;
  text-align: center;
}

.content {
  display: flex;
  gap: 1rem;
  padding: 1rem;
  justify-content: center;
}

.box {
  flex: 1;
  min-width: 200px;
  background-color: #ddd;
  padding: 2rem;
  text-align: center;
  font-size: 1.2rem;
  border-radius: 8px;
  transition: background-color 0.3s;
}

.box:hover {
  background-color: #ccc;
}

/* Responsive: for tablets and below (≤ 768px) */
@media (max-width: 768px) {
  .content {
    flex-direction: column;
    align-items: center;
  }

  .box {
    width: 90%;
    font-size: 1.1rem;
  }
}

/* Responsive: for phones (≤ 480px) */
@media (max-width: 480px) {
  .header {
    padding: 1.5rem 1rem;
  }

  .header h1 {
    font-size: 1.5rem;
  }

  .header p {
    font-size: 1rem;
  }

  .box {
    font-size: 1rem;
    padding: 1.5rem;
  }
}
```

:::

Još jedna važna karakteristika CSS je njegova priroda neprekidnog praćenja, što je izvor reči "cascading" u njegovom imenu.

Ovo znači da stilovi mogu biti naslijeđeni i modifikovani, što omogućava višeslojni dizajn stilova. Saznaćete više o tome kako ovo funkcioniše u narednim lekcijama.

CSS Takođe podržava korišćenje eksternih stilskih fajlova.

Ovo znači da možete da postavite sva svoja pravila stila u zaseban fajl, koji se može povezati sa mnogim stranicama HTML.

Ova mogućnost znatno olakšava održavanje veb sajtova, naročito velikih. Umesto menjanja stilova na svakoj pojedinačnoj stranici, dovoljno je da izmeniš jednu CSS datoteku, a promene će važiti za sve povezane stranice.

U svetu veb razvoja CSS ima ključnu ulogu u kreiranju lepih, responzivnih i korisnicima prilagođenih veb sajtova.

On omogućava programerima da jednostavne HTML dokumente pretvore u privlačna veb iskustva koja angažuju korisnike i poboljšavaju njihovu interakciju sa sadržajem.
