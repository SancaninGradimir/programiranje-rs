# --explanation--

`First Conditional` koristi se za izražavanje stvarnih i mogućih budućih situacija. Gradi se pomoću `if` + `Present Simple` (za uslov) i `will` + osnovni oblik glagola (za rezultat). Na primer:

`If you debug the code, the app will run smoothly.` – Ispravljanje koda će omogućiti da aplikacija radi bez problema.

U ispravnoj rečenici uslov je `If we don't back up regularly`, a rezultat je `we could lose important information`. To pokazuje stvarnu mogućnost.
