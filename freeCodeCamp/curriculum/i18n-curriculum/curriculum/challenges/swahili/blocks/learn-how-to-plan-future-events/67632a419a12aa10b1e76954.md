# --explanation--

`Database` Odnosi se na sistem gde se podaci skladište i obrađuju.

`Activity` odnosi se na operacije ili procese koji se dešavaju u bazi podataka.

- `The database contains customer information and order details.` - Ovo znači da baza podataka skladišti važne poslovne podatke.

- `There was unusual activity in the database last night.` - Ovo znači da se dogodilo nešto neočekivano u bazi podataka.

Zajednički, `database activity` znači sve akcije, promene ili procese koji se dešavaju unutar sistema baze podataka.
