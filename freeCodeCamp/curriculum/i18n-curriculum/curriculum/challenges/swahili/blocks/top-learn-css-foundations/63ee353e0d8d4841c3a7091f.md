# --description--

Dobro, naučio si mnogo do sada. Jedina stvar koja je ostala za sada je naučiti kako da dodaš sve ovo na CSS u tvoj HTML. Postoji tri načina da to uradiš.

CSS eksterni je najčešće korišćeni metod koji ćete naići, i podrazumeva kreiranje zasebnog fajla za CSS i integrisanje ga unutar tagova otvaranja i zatvaranja `<head>` za HTML koristeći element `<link>` (što je element bez tagova zatvaranja):

Prvo, dodajete funkciju bez oznake zatvaranja `<link>` unutar oznaka otvaranja i zatvaranja `<head>` za fajl HTML. Atribut `href` je lokacija fajla CSS, bilo da je u pitanju kompletan URL ili, ono što ovde koristite, URL povezan sa lokacijom fajla HTML. U gornjem primeru, pretpostavljate da su oba vaša fajla u jednom direktorijumu. Atribut `rel` je važan, jer opisuje odnos između fajla HTML i povezanog fajla.

Zatim unutar novog fajla `styles.css`, imaš selektor (`div` i `p`), praćen kružnim zagradama za otvaranje i zatvaranje, što stvara "blok deklaracije". Na kraju, stavljaš bilo koju deklaraciju unutar bloka deklaracije. `color: white; ` je jedna deklaracija, gde je `color` atribut i `white` vrednost, a `background-color: black;` je druga deklaracija.

Napomena o imenima fajlova: `styles.css` je ime koje ste izabrali za ovaj fajl. Možete se na fajl pozvati bilo kojim imenom koje želite, pod uslovom da je tip fajla `.css`, iako su najčešće korišćeni "style" ili "styles".

Prednost keša za ovu metodu je:

1. Razdvaja HTML od vašeg CSS, što čini fajl HTML manjim i čistijim.
2. Morate urediti CSS samo na jednom mestu, što je posebno lako za veb-sajtove sa mnogim stranicama koje dele iste stilove.
