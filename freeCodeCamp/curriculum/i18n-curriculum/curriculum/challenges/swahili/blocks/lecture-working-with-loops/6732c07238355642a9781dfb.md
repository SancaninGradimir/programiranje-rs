# --interactive--

Naredba `break` koristi se za prevremeni izlazak iz petlje, dok se naredba `continue` koristi za preskakanje trenutne iteracije i nastavak sledeće.

Ovde je primer korišćenja naredbe `break` u ciklusu `for`:

:::interactive_editor

```js
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}
```

:::

U gornjem primeru petlja počinje da broji od `0`, a sve dok je `i` manje od `10`, nastaviće sa izvršavanjem.

U okviru petlje, proveravamo da li `i` je jednako `5`. Ako je jednako, koristimo naredbu `break` iz ranije petlje. Ako nije jednako, ispisujemo vrednost `i` na konzolu. Takođe, rezultat koda će ispisati brojeve `0`, `1`, `2`, `3`, i `4`.

Izjava `break` je korisna kada želite rano izaći iz petlje na osnovu određenog uslova. Na primer, ako tražite određenu vrednost u nizu podataka, možete koristiti izjavu `break` iz petlje čim pronađete tu vrednost.

Ponekad želite da preskočite određenu petlju bez potpunog izlaska. Ovde se koristi naredba `continue`. Evo primera korišćenja naredbe `continue` u petlji `for`:

:::interactive_editor

```js
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    continue;
  }
  console.log(i);
}
```

:::

Kao što je navedeno ranije, inicirali smo `i` da bude `0` i imamo status koji će omogućiti ciklusu nastavak projekta `i`, koji je pod `10`.

U petlji, kada `i` je jednako `5`, koristimo naredbu `continue` da preskočimo trenutnu iteraciju i nastavimo sa sledećom.

Rezultat ovog koda biće ispis brojeva `0`, `1`, `2`, `3`, `4`, `6`, `7`, `8` i `9`. Broj `5` je preskočen zbog naredbe `continue`.

Još jedna mogućnost sa naredbama `break` i `continue` jeste korišćenje oznaka (labela) za određivanje iz koje petlje želiš da izađeš ili koju želiš da nastaviš.

Ovo je korisno kada imaš ugnježdene petlje i želiš da upravljaš tokom spoljne petlje iz unutrašnje.

Ovo je primer korišćenja etikete i izjave za `break`:

:::interactive_editor

```js
outerLoop: for (let i = 0; i < 3; i++) {
  innerLoop: for (let j = 0; j < 3; j++) {
    if (i === 1 && j === 1) {
      break outerLoop;
    }
    console.log(`i: ${i}, j: ${j}`);
  }
}
```

:::

U ovom primeru, imamo spoljni krug od `for` sa oznakom `outerLoop` i unutrašnji krug od `for` sa oznakom `innerLoop`.

Kada `i` je jednako `1` i `j` je jednako `1`, koristimo izjavu `break` i labelu `outerLoop` iz spoljne petlje ranije. Ovo će biti dostupno u svim unutrašnjim i spoljnim petljama.

Rezultati ovog koda će biti prikazani na konzoli, kao što sledi:

```md
"i: 0, j: 0"
"i: 0, j: 1"
"i: 0, j: 2"
"i: 1, j: 0"
```

Često ćete morati da koristite oznake i izjave za `break` i `continue`, ali je dobro znati da imate tu opciju ako vam zatreba.
