# --description--
Konverzija elementa u mrežu utiče samo na svojstva njegovog direktnog roditelja. Stoga, konvertovanjem direktnog roditelja u mrežu, dobijate mrežu unutar mreže.

Na primer, postavljanjem atributa `display` i `grid-template-columns` za element sa klasom `item3`, kreirate mrežu unutar vaše mreže.

# --instructions--
Promeni element sa klasom `item3` u strukturalnu mrežu sa dve redne širine od `auto` i `1fr`, koristeći `display` i `grid-template-columns`.

# --hints--
Klasa `item3` mora imati atribut `grid-template-columns` sa vrednostima `auto` na `1fr`.

```js
assert.match(
  code,
  /.item3\s*?{[\s\S]*grid-template-columns\s*?:\s*?auto\s*?1fr\s*?;[\s\S]*}/gi
);
```

Klasa `item3` treba da ima svojstvo `display` sa vrednošću `grid`.

```js
assert.match(code, /.item3\s*?{[\s\S]*display\s*?:\s*?grid\s*?;[\s\S]*}/gi);
```
