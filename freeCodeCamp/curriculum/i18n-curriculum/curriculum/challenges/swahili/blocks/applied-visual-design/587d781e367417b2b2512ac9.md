# --description--

CSS tretira svaki HTML element kao sopstvenu kutiju, što se obično naziva <dfn>CSS box model</dfn>. Blok elementi automatski počinju na novom redu (zamislite heading-e, paragrafe i div-ove) dok su inline elementi smešteni unutar okolnog sadržaja (poput slika ili span-ova). Podrazumevani raspored elemenata na ovaj način naziva se <dfn>normalni tok</dfn> dokumenta, ali CSS nudi position svojstvo da ga nadjača.

Kada je pozicija elementa postavljena na `relative`, omogućava vam da odredite kako CSS treba da ga pomeri *u odnosu* na njegovu trenutnu poziciju u normalnom toku stranice. Kombinuje se sa CSS offset svojstvima `left` ili `right`, i `top` ili `bottom`. Ona ukazuju na to koliko piksela, procenata ili em-a pomeriti element *od* mesta gde bi inače bio. Sledeći primer pomera paragraf 10 piksela od dna:

```css
p {
  position: relative;
  bottom: 10px;
}
```

Promena pozicije elementa u relativnu ne uklanja ga iz normalnog toka - druge okolne stavke i dalje se ponašaju kao da je element na svojoj originalnoj poziciji.

**Napomena:** Pozicioniranje vam daje veliku slobodu i moć nad vizuelnim rasporedom stranice. Dobro je zapamtiti da bez obzira na poziciju elemenata, HTML oznake treba da budu organizovane i logične kada se čitaju odozgo prema dole. Ovo je način na koji korisnici sa oštećenjem vida (koji se oslanjaju na asistivne uređaje kao što su čitači ekrana) pristupaju vašem sadržaju.

# --instructions--

Promenite `position` od `h2` u `relative` i koristite CSS offset da ga pomerite 15 piksela od `top` gde se nalazi u normalnom toku. Obratite pažnju da nema uticaja na pozicije okolnih h1 i p elemenata.

# --hints--

`h2` element treba da ima `position` svojstvo postavljeno na `relative`.

```js
const h2Element =document.querySelector('h2')
const h2Style = window.getComputedStyle(h2Element);
assert.equal(h2Style?.position, 'relative');
```

Vaš kod treba da koristi CSS offset da postavi `h2` 15px od `top` mesta gde se normalno nalazi.

```js
const h2Element =document.querySelector('h2')
const h2Style = window.getComputedStyle(h2Element);
assert.equal(h2Style?.top, '15px');
```
