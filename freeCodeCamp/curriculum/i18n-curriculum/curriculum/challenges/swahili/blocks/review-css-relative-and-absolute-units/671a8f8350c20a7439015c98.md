## Direktne jedinice

- **`px` (Pikseli)**: Ovo je direktna jedinica fiksne veličine u CSS. To je najčešće korišćena direktna jedinica i pruža preciznu kontrolu nad dimenzijama. `1px` je uvek jednako 1/96 inča.
- **`in` (Inči)**: Ovo je direktna jedinica koja odgovara 96px.
- **`cm` (Centimetri)**: Ovo je direktna jedinica koja odgovara 25.2/64 inča.
- **`mm` (Milimetri)**: Ovo je direktna jedinica koja odgovara 1/10 centimetra.
- **`q` (Četvrtmilimetar)**: Ovo je direktna jedinica koja odgovara 1/40 centimetra.
- **`pc` (Pika)**: Ovo je direktna jedinica koja odgovara 1/6 inča.
- **`pt` (Pointer)**: Ovo je direktna jedinica koja odgovara 1/72 inča.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="units">
  <div class="unit px">px</div>
  <div class="unit inch">in</div>
  <div class="unit cm">cm</div>
  <div class="unit mm">mm</div>
  <div class="unit q">q</div>
  <div class="unit pc">pc</div>
  <div class="unit pt">pt</div>
</div>
```

```css
.units {
  display: flex;
  gap: 12px;
  flex-wrap: wrap;
  align-items: flex-end;
}

.unit {
  background: steelblue;
  color: white;
  text-align: center;
  padding: 4px;
}

.px {
  width: 40px;
  height: 40px;
}

.inch {
  width: 0.5in;
  height: 0.5in;
}

.cm {
  width: 1cm;
  height: 1cm;
}

.mm {
  width: 10mm;
  height: 10mm;
}

.q {
  width: 40q;
  height: 40q;
}

.pc {
  width: 6pc;
  height: 6pc;
}

.pt {
  width: 36pt;
  height: 36pt;
}
```

## Povezani elementi

- **Procenat**: Ove povezane jedinice omogućavaju postavljanje veličine, dimenzija i drugih atributa kao dela dimenzije roditeljskog kontejnera. Na primer, ako koristite `width: 50%;` na elementu, on će zauzeti polovinu širine svog roditeljskog kontejnera.
- **Jedinica `em`**: Ove jedinice se odnose na veličinu fonta elementa. Kada koristite `ems` za atribut `font-size`, veličina teksta će biti bazirana na veličini fonta roditeljskog elementa.
- **Jedinica `rem`**: Ove jedinice se odnose na veličinu fonta korenovog elementa, koji je element `html`.
- **Jedinica `vh`**: `vh` označava `"viewport height"`, a `1vh` je jednako 1% visine vidljivog područja.
- **Jedinica `vw`**: `vw` označava `"viewport width"`, a `1vw` je jednako 1% širine vidljivog područja.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="parent">
  <div class="box percent">50%</div>
  <div class="box em">2em</div>
  <div class="box rem">2rem</div>
  <div class="box vh">10vh</div>
  <div class="box vw">10vw</div>
</div>
```

```css
html {
  font-size: 16px;
}

.parent {
  width: 200px;
  font-size: 20px;
  border: 2px dashed #555;
  padding: 10px;
  display: flex;
  flex-direction: column;
  gap: 12px;
}

.box {
  background: seagreen;
  color: white;
  text-align: center;
  padding: 6px;
}

.percent {
  width: 50%;
}

.em {
  font-size: 2em;
}

.rem {
  font-size: 2rem;
}

.vh {
  height: 10vh;
}

.vw {
  width: 10vw;
}
```

## Funkcija za `calc`

- **Funkcija za `calc()`**: Za funkciju `calc()`, možete izvesti proračune direktno unutar vaših stilova kako biste odredili vrednosti svojstava pomoću transformacija. To znači da možete kreirati interfejse koji se prilagođavaju uređaju tako što ćete računati dimenzije na osnovu veličine vidokruga ili drugih elemenata.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="calc-box">calc()</div>
```

```css
.calc-box {
  width: calc(100% - 40px);
  padding: 10px;
  background: steelblue;
  color: white;
  text-align: center;
  border: 2px solid black;
}
```
