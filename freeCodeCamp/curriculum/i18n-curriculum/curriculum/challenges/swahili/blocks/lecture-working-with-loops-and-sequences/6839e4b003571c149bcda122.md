```py

```

Iako to funkcioniŇ°e, postoji jednostavniji nańćin pomońáu funkcije `enumerate()`. Funkcija `enumerate()` prati indeks svakog elementa u iterabilnom objektu i vrańáa objekat tipa enumerate.

Ako prosledimo listu `languages` funkciji `enumerate()` i njen rezultat pretvorimo u listu pomońáu funkcije `list()`, dobińáemo sledeńáe:

```python

list(enumerate(languages))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

Svaki element u objekta koji se enumerira (sada je lista) je tuple koji sadrŇĺi brojańć, prańáen vrednoŇ°ńáu iz iterabilnog objekta prosleńĎenog funkciji `enumerate()`.

Sada izmenimo prethodni primer tako da koristi funkciju `enumerate()`:

```python

    print(f'Index {index} and language {language}')
```

Za svaki tuple iz enumerate objekta raspakujemo indeks i vrednost u promenljive `index` i `language`. Na kraju se obe promenljive koriste u f-stringu koji se ispisuje u konzoli pri svakoj iteraciji petlje.

```md

Index 1 and language English
Index 2 and language Russian

```

Time viŇ°e nije potrebno ruńćno praviti i aŇĺurirati promenljivu `index`.

```python

    print(f'Index {index} and language {language}')
```

Evo kako ńáe izlaz izgledati u konzoli:

```md

```

Do sada smo prolazili samo kroz jednu listu. Ali Ň°ta ako treba da prolaziŇ° kroz viŇ°e iterabilnih objekata istovremeno? Za to moŇĺeŇ° da koristiŇ° funkciju `zip()`, koja spaja liste u parove elemenata i vrańáa iterator tuple-ova.

Ako prońĎemo listu `developers` i `ids` u funkciju `zip()` i promenimo njenu vrańáenu vrednost u listu za funkciju `list()`, bińáe prikazano ovako:

```python

ids = [1, 2, 3, 4]

list(zip(developers, ids))
# [('Naomi', 1), ('Dario', 2), ('Jessica', 3), ('Tom', 4)]
```

Ovo je primer koriŇ°ńáenja funkcije `zip()` zajedno sa petljom `for` koja iterira preko `developers` i `ids`:

```python

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')
```

U ovom primeru, `zip()` spaja dve liste u parove elemenata i vrańáa strukturu tuple. Petlja `for` zatim raspakuje svaki tuple na `name` i `id`. Na kraju, za svaku izjavu za Ň°tampanje, Ň°tampamo svaki `name` i `id` iz lista `ids` i `developers` respektivno. Ovde je kako rezultati izgledaju na konzoli:

```md

ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
```

Funkcija za `enumerate()` i `zip()` je veoma mońána, a kada se kombinuje sa petljama, moŇĺe uńćiniti vaŇ° kod krańáim.
