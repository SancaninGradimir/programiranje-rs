# --description--
Drugi način za predstavljanje grafa je postavljanje u <dfn>matrica susednosti</dfn>. <dfn>Matrica susednosti</dfn> je dvodimenzionalni niz podataka (2D) gde svaki unutrašnji red ima isti broj elemenata kao i glavni red. Drugim rečima, to je matrica ili mreža brojeva, gde ti brojevi predstavljaju ivice.

**Napomena**: Brojevi na vrhu i lijevoj strani matrice su samo oznake čvorova. Unutar matrice, broj znači da postoji korelacija između brojeva redova i kolona koji predstavljaju čvorove. Konačno, nula označava da nema korelacije ili veze.

<pre>
    1 2 3
  \------
1 | 0 1 1
2 | 1 0 0
3 | 1 0 0
</pre>

Gore je veoma jednostavan nedirekcijski graf sa tri čvora, gde je prvi čvor povezan sa drugim i trećim čvorom. Ispod je implementacija JavaScript istog.

```js
var adjMat = [
  [0, 1, 1],
  [1, 0, 0],
  [1, 0, 0]
];
```

Razlika od liste susjedstva je što svaki "red" matrice mora imati isti broj elemenata i čvorova u grafu. Ovde imamo matricu tri na tri, što znači da imamo tri čvora u našem grafu. Usmjeren graf bi izgledao ovako. Ispod je graf gdje prvi čvor ima ivicu usmjerenu ka drugom čvoru, a drugi čvor ima ivicu usmjerenu ka trećem čvoru.

```js
var adjMatDirected = [
  [0, 1, 0],
  [0, 0, 1],
  [0, 0, 0]
];
```

Grafovi takođe mogu imati <dfn>uzito</dfn> na svojim ivicama. Do sada, imali smo ivice <dfn>yasiyo na uzito</dfn> gde je postojanje ili nepostojanje ivice binarno (`0` ili `1`). Možete imati različite težine u zavisnosti od vašeg programa.

# --instructions--
Kreiraj matricu susedstva nedirektovanog grafa sa pet čvorova. Ova matrica treba da bude u redu organizovanih podataka. Ovi pet čvorova imaju veze između prvog i četvrtog čvora, prvog i trećeg čvora, trećeg i petog čvora, i četvrtog i petog čvora. Težina svih ivica je jedan.
