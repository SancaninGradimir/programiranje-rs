# --explanation--

`Specify` to znači reći jasno ili objasniti nešto detaljno. Na primer:

`Please specify the time and place of the meeting.` – To znači reći tačno kada i gde će se održati sastanak.

`What makes it` je nominalna fraza. Skup reči koji se koristi kao imenica u rečenici. Opisuje uzrok ili izvor nečega. Na primer:

`She described what makes it so effective.` – Ovo znači da je objasnio razlog zašto nešto efikasno radi. Ovde se odeljak `what makes it so effective` koristi kao funkcionalni element `described`.
