# --description--

JavaScript se smatra jezikom sa promenljivim tipovima tokom izvršavanja. To znači da parametri mogu primiti bilo koju vrednost tokom izvršavanja – deklarisanje parametra kao broja ne sprečava dodelu string vrednosti kasnije, a argumenti funkcije mogu biti prosleđeni bilo koju vrednost.

Izazov jezika sa promenljivim tipovima je što nedostatak sigurnosti tipa može dovesti do grešaka ako niste pažljivi. TypeScript dodaje jezik JavaScript uključivanjem statičkih tipova, što pomaže u otkrivanju tih grešaka čak i pre nego što deployujete svoj kod. Ali kako to radi?

Razmotrite ovu funkciju JavaScript:

```js
const getRandomValue = (array) => {
  return array[Math.floor(Math.random() * array.length)];
}
```

Naša funkcija ``getRandomValue`` koja uzima red podataka i vraća slučajnu vrednost iz tog reda. Ali zato što `JavaScript` ne proverava tip, ništa ne sprečava pozivanje te funkcije sa brojem:

```js
console.log(getRandomValue(10));
```

Izlaz konzole za ovaj primer će vratiti `undefined` jer je očekivao red podataka umesto brojeva.

Ali za TypeScript možete definisati vrstu parametra niza podataka:

```js
const getRandomValue = (array: string[]) => {
  return array[Math.floor(Math.random() * array.length)];
}
```

Ova definicija tipa kaže TypeScript da argument red mora biti niz znakova. Zatim kada pomenete `getRandomValue` i dodelite mu broj, dobijate drugačiji tip greške nazvan greška kolektora kodova.

Mnoge okruženja za izvršavanje JavaScript, poput pretraživača ili Node.js, ne mogu direktno pokrenuti TypeScript. Umesto toga, prvo morate kompajlirati ili prevesti kod iz TypeScript u standardni JavaScript. Možete to učiniti pomoću ugrađenog kompajlera koji dolazi sa jezikom TypeScript. Kada pokrenete kompajler, TypeScript će proveriti vaš kod i baciti grešku za bilo kakve neusklađene tipove - kao što je prosleđivanje broja funkciji koja očekuje niz podataka.

Naučićete više o tome kako ove vrste funkcionišu u budućim lekcijama. Ali upravo ova dodatna sigurnost čini TypeScript jezikom koji je privlačan mnogim programerima i organizacijama.
