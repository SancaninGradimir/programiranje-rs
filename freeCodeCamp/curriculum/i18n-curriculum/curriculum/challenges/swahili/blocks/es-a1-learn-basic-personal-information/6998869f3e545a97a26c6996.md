# --explanation--

Kompletno pitanje je `¿Cuál es tu nombre y apellido?`

- `¿Cuál...?` – Traži specifične informacije.

- `es` – iz glagola `ser`, saglašava se sa imenicom.

- `tu` – Indikator vlasništva koji prikazuje informacije pripada slušaocu.

Ovaj format je uobičajen kada se traže lični podaci. Primer:

`¿Cuál es tu número?` - Koji je vaš broj?
