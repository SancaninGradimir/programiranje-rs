# --description--

Najkompleksnije izjave za if mogu da sadrže mnogo uslova. Zato se logički operatori koriste za kombinovanje više uslova. Logički operatori su `&&`, `||` i `!`, koji se koriste za predstavljanje `and`, `or` i `not`, respektivno.

Logički operator `||` se koristi za kombinovanje dve uslova sa vrednostima tačno ili netačno. Vraća `true` ako je bar jedan od uslova `true`. Inače, vraća `false`:

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a > b || a > c) {
  console.log("At least one of the conditions is true");
} else {
  console.log("Both of the conditions are false");
}
```

Logički operator `&&` se koristi za spajanje dva uslova sa vrednošću tačno ili netačno. Vraća `true` samo ako su oba uslova `true`. Inače, vraća `false`:

```javascript
let a = 5;
let b = 10;
let c = 15;

if (a < b && a < c) {
  console.log("Both of the conditions are true");
} else {
  console.log("At least one of the conditions is false");
}
```

Logički operator `!` koristi se za negiranje stanja vrednosti tačno ili netačno. Vraća `true` ako je stanje `false`. Inače, vraća `false`:

```javascript
let a = 5;
let b = 10;

if (!(a > b)) {
  console.log("The condition is false");
} else {
  console.log("The condition is true");
}
```

U gornjem primeru, korišćen je dodatni par standardnih vitičastih zagrada kako bi kod bio lakši za čitanje. Nije neophodno ih koristiti.

# --questions--

## --text--

Dato vam je zadatak da napišete izveštaj `if` koji proverava sledeće uslove u veb aplikaciji radi prikazivanja `Welcome!` za korisnika:

Korisnik mora imati premium nalog (`isPremium`) ili biti član više od jedne godine (`membershipDuration` > 12 meseci).

Korisnik ne sme da bude trenutno isključen od usluge (`!isBlocked`).

Koji je izveštaj if koji proverava ove uslove sa preciznošću?

## --answers--

```javascript
if (isPremium && membershipDuration > 12 && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (isPremium || (membershipDuration > 12 && !isBlocked)) {
  console.log("Welcome!");
}
```

---

```javascript
if ((isPremium || membershipDuration > 12) && !isBlocked) {
  console.log("Welcome!");
}
```

---

```javascript
if (!isPremium || membershipDuration <= 12 || isBlocked) {
  console.log("Welcome!");
}
```
