# --explanation--

`Haven't` je skraćenica od `have not`, koja se koristi u trenutku `Present Perfect` da pokaže da akcija nije nastupila.

`Negative` opisuje nešto neprijatno ili kritično. Na primer:

`The user left a negative review about the app's performance.` - Ovde, `negative` objašnjava neugodno poreklo verifikacije.

`Comment` predstavlja komentar ili deo mrežnog sadržaja koji je objavio osoba. Njegov množinski oblik je `comments`. Primer:

`I received a comment suggesting more features in the app.` - U ovom kontekstu, `comment` opisuje povratne informacije koje je korisnik dao o programu.
