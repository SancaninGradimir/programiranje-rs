# --interactive--

Kada počnete da učite JavaScript, brzo ćete shvatiti da je rad aritmetičkih proračuna uobičajena vežba. Iako osnovni kalkulatorski tasteri mogu da obrade jednostavne proračune, JavaScript nudi nešto `Math` ugrađeno za rešavanje zahtevnijih matematičkih izazova.

Ovaj alat za pomoć pruža razne načine koji pojednostavljuju obavljanje naprednih proračuna i upravljanje brojevima. Hajde da istražimo ove metode i vidimo kako vam mogu olakšati iskustvo kodiranja.

Metoda `Math.random()` generiše neuredne brojeve između `0` (uključujući) i `1` (isključujući). Ovo znači da mogući rezultati mogu biti `0`, ali neće dostići `1`. Evo primera korišćenja metode `Math.random()`:

:::interactive_editor

```js
const randomNum = Math.random();

console.log(randomNum);
// any number between 0 and 1 – 0 inclusive and 1 exclusive
```

:::

`Math.min()` na `Math.max()` sve uzima skup brojeva i vraća minimalnu i maksimalnu vrednost, odnosno. Evo primera korišćenja obe metode:

:::interactive_editor

```js
const smallest = Math.min(1, 5, 3, 9);
console.log(smallest); // 1

const largest = Math.max(1, 5, 3, 9);
console.log(largest); // 9
```

:::

`console.log()` prvi će imati broj `1`, jer `1` je najmanji u toj listi brojeva. I `console.log()` drugi će imati broj `9`, jer `9` je najveći u toj listi brojeva.

Ako želite da zaokružite broj naviše ili naniže do najbližeg celog broja, možete koristiti metode `Math.ceil()` i `Math.floor()`. Evo primera korišćenja `Math.ceil()`:

:::interactive_editor

```js
console.log(Math.ceil(4.3)); // 5
```

:::

`Math.ceil()` će zaokruži `4.3` do najbližeg celog broja, koji je `5` u ovom slučaju. Sada, hajde da istražimo zaokruživanje broja naviše:

:::interactive_editor

```js
console.log(Math.floor(4.7)); // 4
```

:::

`Math.floor()` će zaokružiti `4.7` na najbliži ceo broj, koji je `4` u ovom slučaju. `Math.round()` je kombinacija `Math.ceil()` i `Math.floor()`. Zaokružuje broj na najbliži ceo broj, uzimajući u obzir decimalni deo:

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

Dakle, ako je decimalni deo manji od `5`, broj se zaokružuje nadole. A ako je decimalni deo `5` ili veći, broj se zaokružuje naniže. Praktična upotreba `Math.floor()` i `Math.random()` je kreiranje broja sa nosivim kapacitetom bez redosleda između dva cela broja. Ovo je njegova sintaksa:

:::interactive_editor

```js
const max = 10;
const min = 5;
const randomNum = Math.floor(Math.random() * (max - min + 1)) + min;
console.log(randomNum);
```

:::

Kreiranje slučajnog broja nosioca između `20` i `1` biće kao ovo:

:::interactive_editor

```js
const randomNumBtw1And20 = Math.floor(Math.random() * 20) + 1;
console.log(randomNumBtw1And20);
```

:::

Drug drugi način pomoći od `Math` je način `Math.trunc()`. `Math.trunc()` uklanja decimalni deo broja, vraćajući samo ceo broj, bez zaokruživanja:

:::interactive_editor

```js
console.log(Math.trunc(2.9)); // 2
console.log(Math.trunc(9.1)); // 9
```

:::

Ako treba da pronađete kvadratni koren ili koren broja, možete koristiti metode `Math.sqrt()` i `Math.cbrt()`, respektivno:

:::interactive_editor

```js
console.log(Math.sqrt(81)); // 9
console.log(Math.cbrt(27)); // 3
```

:::

Prva jednačina za pisanje će pisati `9` jer je kvadratni koren od `81` jednak `9`, dok će druga jednačina za pisanje pisati `3` jer je koren kružnice od `27` jednak `3`. Ako vam treba da pronađete punu vrednost broja, možete koristiti metod `Math.abs()`:

:::interactive_editor

```js
console.log(Math.abs(-5)); // 5
console.log(Math.abs(5)); // 5
```

:::

`Math.abs()` vraća punu vrednost broja, pretvarajući negativno u pozitivno. Poslednji metod koji ćemo istražiti je metod `Math.pow()`:

:::interactive_editor

```js
console.log(Math.pow(2, 3)); // 8
console.log(Math.pow(8, 2)); // 64
```

:::

`Math.pow()` uzima dva broja i podiže prvi na stepen drugog. Postoji mnogo više metoda koje poseduje nešto zvano `Math`, koje možete sami istražiti. Međutim, ovo su samo neke od metoda korišćenih u kodovima JavaScript.
