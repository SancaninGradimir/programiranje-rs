## Python u Vašem Unutrašnjem Okruženju

## Upotreba Python Interaktivnog Shell-a

- **Opis**: Interaktivna shell je program koji vam omogućava da ukucate komandu direktno i vidite rezultate. Otvorite terminalnu aplikaciju i ukucajte `python`, a zatim pritisnite `Enter`. Ovo će pokrenuti interaktivnu shell za Python. Znak `>>>` pokazuje da Python čeka da ukucate komandu. Pokušajte da izbacite (ili: pokrenete) `"Hello, world!"` u terminalu:

```bash
print("Hello, world!")
```

Trebalo bi da vidite tekst koji izgleda ovako:

```bash
>>> print("Hello, world!")
Hello, world!
```

Nakon što se tekst objavi, Python vraća se na sledeće:

```bash
>>>
```

Ovo znači da možete napisati još jednu komandu.

- **Ciklus čitanja, evaluacije, štampanja, ponavljanja (REPL)**: Svaki put kada napišete komandu, interpreter će pročitati unos, izvršiti ga, ispisati rezultate i ponoviti prikaz ``>>>`` ili nastavite sa pisanjem više komandi.

  - Imena parametara sa više reči razdvajaju se donjim crtama. Primer: `snake_case`.

## Tipovi standardnih podataka u Python

- **Uvod**: Python je jezik koji dinamički menja tipove podataka tokom izvršavanja, kao što je JavaScript, pa vam nije potrebno deklarisati tipove podataka za parametre. Jezik zna tip parametra na osnovu onoga što unesete.
- **Celobrojni broj**: Broj bez decimalnih delova:

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **Decimalni broj (Float)**: Broj sa decimalnim delovima:

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **Niz slova/karaktera (String)**: Niz slova postavljen unutar navodnika:

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **Logička vrednost (Boolean)**: Vrednost koja predstavlja `True` ili `False`:

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **Skup**: Neporedjena kolekcija jedinstvenih elemenata:

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **Kamusi (Dictionary)**: Kolekcija parova ključ-vrednost, unutar okruglih zagrada:

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **Tuple**: Kolekcija uređene i nepromenljive vrednosti, postavljena unutar standardnih zagrada:

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **Range**: Niz brojeva, često se koristi u petljama:

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **Orodha (List)**: Organizovana kolekcija elemenata koja podržava različite vrste podataka:

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**: Specifična vrednost koja predstavlja odsustvo vrednosti:

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## Nepromenljive i promenljive vrste

- **Nepromenljive vrste**: Ove vrste ne mogu se promeniti nakon deklarisanja, iako možete postaviti njene parametre da indikuju nešto novo, što se naziva ponovno definisanje (ili re-deklaracija). One uključuju integer, decimalni broj, kompleksan broj, boolean vrednost (istina ili laž), niz znakova, tuple, range, i `None`.
- **Promenljive vrste**: Ove vrste mogu se promeniti nakon deklarisanja. Možete dodati, ukloniti ili modifikovati njihove elemente. One uključuju kolekivne tipove kao što su lista, set i rečnik.
- **Funkcija `type()`**: Da biste videli tip parametra, možete koristiti funkciju `type()` na sledeći način:

```py
greeting = 'Hello there!'
age = 21

print(type(greeting)) # <class 'str'>
print(type(age)) # <class 'int'>
```

- **Funkcija `isinstance()`**: Koristi se za proveru da li parametar odgovara određenom tipu podataka:

```py
greeting = 'Hello world'
name = 'John Doe'

print(isinstance(greeting, str)) # True
print(isinstance(name, int)) # False
```

## Rad sa sekvencama slova

- **Opis**: Kao što ste se setili iz JavaScript, niz slova je nepromenljiv, što znači da ga ne možete promeniti nakon kreiranja. U Python, možete koristiti pojedinačne ili dvostruke navodnike. Savetuje se da izaberete jedno pravilo i da ga sledite:

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **Dobijanje karaktera iz sekvence karaktera**: Karakteri se mogu dohvatiti iz niza pomoću notacije uglastih zagrada, kao što sledi:

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **Escape znakovi u sekvenci**: Možete koristiti povratnik (backslash) (`\`) ako vaš niz sadrži navodnike kao što je sledeće:

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **Konkatenacija niza karaktera**: Da biste spojili sekvencu karaktera, možete koristiti operator `+`, kao što sledi:

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

Drug način za povezivanje sekvence je korišćenje operatora `+=`. Ovo se koristi za spajanje i distribuciju u jednom koraku, kao što sledi:

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**: Ovo je prečica za formatirane string literal-e. Omogućava vam da izvršite umetanje vrednosti i takođe spajanjem sa kratkom i čitljivom sintaksom:

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **Ekstrakcija dela niza znakova (String slicing)**: Ovo je kada želite da izvučete deo niza. Ovde je osnovna sintaksa:

```py
str[start:stop:step]
```

Početna pozicija ukazuje model od kojeg se uzima deo. Krajnja pozicija je mesto gde deo treba da se završi, a ta pozicija nije inkluzna. Korak pokazuje rastanak za preuzimanje delova. Evo primera:

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **Dobijanje dužine niza znakova**: Funkcija ``len()`` se koristi za vraćanje broja znakova u nizu:

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## Rad s operacijom od `in`

- **Funkcija za `in`**: Ovo vraća vrednost tačno/netačno koja pokazuje da li se slova ili karakteri nalaze unutar niza ili ne:

```py
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

## Standardni načini za nizove karaktera

- **`upper()`**: Ova sekvenca metoda vraća novu sekvencu sa svim velikim slovima:

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`lower()`**: Ovaj metod sekvence vraća novu sekvencu sa svim slovima u malim slovima:

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`strip()`**: Ova metoda vraća kopiju niza i uklanja određene početne i krajnje karaktere (ako nema argumenata, uklanja razmakove sa početka i kraja).

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**: Ovo vraća novu sekvencu gde su svi događaji starog redosleda zamenjeni novim redosledom.

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**: Koristi se za podelu sekvence u listu pomoću delimintera. Delimiter je niz koji opisuje mesto na kojem treba da se izvrši podela.

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**: Koristi se za spajanje iterativnih elemenata u niz znakova pomoću separatora. Iterable je kolekcija elemenata po kojima se može iterirati, kao što su lista, niz znakova ili tupel.

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`startswith(prefix)`**: Ova funkcija niza vraća booleanski vrednost (tačno ili netačno) ako niz počinje sa definisanim prefiksom:

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`endswith(suffix)`**: Ova funkcija sekvence vraća vrednost tačno ili netačno ako sekvenca završava sa navedenim zadnjim karakterom:

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`find()`**: Ova putanja sekvence vraća instancu prvog događaja fragmenta sekvence. Ako nije pronađeno, vraća `-1`:

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`count(substring)`**: Ova sekvencijalna metoda broji koliko puta se podsekvenca pojavljuje:

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`capitalize()`**: Ova metoda niza vraća novi niz sa velikim početnim slovom i ostalim malim:

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`isupper()`**: Ova sekvenca metoda vraća `True` ako su svi znakovi velika slova i `False` inače:

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`islower()`**: Ova sekvenca putanja vraća `True` ako su svi znakovi mala slova i `False` inače:

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`title()`**: Ova funkcija sekvence vraća novu sekvencu sa velikim početnim slovom svake reči:

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`maketrans()`**: Ovaj sekvencijalni metod se koristi za kreiranje tabele mape karaktera jedan na jedan radi prevoda. Često se koristi zajedno sa `translate()`, koja koristi tu tabelu na sekvenci i vraća prevedene rezultate.

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## Standardne operacije koje se koriste sa celim i decimalnim brojevima

- **Osnovne matematičke operacije**: U Python, možete izvesti osnovne matematičke operacije sa celim brojevima i decimalnim brojevima, uključujući sabiranje, oduzimanje, množenje i deljenje:

```py
int_1 = 56
int_2 = 12
float_1 = 5.4
float_2 = 12.0

# Addition

print('Integer Addition:', int_1 + int_2) # Integer Addition: 68
print('Float Addition:', float_1 + float_2) # Float Addition: 17.4

# Subtraction

print('Int Subtraction:', int_1 - int_2) # Int Subtraction: 44
print('Float Subtraction:',  float_2 - float_1) # Float Subtraction: 6.6

# Multiplication

print('Int Multiplication:', int_1 * int_2) # Int Multiplication: 672
print('Float Multiplication:', float_2 * float_1) # Float Multiplication: 64.80000000000001

# Division

print('Division:', int_1 / int_2) # Division: 4.666666666666667
print('Float Division:', float_2 / float_1) # Float Division: 2.222222222222222
```

Kada dodate decimalni broj i ceo broj, rezultat će biti konvertovan u decimalni broj, kao što sledi:

```py
int_1 = 56
float_1 = 5.4

print(int_1 + float_1) # 61.4
```

- **Modulo operator (`%`)**: Ovo vraća ostatak kada se jedan broj deli sa drugim brojem:

```py
int_1 = 56
int_2 = 12

print(int_1 % int_2) # 8
```

- **Celobrojno deljenje (`//`)**: Ovaj operator se koristi za deljenje dva broja i zaokruživanje rezultata na najbliži ceo broj.

```py
int_1 = 56
int_2 = 12

print(int_1 // int_2) # 4
```

- **Eksponent snage (`**`)**: Ovaj eksponent se koristi za podizanje broja na drugu potenciju:

```py
int_1 = 4
int_2 = 2

print(int_1 ** int_2) # 16
```

- **Funkcija za `float()`**: Možete koristiti ovu funkciju da biste konvertovali ceo broj u decimalni broj.

```py
num = 4

print(float(num)) # 4.0
```

- **Funkcija `int()`**: Možete koristiti ovu funkciju za konverziju decimalnog broja u ceo broj.

```py
num = 4.0

print(int(num)) # 4
```

- **Funkcija za `round()`**: Koristi se za zaokruživanje brojeva na najbliži ceo broj:

```py
num_1 = 3.4
num_2 = 7.7

print(round(num_1)) # 3
print(round(num_2)) # 8
```

- **Funkcija `abs()`**: Koristi se za vraćanje pune vrednosti broja:

```py
num = -13

print(abs(num)) # 13
```

- **Funkcija `pow()`**: Koristi se za podizanje broja na drugu stepenicu:

```py
result = pow(2, 3) 
print(result)  # 8
```

## Optimizovana distribucija

- **Opis**: Napredno dodeljivanje spaja binarnu operaciju i dodeljivanje u jednom koraku. Uzima varijablu, izvodi operaciju sa drugom vrednošću, i čuva rezultat nazad u tu varijablu.

```py
# Addition assignment 
my_var = 10
my_var += 5

print(my_var) # 15

# Subtraction assignment
count = 14
count -= 3

print(count) # 11

# Multiplication assignment 
product = 65
product *= 7

print(product) # 455

# Division assignment 
price = 100
price /= 4

print(price) # 25.0

# Floor Division assignment 
total_pages = 23
total_pages //= 5

print(total_pages) # 4

# Modulo assignment 
bits = 35
bits %= 2

print(bits) # 1

# Exponentiation assignment 
power = 2
power **= 3

print(power) # 8
```

Postoje takođe i druge operacije sa poboljšanom alokacijom, poput onih za bitwise operacije. Obuhvataju `&=`, `^=`, `>>=`, i `<<=`.

## Raditi sa uređajem

- **Opis**: Funkcija je deo koda koji se može ponovo koristiti, koji prima ulaz (argument) i vraća rezultate. Da biste pozvali funkciju, morate navesti ime funkcije praćeno standardnim zagradama:

```py
# Defining a function

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4) # function call
print(result) # 7
```

Ako funkcija ne vraća vrednosti eksplicitno, vrednost za default od return je `None`:

```py
def greet():
    print('hello') 

result = greet() # hello
print(result) # None
```

Takođe možete da pružite vrednosti za default za parametre kao što sledi:

```py
def get_sum(num_1, num_2=2):
    return num_1 + num_2

result = get_sum(3) 
print(result) # 5
```

Ako pozovete funkciju bez ispravnog broja argumenata, dobićete `TypeError`:

```py
def calculate_sum(a, b):
    print(a + b)

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

## Opseg u Python

- **Lokalni opseg**: Ovo je mesto gde se varijabla deklarše unutar funkcije ili klase i koja može biti dostupna samo unutar te specifične funkcije ili klase.

```py
def my_func():
    num = 10
    print(num)
```

- **Opseg (Scope)**: Ovo je situacija u kojoj funkcija ugnježdena unutar druge funkcije može pristupiti parametrima spoljašnje/roditeljske funkcije.

```py
def outer_func():
    msg = 'Hello there!'
    
    def inner_func():
        print(msg)
    inner_func()

print(outer_func()) # Hello there!
```

- **Globalni opseg**: Ovo se odnosi na parametre deklarisane van funkcija ili klasa koji mogu biti dostupni iz bilo kog dela programa.

```py
tax = 0.70 

def get_total(subtotal):
    total = subtotal + (subtotal * tax)
    return total

print(get_total(100))  # 170.0
```

- **Ugrađeni interni opseg**: Imena sačuvana u Python za prethodno deklarisane funkcije, module, ključne reči i entitete.

```py
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```

## Opereta za poređenje

- **Sawa (`==`)**: Ovo proverava da li su dve vrednosti jednake:

```py
print(3 == 4) # False
```

- **Sio sawa (`!=`)**: Ovo proverava da li su dve vrednosti različite:

```py
print(3 != 4) # True
```

- **Najveći od svih (`>`)**: Ovo proverava da li je jedna vrednost veća od druge:

```py
print(3 > 4) # False
```

- **Najmanja vrednost (`<`)**: Proverava da li je jedna vrednost manja od druge:

```py
print(3 < 4) # True
```

- **Veće ili jednako (`>=`)**: Ovo proverava da li je jedna vrednost veća ili jednaka drugoj:

```py
print(3 >= 4) # False
```

- **Manje ili jednako (`<=`)**: Ovo proverava da li je jedna vrednost manje ili jednaka drugoj:

```py
print(3 <= 4) # True
```

## Rad sa `if`, `elif` i `else` Statements

- **`if` Statements**: Ovo su uslovi koji se koriste za određivanje da li je nešto tačno ili netačno. Ako je uslov potvrđen kao `True`, onda će se taj deo koda izvršiti.

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **`elif` Klauzula**: Ovo su uslovi koji prate informaciju o `if`. Ako je uslov `elif` potvrđen da je `True`, onda će taj deo koda biti izvršen.

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **`else` Klauzula**: Ovo će biti izvršeno ako druge uslove ne potvrđuju da `True`.

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

Takođe možete koristiti informacije iz `if` koje su uključene unutra, kao što sledi:

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## Vrednosti istine i laži

- **Opis**: U Python, svaka vrednost ima inherentnu istinitost ili lažnost, ili unutrašnje osećanje da treba da se tretira kao `True` ili `False` u kontekstu logike. Većina vrednosti smatra se istinitim, tj., dokazano je da su `True` u kontekstu logike. Ostale su lažne, što je dokazano kao `False`. Evo primera lažnih vrednosti:

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

Ostale vrednosti, kao što su brojevi različiti od nule i ne-prazan niz znakova, su tačne.

## Rad sa funkcijom `bool()`

- **Opis**: Ako želite proveriti da li je vrednost istinita ili lažna, možete koristiti ugrađenu funkciju `bool()`. Ona konvertuje vrednost na ekvivalent istinite ili lažne vrednosti i vraća `True` za istinite vrednosti i `False` za lažne vrednosti. Evo nekoliko primera:

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## Algoritam za vrednosti tačno i netačno i smanjenje ciklusa

- **Opis**: Ovo su specijalni operatori koji vam omogućavaju da spojite više izraza i kreirate složeniju logiku odlučivanja u vašem kodu. Postoji tri operatera za vrednosti istinito i lažno u Python: `and`, `or`, i `not`.
- **Operater `and`**: Ovaj operater uzima dva operanda i vraća prvi operand ako je lažan, inače vraća drugi operand. Oba operanda moraju biti istinita da bi izraz proizveo vrednost tačno.

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

Možete takođe koristiti operaciju `and` na sledećim uslovima:

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **Operater za `or`**: Ovaj operator vraća prvi operand ako je tačan, inače vraća drugi operand. Izraz `or` ima vrednost istinu (true) ako je barem jedan operand tačan. Evo primera:

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

Kao što je slučaj sa operetom `and`, možete koristiti operetu `or` na uslovima kao što su sledeći:

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **Skraćivanje ciklusa**: Operacije za `and` i `or` poznate su kao operacije skraćivanja ciklusa. Skraćivanje ciklusa znači da Python proverava vrednosti od levi ka desni i zaustavlja se čim detektuje konačan rezultat.
- **Operacija `not`**: Ova operacija uzima jedan operand i konvertuje njegovu tačnu ili netačnu vrednost. Konvertuje tačne vrednosti u `False`, a netačne vrednosti u `True`. Za razliku od prethodnih operacija, `not` uvek vraća `True` ili `False`. Evo primera:

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

Ovde je primer operete `not` u uslovu:

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```

## Tuple u Python

- **Opis**: Tuple je tip podataka Python koji se koristi za kreiranje usrednice uređena vrednosti. Tuple mogu imati mešoviti tip podataka:

```py
developer = ('Alice', 34, 'Rust Developer')
```

- Tupel je nemenjiv, što znači da se elementi unutar tupela ne mogu promeniti nakon kreiranja. Ako pokušate da modifikujete element unutar tupela, dobićete `TypeError`:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object does not support item assignment
"""
```

- **Dobijanje elemenata iz tupaola**: Da biste dobili element iz tupaola, koristite notaciju zagrada i indeks:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

- Negativni indeks se može koristiti za dobijanje elemenata počevši od kraja tupela:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

- Ako pokušate da postavite vrednost koja je veća ili jednaka dužini tupa, dobićete `IndexError`:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
"""
```

- Tuple se takođe može kreirati korišćenjem komponente `tuple()`. Unutar komponente, možete postaviti različite iterabilne objekte kao što su niz karaktera, lista i čak drugi tuple.

```python
developer = 'Jessica'

print(tuple(developer)) 
# Result: ('J', 'e', 's', 's', 'i', 'c', 'a')
```

- **Provera elemenata u tuplu**: Da biste proverili da li je element u tuplu, možete koristiti ključnu reč `in` na sledeći način:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **Izvlačenje elemenata iz tupola**: Elementi se mogu izvlačiti iz tupola na sledeći način:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer
```

- Ako trebate sakupiti preostale elemente iz tupaola, možete koristiti zvezdasti operator (`*`) kao što sledi:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer
```

- **Izvlačenje dela tupa**: Izvlačenje se može koristiti za izvlačenje dela tupa. Na primer, stvari `pie` i `cookies` mogu se smatrati različitim tupelom:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

- **Brisanje elemenata iz tupa**: Brisanje elementa iz tupa će dodati `TypeError` jer su tupi nepromenljivi:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

- **Kada koristiti tuple umesto liste?**: Ako vam je potreban kolekcija promenljivih elemenata gde možete dodati, ukloniti i izmeniti elemente, onda koristite listu. Ako znate da radite sa fiksnom i nepromenljivom kolekcijom podataka, onda koristite tuple.

programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')

- **`sorted()`**: Koristi se za sortiranje elemenata u bilo kom iterabilnom objektu i vraća novu, sortiranu listu. Evo primera kreiranja nove liste brojeva korišćenjem funkcije `sorted()`:

programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

Java
Python
C++

c
o
d
e

categories = ['Fruit', 'Vegetable']

    guess = int(input('Guess the number (1-5): '))

- **Instrukcije za `break` na `continue`**: Koriste se u kola za promenu izvršavanja kola.

- Deklaracija `break` izvršava se jednom po ciklusu kada je uslov zadovoljen. Evo primera korišćenja deklaracije `break` za listu `developer_names`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

    if developer == 'Naomi':

- Izjava `continue` se koristi za preskakanje trenutne ponavljanja i nastavak sa sledećom ponavljanjem u ciklusu. Evo primera korišćenja izjave `continue` umesto izjave `break`:

developer_names = ['Jess', 'Naomi', 'Tom']

    print(developer)
```

words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

    print(num)
```

- Ako argument `start` nije dostupan, podrazumevana opcija je `0`. Za serijski niz punih brojeva, povećanje će biti dodato sa `1`. Možete koristiti opcioni argument `step` da promenite vrednost inkrementa. Evo primera kreiranja serije punih brojeva čak i od 2 do 10 (bez uključivanja 11):

```py
for num in range(2, 11, 2):
    print(num)
```

- Ako pružite bilo koji argument funkciji za `range()`, onda ćete dobiti `TypeError`.

- Funkcija `range()` prihvata samo cele brojeve za argument, a ne decimalne brojeve. Korišćenje decimalnih brojeva će izazvati grešku tipa `TypeError`:

```py
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

- Možete koristiti negativan ceo broj za argument `step` da kreirate opadajuću sekvencu celih brojeva:

```py
for num in range(40, 0, -10):
    print(num)
```

- Funkcija `list` se takođe može koristiti za kreiranje liste celih brojeva koristeći `range()` kao komponentu. Komponenta `list` se koristi za konverziju iterabilnog objekta u listu. Evo primera za kreiranje liste celih brojeva, čak i između 2 i 10 (uključujući):

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

## Funkcija `enumerate()` i `zip()` u Python

- **`enumerate()`**: Koristi se za iteriranje kroz sekvencu i praćenje indeksa svakog elementa u toj sekvenci. Funkcija ``enumerate()`` koja uzima *iterable* kao argument i vraća objekat ``enumerate`` koji uključuje indeks i vrednost svakog elementa u *iterable*-u.

languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):

# Result
# Index 0 and language Spanish
# Index 1 and language English

# Index 3 and language Chinese

- Funkcija `enumerate()` se takođe može koristiti van opsega `for`:

languages = ['Spanish', 'English', 'Russian', 'Chinese']

print(list(enumerate(languages)))

```

- Funkcija `enumerate()` prima takođe opcionalni argument `start` koji određuje početnu vrednost za brojanje. Ako ovaj argument nije dostavljen, brojanje počinje na `0`.

- **`zip()`**: Koristi se za iteriranje preko više iterabilnih objekata istovremeno. Evo primera korišćenja funkcije `zip()` za iteriranje preko liste `developers` i `ids`:

developers = ['Naomi', 'Dario', 'Jessica', 'Tom']

## List comprehensions u Python

- **Definicija**: List comprehension vam omogućava da kreirate novu listu u jednoj liniji spajanjem petlje i uslova direktno unutar uglastih zagrada. Ovo čini kod kraćim i često lakšim za čitanje.

```py
even_numbers = [num for num in range(21) if num % 2 == 0]

```

## Iterabilna funkcija

- **`filter()`**: Koristi se za filtriranje elemenata iz iterabljive kolekcije po određenim uslovima. Vraća iterator koji sadrži samo elemente koji zadovoljavaju uslov. Evo primera kreiranja nove liste reči koje su duže od četiri slova:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**: Koristi se za primenu funkcije na svakom elementu u iterabilnom objektu i vraćanje novog iterabilnog objekta sa rezultatima. Evo primera korišćenja funkcije `map()` za konverziju liste temperatura iz Celzija u Farenhajta:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**: Koristi se za dobijanje zbira iz iterabilnog objekta kao što su lista ili tuple. Evo primera korišćenja funkcije ``sum()``:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- Možete takođe proslediti opcionalni argument ``start`` koji postavlja početnu ukupnu vrednost. Evo ažuriranog primera koji koristi argument ``start`` kao pozicionalni argument:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- Takođe možete koristiti argument `start` kao tekstualni argument umesto ovoga:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

## Lambda funkcija

- **Definicija**: Lambda funkcija u Python je kratak način za kreiranje anonimne funkcije (anonimna funkcija).
- Lambda funkcija se često koristi kao argument za drugu funkciju. Evo primera lambda funkcije:

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

- Najbolje prakse za korišćenje lambda funkcije su da je ne prosleđujete kao argument, već da ih održavate jednostavnim i razumljivim, te da ih koristite samo jednom za kratke funkcije.

## Rečnik

- **Rečnik**: Rečnik je interni struktura podataka koja skladišti kolekcije parova ključ-vrednost. Ključevi moraju biti nemenjivi tipovi podataka. Ovo je opšta sintaksa rečnika za Python:

```python
dictionary = {
    key1: value1,
    key2: value2
}
```

- **Komponenta za `dict()`**: Komponenta `dict()` je alternativa za kreiranje rečnika. Prolazi listu tuple-a kao argument komponenti `dict()`. Ove tuple imaju ključeve kao prvi element i vrednosti kao drugi element.

```python
pizza = dict([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250), ('toppings', ['mozzarella', 'basil'])])
```

- **Notacija vitičastih zagrada (Bracket Notation)**: Da biste dobili vrednost para ključ-vrednost, možete koristiti sintaksu poznatu kao notacija vitičastih zagrada.

```python
dictionary[key]
```

## Uobičajeni načini za rečnik

- **`get()`**: Putanja za `get()` dobija vrednost vezanu za ključ. Slično je notaciji zagrada, ali omogućava postavljanje podrazumevane vrednosti, sprečavajući greške ako ključ ne postoji.

```python
dictionary.get(key, default)
```

- **`keys()` na `values()`**: Metode za `keys()` na `values()` vraćaju objekat pogleda sa svim ključevima i svim vrednostima iz rečnika, respektivno. Objekat pogleda je način za pregledavanje sadržaja u rečniku bez kreiranja zasene kopije podataka.

```python
pizza = {
    'name': 'Margherita Pizza',
    'price': 8.9,
    'calories_per_slice': 250
}

pizza.keys()
# dict_keys(['name', 'price', 'calories_per_slice'])

pizza.values()
# dict_values(['Margherita Pizza', 8.9, 250])
```

- **`items()`**: Metoda `items()` vraća objekat pogleda koji sadrži sve parove ključ-vrednost u rečniku, uključujući i ključeve i vrednosti.

```python
pizza.items()
# dict_items([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250)])
```

- **`clear()`**: Metoda `clear()` uklanja sve parove ključ-vrednost iz rečnika.

```python
pizza.clear()
```

- **`pop()`**: Metoda `pop()` uklanja par ključ-vrednost sa ključem navedenim kao prvi argument i vraća njegovu vrednost. Ako ključ ne postoji, vraća podrazumevanu vrednost kao drugi argument. Ako ključ ne postoji i nije navedena podrazumevana vrednost, će biti izbačena greška `KeyError`.

```python
pizza.pop('price', 10)
pizza.pop('total_price') # KeyError
```

- **`popitem()`**: U Python 3.7 i više, metoda `popitem()` uklanja poslednji postavljen element.

```python
pizza.popitem()
```

- **`update()`**: Metoda `update()` spaja parove ključ-vrednost sa parovima ključ-vrednost drugog rečnika. Ako imaju isti ključ, njihove vrednosti se prepisuju. Novi ključevi će biti dodati kao novi parovi ključ-vrednost.

```python
pizza.update({ 'price': 15, 'total_time': 25 })
```

products = {

for product in products.keys():

Postavljanje atributa i polja u različite parametre ciklusa je uobičajen način za korišćenje `enumerate()`. Na primer, za `products.items()`, možete dobiti ceo par vrednosti ključa zajedno sa indeksom:

for index, product in enumerate(products.items()):

for index, product in enumerate(products.items(), 1):

## Množine

- **Setovi**: Setovi su struktura podataka ugrađena u Python koja ne dozvoljava duplikate vrednosti. Setovi su promenljivi i bez redosleda, što znači da njihovi elementi nisu sačuvani po određenom nizu, pa ih ne možete pristupiti pomoću indeksa ili ključeva. Takođe, setovi mogu sadržati samo imutabilne tipove podataka, kao što su brojevi, niz znakova i tuple.

- **Definisati set**: Da biste definisali set, potrebno je da napišete njegove elemente unutar zaokobljenih zagrada i razdvojite ih zarezom.

```python
my_set = {1, 2, 3, 4, 5}
```

- **Definisati set prazno**: Ako treba da definišete set prazno, morate koristiti funkciju `set()`. Samo pisanje praznih vitičastih zagrada direktno će kreirati rečnik.

```python
set() # Set
{}    # Dictionary
```

## Standardni načini za set

- **`add()`**: Možete dodati funkciju na set putem `add()`, prosleđujući novu funkciju kao argument.

```python
my_set.add(6)
```

- **`remove()` na `discard()`**: Kada uklonite element iz set, možete koristiti metodu `remove()` ili `discard()`, prosleđujući element koji želite da uklonite kao argument. Metoda `remove()` će baciti grešku/izuzetak `KeyError` ako element nije pronađen, dok `discard()` neće bacati grešku.

```python
my_set.remove(4)
my_set.discard(4)
```

- **`clear()`**: Putanja za `clear()` uklanja sve elemente iz set.

```python

```

## Operacije za kalkulator set

- **`issubset()` na `issuperset()`**: Putanje za `issubset()` i `issuperset()` proveravaju da li set je podskup ili nadskup drugog set, respektivno.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 5}

print(your_set.issubset(my_set)) # True
print(my_set.issuperset(your_set)) # True
```

- **`isdisjoint()`**: Put `isdisjoint()` proverava da li dva seta nemaju slične elemente.

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

print(my_set.isdisjoint(your_set)) # True
```

- **Operater unije (`|`)**: Operater unije `|` vraća set novi sa svim elementima iz oba skupa.

```python
my_set = {1, 2, 3}
your_set = {4, 5, 6}

my_set | your_set # {1, 2, 3, 4, 5, 6}
```

- **Operater preseka (intersection) (`&`)**: Operater preseka `&` vraća novi set sa elementima koji su prisutni u oba skupa.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set & your_set # {2, 3, 4}
```

- **Operater razlike (difference) (`-`)**: Operater razlike `-` vraća set novi skup sa elementima prvog skupa koji nisu u drugom skupu.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set - your_set # {1, 5}
```

- **Operater simetrične razlike (symmetric difference) (`^`)**: Operater simetrične razlike `^` vraća set sa novim elementima koji su u prvom ili drugom skupu, ali ne i oba.

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

my_set ^ your_set # {1, 5, 6}
```

- **Operater za proveru prisutnosti (`in`)**: Može se proveriti da li je element u set ili ne, koristeći operater `in`.

```python
print(5 in my_set)
```

## Izjava o import

- **Deklaracija import**: Da biste pristupili elementima navedenim u ugrađenim modulima, koristite deklaraciju import. Deklaracije import obično se pišu na vrhu fajla. Deklaracije import rade isto za funkcije, klase, konstante, parametre i sve ostale elemente navedene u modulima.

- **Osnovna izjava za import**: Možete koristiti glavnu reč `import` praćenu imenom modula:

```python
import module_name
```

Zatim, ako treba da pozovete funkciju iz tog modula, koristićete notaciju tačke, naziv modula praćen nazivom funkcije.

```python
module_name.function_name()
```

Na primer, napišite sledeće u svoj kod da učitate modul `math` i pronađete kvadratni koren iz 36:

```python
import math

math.sqrt(36)
```

- **Uvoz modula pod drugačijim imenom**: Ako je potrebno uvesti module pod drugim imenom (poznatom kao "alias"), možete koristiti ključnu reč `as` kao alias na kraju izjave import. Ovo se često koristi za dugačka imena modula ili za izbegavanje kolizija imena.

```python
import module_name as module_alias
```

Na primer, referenciranje modula `math` kao `m` u vašem kodu, možete postaviti alias ovako:

```python
import math as m
```

Zatim, možete dobiti komponente modula koristeći alias:

```python
m.sqrt(36)
```

- **Uključivanje specifičnih elemenata**: Ako vam ne treba sve iz modula, možete uključiti specifične elemente koristeći `from`. U ovom slučaju, izjava import počinje sa `from`, praćena nazivom modula, zatim glavnom reči `import`, i na kraju imenima elemenata koje želite uključiti.

```python
from module_name import name1, name2
```

Zatim, možete koristiti ova imena bez modulatorskog prefiksa u vašoj skripti za Python. Na primer:

```python
from math import radians, sin, cos

angle_degrees = 40
angle_radians = radians(angle_degrees)

sine_value = sin(angle_radians)
cos_value = cos(angle_radians)

print(sine_value) # 0.6427876096865393
print(cos_value)  # 0.766044443118978
```

Ovo je pomoć, ali može uzrokovati sukobe imena ako već imaš funkcije ili parametre sa istim imenima. Molimo te, imaj ovo na umu kada odlučuješ koji tip deklaracije da koristiš za import.

Ako treba da date ovim imenima alias, možete to uraditi i korišćenjem glavne reči `as` praćeno aliasom.

```python
from module_name import name1 as alias1, name2 as alias2
```

- **Izjava za import sa zvezdama (from modul import `*`)**: Zvezda mu govori Python da želiš da sve prebaci u taj modul, ali želiš da to uradi tako da ne moraš da koristiš ime modula kao prefiks.

```python
from module_name import *
```

Na primer, ako koristite ovo da učitate modul `math`, moći ćete pozvati bilo koju funkciju navedenu u tom modulu bez prikazivanja naziva modula kao prefiksa.

```python
from math import *
print(sqrt(36))  # 6.0
```

Međutim, ovo se obično ne preporučuje jer može uzrokovati kolizije imena i otežati određivanje odakle ta imena potiču.

## `if __name__ == '__main__'`

Kada se datoteka Python izvršava direktno, Python postavlja vrednost ovog parametra na niz karaktera `"__main__"`. Ali ako se datoteka Python uvozi kao modul u drugi skript za Python, vrednost parametra `__name__` postavlja se na ime tog modula.

To je razlog da često naiđete na ove uslove u skriptama za Python. One imaju kod koji se želi pokrenuti **samo** ako je skripta za Python izvršena kao glavni program.

if __name__ == '__main__': 
    # Code
```

## Uobičajene greške u Python

- **SyntaxError**: Greška koju Python vraća kada vaš kod ne prati pravila sintakse. Na primer, kod `print("Hello there"` će izazvati sintaksnu grešku sa porukom `SyntaxError: '(' was never closed`, jer nedostaju zatvarajuće zagrade.
- **NameError**: Python vraća grešku `NameError` kada pokušavate da pristupite parametru ili funkciji koju niste definisali. Na primer, ako imate liniju `print(username)` u vašem kodu bez prethodno definisanog parametra `username`, dobićete NameError sa porukom `NameError: name 'username' is not defined`.
- **TypeError**: Ovo je greška koju Python vraća kada obavljate operaciju na nekompatibilnim tipovima podataka. Na primer, ako pokušavate da dodate niz znakova na broj, dobićete grešku `TypeError: can only concatenate str (not "int") to str`.
- **IndexError**: Dobićete grešku `IndexError` ako pokušate da pristupite indeksu koji ne postoji u listi ili bilo kojoj drugoj sekvenci, kao što su tuple i niz znakova. Na primer, u nizu znakova `Hello world`, indeks poslednjeg znaka je `10`. Ako nastavite da pristupate znakovima na ovaj način, `greet = "hello world"; print(greet[11])`, dobićete grešku sa porukom `IndexError: string index out of range`.
- **AttributeError**: Python vraća ovu grešku kada pokušavate da koristite metod ili atribut koji ne postoji na objektu tog tipa. Na primer, pozivanje `.append()` na nizu znakova kao što je `"hello".append("!")` vodi do greške sa porukom `AttributeError: 'str' object has no attribute 'append'`.

## Efikasne metode za ispravljanje grešaka u Python

- **Korišćenje funkcije `print`**: Unos stanja za `print()` u različitim delovima vašeg koda tokom debagovanja pomaže vam da vidite vrednosti promenljivih i kako teče kod.
- **Korišćenje ugrađenog debugger-a Python (`pdb`)**: Python pruža modul `pdb` za debagovanje. To je deo standardne biblioteke Python, pa je uvek dostupan za korišćenje. Sa `pdb`, možete postaviti *breakpoint* (tačku za pauziranje) na funkciju `set_trace()` kako biste započeli korak po korak izvršavanje koda i interaktivno pregledali promenljive.
- **Korišćenje alata za debagovanje u IDE-u**: Mnogi razvojni okruženja (IDE) i editori koda kao što su Pycharm i VS Code nude alate za debagovanje sa *breakpoints*-ovima, korak po korak izvršavanjem, inspekcijom promenljivih i drugim funkcijama za debagovanje.

## Rešavanje grešaka

- **`try...except`**: Ovo se koristi za izvršavanje dela koda koje može izazvati grešku. Blok `try` je mesto gde očekujete da će doći do greške, dok blok `except` hvata specifičnu grešku i izvršava se ako ta greška nastane. Evo primera:

  ```py
  try:
    print(22 / 0)
  except ZeroDivisionError:
    print('You can\'t divide by zero!')
    # You can't divide by zero!
  ```

Takođe možete povezati više delova `except` kako biste obradili više vrsta grešaka:

  ```py
  try:
    number = int(input('Enter a number: '))
    print(22 / number)
  except ZeroDivisionError:
    print('You cannot divide by zero!')
    # You cannot divide by zero! prints when you enter 0
  except ValueError:
    print('Please enter a valid number!')
    # Please enter a valid number! prints when you don't enter an integer
  ```

- **`else` na `finally`**: Ove sekcije dodaju `try...except`. Ako ne dođe do greške, sekcija `else` se izvršava. Sekcija `finally` se izvršava uvek, bez obzira na to da li je došlo do greške ili ne.

  ```py
  try:
    result = 100 / 4
  except ZeroDivisionError:
    print('You cannot divide by zero!') # This will not run
  else:
    print(f'Result is {result}') # Result is 25.0
  finally:
    print('Execution complete!') # Execution complete!
  ```

- **Karakteristika grešaka**: Ovo vam omogućava da dobijete samu grešku radi bolje korekcije i direktnog štampanja poruke o grešci. Da biste dobili karakteristiku grešaka, morate koristiti ključnu reč `as`. Evo primera:

  ```py
  try:
      value = int('This will raise an error')
  except ValueError as e:
      print(f'Caught an error: {e}')
      # Caught an error: invalid literal for int() with base 10: 'This will raise an error'
  ```

- **Izjava za `raise`**: Ovo vam omogućava da ručno bacite grešku. Možete ga koristiti za bacanje greške kada se ispuni određeni uslov. Evo primera:

  ```py
  def divide(a, b):
      if b == 0:
          raise ZeroDivisionError('You cannot divide by zero')
      return a / b
  ```## Indikacija greške

- Izjava za `raise` takođe je važna kada kreirate svoje prilagođene greške, jer ga možete koristiti da bacite izuzetak sa specifičnom porukom. Evo primera toga:

  ```py
  class InvalidCredentialsError(Exception):
      def __init__(self, message="Invalid username or password"):
          self.message = message
          super().__init__(self.message)

  def login(username, password):
      stored_username = "admin"

  ```

      print(f"Login failed: {e}")
  else:
      print(message)

  # successful login attempt
  try:
      message = login("admin", "password123")
  except InvalidCredentialsError as e:
      # This block is not executed because the login was successful

  else:
      # The else block runs if the 'try' block completes without an exception

  ```

Izjava `raise` se takođe može koristiti zajedno sa glavnom rečju `from` za povezivanje grešaka, prikazujući vezu između različitih grešaka:

## Putevi za dunder (za magiju)

- **Definicija**: Specijalne putanje koje počinju i završavaju sa dve podvučnice (primer, `__init__`, `__len__`, `__str__`, `__eq__`). Python koristi ove interne za ugrađene operacije.

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)

```

- **Matematičke operacije kao sabiranje, oduzimanje, množenje, deljenje i druge**. `__add__()` se koristi za sabiranje, `__sub__()` za oduzimanje, `__mul__()` za množenje, i `__truediv__()` za deljenje.

  - **Operacije poređenja kao jednako, manje od, veće od i druge**. `__eq__()` se koristi za proveru jednakosti, `__lt__()` za manje od, `__gt__()` za veće od, itd.
        
  - **Iterativne operacije kao činjenje nečega ponovljivim i pomeranje kroz elemente**. `__iter__()` se koristi za vraćanje iteracionog elementa, a `__next__()` za dobijanje sledećeg elementa.

class Cart:

## Šta je instalacija?

- **Enkapsulacija**: Kombinovanje atributa i metoda objekta u jednu jedinicu. Omogućava skrivanje internog stanja objekta iza jednostavnog skupa javnih atributa i metoda koje funkcionišu kao vrata. Iza tih vrata nalaze se privatni atributi i metode koji kontrolišu kako se podaci menjaju i ko može da ih vidi.
- **Primer enkapsulacije**: Ako želite da pratite stanje novčanika, dozvoljavate deponovanje i povlačenje novca, ali ne želite da bilo ko direktno pristupi stanju novčanika:

```py
class Wallet:
   def __init__(self, balance):
       self.__balance = balance # Private attribute

   def deposit(self, amount):
       if amount > 0:
           self.__balance += amount # Add to the balance safely

   def withdraw(self, amount):
       if 0 < amount <= self.__balance:
           self.__balance -= amount # Remove from the balance safely

account = Wallet(500)
print(account.__balance) # AttributeError: 'Wallet' object has no attribute '__balance'
```

- **Razlika između postavljanja modifikatora pristupa u jednom i dva reda za svojstvo**: Postavljanje modifikatora pristupa u jednom redu za svojstvo i metodu znači da je namenjeno za interno korišćenje. Ovo predstavlja ugovor, a ne sprečava pristup svojstvu izvana. Postavljanje modifikatora pristupa u dva reda za svojstvo i metodu potpuno onemogućava pristup izvan njihove klase.

## Šta su primaoci i pošiljaoci?

- **Wapokeaji na wapezaji**: Metode koje vam omogućavaju da kontrolišete kako su svojstva klase dostupna i modifikovana. Dobijate vrednost za wapokeaji, a postavljate vrednost za wapezaji.
- **Sifa**: Oni povezuju wapokeaji i wapezaji, omogućavajući pristup podacima. Izvršavaju dodatnu logiku iza scene kada dobijate, postavljate ili brišete vrednost.
- **Zašto sifa umesto metode**: Svojstva se koriste umesto metoda radi bolje čitljivosti i čistog koda. Omogućavaju vam da dobijate vrednosti pomoću dot notacije, kao obična svojstva, bez zagrada.
- **Kreiranje wapokeaji**: Da kreirate wapokeaji, koristite dekorator `@property`. Evo wapokeaji koji dobija krug kruga:

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self): # A getter to get the radius
        return self._radius
  
    @property
    def area(self):  # A getter to calculate area
        return 3.14 * (self._radius ** 2)

my_circle = Circle(3)

print(my_circle.radius) # 3
print(my_circle.area) # 28.26
```

Korišćenje self.radius unutar `__init__()` osigurava da programeri funkcionišu tokom kreiranja objekta, tako da se nevažeći vrednosti radijusa uhvate odmah.

- **Kreiranje komponente**: Da biste kreirali komponentu koja će postaviti krug, morate navesti drugu putanju sa istim imenom i koristiti `@<property_name>.setter` na njoj:

```py
class Circle:
    def __init__(self, radius):
        self.radius = radius # Calling the setter

    @property
    def radius(self):  # A getter to get the radius
        return self._radius

    @radius.setter
    def radius(self, value):  # A setter to set the radius
        if value <= 0:
            raise ValueError('Radius must be positive')
        self._radius = value

my_circle = Circle(3)
print('Initial radius:', my_circle.radius) # Initial radius: 3

my_circle.radius = 8
print('After modifying the radius:', my_circle.radius) # After modifying the radius: 8
```

- **Kako Python obrađuje primaoca i pošiljaoce**: Kada definiš primaoca i pošiljaoce, Python automatski poziva svaki put kada koristiš standardnu sintaksu svojstva kao što je ovo:

```py
my_circle.radius # This will call the getter
my_circle.radius = 4 # This will call the setter
```

Unutar provajdera, nije prikladno navesti ime samog atributa jer će to uzrokovati `RecursionError`.

- **Stripper**: Nakon postavljanja i dohvaćanja vrednosti za pošiljaoca i primaoca, možete kontrolisati kako se briše pomoću `deleter`. Stripper izvršava specifičnu logiku kada koristite izjavu `del` na svojstvu. Za kreiranje stripper-a, koristite dekorator `@<property_name>.deleter`.

```py
  # Deleter
    @radius.deleter
    def radius(self):
        print("Deleting radius...")
        del self._radius
```

## Šta je nasleđivanje i kako pomaže u ponovnom korišćenju koda?

- **Nasleđivanje**: Proces u kojem podklasa koristi svojstva i metode roditeljske klase. Nasleđivanje pomaže ponovnoj upotrebi koda, pruža jasnu hijerarhiju i omogućava modifikovanje ponašanja bez ponovnog pisanja koda. Kada se implementira nasleđivanje, podklasa preuzima ime roditeljske klase:

```py
class Parent:
    # Parent attributes and methods

class Child(Parent):
    # Child inherits, extends, and/or overrides where necessary
```

- **Direktno i višestrano nasleđe**: Kada dete nasleđuje osobine i putanje od jednog roditelja, kao što vidite gore, taj proces se naziva **direktno nasleđe**. Kada dete nasleđuje osobine i putanje od više od jednog roditelja, to je **višestrano nasleđe**. Sintaksa mu je ovakva:

```py
class Parent:
    # Attributes and methods for Parent

class Child:
    # Attributes and methods for Child

class GrandChild(Parent, Child):
    # GrandChild inherits from both Parent and Child
    # GrandChild can combine or override behavior from each
```

- **Metoda za `super()`**: Metoda koja vam omogućava da pozovete metodu iz roditeljske klase, kada klasa ima drugačiju implementaciju te metode, ili dodaje tu metodu, bez potrebe za ponovnim pisanjem koda.

## Šta je polimorfizam i kako pomaže u táiспользовању koda?

```py
class A:
   def action(self): ...

class B:
   def action(self): ...

class C:
   def action(self): ...

Class().method()  # Works for A, B, or C
```

- **Polimorfizam zavisno od nasleđa**: Roditelj pruža putanju, a svaka klasa deteta je modifikuje u zavisnosti od njihove upotrebe.

## Šta je promena imena i kako funkcioniše?

- **Promena imena**: Proces u kojem Python unutar sebe menja ime svojstva koje ima dvolinijski prefiks dodavanjem novog reda i imena klase kao prefiksa, menjajući `__attribute` u `_ClassName__attribute`.
- **Svrha promene imena**: Glavna svrha je sprečavanje slučajnog nasleđivanja svojstava i metoda prilikom korišćenja nasleđivanja. Evo koda koji ovo olakšava za razumevanje:

```py
class Parent:
    def __init__(self):
        self.__data = 'Parent data'

class Child(Parent):

        super().__init__()

c = Child()

## Šta je sažimanje i kako pomaže u strukturiranju složenih sistema?

- **Apstrakcija**: Programski koncept u kojem su kompleksni detalji implementacije objekta ili sistema skriveni, a prikazuju se samo ključne karakteristike. U Python i drugim programskim jezicima, apstrakcija pojednostavljuje složene sisteme povećanjem ponovne upotrebljivosti.
- **Primer apstrakcije**: Dobar primer apstrakcije u svakodnevnom životu je automobil koji vam omogućava da koristite samo točkove, pedale i menjač bez znanja kako rade motor ili kočnice.
- **Kako Python implementira apstrakciju**: Python vrši apstrakciju putem modula `abc`. Modul pruža klasu `ABC` (apstraktna bazna klasa) i dekorator `@abstractmethod`. Apstraktna bazna klasa (ABC) definiše zajedničke metode i svojstva koje izvedene klase moraju implementirati. Ne može se direktno instancirati.
- **Kako je apstraktna metoda definisana**: Apstraktna metoda je definisana sa `@abstractmethod` i mora biti prepisana u izvedenim klasama, čak i ako ima podrazumevanu implementaciju. Osnovna sintaksa apstrakcije je sledeća:

```py
from abc import ABC, abstractmethod

# Define an abstract base class
class AbstractClass(ABC):
    @abstractmethod
    def abstract_method(self):
        pass

# Concrete subclass that implements the abstract method
class ConcreteClassOne(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassOne')

# Another concrete subclass
class ConcreteClassTwo(AbstractClass):
    def abstract_method(self):
        print('Implementation in ConcreteClassTwo')
```

## Algoritmi u Big O notaciji

- **Algoritmi**: Skup jasnih instrukcija za rešavanje problema ili izvršavanje zadatka. Algoritmi moraju biti kompletni sa konačnim brojem koraka, a svaki korak mora biti tačan i neuključiv.

- **Big O notacija**: Opisuje najgori slučaj performansi, ili stopu rasta, algoritma kada veličina ulaza raste. Razmatra kako rast potrošnje resursa u odnosu na veličinu ulaza, zanemarujući konstante faktore i niže redove članova.

### Normalna kompleksnost vremena

- **O(1) - Konstantno vreme**: Algoritmi zahtevaju isto vreme bez obzira na veličinu ulaznih podataka.

```python
def check_even_or_odd(number):
    if number % 2 == 0:
        return 'Even'
    else:
        return 'Odd'
```

- **O(log n) - Logaritamsko vreme**: Vreme raste polako kako raste ulaz. Često se koristi u algoritmima koji ponavljajuće smanjuju veličinu problema za polovinu (kao što je Binarna pretraga).

- **O(n) - Linearno vreme**: Vreme izvršavanja raste proporcionalno veličini ulaza.

```python
for grade in grades:
    print(grade)
```

- **O(n log n) - Logaritamsko vreme**: Tipična složenost efikasnih algoritama sortiranja kao što su Merge Sort i Quick Sort.

- **O(n²) - Kvadratično vreme**: Vreme izvršavanja raste kvadratno. Često se javlja u ugnježdene petlje.

```python
for i in range(n):
    for j in range(n):
        print("Hello, World!")
```

### Kompleksnost prostora

- **O(1) - Konstantni prostor**: Algoritam koristi konstantnu količinu memorije bez obzira na veličinu ulaza.
- **O(n) - Linearni prostor**: Potrošnja memorije raste proporcionalno veličini ulaza.
- **O(n²) - Kvadratni prostor**: Potrošnja memorije raste kvadratno u odnosu na veličinu ulaza.

## Metode za rešavanje problema

- **Razumevanje problema**: Pročitajte izjavu o problemu nekoliko puta. Identifikujte ulaz, očekivani rezultat i način na koji se ulaz pretvara u rezultat.

- **Pseudokod**: Opis logike algoritma na visokom nivou koja ne zavisi od bilo kog jezika. Koristi uobičajen jezik pomešan sa programskim strukturama kao što su `IF`, `ELSE`, `FOR`, `WHILE`.

```md
GET original_string
SET reversed_string = ""
FOR EACH character IN original_string:
  ADD character TO THE BEGINNING OF reversed_string
DISPLAY reversed_string
```

- **Izuzetni slučajevi**: Specifičan, validan unos koji se dešava na granicama onoga što algoritam treba da obrađuje. Uvek razmatrajte i testirajte izuzetne slučajeve.

## Redovi podataka

- **Trajno polje**: Ima veličinu postavljenu tokom inicijalizacije. Komponente se skladište u susednim memorijskim oblastima. Veličina ne može biti promenjena tokom izvršavanja programa.

- **Dinamički nizovi**: Mogu rasti ili smanjiti se automatski tokom izvršavanja programa. On upravlja promenama veličine tako što automatski kopira u veći niz kada je to potrebno.

### Alati za Python (Redovi za izmene)

```python
numbers = [3, 4, 5, 6]

# Access elements
numbers[0]  # 3

# Update elements
numbers[2] = 16

# Add elements
numbers.append(7)
numbers.insert(3, 15)  # Insert at specific index

# Remove elements
numbers.pop(2)  # Remove at specific index
numbers.pop()   # Remove last element
```

### Kompleksnost vremena za slojeve promena

- **Pristup**: O(1)
- **Ubacivanje na kraj**: O(1) prosečno, O(n) vreme skaliranja veličine
- **Ubacivanje u sredinu**: O(n)
- **Brisanje**: O(n) za sredinu, O(1) na kraju

## Ustani

- **Stack**: Struktura podataka tipa Last-In, First-Out (LIFO). Elementi se dodaju i brišu samo o vrhu.

- **Operacija pusha**: Dodaje element na vrh steka. Vremenska složenost: O(1).

- **Operacija uklanjanja**: Uklanja element sa vrha steka. Vremenska složenost: O(1).

```python
# Using Python list as stack
stack = []

# Push operations
stack.append(1)
stack.append(2)
stack.append(3)

# Pop operations
top_element = stack.pop()  # Returns 3
```

## Foleni

- **Red**: Struktura podataka tipa First-In, First-Out (FIFO). Elementi se dodaju sa zadnje strane i uklanjaju sa prednje strane.

- **Operacija dodavanja u red**: Dodavanje elementa na zadnji kraj reda. Vremenska složenost: O(1).

- **Operacija uklanjanja sa prednje strane koljke**: Uklanjanje elementa sa prednje strane koljke. Vremenska složenost: O(1).

```python
from collections import deque

# Using deque for efficient queue operations
queue = deque()

# Enqueue operations
queue.append(1)
queue.append(2)
queue.append(3)

# Dequeue operations
first_element = queue.popleft()  # Returns 1
```

## Povezane liste

- **Povezana lista**: Linearna struktura podataka gde svaki čvor sadrži podatke i referencu ka sledećem čvoru. Čvorovi su povezani kao lanac.

### Samostalni povezani liste

- **Struktura**: Svaki čvor ima podatke i jednu referencu ka sledećem čvoru.
- **Traversiranje**: Može se kretati samo napred, od glave do repa.
- **Čvor glave**: Prvi čvor u listi, obično je jedini čvor dostupan direktno.
- **Čvor repa**: Poslednji čvor u listi, prikazuje `None`.

### Operacije vremenske složenosti

- **Ubacivanje na početak**: O(1)
- **Ubacivanje na kraj**: O(n) - mora se proći do kraja
- **Ubacivanje u sredinu**: O(n) - mora se proći do pozicije
- **Brisanje sa početka**: O(1)
- **Brisanje sa kraja**: O(n) - mora se proći da bi se dobio prethodni čvor
- **Brisanje u sredini**: O(n) - mora se proći da bi se pronašao čvor

### Dvostruke liste

- **Struktura**: Svaki čvor ima podatke i dve reference: sledeći čvor i prethodni čvor.
- **Traversiranje**: Može se kretati u oba pravca.
- **Memorija**: Zahteva više memorije nego samo spojene liste zbog dodatne reference.

## Mapovi za hash na skupu

### Karte i karte za hash

- **Mapa (Apstraktni tip podataka)**: Upravlja kolekcijama parova ključ-vrednost. Svaki ključ mora biti jedinstven, ali vrednosti mogu se ponavljati.

- **Hash mapa**: Stvarno implementiranje ADT mape korišćenjem metode hasovanja (hashing). Koristi funkciju hasovanja da kreira vrednosti hasa za ključeve, koje određuju mesto skladištenja u osnovnom nizu.

### Rečnici za Python (Mapa za hash)

```python
# Creating dictionaries
my_dictionary = {
    "A": 1,
    "B": 2, 
    "C": 3
}

# Alternative creation
my_dictionary = dict(A=1, B=2, C=3)

# Access and modify
value = my_dictionary["A"]  # 1
my_dictionary["A"] = 4      # Update value
del my_dictionary["A"]      # Remove key-value pair

# Check membership
"C" in my_dictionary

# Get keys, values, items
my_dictionary.keys()
my_dictionary.values()
my_dictionary.items()
```

### Vremenska složenost za hash mape

- **Prosečan slučaj**: O(1) za ubacivanje, dohvaćanje, brisanje
- **Najgori slučaj**: O(n) kada postoji mnogo kolizija hasha

### Skup

- **Set:** Neuređene kolekcije jedinstvenih elemenata. Nema duplikata, niti se održava specifičan redosled.

- **Samo nepromenljivi elementi**: Setovi mogu sadržati nepromenljive tipove podataka (broj, niz karaktera, tuple) jer vrednosti hešovanja moraju ostati konstantne.

```python
# Creating sets
numbers = {1, 2, 3, 4}
empty_set = set()  # Must use set(), not {}

# Add and remove elements
numbers.add(5)
numbers.remove(4)      # Raises KeyError if not found
numbers.discard(4)     # No error if not found

# Set operations
set_a = {1, 2, 3, 4}
set_b = {2, 3, 4, 5, 6}

# Union, intersection, difference, symmetric difference
set_a.union(set_b)                    # or set_a | set_b
set_a.intersection(set_b)             # or set_a & set_b
set_a.difference(set_b)               # or set_a - set_b
set_a.symmetric_difference(set_b)     # or set_a ^ set_b

# Subset and superset checks
set_a.issubset(set_b)
set_a.issuperset(set_b)
set_a.isdisjoint(set_b)

# Membership testing
5 in numbers
```

### Vremenska složenost za skup

- **Prosečan slučaj**: O(1) za dodavanje, brisanje, proveru članstva
- **Najgori slučaj**: O(n) usled kolizija haša

## Kolizije haša

- **Kolizija hasha**: Dešava se kada dva različita ključa generišu istu vrednost hasha.

- **Metode rešavanja kolizija**:
  - **Lančasto povezivanje (Chaining)**: Svaki red prikazuje spojenu listu koja čuva sve elemente sa istom vrednošću hasha.
  - **Otvorena adresa (Open Addressing)**: Pronađi sledeći dostupan unos koristeći prethodno definisan niz.

## Kada koristiti svaku strukturu podataka

- **Lista**: Kada vam je potreban redosni pristup, definisana struktura i ne znate veličinu unapred
- **Stack**: Za LIFO operacije (funkcija pop, evaluacija izraza, povratak nazad)
- **Red (Queue)**: Za FIFO operacije (planiranje zadataka, pretraživanje po širini/BFS)
- **Povezana lista (Linked List)**: Kada je potrebno često dodavanje/brisanje na početku, nepoznata veličina, nije potreban nasumični pristup
- **Hash mape (Hash Maps)**: Za brzi pristup parovima ključ-vrednost, brojanje događaja, keširanje (cache)
- **Set (Skup)**: Za proveru jedinstvenosti, operacije skupa za izračunavanje, uklanjanje duplikata

## Algoritmi za pretraživanje

Algoritmi za pretraživanje omogućavaju pronalaženje cilja unutar određene liste stavki.

U nauci o računarstvu, postoje dva algoritma pretraživanja koja morate znati. To su **linearno pretraživanje** i **binarno pretraživanje**. Važno je razumeti razliku između ova dva algoritma i kada koristiti svaki od njih.

### Pretraživanje linije

- Linearno pretraživanje iterira kroz niz elemenata, proveravajući svaki element od početka sve dok se ne pronađe željeni element.
- Ako se željeni element pronađe, vraća se element koji je u nizu.
- Ako željeni element nije pronađen, vraća `-1`, što znači **nevažeći element** u mnogim programskim jezicima.
- Jer linearno pretraživanje proverava svaki element sve dok ne pronađe željeni, nije efikasno za velike nizove elemenata.
- Vremenska složenost linearnog pretraživanja je `O(n)` jer vreme potrebno za pretragu kroz niz raste sa redom i veličinom niza.
- Prostorna složenost linearnog pretraživanja je `O(1)` jer ne zahteva dodatni prostor za pretragu kroz niz.

### Binarna pretraga

- Binarna pretraga radi tako što deli listu elemenata na pola, i proverava da li je ciljna vrednost u sredini liste.
- Uslov za binarnu pretragu je da elementi u listi moraju biti u rastućem redosledu.
- Binarna pretraga je najefikasniji algoritam za pretraživanje kroz veliku listu elemenata jer deli listu na pola i ignoriše pol koji ne sadrži cilj.
- Ako se ciljni element pronađe u sredini liste, vraća se indeks tog ciljnog elementa.
- Ako element nije pronađen, algoritam proverava da li je cilj u levom ili desnom delu liste.
- Nastavlja da deli preostale delove liste na pola sve dok se ciljni element ne pronađe.
- Ako ciljni element na kraju nije pronađen u listi, vraća se `-1`.
- Vremenska složenost binarnog pretraživanja je `O(log n)` jer vreme potrebno za pretraživanje kroz listu raste logaritamički sa veličinom liste.
- Prostorna složenost binarnog pretraživanja je `O(1)` jer ne zahteva dodatni prostor za pretraživanje kroz listu.

### Kako se linearna pretraga razlikuje od binarne pretrage

- Binarna pretraga je efikasnija za velike liste elemenata u poređenju sa linearnom pretragom.
- Vremenska složenost linearne pretrage je `O(n)` jer vreme potrebno za pretraživanje kroz listu raste linearno sa veličinom liste.
- Vremenska složenost binarne pretrage je `O(log n)` jer vreme potrebno za pretraživanje kroz listu raste logaritamski sa veličinom liste.

## Algoritmi za sortiranje i metod razdvajanja-vladanja

U računarskoj nauci, podela i osvajanje je tehnika koja se koristi za razdvajanje problema na manje probleme kako bi bilo lakše da se reše. Rekurzija je metoda koja se često koristi u principu podele i osvajanja, i predstavlja moćnu strategiju koja se koristi za implementaciju mnogih efikasnih algoritama sortiranja kao što su *merge sort*.

### Merge sort

- Merge sort je sortirajući algoritam koji koristi pristup podelama i osvajanju (divide-and-conquer).
- Funkcioniše deljenjem niza rekursivno u sve manje nizove, sve dok svaki podniz ne sadrži samo jedan element.
- Nakon toga, te manji nizove ponovno spaja u sortirano poređenje.
- Vremenska složenost Merge sort-a je `O(n log n)` jer se niz ponavljajuće deli na pola `(log n)` i zatim spaja zajedno `(O(n))`.
- Prostorna složenost Merge sort-a je `O(n)` jer nije sortirajući algoritam koji se izvršava na mestu (in-place).

## Sažetak grafa

Graf je skup čvorova (nodi) povezanih ivicama (vezaima). Svaki čvor može biti povezan sa mnogo drugih čvorova, stvarajući mrežu. Različiti tipovi grafova su:

- Usmereni: Grafovi imaju smer (od jednog čvora ka drugom), često se prikazuju pravim linijama i strelicama.
- Nesmerni: Grafovi nemaju smer, prikazuju se jednostavnim linijama.
- Čvorovi: Svaki čvor ima labelu ili identifikator.
- Sa ciklusima: Imaju cikluse (putanja koja počinje i završava na istom čvoru).
- Bez ciklusa (DAG): Nemaju cikluse.
- Grafovi sa etiketama/labelama: Svaki graf ima labelu, koja se obično piše blizu odgovarajućeg grafa.
- Sa težinama: Grafovi imaju težinu (vrednost) povezanu s njima, koju je moguće koristiti za izvođenje matematičkih operacija.
- Diskonektovani/Razdvojeni: Imaju dva ili više čvorova koja nisu povezana sa bilo kojim grafom.

Graf se koristi u različitim aplikacijama kao što su mape, mreže, sistemi preporuka i rešavanje zavisnosti.

## Kroz graf

Ovo uključuje posetu svih čvorova u grafu. Dva glavna algoritma su:

- **Širokopredno pretraživanje (BFS)**
  - Koristi red.
  - Istražuje sloj po sloj.
  - Pronalazi najkraći put u neotežnjenim grafikama.

- **Dubinska pretraga (DFS)**
  - Koristi stos (ili rekurziju).
  - Istražuje celu granu pre povratka (backtrackinga).
  - Pogodan je za otkrivanje ciklusa i pronalaženje putanja.

## Reprezentacija grafa

Grafik se može predstaviti na dva glavna načina:

- **Lista susjedstva**
  - Svaki čvor ima listu svojih susjeda.
  - Koristi malo memorije za razrijedke grafove.
  - Lako je iterirati kroz susjedima.

- **Matrica susjednosti**
  - Dvodimenzionalna matrica gdje redovi i kolone predstavljaju čvorove.
  - Koristi mnogo prostora za velike grafove.
  - Brzo provjeriti da li postoji veza između dva čvora.

## Drveće

Drvo je specijalna vrsta grafa koja nema cikluse i povezana je. Glavne karakteristike su:

- Nema ciklusa ili je cikličan (putanja gde su početni i krajnji čvor isti).
- Mora biti povezan (svaki čvor može biti dostignut iz bilo kog drugog čvora).

### Uobičajene vrste drveća

Uobičajene vrste drveća su:

- Binarna stabla
  - Svaki čvor ima najmanje dva potomka, levi i desni potomak.

- Binarna stabla za pretraživanje (BST)
  - Binarno stablo gde je svaki levi sin manji od svog roditelja, a svaki desni sin veći od svog roditelja.

## Miti o riječima (pokušaji)

Takođe se nazivaju stablo prefiksa i koriste se za skladištenje skupova sekvenci slova, gde svaki čvor predstavlja slovo.

Prefiks se deli samo jednom, i efikasno radi za zadatke kao što su dovršavanje teksta i provera pravopisa.

Operacije pretrage i umetanja imaju vremensku složenost od O(L), gde je L dužina niza znakova.

## Funkcionalnosti visokog prioriteta

Red sa prioritetom je tip apstraktnih podataka gde svaki element ima prioritet.

Red na staku uzima samo redosled unosa, dok prioritetni red koristi prioritet elemenata.

Standardni redovi prate FIFO (Prvo uđe, prvo izađe), a stike prate LIFO (Poslednje uđe, prvo izađe). Međutim, u redovima sa prioritetom, elementi sa visokim prioritetom obrađuju se prvi bez obzira na redosede unosa.

## Snop

To je struktura podataka specifičnog stabla sa posebnim svojstvom koje se zove klasterovo svojstvo.

Svojstvo grafa određuje odnos između roditeljskog čvora i dječijeg čvora. Postoji dve vrste grafova:

- Max-Heap
  - Vrednost svakog roditeljskog čvora je veća ili jednaka vrednostima njegovih dece.
  - Najveći element se nalazi na korenu.

- Minimalni heap (Min-Heap)
  - Vrednost svakog roditeljskog čvora je manja ili jednaka vrednosti njegovih potomaka.
  - Najmanji element se nalazi na korenu.

### Primer modula za `heapq` od Python

```py
import heapq

# Create empty heap
my_heap = []

# Insert elements
heapq.heappush(my_heap, 9)
heapq.heappush(my_heap, 3)
heapq.heappush(my_heap, 5)
print(my_heap) # [3, 9, 5]

# Remove smallest element
print(heapq.heappop(my_heap))  # 3
print(my_heap) # [5, 9]

# Push + Pop in one step
print(heapq.heappushpop(my_heap, 7)) # 5
print(my_heap) # [7, 9]

# Transform list into heap
nums = [5, 7, 3, 1]
heapq.heapify(nums)
```

### Korišćenje prioriteta

```py
my_heap = []
heapq.heappush(my_heap, (3, "A"))
heapq.heappush(my_heap, (2, "B"))
heapq.heappush(my_heap, (1, "C"))

# Removes lowest number = highest priority
print(heapq.heappop(my_heap))  # (1, "C")
```

## Uvod u program promena

- **Definicija**: Dinamičko programiranje je algoritamska tehnika koja rešava složene probleme tako što ih razlaže na manje podprobleme i skladišti rezultate kako bi se izbegli ponovni proračuni.
- **Preklapajući podproblemi**: Preklapajući podproblemi su manji problemi koji se često javljaju prilikom rešavanja većeg problema. Umesto da se stalno ponovo računa, skladištimo njihova rešenja.
- **Optimalna struktura problema**: Optimalno rešenje celog problema poseduje optimalna rešenja za manje podprobleme. To znači da možemo izgraditi optimalno rešenje kombinovanjem optimalnih rešenih manjih delova.

## Rešenje za program promena

- **Memoization (Top-down pristup)**: Memoizacija čuva rezultate skupih poziva funkcija i vraća sačuvane rezultate kada se ponovo pojavi isti ulaz.

```py
def climb_stairs_memo(n, memo={}):
    """Dynamic programming with memoization"""
    # Check if we've already calculated this value
    if n in memo:
        return memo[n]  # Return cached result - O(1) lookup!
    
    # Base cases
    if n <= 2:
        return n
    
    # Calculate once and store in memo for future use
    memo[n] = climb_stairs_memo(n-1, memo) + climb_stairs_memo(n-2, memo)
    return memo[n]
```

- **Tabulacija (Metoda od dna ka vrhu)**: Tabulacija izgrađuje rešenja od najnižeg nivoa, popunjavajući tabelu rešenjima za manje probleme.

```py
def climb_stairs_tabulation(n):
    """Dynamic programming with tabulation"""
    if n <= 2:
        return n
    
    # Create array to store results for all steps from 0 to n
    dp = [0] * (n + 1)
    dp[1] = 1  # 1 way to reach step 1
    dp[2] = 2  # 2 ways to reach step 2
    
    # Build up the solution iteratively
    for i in range(3, n + 1):
        # Ways to reach step i = ways to reach (i-1) + ways to reach (i-2)
        dp[i] = dp[i-1] + dp[i-2]
    
    return dp[n]
```
