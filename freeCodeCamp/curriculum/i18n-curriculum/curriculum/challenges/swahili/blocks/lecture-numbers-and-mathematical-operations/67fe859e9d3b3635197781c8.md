# --description--

Čeli brojevi i decimalni brojevi su glavni tipovi numeričkih podataka u Python. Korišćenjem njih, možete sačuvati numeričke podatke i izvršiti matematičke operacije.

Istražićemo šta su celi brojevi i decimalni brojevi, kako da se izvode matematički proračuni koristeći ih, i nekoliko metoda Python koje pružaju načine za rad sa oba.

Cjelobrojni brojevi su cjelobrojni brojevi bez decimalnih tačaka, bilo da su pozitivni ili negativni:

```python
my_int_1 = 56
my_int_2 = -4

print(type(my_int_1)) # <class 'int'>
print(type(my_int_2)) # <class 'int'>
```

Ovo je kako izvršiš operaciju sabiranja za ceo broj:

```python
my_int_1 = 56
my_int_2 = 12

sum_ints = my_int_1 + my_int_2
print('Integer Addition:', sum_ints) # Integer Addition: 68
```

Ovo je kako radiš operaciju izvlačenja kompletnog broja:

```python
my_int_1 = 56
my_int_2 = 12

# Subtraction
diff_ints = my_int_1 - my_int_2
print('Integer Subtraction:', diff_ints) # Integer Subtraction: 44
```

Ovo je kako se vrši operacija množenja za cele brojeve:

```python
my_int_1 = 12
my_int_2 = 4

# Multiplication
product_ints = my_int_1 * my_int_2
print('Integer Multiplication:', product_ints) # Integer Multiplication: 48
```

Ovo je kako izvršavate operaciju deljenja za celobrojni broj:

```python
my_int_1 = 56
my_int_2 = 12

# Division
div_ints = my_int_1 / my_int_2
print('Division:', div_ints) # Division: 4.666666666666667
```

Brojevi sa decimalom su pozitivni ili negativni brojevi koji imaju decimalnu tačku, kao što su `3.14`, `-0.5`, ili `0.0`.

```python
my_float_1 = -12.0
my_float_2 = 4.9

print(type(my_float_1)) # <class 'float'>
print(type(my_float_2)) # <class 'float'>
```

Ovako se izvodi operacija sabiranja decimalnih brojeva:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_addition = my_float_1 + my_float_2
print('Float Addition:', float_addition) # Float Addition: 17.4
```

Ovo je kako se izvodi operacija sa decimalnim brojevima:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_subtraction = my_float_2 - my_float_1
print('Float Subtraction:', float_subtraction) # Float Subtraction: 6.6
```

Evo kako se izvodi operacija množenja decimalnih brojeva:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_multiplication = my_float_2 * my_float_1
print('Float Multiplication:', float_multiplication) # Float Multiplication: 64.80000000000001
```

Na ovakav način vršite operaciju deljenja decimalnih brojeva:

```python
my_float_1 = 5.4
my_float_2 = 12.0

float_division = my_float_2 / my_float_1
print('Float Division:', float_division) # Float Division: 2.222222222222222
```

Ako dodate cijeli broj i decimalni broj, rezultat će biti automatski konvertovan u decimalni broj:

```python
my_int = 56
my_float = 5.4

sum_int_and_float = my_int + my_float

print(sum_int_and_float) # 61.4
print(type(sum_int_and_float)) # <class 'float'>
```

Ovo je tačno i za druge osnovne matematičke operacije, kao što su oduzimanje, množenje i deljenje. Ako kombinujete cele brojeve i decimalne brojeve, Python će vratiti decimalni broj kao rezultat.

Takođe možete raditi složenije aritmetičke proračune, kao što je pronalaženje ostatka od dva broja koristeći operator modulo, celobrojno deljenje i moć za oba broja, celobrojne i decimalne brojeve.

Operater modula (`%`) vraća ostatak kada se vrednost sa leve strane deli sa vrednošću sa desne strane:

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

mod_ints = my_int_1 % my_int_2
mod_floats = my_float_2 % my_float_1

print('Integer Modulo:', mod_ints) # Integer Modulo: 8
print('Float Modulo:', mod_floats) # Float Modulo: 1.1999999999999993
```

Celobrojno deljenje deli dva broja i vraća najveći ceo broj koji ne prelazi rezultat. Ovo se radi korišćenjem operatera dvostruke kose crte (`//`):

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

floor_div_ints = my_int_1 // my_int_2
floor_div_floats = my_float_2 // my_float_1

print('Integer Floor Division:', floor_div_ints) # Integer Floor Division: 4
print('Float Floor Division:', floor_div_floats) # Float Floor Division: 2.0
```

Moć podiže brojeve do druge moći, i radi se korišćenjem operatera dve zvezdice (`**`):

```python
my_int_1 = 56
my_int_2 = 12

my_float_1 = 5.4
my_float_2 = 12.0

exp_ints = my_int_1 ** my_int_2
exp_floats = my_float_1 ** my_float_2

print('Integer Exponentiation:', exp_ints) # Integer Exponentiation: 951166013805414055936
print('Float Exponentiation:',  exp_floats) # Float Exponentiation: 614787626.1765089
```

Ponekad možete otkriti da rezultati operacija koje uključuju decimalne brojeve imaju više decimalnih cifara nego što ste očekivali. Na primer, zbir `0.1 + 0.2` je jednak `0.30000000000000004` umesto `0.3`.

Ovo se dešava zato što brojevi budu skladišteni u binarnom formatu, a neki delovi ne mogu biti tačno predstavljeni u binaru. Kao rezultat toga, skladišti se kao konačna aproksimacija, jer deo `1/3` ne može biti predstavljen brojem decimalnih cifara i zaokružen je nakon određenog broja njegovih beskonačnih cifara (`0.33333...`).

Ovo uzrokuje male rotacione greške.

Python takođe pruža funkciju integrisanu za konverziju podataka brojeva ili sekvenca karaktera u cele ili decimalne brojeve.

Funkcija za `float()` vraća decimalni broj konstruisan iz datog broja:

```python
my_int_1 = 56
my_float_1 = float(my_int_1)

print(my_float_1)  # 56.0
print(type(my_float_1))  # <class 'float'>
```

Funkcija `int()` vraća pun broj sastavljen iz datog broja:

```python
my_float = 12.92563
my_int = int(my_float)

print(my_int)  # 12
print(type(my_int))  # <class 'int'>
```

Pia, možete koristiti ugrađene funkcije za konverziju niza karaktera u decimalni broj ili ceo broj:

```python
my_str_int = '45'
my_str_float = '7.8'

converted_int = int(my_str_int)
converted_float = float(my_str_float)

print(converted_int, type(converted_int))  # 45 <class 'int'>
print(converted_float, type(converted_float))  # 7.8 <class 'float'>
```

Postoji drugi način Python koji omogućava rad sa celim brojevima i decimalnim brojevima.

- `round()`: Zaokružuje broj na zadati broj decimalnih mesta. Podrazumevano, ova funkcija zaokružuje do najbližeg celog broja i vraća ceo broj bez decimalnih mesta:

```python
my_int_1 = 4.798
my_int_2 = 4.253

rounded_int_1 = round(my_int_1)
rounded_int_2 = round(my_int_2, 1)

print(rounded_int_1) # 5
print(rounded_int_2) # 4.3
```

- `abs()`: Vrati punu vrednost broja,

```python
num = -15

absolute_value = abs(num)
print(absolute_value) # 15
```

- `pow()`: podiže broj na drugu potenciju ili izračunava modularnu eksponencijaciju.

```python
result_1 = pow(2, 3)  # Equivalent to 2 ** 3
print(result_1)  # 8

result_2 = pow(2, 3, 5)  # (2 ** 3) % 5
print(result_2)  # 3
```
