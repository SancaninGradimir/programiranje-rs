# --description--

Arrow funkcija je najmoderniji način za pisanje funkcije u JavaScript. Pruža kraću sintaksu u poređenju sa tradicionalnom deklaracijom funkcija. Arrow funkcija se piše korišćenjem sintakse `=>`.

```js
const add = (a, b) => {
  return a + b;
}
```

U gornjem primeru, strelica funkcija ``add`` prima dva parametra ``a`` i ``b`` i vraća njihov zbir. Ključna reč ``return`` koristi se za vraćanje rezultata operacije sabiranja.

Ako funkcija sa strelicom ima samo jedan matematički izraz, zagrade `{}` i glavni naziv `return` mogu se izostaviti. Taj matematički izraz biće vraćen na indirektan način.

```js
const add = (a, b) => a + b;
```

U gornjem primeru, strelična funkcija `add` prima dva parametra `a` i `b` i vraća njihov zbir. Glavna reč `return` i vitičaste zagrade `{}` su ostavljene, a rezultat operacije sabiranja vraća se indirektno.
