# --interactive--

Algoritam Cascade je proces koji koristi pregledač da utvrdi koje će pravilo od CSS biti korišćeno kada postoji mnogo stilova koji ciljaju jedan element. Osigurava da se koristi najpogodniji stil, u skladu sa dobro definisanim setom pravila.

Proces počinje sa **vezom**. Pretraživač prvo filtrira sva pravila za CSS ili dobija ona koja se primenjuju na element o kojem se govori. Ovo uključuje poređenje selektora i uzimanje u obzir potencijalno primenljivih media upitnika (media queries).

Media query je CSS tehnika koja se koristi za primenu stilova na osnovu karakteristika uređaja ili prikaza, kao što su njegova širina, visina ili orijentacija.

Zatim, algoritmi uzimaju u obzir **poreklo i značaj**. CSS može poticati iz različitih izvora: bazni stilovi pregledača (user-agent), stilovi postavljeni od strane korisnika, i stilovi kreirani od strane autora (vi).

Nakon razmatranja porekla, algoritam procenjuje važnost svakog pravila, dajući prednost pravilima sa `!important`, koja nadjačavaju druga pravila bez obzira na njihovo poreklo.

Nakon filtriranja po poreklu i važnosti, algoritam proverava **specifičnost**. Kada se primenjuju dva pravila istog porekla i istog nivoa važnosti, primeniće se pravilo sa većom specifičnošću.

Specifičnost je mera koliko je selektor precizan, pri čemu precizniji selektori imaju prednost u odnosu na opštije selektore.

Na kraju, ako je sve ostalo isto, na snagu stupa **redosled pojavljivanja**. Kada dva pravila imaju istu specifičnost, primeniće se ono koje se poslednje pojavljuje u CSS-u.

Zbog toga redosled kojim pišete stilove može uticati na rezultat.

Tuchukulie mfano.

U HTML datoteci postoji jedan element pasusa. Zatim se u CSS-u nalaze dva pravila, od kojih svako cilja taj element pasusa.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>example paragraph</p>
```

```css
p {
  color: blue;
}

p {
  color: green; 
}
```

:::

Prvi zakon postavlja boju teksta svih pasusa na plavu, dok drugi zakon postavlja boju teksta svih pasusa na zelenu.

Pa, koja boja će biti korišćena? Boja zelena će biti primenjena za elemente paragrafa.

Uzimajući u obzir odnos, poreklo i važnost, posebne sposobnosti i vizuelni raspored, Algoritam Cascade osigurava da CSS vaš radi prediktivno, omogućavajući vam da dizajnirate veb stranice sa kompleksnijom i detaljnijom strukturom.
