# --interactive--

## Osnovi sekvence slova

- **Definicija**: String je sekvenca karaktera koja je obuhvaćena jednostrukim navodnicima, dvostrukim navodnicima ili backtick-ovima. String je osnovni tip podataka i on je nepromenljiv (immutable). Nepromenljiv znači da kada se string kreira, ne može biti izmenjen.
- **Pristup karakteru iz stringa**: Da biste pristupili karakteru unutar stringa, možete koristiti indeksiranje pomoću uglastih zagrada i uneti numerički indeks. Indeks je pozicija karaktera unutar stringa, i počinje od nule.

:::interactive_editor

```js
const developer = "Jessica";
console.log(developer[0]); // J
```

:::

- **`\n` (Znak za novi red)**: Možete kreirati novi red u sekvenci karaktera koristeći znak novog reda `\n`.

:::interactive_editor

```js
const poem = "Roses are red,\nViolets are blue,\nJavaScript is fun,\nAnd so are you.";
console.log(poem);
```

:::

- **Izbegavanje sekvence znakova**: Možete eskapirati znake u sekvenci karaktera postavljanjem kosog crta (`\`) ispred navodnika.

:::interactive_editor

```js
const statement = "She said, \"Hello!\"";
console.log(statement); // She said, "Hello!"
```

:::

## Template literals za interpolaciju vrednosti

- **Objašnjenje**: Placeholder karakteri su obuhvaćeni backtick oznakama (`). Omogućavaju lako upravljanje nizovima znakova, uključujući direktno ubacivanje parametara unutar niza znakova, što je funkcija poznata kao injekcija vrednosti.

:::interactive_editor

```js
const name = "Jessica";
const greeting = `Hello, ${name}!`; 
console.log(greeting); // "Hello, Jessica!"
```

:::

## ASCII, putanja za `charCodeAt()` i putanja za `fromCharCode()`

- **ASCII**: ASCII (American Standard Code for Information Interchange) je standard kodiranja znakova koji se koristi za predstavljanje osnovnih engleskih slova korišćenjem numeričkih vrednosti. Inicijalno treniranje inicijalizuje `charCodeAt()` i `fromCharCode()` koristeći ASCII primere.
- **Unicode**: Sekvenca znakova JavaScript koristi Unicode unutar sebe, posebno kodiranje UTF-16. Za prvih 128 znakova (osnovna latinska slova, brojevi i uobiđi simboli), vrednosti Unicode liče na ASCII kod. Zato što modeli zavisni od ASCII funkcionišu u JavaScript.
- **Metoda `charCodeAt()`**: Ova metoda vraća jedinicu koda UTF-16 znaka u datoj formati. Za osnovna latinska slova, ova vrednost odgovara ASCII kodu.

:::interactive_editor

```js
const letter = "A";
console.log(letter.charCodeAt(0));  // 65
```

:::

- **Njia ya `fromCharCode()`**: Ova putanja konvertuje ASCII kod u odgovarajuće slovo.

:::interactive_editor

```js
const char = String.fromCharCode(65);
console.log(char);  // A
```

:::

## Drugi uobičajeni načini za sekvencu slova

- **Funkcija `indexOf`**: Ova funkcija se koristi za pretraživanje podniza unutar niza karaktera. Ako se podniz pronađe, ``indexOf`` vraća indeks (ili poziciju) prvog pojavljivanja tog podniza. Ako se podniz ne pronađe, ``indexOf`` vraća -1, što ukazuje da pretraga nije uspela.

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.indexOf("fox")); // 16
console.log(text.indexOf("cat")); // -1
```:::

- **Funkcija `includes()`**: Ova funkcija se koristi za proveru da li niz znakova sadrži određenu podsekvencu. Ako je podsekvenca pronađena unutar niza znakova, ova funkcija vraća true. Inače, vraća false.

:::interactive_editor

```js
const text = "The quick brown fox jumps over the lazy dog.";
console.log(text.includes("fox")); // true
console.log(text.includes("cat")); // false
```

:::

- **Njia ya `slice()`**: Ova funkcija uzima deo niza znakova i vraća novi niz znakova, bez menjanja originalnog niza. Prihvata dva parametra: početno ograničenje i opcionalno krajnje ograničenje.

:::interactive_editor

```js
const text = "freeCodeCamp";
console.log(text.slice(0, 4));  // "free"
console.log(text.slice(4, 8));  // "Code"
console.log(text.slice(8, 12)); // "Camp"
```

:::

- **Metoda za `toUpperCase()`**: Ova metoda konvertuje sva slova u VELIKA SLOVA i vraća novu sekvencu znakova sa svim velikim slovima.

:::interactive_editor

```js
const text = "Hello, world!";
console.log(text.toUpperCase()); // "HELLO, WORLD!"
```

:::

- **Put za `toLowerCase()`**: Ova funkcija konvertuje sva slova u nizu karaktera u mala slova.

:::interactive_editor

```js
const text = "HELLO, WORLD!"
console.log(text.toLowerCase()); // "hello, world!"
```

:::

- **Metoda za `replace()`**: Ova metoda omogućava pretraživanje određene vrednosti (kao reči ili slova) u nizu karaktera i zamenjivanje je drugom vrednošću. Ova metoda vraća novi niz karaktera sa izmenama i ostavlja originalni niz nepromenjen jer nije dozvoljena promena niza karaktera JavaScript.

:::interactive_editor

```js
const text = "I like cats";
console.log(text.replace("cats", "dogs")); // "I like dogs"
```

:::

- **Put za `replaceAll()`**: Ovaj put omogućava pronalaženje svih pojavljivanja određene vrednosti (reči, karaktera ili obrasca) u nizu znakova i zamenjivanje je drugom vrednošću. Radi kao `replace()`, ali umesto da zaustavi nakon prvog pogađanja, ažurira svako pogađanje pronađeno u nizu znakova.

:::interactive_editor

```js
const text = "I love cats and cats are so much fun!";
console.log(text.replaceAll("cats", "dogs")); // "I love dogs and dogs are so much fun!"
```

:::

- **Putanja za `repeat()`**: Ova putanja se koristi za ponavljanje niza slova više puta, kako je navedeno.

:::interactive_editor

```js
const text = "Hello";
console.log(text.repeat(3)); // "HelloHelloHello"
```

:::

- **Metoda za `trim()`**: Ova metoda se koristi za uklanjanje praznih razmaka sa početka i kraja niza znakova.

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trim()); // "Hello, world!"
```

:::

- **Njia ya `trimStart()`**: Ova metoda uklanja prazne razmake sa početka (ili "početak") niza znakova.

:::interactive_editor

```js
const text = "  Hello, world!  ";
console.log(text.trimStart()); // "Hello, world!  "
```

:::

- **Njia ya `trimEnd()`**: Ovaj put uklanja prazan razmak sa kraja niza znakova.

:::interactive_editor

```js
const text = " Hello, world! ";
console.log(text.trimEnd()); // "  Hello, world!"
```

:::

- **Metoda `prompt()`**: Ova metoda `window` se koristi za dohvaćanje informacija od korisnika putem prozora za razgovor. Ova metoda prihvata dva argumenta. Prvi argument je poruka vidljiva unutar prozora za razgovor, obično podstiče korisnika da unese informacije. Drugi argument je podrazumevana vrednost koja je opcionalna i popuniće polje za unos na početku.

```js
const answer = window.prompt("What's your favorite animal?"); // This will change depending on what the user answers
```
