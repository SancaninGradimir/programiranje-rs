# --description--

U prethodnim lekcijama, naviknut ste da definišete funkciju koristeći glavni rečnik `def` kao što je ovo:

```py
def square(num):
    return num ** 2

print(square(4)) # 16
```

Ali kada radiš sa funkcijama višeg reda kao što su `map()` i `filter()`, možeš da koristiš anonimnu funkciju unutar jedne linije. Tu na scenu stupaju lambda funkcije.

Ovo je kako funkcija za `square()` izgleda kada se konvertuje u lambda funkciju:

```py
lambda num: num ** 2
```

Kao što je ranije pomenuto, lambda funkcije nemaju ime, pa ova funkcija više nije povezana sa imenom `square`. Lambda funkcije su odlične kada ih koristiš u funkcijama višeg reda, kao u ovom primeru:

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

U ovom primeru imamo listu brojeva i želimo da napravimo novu listu pozitivnih parnih brojeva. Zato prosleđujemo lambda funkciju kao jedan od argumenata funkciji `filter()` kako bismo dobili novu listu koja sadrži brojeve `2` i `4`.

Kada radiš sa lambda funkcijama, važno je da poznaješ dobre prakse. Na primer, nije dobra praksa da lambda funkciju dodeliš promenljivoj na ovaj način:

```py
numbers = [1, 2, 3, 4, 5]

square = lambda x: x ** 2
squared_numbers = list(map(square, numbers))
print(squared_numbers) # [1, 4, 9, 16, 25]
```

Time se gubi smisao korišćenja anonimne funkcije. U ovom slučaju treba da koristiš običnu funkciju, ovako:

```py
numbers = [1, 2, 3, 4, 5]

def square(num):
    return num ** 2

squared_numbers = list(map(square, numbers))
print(squared_numbers) # [1, 4, 9, 16, 25]
```

Takođe bi trebalo da izbegavaš kreiranje lambda funkcija koje su nepotrebno složene ili teške za čitanje, kao što je ova:

```py
result = (lambda x: (x**2 + 2*x - 1) if x > 0 else (x**3 - x + 4))(3)
print(result)  # 14
```

Iako ova funkcija radi ispravno i vraća rezultat `14`, nije laka za čitanje niti proveru. U ovom slučaju bolje je napraviti posebnu funkciju sa `if/else` naredbom, a zatim pozvati tu funkciju:

```py
def calculate_expression(x):
    if x > 0:
        return x**2 + 2*x - 1
    else:
        return x**3 - x + 4

print(calculate_expression(3))  # 14
```

I obične funkcije i lambda funkcije imaju svoju primenu u Python programima. Ako radiš sa jednostavnim izrazom u jednoj liniji, možeš razmotriti korišćenje lambda funkcije. U suprotnom, obična funkcija je bolji izbor.
