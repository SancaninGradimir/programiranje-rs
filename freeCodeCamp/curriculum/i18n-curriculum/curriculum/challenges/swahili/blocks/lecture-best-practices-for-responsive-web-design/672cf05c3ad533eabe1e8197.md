# --interactive--

Media query je važan deo dizajna veb stranica koji se prilagođava uređaju, omogućavajući programerima da koriste različite stilove na osnovu karakteristika uređaja korisnika ili pretraživača. Oni pružaju način za personalizovanje prikazivanja sadržaja za različite uređaje bez menjanja samog sadržaja.

U svojoj suštini, upit za medija uključuje tip medija i jedan ili više matematičkih izraza koji proveravaju specifične uslove. Ako su ovi uslovi istiniti, primenjuju se odgovarajući stilovi. Ovaj sistem omogućava kreiranje struktura koje se prilagođavaju veličini ekrana, rezoluciji i različitim sposobnostima uređaja.

Osnovna sintaksa upita za medija u CSS je sledeća:

```css
@media mediatype and (feature: value) {
  /* CSS rules go here */
}
```

U ovoj strukturi, mediatype opisuje tip medija koji se tiče upit za pretraživanje, a par atributa: vrednost opisuje uslov koji mora biti ispunjen da bi stili bili korišćeni.

Tipovi medija opisuju opštu kategoriju uređaja. Pogledajmo najčešće korišćene tipove medija:

- `all` važi za sve uređaje. Ovo je podrazumevana opcija ako nije naveden tip medija.

- `print` namenjen je dokumentima za štampu i dokumentima koji se prikazuju u režimu pregleda za štampanje.

- `screen` Namijenjeno je posebno za ekran.

U prošlosti, postojalo je više tipova medija kao što su `handheld` i `tv`, ali mnogi od njih su prestali da se koriste, a umesto toga koriste se komponente za preciznije ciljanje uređaja.

Media karakteristike opisuju određena svojstva korisničkog agenta, izlaznog uređaja ili okruženja. Pogledajmo neke od najčešće korišćenih media karakteristika. `width` i `height` odnose se na širinu i visinu prikaza (viewport-a) i često se koriste zajedno sa prefiksima `min-` ili `max-` za definisanje opsega. Evo primera:

```css
@media screen and (min-width: 768px) {
  /* Styles for screens at least 768px wide */
}
```

`aspect-ratio` opisuje odnos između širine i visine prikaza (viewport-a). Evo primera:

```css
@media screen and (aspect-ratio: 16/9) {
  /* Styles for screens with a 16:9 aspect ratio */
}
```

Funkcija `orientation` pokazuje da li je uređaj u horizontalnom ili vertikalnom položaju. Evo primera:

```css
@media screen and (orientation: landscape) {
  /* Styles for landscape orientation */
}
```

Atribut ``resolution`` opisuje rezoluciju izlaznog uređaja u tačkama po inču (dpi) ili tačkama po centimetru (dpcm). Evo primera:

```css
@media screen and (min-resolution: 300dpi) {
  /* Styles for high-resolution screens */
}
```

Funkcija `hover` pokušava da vidi stvari pregledavanjem putem glavne metode unosa. Evo primera:

```css
@media (hover: hover) {
  /* Styles for devices that support hover */
}
```

Funkcija `prefers-color-scheme` detektuje da li je korisnik tražio svetlu ili tamnu temu. Evo primera:

```css
@media (prefers-color-scheme: dark) {
  /* Styles for dark mode */
}
```

Media upiti takođe mogu kombinovati više uslova korišćenjem logičkih operatora. Operator `and` se koristi za kombinovanje više medijalnih karakteristika, dok se `not` i `only` mogu koristiti za negiranje ili razdvajanje media upita. Evo primera koji kombinuje više karakteristika:

```css
@media screen and (min-width: 768px) and (orientation: landscape) {
  /* Styles for landscape screens at least 768px wide */
}
```

Takođe je moguće odrediti više argumenata u listi razdvojene zarezima, što funkcioniše kao operator „ili“:

```css
@media screen and (min-width: 768px), print {
  /* Styles for screens at least 768px wide OR for print */
}
```

Kada radiš sa media query-jima, važno je imati na umu redosled primene stilova. Media query-ji ne povećavaju specifičnost selektora – oni samo grupišu uslovna pravila. Uobičajena CSS pravila o redosledu i dalje važe unutar svakog media query-ja.

U upotrebi, media query se često koristi za kreiranje prilagodljivog rasporeda u zavisnosti od uređaja. Uobiştiji primer je definisanje osnovnog stila za mobilne uređaje, a zatim korišćenjem media query-ja za poboljšanje rasporeda za velike ekrane:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Integer nec odio. Praesent libero. Sed cursus ante dapibus diam.</p>
  <p>Sed nisi. Nulla quis sem at nibh elementum imperdiet. Duis sagittis ipsum. Praesent mauris. Fusce nec tellus sed augue semper porta.</p>
  <p>Mauris massa. Vestibulum lacinia arcu eget nulla. Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos.</p>
</div>
```

```css
/* Base styles for mobile */
.container {
  width: 100%;
  padding: 15px;
}

/* Styles for tablets */
@media screen and (min-width: 768px) {
  .container {
    width: 750px;
    margin: 0 auto;
  }
}

/* Styles for desktops */
@media screen and (min-width: 1024px) {
  .container {
    width: 960px;
  }
}
```

:::

Ovaj pristup, poznat kao prilagodljiv dizajn „prvo za mobilne telefone“ (mobile-first), osigurava da osnovni stilovi odgovaraju mobilnim uređajima, a poboljšanja se dodaju za veće ekrane.

Zaključno, Media Query je moćan alat u CSS koji omogućava kreiranje responsivnog i adaptivnog web dizajna. Razumevanjem kako koristiti različite medije i funkcije, programeri mogu kreirati sajtove koji pružaju bolje korisničko iskustvo na raznim uređajima i preferencijama. Kako se web tehnologije razvijaju, ažuriranje znanja o novim medijskim funkcijama može pomoći u kreiranju sofisticovanijih i korisnički pristupačnijih dizajna.
