# --explanation--

`Issue` U ovom kontekstu, to znači problem ili poteškoću koja zahteva rešavanje. U rečenici/kontekstu, radi se o neispravnim sertifikatima za prijavu u bazu podataka. Primer:

- `There was an issue with the system configuration.` - Problem je bio povezan sa konfiguracijom sistema.

- `The main issue was a missing file.` - Problem je nastao jer se fajl nalazi na pogrešnoj lokaciji.

`Them` je glagol koji se odnosi na mnogo imenica spomenutih ranije. U pitanju Lise, `them` se odnosi na `credentials`. Primer:

- `The files were corrupted. Did you fix them?` - `them` Odnosi se na fajlove.

- `The permissions were incorrect. I updated them this morning.` - `them` odnosi se na dozvolu.

Pitanje Lise je da li je Džejms proverio ulazne sertifikate/kredencijale.
