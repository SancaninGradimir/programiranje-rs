# --interactive--

Dijalozi vam omogućavaju prikazivanje važnih informacija ili radnji korisnicima. Korišćenjem ugrađenog HTML `dialog` elementa možete jednostavno kreirati ove dijaloge (modalne i nemodalne) u svojim veb aplikacijama.

Modalni dijalog je tip dijaloga koji primorava korisnika da stupi u interakciju sa njim pre nego što dobije pristup drugim delovima aplikacije ili veb stranice. Potpuno blokira interakciju sa drugim sadržajem dok korisnik ne završi radnju, kao što je zatvaranje dijaloga ili slanje forme.

Nasuprot tome, nemodalni dijalog omogućava korisniku da nastavi interakciju sa drugim delovima stranice ili aplikacije čak i dok je dijalog otvoren. On ne blokira pristup drugom sadržaju.

U ovoj lekciji naučićete kako da otvorite i zatvorite modal pomoću metoda `showModal()` i `close()`.

Kada želite da osigurate da se korisnik fokusira na određenu radnju ili poruku modala, možete otvoriti modalni dijalog pomoću metode `showModal()`. Ovo će dodati pozadinski sloj preko drugih elemenata stranice i odvojiti ih. Ovo je najbolje za modale koji prikazuju forme, potvrde i važne informacije koje zahtevaju radnju korisnika.

Evo HTML-a za modalni dijalog:

:::interactive_editor

```html
<dialog id="my-modal">
  <p>This is a modal dialog.</p>
</dialog>
```

:::

Za sada ne možete videti ništa na stranici jer je modal zatvoren pri početnom prikazu. Modal možete otvoriti direktno pomoću metode `showModal()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
dialog.showModal();
```

:::

Rezultat u pregledaču će prikazati modal sa tekstom `This is a modal dialog.`

Bolje je dati korisniku kontrolu. Da biste to postigli, možete dodati osluškivač događaja klika na dugme i koristiti metodu `showModal()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.showModal();
});
```

:::

Ako treba da prikažete dijalog dok dozvoljavate interakciju sa sadržajem izvan dijaloga, možete koristiti metodu `show()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});
```

:::

Da biste zatvorili modal, možete dodati dugme unutar `dialog` elementa modala i koristiti metodu `close()`:

:::interactive_editor

```html
<dialog id="modal">
  <p>This is a modal dialog.</p>
  <button id="close-modal-btn">Close Modal</button>
</dialog>
<button id="open-modal-btn">Open Modal Dialog</button>
<script src="index.js"></script>
```

```js
const dialog = document.getElementById("modal");
const openButton = document.getElementById("open-modal-btn");
const closeButton = document.getElementById("close-modal-btn");

openButton.addEventListener("click", () => {
  dialog.show();
});

closeButton.addEventListener("click", () => {
  dialog.close();
});
```

:::
