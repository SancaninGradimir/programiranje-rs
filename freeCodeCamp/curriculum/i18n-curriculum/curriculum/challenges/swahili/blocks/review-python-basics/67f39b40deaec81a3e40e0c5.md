## Python Šta je to?

- **Uvod**: Python je programski jezik opšte namene, poznat po svojoj jednostavnosti i lakoj primeni. Python se koristi u mnogim oblastima, kao što su data nauka i mašinsko učenje, web razvoj, pisanje skripti i direktno funkcionisanje zadataka, ugrađeni sistemi i IoT, i još mnogo toga.
- **Primene**: Python se koristi u data nauci, mašinskom učenju, web razvoju, sajber bezbednosti, direktnom funkcionisanju zadataka i mikroračunarima kao što su Raspberry Pi i matične ploče kompatibilne sa MicroPython.

## Varijable

- **Deklarisanje parametara**: Da biste deklarisali parametar, počinjete sa imenom parametra praćenim operatorom dodeljivanja (`=`) a zatim vrednost. Ovo može biti broj, niz karaktera, tačna ili netačna vrednost, itd. Evo nekoliko primera:

```py
name = 'John Doe'
age = 25
```

- **Konvencije imenovanja parametara**: Ovde su konvencije imenovanja koje treba da koristite za parametre:

- Imena varijabli mogu počinjati samo sa slovom ili donjom crtom (\_), a ne brojem.
  - Imena varijabli mogu sadržati samo slova i brojeve (a-z, A-Z, 0-9) i donje crte (\_).
  - Imena varijabli su osetljiva na veličinu slova — `age`, `Age`, i `AGE` se sve smatraju različitim.
  - Imena varijabli ne smeju biti rezervne ključne reči za Python kao što su `if`, `class`, ili `def`.
  - Imena varijabli sa više reči razdvajaju se donjim crtama. Primer `snake_case`.

## Tipovi standardnih podataka u Python

- **Uvod**: Python je jezik čije se tipovi menjaju tokom izvršavanja kao JavaScript, što znači da nema potrebe eksplicitno deklarisati tipove parametara. Jezik zna tip parametra na osnovu onoga što unesete u taj parametar.
- **Celi broj**: Broj bez decimalnih mesta:

```py
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- **Decimalni broj**: Broj sa decimalnim delovima:

```py
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- **Niz slova**: Niz slova koji je postavljen unutar navodnika:

```py
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- **Vrednost Tačno ili Netačno**: Vrednost koja predstavlja `True` ili `False`:

```py
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- **Skup**: Neporedjena kolekcija jedinstvenih elemenata:

```py
my_set_var = {7, 5, 8}
print('Set:', my_set_var) # Set: {7, 5, 8}
```

- **Diktionar**: Kolekcija parova ključ-vrednost, koji su zatvoreni unutar uglastih zagrada:

```py
my_dictionary_var = {"name": "Alice", "age": 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- **Tuple**: Kolekcija koja je strukturirana i nepromenljiva, zatvorena unutar običnih zagradi:

```py
my_tuple_var = (7, 5, 8)
print('Tuple:', my_tuple_var) # Tuple: (7, 5, 8)
```

- **Niz brojeva**: Niz brojeva, često korišćen u petljama:

```py
my_range_var = range(5)
print(my_range_var) # range(0, 5)
```

- **Lista**: Struktuirana kolekcija komponenti koja podržava različite vrste podataka:

```py
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- **None**: Specifična vrednost koja predstavlja odsustvo vrednosti:

```py
my_none_var = None
print('None:', my_none_var) # None: None
```

## Vrste nepromenjivog i promenljivog

- **Nepromenljivi tipovi**: Ovi tipovi se ne mogu promeniti nakon deklarisanja, iako možete preusmeriti njihove parametre na nešto novo, što se naziva ponovno kastovanje (recasting). Oni uključuju cele brojeve, decimalne brojeve, kompleksne brojeve, logičke vrednosti (tačno ili netačno), niz znakova (string), tuple, sekvence i `None`.
- **Promenljivi tipovi**: Ovi tipovi se mogu promeniti nakon deklarisanja. Možete dodati, ukloniti ili modifikovati njihove elemente. Oni uključuju kolekivne tipove kao što su liste, setovi i dikcionari.
- **Funkcija `type()`**: Da biste videli tip parametra, možete koristiti funkciju `type()` na sledeći način:

```py
greeting = 'Hello there!'
age = 21

print(type(greeting)) # <class 'str'>
print(type(age)) # <class 'int'>
```

- **Funkcija `isinstance()`**: Koristi se za proveru da li parametar odgovara određenom tipu podataka:

```py
greeting = 'Hello world'
name = 'John Doe'

print(isinstance(greeting, str)) # True
print(isinstance(name, int)) # False
```

## Rad sa nizovima znakova

- **Definicija**: Kao što ste zapamtili iz JavaScript, niz slova je nepromenljiv, što znači da ga ne možete promeniti nakon kreiranja. U Python, možete koristiti pojedinačne ili dvostruke navodne znakove. Preporučuje se da odaberete jedno pravilo i sledite ga:

```py
developer = 'Jessica'
city = "Los Angeles"
```

- **Dobijanje karakter(a) iz sekvence karaktera**: Možete dobiti karakter iz sekvence karaktera koristeći notaciju uglastih zagrada, kao što sledi:

```py
my_str = 'Hello world'

print(my_str[0])  # H
print(my_str[6])  # w

print(my_str[-1])  # d
print(my_str[-2]) # l
```

- **Izbegavanje sekvence karaktera**: Možete koristiti kosu crtu (`\`) ako vaša sekvenca karaktera sadrži navodnike kao što je sledeće:

```py
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

- **Konkatenacija niza karaktera**: Da biste spojili niz znakova, možete koristiti operator `+` kao što sledi:

```py
developer = 'Jessica'
print('My name is ' + developer + '.') # My name is Jessica.
```

Drug način za spajanje niza znakova je korišćenje operatora `+=`. Ovo se koristi za konkatenaciju i segmentaciju u jednom koraku, kao što sledi:

```py
greeting = 'My name is '
developer = 'Jessica.'

greeting += developer
print(greeting) # My name is Jessica.
```

- **`f-strings`**: Ovo je skraćenica za uređenu sekvencu karaktera. Omogućava vam da izvršite unos vrednosti i takođe uspostavite sa kratkom i čitljivom sintaksom:

```py
developer = 'Jessica'
greeting = f'My name is {developer}.'
print(greeting) # My name is Jessica.
```

- **Извлачење подпореже низа символа**: Ovo je mesto gde možete izvući delove sekvence karaktera. Ovde je osnovna sintaksa:

```py
str[start:stop:step]
```

Broj za početak pokazuje tačku od koje treba započeti ekstrakciju. Broj za kraj je mesto gde bi deo trebalo da se završi. Ovaj broj nije uključen. Broj koraka ukazuje interval za povećanje ekstrakcije. Evo primera:

```py
message = 'Python is fun!'

print(message[0:6])  # Python
print(message[7:])  # is fun!
print(message[::2])  # Pto sfn
```

- **Dobijanje dužine niza znakova**: Funkcija `len()` se koristi za vraćanje broja znakova u nizu:

```py
developer = 'Jessica'

print(len(developer)) # 7
```

## Rad s operacijom od `in`

- **Opereta ya `in`**: Ovo vraća vrednost tačno ili netačno koja određuje da li znakovi ili slova postoje u nizu karaktera, ili ne:

```py
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

## Uobičajene metode sekvenci karaktera

- **`str.upper()`**: Ovo vraća novu sekvencu znakova sa svim slovima konvertovanim u VELIKA SLOVA:

```py
developer = 'Jessica'

print(developer.upper()) # JESSICA
```

- **`str.lower()`**: Ovo vraća novu sekvencu znakova sa svim slovima konvertovanim u mala slova:

```py
developer = 'Jessica'

print(developer.lower()) # jessica
```

- **`str.strip()`**: Ovo vraća kopiju niza znakova koji uklanja određene znakove sa početka i kraja (ako nije naveden argument, uklanja vodeće i zakažne razmake).

```py
greeting = '  hello world  '

trimmed_my_str = greeting.strip()
print(trimmed_my_str)  # 'hello world'
```

- **`replace()`**: Ovo vraća novu sekvencu karaktera gde su svi događaji stare sekvence zamenjeni novom sekvencom.

```py
greeting = 'hello world'

replaced_my_str = greeting.replace('hello', 'hi')
print(replaced_my_str)  # 'hi world'
```

- **`split()`**: Ovo se koristi za razdvajanje niza znakova u listu koristeći definisani separator. Separator je niz znakova koji određuje mesto na kojem treba da se podeli.

```py
dashed_name = 'example-dashed-name'

split_words = dashed_name.split('-')
print(split_words)  # ['example', 'dashed', 'name']
```

- **`join()`**: Ovo se koristi za spajanje elemenata iterabilne kolekcije u niz znakova pomoću funkcije. Iterabilna kolekcija je kolekcija elemenata koja se može iterirati, kao što su lista, niz znakova ili tuple.

```py
example_list = ['example', 'dashed', 'name']

joined_str = ' '.join(example_list)
print(joined_str)  # example dashed name
```

- **`str.startswith(prefix)`**: Ovo vraća vrednost tačno/netačno koja ukazuje da li niz znakova počinje sa definisanim prefiksom:

```py
developer = 'Naomi'

result = developer.startswith('N')
print(result)  # True
```

- **`str.endswith(suffix)`**: Ovo vraća booleanski vrednost koja ukazuje da li niz znakova završava sa navedenim sufiksom:

```py
developer = 'Naomi'

result = developer.endswith('N')
print(result)  # False
```

- **`str.find()`**: Ovo vraća šablon prvog događaja male sekvence. Ako nije dostupan, `-1` se vraća:

```py
developer = 'Naomi'

result = developer.find('N')
print(result)  # 0

city = 'Los Angeles'
print(city.find('New')) # -1
```

- **`str.count(substring)`**: Ovo broji koliko puta manja sekvenca se pojavljuje u nizu karaktera:

```py
city = 'Los Angeles'
print(city.count('e')) # 2
```

- **`str.capitalize()`**: Ovo vraća novu sekvencu slova sa velikim početnim slovom i ostalim malim:

```py
dessert = 'chocolate cake'
print(dessert.capitalize()) # Chocolate cake
```

- **`str.isupper()`**: Ovo vraća `True` ako su svi znakovi u nizu velika slova i `False` inače:

```py
dessert = 'chocolate cake'
print(dessert.isupper()) # False
```

- **`str.islower()`**: Ovo vraća `True` ako su sva slova u nizu mala i `False` inače:

```py
dessert = 'chocolate cake'
print(dessert.islower()) # True
```

- **`str.title()`**: Ovo vraća novu sekvencu slova tako da je prvo slovo svake reči veliko:

```py
city = 'los angeles'
print(city.title()) # Los Angeles
```

- **`str.maketrans()`**: Ova metoda se koristi za kreiranje tabele odgovaranja slova po slovo radi prevođenja. Često se koristi zajedno sa metodom `translate()` koja koristi tu tabelu u sekvenci znakova i vraća prevedene rezultate.

```py
trans_table = str.maketrans('abc', '123')
print(trans_table) # {97: 49, 98: 50, 99: 51}

result = 'abcabc'.translate(trans_table)
print(result)  # 123123
```

## Operacije koje se koriste sa celobrojnim i decimalnim brojevima

- **Osnovne matematičke operacije**: U Python, možete izvršiti osnovne matematičke operacije sa celim brojevima i decimalnim brojevima, uključujući sabiranje, oduzimanje, množenje i deljenje:

```py
int_1 = 56
int_2 = 12
float_1 = 5.4
float_2 = 12.0

# Addition

print('Integer Addition:', int_1 + int_2) # Integer Addition: 68
print('Float Addition:', float_1 + float_2) # Float Addition: 17.4

# Subtraction

print('Int Subtraction:', int_1 - int_2) # Int Subtraction: 44
print('Float Subtraction:',  float_2 - float_1) # Float Subtraction: 6.6

# Multiplication

print('Int Multiplication:', int_1 * int_2) # Int Multiplication: 672
```

```py
int_1 = 56
```


- **Funkcija za `float()`**: Možete koristiti ovu funkciju da biste konvertovali ceo broj u decimalni broj.

- **Funkcija `int()`**: Možete koristiti ovu funkciju za konverziju decimalnog broja u ceo broj.

```py
print(int(num)) # 4
```

- **Funkcija `round()`**: Koristi se za zaokruživanje broja na najbliži ceo broj:

- **Funkcija za `pow()`**: Ovo se koristi za dohvaćanje broja drugom metodom:

## Rad sa zaposlenima

- **Definicija**: Funkcije su ponovno upotrebljivi delovi koda koji primaju ulaz (argument) i vraćaju rezultate. Da biste pozvali funkciju, morate navesti naziv funkcije praćen standardnim zagradama:

```py
def greet():
```

```py
def get_sum(num_1, num_2=2):

result = get_sum(3) 
print(result) # 5
```

```py
def calculate_sum(a, b):

calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

## Rad sa izjavama za `if`, `elif` i `else`

- **Matamko ya `if`**: Ovo su uslovi koji se koriste za određivanje da li je nešto tačno ili netačno. Ako je uslov potvrđen da `True`, onda će se taj deo koda izvršiti.

```py
age = 18

if age >= 18:
    print('You are an adult') # You are an adult
```

- **Član `elif`**: Ovo su uslovi koji prate izjavu od `if`. Član `elif` je primenjiv samo ako su svi prethodni uslovi potvrđeni kao `False` i njegov uslov je potvrđen kao `True`.

```py
age = 16

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')  # You are a teenager
```

- **Klauzula za `else`**: Ovo će biti izvršeno ako ne postoji nijedan drugi uslov koji je potvrđen kao `True`.

```py
age = 12

if age >= 18:
    print('You are an adult')
elif age >= 13:
    print('You are a teenager')
else:
    print('You are a child')  # You are a child
```

Takođe možete koristiti izjave za `if` kako je postavljeno u nastavku:

```py
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

## Vrednosti Istine i Laži

- **Definicija**: U Python, svaka vrednost ima prirodnu vrednost istinito ili netačno, ili ugrađeno osećanje ako treba da se tretira kao `True` ili `False` u kontekstu logike. Većina vrednosti se smatra istinitom, tj. potvrđuje da je `True` u kontekstu logike. Ostale su netačne, smatra se da su `False`. Evo primera lažnih vrednosti:

```md
None
False
Integer 0
Float 0.0
Empty strings ''
```

Druge vrednosti, kao što su brojevi različiti od nule i ne-prazan niz slova, su validne.

## Rad sa funkcijom `bool()`

- **Opis**: Ako želite da proverite da li je vrednost istinita ili lažna, možete koristiti funkciju unutar `bool()`. Ona konvertuje vrednost u jednaku istinitoj ili lažnoj i vraća `True` za istinite vrednosti i `False` za lažne vrednosti. Ovde su nekoliko primera:

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

## Opereta za Vrednost Istinitog ili Nistinitog i Sekcionisanje puta

- **Definicija**: Ovo su specijalni operatori koji vam omogućavaju da kombinujete više matematičkih izraza kako biste kreirali složenu logiku odlučivanja u vašem kodu. Postoje tri operatera sa vrednošću tačno ili netačno u Python: `and`, `or` i `not`.
- **Operater `and`**: Ovaj operator uzima dva operanda i vraća prvi operand ako je netačan, inače vraća drugi operand. Oba operanda moraju biti tačna da bi se izraz generisao kao tačan.

```py
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

Možete takođe koristiti operaciju `and` na sledećim uslovima:

```py
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

- **Opereta ya `or`**: Ovaj operator vraća prvi operand ako je tačan, inače vraća drugi operand. Izraz `or` je tačan ako je bar jedan operand tačan. Evo primera:

```py
age = 19
is_employed = False

print(age or is_employed) # 19
```

Kao što je slučaj sa operetom `and`, možete koristiti operetu `or` na uslovima kao što su sledeći:

```py
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

- **Prekid puta**: Operatori za `and` i `or` poznati su kao operateri prekida puta. Prekid puta znači da Python proverava vrednosti od leve ka desnoj i zaustavlja se čim detektuje konačan rezultat.
- **Operater `not`**: Ovaj operater uzima jedan operand i konvertuje njegovu istinitu ili neistinitu vrednost. Konvertuje istinite vrednosti u `False`, a netačne (false) vrednosti u `True`. Za razliku od prethodnih operatera koje smo gledali, `not` uvek vraća `True` ili `False`. Evo primera:

```py
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

Ovde je primer operete `not` u uslovu:

```py
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```
