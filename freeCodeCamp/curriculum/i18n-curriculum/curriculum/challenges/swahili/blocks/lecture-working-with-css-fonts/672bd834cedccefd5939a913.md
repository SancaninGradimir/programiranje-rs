# --description--

Pre nego što uđemo unutra, treba da znate šta je pravilo 'at' u CSS. Pravila 'at' su deklaracije koje daju instrukcije pretraživaču. Možete ih koristiti za definisanje različitih elemenata u listu stilova, kao što su upiti medija (media queries), važni elementi/okviri, font tipka i još mnogo toga. Naučićete više o konceptima kao što su upiti medija i važni elementi u narednim lekcijama. Trenutno, naučićete o pravilu 'at' za `@font-face`.

Za `@font-face`, možete definisati specifične fontove navođenjem fajla fonta, stila i drugih važnih parametara, kao što su težina i stil. Ovo je osnovna sintaksa. Možete videti `@font-face` praćen blokom u zagradi:

```css
@font-face {
  /* Descriptors */
}
```

Unutar uglastih zagrada, moraćete uključiti specifikacije za definisanje izgleda vašeg fonta. Pogledajmo neke od najčešće korišćenih parametara. Parametar `font-family` opisuje ime koje ćete koristiti u stilskom listu da biste se na taj font pozvali. Na primer, recimo da definišete `@font-face rule`. Ima parametar `font-family` definisan sa vrednošću `MyCustomFont`:

```css
@font-face {
  font-family: "MyCustomFont";
}
```

U tom slučaju, biće potrebno da koristite to ime u vašem stilskom listu gde god želite postaviti tu specifičnu font porodicu za sve elemente koji odgovaraju selektoru CSS. Na primer, koristili biste kao vrednost za `font-family`:

```css
body { 
  font-family: "MyCustomFont"; 
}
```

Međutim, da bi pravilo `at` za ``@font-face`` bilo važeće, morate navesti i ``src``. Ovo sadrži reference na font resurse. To je lista eksternih referenci ili imena lokalno instaliranih fontskih karakteristika, razdvojena zarezima. Takođe može da sadrži napomene o strukturi i tehnologiji font resursa.

U kodu koji se nalazi ispod, `src` je definisan. Vrednost je lista URL razdvojenih zarezima i postavljene na različite redove radi poboljšanja čitljivosti:

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2"),
    url("path/to/font.woff"),
    url("path/to/font.otf");
}
```

Možete pozvati funkciju `url()` da biste uključili datoteke u svoj CSS. U ovom slučaju uključujemo datoteke fontova. Putanju do datoteke potrebno je navesti unutar običnih zagrada i navodnika, zajedno sa ekstenzijom datoteke. Radi bolje čitljivosti, svaki izvor možete napisati u posebnom redu, ali poslednji mora da se završava tačkom i zarezom.

Za svaki izvor fonta možete navesti i njegov format. To nije obavezno. To je samo nagoveštaj pregledaču o formatu fonta. Ako format nije naveden, resurs će biti preuzet, a format će biti otkriven nakon preuzimanja. Ako je format nevažeći, resurs neće biti preuzet.

Miundo inayowezekana ya fonti ni pamoja na `collection`, `embedded-opentype`, `opentype`, `svg`, `truetype`, `woff`, na `woff2`.

Ovo je primer sa strukturama fonta. Pogledajte kako pišemo specifičan format unutar običnih zagrada i navodnika:

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
    url("path/to/font.otf") format("opentype"),
    url("path/to/font.woff") format("woff");
}
```

U ovom primeru, možete videti da identifikujemo format WOFF2, format OpenType i format WOFF.

woff znači "Web Open Font Format." Razlika između WOFF-a i WOFF2 je algoritam koji se koristi za kompresiju podataka. OpenType je proširivi format računarskih fontova koji su uspostavili Microsoft i Adobe, omogućavajući korisnicima da dobiju dodatne funkcije u fontovima. Široko se koristi u velikim operativnim sistemima.

Pored određivanja strukture, takođe možete definisati tehnologiju resursa fonta. Ovo je takođe opcionalno. Evo primera gde definišemo tehnologiju resursa drugog fonta.

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
    url("path/to/font.otf") format("opentype") tech(color-COLRv1),
    url("path/to/font.woff") format("woff");
}
```

Ovo je osnova pravila `@font-face`. Sa ovim pravilom 'at', možete specificirati fajl fonta za definisanje specifičnih fontova za vaše jedinstvene dizajne.
