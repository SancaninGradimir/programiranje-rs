# --description--

Razmotrićemo kriterijume i argumente u kontekstu sledećeg praktičnog primera:

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal('Goat'));
```

U JavaScript, parametri su stvari unutar običnih zagrada `()` u deklaraciji funkcije. Argumenti funkcije su stvarni vrednosti koje odlučujete da prosledite toj funkciji.

U gornjem primeru, opis funkcije je napisan na prvoj liniji: `function favoriteAnimal(animal)`. Parametar `animal` nalazi se unutar običnih zagrada. Takođe možete promeniti `animal` i `pet`, `x`, ili `blah`. Međutim, u ovom slučaju, pominjanje parametra `animal` daje osobi koja čita vaš kod malo konteksta kako ne bi morala da pogađa šta `animal` može biti kasnije.
