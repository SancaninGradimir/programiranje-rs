# --description--

`Changes` U ovom kontekstu znači promjene ili izmjene napravljene u kodu.

`Descriptive` znači davati opis o tome kako nešto izgleda, oseća, zvuči, miriše ili ukus. To je pridev koji se koristi za pružanje više informacija o imenici i čine opis jasnijim i vidljivijim. Primer, u rečenici `she wears a beautiful dress`, reč `beautiful` je `descriptive` jer opisuje slušaoca više o haljini.

`message` To je deo informacija ili komunikacija koju jedna osoba šalje drugoj osobi. Može biti tekstualni sadržaj, reči, ili čak poslat putem e-pošte ili poruke.
