# --description--

Za cele brojeve $a$ i $b$, definišemo $D(a, b)$ kao površinu ograničenu parabolom $y = x^2$ i linijom $y = ax + b$: $D(a, b) = \\{ (x, y) | x^2 ≤ y ≤ ax + b \\}$.

$L(a, b)$ se definiše kao broj rešetkastih tačaka koje su unutar $D(a, b)$. Na primer, $L(1, 2) = 8$ i $L(2, -1) = 1$.

Takođe definišemo $S(N)$ kao zbir $L(a, b)$ za sve parove ($a$, $b$) gde je površina $D(a, b)$ računski broj i $|a|,|b| \leq N$.

Možemo potvrditi da $S(5) = 344$ i $S(100) = 26\\,709\\,528$.

Tafuta $S({10}^{12})$. Toa jibu lako $\bmod {10}^8$.
