# --interactive--

Da vidimo kako da napravite čvor koristeći `innerHTML` i `createElement()`.

Početi ćemo sa `innerHTML`.

`innerHTML` je svojstvo objekata `Element` koje možete koristiti za postavljanje atributa HTML. Sa `innerHTML`, možete postaviti strukturu HTML postojećeg elementa koristeći niz karaktera, kreirajući sve potrebne čvorove.

Ovo je primer. Imamo prazan `div` i dodaćemo nekoliko elemenata unutar njega koristeći osobinu `innerHTML`:

```html
<div id="container">
  <!-- Add new elements here -->
</div>
```

Prvo, moramo izabrati element koristeći njegov ID:

```js
const container = document.getElementById("container");
```

Zatim postavljamo osobinu `innerHTML` kontejnera na niz znakova. Taj niz mora sadržati sve potrebne oznake za predstavljanje čvorova i strukture koju želite da kreirate. Možete ga zamisliti kao pisanje HTML-a unutar niza znakova.

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
container.innerHTML = "<ul><li>Cheese</li><li>Tomato</li></ul>";
```

:::

Postavićemo `innerHTML` ovog elementa da bude neuređeni spisak sastojaka za picu (sir i paradajz).

Takođe možete zapisati taj niz u više redova ako to čini oznake lakšim za razumevanje. Važno je da oznake pravilno predstavljaju HTML strukturu koju želite da napravite.

Nakon pokretanja ovog koda, videćete strukturu HTML dok pregledavate svoje pravopisne znake.

```html
<div id="container">
  <ul>
    <li>Cheese</li>
    <li>Tomato</li>
  </ul>
</div>
```

Novi čvorovi su dinamički kreirani i dodati u DOM nakon što je niz obrađen.

`innerHTML` može biti veoma koristan u određenim situacijama. Međutim, ima određene bezbednosne rizike kojih treba da budete svesni. Ne bi trebalo da ga koristite ako nemate potpunu kontrolu nad tim nizom.

Na primer, ako korisnik unese sekvencu, ne biste trebali koristiti `innerHTML` jer korisnik može uneti opasan sadržaj na vaš sajt. Zbog toga, obično se savetuje da umesto toga koristite `textContent` za unos običnog teksta.

Drugi način za kreiranje novog čvora je korišćenje metode `createElement()`. Ovim novom putem, možete kreirati novi element navođenjem imena njegove oznake (ili: etikete).

Na primer, ako želite kreirati vizuelni element slike, proći ćete pored oznake `img` kao niz karaktera kada pozovete ovu funkciju:

```js
document.createElement("img");
```

I možete postaviti ovu novu stavku u parametar:

```js
const img = document.createElement("img");
```

Metoda za `createElement()` vraća novi objekat od `HTMLElement` ako je dokument `HTMLDocument`. Inače, vraća objekat od `Element`.

Kada imate ovu novu funkciju spremnu, možete je dodati na DOM kao dete druge postojeće funkcije koristeći način `appendChild()`, ili ga možete umetnuti na specifično mesto koristeći drugi način. Izaberite onaj koji vam najviše odgovara.

:::interactive_editor

```html
<div id="container"></div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
const img = document.createElement("img");
img.src = "https://cdn.freecodecamp.org/curriculum/cat-photo-app/lasagna.jpg";
img.alt = "A slice of lasagna on a plate.";
container.appendChild(img);
```

:::

Kreiranje čvorova korišćenjem `innerHTML` i `createElement()` omogućava vam da snažno kontrolišete strukturu i sadržaj vaših sajtova. Kombinovanjem ovih tehnika, možete izgraditi interaktivne veb aplikacije.
