# --description--

Reč `might` je modalni glagol koji se koristi za izražavanje mogućnosti ili sposobnosti. Forma `be able to` pokazuje sposobnost.

Pamoja, `might be able to` ukazuje da postoji mogućnost postizanja nečega, ali nije sigurno.

Na primer, `She might be able to finish the work by tomorrow.` Ovo znači da je moguće da će završiti taj zadatak u tom trenutku, ali nije sigurno.
