# --description--

Ako je dat CSS niz boja `rgb(r, g, b)`, vrati njegov ekvivalent u heksadecimalnom formatu.

Ovde su neki primerni rezultati za određen unos:

| Ingizo   | Matokeo   |
|---------|----------|
| `"rgb(255, 255, 255)"`| `"#ffffff"` |
| `"rgb(1, 2, 3)"` | `"#010203"` |

- Pretvorite sva slova u mala.
- Vratite `#` praćeno šest slova. Nemojte koristiti skraćene vrednosti.

# --solutions--

```js
function rgbToHex(rgb) {
  const match = rgb.match(/\d+/g);
  const [r, g, b] = match.map(num =>
    Math.max(0, Math.min(255, parseInt(num)))
      .toString(16)
      .padStart(2, '0')
  );

  return `#${r}${g}${b}`;
}
```
