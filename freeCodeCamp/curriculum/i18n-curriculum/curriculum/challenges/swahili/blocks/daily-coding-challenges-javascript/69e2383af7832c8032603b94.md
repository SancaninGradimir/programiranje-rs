# --description--
Dado je niz slova kafe narudžbine; identifikujte sve stavke sa menija i vratite organizovanu porudžbinu.

Koristi sledeće stavke sa menija i cene:

| Kitu | Bei | 
| - | - |
| `"cold brew"` | $4.50 |
| `"oat latte"` | $5.00 |
| `"cappuccino"` | $4.75 |
| `"espresso"` | $3.00 |
| `"vanilla syrup"` | $0.75 |
| `"caramel drizzle"` | $0.60 |
| `"extra shot"` | $0.50 |
| `"oat milk"` | $0.75 |
| `"cream"` | $0.75 |

Vratite sekvencu karaktera sa odgovarajućim stavkama povezanim sa `" + "`, praćen dve tačke i razmak (`": "`), i ukupnom cenom.

Na primer, ako primite `"I'd like an oat latte with vanilla syrup and an extra shot please."`, vratite `"oat latte + vanilla syrup + extra shot: $6.25"`

Predmeti bi trebalo da se prikazuju u redosledu kako su na meniju, a ukupan iznos mora uvek imati dve decimalne za cela vremena.

```js
function formatCoffeeOrder(order) {
  const menu = [
    { name: "cold brew", price: 4.50 },
    { name: "oat latte", price: 5.00 },
    { name: "cappuccino", price: 4.75 },
    { name: "espresso", price: 3.00 },
    { name: "vanilla syrup", price: 0.75 },
    { name: "caramel drizzle", price: 0.60 },
    { name: "extra shot", price: 0.50 },
    { name: "oat milk", price: 0.75 },
    { name: "cream", price: 0.75 },
  ];

  const found = menu.filter(item => order.toLowerCase().includes(item.name));
  const total = found.reduce((sum, item) => sum + item.price, 0);
  return `${found.map(item => item.name).join(" + ")}: $${total.toFixed(2)}`;
}
```
