# --interactive--

Kreiranje glatkih animacija na web stranici može biti izazovno, posebno ako ne znate kako pravilno da upravljate tajmingom. Dobra vest je da API od `requestAnimationFrame()` to olakšava.

`requestAnimationFrame()` je način koji vam omogućava da planirate sledeći korak vaše animacije pre nego što se ekran osveži, rezultujući glatkim i vizuelno privlačnim iskustvom.

Osvježavanje ekrana znači trenutak kada pretraživač ponovo prikazuje izgled web stranice. To se dešava više puta u sekundi, obično otprilike 60 puta (ili 60 frejmova po sekundama) na većini ekrana.

Da biste koristili metodu `requestAnimationFrame()`, što morate učiniti je da ga pozovete i date mu callback funkciju unutar njega:

```js
requestAnimationFrame(callback);
```

Kada se pominje ``requestAnimationFrame()``, to mora prvo biti urađeno unutar funkcije koja obrađuje animaciju, kao što je ``animate()``, zajedno sa funkcijom za ažuriranje animacije, koja se obično zove ``update()``:

```js
function animate() {
 // Update the animation...
 // for example, move an element, change a style, and more.
 update();
 // Request the next frame
 requestAnimationFrame(animate);
}
```

Funkcija ``update()`` radi čuda. Unutra, možete promeniti sve što želite da animirate. Na primer, ažuriranje stila ili promena pozicije elementa:

```js
function update() {
 element.style.transform = `translateX(${position}px)`;
 position += 2;
}
```

Ono što pokreće animaciju jeste pozivanje `requestAnimationFrame()` i prosleđivanje funkcije `animate`, ovog puta izvan same funkcije `animate`:

```js
requestAnimationFrame(animate);
```

Ciklus će se nastaviti dok ga ne pustiš.

Sada pogledajmo drugi primer. HTML za ovaj primer je `div` element sa tekstom `freeCodeCamp is Awesome`. CSS pretvara `div` u pravougaonik i skriva sve što izlazi iz vidljive oblasti sa leve ili desne strane. JavaScript kod pomera pravougaonik `2px` udesno pri svakom pozivu `requestAnimationFrame(animate)`, pri čemu funkcija `animate` služi kao povratna funkcija:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div id="rect" class="rect">freeCodeCamp is Awesome</div>
<script src="index.js"></script>
```

```css
body {
  overflow-x: hidden;
}

.rect {
  display: flex;
  align-items: center;
  justify-content: center;
  width: 400px;
  height: 250px;
  border-radius: 5px;
  background-color: #1b1b32;
  color: #f5f6f7;
  font-size: 2rem;
  position: absolute;
}
```

```js
const rect = document.getElementById("rect");

let position = 0;

function update() {
  // Move the rectangle 2px to the right
  rect.style.left = position + "px";
  position += 2;

  if (position > window.innerWidth) {
    // Move the rectangle just outside the left side of the screen
    position = -rect.offsetWidth;
  }
}

function animate() {
  update();

  //request the next frame
  requestAnimationFrame(animate);
}

// Start the animation
requestAnimationFrame(animate);
```

:::

Rezultati u pretraživaču će biti animirana kartica zaglavlja koja pluta na ekranu.
