## Python u Vašem Unutrašnjem Okruženju

- **Instalacija**: Najbolji način za instaliranje Python na Windows, Mac i Linux je preuzimanje instalatora sa zvaničnog sajta Python (`https://www.python.org/`).
- **Terminal**: Tekstualni interfejs koji vam omogućava da komunicirate sa svojim računarom unosom komandi. Svaki operativni sistem dolazi sa osnovnim terminalnim programom. Na macOS-u, možete koristiti Terminal aplikaciju. Na Windows-u, možete koristiti Command Prompt ili PowerShell. Na Linux-u, svako desktop okruženje ima svoj osnovni terminalni program, kao što su GNOME Terminal ili Konsole.
- **IDE**: IDE je skraćenica od Integrated Development Environment i ima funkcije kao što su uređivač koda, alati za debagovanje, terminal i više.
- **Popularni uređivači koda i IDE-ovi za Python**: VS Code, PyCharm i Spyder
- **Pokretanje koda na vašem računaru**: Da pokrenete kod Python, imate nekoliko opcija. Prva opcija je kliknuti na dugme za pokretanje u VS Code koje izgleda kao dugme za reprodukciju u gornjem desnom uglu. To će otvoriti terminal i pokrenuti vaš skript Python tamo.

Druga opcija je otvoriti terminal i pokrenuti program ručno. Evo primera kako da se pokrene program pod nazivom `main.py`:

```bash
python main.py
```

Morate pokrenuti ovu komandu iz foldera gde je `main.py` sačuvan. Na primer, ako je `main.py` unutar foldera pod nazivom `python-projects`, prvo koristite sledeće u terminalu:

```bash
cd python-projects
```

Zatim trči:

```bash
python main.py
```

## Upotreba Python Interaktivnog Shell-a

- **Opis**: Interaktivna školjka (shell) je program koji vam omogućava da direktno unosite komande i odmah vidite rezultate. Otvorite terminal i unesite `python`, a zatim pritisnite `Enter`. Time ćete pokrenuti Python interaktivnu školjku. Simbol `>>>` pokazuje da Python čeka da unesete komandu. Pokušajte da prikažete `"Hello, world!"` u terminalu:

```bash
print("Hello, world!")
```

Trebalo bi da vidite tekst koji izgleda ovako:

```bash
>>> print("Hello, world!")
Hello, world!
```

Nakon što se tekst objavi, Python vraća se na sledeće:

```bash
>>>
```

Ovo znači da možete napisati još jednu komandu.

- **Ciklus Čitanje, Evaluacija, Ispisivanje, Ponavljanje (REPL)**: Svaki put kada napišete komandu, interpretator će pročitati unos, evaluirati ga, ispisati rezultate i ponoviti prikaz `>>>` ili nastavite sa unosom više komandi.
