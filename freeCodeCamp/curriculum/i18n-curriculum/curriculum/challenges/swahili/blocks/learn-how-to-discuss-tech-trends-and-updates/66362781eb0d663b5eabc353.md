# --description--

`Serverless computing` je primer implementacije `cloud computing` gde cloud provajder upravlja serverima i dinamički upravlja raspodelom mašinskih resursa. Cena zavisi od stvarno korišćenog obima resursa aplikacijom, umesto unapred kupljenih jedinica kapaciteta. Naziva se `serverless` jer programeri ne moraju da upravljaju serverima - oni samo učitavaju svoj kod, a cloud provajder se bavi svim ostalnim.

Na primer, u `serverless computing`, možete učitati funkciju za cloud uslugu, i ona radi samo kada je potrebno.
