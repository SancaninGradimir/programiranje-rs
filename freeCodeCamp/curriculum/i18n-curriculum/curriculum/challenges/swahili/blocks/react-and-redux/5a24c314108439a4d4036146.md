# --description--

Funkcija `mapDispatchToProps()` koristi se za obezbeđivanje kreatora akcija vašim React komponentama kako bi mogle da šalju akcije prema Redux skladištu. Slična je strukturi funkcije `mapStateToProps()` koju ste napisali u prethodnom izazovu. Vraća objekat koji mapira dispatch akcije na imena svojstava, koja zatim postaju `props` komponente. Međutim, umesto vraćanja dela `state`, svako svojstvo vraća funkciju koja poziva `dispatch` sa kreatorom akcije i svim potrebnim podacima. Ovom `dispatch` pristupu imate pristup zato što se prosleđuje u `mapDispatchToProps()` kao argument kada definišete funkciju, slično kao što ste prosledili `state` u `mapStateToProps()`. Ispod površine, React Redux koristi `store.dispatch()` iz Redux-a za izvršavanje ovih dispatch poziva u `mapDispatchToProps()`. Ovo je slično načinu na koji koristi `store.subscribe()` za komponente povezane sa `state`.

Na primer, imate kreatora akcije `loginUser()` koji prima `username` kao podatak akcije. Objekat koji bi se vratio iz `mapDispatchToProps()` za ovog kreatora akcije izgledao bi ovako:

```jsx
{
  submitLoginUser: function(username) {
    dispatch(loginUser(username));
  }
}
```

# --instructions--

Kodni editor pruža fabriku akcija nazvanu `addMessage()`. Napisite funkciju za `mapDispatchToProps()` koja uzima `dispatch` kao argument, a zatim vraća objekat. Taj objekat treba da ima svojstvo `submitNewMessage` postavljeno za dispatch funkciju, koja uzima parametar za dodavanje nove poruke kada šalje `addMessage()`.
