# --description--

Kao što si naučio u prethodnim modulima, petlje se koriste za ponavljanje bloka koda određeni broj puta. U ovoj lekciji naučićeš kako da radiš sa različitim vrstama petlji u Python-u.

Prvi ciklus ćemo vam objasniti, a to je ciklus `for`. Evo primera kako da koristite ciklus `for` za iteriranje kroz listu i štampanje svakog elementa na konzolu:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)
```

Rezultati će biti:

```md
Rust
Java
Python
C++
```

Obrati pažnju da je `print(language)` uvučen unutar petlje. Bez tog uvlačenja dobićeš `IndentationError`:

```python
"""
Traceback (most recent call last):
  File "<stdin>", line 4, in <module>
IndentationError: expected an indented block after 'for' statement on line 3
"""
```

Takođe možete koristiti petlju za `for` da iterirate kroz druge iterabilne stvari, kao što je niz znakova. Evo primera korišćenja petlje za `for` da iterira kroz niz znakova `code` i štampa svaki znak:

```py
for char in 'code':
    print(char)
```

Matokeo yatakuwa:

```md
c
o
d
e
```

Kao i u JavaScript-u, i u Python-u možeš da ugnjezdiš `for` petlje. Evo primera korišćenja ugnježdene `for` petlje:

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)
```

Spoljašnja petlja prolazi kroz svaku `category` u listi `categories`. Za svaku `category`, unutrašnja petlja prolazi kroz svaki `food` u listi `foods`. Evo šta će biti ispisano u konzoli:

```md
Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
```

Druga vrsta petlje koju možeš da koristiš u Python-u je `while` petlja. Ova vrsta petlje će ponavljati blok koda sve dok uslov ne postane `False`. Evo primera korišćenja `while` petlje za igru pogađanja:

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')
```

U ovom primeru imamo promenljivu `secret_number` sa vrednošću `3` i početnu pretpostavku `0`. Zatim koristimo funkciju `input` da dobijemo unos od korisnika, pretvaramo uneti tekst u ceo broj pomoću funkcije `int()`, i čuvamo tu vrednost u promenljivoj `guess`. Ako korisnik pogodi unosom broja `3`, `while` petlja se prekida i u konzoli se ispisuje poruka `You got it!`. U suprotnom, u konzoli se ispisuje poruka `Wrong! Try again.`, a petlja se nastavlja i ponovo traži od korisnika da pokuša.

Evo primera izlaza:

```md
Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
```

Kao što je u JavaScript, Python podržava izjave za `break` i `continue`.

Izjava `break` se koristi za zaustavljanje izvršavanja kola. Evo primera korišćenja izjave `break` za listu `developer_names`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

U ovom primeru, iteriramo kroz listu `developer_names` i ispisujemo svaki naziv na konzolu. Ako je naziv jednak `Naomi`, onda prekidamo petlju. Ovo uzrokuje da se samo naziv `Jess` ispisa na konzolu.

Izjava za `continue` se koristi za preskakanje trenutne iteracije ciklusa i nastavak sa sledećom iteracijom. Hajde da promenimo početni primer koristeći izjavu `continue` umesto `break`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

Sada će rezultati na konzoli biti drugačiji. Imena `Jess` i `Tom` su odštampana jer izjava `continue` preskače drugu iteraciju petlje kada je `developer` jednako `Naomi`, i ne štampa to ime na konzoli.

Petlje ``for`` i ``while`` mogu se kombinovati sa klauzulom ``else``, što se dešava samo kada petlja nije završena za izjavu ``break``. Evo primera korišćenja više petlji za ``for``:

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

U ovom primeru imamo listu nasumičnih reči, a petlja `for` se koristi za iteraciju kroz svaku reč. Unutar spoljne petlje `for`, imamo još jednu petlju `for` koja iterira kroz svaki karakter svake reči. Ako je mala slova verzija samoglasnik, ispisujemo reč praćenu njenim samoglasnicima, a zatim prekidamo unutrašnju petlju. Ako reč nema samoglasnika, onda ispisujemo poruku koja to pokazuje.

Ovo je kako rezultati izgledaju na konzoli:

```md
'sky' has no vowels
'apple' contains the vowel 'a'
'rhythm' has no vowels
'fly' has no vowels
'orange' contains the vowel 'o'
```

Circuiti su veoma uobičajeni u Python, tako da je važno da se naviknete na njih. U narednim lekcijama, naučićete kako da radite sa funkcijom `enumerate()` i `range()` u circuitima.
