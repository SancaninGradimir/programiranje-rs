# --explanation--

Upotreba `that` nakon `I think` je opcionalna u rečenici poput ove. Reći `I think that it's vital we make these updates.` je takođe ispravno. Međutim, u svakodnevnom razgovoru, uobičajeno je izostaviti `that` radi glatke i brže rečenice. Na primer:

`I think that we need more time to finish the project.` – Ovaj dizajn je tačan. Mnogi ljudi bi rekli `I think we need more time…`, međutim.
