# --interactive--

CSS ne primenjuje nijedoke senke na tekst podrazumevano. Ovo je primer pasusa bez ikakvih senki. Ali, ako treba da dodate senke, možete koristiti svojstvo `text-shadow`.

```css
text-shadow: /* Values */
```

U CSS, možete opisati senku pomoću kombinacije udaljenosti `X`, udaljenosti `Y`, kruga zamućenja (blur), i boje. Prvo, morate specificirati `X` i `Y` udaljenosti, koje predstavljaju horizontalnu i vertikalnu udaljenost senke od teksta, respektivno. Ove vrednosti su neophodne.

Ovo je primer kako koristiti `X` i `Y` daleko u sjeni. Koristimo svojstvo `text-shadow`, sa x-offsetom od `3px` i y-offsetom od `2px`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 3px 2px;
}
```

:::

U pretraživaču, tekst i senka će izgledati isto, ali možemo prilagoditi boju senke specifikovanjem vrednosti pre ili posle udaljenosti. Izaberimo boju senke. Ovde ćemo koristiti heksadecimalnu boju, ali možete koristiti bilo koji validan tip boje.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 3px 2px #00ffc3;
}
```

:::

Boju pišemo posle vrednosti pomeraja, ali je možete napisati i pre njih ako vam tako više odgovara. Oba načina su ispravna:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: #00ffc3 3px 2px;
}
```

:::

U pregledaču će senka imati posebnu boju, pa je možemo lako razlikovati od teksta. Kada možemo da razlikujemo senku od teksta, važno je i razumeti kako pozitivne i negativne vrednosti `X` i `Y` pomeraja utiču na njen položaj. Pozitivne vrednosti `X` i `Y` pomeraja pomeraju senku udesno i nadole, dok je negativne vrednosti pomeraju ulevo i nagore. Evo primera koji koristi negativne vrednosti:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: -3px -2px #00ffc3;
}
```

:::

Sada će se senka nalaziti ulevo i iznad teksta. Dobro. Međutim, senka još uvek ne izgleda naročito dobro, jer izgleda kao običan tekst postavljen iza originala. Da bi izgledala prirodnije, potrebno je dodati treću vrednost – radijus zamućenja (blur radius). Ova vrednost nije obavezna, ali čini senku mekšom i prirodnijom. Podrazumevana vrednost radijusa zamućenja je nula. Što je vrednost veća, zamućenje je izraženije, pa senka izgleda blaže.

Ovde postavljamo radijus zamućenja na `3px` i vraćamo se na pozitivne vrednosti pomeraja:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 3px 2px 3px #00ffc3;
}
```

:::

Sada već počinje da liči na pravu senku. Mekša je i prirodnija, pa pažnja ostaje na glavnom tekstu, dok senka dodaje vizuelni efekat u pozadini. Odavde možete fino podešavati vrednosti pomeraja, boje i zamućenja dok ne dobijete željeni izgled.

Takođe je korisno znati da tekst može imati više senki istovremeno. Potrebno je samo da ih navedete u jednom svojstvu `text-shadow`, odvojene zarezima. Senke će biti složene po slojevima, od napred ka pozadi, pri čemu je prva senka na vrhu.

Evo primera pasusa sa tri sloja senki. Primetite da su sve senke definisane u jednom svojstvu `text-shadow` i razdvojene zarezima:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Hello, World!</p>
```

```css
p {
  text-shadow: 
    3px 2px 3px #00ffc3, 
    -3px -2px 3px #0077ff, 
    5px 4px 3px #dee7e5;
}
```

:::

`text-shadow` je moćna karakteristika CSS za učinjavanje vašeg teksta lepšim. Takođe može poboljšati ceo dizajn vaše veb aplikacije. Izabravanjem udaljenosti, zamućenja (blur) i boje, možete kreirati različite vrste senkarnih efekata.
