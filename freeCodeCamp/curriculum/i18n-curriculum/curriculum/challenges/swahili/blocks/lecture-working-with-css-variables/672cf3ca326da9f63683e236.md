# --interactive--

Kanuni `@property` je moćno svojstvo CSS koje omogućava programerima da objavljuju standardne atribute i imaju veću kontrolu nad njihovim ponašanjem, uključujući način na koji se menjaju i njihove početne vrednosti.

Ovaj pravilnik nudi način povećanja performansi standardnih svojstava CSS i pruža veću fleksibilnost u njegovoj upotrebi.

Osnovna sintaksa pravila `@property` je sledeća:

```css
@property --property-name {
  syntax: '<type>';
  inherits: true | false;
  initial-value: <value>;
}
```

`--property-name` je ime običnog atributa koji deklarišete. Kao svi obični atributi, mora početi sa dvostrukim dvotačkom.

`syntax` deklarira tip atributa, koji može biti stvari kao što su `<color>`, `<length>`, `<number>`, `<percentage>`, ili složeniji tipovi.

`inherits` objašnjava kako svojstvo treba da nasledi svoju vrednost od roditeljskog elementa.

`initial-value` postavlja podrazumevanu vrednost svojstva.

Ovdje je praktičan primjer korištenja pravila `@property`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
@property --main-color {
  syntax: '<color>';
  inherits: false;
  initial-value: #3498db;
}

.button {
  background-color: var(--main-color);
}
```

:::

U ovom primeru, deklarišemo standardno svojstvo ``--main-color`` kao vrednost za ``color``, postavljajući ga da nije nasleđeno, i dajući mu početnu vrednost od ``#3498db``.

Glavna prednost pravila `@property` je što omogućava animaciju standardnih svojstava, nešto što nije bilo moguće ranije:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="gradient-box"></div>
```

```css
@property --gradient-angle {
  syntax: '<angle>';
  inherits: false;
  initial-value: 0deg;
}

.gradient-box {
  width: 100px;
  height: 100px;
  background: linear-gradient(var(--gradient-angle), red, blue);
  transition: --gradient-angle 0.5s;
}

.gradient-box:hover {
  --gradient-angle: 90deg;
}
```

:::

Gornji kod kreira gradijent boje koji se postepeno menja kada se klikne na element, nešto što nije bilo moguće sa standardnim svojstvima.

Sada, hajde da diskutujemo kako pravilo `@property` radi sa alternativama.

Alternativni način je važan u CSS kako bi se osiguralo da stilovi pravilno funkcionišu u pregledačima koji ne podržavaju određene funkcije. Za `@property`, fallback radi na dva nivoa: po sopstvenom pravilu i korišćenjem zajedničkog svojstva.

Za pretraživače koji ne podržavaju kanon `@property`, možete pružiti alternativu deklarisanjem standardnog atributa na uobičajen način:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
```

```css
:root {
  --main-color: #3498db;
}

@property --main-color {
  syntax: '<color>';
  inherits: false;
  initial-value: #3498db;
}

body {
  background-color: var(--main-color);
}
```

:::

U ovom slučaju, pretraživači koji podržavaju `@property` će koristiti najstrože objavljenu verziju, dok će drugi se vratiti na standardnu deklaraciju uobičajene karakteristike.

Kada koristite standardnu osobinu, možete pružiti alternativnu vrednost koristeći akciju za `var()`, kao što radite sa uobičajenim atributima:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
.button {
  background-color: var(--main-color, #3498db);
}
```

:::

Ovo osigurava da čak i ako atribut `--main-color` nije deklarisan ili je nevažeći, dugme će i dalje biti vidljivo sa pozadinskom bojom.

Kanuni za `@property` takođe dozvoljavaju složenije alternative. Na primer, možete ga koristiti za pružanje bezbednih alternativa za vrste:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="box">Colored box</div>
```

```css
@property --padding {
  syntax: '<length>';
  inherits: false;
  initial-value: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkred;
  color: white;
  padding: var(--padding);
}
```

:::

U ovom slučaju, ako neko pokuša da postavi `--padding` sa nevažešćom vrednošću, kao što je boja, pretraživač će se vratiti na podrazumevanu vrednost od `10px` i održati tipsku bezbednost.

Kanuni ya CSS `@property` se smatra široko dostupnim u svim glavnim motorima savremenih pregledača (kao što su Chrome, Edge, Opera, Firefox i Safari) od početka 2024. godine. Ako želite da podržite starije pregledače, morate pružiti CSS alternativu kako je opisano gore.

Zaključno, standard `@property` predstavlja veliki napredak u načinu na koji radimo sa standardnim svojstvima u CSS. Pruža veću kontrolu i omogućava nove mogućnosti, kao što je animacija standardnih svojstava. Međutim, važno je koristiti ga promišljeno i uvek pružiti odgovarajuće alternative kako bi se osigurao dobar korisnički doživljaj u svim pretraživačima. Kao i kod bilo koje moderne web tehnologije, ključ je poboljšanje iskustva za pretraživače koji ga podržavaju, uz osiguravanje osnovne funkcionalnosti za one koji ne podržavaju.
