# --description--

Niz celih brojeva $S = \\{s_i\\}$ naziva se nizom dužine $n$ ako ima $n$ elemenata i svaki element $s_i$ zadovoljava $1 \le s_i \le n$. Ukupan broj različitih nizova dužine $n$ je $n^n$.

Kwa mfano, mfululizo $S = \\{1, 5, 5, 10, 7, 7, 7, 2, 3, 7\\}$ ni mfululizo wa 10.

Za bilo koji niz $S$, neka $L(S)$ označava dužinu najdužeg uzastopnog podniza sa jednakim vrednostima. Na primer, za gore dati niz $S$, važi $L(S) = 3$, zbog tri uzastopne sedmice.

Izračunajte $f(n) = \sum L(S)$ za sve sekvence celih brojeva $n$ $S$.

Kwa mfano, $f(3) = 45$, $f(7) = 1\\,403\\,689$ na $f(11) = 481\\,496\\,895\\,121$.

Pronađi $f(7\\,500\\,000)\bmod 1\\,000\\,000\\,009$.
