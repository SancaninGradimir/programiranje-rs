# --description--

Sada znate za najpopularnije načine iteracije po vrednostima i nizovima podataka, postoje još manje načina za iteraciju po nizovima podataka. Nizovi podataka imaju ugrađenu metodu pod nazivom `map()`, koja se koristi za kreiranje novog niza korišćenjem funkcije za svaki element originalnog niza. Metoda `map()` ne modifikuje originalni niz. Sintaksa metode `map()` je sledeća:

```javascript
const array = [1, 2, 3, 4, 5];

const newArray = array.map((arrayValue) => {
  return arrayValue * 2;
});

console.log(newArray); // Output: [2, 4, 6, 8, 10]
```

Metoda `map()` kreira novi red koristeći funkciju `(arrayValue) => { return arrayValue * 2; }` za svaki element prvobitnog reda. Ovo je važno posebno kada želite da modifikujete elemente reda bez menjanja originalnog reda.
