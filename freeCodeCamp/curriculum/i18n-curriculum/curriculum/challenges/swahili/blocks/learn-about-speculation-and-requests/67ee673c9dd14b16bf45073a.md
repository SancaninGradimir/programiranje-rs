# --explanation--

`Unexpected behavior` Odnosi se na rezultate koji se razlikuju od očekivanih. Ovo može uključivati greške, netačne proračune ili preskočenje važnih slučajeva. Neki primeri nelagano ponašanja u petljama su:

- `Incorrect output` - Program radi, ali daje loše rezultate.

- `Infinite repetition` - Ciklus se nastavlja zauvek.

- `Unexpected termination` - Petlja može izazvati grešku koja prekida izvršavanje.

- `System crash` - Retko je, ali moguće zbog problema povezanih sa memorijom.
