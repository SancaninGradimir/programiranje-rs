# --interactive--

Kopiranje tipa leptira serije podataka je nova serija koja sadrži iste elemente kao i originalna serija. Ako ta serija sadrži samo osnovne vrednosti, poput brojeva ili sekvence slova, nova serija je potpuno drugačija. Međutim, ako serija sadrži drugu seriju unutar sebe, kako original tako i kopija imaju referencu na te interne serije. To znači da ako promenite nešto unutar podeljene interne serije, videćete te promene u obe serije.

Kopije leptira su pomoć kada je potrebno promeniti strukturu višeg nivoa, kao što je dodavanje, uklanjanje ili menjanje redosleda elemenata, bez menjanja originalnog sloja ili unutrašnjeg sloja.

Postoje nekoliko načina za kreiranje kopija redova podataka o leptirima, i istražićemo neke popularne metode: `concat()`, `slice()`, i operaciju raspršivanja (spread).

Počnimo sa metodom `concat()`. Ova metoda kreira novi red spajanjem dva ili više redova. Kada se koristi samo sa jednim redom, efikasno kreira leptir kopiju. Evo primera:

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const copyArray = [].concat(originalArray);

console.log(copyArray); // [1, 2, 3]
console.log(copyArray === originalArray); // false
```

:::

U ovom primeru, koristimo metodu `concat()` za spajanje praznog reda sa `originalArray`. Ovo će stvoriti novi red koji je leptir kopija od `originalArray`.

`copyArray` ima elemente slične `originalArray`, ali je drugačiji tip sloja, zato stroga provera jednakosti (`===`) vraća `false`.

Drugi način za kreiranje kopije leptira je `slice()`. Kada se pozove bez argumenata, `slice()` vraća kopiju leptira celog reda. Ovako funkcioniše:

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const copyArray = originalArray.slice();

console.log(copyArray); // [1, 2, 3]
console.log(copyArray === originalArray); // false
```

:::

U ovom slučaju, `originalArray.slice()` kreira novi red koji je kopija od `originalArray`. Takođe, `copyArray` ima iste elemente, ali je druga stavka reda.

Opereta širenja (`...`), deklarisan u ES6, nudi drugi kraći način za kreiranje leptir kopija nizova. Evo primera:

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const copyArray = [...originalArray];

console.log(copyArray); // [1, 2, 3]
console.log(copyArray === originalArray); // false
```

:::

Operacija širenja (`...`) širi elemente `originalArray` u novi red, efikasno stvarajući leptir kopiju. Važno je razumeti da sve ove metode kreiraju nove redne stavke, što znači da možete izmeniti kopiju bez uticaja na originalni red. Na primer:

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const copyArray = [...originalArray];

copyArray.push(4);
console.log(originalArray); // [1, 2, 3]
console.log(copyArray);     // [1, 2, 3, 4]
```

:::

U ovom primeru, dodavanje elementa u `copyArray` neće uticati na `originalArray`.

Ukratko, plitke kopije nizova mogu se lako napraviti pomoću metoda `concat()`, `slice()` ili spread operatora. Ove metode su korisne za pravljenje novih nizova koji se mogu nezavisno menjati bez uticaja na originalni niz.
