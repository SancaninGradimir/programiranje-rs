# --description--

React je biblioteka za prikaze otvorenog izvora, kreirana i održavana od strane Facebooka. To je odličan alat za prikaz korisničkog interfejsa (UI) moderne veb aplikacije.

`React koristi sintaktičku ekstenziju JavaScript pod nazivom JSX, koja omogućava pisanje HTML direktno unutar JavaScript. Ovo ima nekoliko prednosti. Omogućava korišćenje pune moći programa JavaScript unutar HTML, i pomaže da vaš kod bude čitljiv. U velikoj meri, JSX je sličan HTML koji ste već naučili, međutim postoje neke ključne razlike koje ćemo pokriti u ovim izazovima.`

Na primer, pošto je JSX sintaksno proširenje JavaScript-a, možeš pisati JavaScript direktno unutar JSX-a. Da bi to uradio, kod koji želiš da se tretira kao JavaScript stavi u vitičaste zagrade: `{ 'this is treated as JavaScript code' }`. Zapamti ovo, jer će se koristiti u mnogim narednim izazovima.

Međutim, pošto JSX nije važeći JavaScript, JSX kod mora biti preveden u JavaScript. Babel je popularan transpajler za taj proces. Radi tvoje praktičnosti, već je uključen u ovim izazovima. Ako napišeš sintaksno neispravan JSX, prvi test u izazovu neće proći.

Treba napomenuti da su ove izazove ispod haube nazvani `ReactDOM.render(JSX, document.getElementById('root'))`. Ovaj poziv funkcije je ono što stavlja vaš JSX u lagani prikaz/predstavljanje React od DOM. React koristi snimke (snapshot) svog DOM kako bi poboljšao ažuriranje samo za određene delove stvarnog DOM.
