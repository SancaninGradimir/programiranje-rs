# --description--

Onyesho uslova u React vam omogućava da kreirate dinamičke korisničke interfejse. Koristi se za prikaz različitog sadržaja u zavisnosti od određenih uslova ili stanja unutar vaše aplikacije.

Najčešće korišćene metode za prikaz uslova uključuju upotrebu izraza `if`, funkcije tri-argumentnog kalkulatora (`?:`) i logičke operacije AND (`&&`).

Najjednostavniji način za prikaz uslova je korišćenje deklaracije `if`. Evo primera:

```jsx
function Greeting({ isLoggedIn }) {
  if (isLoggedIn) {
    return <h1>Welcome back!</h1>;
  }
  return <h1>Please sign in</h1>;
}
```

U ovom primeru, deo `Greeting` proverava parametre `isLoggedIn`. Ako je `true`, vraća poruku dobrodošlice, inače podstiče korisnika da se prijavi.

Evo primera koji koristi komponentu `Greeting` unutar roditeljske komponente `App`:

```jsx
function App() {
  return (
    <div className="App">
      <Greeting isLoggedIn={false} />
    </div>
  );
}
```

Za jednostavne uslove, ternarni operator (`?:`) se često koristi direktno u JSX-u. To omogućava uslovno prikazivanje u jednoj liniji, što kod može učiniti kraćim.

```jsx
function Greeting({ isLoggedIn }) {
  return <h1>{isLoggedIn ? "Welcome back!" : "Please sign in."}</h1>;
}
```

Ovaj kod postiže rezultate slične prethodnom primeru, ali na sažetiji način. Ternarni operator proverava ``isLoggedIn`` i prikazuje odgovarajuću poruku.

Drugi uobičajeni način za prikaz uslova je korišćenje logičkog operatora AND (`&&`). Ovo je posebno korisno kada želite da prikažete nešto, ili da ne prikažete ništa, u zavisnosti od uslova:

```jsx
function Notification({ message }) {
  return (
    <div>
      {message && <p>{message}</p>}
    </div>
  );
}
```

U ovom primeru, element paragrafa sa porukom je prikazan samo ako su uslovi za `message` tačni. Ako je `message` lažna vrednost - što znači prazan niz karaktera, `null`, ili `undefined`, ništa se ne prikazuje na ekranu.

Koristeći ove metode uslovnog prikazivanja, možete izgraditi aplikacije koje su više angažujuće i korisnički prijateljske, a koje se menjaju na osnovu podataka i interakcije korisnika.
