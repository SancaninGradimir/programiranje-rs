# --description--

Modifikovani Collatz niz celih brojeva se dobija od početne vrednosti $a_1$ na sledeći način:

$a_{n + 1} = \frac{a_n}{3}$ ako je $a_n$ deljivo sa 3 bez ostataka. Ovo ćemo nazvati veliki pad, "D".

$a_{n + 1} = \frac{4a_n + 2}{3}$ ako se $a_n$ deli sa 3 i ostaje ostatak 1. Ovo ćemo nazvati uspon, "U".

$a_{n + 1} = \frac{2a_n - 1}{3}$ Ako se $a_n$ podeli sa 3, ostatak je 2. To ćemo nazvati manjim korakom spuštanja, "d".

Serija se završava kada $a_n = 1$.

Ako nam je dat bilo koji kompletan broj, možemo navesti niz koraka. Na primer ako $a_1 = 231$, onda niz $\\{a_n\\} = \\{231, 77, 51, 17, 11, 7, 10, 14, 9, 3, 1\\}$ odgovara koraku "DdDddUUdDD".

Bez sumnje, postoji još jedna sekvenca koja prethodi i toj istoj sekvenci "DdDddUUdDD...".

Na primer, ako je $a_1 = 1004064$, onda niz je DdDddUUdDDDdUDUUUdDdUUDDDUdDD.

Zapravo, 1004064 je $a_1$ najmanji mogući veći od ${10}^6$, koji prethodi nizu DdDddUUdDD.

Je li $a_1$ manje od ${10}^{15}$ IP adrese koja prethodi sekvenci "UDDDUdddDDUDDddDdDddDDUDDdUUDd"?
