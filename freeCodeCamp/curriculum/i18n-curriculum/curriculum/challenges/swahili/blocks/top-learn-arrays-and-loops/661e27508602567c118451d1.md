# --description--

Sekvence slova i brojeva mogu biti naše osnovne strukture, ali kako vaš tekst postaje složeniji, biće vam potreban način za rukovanje velikim količinama toga. Srećom, JavaScript ima nekoliko tipova podataka koji se koriste posebno za to. Red podataka je organizovana kolekcija stavki (sekvenca slova, brojeva ili drugih stvari).

Nizovi podataka su način za skladištenje više vrednosti u jednom atributu. To je poseban tip objekta koji ima karakteristiku dužine i sekvencu atributa sa brojevima. Svaki atribut sa brojem naziva se element, a svaki element može da skladišti vrednost bilo kog tipa.

Primer reda podataka je:

```javascript
const fruits = ['apple', 'banana', 'orange'];
```
