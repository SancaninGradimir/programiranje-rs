# --interactive--

U JavaScript, komponenta je posebna vrsta funkcije koja se koristi za kreiranje i inicijalizovanje stvari. Naziva se korišćenjem glavne reči `new` i može inicijalizovati parametre i metode na novokreiranom objektu.

U ovom tutorijalu, ćemo pogledati kako da radimo sa komponentom `Object()`. Komponenta `Object()` kreira novi prazan objekat. Evo primera:

```js
new Object()
```

Kada izvršite poziv `new Object()`, vraćate nešto novo koje se može koristiti za skladištenje vrednosti.

Komponenta ``Object()`` može se koristiti sa ili bez glavne riječi ``new``. Kada se pozove kao funkcija bez ``new``, radi drugačije u zavisnosti od tipa proslijeđene vrijednosti. Evo primjera korištenja komponente ``Object()`` bez glavne riječi ``new``:

:::interactive_editor

```js
const num = 42;
const numObj = Object(num); // Creates an object wrapper for the number

console.log(numObj);
console.log(typeof numObj); // "object"
```

:::

Kao što možete videti u drugom `console.log` primeru, `numObj` je objekat. Ovo se dešava zato što smo koristili konstruktor `Object()` da pretvorimo brojčani unos u objekat.

Šta se dešava ako pokušamo da prosledimo `null` ili `undefined` konstruktoru `Object()`?

:::interactive_editor

```js
const newObj = new Object(undefined);
console.log(newObj); // {}
```

:::

Konkretno, rezultat će biti prazan objekat. Druga upotreba konstruktora `Object()` je kada radite sa vrednošću nepoznatog tipa i morate biti sigurni da je objekat. Pogledajmo sledeći primer:

:::interactive_editor

```js
function toObject(value) {
  if (value === null || value === undefined) {
    return {};
  }

  if (typeof value === "object") {
    return value;
  }

  return Object(value);
}

console.log(toObject(null));

console.log(toObject(true));

console.log(toObject([1, 2, 3]));
```

:::

U ovom primeru imamo funkciju nazvanu `toObject`. Drugi uslov proverava da li je vrednost tipa objekat i vraća vrednost ako je uslov `true`. Ovaj uslov proverava objekte zajedno sa nizovima, jer su nizovi posebna vrsta objekata.

Ako nijedan od uslova nije tačan, funkcija će vratiti `Object(value)`, koji konvertuje ulaz u objekat. Ovo radi sa vrednostima kao što su brojevi, nizovi znakova i vrednosti tačno ili netačno.

U većini slučajeva nećete koristiti konstruktor `Object()` za kreiranje novih objekata, jer ćete umesto toga koristiti sintaksu objektnog literala (na primer, `const objectLiteral = { name: "Beau" }`). Ipak, korisno je razumeti osnove rada sa konstruktorom Object.
