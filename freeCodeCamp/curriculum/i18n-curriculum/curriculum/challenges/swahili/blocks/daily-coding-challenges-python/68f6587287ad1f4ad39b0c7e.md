# --description--
Uklanjanje sekvence karaktera koja predstavlja naziv fajla, vratite mu ekstenziju.

- Ekstenzija je deo imena fajla koji sledi nakon poslednje tačke (`.`).
- Ako ime fajla nema tačku ili se završava tačkom, vrati `"none"`.
- Ekstenzija treba da se vrati onakva kakva jeste, sa očuvanim velikim i malim slovima.

# --solutions--

```py
def get_extension(filename):
    last_dot = filename.rfind('.')
    
    if last_dot == -1 or last_dot == len(filename) - 1:
        return "none"
    
    return filename[last_dot + 1:]
```
