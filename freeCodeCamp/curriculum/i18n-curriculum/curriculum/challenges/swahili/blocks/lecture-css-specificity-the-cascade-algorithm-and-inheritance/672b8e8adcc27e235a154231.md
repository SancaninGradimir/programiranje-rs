# --interactive--

Selektori tipa, poznati takođe kao selektori elemenata, ciljaju elemente na osnovu imena njihove oznake/tag-a.

Ovi selektori su osnova u CSS i omogućili su korišćenje stilova za sve instance specifičnog elementa od HTML.

Selektori tipa su laki za upotrebu i lako se pišu kao naziv oznake elementa kojem želite dati stil.

Ovo je primer tipskog selektora koji cilja na sve elemente paragrafa na stranici:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  color: blue;
}
```

:::

U ovom primeru svi elementi `p` imaće boju teksta postavljenu na `blue`.

Selektori tipa imaju nižu specifičnost u poređenju sa drugim selektorima. Njihova vrednost specifičnosti je `(0, 0, 0, 1)`.

To znači da će selektori tipa biti nadjačani selektorima klase, ID selektorima i inline stilovima, ali će se primenjivati ako takva pravila više specifičnosti ne postoje.

Pogledajmo primer u kojem selektori klase nadjačavaju stilove selektora tipa.

Evo primera sa dva elementa pasusa:

:::interactive_editor

```html
<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

:::

Oba elementa pasusa imaju klasu `para`.

U CSS datoteci selektor tipa cilja elemente pasusa, dok selektor klase cilja elemente sa klasom `para`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para">I am a paragraph</p>
<p class="para">Here is another paragraph</p>
```

```css
p {
  color: blue;
}

.para {
  color: red;
}
```

:::

Sve linije na stranicama sa klasom `para` će imati boju teksta postavljenu na `red` umesto `blue`, jer selektori klas objekata imaju veću važnost od selektora tipova.
