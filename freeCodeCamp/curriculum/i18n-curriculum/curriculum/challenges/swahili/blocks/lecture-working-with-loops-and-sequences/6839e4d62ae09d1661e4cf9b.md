# --description--

Tokom prethodnih lekcija, navikli ste se da radite sa ovakvim krugovima:

```py
even_numbers = []

for num in range(21):
    if num % 2 == 0:
        even_numbers.append(num)

print(even_numbers)
```

Ovaj primer pravi novu praznu listu pod nazivom `even_numbers` i prolazi kroz niz brojeva od `0` do `20`. Unutar petlje proverava se da li trenutni broj daje ostatak `0` pri deljenju sa `2`. Time se utvrđuje da li je broj paran. Ako je uslov `True`, trenutni `num` se dodaje na kraj liste `even_numbers`. Na kraju se lista `even_numbers` ispisuje u konzoli.

Iako ovaj kod radi, postoji kraći način da se ovo napiše koristeći list comprehension umesto toga. List comprehension vam omogućava da kreirate novu listu u jednoj liniji kombinujući petlju i uslov direktno unutar uglastih zagrada. To čini kod kraćim i često lakšim za čitanje.

Evo izmenjenog primera od malopre koji koristi uglaste zagrade:

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

U ovom izmenjenom primeru lista `even_numbers` napravljena je u jednoj liniji koda. List comprehension prolazi kroz brojeve od `0` do `20` i uključuje samo one koji su deljivi sa `2`. Ovaj pristup je kraći i uklanja potrebu za posebnom petljom i uslovom.

Pogledajmo još jedan primer kako bismo bolje razumeli kako funkcioniše list comprehension:

```py
numbers = [1, 2, 3, 4, 5]
result = [(num, 'Even') if num % 2 == 0 else (num, 'Odd') for num in numbers]
print(result)
```

U ovom primeru imamo listu brojeva i želimo da napravimo novu listu torki koje pokazuju da li je broj paran ili neparan. U prvom delu list comprehension-a koristimo `if` izraz da proverimo da li je broj deljiv sa `2`. Ako jeste, rezultat je torka koja sadrži broj i reč `Even`. U suprotnom, rezultat je torka koja sadrži broj i reč `Odd`.

Ovako izgleda rezultat kada se ispiše u konzoli:

```py
[(1, 'Odd'), (2, 'Even'), (3, 'Odd'), (4, 'Even'), (5, 'Odd')]
```

Još jedan način za pravljenje liste iz postojećeg iterabilnog objekta jeste funkcija `filter()`. Evo primera kreiranja nove liste reči koje imaju više od četiri slova:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

Funkcija `filter()` koristi se za izdvajanje elemenata iz iterabilnog objekta koji ispunjavaju određeni uslov. Funkcija `filter()` prima funkciju i iterabilni objekat kao argumente. U ovom primeru prosleđujemo funkciju `is_long_word` funkciji `filter()` kako bismo proverili da li trenutna reč ima više od `4` slova. Sve reči koje imaju više od `4` slova dodaju se u novu listu koja se čuva u promenljivoj `long_words`.

Pored funkcije `filter()`, postoje i druge korisne funkcije za rad sa listama. Jedna od njih je funkcija `map()`, koja prima iterabilni objekat i primenjuje funkciju na svaki njegov element. Evo primera korišćenja funkcije `map()` za pretvaranje liste temperatura iz Celzijusa u Farenhajte:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

Funkcija kao `filter()`, `map()` prima funkciju i iterable kao svoje argumente. Funkcija `to_fahrenheit` uzima temperaturu i konvertuje je iz Celzija u Farenhajta.

Poslednja funkcija koju ćemo pregledati je funkcija `sum()`. Ova funkcija se koristi za dobijanje zbira iz iterabilnog objekta kao što su lista ili tuple. Evo primera korišćenja funkcije `sum()`:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

Takođe možete proslediti opcionarni argument `start` koji postavlja podrazumevanu vrednost za ukupno. Evo ažuriranog primera koji koristi argument `start` kao pozicioni argument:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

Takođe možete izabrati da koristite argument `start` kao argument reči, umesto ovoga:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

Oba izdanja će dati iste rezultate, ali formulacija je malo jasnija.

List comprehension uz ostale funkcije kao `map()`, `filter()`, i `sum()` može izgledati zbunjujuće u početku. Ali uz dovoljno prakse i vremena, počete ćete se osećati mnogo bolje koristeći ovo u vašim aplikacijama za Python.
