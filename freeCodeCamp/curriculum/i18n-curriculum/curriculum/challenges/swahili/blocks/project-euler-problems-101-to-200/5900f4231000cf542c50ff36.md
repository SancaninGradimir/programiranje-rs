# --description--

Neka je $N$ pozitivan ceo broj i neka je podeljen na $k$ jednakih delova, $r = \frac{N}{k}$, tako da je $N = r + r + \cdots + r$.

Neka je $P$ proizvod tih delova, $P = r × r × \cdots × r = r^k$.

Na primer, ako se 11 podeli na pet jednakih delova, 11 = 2.2 + 2.2 + 2.2 + 2.2 + 2.2, tada je $P = {2.2}^5 = 51.53632$.

Za $M(N) = P_{max}$ za određenu vrednost $N$.

Ispostavlja se da se maksimalna vrednost za $N = 11$ dobija kada se jedanaest podeli na četiri jednaka dela, što daje $P_{max} = {(\frac{11}{4})}^4$; odnosno, $M(11) = \frac{14641}{256} = 57.19140625$, što je konačan decimalni broj.

Međutim, za $N = 8$ maksimalna vrednost se dobija deljenjem na tri jednaka dela, pa je $M(8) = \frac{512}{27}$, što je beskonačan decimalni zapis.

Neka je $D(N) = N$ ako je $M(N)$ beskonačan decimalni broj, a $D(N) = -N$ ako je $M(N)$ konačan decimalni broj.

Kwa mfano, $\sum D(N)$ kwa $5 ≤ N ≤ 100$ ni 2438.

Izračunaj $\sum D(N)$ za $5 ≤ N ≤ 10000$.
