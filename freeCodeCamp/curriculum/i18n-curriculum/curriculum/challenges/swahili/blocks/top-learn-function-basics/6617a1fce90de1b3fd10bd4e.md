# --description--

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal('Goat'));
```

Postavljanjem `animal` unutar standardnih zagrada funkcije `favoriteAnimal()`, govoriš JavaScript da ćeš poslati određenu vrednost svojoj funkciji `favoriteAnimal`. To znači da je animal samo zamena za određenu buduću vrednost. Ali koju vrednost šalješ?

Poslednji red, `favoriteAnimal('Goat')`, je kada pozovete svoju funkciju `favoriteAnimal` i prosledite vrednost `'Goat'` unutar te funkcije. Ovde, `'Goat'` je vaš argument. Kažete funkciji `favoriteAnimal`: "Molimo vas pošaljite `'Goat'` funkciji `favoriteAnimal` i koristite `'Goat'` gde god je predstavnik `'animal'`." Zbog fleksibilnosti koju pruža korišćenje parametra, možete deklarisati bilo koje životinjsko biće kao vaš ljubimac.

Ovde je dijagram koji pomaže da se vidi kako parametri prolaze u funkciju, i kako se vrednosti vraćaju iz nje.

Znate da pozivanjem `favoriteAnimal()` unutar `console.log()` sa argumentom `'Goat'`, dobijate vrednost funkcije return, string karaktera `"Goat is my favorite animal!"`, koja je odštampana na konzolu. Prosljeđuje se poziv funkcije `favoriteAnimal('Goat')` kao argument u drugom pozivu funkcije - `log()`.
