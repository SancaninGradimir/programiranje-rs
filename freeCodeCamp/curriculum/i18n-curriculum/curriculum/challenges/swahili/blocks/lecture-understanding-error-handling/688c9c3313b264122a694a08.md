# --description--

U Python, rukovanje greškama je važan deo pisanja robusnog i otpornog na greške softvera. Omogućava vam da predvidite, uhvatite i reagujete na greške strukturizovano.

Obrada grešaka je proces hvatanja i upravljanja greškama koje nastaju tokom izvršavanja programa, kako bi se sprečilo da vaš kod naglo sruši.

Python pruža funkcije za `try`, `except`, `else`, na `finally` za rukovanje greškama. Evo osnovnog primera:

```py
try:
    x = 10 / 0
except ZeroDivisionError:
    print("You can't divide by zero!")
```

- `try`: Kodni fragment koji predviđa moguću grešku.

- `except`: Ovaj deo se izvršava ako se unutar try dogodi greška navedenog tipa.

- U ovom slučaju, deljenje sa nulom izaziva `ZeroDivisionError`, što je zatim uhvaćeno i obrađeno.

Ovde je primer koji takođe pokazuje kako koristiti sekcije za `else` i `finally`:

```py
try:
    x = 10 / 2
except ZeroDivisionError:
    print("You can't divide by zero!")
else:
    print('Division successful:', x)
finally:
    print('This block always runs.')
```

- `else`: Izvršiće se ako nije došlo do greške u sekciji try.

- `finally`: Izvršava se bez obzira na to da li je došlo do greške ili ne. Važno je za zadatke čišćenja, kao što su zatvaranje fajlova ili oslobađanje resursa.

Takođe možete uhvatiti mnogo grešaka u različitim delovima od `except`:

```py
try:
    number = int('abc')
    result = 10 / number
except ValueError:
    print('That was not a valid number.')
except ZeroDivisionError:
    print("Can't divide by zero.")
```

Korišćenjem više `except` blokova možeš učiniti obradu grešaka preciznijom i korisnijom.

Takođe možete koristiti objekat greške, koji se obično naziva drugim imenom koristeći reč ``as``. Ovde koristimo ``e`` kao alternativno ime za objekat greške:

```py
try:
    x = 1 / 0
except ZeroDivisionError as e:
    print(f'Error occurred: {e}')
```

Upotrebom `e`, možete dobiti stvarnu poruku o grešci ili nešto za snimanje ili pronalaženje grešaka.

Možete takođe uhvatiti više grešaka u jednom bloku `except` navodeći te greške kao tupu:

```py
try:
    number = int(input('Enter a number: '))
    result = 10 / number
except (ValueError, ZeroDivisionError) as e:
    print(f'Error occurred: {e}')
```

Upravljanje greškama omogućava vašim programima da se oporave od grešaka na elegantan način. Korišćenjem `try`, `except`, `else` i `finally`, možete predvideti potencijalne probleme i izgraditi robusnije programe.
