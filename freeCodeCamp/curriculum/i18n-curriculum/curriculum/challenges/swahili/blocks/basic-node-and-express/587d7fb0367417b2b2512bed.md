# --description--

Rad na ovim zadacima uključuje pisanje vašeg koda koristeći jedan od sledećih načina:

- Kopirajte <a href="https://github.com/freeCodeCamp/boilerplate-express/" target="_blank" rel="noopener noreferrer nofollow">ovaj GitHub repozitorijum</a> i završite ove zadatke na vašem lokalnom računaru.
- Koristite vaš omiljeni website builder da završite projekat. Osigurajte da uključujete sve fajlove iz našeg GitHub repozitorijuma.

Tokom procesa razvoja, važno je da imate mogućnost da vidite šta se dešava u vašem kodu.

Node je samo JavaScript okruženje. Kao JavaScript na klijentskoj strani, možete koristiti konzolu da prikažete važne informacije za otklanjanje grešaka. Na vašem lokalnom računaru, videćete rezultate konzole u terminalu.

Preporučujemo da držite terminal otvoren dok radite na ovim zadacima. Čitajući rezultate u terminalu, videćete bilo koje greške koje mogu da se pojave.

Server mora biti ponovo pokrenut nakon što napravite izmene u njegovim fajlovima.

Možete zaustaviti server iz terminala koristeći `Ctrl + C` i odmah ga ponovo pokrenuti koristeći Node (`node mainEntryFile.js`) ili koristeći start skriptu iz `package.json` fajla sa `npm run`.

Na primer, skripta `"start": "node server.js"` će se pokrenuti iz terminala koristeći `npm run start`.

Da biste omogućili automatsko ponovno pokretanje servera kada se fajl sačuva, Node nudi `--watch` zastavicu koju možete dodati u vašu start skriptu `"start": "node --watch server.js"` ili možete instalirati npm paket kao što je `nodemon`. Ostavićemo ovo kao vežbu za vas.
