# --interactive--

Kada stilizuješ veb sajt, jedan od najfleksibilnijih i najlakših načina za rad sa bojama jeste korišćenje HSL modela boja. HSL predstavlja Hue (nijansa), Saturation (zasićenost) i Lightness (osvetljenost) — tri osnovne komponente koje određuju boju.

HSL model boja predstavlja boje na način koji je bliži ljudskom načinu opažanja boja. U ovoj lekciji naučićeš šta je HSL model boja i kako se funkcija `hsl()` koristi u CSS-u.

Hue je vrsta boje, predstavljena kao ugao na točkiću boja, koji počinje od `0` do `360` stepeni. Na primer, `0` stepeni predstavlja crvenu, `120` stepeni je zelena, a `240` stepeni je plava. Da biste promenili Hue, menjate različite boje na točkiću boja.

Saturacija se odnosi na intenzitet ili čistoću boje. Meri se u procentima od `0%` (boja bez boje, siva) do `100%` (najjačije stanje te boje). Boja sa saturacijom `100%` je potpuno svetla, dok boja sa saturacijom `0%` izgleda kao siva senka.

Svetlina određuje kako je boja svetla ili tamna, takođe se meri u procentima. Vrednost svetline za `0%` će dati crnu, `50%` će vam dati normalan ton nijanse, a `100%` će rezultirati belom.

U CSS-u se funkcija `hsl()` koristi za definisanje boja pomoću HSL modela. Savremena sintaksa koristi vrednosti razdvojene razmacima:

```css
element {
  color: hsl(hue saturation lightness);
}
```

Takođe možete naići na zastarelu sintaksu koja koristi zarez kao separator u postojećem kodu, što je i dalje podržano od strane pregledača:

```css
element {
  color: hsl(hue, saturation, lightness);
}
```

Tuchambue kwa mfano:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
body {
  background-color: hsl(0 0% 1% / 1);
}

p {
  color: hsl(120 100% 50%);
}
```

:::

U ovom slučaju, nijansa (`hue`) je `120` stepeni, što odgovara zelenoj boji. Zasićenost (`saturation`) je `100%`, pa će zelena biti potpuno intenzivna. Osvetljenost (`lightness`) je `50%`, što predstavlja njenu standardnu nijansu — ni previše tamnu ni previše svetlu. Kao rezultat, tekst pasusa biće svetlozelene boje.

Glavna prednost HSL modela boja je njegova lako razumljiva priroda. To olakšava podešavanje intenziteta boje ili svetline menjanjem vrednosti saturacije i svetline bez promene osnovne boje (ton).

Na primer, ako želiš da napraviš različite nijanse ili svetlije tonove iste boje, dovoljno je da promeniš vrednost osvetljenosti (`lightness`).

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="light">This is a light blue div.</div>
<div class="dark">This is a dark blue div.</div>
```

```css
div.light {
  background-color: hsl(240 100% 80%);
}

div.dark {
  background-color: hsl(240 100% 20%);
  color: hsl(0 0% 100%);
}
```

:::

Ovde, oba dela od `div` koriste isti ton (`240` stepeni, koji je plavi), ali jedno ima svetlina od `80%` (svetli plavi ton), a drugo ima svetlina od `20%` (tamni plavi ton).

Kao primer za RGB, funkcija ``hsl()`` takođe podržava opcijalnu alfa vrednost za kontrolu neprozirnosti. Dodajete je nakon separatora ``/`` kao četvrti parametar. Ovo je osnovna sintaksa:

```css
element {
  background-color: hsl(hue saturation lightness / alpha);
}
```

Na primer:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: hsl(0 100% 50% / 0.5);
}
```

:::

Ovaj kod će proizvesti `div` iza crvene boje sa polu opacitetom, gde je nijansa postavljena na `0` stepeni (crvena), saturacija je `100%`, svetlina je `50%`, a alfa je `0.5` (opacitet od 50%). Takođe možete definisati alfa kao procenat — na primer, `hsl(0 100% 50% / 50%)` je isto što i `hsl(0 100% 50% / 0.5)`.

Takođe ćete često videti funkciju `hsla()` u postojećim CSS. Ovo je bio raniji način za uključivanje transparentnosti u HSL boje, korišćenje vrednosti razdvojenih zarezima:

```css
element {
  background-color: hsla(hue, saturation, lightness, alpha);
}
```

Na primer, `hsla(0, 100%, 50%, 0.5)` je ekvivalentno starom `hsl(0 100% 50% / 0.5)`. Oba su podržana od strane pregledača, ali moderna sintaksa `hsl()` sa razdvajanjem `/` je sada preporučljivija.

Model boja HSL je važan posebno kada treba da kreirate podešavanja boje i lako prilagodite nijanse ili svetle tonove.

Pošto HSL razdvaja ton od saturacije i svetline, možete postaviti stabilnu paletu boja dok menjate svetlinu za senke ili osvetljenje. Ovaj nivo prilagodljivosti čini HSL omiljenim kod dizajnera i programera.

Štaviše, HSL čini vrednosti boja lakšim za čitanje i razumevanje u poređenju sa drugim metodama kao što su heks kodovi ili RGB. Možete jasno videti nameru iza boje i prilagoditi je shodno potrebi, posebno kada testirate različite dizajne.

Korišćenjem HSL modela boja, dobijate veću fleksibilnost u kontrolisanju tonova boje, saturacije i svetline, čime ga čini moćnim alatom za kreiranje bogatih i živopisnih dizajna u CSS.
