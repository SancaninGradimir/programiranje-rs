## Poređenje po tipovima podataka `null` i `undefined`

- **Poređenje sa `undefined`**: Paramar je `undefined` kada je deklarisan, ali mu nije dodeljena vrednost. Ovo je podrazumevana vrednost za nedeklarisane parametre i funkcionalne parametre kojima nije dodata argumentacija. `undefined` se menja u `NaN` u kontekstu brojeva, što čini poređenje svih brojeva sa `undefined` da vrati `false`.

:::interactive_editor

```js
console.log(undefined < 0); // false (NaN < 0 is false)
console.log(undefined >= 0); // false (NaN >= 0 is false)
```

:::

- **Poređenje sa `null`**: Tip od `null` predstavlja namerno odsustvo vrednosti. `null` se menja u `0` u kontekstu brojeva, što može izazvati neočekivano ponašanje pri poređenju brojeva:

:::interactive_editor

```js
console.log(null < 0); // false (0 < 0 is false)
console.log(null >= 0); // true (0 >= 0 is true)
```

:::

- Kada koristiš operator jednakosti (`==`), `null` na `undefined` je jednako samo svakom i njihovoj duši:

:::interactive_editor

```js
console.log(null == undefined); // true
console.log(null == 0); // false
console.log(undefined == NaN); // false
```

:::

- Međutim, kada koristite operator stroge jednakosti (`===`), koji proverava vrednost i tip bez nametanja tipa, `null` i `undefined` nisu jednaki:

:::interactive_editor

```js
console.log(null === undefined); // false
```

:::

## Sekcija od `switch`

- **Opis**: Funkcija `switch` proverava matematički izraz i upoređuje njegovu vrednost sa nizom uslova u `case`. Kada se pronađe podudarnost, izvršava se deo koda povezan sa tim slučajem. Funkcija `break` treba da bude postavljena na kraj svakog slučaja, kako bi završila svoje izvršenje i prešla na sledeći. Slučaj `default` je opcionalan slučaj i izvršava se samo ako nijedan drugi slučaj ne odgovara. Slučaj `default` postavlja se na kraj funkcije `switch`.

:::interactive_editor

```js
const dayOfWeek = 3; 

switch (dayOfWeek) {
  case 1:
    console.log("It's Monday! Time to start the week strong.");
    break;
  case 2:
    console.log("It's Tuesday! Keep the momentum going.");
    break;
  case 3:
    console.log("It's Wednesday! We're halfway there.");
    break;
  case 4:
    console.log("It's Thursday! Almost the weekend.");
    break;
  case 5:
    console.log("It's Friday! The weekend is near.");
    break;
  case 6:
    console.log("It's Saturday! Enjoy your weekend.");
    break;
  case 7:
    console.log("It's Sunday! Rest and recharge.");
    break;
  default:
    console.log("Invalid day! Please enter a number between 1 and 7.");
}
```

:::
