# --description--

Mateo spominje svoj uzrast na španskom koristeći sledeću strukturu, i glagol `tener` ("imati"):

`Tengo` + broj rečima + `años`.

Napomena, španske brojeve od 21 do 29 počinju sa `veinti-`, praćenim odgovarajućim brojem. Primer:

25 je `veinticinco`. Ovo nastaje uklanjanjem `e` i dodavanjem `i` pre broja `cinco`.

Brojevi od 1 do 9 na španskom su:

| Broj | Španski Izraz |
| :---: | :---: |
| 1 | `uno` |
| 2 | `dos` |
| 3 | `tres` |
| 4 | `cuatro` |
| 5 | `cinco` |
| 6 | `seis` |
| 7 | `siete` |
| 8 | `ocho` |
| 9 | `nueve` |
