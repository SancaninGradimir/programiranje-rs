# --description--

Koeficijent binoma $\displaystyle\binom{10}{3} = 120$.

$120 = 2^3 × 3 × 5 = 2 × 2 × 2 × 3 × 5$, i $2 + 2 + 2 + 3 + 5 = 14$.

Dakle, ukupan broj elemenata u binomnom koeficijentu od $\displaystyle\binom{10}{3}$ je $14$.

Pronađi zbir članova u binomialnom koeficijentu $\binom{20\\,000\\,000}{15\\,000\\,000}$.
