## Uvod u dinamičko programiranje

- **Definicija**: Dinamičko programiranje je algoritamska tehnika koja rešava složene probleme razbijanjem na manje, jednostavnije probleme i skladištenjem rezultata kako bi se izbegavali ponovni proračuni.
- **Preklapajući podproblemi**: Slični manji problemi često nastaju prilikom rešavanja većeg problema. Umesto da izračunavamo ove male probleme više puta, skladištimo njihovo rešenje.
- **Optimalna struktura podproblema**: Optimalno rešenje jednog problema ima optimalna rešenja njegovih manjih delova. To znači da možemo izgraditi optimalno rešenje spajanjem optimalnih rešenja manjih delova.

## Rešenje za dinamički program

- **Memoization (Metoda odozgo prema dole)**: Memoizacija čuva rezultate skupih poziva funkcija i vraća sačuvane rezultate kada se ponovi isti ulaz.

```py
def climb_stairs_memo(n, memo={}):
    """Dynamic programming with memoization"""
    # Check if we've already calculated this value
    if n in memo:
        return memo[n]  # Return cached result - O(1) lookup!
    
    # Base cases
    if n <= 2:
        return n
    
    # Calculate once and store in memo for future use
    memo[n] = climb_stairs_memo(n-1, memo) + climb_stairs_memo(n-2, memo)
    return memo[n]
```

- **Tabulacija (Metoda od dna ka vrhu)**: Tabulacija gradi rešenja od dna ka vrhu, popunjavajući tabelu rešenjima za manje izazove.

```py
def climb_stairs_tabulation(n):
    """Dynamic programming with tabulation"""
    if n <= 2:
        return n
    
    # Create array to store results for all steps from 0 to n
    dp = [0] * (n + 1)
    dp[1] = 1  # 1 way to reach step 1
    dp[2] = 2  # 2 ways to reach step 2
    
    # Build up the solution iteratively
    for i in range(3, n + 1):
        # Ways to reach step i = ways to reach (i-1) + ways to reach (i-2)
        dp[i] = dp[i-1] + dp[i-2]
    
    return dp[n]
```

## Stvarna upotreba dinamičkog programa

- **Optimizacija putanje podataka**: GPS sistemi koriste algoritme dinamičkog programiranja za pronalaženje najkraćeg puta između lokacija.
- **Obrada teksta**: Alati za proveru pravopisa i funkcija dovršavanja teksta često se oslanjaju na dinamičko programiranje za izračunavanje udaljenosti uređivanja između reči.
- **Kreiranje finansijskih modela**: Investicione strategije i optimizacija portfolija često koriste metode dinamičkog programiranja.
- **Alokacija resursa**: Problem plekovnika (knapsack) i njegove vrste nastaju u planiranju rasporeda, budžetiranju i upravljanju resursima.

## Kada se koristi dinamički program

Trebalo bi uzeti u obzir korišćenje dinamičkog programa u sledećim slučajevima:

- Problem se može razbiti na manji, ponavljajuće probleme/izazove.
- Problem ima strukturu malih problema.
- Standardno rekurzivno rešenje bi zahtevalo rekurzivne proračune.
- Potrebno je optimizovati po pitanju vremenske složenosti na uštrb prostorne složenosti.
