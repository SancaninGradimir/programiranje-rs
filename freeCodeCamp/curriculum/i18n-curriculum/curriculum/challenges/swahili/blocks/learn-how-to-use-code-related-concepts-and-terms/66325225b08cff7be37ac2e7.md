# --description--

Povezani reprezentacije kao `who`, `whom`, `which`, i na `that` se koriste za spajanje fraza povezane sa imenicama ili pridevima.

`Who` na `whom` se odnose na ljude. Razlika je u tome što `who` je subjekat glagola, dok `whom` se koristi kao glagolski element ili glagolski afiks.

`Which` koristi se za određene stvari ili životinje, i `that` može se koristiti za ljude, stvari ili životinje u opisnim klauzulama.

Primeri:

`The programmer who wrote this code is brilliant.` (`who` odnosi se na programsko okruženje)

`The artist whom we met at the gallery is internationally recognized.` (`whom` se odnosi na umetnika)

`The tool that you lent me was useful.` (`that` odnosi se na uređaj)

`The book, which I bought yesterday, is already a bestseller.` (`which` odnosi se na knjigu)

### --feedback--

`which` Obično se koristi za stvari, a ne za ljude. Ispravan pridev ovde bi trebalo da bude `who`.
