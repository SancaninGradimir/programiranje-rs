# --description--

U React-u, ažuriranje nizova u state-u je veoma jednostavno, ali se lako može napraviti greška, posebno ako dolazite iz običnog JavaScript-a gde možete direktno menjati nizove.

Katika React, stanje se mora tretirati kao nepromenljivo kako bi moglo detektovati promene i izvršiti tačne ažurmane na korisničkom interfejsu.

Istražite kako možete ispraviti redove podataka koji su u stanju u React.

Jedna od najčešćih grešaka prilikom ažuriranja nizova u React state-u jeste direktna promena niza korišćenjem metoda kao što su `push()`, `pop()` ili `splice()`. Ove metode menjaju originalni niz, a React to ne prihvata.

React se oslanja na novu referencu niza kako bi prepoznao promene, tako da direktna promena niza može sprečiti ponovno prikazivanje komponente kako se očekuje.

Evo primera korišćenja metode `push()` za dodavanje elementa u niz u state-u, što neće raditi:

```jsx
import { useState } from "react";

function ItemsList() {
  const [items, setItems] = useState([
    { id: 0, name: "Item 1" },
    { id: 1, name: "Item 2" },
    { id: 2, name: "Item 3" },
  ]);

  const addItem = () => {
    const newItem = { id: items.length + 1, name: `Item ${items.length + 1}` };
    items.push(newItem); // This modifies the original array
    setItems(items); // React will not detect this change
  };

  return (
    <div>
      <button onClick={addItem}>Add Item</button>
      <ul>
        {items.map((item) => (
          <li key={item.id}>{item.name}</li>
        ))}
      </ul>
    </div>
  );
}

export default ItemsList;
```

Kada kliknete na dugme `Add Item`, ništa se ne dešava na korisničkom interfejsu.

Takođe može biti primamljivo ukloniti elemente iz niza korišćenjem metode `pop()`:

```js
const removeItem = () => {
  items.pop(); // Modifies the original array
  setItems(items); // React will not detect this change, either
};
```

Da biste ažurirali niz u state-u, ključ je da napravite novi niz, izvršite promene i prosledite ga React-u, umesto da menjate postojeći niz.

Pošto je to novi niz, React će znati da se state promenio, što će izazvati ponovno prikazivanje komponente.

Evo kako možete dodati element u niz `items` koristeći spread operator:

```js
const addItem = () => {
  const newItem = {
    id: items.length + 1,
    name: `Item ${items.length + 1}`,
  };

  // Creates a new array
  setItems((prevItems) => [...prevItems, newItem]);
};
```

`[...prevItems, newItem]` kreira novi red kopiranjem svih stavki iz reda `items` koji se nalazi u stanju, a zatim dodaje `newItem` na kraju, što dodaje `id` i broj stavke.

Ako želite da uklonite nešto iz niza, možete koristiti metodu `filter()`, koja vraća novi niz nakon filtriranja onoga što želite ukloniti:

```js
const removeItem = (id) => {
  setItems((prevItems) => prevItems.filter((item) => item.id !== id));
};
```

Evo kompletnog koda:

```jsx
import { useState } from "react";

function ItemsList() {
  const [items, setItems] = useState([
    { id: 0, name: "Item 1" },
    { id: 1, name: "Item 2" },
    { id: 2, name: "Item 3" },
  ]);

  const addItem = () => {
    const newItem = { id: items.length + 1, name: `Item ${items.length + 1}` };
    setItems((prevItems) => [...prevItems, newItem]); // Creates a new array
  };

  const removeItem = (id) => {
    setItems((prevItems) => prevItems.filter((item) => item.id !== id)); // Creates a new array
  };

  return (
    <div>
      <button onClick={addItem}>Add Item</button>
      <ul>
        {items.map((item) => (
          <li key={item.id}>
            {item.name}{" "}
            <button onClick={() => removeItem(item.id)}>Remove</button>
          </li>
        ))}
      </ul>
    </div>
  );
}

export default ItemsList;
```

Ovo su neki uobičajeni načini na koje možete popraviti red unutar stanja.
