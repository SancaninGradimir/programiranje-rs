# --description--

`constant` U programu je slično varijabli, ali kada mu se dodeli vrednost, ta vrednost ne može biti promenjena. `Constants` Koristi se kada imate vrednosti koje znate da neće morati da se menjaju u celom programu. Često se piše velikim slovima kako bi se razlikovalo od varijabli.

Na primer, `const PI = 3.14;` U mnogim programskim jezicima, `const` se koristi za deklarisanje konstante. Ovde, `PI` je `constant` čija je vrednost dodeljena `3.14`.
