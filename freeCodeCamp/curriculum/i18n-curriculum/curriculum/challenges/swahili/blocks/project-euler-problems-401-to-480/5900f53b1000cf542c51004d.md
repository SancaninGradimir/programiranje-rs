# --description--

3-smooth broj je celi broj koji nema premični faktor veći od 3. Za celi broj $N$, definišemo $S(N)$ kao skup 3-smooth brojeva manjih ili jednakih $N$. Na primer, $S(20) = \\{1, 2, 3, 4, 6, 8, 9, 12, 16, 18\\}$.

Definisemo $F(N)$ kao broj promena $S(N)$, gde svaki element dolazi nakon svih svojih premih delitelja.

Ovo je jedan od mogućih promena za $N = 20$.

-   1, 2, 4, 3, 9, 8, 16, 6, 18, 12.

Ovo nije važeća transformacija zato što 12 dolazi pre svog delitelja 6.

-   1, 2, 4, 3, 9, 8, 12, 16, 6, 18.

Možemo potvrditi da $F(6) = 5$, $F(8) = 9$, $F(20) = 450$ i $F(1000) ≈ 8.8521816557e\\,21$.

Pronađite $F({10}^{18})$. Odgovor napišite u naučnoj notaciji, zaokružen na deset cifara posle decimalne tačke. Koristite malo slovo `e` za razdvajanje mantise i eksponenta. Na primer, ako je odgovor $112\\,233\\,445\\,566\\,778\\,899$, format odgovora bi bio `1.1223344557e17`.
