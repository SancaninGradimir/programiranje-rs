## Lažni kod

Pseudokod je pisanje logike vašeg programa prirodnim jezikom umesto koda. Pomaže u pojednostavljanju i dubokoj razradi koraka koje vaš program mora da prođe kako bi rešio problem.

Evo primera kako dummy kod programa koji štampa sve brojeve do broja unet kao ulaz može izgledati:

```md
When the user inputs a number
Initialize a counter variable and set its value to zero
While counter is smaller than user inputted number increment the counter by one
Print the value of the counter variable
```

Ovo je osnovni program koji pokazuje kako izgleda lažni kod. Biće uključeno više primera lažnog koda u segmente vredne distribucije.

## --answers--

Pseudokod je posebno koristan za početnike jer im pomaže da brzo nauče programski jezike, zaobilazeći složenu sintaksu.

---

Pseudo-kod olakšava komunikaciju sa ljudima koji nemaju programersko znanje, pomaže pri izgradnji koda, pruža odličnu osnovu za dokumentaciju i pomaže u otkrivanju grešaka.

---

Veštački kod potpuno eliminiše potrebu programera da razumeju ili koriste programski jezike, čineći ga moćnim alatom za profesionalne developere.
