## Rad sa HTML, CSS, na JavaScript

Kada HTML i CSS pružaju strukturu sajta, JavaScript unosi interaktivnost na sajt omogućavanjem složenih funkcija, kao što je rukovanje unosima korisnika, animiranje elemenata i čak izgradnja punih web aplikacija.

## Tipovi podataka u JavaScript

Tipovi podataka pomažu programu da razume vrstu podataka sa kojima radi, bilo da je to broj, tekst ili nešto drugo.

- **Broj**: Broj predstavlja cele brojeve i decimalne brojeve. Primjeri celih brojeva su 7, 19 i 90.
- **Decimalni broj**: Decimalni broj je broj koji ima decimalnu tačku. Primjeri su 3.14, 0.5 i 0.0001.
- **String**: String je sekvenca karaktera, ili tekst, koji je postavljen unutar navodnika. `"I like coding"` i `'JavaScript is fun'` su primjeri stringova.
- **Logička vrednost**: Logička vrednost predstavlja jednu od dve moguće vrednosti: `true` ili `false`. Možete koristiti logičku vrednost za predstavljanje stanja, kao što je `isLoggedIn = true`.
- **Undefined i Null**: Vrednost `undefined` je varijabla deklarisana, ali joj nije dodeljena vrednost. Vrednost `null` je prazna vrednost, ili varijabla kojoj je namerno dodeljena vrednost `null`.
- **Objekt**: Objekt je kolekcija parova ključ-vrednost. Ključ je ime svojstva, a vrednost je vrednost tog svojstva.

Ovde, predmet za `pet` ima tri svojstva ili funkcije: `name`, `age`, i `type`. Vrednost je `Fluffy`, `3`, i `dog`, u tom redosledu.

```js
let pet = {
  name: "Fluffy",
  age: 3,
  type: "dog"
};
```

- **Identifikator (Symbol)**: Tip podataka Symbol je jedinstvena i nepromenljiva vrednost koja se može koristiti kao identifikator svojstava objekta.

U ovom primeru ispod, kreirane su dve oznake sa istim opisima, ali ne odgovaraju.

:::interactive_editor

```js
const crypticKey1= Symbol("saltNpepper");
const crypticKey2= Symbol("saltNpepper");
console.log(crypticKey1 === crypticKey2); // false
```

:::

- **BigInt**: Kada je broj prevelik za tip podataka `Number`, možete koristiti tip podataka BigInt da predstavite cele brojeve bilo koje dužine.

Dodavanjem `n` na kraj broja, možete kreirati BigInt.

```js
const veryBigNumber = 1234567890123456789012345678901234567890n;
```

## Parametri u JavaScript

- Kriterijumi mogu biti objavljeni korišćenjem ključne reči `let`.

```js
let cityName;
```

- Da biste postavili vrednost za parametar, možete koristiti operator dodeljivanja `=`.

```js
cityName = "New York";
```

- Parametri definisani korišćenjem `let` mogu dobiti novu vrednost ponovo.

:::interactive_editor

```js
let cityName = "New York";
cityName = "Los Angeles";
console.log(cityName); // Los Angeles
```

:::

- Osim `let`, takođe možete koristiti `const` da deklarišete parametar. Međutim, parametr `const` više ne može dobiti novu vrednost.

```js
const cityName = "New York";
cityName = "Los Angeles"; // TypeError: Assignment to constant variable.
```

- Varijable deklarisane korišćenjem `const` se koriste za deklarisanje konstanti, koje se ne smeju menjati u celom kodu, kao što su `PI` ili `MAX_SIZE`.

## Izjave za pozivanje parametara

- Imena varijabli bi trebalo da budu razumljiva i smislena.
- Imena varijabli bi trebalo da budu u camelCase formatu, kao što su `cityName`, `isLoggedIn`, i `veryBigNumber`.
- Imena varijabli ne smeju počinjati brojem. Treba da počinju slovom, kao što su `_` ili `$`.
- Imena varijabli ne bi trebalo da sadrže razmake ili specijalne znakove, osim `_` i `$`.
- Imena varijabli ne smeju biti rezervisane ključne reči.
- Imena varijabli su osetljiva na veličinu slova (case-sensitive). `age` i `Age` su različite varijable.

## Niz karaktera i imutabilnost niza karaktera u JavaScript

- Niz znakova je sekvenca znakova koja je postavljena unutar navodnika. Možete kreirati niz znakova koristeći jednake ili dvostruke navodnike.

```js
let correctWay = 'This is a string';
let alsoCorrect = "This is also a string";
```

- Niz znakova nije promenljiv u JavaScript. To znači da kada se niz znakova kreira, ne možete promeniti znak unutar tog niza. Međutim, i dalje možete postaviti novi niz znakova sa novom vrednošću.

```js
let firstName = "John";
firstName = "Jane"; // Reassigning the string to a new value
```

## Povezivanje niza znakova u JavaScript

- Konkatenacija je proces spajanja nekoliko sekvenci karaktera ili spajanja sekvence karaktera sa varijablama koje sadrže tekst. Operater `+` je jedan od najlakših i najčešće korišćenih načina za konkatenaciju sekvenci karaktera.

:::interactive_editor

```js
let studentName = "Asad";
let studentAge = 25;
let studentInfo = studentName + " is " + studentAge + " years old.";
console.log(studentInfo); // Asad is 25 years old.
```

:::

- Ako treba da dodate ili prijedite na postojeću sekvencu karaktera, onda možete koristiti operater `+=`. Ovo je pomoć kada želite da izgradite na sekvencu karaktera dodavanjem dodatnog teksta privremeno.

:::interactive_editor

```js
let message = "Welcome to programming, ";
message += "Asad!";
console.log(message); // Welcome to programming, Asad!
```

:::

- Drugi način za spajanje sekvenci karaktera je korišćenje metode `concat()`. Ova metoda spaja dve ili više sekvenci karaktera zajedno.

:::interactive_editor

```js
let firstName = "John";
let lastName = "Doe";
let fullName = firstName.concat(" ", lastName);
console.log(fullName); // John Doe
```

:::

## Snimanje poruke za `console.log()`

- Putanja za `console.log()` se koristi za snimanje poruka u konzolu. To je važan alat za otklanjanje grešaka i testiranje vašeg koda.

:::interactive_editor

```js
console.log("Hello, World!");
// Output: Hello, World!
```

:::

## Znakovi za dvostruke navodnike u JavaScript

- Dvostruke navodne crte se koriste posebno za označavanje kraja govora. Ovo pomaže motoru JavaScript da razume razliku ličnih instrukcija, što je važno za tačno izvršenje.

```js
let message = "Hello, World!"; // first statement ends here
let number = 42; // second statement starts here
```

- Dvojne navodne oznake pomažu u sprečavanju zbunjenosti tokom izvršavanja koda i osiguravaju da su instrukcije pravilno završene.

## Komentari u JavaScript

- Svaka linija koda deklarisana kao komentar nije obrađena od strane motora JavaScript. Komentari se koriste za objašnjavanje koda, pisanje napomena ili privremeno onemogućavanje koda.
- Komentari jedne linije kreiraju se korišćenjem `//`.

```js
// This is a single-line comment and will be ignored by the JavaScript engine
```

- Višelinijski komentari se kreiraju korišćenjem `/*` za početak komentara i `*/` za kraj komentara.

```js
/*
This is a multi-line comment.
It can span multiple lines.
*/
```

## JavaScript kao jezik sa tipovima podataka koji se menjaju tokom izvršavanja

- JavaScript je jezik sa promenljivim tipovima podataka tokom izvršavanja, što znači da ne morate navoditi tip podatka parametra kada ga deklarišete. Motor JavaScript automatski određuje tip podataka na osnovu vrednosti dodeljene parametru.

```js
let error = 404; // JavaScript treats error as a number
error = "Not Found"; // JavaScript now treats error as a string
```

- Drugi jezici, kao što je C#, koji nisu tipa koji se menja tokom izvršavanja (runtime), će izazvati grešku:

```csharp
int error = 404; // value must always be an integer
error = "Not Found"; // This would cause an error in C#
```

## Korišćenje operacije `typeof`

- Operater `typeof` se koristi za proveru tipa podataka parametra. Vraća niz znakova koji ukazuje tip podataka parametra.

:::interactive_editor

```js
let age = 25;
console.log(typeof age); // "number"

let isLoggedIn = true;
console.log(typeof isLoggedIn); // "boolean"
```

:::

- Međutim, postoji poznato svojstvo u JavaScript o `null`. Operacija `typeof` vraća `"object"` na vrednosti `null`.

:::interactive_editor

```js
let user = null;
console.log(typeof user); // "object"
```
