# --description--

HTML i CSS više su povezane sa prikazivanjem informacija. Ne koriste se za pisanje logike programa. JavaScript, koje ćete naučiti u sledećem odeljku, koristi se da bi veb stranice nešto činile. Možete učiniti mnogo stvari koristeći HTML i CSS, i sigurno će vam trebati oba. U celokupnom kursu, sledeće lekcije se fokusiraju na davanju vam alata koji su potrebni za uspeh kada uđete u sadržaj JavaScript.
