# --description--

Za sve radionice i laboratorije, koristićeš editor Python od freeCodeCamp-a. Ali važno je naučiti kako da konfigurišeš Python na svoj lokalni računar.

Najlakši način za instaliranje Python na Windows-u i Mac-u je preuzimanje instalacionog fajla sa zvanične web stranice Python. Takođe ćemo razgovarati o tome kako pokrenuti Python na Linux-u kasnije u ovom kursu.

Idite `https://www.python.org/` i saведите miša preko "Downloads". Pojaviće se prozor koji prikazuje trenutnu verziju Python za vaš operativni sistem (OS).

Pokazaćemo kako instalirati Python na računar koji koristi macOS prvo:

- Kliknite na dugme koje prikazuje trenutnu verziju Python (iz prethodnog dijalog prozora) i automatski ćete početi preuzimanje instalacionog fajla `.pkg`.
- Nakon što se fajl `.pkg` uspešno preuzme, otvorite ga, a zatim kliknite na "Continue" u otvaranom prozoru.
- Nastavite da kliktate na dugme "Continue" sve dok ne stignete do sekcije "Installation Type". Tamo, kliknite na dugme "Install".
- Unesite svoju lozinku ako je potrebno, a zatim pokrenite instalaciju.
- Nakon toga, trebalo bi da dobijete poruku čestitanja koja kaže da je Python uspešno instaliran.
- Kliknite na dugme "Close", i gotovo!

Možete proveriti instalaciju otvaranjem vašeg termina i pokretanjem `python --version` ili `python3 --version`.

Takođe možete otvoriti interpretator za Python pokretanjem `python` ili `python3` u terminalu. Interpretator za Python je program koji čita kod Python, prevodi ga u instrukcije koje računar razume i izvršava te instrukcije.

<dfn>Terminal</dfn> je tekstualni interfejs koji vam omogućava da interagujete sa svojim računarom pisanjem komandi. Svaki operativni sistem dolazi sa osnovnim terminal programom. Na macOS-u, možete koristiti Terminal aplikaciju. Na Windowsu, možete koristiti Command Prompt ili PowerShell. U Linuksu, svako desktop okruženje ima svoj osnovni terminal program, kao što su GNOME Terminal ili Konsole.

Zapamtite da, u nekim starijim sistemima macOS i Linux, `python` može biti sačuvan za Python 2, dok je `python3` za Python 3 posebno. Ako pokrećete `python --version` i vidite verziju od Python 2 kao `Python 2.7.18`, onda je moguće da vaš sistem zavisi od nekog programa razvijenog za stariju verziju Python. Ako je tako, trebalo bi vam da koristite `python3` da pokrenete vaš kod Python od sada. Python 2 je kraj njegovog životnog veka i ne bi trebalo koristiti za bilo kakav novi razvoj.

Instaliranje Python na Windows-u, pratite ove korake:

- Idite na `https://www.python.org/` i sačekajte miša iznad „Downloads“. Trebalo bi da vidite prozor koji kaže "Download for Windows" i dugme za preuzimanje sa trenutnom verzijom Python.
- Kliknite na broj verzije, i automatski ćete početi da preuzimate izvršnu datoteku za Windows (`.exe`).
- Kada završite preuzimanje instalacione datoteke Python za Windows, dvaput kliknite na nju i sledite uputstva.
- Kada vidite opciju `Add python.exe to Path`, označite tu opciju, a zatim kliknite na `Install Now`. To će vam olakšati stvari u budućnosti.

Možete potvrditi instalaciju otvaranjem shell-a komandne linije kao što je PowerShell i pokretanjem `python --version`. Takođe možete otvoriti interpretator za Python pokretanjem `python`.

Na Python na Linuxu, glavne distribucije kao što su Ubuntu, Debian i Fedora dolaze sa Python već.
