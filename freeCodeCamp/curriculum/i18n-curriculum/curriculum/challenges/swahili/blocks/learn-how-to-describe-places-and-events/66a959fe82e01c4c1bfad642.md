# --explanation--

`Might` koristi se za pokazivanje mogućnosti. To znači da je nešto moguće, ali nije potvrđeno. Na primer:

- `I might go to the party if I finish my work.` - Možda ću ići, ali još uvek nisam potvrdio/potvrdila.

- `She might call you later.` - Moguće je da će pozvati, ali nije sigurno.
