# --interactive--

Njia ya `map` je moćna funkcija i široko korišćena u JavaScript koja radi na redovima podataka. Dizajnirana je da kreira novi red podataka koristeći određenu funkciju na svakom elementu originalnog reda.

Ova metoda ne menja originalni red, već vraća novi red sa rezultatom funkcije koja je korišćena na svakom elementu.

Ovo je primer korišćenja metode `map` u redu brojeva:

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];
const doubled = numbers.map((num) => num * 2);
console.log(numbers); // [1, 2, 3, 4, 5]
console.log(doubled); // [2, 4, 6, 8, 10]
```

:::

Da bismo kreirali novi red u kojem je svaki broj pomnožen sa dva, koristimo metodu `map`. Metoda `map` prihvata funkciju povratnog poziva (callback), gde se ta funkcija izvršava na svakom elementu reda.

U ovom slučaju, svaki broj u nizu biće pomnožen sa `2`. Rezultat će biti novi niz brojeva `2, 4, 6, 8, 10`.

Funkcija povratnog poziva može da primi do tri argumenta.

Prvi argument predstavlja element koji se trenutno obrađuje.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element) => {
  console.log("Element:", element);
  return element * 2;
});
```

:::

Drugi argument je indeks elementa koji se trenutno obrađuje.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index) => {
  console.log("Element:", element);
  console.log("Index:", index);
  return element * 2;
});
```

:::

Treći argument je niz nad kojim je pozvana metoda `map`.

:::interactive_editor

```js
const numbers = [3, 4, 5, 6, 7].map((element, index, array) => {
  console.log("Element:", element);
  console.log("Index:", index);
  console.log("Array:", array);
  return element * 2;
});
```

:::

Razumevanje i efikasna upotreba metode `map` mogu značajno poboljšati tvoju sposobnost rada sa nizovima u JavaScript-u. U narednim lekcijama detaljnije ćemo se baviti naprednom upotrebom metode `map` i istražiti kako ona može biti moćan alat za izgradnju efikasnih i robusnih aplikacija.
