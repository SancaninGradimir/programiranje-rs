# --description--

Ako želiš da imaš listu stavki čiji red nije važan, kao što je lista za kupovinu stvari koje se mogu kupiti u bilo kom redosledu, onda možeš koristiti neuređenu listu.

Neredovni spiskovi se kreiraju korišćenjem elementa `<ul>`, i svaki element unutar spiska kreira se korišćenjem elementa stavke spiska `<li>`.
