# --interactive--

Prethovna obuka vas je upoznala sa `console.log()`, ali ova lekcija će detaljnije pokriti njenu svrhu i upotrebu.

Katika JavaScript, `console.log()` je jednostavan, ali moćan alat koji se koristi za prikazivanje poruka ili pružanje informacija u konzoli pretraživača. Obično ga koriste programeri da pronađu greške i ispitaju kod dok rade na svojim aplikacijama.

Možete koristiti `console.log()` da napišete tekst ili parametre u konzoli i osigurate da vaš kod radi ispravno.

Da biste koristili `console.log()`, pozivate ovu funkciju postavljanjem vrednosti ili poruke koju želite da izbacite unutar običnih zagrada. Evo nekoliko primera:

:::interactive_editor

```js
console.log("Hello, world!");

let num = 5;
console.log(num); // 5
```

:::

Prvi primer štampa `Hello, world!` u konzoli pregledača, dok drugi primer štampa vrednost `5`.

Evo još jednog primera kako raditi sa `console.log()`:

:::interactive_editor

```js
let name = "Alice";
console.log("Hello, " + name + "!"); // Hello, Alice!
```

:::

Takođe možete proslediti više vrednosti za `console.log()`, razdvojene zarezima. Primer:

:::interactive_editor

```js
let name = "Alice";
let age = 25;
console.log("Name:", name, "Age:", age); // Name: Alice Age: 25
```

:::

Ovo je pomoć za pisanje više izveštaja istovremeno.

`console.log()` vam pomaže da pratite kako se vaš kod izvršava, čineći ga lakšim za pronalaženje grešaka i razumevanje kako vaša aplikacija funkcioniše.
