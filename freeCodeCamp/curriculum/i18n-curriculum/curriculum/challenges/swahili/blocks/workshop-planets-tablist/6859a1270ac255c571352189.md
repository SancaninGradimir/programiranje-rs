# --description--

Sada je vreme da kreiramo strukturu za svaki od panela koji će prikazivati činjenice o svakoj planeti.

Počnite dodavanjem tri `div` elementa, svaki sa atributom `role` postavljenim na `"tabpanel"`. Unutar svakog `div` elementa, treba da postoji `p` element.

# --hints--

You should have three `div` elements with a `role` attribute set to `"tabpanel"`.

```js
assert.lengthOf(document.querySelectorAll('div[role="tabpanel"]'), 3);
```

Each `tabpanel` element should have a `p` element inside it.

```js
assert.lengthOf(document.querySelectorAll('div[role="tabpanel"] p'), 3);
```
