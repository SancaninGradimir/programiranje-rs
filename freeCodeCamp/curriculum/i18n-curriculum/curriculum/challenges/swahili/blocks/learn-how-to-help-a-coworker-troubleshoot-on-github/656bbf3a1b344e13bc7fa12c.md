# --description--

`Strange` znači nešto neobično ili neočekivano. To je kao da pronađeš nešto drugačije od onoga što obično vidiš. Na primer: `A strange sound in a car might mean a problem.`

`error` je greška, posebno ona koja uzrokuje da nešto ne radi kako treba. Na primer: `An error in a code means there is a mistake that needs fixing.`

# --questions--

## --text--

## --answers--

Nova funkcija i uspešni rezultati

### --feedback--

`Strange` ukazuje na nešto neočekivano, a `error` pokazuje problem, ne uspeh.
