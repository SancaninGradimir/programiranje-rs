# --description--

Za prirodni broj $n$, definiše se $C(n)$ kao broj celih brojeva $x$, za koje je $1 < x < n$ i $x^3 \equiv 1\bmod n$.

Kada $n = 91$, postoji 8 mogućih vrijednosti za $x$, tj.: 9, 16, 22, 29, 53, 74, 79, 81. Stoga, $C(91) = 8$.

Pronađi zbir pozitivnih brojeva za koje je $C(n)=242$, u opsegu $n \leq {10}^{11}$.
