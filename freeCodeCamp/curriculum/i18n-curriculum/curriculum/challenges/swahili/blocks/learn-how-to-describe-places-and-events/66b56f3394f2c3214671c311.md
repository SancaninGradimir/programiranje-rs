# --explanation--

`To roll out` To znači lansirati ili pokrenuti nešto novo, kao što je proizvod, usluga ili projekat. Često se koristi u kontekstu biznisa i tehnologije. Na primer:

- `The company plans to roll out a new software update next month.` - Kompanija će lansirati ažuriranje sledećeg meseca.

`By the end of` koristi se za postavljanje krajnjeg roka ili prikaz vremena do kojeg nešto mora biti završeno. Primer:

- `I need to finish this report by the end of the day.` - Izveštaji moraju biti završeni pre kraja dana.
