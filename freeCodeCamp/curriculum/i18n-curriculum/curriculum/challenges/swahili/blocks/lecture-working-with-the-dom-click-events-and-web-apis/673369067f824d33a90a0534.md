# --interactive--

Kada korisnik klikne dugme ili dođe do promene u formi, to se naziva događaj. U vašim aplikacijama biće potrebno da imate način da slušate ove događaje i reagujete na njih.

Metoda `addEventListener()` koristi se za osluškivanje događaja. Prima dva argumenta: događaj koji želite da slušate i funkciju koja će biti pozvana kada se događaj desi.

Evo osnovne sintakse:

```js
element.addEventListener("event", listener);
```

Element je HTML element koji se prati zbog događaja, a događaj opisuje vrstu događaja koji se osluškuje, kao što je `"click"`.

Osluškivač je objekat koji prima informacije kada se događaj desi. Najčešće je to funkcija koju definišete za obradu događaja.

Evo primera:

```js
element.addEventListener("click", () => {
  // code to run when the click event occurs
});
```

Takođe možete izabrati da prosledite referencu funkcije na ovaj način:

```js
function handleClick() {
  // code to run when the click event occurs
}

element.addEventListener("click", handleClick);
```

Ponekad želite da kreirate različitu funkciju za rukovanje događajem. To može učiniti vaš kod lakšim za čitanje i održavanje.

Argument slušaoca takođe može biti `null`, ili može biti nešto što koristi interfejs `EventListener`. Interfejs `EventListener` opisuje metodu nazvanu `handleEvent()`. Ova metoda se automatski poziva svaki put kada se dogodi događaj koji se sluša. Korišćenjem ovog interfejsa omogućava se jednoj stvari da rukuje sa više događaja, ako je potrebno.

U većini slučajeva nećete morati da koristite ovaj interfejs, već ćete proslediti funkciju koja će biti pozvana kada se događaj desi.

Sada pogledajmo primer da bismo bolje razumeli kako ovo funkcioniše:

U ovom primeru imamo `button` element sa identifikatorom `btn`:

```html
<button id="btn">Show alert</button>
```

Želimo da pratimo događaj `"click"` na ovom dugmetu i prikažemo upozorenje kada se dugme klikne.

Prvo moramo dobiti pristup funkciji `button` u našem kodu JavaScript. Zatim moramo dodati slušalicu događaja da prati događaj klika i prikaže upozorenje kada se dugme klikne:

:::interactive_editor

```html
<button id="btn">Show alert</button>
<script src="index.js"></script>
```

```js
const btn = document.getElementById("btn");

btn.addEventListener("click", () => alert("You clicked the button"));
```

:::

Sada, svaki put kada se klikne na dugme, korisnik će videti upozoravajuću poruku prikazanu na ekranu kao ovo:

```md
You clicked the button
```

Drugi tip događaja koji možete pratiti je događaj `"input"`. Ovaj događaj se pokreće kada se vrednost elementa unosa promeni.

Evo primera:

```html
<input type="text" id="input" placeholder="Type something" />
```

Želimo da pratimo događaj unosa na ovom polju za unos i da snimamo vrednost unosa u konzolu svaki put kada korisnik nešto ukuca.

Prvo moramo doći do pristupa funkcionalnosti `input` u našem kodu JavaScript. Zatim moramo dodati slušaoca događaja da prati događaj unosa i snima vrednost unosa u konzolu svaki put kada korisnik nešto ukuca:

:::interactive_editor

```html
<input type="text" id="input" placeholder="Type something" />
<script src="index.js"></script>
```

```js
const input = document.getElementById("input");

input.addEventListener("input", () => {
  console.log(input.value);
});
```

:::

Postoji mnogo više događaja koje možete slušati koristeći način `addEventListener()`. Neki su uobičajeni događaji `mouseover`, `mouseout`, `keydown`, `keyup`, i `submit`.

U narednim lekcijama obradićemo još događaja i kako koristiti metodu `addEventListener()` za njihovo praćenje.
