# --description--

Definišemo zaokruženi kvadratni koren pozitivnog celog broja $n$ kao kvadratni koren broja $n$ zaokružen na najbliži ceo broj.

Sledeći metod (posebno Heronova metoda modifikovana za integerne kalkulacije) pronalazi približnu kvadratnu korenu $n$:

Neka je $d$ broj cifara broja $n$.

Ako je $d$ neparan broj, postavite $x_0 = 2 × {10}^{\frac{d - 1}{2}}$.

Ako je $d$ paran broj, postavite $x_0 = 7 × {10}^{\frac{d - 2}{2}}$.

Ponavlaj:

$$x_{k + 1} = \left\lfloor\frac{x_k + \left\lceil\frac{n}{x_k}\right\rceil}{2}\right\rfloor$$

hadi $x_{k + 1} = x_k$.

Primer, hajde da nađemo kvadratni koren od $n = 4321$.

$n$ ina tarakimu 4, kwa hiyo $x_0 = 7 × {10}^{\frac{4-2}{2}} = 70$.

$$x_1 = \left\lfloor\frac{70 + \left\lceil\frac{4321}{70}\right\rceil}{2}\right\rfloor = 66 \\\\
x_2 = \left\lfloor\frac{66 + \left\lceil\frac{4321}{66}\right\rceil}{2}\right\rfloor = 66$$

Pošto je $x_2 = x_1$, ovde se zaustavljamo. Dakle, nakon samo dve iteracije dobili smo da je zaokruženi kvadratni koren broja 4321 jednak 66 (stvarni kvadratni koren je 65.7343137…).

Broj iteracija potrebnih za ovu metodu iznenađujuće je mali. Na primer, možemo pronaći obrnuti kvadratni koren petocifrenog celog broja ($10\\,000 ≤ n ≤ 99\\,999$) uz prosečno 3.2102888889 iteracija (prosečna vrednost zaokružena na 10 decimalnih mesta).

Koristeći gore opisani metod, koji je prosečan broj iteracija potreban za dobijanje zaokruženog kvadratnog korena broja sa 14 cifara (${10}^{13} ≤ n < {10}^{14}$)? Zaokružite odgovor na 10 decimalnih mesta.

**Napomena:** Oznake $⌊x⌋$ i $⌈x⌉$ predstavljaju funkciju poda i funkciju plafona.
