# --description--

Proverite da li je `length` potpun pozitivan broj koristeći vašu funkciju `isEven`. Ako je tako, vratite prosečnu vrednost broja iz polja `middle` i sledećeg broja. Ako nije potpun pozitivan broj, vratite broj iz polja `middle` – moraćete da povećate vrednost `middle`.

# --hints--

Trebalo bi da vratiš vrednost iz funkcije `median`.

```js
assert.exists(median([2,4,6,8]));
```

```js
assert.strictEqual(median([2,4,6,8]),5);
```

```js

assert.strictEqual(median([3,6,9,12,15]),9);
```
