# --interactive--

**NAPOMENA**: Neki primeri interakcije mogu koristiti CSS koje još niste naučili. Ne brinite zbog razumevanja celog koda. Cilj ovih primera je da vam pokaže pregled ovih koncepata strukture kako biste bolje razumeli kako stvari funkcionišu. Da biste videli pregled, moraćete da omogućite interaktivni editor.

Progressivno otkrivanje je dizajnerski obrazac koji se koristi da pokaže korisnicima samo sadržaj vezan za njihove trenutne aktivnosti, skrivajući sve ostalo. Ovo se radi kako bi se smanjilo kognitivno opterećenje i učinilo da korisničko iskustvo bude lakše za razumevanje.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="progressive-disclosure">
  <button id="show-details-btn" aria-expanded="false" aria-controls="extra-content">
    Show More Details
  </button>
  <div id="extra-content" class="hidden" tabindex="-1">
    <p>
      Here are additional details that are revealed only when you want to see them.
      This keeps the interface clean and reduces cognitive load.
    </p>
  </div>
</div>
<script src="index.js"></script>

```

```js
const btn = document.getElementById('show-details-btn');
const content = document.getElementById('extra-content');

btn.addEventListener('click', () => {
  const isHidden = content.classList.toggle('hidden');
  btn.setAttribute('aria-expanded', !isHidden);
  if (!isHidden) {
    content.focus();
  }
});

```

Na primer, kada koristiš Google pretragu, većina pretraga je jednostavna. Zato ima smisla imati jednostavan korisnički interfejs sa malo opcija. Međutim, u retkim slučajevima kada ti je potrebna napredna pretraga, Google nudi opciju napredne pretrage koja prikazuje dodatne mogućnosti.

Google-ovu opciju napredne pretrage možeš pronaći u meniju podešavanja na početnoj stranici Google-a ili otići na `google.com/advanced_search`.

Još jedan primer progresivnog otkrivanja je dugme „Više detalja“ na proizvodima u internet prodavnici. Kada posetiš sajt kao što je Amazon, videćeš listu slika proizvoda raspoređenih po kategorijama. To omogućava korisnicima da jednostavno kupuju bez preopterećenosti informacijama.

Ako je korisnik zainteresovan za određeni proizvod, može kliknuti na njegovu sliku kako bi video više detalja. Prikazivanje svih informacija odjednom može ga preopteretiti. Zato progresivno otkrivanje omogućava korisniku da vidi više informacija tek kada to želi.

```js
document.querySelectorAll('.details-btn').forEach(button => {
  button.addEventListener('click', () => {
    const product = button.closest('.product');
    const details = product.querySelector('.product-details');
    const isHidden = details.classList.toggle('hidden');
    product.setAttribute('aria-expanded', !isHidden);
    if (!isHidden) {
      details.focus();
      button.textContent = 'Hide details';
    } else {
      button.textContent = 'More details';
    }
  });
});

```
