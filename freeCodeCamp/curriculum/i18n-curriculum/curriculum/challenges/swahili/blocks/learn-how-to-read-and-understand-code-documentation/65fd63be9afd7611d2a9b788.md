# --description--

Deo `introductory` je početak dokumenta koji identifikuje glavnu temu. Primer: `The introductory chapter of the book outlines the key concepts.`

`paragraph` je deo teksta o nekoj temi. Kao što je, `Each paragraph in the blog post focuses on a different technology trend.`

`To cover` znači obrađivanje ili uključivanje specifičnog sadržaja. Na primer, u tehničkom vodiču, `This section covers the steps to install software.`
