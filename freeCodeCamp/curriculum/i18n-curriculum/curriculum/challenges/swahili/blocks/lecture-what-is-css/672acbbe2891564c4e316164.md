# --interactive--

CSS odnosi se na stilove veb stranice. Svi ovi stilovi su kreirani različitim pravilima za CSS.

Zakon o CSS ima dvije glavne dijelove: **opcije** i dio **deklaracija**.

Istražimo osnovnu sintaksu:

```css
selector {
  property: value;
}
```

Selektor je struktura koja se koristi u CSS za identifikovanje i ciljanje specifičnih elemenata od HTML radi dekoracije.

Primeri selektora uključuju selektore tipa, selektore klase i ID selektore.

Okrugle zagrade prisutne u osnovnoj sintaksi poznate su kao deo izjave. Deo izjave koristi skup obrazaca za određene selektore, ili više selektora.

Unutar deklaracionog dela, biće niz deklaracija. Svaka deklaracija ima svojstvo i vrednost.

Svojstvo je CSS identifikator koji određuje šta se stilizuje. Primer svojstva je `background-color`.

Vrednost je konkretna postavka za to svojstvo. Na primer, ako je svojstvo `background-color`, vrednost može biti `purple`, čime se pozadina postavlja na ljubičastu boju.

Nakon svakog naziva svojstva moraš staviti dvotačku, a nakon svake vrednosti tačku-zarez.

Sada kada znaš sintaksu CSS pravila, pogledajmo primer. Otvori interaktivni editor i klikni na karticu `styles.css` da vidiš CSS kod.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1>Learning about CSS</h1>

<p>Example paragraph element</p>
<p>Another example paragraph element</p>
```

```css
p {
  color: blue;
}
```

:::

U ovom pravilu za CSS, selektor tipa cilja na sve elemente paragrafa na stranici.

Unutar dela deklaracije, postoji jedna izjava.

Deklaracija ima atribut od `color` i vrednost postavljenu na `blue`. Ovo će promeniti boju teksta za sve elemente paragrafa na `blue`.

Ako želite koristiti isti set stilova za više odabira, možete kreirati listu opcija gde je svaka opcija razdvojena zarezom.

Evo primera dekorisanja više opcija (za interakciju sa primerom, uključite interaktivni editor):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1 id="title">Example heading</h1>
<h2 class="subheading">Example subheading</h2>
<p>This paragraph is not affected by the selector.</p>
```

```css
#title,
.subheading {
  color: navy;
}
```

:::

U ovoj listi selektora, postoji selektor za `id` koji cilja element HTML sa vrednošću `id` od `title`. Svi selektori za `id` moraju početi sa znakom haša `#`.

Zatim sledi zarez, pa selektor klase koji cilja sve HTML elemente čija je vrednost atributa `class` jednaka `subheading`. Svi selektori klase moraju počinjati tačkom `.`.

Svi selektori sa liste dobiće isti stil, a boja teksta biće `navy`.
