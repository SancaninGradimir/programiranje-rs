# --description--

React pruža mnogo ugrađenih hook-ova koji vam omogućavaju da implementirate različite funkcije u vašim projektima. Ovo uključuje `useState`, `useEffect`, `useContext`, i drugi.

Aliako, biće vam potrebno dodati funkciju koja nema ni jedan ugrađen hook koji bi mogao da pomogne. Srećom, možete kreirati sopstvene custom hooks u React.

Custom hooks nisu tako komplikovani koliko izgledaju. To je ponovno upotrebljiva funkcija koja vam omogućava dijeljenje logike između više komponenti. To znači da je ponovna upotrebljivost još jedan razlog zašto biste željeli sami napraviti svoj hook.

Sa *custom hook*-om, možete izdvojiti logiku iz bilo kog dela koji ga koristi, kao što je dohvaćanje podataka, upravljanje stanjem, menjanje stanja, nuspojave poput praćenja statusa korisnika da li je online ili offline, i slično.

Zatim možete koristiti taj import hook na bilo kom mestu, tako da možete pažljivo prikazati i predstaviti unutar tih sekcija. To znači malo ponavljanje i malo kopiranje, što znači nekoliko delova gde možete izvršiti izmene kada želite da napravite bilo kakvo ažuriranje.

Sada, hajde da istražimo kako možete kreirati sopstveni/svojstveni custom hook.

U React, svi ugrađeni hookovi počinju sa rečju `use`, pa vaš custom hook mora pratiti isti konvencioni sistem. Ime vašeg custom hook-a takođe bi trebalo jasno da objasni šta radi.

Dakle, ako je tvoj custom hook...

- hupata data, unaweza kuitaja `useFetch`

- Promena stvari u on/off stanju, možeš ga pomenuti `useToggle`

- ili ako implementira debouncing, `useDebounce` je lijep naziv

Želimo da napravimo custom hook za dodavanje debouncinga u vašu aplikaciju.

Debounsing je programska tehnika koja smanjuje učestalost izvršavanja funkcije. Radi tako što čeka da korisnik prestane sa radnjom na određeni vremenski period pre nego što izvrši funkciju. Na primer, u polju za pretragu, umesto da poziva API za svaki pritisnut karakter, debounsing čeka dok korisnik ne prestane sa kucanjem na, recimo, 500 milisekundi.

Da biste kreirali debouncing custom hook, prvo je potrebno kreirati fajl `useDebounce.jsx` ili `useDebounce.js`. Po standardu, fajlovi custom hook-ova koje kreirate čuvaju se u fascikli `hooks`.

Možete koristiti neke ugrađene hook-ove unutar sopstvenog custom hook-a. Za debouncing su vam potrebni `useState` i `useEffect` hook-ovi, zato ih importujte na vrhu fajla:

```js
import { useState, useEffect } from "react";
```

Sledeće, kreiraj funkciju za `useDebounce` koja prima `value` i `delay` kao parametre. `value` je resurs za koji želiš da sačekaš, a `delay` je vreme čekanja. Pošto želiš da sačekaš određeno vreme, funkcija `setTimeout` i `clearTimeout` će biti korisne:

```js
function useDebounce(value, delay) {
  const [debouncedValue, setDebouncedValue] = useState(value);

  useEffect(() => {
    const handler = setTimeout(() => {
      setDebouncedValue(value);
    }, delay);

    return () => {
      clearTimeout(handler);
    };
  }, [value, delay]);

  return debouncedValue;
}

export { useDebounce };
```

Stanje `debouncedValue` čuva i vraća kašnjavu vrednost, koja se menja samo nakon definisanog perioda čekanja.

`useEffect` je to mesto gde prava magija nastaje. Kao što ste zapamtili sa prošle lekcije, sve što se dešava van ciklusa prikazivanja od React, kao što su postavljanje i brisanje alarma, je sporedna akcija, i treba da koristite hook za `useEffect` da ih pokrenete.

Unutar hooka ``useEffect`` ovde, koristiš ``setTimeout`` da postaviš ``debouncedValue``. Zatim vraćaš funkciju čišćenja koja koristi ``clearTimeout`` za brisanje prethodno vremena čekanja svaki put kada ``value`` ili ``delay`` promeni, ili deo bude uklonjen.

I ovde postoji deo `FootballerSearch` koji koristi hook `useDebounce` da odloži pretragu za 1 sekund nakon što korisnik prestane da kuca:

```jsx
import { useState, useEffect } from "react";
import { useDebounce } from "./hooks/useDebounce";
import footballers from "./footballers";

const FootballerSearch = () => {
  const [query, setQuery] = useState("");
  const debouncedQuery = useDebounce(query, 1000); // Start searching 1 second after the user stops typing

  useEffect(() => {
    if (debouncedQuery) {
      const results = footballers.filter((footballer) =>
        footballer.toLowerCase().includes(debouncedQuery.toLowerCase()),
      );
      console.log("Search results:", results);
    } else {
      console.log("Search results: []");
    }
  }, [debouncedQuery]);

  return (
    <>
      <h1 style={{ textAlign: "center" }}>Footballer Search App</h1>
      <div style={{ textAlign: "center" }}>
        <input
          style={{ padding: "0.5rem", width: "30%" }}
          type="text"
          value={query}
          onChange={(e) => setQuery(e.target.value)}
          placeholder="Search for a footballer..."
        />
      </div>
    </>
  );
};

export default FootballerSearch;
```

Kao što možete da vidite, parametar `debouncedQuery` omogućava hook `useDebounce` sa stanjem argumenta pretrage (ono što korisnik kuca), i odlaganje od 1.000 milisekundi, ili 1 sekundu. Sama pretraga se obrađuje unutar hook-a `useEffect`, a rezultati pretrage se pišu u konzolu.
