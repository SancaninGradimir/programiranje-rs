# --description--

Postoji samo deset načina da se izabere tri od pet, 12345:

<div style='text-align: center;'>123, 124, 125, 134, 135, 145, 234, 235, 245, i 345</div>

U kombinacijama, koristimo simbol, $\\displaystyle \\binom 5 3 = 10$

Generalno, $\\displaystyle \\binom n r = \\dfrac{n!}{r!(n-r)!}$, gde je $r \\le n$, $n! = n \\times (n-1) \\times ... \\times 3 \\times 2 \\times 1$, i na $0! = 1$.

Do $n = 23$, vrednost prelazi milion: $\\displaystyle \\binom {23} {10} = 1144066$.

Koliko, ne mora da budu različite, vrednosti $\\displaystyle \\binom n r$ za $1 \\le n \\le 100$, koje su veće od miliona?

```js
function combinatoricSelections(limit) {
    const factorial = n =>
        Array.apply(null, { length: n })
            .map((_, i) => i + 1)
            .reduce((p, c) => p * c, 1);

    let result = 0;
    const nMax = 100;

    for (let n = 1; n <= nMax; n++) {
        for (let r = 0; r <= n; r++) {
            if (factorial(n) / (factorial(r) * factorial(n - r)) >= limit)
                result++;
        }
    }

    return result;
}
```
