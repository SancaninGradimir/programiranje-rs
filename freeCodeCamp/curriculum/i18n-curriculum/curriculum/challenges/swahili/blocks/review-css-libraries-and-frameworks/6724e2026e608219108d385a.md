## Sistemi rada za CSS

- **Radni sistemi CSS**: Radni sistemi CSS mogu ubrzati vaš radni proces, stvoriti dosledan vizuelni stil na celoj veb stranici, učiniti da vaš dizajn izgleda konzistentno u mnogim pretraživačima i dati vašem kodu CSS bolju organizaciju.
- **Popularni radni sistemi CSS**: Neki popularni radni sistemi CSS su Tailwind CSS, Bootstrap, Materialize i Foundation.
- **Mogući nedostaci**:
  - CSS koji pruža radni sistem može izazvati konflikt sa vašim standardnim CSS.
  - Vaš sajt može izgledati kao drugi sajtovi koji koriste isti sistem.
  - Veliki sistemi mogu uzrokovati probleme sa performansama.

## Dve vrste radnih sistema za CSS

- **Sistemi funkcija CSS koji zavise od korišćenja malih klasa**: Ovi sistemi imaju male klase sa specifičnim namenama, kao što su postavljanje ivice (border), unutrašnjeg prostora ili pozadinske boje. Ove male klase možete primeniti direktno na elementima HTML kako je potrebno. Tailwind CSS se smatra sistemom funkcija CSS koji zavisi od korišćenja malih klasa.

Ovo je primer korišćenja Tailwind CSS za stilizovanje dugmeta.

```html
<button class="bg-blue-500 text-white font-bold py-2 px-4 rounded-full hover:bg-blue-700">
  Button
</button>
```

- **Okviri za CSS bazirani na komponentama**: Ovi okviri imaju gotove komponente sa unapred definisanim stilovima koje možete dodati na vaš veb sajt. Ove komponente su dostupne u zvaničnoj dokumentaciji okvira CSS, i možete ih prilagoditi i integrisati u svoj projekat. Bootstrap se smatra komponentno baziranim okvirom CSS.

Evo primera korišćenja Bookstrapa za kreiranje grupe stavki. Umesto korišćenja malih klasa na elementima `HTML`, dodati ćete ceo deo, uključujući i strukturu `HTML`.

```html
<div class="card" style="width: 25rem;">
  <ul class="list-group list-group-flush">
    <li class="list-group-item">HTML</li>
    <li class="list-group-item">CSS</li>
    <li class="list-group-item">JavaScript</li>
  </ul>
</div>
```

## Tailwind CSS

Tailwind je okvir za CSS koji zavisi od korišćenja malih klasa. Umesto pisanja standardnih pravila CSS, gradite svoje strukture kombinovanjem malih utilitarnih klasa direktno u vaš HTML.

### Alat za dinamički dizajn

Tailwind koristi pseudo-elemente kao `sm:`, `md:`, i `lg:` za primenu stilova za različite veličine ekrana.

```html
<div class="w-full md:w-1/2 lg:flex-row">Responsive layout</div>
```

### Alati za Flexbox

Klase kao `flex`, `flex-col`, `justify-around`, i `items-center` olakšavaju kreiranje dinamičkih podešavanja.

```html
<div class="flex flex-col md:flex-row justify-around items-center">
  <p>Column on small screens</p>
  <p>Row on medium and larger screens</p>
</div>
```

### Alat za Grid

Tailwind uključuje alate za CSS Grid, kao što su `grid`, `grid-cols-1`, i `md:grid-cols-3`.

```html
<div class="grid grid-cols-1 md:grid-cols-3 gap-8">
  <div class="bg-gray-100 p-4">Column 1</div>
  <div class="bg-gray-100 p-4">Column 2</div>
  <div class="bg-gray-100 p-4">Column 3</div>
</div>
```

### Alati za svemir

Alati kao `mt-8`, `mx-auto`, `p-4`, na `gap-4` pomaže u kreiranju odgovarajućih prostora bez pisanja CSS.

```html
<div class="mt-8 p-4 bg-indigo-600 text-white">Spaced content</div>
```

### Alati za tipografiju

Alat kao `uppercase`, `font-bold`, `font-semibold`, i na `text-4xl` kontroliše izgled teksta.

Možete postaviti promenljivu veličinu fonta u sekcijama za prekid/razdvajanje, kao što su `text-3xl` `md:text-5xl`.

```html
<h1 class="text-3xl md:text-5xl font-semibold text-center">Responsive Heading
</h1>
```

### Boje i stanja za hover

Tailwind nudi mnoge boje, kao što su `text-red-700`, `bg-indigo-600`, i `bg-gray-100`.

Klase kao `hover:bg-pink-600` olakšavaju interakcione efekte.

```html
<a href="#" class="bg-pink-500 hover:bg-pink-600 text-white px-4 py-2 rounded-md">
  Hover Me
</a>
```

### Granice, pete, i uticaj

- **Granice**: `border-2 border-red-300` dodaje granice sa definisanom debljinom i bojom.
- **Prsten/Krug**: `ring-1 ring-gray-300` kreira kružne efekte koji se često koriste za pažnju ili kartice.
- **Zaobljeni i prošireni uglovi**: Klase kao što su `rounded-md`, `rounded-xl`, i `scale-105` dodaju estetiku.

```html
<div class="p-6 rounded-xl ring-2 ring-fuchsia-500 scale-105">
  Highlighted card
</div>
```

### Bojni prelaz

Tailwind podržava alate za gradijente boja, kao `bg-gradient-to-r from-fuchsia-500 to-indigo-600`.

```html
<div class="p-4 text-white bg-gradient-to-r from-fuchsia-500 to-indigo-600">
  Gradient background
</div>
```

## Početni procesori za CSS

- **Preprocesor za CSS**: Preprocesor CSS je alat koji povećava sposobnost standardnog CSS. Analizira kod sa proširenom sintaksom i kompajlira ga u stvarni fajl CSS. Može pomoći pri pisanju čistog, ponovo upotrebljivog, neponavljajućeg i skalabilnog koda za složene projekte CSS.
- **Funkcionalnosti**: Neke funkcionalnosti koje pružaju preprocesori CSS su parametri, mixini, interne organizacije i nasleđivanje selektora.
- **Popularni preprocesori za CSS**: Neki popularni preprocesori za CSS su Sass, Less i Stylus.  
- **Mogući nedostaci**:
  - Obrada pravila CSS u standardni CSS može izazvati dodatno opterećenje.
  - Procesirani kod može biti težak za debagovanje (pronalaženje grešaka).

## Sass

- **Sass**: To je jedan od popularnih preprocesora za CSS. Sass znači "Syntactically Awesome Style Sheets."
- **Funkcionalnosti koje podržava Sass**: Sass podržava funkcije kao što su varijable, ugnježđena pravila za CSS, moduli, mixini, nasleđivanje i osnovni operateri kalkulacije.

## Dve sintakse koje Sass podržava

- **SCSS Sintaksa**: SCSS (Sassy CSS) dodaje osnovnu sintaksu za CSS. To je najčešće korišćena sintaksa za Sass. SCSS fajlovi imaju prefiks `.scss`.

Ovo je primer definisanja i korišćenja varijable u SCSS-u.

```scss
$primary-color: #3498eb;

header {
  background-color: $primary-color;
}
```

- **Sintaksa zasnovana na uvlačenju**: Sintaksa zasnovana na uvlačenju bila je originalna sintaksa jezika Sass i poznata je kao „Sass sintaksa“.

Ovo je primer definisanja i korišćenja parametra u poziciono strukturiranoj sintaksi.

```sass
$primary-color: #3498eb

header
  background-color: $primary-color
```

### Mixin

- **Mixin**: Mixin vam omogućava da kombinujete više svojstava od CSS i njihove vrednosti pod imenom, te ponovo koristite taj deo koda CSS u vašem stilskom listu.

Ovo je primer definisanja mixina u SCSS sintaksi. U ovom slučaju, mixin se zove `center-flex`. Ima tri svojstva CSS za centriranje stvari korišćenjem flexbox-a.

```scss
@mixin center-flex {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

Evo primera korišćenja mixina koji ste definisali.

```scss
section {
  @include center-flex;
  height: 500px;
  background-color: #3289a8;
}
```
