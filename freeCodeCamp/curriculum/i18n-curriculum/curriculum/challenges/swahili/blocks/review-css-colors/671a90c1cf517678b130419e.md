## Teorija boja

- **Objašnjenje teorije boja**: Ovo je lekcija o tome kako boje interaguju i kako utiču na našu percepciju. Pokriva odnos boja, harmoniju boja i psihološki efekat boja.
- **Osnovne boje**: Ove boje, koje su žuta, plava i crvena, su osnovne boje od kojih se sve ostale boje izvode.
- **Sekundarne boje**: Ove boje nastaju mešanjem jednakih količina dve osnovne boje. Zelena, narandžasta i ljubičasta su primeri sekundarnih boja.
- **Tercijarne boje**: Ove boje nastaju mešanjem osnovne i susedne sekundarne boje. Žuto-zelena, plavo-zelena i plavo-ljubičasta su primeri tercijarnih boja.
- **Tople boje**: Ove boje, koje su crvena, narandžasta i žuta, unose osećaj udobnosti, topline i energije.
- **Hladne boje**: Ove boje, koje su plava, zelena i ljubičasta, unose osećaj mira, smirenosti i profesionalizma.
- **Kolačić boja**: Kolačić boja je kružni prikaz koji pokazuje kako su boje međusobno povezane. Ključan alat za dizajnere jer ih pomaže da izaberu kombinaciju boja.
- **Analogne boje**: Ova kombinacija boja stvara harmoničan i miran doživljaj. Sastoji se od boja koje su susedne na kolačiću boja.
- **Komplementarne boje**: Ova kombinacija boja unosi veliki kontrast i vizuelni efekat. Njihove boje se nalaze na potpuno suprotnim stranama kolačića boja, u odnosu jedna na drugu.
- **Triadni šemi boja**: Ova kombinacija boja ima snažne boje. Nastaje od boja koje su razmaknute približno jednakim rastojanjem. Kada se povežu, formiraju jednstrukvi trougao na kolačiću boja.
- **Monohromatska šema boja**: Ovom kombinacijama boja sve boje potiču od jedne osnovne boje menjanjem svetline, tamnje i zasićenosti. Ovo unosi osećaj jedinstva i harmonije, istovremeno stvarajući varijaciju.

## Različiti načini rada sa bojama u CSS

- **Nazivne boje**: Ove su preddefinirane nazive boja koje prepoznaju pregledači. Primeri su `blue`, `darkred`, `lightgreen`.
- **Funkcija za `rgb()`**: RGB predstavlja Red, Green i Blue — osnovne boje svetlosti. Ove tri boje se kombinuju različitom intenzitetom kako bi stvorile mnogo boja. Funkcija `rgb()` omogućava vam da definišete boju koristeći RGB model boja.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>RGB Color</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

- **Funkcija za `rgba()`**: Ova funkcija dodaje četvrtu vrednost, alfa, koja kontroliše transparentnost boje. Ako nije navedeno, alfa vrednost je `1` podrazumevano.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>RGBA Background</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

- **Funkcija `hsl()`**: HSL predstavlja Hue, Saturation i Lightness — tri glavne komponente koje definišu boju.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>HSL Color</p>
```

```css
p {
  color: hsl(120, 100%, 50%);
}
```

:::

- **Funkcija `hsla()`**: Ova funkcija dodaje četvrtu vrednost, alfa, koja kontroliše opacitet boje.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div>HSLA Background</div>
```


```css
div {
  background-color: hsla(0, 100%, 50%, 0.5);
}
```

:::

- **Heksaadecimalni**: Heksa kod (skraćeno od heksaadecimalnog koda) je sekvenca od šest karaktera koja se koristi za predstavljanje boja u formatu RGB. "Hex" se odnosi na sistem brojanja sa bazom 16, koji koristi cifre od 0 do 9 i slova A do F.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h1 class="hex-text">Hex Text</h1>
<p class="hex-bg">Hex Background</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}

p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

## Karakteristika za `box-shadow`

- **Opis**: Svojstvo `box-shadow` postavlja jednu ili više senki oko elementa.
- **Ofset vrednosti**: Morate postaviti horizontalne (`offset-x`) i vertikalne (`offset-y`) vrednosti. Pozitivna vrednost `offset-x` pomera senku udesno, dok negativna vrednost pomeri ulevo. Pozitivna vrednost `offset-y` pomera senku nadole, dok negativna vrednost pomera nagore. Ako je vrednost `0`, nije potrebno navoditi jedinicu.
- **Radijus zamućenja (blur)**: Ova opcionalna vrednost kontroliše kako senka izgleda zamućeno. Ako nije navedena, podrazumevana je `0`, što stvara oštre ivice. Veća vrednost daje mekšu senku.
- **Radijus širenja (spread)**: Ova opcionalna vrednost kontroliše koliko se senka širi ili smanjuje. Ako nije navedena, podrazumevana je `0`.
- **Boja senke**: Možete postaviti boju korišćenjem imenovanih boja, heksadecimalnih vrednosti, `rgb()`, `rgba()`, `hsl()`, ili funkcije `hsla()`.
- **Ključna reč `inset`**: Dodavanje ključne reči `inset` postavlja senku unutar elementa umesto spolja.
- **Postavljanje više senki kutije**: Možete postaviti više senki razdvajanjem zarezima. Senke se slažu slojevima od prednjeg ka zadnjem.

### Sintaksa

```css
box-shadow: offset-x offset-y blur-radius spread-radius color;
```

## Nagibanje linije i nagibanje kruga

- **Gradient linije**: Ovaj gradijent stvara sporu mešavinu boja duž prave linije. Možete kontrolisati smer ove linije koristeći ključne reči kao što su `to top`, `to right`, `to bottom right`, ili uglove kao što su `45deg`. Možete koristiti bilo koju validnu boju iz CSS i više tačaka boja kako god želite.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient">Linear Gradient</div>
```

```css
.linear-gradient {
  background: linear-gradient(45deg, red, #33FF11, rgba(100, 100, 255, 0.5));
  height: 40vh;
}
```

:::

- **Kružna visina**: Ova visina stvara kružni ili okrugli oblik koji proizlazi iz centra.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient">Radial Gradient</div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```
