# --explanation--

`To touch on` može značiti spominjanje nešto kratko ili obrađivanje teme. Često se koristi kada se pokriva tema bez ulaska u veliku detaljnost. Na primer:

- `The speaker touched on the new software update during the meeting.` - Govornik je spomenuo ažuriranje programa ukratko, ali nije ušao u detalje.
