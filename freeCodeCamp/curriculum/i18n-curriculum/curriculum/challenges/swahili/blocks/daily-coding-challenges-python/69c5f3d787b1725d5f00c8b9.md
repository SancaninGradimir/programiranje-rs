# --description--
Danas je Svetski dan. Ako dobijete red podataka stvari koje ste očistili, vratite zbir vaših ocena za čišćenje prema pravilima navedenim ispod.

Predmeti koji su izdati biće jedan od sledećih:

| Artikal | Osnovna vrednost |
| - | - |
| `"bottle"` | 10 |
| `"can"` | 6 |
| `"bag"` | 8 |
| `"tire"` | 35 |
| `"straw"` | 4 |
| `"cardboard"` | 3 |
| `"newspaper"` | 3 |
| `"shoe"` | 12 |
| `"electronics"` | 25 |
| `"battery"` | 18 |
| `"mattress"` | 38 |

Retki predmet je predstavljen sa `["rare", value]`. Na primer, `["rare", 80]`. Retki predmeti ne dobijaju bonus za niz.

- Bonus za niz: Ako se element pojavljuje uzastopno, dobija rastuće bonus poene.
  - Prvi uzastopni događaj: osnovna vrednost
  - Drugi: osnovna vrednost + 1
  - Treći: osnovna vrednost + 2
  - itd.
- Koeficijent za peti element: Svaki peto prikupljeni element dobija koeficijent množenja.
  - Peti element: *2
  - Deseti element: *3
  - itd.

- Tumia kizidishi baada ya kuhesabu bonasi zozote.

```py
def get_cleanup_score(items):
    values = {
        'bottle': 10, 'can': 6, 'bag': 8, 'tire': 35,
        'straw': 4, 'cardboard': 3, 'newspaper': 3,
        'shoe': 12, 'electronics': 25, 'battery': 18, 'mattress': 38
    }
    total = 0
    prev_item = None
    streak = 0
    for i, item in enumerate(items):
        is_rare = isinstance(item, list) and item[0] == 'rare'
        if is_rare:
            base = item[1]
            prev_item = None
            streak = 0
        else:
            base = values[item]
            streak = streak + 1 if item == prev_item else 0
            prev_item = item
        points = base if is_rare else base + streak
        if (i + 1) % 5 == 0:
            multiplier = (i + 1) // 5 + 1
            points *= multiplier
        total += points
    return total
```
