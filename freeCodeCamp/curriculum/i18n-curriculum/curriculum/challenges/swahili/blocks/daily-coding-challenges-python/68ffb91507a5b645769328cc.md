# --description--
S obzirom na datum rođenja osobe sa formatom ``YYYY-MM-DD``, izračunajte godine te osobe do datuma 27. novembra 2025.

- Prihvatite da su svi datumi rođenja važeći pre 27. novembra 2025.
- Unesite starost kao celobrojni broj.
- Uverite se da uzimate u obzir da li je ta osoba već proslavila svoj rođendan u godini 2025.

```py
from datetime import datetime
def calculate_age(birthday):
    today = datetime(2025, 11, 27)
    birth_date = datetime.strptime(birthday, "%Y-%m-%d")

    age = today.year - birth_date.year

    if (today.month, today.day) < (birth_date.month, birth_date.day):
        age -= 1

    return age
```
