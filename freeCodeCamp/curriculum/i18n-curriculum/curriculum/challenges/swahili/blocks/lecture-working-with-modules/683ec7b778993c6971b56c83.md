# --description--

U programskom razvoju, biblioteka je kao alatna kutija za programere.

Umesto da morate sami implementirati svaki deo koda od nule, biblioteka vam pruža unapred napisan i ponovno upotrebljiv kod, kao što su funkcije, klase objekata i strukture podataka koje možete koristiti u svojim projektima.

Python posjeduje široku standardnu biblioteku sa brojnim ugrađenim modulima. Sve su dobro dizajnirana i testirana rešenja za brojne izazove i vežbe koje ćete naići svakog dana kao razvijatelj, kao što su:

* Interakcija sa operativnim sistemom.
    
* Rad sa fajlovima.
    
* Mreže.
    
* Rad sa datumima i vremenom.
    
* Izvršavanje matematičkih operacija.
    
* Upotreba svakodnevnog jezika.
    
* Detektovanje i ispravljanje grešaka koda.
    
* I mnogo više!

Primeri popularnih ugrađenih modula su `math`, `random`, `re` (skraćeno za "uobičajena upotreba"), i `datetime`.

Moduli za `math` imaju funkcije koje pomažu u izvršavanju složenijih matematičkih operacija.

Modul za `random` je alat za generisanje nasumičnih brojeva.

Modul `re` se koristi za rad sa regularnim izrazima.

U modulu za `datetime` postoji pomoć za rad sa datumom i vremenom u Python.

Ali kako se može pristupiti parametrima, konstantama, funkcijama i klasama objekata definisanim u ovim ugrađenim modulima?

Koristite deklaraciju import. Ove deklaracije vam omogućavaju da uvezete module u vaš skript Python. Deklaracije import se obično pišu na vrhu fajla. Takođe, možete ih promeniti u skladu sa svojim potrebama. Prvo, koristite deklaraciju `import`, praćenu imenom modula:

```python
import module_name
```

Pretpostavimo da želite uvesti modul `math`. U tom slučaju, napisaćete ovo u vaš fajl:

```python
import math
```

Zatim, ako treba da pozoveš funkciju iz tog modula u tvom skriptu za Python, koristićeš dot notaciju (tačku), pišući ime modula praćeno imenom funkcije:

```python
module_name.function_name() 
```

Na primer, da dobijete kvadratni koren 36, napisaćete `math` praćen tačkom pa `sqrt`, funkciju za kvadratni koren, a unutar običnih zagrada prosledićete bilo koji potreban argument. U ovom slučaju, moramo proslediti broj čiji želimo kvadratni koren:

```python
math.sqrt(36)
```

Ovo je najosnovnija verzija deklaracije za import, ali postoji druga alternativa.

Ako želite uvesti module pod drugim imenom (poznato i kao "alias ime"), možete koristiti ovu sintaksu, tako što ćete napisati `as` praćeno alias imenom na kraju deklaracije import:

```python
import module_name as module_alias
```

Ovo se često koristi za skraćivanje dugih imena modula ili izbegavanje kolizija imena.

Na primer, referenciranje modula `math` kao `m` u vašem kodu, možete postaviti alias ime, kao što je ovo:

```python
import math as m 
```

Zatim, možete dobiti komponente modula koristeći alias:

```python
m.sqrt(36)
```

Ali, nekada nije potrebno da doneseš sve iz modula. Možda ti trebaju samo jedna ili dve specifične funkcije ili klase. Python ima ono što ti treba u toj situaciji.

Sada deklaracija za import počinje sa `from`, praćena imenom modula, zatim ključna reč `import` praćena imenima elemenata koje želite da unesete:

```python
from module_name import name1, name2
```

Zatim, možete koristiti ova imena bez prefiksa modula u vašem skriptu za Python.

Ako želite dati ovim imenima nadimke, možete to učiniti korišćenjem ključne reči `as` nakon svakog, praćeno nadimkom koji želite da koristite:

```python
from module_name import name1 as alias1, name2 as alias2
```

Pretpostavimo da želite uvesti funkcije za radijane, sinus i kosinus iz modula `math` samo. Napišite:

```python
from math import radians, sin, cos
```

Sada možete pozvati ove funkcije direktno u vašem kodu, bez modula `math` kao prefiks.

Ovde imamo još detaljniji primer:

Dobijanje sinusa i kosinusa određenog ugla koji je inicijalno postavljen u stepenima, možemo pozvati funkciju `radians` da ga konvertuje u radijane, zatim pozvati sin i kosinus funkcije, prosleđujući taj ugao u radijanima:

```python
from math import radians, sin, cos

angle_degrees = 40
angle_radians = radians(angle_degrees)

sine_value = sin(angle_radians)
cos_value = cos(angle_radians)

print(sine_value) # 0.6427876096865393
print(cos_value)  # 0.766044443118978
```

Pogledajte kako pozivamo funkcije direktno, bez imena modula kao prefiksa. To je zato što smo funkcije uveli ovim alternativnim sintaksom.

Ovo je upozorenje, ali može izazvati sukobe imena ako već imate funkcije ili parametre sa istim imenima definisane u samoj skripti Python. Stoga je to nešto što treba uzeti u obzir kada odlučujete koji tip deklaracije import želite da koristite.

Konačno, dobijamo deklaraciju za import koja se završava sa zvezdikom. Zvezdica govori Python da želiš da prebaciš sve u taj modul, ali želiš to tako da ne moraš da koristiš ime modula kao prefiks:

```python
from module_name import *
```

Na primer, ako ovo uradite prilikom učitavanja modula `math`, moći ćete pozvati bilo koju funkciju opisanu u tom modulu bez spominjanja imena modula kao prefiksa. Evo primera:

```python
from math import *
print(sqrt(36))  # 6.0
print(pow(5, 2)) # 25.0
print(exp(1))    # 2.718281828459045
```

Međutim, ovo se obično ne preporučuje jer može uzrokovati konflikte imena i otežati da se sazna odakle potiču određena imena.

Deklaracija za import radi isto sa funkcijama, klasama objekata, konstantama, parametrima i svim drugim elementima definisanim u modulu.

Ovo je stabilan primer iz modula `math`, broj `pi`:

```python
import math
print(math.pi)
```

Ovaj primer prikazuje klasu objekata iz modula `datetime`. Kreiramo objekat datuma koji predstavlja 15. jula 1959. godine. Zatim, stavljamo taj objekat od `date` u parametar i dobijamo dan, mesec i godinu zasebno koristeći notaciju tačke:

```python
import datetime
birthday = datetime.date(1959, 7, 15)
print(birthday.day)    # 15
print(birthday.month)  # 7
print(birthday.year)   # 1959
```

Možete pronaći više informacija o sadržaju tog modula u zvaničnoj dokumentaciji za Python.

Dobro. Sada znaš više o modulima, takođe bi trebalo da saznaš i ovu veoma važnu frazu u skriptama za Python, jer imaju vrlo blisku vezu:

```python
if __name__ == '__main__': 
    # Code
```

`__name__` je poseban kriterijum koji je interno kreiran u Python.

Kada se fajl Python izvršava automatski, Python postavlja vrednost ove varijable na niz karaktera `"__main__"`.

Međutim, ako je fajl Python uključen kao modul unutar druge skripte od Python, vrednost parametra `__name__` postavlja se na ime tog modula (obično naziv fajla bez ekstenzije `.py`).

Ovo je razlog zašto često naiđete na ove uslove u skripovima Python. Ima kod koji želi da se pokrene **samo ako** skripta Python radi kao glavni program:

```python
if __name__ == '__main__': 
    # Code
```

Ali ako se skripta uvozi kao modul, kod unutar tog dela neće raditi.

Ovo je korisno jer omogućava skriptama Python da imaju dvostru namenu. Mogu se pokrenuti direktno za izvršavanje glavne logike, ili ugraditi u drugi modul bez izvršavanja glavne logike.
