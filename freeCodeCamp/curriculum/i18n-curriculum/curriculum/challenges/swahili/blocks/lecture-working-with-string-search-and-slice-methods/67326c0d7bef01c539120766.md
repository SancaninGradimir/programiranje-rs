# --interactive--

Kada radiš sa nizom karaktera u JavaScript, postoji mnogo slučajeva gde možda treba da proveriš da li string sadrži podstring, koji je manji deo tog niza.

Na primer, možda želite da proverite da li unos korisnika sadrži određenu reč ili slovo pre izvršavanja neke akcije. Jedan način da se ovo postigne je korišćenjem metode `includes()`.

Metoda `includes()` koristi se za proveru da li niz znakova sadrži podniz znaka. Ako je podniz pronađen unutar niza, metoda vraća `true`; inače, vraća `false`.

Evo osnovne sintakse:

```js
string.includes(searchValue);
```

Sintaktički, `searchValue` je niz znakova koji želite tražiti unutar niza. I ovde je primer:

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("awesome");

console.log(result);  // true
```

:::

U ovom primeru, reč `awesome` se nalazi unutar sekvence `JavaScript is awesome!`, stoga funkcija `includes()` vraća `true`.

Važno je napomenuti da putanja `includes()` je osetljiva na velika i mala slova. To znači da zahteva tačno podudaranje karaktera, uključujući veličinu tih karaktera.

Na primer:

:::interactive_editor

```js
let phrase = "JavaScript is awesome!";
let result = phrase.includes("Awesome");

console.log(result);  // false
```

:::

Pošto `Awesome` (ako ima velika slova `A`) ne odgovara ni `awesome` (ako ima mala slova `a`), rezultat je `false`.

Takođe možete koristiti metodu `includes()` da proverite kratku sekvencu slova koja počinje u određenom uzorku, tako što ćete dati drugi parametar:

:::interactive_editor

```js
let text = "Hello, JavaScript world!";
let result = text.includes("JavaScript", 7);

console.log(result);  // true
```

:::

Ovdje, pretraga kratke sekvence slova `JavaScript` počinje od pozicije 7 unutar sekvence, osiguravajući da preskoči bilo koje znakove pre ove pozicije.

Metoda `includes()` vraća samo rezultate od `true` ili `false`. Ne može pružiti informacije o tome gde je podsekvenca pronađena unutar sekvence ili koliko puta se javlja. Ako vam je potreban taj nivo detalja, druga metoda kao što je `indexOf()` može biti bolja.
