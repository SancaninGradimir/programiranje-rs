# --description--

Funkcija za `range()` se koristi za generisanje sekvence punih brojeva. Evo osnovne sintakse za funkciju `range()`:

```py
range(start, stop, step)
```

Obavezni argument za `stop` je kompletan broj koji predstavlja kraj generisane sekvence brojeva. Evo primera korišćenja funkcije `range()`:

```py
for num in range(3):
    print(num)
```

Sledeći kod generiše niz brojeva između `0` i `2`. Ceo broj `3` nije uključen zato što argument `stop` nije uključen u opseg.

Ako nije naveden argument `start`, onda je podrazumevano `0`. Inače, možete koristiti opcioni argument `start` da započnete niz punih brojeva za puni broj različit od `0`. Evo primera generisanja niza punih brojeva između `1` i `4`:

```py
for num in range(1, 5):
    print(num)
```

Podrazumevano, niz celih brojeva povećava se za `1`. Međutim, ako želiš da promeniš tu podrazumevanu vrednost, možeš koristiti opcioni argument `step`. Evo primera koji generiše niz parnih brojeva između `2` i `10`:

```py
for num in range(2, 11, 2):
    print(num)
```

Kao što je ranije pomenuto, funkcija `range()` zahteva samo jedan obavezni argument. Ako ne proslediš nijedan argument funkciji `range()`, dobićeš `TypeError`:

```md
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: range expected at least 1 argument, got 0
```

Važno je napomenuti da funkcija `range()` prihvata samo cele brojeve kao argumente, a ne decimalne brojeve. Decimalni brojevi su brojevi sa decimalnom tačkom, kao što je `3.4`. Ako pokušaš da proslediš decimalni broj kao argument, dobićeš `TypeError`:

```md
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

Ako želiš da generišeš niz celih brojeva u opadajućem redosledu, možeš koristiti negativan ceo broj kao vrednost argumenta `step`, ovako:

```py
for num in range(40, 0, -10):
    print(num)
```

Sledeći kod ispisuje brojeve `40`, `30`, `20` i `10` tim redosledom u konzoli.

Još jedna stvar koju možeš da uradiš sa funkcijom `range()` jeste da napraviš listu celih brojeva koristeći je zajedno sa konstruktorom `list`. Konstruktor `list` služi za pretvaranje iterabilnog objekta u listu. Evo primera koji generiše listu parnih brojeva između `2` i `10`:

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

Funkcija `range()` predstavlja veoma jednostavan način za generisanje niza celih brojeva u Python-u. Kada je savladaš, verovatno ćeš je često koristiti u svojim Python programima.
