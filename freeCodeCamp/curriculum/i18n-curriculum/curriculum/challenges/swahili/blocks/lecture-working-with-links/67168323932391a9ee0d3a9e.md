# --interactive--

Možda ste već videli kako je hiperveza na web stranici promenila boju u ljubičastu nakon klika. To je zato što se stanje veze promenilo, pa se primenjuje drugačiji stil. Postoji pet različitih stanja koja veza može imati.

Prvo je podrazumevano stanje, koje je `:link`. Ovo stanje predstavlja link koji korisnik još nije posetio, kliknuo ili sa kojim nije interagovao. Možete zamisliti ovo stanje kao pružanje osnovnih stilova za sve linkove na vašoj stranici. Ostala stanja su izgrađena na njemu.

Pronađite interfejs za povezivanje i proverite element u prozoru za prikaz. Trebalo bi da ima podrazumevanu plavu boju. Kada kliknete, promeniću se u ljubičastu boju.

:::interactive_editor

```html
<a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
```

:::

Drugo stanje je `:visited`, koje se koristi kada korisnik već poseti povezanu stranicu. Podrazumevano, ovo menja link u ljubičastu boju – ali možete koristiti CSS da pružite različite vizuelne indikatore za korisnika. Ovo pomaže da se obavesti nekoga da je već pročita deo vaših dokumenata. Ili da sajt veruje jer ga je već koristili.

Kliknite na element u prozoru za prikaz i videćete kako se preželjeni element promeni u smeđu boju. Da biste videli prikaz, moraćete da aktivirate interfejs za interakciju.

**NAPOMENA**: Neki CSS su pruženi sa ovim interaktivnim primerom kako biste videli promene u stilovima veza. Ne brinite o razumevanju koda CSS trenutno, jer ćete naučiti njegovo značenje u budućim modulima.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:visited {
  color: brown;
}
```

:::

Treći status je `:hover`. Ovaj status se koristi kada korisnik pređe mišem preko veze. Ovaj status pomaže da se privuče više pažnje na vezu, osiguravajući korisniku da zaista namerava da klikne.

Priključite interfejs i pokušajte da mišem pređete preko uređaja. Videćete kako se boja menja u crvenu.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:hover {
  color: red;
}
```

:::

Zatim imamo `:focus`. Ovo stanje se koristi kada smo pažljivi sa vezom.

Kliknite na bilo koje prazno mesto u prozoru za prikaz, zatim pritisnite `tab` na vašoj tastaturi. Videćete da se link promeni u zelenu boju. Da biste videli prikaz, moraćete da uključite interfejs za interakciju.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:focus {
  color: green;
}
```

:::

Na hatimaye, imamo `:active`. Ovaj status se koristi za linkove koje korisnik izvršava. Ovo obično znači klikom na link glavnom tasterom miša, levim klikom, u većini slučajeva. Ovaj status pomaže da se korisniku pokaže da je element koji je kliknuo interaktivan.

Sačekajte interaktivni interfejs i kliknite na link. Bićete u mogućnosti da vidite kako se link menja u crnu boju. Ovo se dešava veoma brzo, pa možda ćete morati da kliknete nekoliko puta da biste videli promenu boje.

:::interactive_editor

```html
<head>
    <link href="styles.css" rel="stylesheet" />
</head>

<body>
    <a href="https://freecodecamp.org" target="_blank">Visit freeCodeCamp</a>
</body>
```

```css
a:active {
  color: black;
}
```

:::

Kada koristiš ova stanja za stilizovanje elemenata, postoji poseban redosled pisanja CSS selektora: `link`, `visited`, `hover`, `focus`, pa zatim `active`.

Sada trebate znati kako da prikažete svoj lični stil za linkove na vašoj stranici, istovremeno pružajući važne informacije koje korisnik treba o statusu svakog linka.
