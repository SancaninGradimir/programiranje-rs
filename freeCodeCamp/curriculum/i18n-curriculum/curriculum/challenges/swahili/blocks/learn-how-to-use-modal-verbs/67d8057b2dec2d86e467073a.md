# --explanation--

`port` je deo mrežnog spoja koji se koristi preko mreže za omogućavanje komunikacije između uređaja. Na primer:

`I'm trying to connect to the server, but the port might be closed.` - Mrežni port je možda zatvoren, što sprečava povezivanje sa serverom.

Portovi mogu biti `open` ili `closed`. `Open port` može omogućiti napadačima pristup sistemu ako nije pravilno zaštićen, dok `closed port` ne dozvoljava prolazak podataka.
