# --explanation--

`Exactly` je glagol koji se koristi za pokazivanje potpunog saglasja ili tačnosti sa onim što je druga osoba rekla. `Exact`, s druge strane, to je pridev koji znači tačan ili kompletan u detaljima. Na primer:

- `You are exactly right about the cause of the problem.` – Ovo izražava potpuno slaganje. Pokazuje koliko je neko bio u pravu (pridev).

- `We need the exact measurements to build the new server rack.` – Ovo znači veoma precizne dimenzije. Odnosi se na imenicu umesto glagola ili prideva.
