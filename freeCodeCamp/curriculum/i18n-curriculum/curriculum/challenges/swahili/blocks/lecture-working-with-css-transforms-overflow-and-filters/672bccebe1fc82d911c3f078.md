# --interactive--

Sifa za CSS `filter` je moćan alat koji vam omogućava da koristite vizuelne efekte za elemente na veb stranici. Posebno je korisno za prilagođavanje prezentacije slika, pozadina i čak teksta bez menjanja originalnih svojstava.

Svojstvo `filter` može se koristiti za kreiranje različitih efekata, kao što su zamućenje, promena boja i podešavanje osvetljenosti. Hajde da pogledamo kako funkcioniše svojstvo `filter` i istražimo neke uobičajene primere. Osnovna sintaksa svojstva `filter` je jednostavna:

```css
selector {
  filter: function(amount);
}
```

Ovde `function` predstavlja određeni efekat filtera koji želiš da primeniš, a `amount` je obično vrednost koja određuje jačinu tog efekta. Hajde sada da pogledamo neke funkcije filtera i njihovu uobičajenu upotrebu.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: blur(2px);
}
```

:::

Funkcija `blur` primenjuje Gausovo zamućenje na element. Njena vrednost se zadaje u pikselima i predstavlja poluprečnik zamućenja. Ovo CSS pravilo primenjuje zamućenje od 2 piksela na sve slike na stranici. Efekat `blur` može biti koristan za stvaranje dubine u dizajnu ili sakrivanje delova slike.

Funkcija `brightness` podešava osvetljenost elementa. Vrednost `0%` učiniće element potpuno tamnim, dok će vrednosti veće od `100%` povećati osvetljenost.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: brightness(150%);
}
```

:::

Ovo CSS pravilo povećava osvetljenost elementa slike za `50%`. Podešavanje osvetljenosti može pomoći da slike budu uočljivije ili da se napravi blagi efekat sjaja.

Funkcija `grayscale` pretvara element u nijanse sive. Njena vrednost se izražava u procentima, gde `100%` znači potpuno sivu sliku, a `0%` ostavlja sliku neizmenjenom.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: grayscale(100%);
}
```

:::

Ova pravila će promeniti element slike u potpuno sivo nijanse. `grayscale` Može se koristiti za kreiranje starog izgleda ili smanjenje značajnosti određenih elemenata na stranici.

Funkcija `sepia` primenjuje efekat sepije na element. Kao i `grayscale`, koristi procentualne vrednosti:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: sepia(80%);
}
```

:::

Ovo pravilo primenjuje efekat sepije od `80%` na element slike. Efekat sepije je koristan za stvaranje starinskog ili retro izgleda.

Funkcija `hue-rotate` primenjuje rotaciju nijanse boje na element. Njena vrednost se izražava u stepenima i predstavlja rotaciju na krugu boja.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {
  filter: hue-rotate(90deg);
}
```

:::

Ova pravila rotira boju elementa slike po stepenima `90`. Rotacija boja može se koristiti za kreiranje psihedeličkih efekata ili prilagođavanje ukupnog rasporeda boja slike.

Jedna od najmoćnijih karakteristika svojstva `filter` je mogućnost kombinovanja više efekata. Možete primeniti mnogo filtera na jednom svojstvu razdvajanjem ih razmakom:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute orange cat lying on its back.">
```

```css
img {  
  filter: contrast(150%) brightness(110%) sepia(30%);  
}
```

:::

Ovaj filter koristi povećanje sjaja, blagi porast svetlosti i lagani sepijev efekat za element slike.

Kombinovanjem filtera, možete stvoriti jedinstvene i složene vizuelne efekte koji odgovaraju potrebama vašeg dizajna. Funkcija CSS filter je svestrani alat koji omogućava kreativnu kontrolu nad elementima web stranice.

Iako smo pokrili neke popularne filtre funkcije, postoje i druge dostupne, kao što su `contrast`, `invert`, i `saturate`. Kao i svaka moćna karakteristika, važno je biti pažljiv kako koristite filter da biste poboljšali dizajn bez preopterećenja korisnika ili uticaja na pristupačnost.
