# --interactive--

Metoda ``splice()`` u `JavaScript` je moćan način za modifikovanje nizova podataka. Omogućava vam da dodajete ili uklonite elemente bilo gde u nizu podataka, uključujući i sredinu. Vrednost `return` metode ``splice()`` biće niz uklonjenih elemenata iz niza podataka. Ako ništa nije uklonjeno, vraća se prazan niz.

Važno je razumeti da će ova metoda modifikovati originalni red podataka, menjajući ga na mestu umesto kreiranja novog reda. Ovo je nešto što treba uzeti u obzir kada koristite `splice()`. Evo osnovne sintakse:

```js
array.splice(startIndex, itemsToRemove, item1, item2)
```

`startIndex` opisuje obrazac koji počinje da menja red podataka, dok je `itemsToRemove` opcioni parametar koji pokazuje koliko elemenata treba obrisati. Ako `itemsToRemove` nije navedeno, `splice()` će pronaći sve elemente od `startIndex` do kraja reda podataka. Sledeći parametri (`item1`, `item2`, itd.) su elementi koji se dodaju u red podataka, počevši od početnog obrasca.

Počnimo sa primerom brisanja elemenata u sredini reda podataka:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "mango", "kiwi"];
let removed = fruits.splice(2, 2);

console.log(fruits);  // ["apple", "banana", "kiwi"]
console.log(removed); // ["orange", "mango"]
```

:::

U ovom primeru, `splice(2, 2)` počinje od indeksa `2` i uklanja `2` elementa. Izmenjeni niz će sada sadržati samo `apple`, `banana` i `kiwi`. Sada pogledajmo kako da dodamo elemente u sredinu niza:

:::interactive_editor

```js
let colors = ["red", "green", "blue"];
colors.splice(1, 0, "yellow", "purple");

console.log(colors); // ["red", "yellow", "purple", "green", "blue"]
```

:::

Ovde, `splice(1, 0, "yellow", "purple")` počinje na modelu `1`, briše elemente `0`, i dodaje `yellow` i `purple`. Drugi parametar (`0`) znači da se ne brišu elementi pre dodavanja. Takođe možete koristiti `splice()` za brisanje i dodavanje elemenata istovremeno:

:::interactive_editor

```js
let numbers = [1, 2, 3, 4, 5];
numbers.splice(1, 2, 6, 7, 8);

console.log(numbers); // [1, 6, 7, 8, 4, 5]
```

:::

U ovom slučaju, `splice(1, 2, 6, 7, 8)` počinje na modelu `1`, briše elemente `2` (`2` i `3`), i unosi `6`, `7`, i `8`. Ako želite da originalni red podataka ostane nepromenjen, trebate kreirati kopiju pre korišćenja `splice()`:

:::interactive_editor

```js
let original = [1, 2, 3, 4, 5];
let copy = [...original];
copy.splice(2, 1, 6);

console.log(original); // [1, 2, 3, 4, 5]
console.log(copy);     // [1, 2, 6, 4, 5]
```

:::

U ovom primeru, da bismo napravili kopiju niza `original` bez njegovog menjanja, koristimo spread operator (`...`). Spread operator pravi plitku kopiju elemenata niza `original` u novi niz. Više o tome naučićete u narednim predavanjima.

Kada koristimo `copy.splice(2, 1, 6)`, menja se kopija niza tako što se uklanja element na indeksu `2` (što je `3`) i na njegovo mesto ubacuje novi element `6`.

Uobičajena upotreba `splice()` je brisanje jednog elementa iz reda podataka kada znate njegov šablon:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "mango"];
let indexToRemove = fruits.indexOf("orange");
if (indexToRemove !== -1) {
    fruits.splice(indexToRemove, 1);
}

console.log(fruits); // ["apple", "banana", "mango"]
```

:::

U ovom primeru, prvo koristimo metodu `indexOf()` da dobijemo instancu elementa `orange` u nizu `fruits`. Metoda `indexOf()` vraća instancu kada se element prvi put pojavi ili `-1` ako element nije dostupan u nizu podataka.

Zatim upoređujemo `indexToRemove` sa `-1` kako bismo osigurali da element postoji u nizu podataka pre pokušaja brisanja. Ako `indexToRemove` nije jednako `-1` (što znači da je element pronađen), koristimo `splice()` za brisanje jednog elementa počevši od pozicije `indexToRemove`.

Takođe možete koristiti `splice()` da obrišete red podataka tako što ćete obrisati sve elemente:

:::interactive_editor

```js
let array = [1, 2, 3, 4, 5];
array.splice(0);

console.log(array); // []
```

:::

Iako je `splice()` moćan, treba napomenuti da za veoma velike nizove može imati nisku efikasnost u poređenju sa drugim metodama, posebno kada želite da promenite početak niza podataka. To je zato što `splice()` može zahtevati pomeranje svih sledećih elemenata. U takvim slučajevima, ako želite samo dodati ili izbrisati elemente na kraju niza podataka, metode kao što su `push()`, `pop()`, `unshift()`, i `shift()` mogu biti bolje.

Na kraju, `splice()` je veoma korišćena metoda za modifikovanje redova podataka u JavaScript. Omogućava vam da precizno kontrolišete dodavanje i brisanje elemenata iz bilo kog dela reda podataka. Razumevanje kako efikasno koristiti `splice()` može povećati vašu sposobnost rukovanja redovima podataka u vašem kodu JavaScript.
