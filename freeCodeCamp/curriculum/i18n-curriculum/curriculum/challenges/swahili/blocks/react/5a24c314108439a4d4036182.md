# --description--

Možda si primetio u prethodnom izazovu da postoji nekoliko razlika između HTML stilova i React svojstva `style`, koje je JavaScript objekat. Prvo, neka CSS svojstva koriste camelCase zapis. Na primer, koristi se `fontSize` umesto `font-size`. CSS svojstva sa crticama nisu važeća kao svojstva JavaScript objekata, pa React koristi camelCase. Kao opšte pravilo, sva CSS svojstva koja sadrže crtice pišu se u camelCase obliku u JSX-u.

Sve jedinice dužine za vrednosti svojstava (kao što su `height`, `width`, i `fontSize`) smatraju se da su u `px` osim ako je drugačije navedeno. Ako želite da koristite `em`, na primer, morate obuhvatiti vrednost i jedinice navodnicima, kao što je `{fontSize: "4em"}`. Osim dužinskih vrednosti koje uzimaju `px` kao podrazumevano, sve ostale vrednosti svojstava moraju biti obuhvaćene navodnicima.

# --instructions--

Ako imate veliki set stilova, možete postaviti `object` stila u trajni parametar kako biste održali vaš kod organizovanim. Deklarišite svoj stilski parametar kao globalni parametar na fajlu. Inicijalizujte parametar `styles` i postavite `object` sa tri atributa stila i njihovim vrednostima. Definišite `div` boju od `purple`, veličinu fonta od `40` i ivicu od `2px solid purple`. Zatim postavite atribut `style` jednak parametru `styles`.

# --hints--

Kriterijum za `styles` treba da bude `object` sa tri osobine.

```js
assert(Object.keys(styles).length === 3);
```

Parametar za `styles` mora imati svojstvo `color` postavljeno na vrednost `purple`.

```js
assert(styles.color === 'purple');
```

Kriterijum za `styles` mora imati svojstvo `fontSize` koje je postavljeno na vrednost `40`.

```js
assert(styles.fontSize == 40);
```

Parametar za `styles` treba da ima svojstvo `border` koje je postavljeno na vrednost `2px solid purple`.

```js
assert(styles.border === '2px solid purple');
```

Sekcija treba da prikaže element `div`.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.shallow(React.createElement(Colorful));
    return mockedComponent.type() === 'div';
  })()
);
```

Element za `div` treba da ima stilove definisane od objekta `styles`.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.shallow(React.createElement(Colorful));

      mockedComponent.props().style.color === 'purple' &&
      mockedComponent.props().style.fontSize == 40 &&

  })()
);
```
