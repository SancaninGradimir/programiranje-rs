# --description--

SQL je skraćenica za Structured Query Language. Razvijen je tokom 1970-ih godina, kada je prvobitno bio poznat kao structured English query language (SEQUEL). Naziv je kasnije skraćen na SQL.

SQL je programski jezik koji se koristi za skladištenje i upravljanje podacima u relacionalnim bazama podataka.

Kao što ste naučili u prethodnoj lekciji, relacionе baze podataka organizuju podatke u tabelama. Svaka tabela ima redove i kolone, gde su predstavljeni atributi entiteta i njihovi odnosi. Svaki red predstavlja zapis, dok svaka kolona predstavlja određeni atribut ili polje tog zapisa.

Korišćenjem SQL-a možete dodavati, menjati, brisati, pretraživati i dohvatati podatke iz relacionih baza podataka. On pruža kompletan skup naredbi za izvršavanje upita, filtriranje, sortiranje i agregaciju podataka.

SQL može pomoći u poboljšanju performansi baze podataka tako što izvršava upite samo nad potrebnim podacima i može se lako povezati sa različitim programskim jezicima, omogućavajući direktnu interakciju sa bazom podataka iz vaših aplikacija.

Takođe je prihvaćen kao industrijski standard. Obezbeđuje integritet podataka i ima mnoge bezbednosne funkcije, kao što su podrška za autentifikaciju korisnika i šifrovanje podataka.

Takođe je skalabilan, prenosiv i kompatibilan sa različitim sistemima za upravljanje bazama podataka.

SQL se zasniva na naredbama poznatim kao SQL izrazi ili SQL upiti.

Kao programer, ove izjave ćete napisati koristeći specijalne jezičke karakteristike jezika SQL ili ključne reči. Ovo vam omogućava da izvršite potrebne operacije na vašoj bazi podataka.

Kabla hujaingiza amri za SQL, unahitaji kufungua shell ya psql ili uweze kuingiliana moja kwa moja na hifadhidata ya PostgreSQL. Katika prompt ya amri au terminal, ingiza:

```bash
psql -U <username> -d <database_name>
```

Promeni `username` sa svojim korisničkim imenom i `database_name` sa bazom podataka kojoj želiš da se povežeš.

Ako još uvek niste kreirali svoju bazu podataka, PostgreSQL automatski uključuje podrazumevanu bazu podataka nazvanu `postgres` kada se pokrene. Možete koristiti ovu bazu podataka `postgres` da pratite uputstva.

Kada budete povezani, videćete da se prompt menja u:

```bash
postgres=#
```

Prikazuje ime baze podataka sa kojom se povezuje i čeka komande za SQL.

Pogledajmo neke osnovne SQL naredbe koje treba da poznajete. Primetićete da se, po konvenciji, SQL naredbe završavaju tačkom-zarezom (`;`).

Prvo, možete koristiti ovu komandu za kreiranje baze podataka pod nazivom `my_database`:

```sql
CREATE DATABASE my_database;
```

Možete napisati `CREATE` praćen `DATABASE`, i naziv baze podataka koju želite kreirati. Proverite da su ključne reči komande za SQL napisane velikim slovima.

Zatim, kada imate svoju bazu podataka, možete se na nju povezati.

Jinsi unavyounganishwa na hifadhidata hutegemea mazingira na chombo unachotumia.

U komandnoj liniji ili terminalu možete koristiti prethodno prikazani način. U interaktivnoj psql ljusci možete promeniti bazu podataka pomoću prečice `\c` praćene nazivom baze podataka, kao što je prikazano:

```bash
\c my_database
```

Imajte na umu da su naredba `\c` i druge prečice (one koje počinju obrnutom kosom crtom) deo psql ljuske, a ne deo samog SQL jezika, pa im nije potreban tačka-zarez za završetak naredbe.

Sada kada ste povezani sa novom bazom podataka, videćete da se prompt promenio:

```bash
my_database=#
```

Kada kreirate i povežete se sa bazom podataka, možete kreirati tabelu pomoću ključnih reči `CREATE TABLE`:

```sql
CREATE TABLE products (
  id SERIAL,
  name VARCHAR(255)
);
```

U naredbi pišete `CREATE TABLE`, nakon čega sledi naziv tabele koju želite da kreirate. U ovom slučaju, napraviće tabelu pod nazivom `products`.

Zatim, unutar običnih zagrada, pišete nazive kolona koje tabela treba da ima i određujete tip podataka vrednosti koje će biti sačuvane u svakoj koloni.

Za primer, tabela će imati dve kolone: jednu za identifikator proizvoda i drugu za naziv proizvoda.

Standard konvencija za imena tabela i kolona je korišćenje `case`, pisanje reči malim slovima i razdvajanje ih donjom crtom (``_``). Primer: ``delivery_orders``

Ovo su nekoliko važnih komandi za SQL koje trebate znati na početku.

Sa SQL, možete lako izvršiti pretraga upita, upravljati i analizirati podatke radi donošenja tačnih odluka i rešavanja stvarnih problema.
