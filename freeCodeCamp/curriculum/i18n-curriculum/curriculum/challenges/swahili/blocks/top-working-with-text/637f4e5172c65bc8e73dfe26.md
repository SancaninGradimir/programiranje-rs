# --description--

Ako želite da imate listu stavki čiji red nije važan, kao što je lista za kupovinu stvari koje se mogu nabaviti u bilo kom redosledu, onda možete koristiti neuređenu listu.

Neuređene liste se kreiraju korišćenjem elemenata `<ul>`, a svaki element unutar liste kreira se korišćenjem elemenata lista `<li>`.

Svaki stavka u neuređenu listu počinje tačkom pred navodima:
