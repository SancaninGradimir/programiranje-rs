# --description--

U Express, parametri putanje podataka su promenljive vrednosti uključene u URL. Omogućava vašoj aplikaciji da uhvati delove URL koji se mogu menjati sa svakim zahtevom. Ovo je važno za kreiranje dinamičkih stranica, kao što je prikazivanje specifičnog korisnika, blog posta ili proizvoda na osnovu URL.

Evo primera:

```javascript
// Route with a parameter in the path
app.get("/post/:postId", (req, res) => {
  const postId = req.params.postId;
  res.send(`Viewing post with ID: ${postId}`);
});
```

Ovde, putanja podataka sluša GET zahteve na `/post/:postId`.

* `:postId` je parametar putanje podataka, predstavnik u URL koji će biti zamenjen stvarnom vrednošću. Na primer, zahtev za `/post/42` odgovara ovoj putanji, i `42` postaje vrednost za `postId`.
    
* Možete dobiti parametre putanje podataka koristeći `req.params`. Objekat `req.params` ima sve parametre putanje podataka, i u ovom slučaju, `req.params.postId` ima vrednost iz URL, `42`.

Na kraju, `res.send()` šalje odgovor nazad klijentu. Dakle, ako je URL `/post/42`, klijent će videti `Viewing post with ID: 42`.

Kao što možete videti, parametri ruta čine vaše rute fleksibilnijim i dinamičnijim, bez potrebe da pravite posebnu rutu za svaku moguću vrednost.

Sada kada razumete kako funkcionišu parametri ruta, vratimo se na primer iz prethodne lekcije i pogledajmo kako možete koristiti dinamičke rute za različite kategorije menija:

```javascript
app.get("/menu/:category", (req, res) => {
  const category = req.params.category;
  res.send(`You are viewing the ${category} menu!`);
});
```

Evo sažetka kako to funkcioniše:

1. Mtumiaji anatembelea [`http://localhost:3000/menu/drinks`](http://localhost:3000/menu/drinks).
    
2. Server uparuje ovaj zahtev sa rutom `/menu/:category` i hvata vrednost parametra `category` kao `drinks`.
    
3. Server šalje sledeći odgovor nazad pregledaču: `You are viewing the drinks menu!`
    

Slično tome, ako korisnik poseti [`http://localhost:3000/menu/food`](http://localhost:3000/menu/food), server odgovara sa: `You are viewing the food menu!`

Parametri ruta omogućavaju prikaz dinamičkog sadržaja na osnovu URL-a. Umesto da pravite posebnu rutu za svaku kategoriju, možete koristiti jednu rutu za obradu više kategorija, kao što su pića, hrana, grickalice i druge, jednostavnom promenom URL-a.
