# --description--

Skup tačaka u ravni $S$ naziva se titanički skup ako postoji prava koja prolazi samo kroz dve tačke u $S$.

Primer skupa Titanik je $S = \\{(0, 0), (0, 1), (0, 2), (1, 1), (2, 0), (1, 0)\\}$, gde linija koja prolazi kroz (0, 1) i (2, 0) ne prolazi nikakvu drugu tačku u $S$.

S druge strane, skup {(0, 0), (1, 1), (2, 2), (4, 4)} nije titanijski skup jer linija koja prolazi kroz bilo koje dve tačke u tom skupu takođe prolazi i kroz druge dve tačke.

Za bilo koji pozitivan ceo broj $N$, neka je $T(N)$ broj kombinacija titaničkog $S$ koje svaka tačka ($x$, $y$) zadovoljava uslov $0 ≤ x$, $y ≤ N$. Dokazano je da $T(1) = 11$, $T(2) = 494$, $T(4) = 33\\,554\\,178$, $T(111)\bmod {10}^8 = 13\\,500\\,401$ i $T({10}^5)\bmod {10}^8 = 63\\,259\\,062$.

Pronađi $T({10}^{11})\bmod {10}^8$.
