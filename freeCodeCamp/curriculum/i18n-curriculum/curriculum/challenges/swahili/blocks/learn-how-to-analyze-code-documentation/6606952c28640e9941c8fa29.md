# --description--

Obično, `with` se koristi za prikaz veze ili upotrebe. Na primer, `I write with a pen.` Međutim, u nekim rečenicama, `with` dolazi nakon reči sa kojom se koristi. Ovo se obično dešava kada pokušavate da naglasite nešto ili neku stvar s kojom neko radi ili se bavi, posebno u pitanjima ili izjavama koje se tiču tehnologije ili posla.

Na primer, u tehnologiji, možete reći, `This program I work with is challenging.` Ovde, `with` se koristi na kraju da naglasi program sa kojim je teško raditi. Drugi primer može biti pitanje poput, `What else is he working with?` gde `with` na kraju pita o drugim alatima ili tehnologijama koje se koriste.
