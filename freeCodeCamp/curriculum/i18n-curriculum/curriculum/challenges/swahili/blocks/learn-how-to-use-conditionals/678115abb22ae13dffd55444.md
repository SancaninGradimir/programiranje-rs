Pitanje Boba, `How can we set it up?`, pokazuje da traži informacije ili uputstva o tome kako implementirati automatizaciju. Ovaj tip pitanja je uobičajen kada neko treba vodič ili pojašnjenje.

Ostali primeri za traženje informacija su:

- `Can you explain how this works?`

- `What steps should we take?`

- `Do you know how to implement this feature?`

Ovi tipovi pitanja pomažu u prikupljanju informacija i boljem razumevanju procesa.
