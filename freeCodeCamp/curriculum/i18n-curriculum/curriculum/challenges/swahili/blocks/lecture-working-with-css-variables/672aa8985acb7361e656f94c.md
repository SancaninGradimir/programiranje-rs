# --interactive--

Uobičajena svojstva za CSS, poznata takođe kao parametri za CSS, su stvari deklarisane od strane autora CSS koje zadržavaju specifične vrednosti za ponovnu upotrebu u celom dokumentu. To je moćna karakteristika koja omogućava stilove tabele da budu efikasniji, lakši za održavanje i fleksibilniji.

Sintaksa za deklarisanje običnog atributa je jednostavna. Počinje sa dve kose crte (`--`) praćeno imenom atributa:

```css
:root {
  --main-color: #3498db;
}
```

U ovom primeru, deklarišemo običnu osobinu sa imenom `--main-color` i vrednošću `#3498db`. Pseudo-klasa `:root` se često koristi za deklarisanje globalnih običnih svojstava, jer predstavlja roditeljski element najvišeg nivoa u drvetu DOM.

Da biste koristili standardnu osobinu, koristite funkciju `var()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
:root {
  --main-color: #3498db;
}

.button {
  background-color: var(--main-color);
}
```

:::

Ovo postavlja pozadinsku boju elemenata sa klasom `button` na vrednost sačuvanu u `--main-color`.

Jedna od ključnih komponenti standardnih karakteristika je što prate tok CSS. To znači da ih možete ponovo implementirati za specifične funkcije ili određeni kontekst:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="alert">This is an alert message.</div>
```

```css
.alert {
  --main-color: #e74c3c;
  background-color: var(--main-color);
}
```

:::

U ovom slučaju, elementi sa klasom `alert` će koristiti različitu vrednost od `--main-color`, izdajući međunarodnu deklaraciju.

Standardni atributi takođe podržavaju alternativne vrednosti. Ako standardni atribut nije deklarisan ili je nevažeći, možete navesti alternativnu vrednost:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="text">This is some text.</div>
```

```css
:root {
  --text-color: green;
}

.text {
  color: var(--text-color, green);
}
```

:::

Ovde, ako `--text-color` nije deklarisano, `color` koristiće vrednost od `green` kao alternativnu opciju.

Opšta svojstva su veoma važna za kreiranje struktura koje mogu da se prilagode različitim okruženjima. Možete deklarisati skup svojstava za različita okruženja:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="light-theme">
  <p>This is some light themed text.</p>
</div>
```

```css
:root {
  --bg-color: black;
}

body {
  background-color: var(--bg-color);
}

.light-theme {
  --bg-color: white;
  --text-color: black;
  background-color: var(--bg-color);
  color: var(--text-color);
  padding: 10px;
}

```

:::

Promena tema je laka kao dodavanje ili uklanjanje klase iz elementa `body`.

Standardne svojstva se takođe mogu koristiti zajedno sa medijskim upitima (media queries) za kreiranje dizajna koji se prilagođava uređaju:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="card">
  <h2>Responsive Design</h2>
  <p>Resize the window to see the card adapt!</p>
</div>

```

```css
:root {
  --card-width: 90%;
  --card-bg: #f0f0f0;
  --card-padding: 1rem;
  --text-color: #333;
}

/* Tablet screens and up */
@media (min-width: 600px) {
  :root {
    --card-width: 70%;
    --card-bg: #e8f5e9;
    --card-padding: 1.5rem;
  }
}

/* Desktop screens and up */
@media (min-width: 1024px) {
  :root {
    --card-width: 50%;
    --card-bg: #d0f0ff;
    --card-padding: 2rem;
  }
}

body {
  font-family: system-ui, sans-serif;
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
  background-color: #fafafa;
}

.card {
  width: var(--card-width);
  background-color: var(--card-bg);
  padding: var(--card-padding);
  color: var(--text-color);
  border-radius: 8px;
  box-shadow: 0 2px 6px rgba(0,0,0,0.1);
  text-align: center;
  transition: all 0.3s ease;
}

```

:::

Ovaj način omogućava prilagodljive strukture da budu lake za održavanje, jer možete promeniti vrednost na jednom mestu umesto u celoj vašoj tabeli stilova. Ovo može značajno smanjiti ponavljanje u vašim tabelama stilova, posebno za hardkodovane vrednosti koje se često koriste.

Opšti atributi takođe mogu da se odnose na druge opšte atribute:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Click Me</button>
```

```css
:root {
  --primary-color: #3498db;
  --button-bg: var(--primary-color);
}

button {
  background-color: var(--button-bg);
}

```

:::

Ovo omogućava kreiranje odnosa između različitih stilskih elemenata, i olakšava održavanje konzistentne teme na vašoj veb stranici.

Iako standardne funkcije imaju mnogo prednosti, važno je znati o podršci pretraživača. Dobro su podržane u modernim pregledačima, ali stariji pregledači možda neće podržavati. Uvek imajte alternativni plan kada koristite potpuno nove funkcije.

Na kraju, standardne karakteristike CSS pružaju moćan način za kreiranje stilskih tabela koje su jače, fleksibilnije i lakše za održavanje. One vam omogućavaju kreiranje dizajna koji se mogu menjati po temi, pojednostavljuju dizajn koji se prilagođava uređaju, i omogućavaju promene stila tokom implementacije.

Kako se razvija mrežni razvoj, uobičajene funkcije postaju najvažniji alat u repertoaru programera, nudeći nove mogućnosti kreiranja fleksibilnog i efikasnog CSS.
