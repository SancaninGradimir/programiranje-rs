# --interactive--

Kao što se sećate iz prethodnih lekcija i radionica, možete kreirati funkciju sa nekoliko parametara i pozvati tu funkciju sa argumentima.

Ovo je podsetnik:

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call
getSum(3, 4); // 7
```

Ali šta ako imate funkciju koja se poziva sa više argumenata nego što je deklarisano da prima?

:::interactive_editor

```js
// function definition
function getSum(num1, num2) {
  return num1 + num2;
}

// function call with extra argument
console.log(getSum(3, 4, 5)); // 7
```

:::

JavaScript neće vratiti grešku u ovom slučaju. Umesto toga, ignoriše dodatni argument i samo sabira brojeve `3` i `4` zajedno. Funkcije koje prihvataju promenljiv broj argumenata poznate su kao <dfn>vitendakcije variadic</dfn>.

Ako radiš sa variadičkim funkcijama, onda možeš koristiti nešto poput `arguments`. Ovo nešto koje liči na niz podataka sadrži vrednosti argumenata prosleđenih u funkciju.

Evo primera:

:::interactive_editor

```js
function logArgs() {
  for (const arg of arguments) {
    console.log(arg);
  }
}

logArgs(1, 2, 3);
// result:
// 1
// 2
// 3

logArgs("example"); // "example"
```

:::

Pošto je stav `arguments` sličan redu podataka, možete dobiti argument sa ovim specifičnim primerom:

:::interactive_editor

```js
function getArg() {
  return arguments[1];
}

console.log(getArg(2, 4, 6)); // 4
```

:::

Takođe možete koristiti atribut `length` da biste dobili broj argumenata koje je funkcija primila:

:::interactive_editor

```js
function getArgs() {
  return arguments.length;
}

console.log(getArgs("Example")); // 1
console.log(getArgs("Another", "Example")); // 2
```

:::

Iako je objekat `arguments` prikazan kao prava red, nema ugrađene metode poput `Array` kao što su `includes` ili `push`. Da biste dobili pristup tim metodama, prvo morate konvertovati objekat `arguments` u pravi red koristeći nešto poput `slice`, `Array.from()` ili operaciju raspršivanja:

:::interactive_editor

```js
function hasCat() {
  return [...arguments].includes("cat");
}

console.log(hasCat("dog", "chicken", "cat")); // true
console.log(hasCat("dog", "chicken", "horse")); // false
```

:::

Iako je moguće raditi sa nečim kao što je `arguments` za variadijske funkcije, moderne aplikacije JavaScript obično koriste sintaksu rest parametra. Naučićete više o tome u sledećoj lekciji.
