# --interactive--

U prethodnoj lekciji ste naučili o mreži CSS, koju možete koristiti za kreiranje složenih i fluidnih rasporeda na vašim veb stranicama. U ovoj lekciji ćemo pogledati kako kreirati responsivan grid raspored koristeći komponentu `fr`.

Počinjemo sa ovim oznakama za HTML, koje će predstavljati naš alat mreže:

```html
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

Unutar CSS, postavili smo svojstvo display na grid za kontejner.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 25% 25% 25% 25%;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

`grid-template-columns` koristi se za postavljanje veličine svakog reda. U ovom slučaju, veličina svakog reda biće `25%` kontejnera. Zatim se svojstvo 'gap' koristi za kreiranje razmaka između svakog reda.

Sad smo koristili procenat za veličinu reda, ali možemo koristiti i jedinicu od `fr`.

Jedinica za `fr` je komponentna jedinica koja predstavlja sekciju prostora mrežnog uređaja.

Ovo je kako će kod izgledati kada se izmeni korišćenjem jedinica `fr` umesto procenata.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="grid-container">
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
  <div class="col"></div>
</div>
```

```css
html,
body {
  width: 90%;
  height: 50%;
}

.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  gap: 15px;
  background-color: darkgray;
  height: 100%;
}

.col {
  background-color: darkslateblue;
}
```

:::

Svaki red će zauzeti moj deo postojećeg prostora. Pošto ima četiri reda, svaki red će imati jednak deo postojećeg prostora unutar mrežnog kontejnera.

Kada počneš da praviš rasporede pomoću Grid-a, primetićeš da ćeš često koristiti jedinice `fr`, jer pružaju fleksibilan i proporcionalan način raspodele prostora. One omogućavaju kreiranje responzivnih rasporeda koji se automatski prilagođavaju veličini ekrana, bez potrebe za ručnim podešavanjem vrednosti u pikselima.
