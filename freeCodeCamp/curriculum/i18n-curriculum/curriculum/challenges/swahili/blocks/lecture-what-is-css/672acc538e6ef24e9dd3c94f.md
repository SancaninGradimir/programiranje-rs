# --interactive--

Margine i unutrašnji razmak (padding) su važna CSS svojstva za kreiranje dobro organizovanih, preglednih i vizuelno privlačnih veb stranica.

Margine kontrolišu prostor izvan elementa, pomažući da se odvoji od drugih elemenata i definiše raspored, dok padding kontroliše prostor unutar elementa, poboljšavajući čitljivost sadržaja i vizuelni izgled.

Da biste bolje razumeli razliku između `margin` i `padding`, istražimo nekoliko primera.

Evo primera HTML-a sa tri elementa pasusa na stranici:

:::interactive_editor

```html
<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

:::

Da biste dodali razmak iznad svakog elementa pasusa, možete koristiti svojstvo `margin-top` na sledeći način:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p>Paragraph one</p>
<p>Paragraph two</p>
<p>Paragraph three</p>
```

```css
p {
  margin-top: 20px;
  border: 2px solid black;
}
```

:::

U ovom primeru, koristimo `20px` samo za margine na svakom paragrafskom elementu.

Takođe imamo punu crnu ivicu sa sve četiri strane kako biste lakše videli margine.

Sifa nne tofauti za `margin` ni `margin-top`, `margin-right`, `margin-bottom` na `margin-left`.

Evo primera koji koristi sva četiri svojstva:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin-top: 10px;
  margin-right: 20px;
  margin-bottom: 30px;
  margin-left: 40px;
  border: 2px solid black;
}
```

:::

U ovom primeru, postavljamo vrednosti razmaka na sve četiri strane paragrafskog elementa. Takođe je dodata čvrsta crna granica kako biste bolje videli margine.

Drugi način za korišćenje atributa `margin` je korišćenjem skraćivanja atributa. Možete navesti jednu, dve, tri ili četiri vrednosti odjednom. Pogledaćemo ove opcije zajedno.

Kada koristite jednu vrednost u skraćivanju `margin`, ta ista vrednost će biti primenjena na sve četiri strane ciljnog elementa.

Ovo je primer korišćenja jedne vrednosti u skraćenici za `margin`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px;
}
```

:::

Ovaj kodni primer će postaviti `10px` od `margin` na svih četiri strane elementa paragrafa jednako.

Koristiš dve vrednosti, prva vrednost se koristi za `top` i `bottom`, dok druga vrednost se koristi za `left` i `right` elementa.

Ovo je primer upotrebe dve vrednosti za skraćivanje `margin`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px;
}
```

:::

Ovo će postaviti margine za `top` i `bottom` da bude `10px`, na `20px` za margine za `left` i `right` za element paragrafa.

Ako se dostave tri vrednosti, prva vrednost će biti korišćena za margina `top`, druga vrednost za margine `left` i `right`, a treća vrednost za margina `bottom`.

Ovo je primer za bolje razumevanje:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px;
}
```

:::

Ovo će postaviti margine na `10px` za `top`, `20px` za `left` na `right`, i `30px` za `bottom`.

Kada koristite četiri vrednosti, ovo vam daje više kontrole jer možete definisati svaku vrednost margina za svaku stranu ciljnog elementa nezavisno.

Prva vrednost cilja `top`, druga vrednost cilja `right`, treća vrednost cilja `bottom`, a četvrta vrednost cilja `left`.

Ovde je primer korišćenja skraćene notacije za marginu sa četiri vrednosti:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  margin: 10px 20px 30px 40px;
}
```

:::

Ovo će postaviti margina na `10px` za `top`, `20px` za `right`, `30px` za `bottom`, i `40px` za `left`.

Atribut ``padding`` se koristi za postavljanje razmaka unutar elementa, između njegovog sadržaja i ivice.

Kao i kod svojstva `margin`, četiri svojstva za `padding` su `padding-top`, `padding-right`, `padding-bottom` i `padding-left`.

Evo primera kako da postavite `padding` za element pasusa:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 30px;
  padding-left: 40px;
  border: 2px solid black;
}
```

:::

Ovo će postaviti unutrašnji razmak na `10px` za `top`, `20px` za `right`, `30px` za `bottom` i `40px` za `left`.

Kao što vidiš, `padding` se primenjuje na sadržaj unutar ivice, za razliku od `margin`, koji se primenjuje izvan ivice.

Kao i kod svojstva `margin`, možeš koristiti i skraćeni zapis za svojstvo `padding`.

U skraćenom zapisu svojstva `padding` možeš navesti jednu, dve, tri ili četiri vrednosti.

Ovde je primer korišćenja `padding` za element paragrafa iz prethodnog teksta:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<span>Paragraph one</span>
<p>Paragraph two</p>
<span>Paragraph three</span>
```

```css
p {
  padding: 10px 20px 30px 40px;
  border: 2px solid black;
}
```

:::

U ovom primeru, korišćenjem skraćenice koda postaviće unutrašnji prostor na `10px` za `top`, `20px` za `right`, `30px` za `bottom`, i `40px` za `left` za element paragrafa.
