# --description--

U prethodnoj lekciji, naučili ste kako da radite sa tipom podataka tuple. U ovoj lekciji, naučićete o nekim uobičajenim metodama koje ćete koristiti kada radite sa tuple-ima.

Prvi put koji ćemo naučiti je `count()`. Ovaj put se koristi za određivanje koliko puta element postoji u tuple-u. Evo primera kako proveriti koliko puta niz slova `Rust` postoji u tuple-u pod nazivom `programming_languages`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

Pošto se `Rust` pojavljuje dva puta u tuple-u, metoda `count()` vraća broj `2`. Ako element naveden u funkciji `count()` uopšte ne postoji u tuple-u, povratna vrednost će biti `0`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

Ako nijedan predlog nije proslijeđen u funkciji `count()`, onda Python će vratiti `TypeError`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count()

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: tuple.count() takes exactly one argument (0 given)
"""
```

Sledeća metoda koju ćemo razmotriti je `index()`. Metoda `index()` služi za pronalaženje indeksa na kojem se određeni element nalazi u tuple-u. Evo primera korišćenja metode `index()` za pronalaženje indeksa niske `Java`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

Ako navedeni element ne može da se pronađe, Python će prijaviti `ValueError`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: tuple.index(x): x not in tuple
"""
```

Još jedna stvar koju možeš da uradiš sa metodom `index()` jeste da proslediš opcione argumente za početni i krajnji indeks. Evo primera prosleđivanja opcionog početnog indeksa:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

U ovom primeru određujemo odakle treba da počne pretraga niske `Python`. Prosleđivanjem broja `3` kao drugog argumenta metodi `index()` određujemo da pretraga počne od indeksa `3`. Pošto se `Python` pojavljuje dva puta u tuple-u, metoda `index()` će vratiti indeks `5` umesto indeksa `2` zbog korišćenja opcionog početnog indeksa.

Takođe možeš da proslediš opcioni krajnji indeks. Evo izmenjenog primera koji određuje i početni i krajnji indeks:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

Sada je rezultat indeks `2`, zato što pretraga počinje od indeksa `2` i traje do, ali ne uključujući, indeks `5`.

Još jedna često korišćena funkcija za rad sa tuple-ovima je `sorted()`. U prethodnoj lekciji naučio si o metodi `sort()` za liste. Funkcija `sorted()` može da se koristi nad bilo kojim iterabilnim objektom, uključujući i tuple-ove.

Evo primera kako kreirati novu listu brojeva koristeći funkciju `sorted()`:

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

Funkcija `sorted()` će uvek kreirati novu listu sortirovanih vrednosti. Ovo se razlikuje od metode `sort()` koja organizuje elemente liste na mestu i ne vraća novu listu.

Ako treba da prilagodite ponašanje sortiranja za iterabil, možete koristiti opcione argumente `reverse` i `key`. Evo primera korišćenja argumenta `key` za sortiranje elemenata u tupelu po dužini:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

Ako želite kreirati novu listu vrijednosti u obrnutom redoslijedu, onda možete koristiti argument `reverse` na sljedeći način:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

Tuples su vrsta podataka koja se često koristi u Python. Razumevanje kako raditi sa njima, zajedno sa nekim korisnim metodama i funkcijama, pomoći će vam da napišete efikasniji kod.
