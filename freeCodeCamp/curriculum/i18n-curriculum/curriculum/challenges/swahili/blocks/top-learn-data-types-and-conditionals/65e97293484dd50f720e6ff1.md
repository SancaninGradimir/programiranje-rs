# --description--

Drugi način za poređenje uslovnih izraza je ternarni operator. Ternarni operator je kratak način pisanja izraza `if-else`. Koristi se za procenu uslova i vraćanje vrednosti u zavisnosti od rezultata tog uslova. Sintaksa ternarnog operatora je sledeća:

```javascript
let result = condition ? value1 : value2;
```

U gornjem primeru, ako je uslov tačan, vrednost od `result` biće `value1`. Inače, vrednost od `result` biće `value2`.

Ternarni operateri se često koriste za postavljanje vrednosti za parametar u zavisnosti od uslova. Takođe se koriste za vraćanje vrednosti u zavisnosti od uslova.
