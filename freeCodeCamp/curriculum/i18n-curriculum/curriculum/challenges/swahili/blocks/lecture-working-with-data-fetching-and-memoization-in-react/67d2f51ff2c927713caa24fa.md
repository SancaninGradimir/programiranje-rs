# --description--

U poslednjoj lekciji, naučili ste o memoizaciji i kako radi dugme `useMemo`.

U ovom tutorijalu, naučićete kako dugme `useCallback` i `React.memo` funkcionišu.

U prethodnoj lekciji pomenuli smo da je `useCallback` namenjen memorisanju referenci na funkcije.

Kwa `React.memo`, omogućava vam da sačuvate memorijski deo kako bi sprečio ponovno prikazivanje bez razloga kada se njegovi parametri nisu promenili.

Ovo je osnovna sintaksa za `useCallback`:

```js
const handleClick = useCallback(() => {
  // code goes here
}, [dependency]);
```

Ovde je osnovna sintaksa `React.memo`:

```jsx
const MemoizedComponent = React.memo(({ prop }) => {
  return (
    <>
      {/* Presentation */}
    </>
  )
});
```

Tuchunguze mfano wa kitufe cha `useCallback`:

```jsx
import { useState, useEffect } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  const handleClick = () => {
    setCount((prevCount) => prevCount + 1);
  };

  useEffect(() => {
    console.log("useEffect runs");
  }, [handleClick]);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={handleClick}>Increment</button>
    </div>
  );
}

export default Counter;
```

U ovom delu efekat se izvršava svaki put kada se `handleClick` promeni, zato što se funkcija `handleClick` ponovo kreira pri svakom renderovanju.

Da biste ovo popravili, morate reći React da prikaže funkciju `handleClick` kao jedan stavak u svim prikazima za zapisivanje memorije na dugme `useCallback`, kako bi se ne regenerisao:

```jsx
import { useState, useEffect, useCallback } from "react";

function Counter() {
  const [count, setCount] = useState(0);

  // Memoize the handleClick function with useCallback
  const handleClick = useCallback(() => {
    setCount((prevCount) => prevCount + 1);
  }, []);

  useEffect(() => {
    console.log("useEffect runs");
  }, [handleClick]);

  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={handleClick}>Increment</button>
    </div>
  );
}

export default Counter;
```

Sada funkcija `handleClick` nije ponovo kreirana pri svakom prikazivanju.

Da bismo vam pokazali kako funkcija višeg reda `React.memo` (ili `memo`) i hook `useCallback` rade zajedno, evo komponente `Counter` sa funkcijom `handleClick` kojoj je potreban `useCallback`, ali ga trenutno ne koristi.

```jsx
import { useState, useEffect, useCallback } from "react";
import CounterChild from "./CounterChild";

function Counter() {
  const [count, setCount] = useState(0);
  const [timer, setTimer] = useState(new Date().toLocaleTimeString());

  const handleClick = () => {
    setCount(count + 1);
  };

  useEffect(() => {
    const interval = setInterval(() => {
      setTimer(new Date().toLocaleTimeString());
    }, 1000);

    return () => clearInterval(interval);
  }, []);

  return (
    <div>
      <h1>Time: {timer}</h1>
      <p>Count: {count}</p>
      <button onClick={handleClick}>Increment</button>
      <CounterChild onClick={handleClick} />
    </div>
  );
}

export default Counter;
```

Ova funkcija takođe ima indikator vremena u stanju koje se ažurira svake sekunde. Ovo uzrokuje da se deo ponovo prikaže svaki put kada `timer` promeni, što rezultira ponovnim kreiranjem funkcije `handleClick` prilikom svakog prikazivanja.

Stoga `handleClick` mora da se snimi zapis za `useCallback`.

Ovo je deo `CounterChild`:

```jsx
const CounterChild = ({ onClick }) => {
  console.log("CounterChild component rendered");
  return <button onClick={onClick}>Increment from Child</button>;
};

export default CounterChild;
```

Ovaj deo od `CounterChild` prima parametar sa `onClick`, što mu daje mogućnost da takođe doda indikator iz njega.

Pošto je deo `CounterChild` zavisno od dela `Counter`, takođe će se ponovo prikazati svaki put kada `Counter` bude prikazan zbog promenljivog indikatora. Stoga, `CounterChild` takođe zahteva beleženje.

Bez snimanja memorije, jer se deo ponovo prikazuje zbog indikatora koji se ažurira svake sekunde, deo `CounterChild` takođe se ponovo prikazuje.

Da bi se ovo spriječilo, morate dodati memorijski dio za `CounterChild` za `React.memo`:

```jsx
import React from "react";

const CounterChild = React.memo(({ onClick }) => {
  console.log("CounterChild component rendered");
  return <button onClick={onClick}>Increment from Child</button>;
});

export default CounterChild;
```

Stvari i dalje ne idu dobro čak ni nakon pisanja zapisa `CounterChild` za `React.memo`.

Ovo se dešava zato što funkcija `handleClick` je ponovo kreirana svaki put kada se prikaže, pa tako i zahteva zapisivanje u `useCallback`, kako bi rekla React da funkcija mora ostati ista za sve prikaze:

```js
const handleClick = useCallback(() => {
  setCount((prevCount) => prevCount + 1);
}, [count]);
```

Sada, sekcija će biti prikazana ponovo samo kada se status `count` promeni.
