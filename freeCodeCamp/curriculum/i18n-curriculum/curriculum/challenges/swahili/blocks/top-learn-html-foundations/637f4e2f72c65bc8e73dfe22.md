# --description--

Element `<head>` je ono što postavlja važne meta informacije o vašoj web stranici i stvari koje su potrebne da se vaša web stranica ispravno prikaže u pretraživaču. Unutar `<head>`, ne smete koristiti nijedan element koji prikazuje sadržaj na web stranici.

## Element Meta Character Set
Mora da uvek ima oznaku `meta` za kodiranje `charset` web stranice unutar elementa `head`: `<meta charset="utf-8">`.

Postavljanje kodiranja je veoma važno jer osigurava da će web stranica tačno prikazati specijalne znakove i slova iz različitih jezika u pretraživaču.

## Element za Naslov
Drugi element koji morate uvek uključiti u zaglavlje dokumenta HTML je element `title`:

```html
<title>My First Webpage</title>
```

Funkcija `title` služi za pružanje web stranici naslova koji je čitljiv za ljude i koji se prikazuje na kartici pregledača vaše web stranice.

Ako ne uključite karakteristiku `title`, naslov web stranice biće podrazumevano postavljen na ime fajla. U vašem slučaju, to bi bilo `index.html`, što nije mnogo smisleno za korisnike; ovo bi učinilo veoma teško pronći vašu web stranicu ako korisnik ima otvorene mnogo kartica u pretraživaču.

Postoje još mnogo karakteristika koje se mogu postaviti unutar ``head`` dokumenta `HTML`. Međutim, za sada je važno znati samo o dve karakteristike koje ste ovde predstavili. Naučićete o više karakteristika koje se postavljaju unutar ``head`` u celokupnom programu.

Kada se vratimo na fajl `index.html`, dodaćemo element `head` sa elementom `charset` od `meta` i `title` unutar njega. Zaglavni element ulazi u element HTML i mora da bude uvek prvi element ispod oznake otvaranja `<html>`:


```html
<!DOCTYPE html>

<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Webpage</title>
  </head>
</html>
```

Šta je namena elementa `head`?

## --answers--

Element `head` se koristi za prikaz svih elemenata koji su prikazani na veb stranici.

---

Element `head` se koristi za prikazivanje važnih informacija o vašoj web stranici i služi za tačno prikazivanje web stranica korišćenjem elemenata `meta`.

---

Funkcija `head` služi za prikazivanje sadržaja naslova na web stranici.
