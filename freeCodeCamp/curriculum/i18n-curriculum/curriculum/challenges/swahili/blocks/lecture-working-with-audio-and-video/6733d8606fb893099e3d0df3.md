# --description--

Naučio si o API-jima `HTMLMediaElement` i MediaStreams, ali postoji još nekoliko API-ja koji mogu biti veoma korisni za rad sa video i audio sadržajem.

Prvo je Screen Capture API. Kao što ime sugerira, API vam omogućava da snimate ekran korisnika. API se otvara pozivanjem metode `getDisplayMedia()` objekta `mediaDevices` i korišćenjem vrađenog medijskog toka.

Postoji MediaStream Recording API. Ovaj API radi zajedno sa API MediaStreams-a, omogućavajući vam da snimate MediaStream (ili čak direktno `HTMLMediaElement`). Zatim emituje događaje `dataavailable` sa korisnim teretom (Blob) koji se može zapisati u lokalno skladište fajlova korisnika.

Jezgro ove tehnologije je Media Source Extensions API. Media Source Extensions API omogućava direktno prenositi sliku kamere korisnika na video element korišćenjem svojstva `srcObject`, na primer.

Konačno, Web Audio API pokreće gotovo sve što čuješ na internetu. Ovaj API obuhvata važne objekte kao što su `AudioBuffer` (koji predstavlja privremeni bafer za čuvanje audio podataka) i `AudioContext`.

Ovde završavamo naše lekcije o zvuku i videu!
