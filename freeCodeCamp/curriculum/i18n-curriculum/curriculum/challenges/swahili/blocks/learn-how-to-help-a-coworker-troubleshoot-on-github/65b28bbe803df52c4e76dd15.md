# --description--

Modalni glagoli kao `might` i `can` se koriste za pokazivanje mogućnosti i sposobnosti.

`Might` koristi se za nešto što može da se dogodi ili je tačno, kao što su, `It might rain today`.

`Can` koristi se za razgovor o sposobnosti osobe da nešto uradi, kao što je, `She can speak three languages`. Pokušajmo da koristimo ove reči u rečenici.
