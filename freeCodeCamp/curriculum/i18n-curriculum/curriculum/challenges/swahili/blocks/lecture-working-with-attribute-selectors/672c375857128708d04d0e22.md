# --interactive--

Kada radite sa listama organizovanim u HTML, svojstvo elementa `type` omogućava vam da utvrdite tip broja koji se koristi, kao što su decimalni brojevi, alfabetska slova ili rimski brojevi.

Atribut polja `type` može se postaviti na polje `ol` (poredjeni spisak) za kontrolisanje formata brojeva:

- `1` za listu brojeva (kao što su 1, 2, 3, ...),

- `A` za spisak velikih slova alfabeta (kao A, B, C, ...),

- `a` za listu malih slova alfabeta (poput a, b, c, ...),

- `I` za spisak velikih rimskih brojeva (kao što su I, II, III, ...),

- `i` za liste rimskih brojeva (kao što su i, ii, iii, ...).

Ovo je kako možete koristiti selektor atributa elementa da ciljate na strukturiranu listu sa specifičnim atributom `type`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="A"] {
  color: purple;
  font-weight: bold;
}
```

:::

U ovom primeru, pravilo za CSS će koristiti ljubičastu boju teksta i podebljano pisma za bilo koju strukturiranu listu koja koristi velika slova alfabeta, što je navedeno sa `type="A"`.

Slično tome, možete ciljati strukturirane liste koje koriste rimska brojeva za odabir onih koji imaju `type="i"`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="i">
  <li>Item 1</li>
  <li>Item 2</li>
</ol>
```

```css
ol[type="i"] {
  color: green;
}
```

:::

Ovo pravilo će promeniti boju teksta u zelenu za bilo koju strukturiranu listu koja koristi rimska broja.

Sposobnost sortiranja lista na osnovu svojstva elementa `type` daje vam veliku kontrolu nad izgledom liste u vašem dokumentu, omogućavajući prilagođavanje na osnovu načina na koji je sadržaj strukturiran.
