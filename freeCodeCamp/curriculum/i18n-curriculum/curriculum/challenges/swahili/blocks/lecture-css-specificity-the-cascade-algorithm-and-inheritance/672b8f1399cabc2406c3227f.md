# --interactive--

Ključna reč `!important` unutar CSS koristi se da bi obezbedila najviši prioritet stilskom pravilu, i time može da poništi bilo koju drugu deklaraciju atributa.

Kada se koristi, nateraće pretraživač da primeni definisani stil, bez obzira na validnost drugih selektora.

Recimo da ima paragrafski element HTML koji sadrži stilove unutar linije, kao što sledi:

:::interactive_editor

```html
<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

:::

U ovom primeru element pasusa ima pozadinsku boju `lightblue` i boju teksta `black`.

Datoteka CSS koristi sledeće stilove za element pasusa:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

```css
.para {
  background-color: black;
  color: white;
}
```

:::

Pošto inline stilovi imaju veći prioritet od selektora klase, ID selektora i selektora tipa, crna pozadina i bela boja teksta neće biti primenjene na taj element pasusa.

Da biste nadjačali te inline stilove, možete koristiti ključnu reč `!important` na sledeći način:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="para" style="background-color: lightblue; color: black;">
  This is a paragraph.
</p>
```

```css
.para {
  background-color: black !important;
  color: white !important;
}
```

:::

Ključna reč `!important` piše se posle vrednosti CSS svojstva, a pre tačke i zareza.

Sada će element pasusa imati crnu pozadinu i beli tekst zato što su inline stilovi nadjačani pomoću ključne reči `!important`.

Ključna reč `!important` u CSS-u koristi se da dodeli najviši prioritet CSS pravilu i efikasno nadjača bilo koju drugu deklaraciju, uključujući one sa većom specifičnošću i inline stilove.

Međutim, ključna reč `!important` ne menja specifičnost samog CSS selektora. Ona samo osigurava da se pravilo sa `!important` primeni čak i ako postoje druga pravila sa većom specifičnošću.

Neke upotrebe korišćenja ključne reči `!important` su poništavanje stilova iz biblioteka drugih ljudi ili radnih sistema kada nemate kontrolu nad prethodnim CSS.

Međutim, prekomerno korišćenje ključne reči `!important` može izazvati poteškoće u održavanju i popravljanju vaših grešaka CSS, jer narušava normalan tok stilova i može dovesti do neočekivanih rezultata.

Dakle, bolje je koristiti ključnu reč `!important` u maloj meri.
