# --interactive--

Binarni logički operatori pomažu u proceni dve matematičke izraze i vraćaju rezultate na osnovu njihove istinitosti. Pogledaćemo tri poznata binarna logička operatera: logical AND, logical OR, i nullish coalescing.

Opereta za logički AND predstavljena je sa dva ampersand znaka (`&&`). Proverava da li su obe strane tačne i vraća rezultat. Ako su obe strane tačne, vraća vrednost druge strane, tj. one desne:

:::interactive_editor

```js
const result = true && 'hello';

console.log(result); // hello
```

:::

U gornjem primeru, tekst `hello` je ispisan u konzoli jer su obe strane `true`. Ako je bilo koja strana lažna vrednost, vraća lažnu vrednost:

:::interactive_editor

```js
const result = 0 && 3;

console.log(result); // 0
```

:::

Pošto je `0` lažna vrednost, broj `0` je ispisana u konzolu. A ako su obe strane lažne vrednosti, vraća prvu lažnu vrednost:

:::interactive_editor

```js
const result = false && 0;

console.log(result); // false
```

:::

Pošto je `false` lažno/netačno vrednost, onda je `false` ispisan na konzolu. Logički AND operator je važan kada želite da proverite više uslova i osigurate da su svi istiniti pre nastavka. Evo primera:

:::interactive_editor

```js
if (2 < 3 && 3 < 4) {
 console.log('The if block runs'); 
} else {
 console.log('The else block runs');
} 
```

:::

Ako je `2` manje od `3` I `3` je manje od `4`, onda će string `The if block runs` biti ispisano na konzoli.

Operacija logičkog ILI proverava da li je barem jedna strana tačna. Ako je prva strana tačna, vraća tu vrednost:

:::interactive_editor

```js
const result = 'This is truthy' || false;

console.log(result); // This is truthy
```

:::

Ako je prva strana lažna vrednost, ali druga strana istinita, druga vrednost će biti ispisana na konzoli:

:::interactive_editor

```js
const result = 0 || 'This is truthy';

console.log(result); // This is truthy
```

:::

Uobičajeno je koristiti operater logičkog ILI u izjavama za `if/else` kao što su sledeće:

:::interactive_editor

```js
let userInput;

if (userInput || 'Guest') {
 console.log('A user is present');
} else {
 console.log('No user detected');
}
```

:::

Pošto nismo postavili vrednost za promenu `userInput`, trenutno je `undefined`. Uslov u izveštaju `if` proverava da li je promena `userInput` ili niz karaktera `Guest` tačan. Pošto je niz karaktera `Guest` tačan u kontekstu takve vrednosti, niz karaktera `A user is present` će biti ispisano na konzolu.

Operator za nullish coalescing je napredniji od logičkog OR i logičkog AND. Predstavljen sa dva znaka upitnika (`??`), pomaže u situacijama kada želite da vratite vrednost samo ako je prva `null` ili `undefined`. Evo primera korišćenja operatera nullish coalescing:

:::interactive_editor

```ts
const result = null ?? 'default';

console.log(result); // default
```

:::

Pošto je `null` nulizna vrednost, niz znakova `default` će biti ispisano u konzolu. Operator za kondenzovanje nule (nullish coalescing) je veoma koristan u situacijama gde su `null` ili `undefined` jedine vrednosti koje bi trebalo da izazovu alternativnu putanju ili podrazumevanu vrednost. Evo primera za rukovanje podešavanjima korisničkih preferencija:

:::interactive_editor

```ts
const userSettings = {
 theme: null,
 volume: 0,
 notifications: false,
};

let theme = userSettings.theme ?? 'light';
console.log(theme); // light
```

:::

U gornjem primeru imamo objekat pod nazivom `userSettings` koji sadrži svojstva `theme`, `volume` i `notifications`. Svojstvu `theme` pristupamo pomoću tačkaste notacije, kao u `userSettings.theme`. Više o radu sa objektima naučićeš u sledećoj lekciji. Pošto je korisnička postavka `theme` trenutno postavljena na `null`, u konzoli će biti ispisan string `light`.
