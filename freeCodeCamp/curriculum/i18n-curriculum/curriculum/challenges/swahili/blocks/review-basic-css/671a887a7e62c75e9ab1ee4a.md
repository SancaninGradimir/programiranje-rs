## Osnovi CSS

- **CSS šta je?**: Cascading Style Sheets (CSS) je jezik stilova koji se koristi za primenu stilova na elementima HTML. CSS služi za boje, pozadinske slike, raspored i više.
- **Osnovna struktura pravila CSS**: CSS pravilo CSS sastoji se od dva glavna dela: selektor i blok deklaracija. Selektor je struktura koja se koristi u CSS za identifikovanje i ciljanje specifičnih elemenata HTML radi stilizovanja. Blok deklaracija postavlja skup stilova za jedan ili više selektora.

Ovo je opšta struktura zakona CSS:

```css
selector {
    property: value;
}
```

- **Element `meta name="viewport"`**: Ovaj element `meta` pruža uputstva pregledaču kako da kontroliše dimenzije i skaliranje stranice na različitim uređajima, posebno mobilnim telefonima i tabletima.
- **Podrazumevani stilovi pregledača**: Svaki element HTML imaće postavljene podrazumevane stilove pregledača. Ovo obično uključuje stvari kao što su granice i interni padding po podrazumevanom.

## CSS unutar linije, unutar fajla, i van

- **CSS inline**: Ovi stili su direktno napisani unutar elementa HTML koristeći atribut `style`. Često se koristi CSS inline zbog pravila razdvajanja problema.

Ovo je primer CSS unutar linije:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p style="color: red;">This is a red paragraph.</p>
```

:::

- **CSS unutar fajla**: Ovi stili su napisani unutar oznaka `<style>` u sekciji `head` dokumenta HTML. Ovo može biti korisno za kreiranje kratkih primera koda, ali obično ćete morati da koristite CSS unutar fajla.
- **CSS van fajla**: Ovi stili su napisani u zasebni fajl CSS i povezani sa dokumentom HTML korišćenjem funkcije `link` u sekciji `head`. Za mnoge projekte, koristićete fajl CSS van umesto CSS unutar fajla ili unutar linije.

## Rad sa svojstvima `width` i `height`

- **Svojstvo `width`**: Ovo svojstvo opisuje širinu elementa. Ako širina nije definisana, podrazumevano je postavljena na `auto`. To omogućava pretraživaču da odredi širinu elementa na osnovu sadržaja, roditelja i tipa prikaza.
- **Svojstvo `min-width`**: Ovo svojstvo opisuje minimalnu širinu za element.
- **Svojstvo `max-width`**: Ovo svojstvo opisuje maksimalnu širinu za element.
- **Svojstvo `height`**: Ovo svojstvo opisuje visinu elementa. Slično tome, visina je `auto` podrazumevano, što znači da će se prilagoditi sadržaju unutar njega.
- **Svojstvo `min-height`**: Ovo svojstvo opisuje minimalnu visinu za element.
- **Svojstvo `max-height`**: Ovo svojstvo opisuje maksimalnu visinu za element.

## Različite vrste za mešavinu CSS

- **Kombinacija nasleđivanja**: Ova kombinacija se koristi za ciljanje elemenata koji su potomci pomenutog roditeljskog elementa. Sledeći primer će odrediti sve stavke `li` unutar elemenata `ul`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul>
    <li>Example item one</li>
    <li>Example item two</li>
    <li>Example item three</li>
</ul>
```

```css
ul li {
    background-color: yellow;
}
```

:::

- **Detetška kombinacija (`>`)**: Ova kombinacija se koristi za odabir elemenata koji su direktni deca pomenutog roditeljskog elementa. Sledeći primer cilja sve elemente `p` koji su direktna deca klase `container`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <p>This will get styled.</p>

  <div>
    <p>This will not get styled.</p>
  </div>
</div>
```

```css
.container > p {
  background-color: black;
  color: white;
}
```

:::

- **Kombinacija sledećeg brata (`+`)**: Ova kombinacija bira element koji sledi odmah nakon spomenutog srodnog elementa. Sledeći primer će izabrati paragraf element koji sledi odmah nakon elementa `h2`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<h2>I am a sub heading</h2>

<p>This paragraph element will get a red background.</p>
```

```css
h2 + p {
  background-color: red;
}
```

:::

- **Kombinator za sledeće braće (`~`)**: Ovaj kombinator bira sve braće navedenog elementa koje dolaze posle njega. Sledeći primer primeni će stil samo na drugi pasus jer je jedini koji je braća elementa `ul` i deli istog roditelja.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <p>This will not get styled.</p>
  <ul>
    <li>Example item one</li>
    <li>Example item two</li>
    <li>Example item three</li>
  </ul>
  <p>This will get styled.</p>
</div>
<p>This will not get styled.</p>
```

```css
ul ~ p {
  background-color: green;
}
```

:::

## Elementi linije, bloka i linijskog bloka

- **Inline elementi**: Inline elementi zauzimaju samo potrebnu širinu i ne započinju na novom redu. Ovi elementi teku unutar sadržaja, omogućavajući tekstu i drugim inline elementima da se prikazuju sa njima. Popularni inline elementi su `span`, `a`, i `img`.
- **Blok elementi**: Blok elementi počinju na novom redu i po podrazumevanju zauzimaju celu dostupnu širinu, protežući se preko širine svog kontejnera. Popularni blok elementi su `div`, `p`, i `section`.
- **Inline-blok elementi**: Element možete pretvoriti u `inline-block` koristeći svojstvo `display`. Ovi elementi funkcionišu kao inline elementi, ali mogu imati `width` i `height` kao blok elemente.

## Rub i unutrašnji prostor

- **Svojstvo `margin`**: Ovo svojstvo se koristi za postavljanje razmaka izvan elementa, između granice elementa i elemenata koji ga okružuju.
- **Svojstvo `padding`**: Ovo svojstvo se koristi za postavljanje razmaka unutar elementa, između sadržaja i njegove granice.
- **Skraćeni kod `margin`**: Možete postaviti vrednosti od 1 do 4 za postavljanje ivica. Jedna vrednost koristi se za sve četiri strane; dve vrednosti postavljaju `top` i `bottom`, zatim `right` i `left`; tri vrednosti postavljaju `top`, ravno (`right` i `left`), zatim `bottom`; četiri vrednosti postavljaju `top`, `right`, `bottom`, `left`.
- **Skraćeni kod `padding`**: Možete postaviti vrednosti od 1 do 4 za postavljanje unutrašnjeg razmaka (padding). Jedna vrednost koristi se za sve četiri strane; dve vrednosti postavljaju `top` i `bottom`, zatim `right` i `left`; tri vrednosti postavljaju `top`, ravno (`right` i `left`), zatim `bottom`; četiri vrednosti postavljaju `top`, `right`, `bottom`, `left`.

## Jedinstvenost CSS

- **Inline CSS specifičnost**: Inline CSS ima najveću specifičnost jer se primenjuje direktno na element. Nadjačava svaki interni ili eksterni CSS. Vrednost specifičnosti za inline stilove je (1, 0, 0, 0).
- **Interni CSS**: Interni CSS definiše se unutar elementa `style` u odeljku `head` HTML dokumenta. Ima manju specifičnost od inline stilova. I interni i eksterni CSS imaju isti nivo specifičnosti koji zavisi od korišćenih selektora, a ne od mesta gde se nalaze.
- **Eksterni CSS**: Eksterni CSS se povezuje pomoću elementa `link` u odeljku `head` i nalazi se u posebnoj `.css` datoteci. Kao i interni CSS, njegova specifičnost zavisi od korišćenih selektora. Kada dva pravila imaju istu specifičnost, redosled u izvornom kodu određuje pobednika: prednost ima pravilo koje se pojavljuje kasnije u dokumentu. Eksterni CSS pruža mnogo bolje održavanje velikih projekata.
- **Univerzalni selektor (`*`)**: Posebna vrsta CSS selektora koja odgovara svakom elementu u dokumentu. Često se koristi za primenu stilova na sve elemente na stranici, što može pomoći pri resetovanju ili ujednačavanju stilova između različitih pregledača. Univerzalni selektor ima najnižu specifičnost od svih selektora. Doprinosi vrednošću 0 svim delovima specifičnosti (0, 0, 0, 0).
- **Selektori tipa**: Ovi selektori ciljaju elemente na osnovu naziva njihove oznake (tag-a). Selektori tipa imaju nižu specifičnost u odnosu na ostale selektore. Njihova vrednost specifičnosti je (0, 0, 0, 1).
- **Selektori klase**: Ovi selektori se označavaju tačkom (`.`) praćenom nazivom klase. Vrednost specifičnosti selektora klase je (0, 0, 1, 0). To znači da mogu da nadjačaju selektore tipa, ali ih mogu nadjačati ID selektori i inline stilovi.
- **ID selektori**: ID selektori se označavaju znakom tarabe (`#`) praćenim nazivom ID-ja. Imaju veoma visoku specifičnost, veću od selektora tipa i klase, ali manju od inline stilova. Vrednost specifičnosti ID selektora je (0, 1, 0, 0).
- **Ključna reč `!important`**: Koristi se da bi se CSS pravilu dao najviši prioritet, omogućavajući mu da nadjača bilo koju drugu deklaraciju svojstva. Kada se koristi, primorava pregledač da primeni navedeni stil bez obzira na specifičnost drugih selektora. Koristite `!important` oprezno, jer može otežati održavanje i otklanjanje grešaka u CSS-u.
- **Cascade algoritam**: Algoritam koji određuje koja CSS pravila će biti primenjena kada više stilova cilja isti element. Osigurava da se primeni najprikladniji stil na osnovu jasno definisanih pravila.
- **CSS nasleđivanje**: Proces u kojem se stilovi prenose sa roditeljskih elemenata na njihove potomke. Nasleđivanje omogućava da stilove definišete na višem nivou stabla dokumenta i primenite ih na više elemenata bez pojedinačnog definisanja.
