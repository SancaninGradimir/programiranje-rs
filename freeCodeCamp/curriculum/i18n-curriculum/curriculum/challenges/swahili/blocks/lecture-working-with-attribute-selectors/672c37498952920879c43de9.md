# --interactive--

Kada razvijate sajtove koji podržavaju više jezika ili obrađujete specifične atribute podataka, često je potrebno organizovati elemente na osnovu jezika koji se govore ili specifičnih vrednosti podataka.

Svojstva za `lang` i `data-lang` se koriste mnogo za ove svrhe, a birač svojstava elemenata u CSS omogućio je efikasno korišćenje stilova zasnovanih na ova svojstva.

Svojstvo `lang` se koristi u HTML za određivanje jezika sadržaja unutar elementa. Možda želite da organizujete različite elemente u zavisnosti od napisanih jezika, posebno na višejezičnom veb-sajtu.

Ovo je primer kako možete koristiti selektor atributa elementa za ciljanje elemenata sa specifičnim atributom/svojstvom `lang`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">This is an English paragraph.</p>
<p lang="fr">Ceci est un paragraphe en français.</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

Ovo pravilo za CSS koristi stil pisanja velikim slovima za bilo koji element paragrafa čije je svojstvo `lang` postavljeno na engleski (`en`). Ovo može biti korisno za prikaz engleskog teksta u dokumentu koji uključuje više jezika.

Takođe, možete koristiti selektor atributa elementa za ciljanje elemenata sa atributom `data-lang`.

Atributi specifičnih podataka kao `data-lang` se često koriste za skladištenje dodatnih informacija u elementima, kao što je određivanje jezika koji se koristi unutar određenog dela teksta.

Ovo je kako možete koristiti CSS da ciljate i organizujete elemente prema svojstvu `data-lang`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p data-lang="fr">Ceci est un paragraphe en français.</p>
<p data-lang="en">This is a paragraph in English.</p>
```

```css
p[data-lang="fr"] {
  color: blue;
}
```

:::

U ovom slučaju, bilo koji element iz `p` sa atributom `data-lang` postavljen na francuski (`fr`) biće sa tekstualnom bojom plava. Ovo vam omogućava da brzo identifikujete i organizujete sekcije sadržaja na osnovu informacija o jeziku sačuvanih u atributu `data-lang`.

Detektori značajki kao što su ovi pružaju moćan način za korišćenje uslovnih stilova zasnovano na metapodative unutar vašeg HTML, i čineći vaše veb stranice kontekstualnijim i dinamičnijim.
