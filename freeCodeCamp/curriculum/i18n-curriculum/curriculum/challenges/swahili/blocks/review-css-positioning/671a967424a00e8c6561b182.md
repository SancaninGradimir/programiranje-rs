## Rad sa floatovima

- **Definicija**: Floats se koriste za uklanjanje elementa iz njegovog normalnog toka na stranici i postavljanje ga sa leve ili desne strane njegovog kontejnera. Ovo uzrokuje da tekst obavija flotirani element.

```css
float: left;
float: right;
```

- **Kufuta floats**: Svojstvo `clear` se koristi za određivanje da li element treba da bude pomeren ispod plutajućeg elementa. Kada postoji mnogo elemenata raspoređenih jedan pored drugog, mogu nastati problemi sa spajanjem ili vezivanjem u strukturi. Stoga je metoda `clearfix` kreirana da reši ovaj problem.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="clearfix">
  <div class="box left">Left</div>
  <div class="box right">Right</div>
</div>
```

```css
.box {
  width: 100px;
  height: 60px;
  color: white;
  text-align: center;
  line-height: 60px;
}

.left {
  float: left;
  background: teal;
}

.right {
  float: right;
  background: purple;
}

.clearfix::after {
  content: "";
  display: block;
  clear: both;
}
```

:::

## Konfiguracija static, u odnosu na apsolutno

- **Upangaji wa static**: Ovo je standardni tok dokumenta. Elementi se postavljaju od vrha ka dnu i od leve ka desne, sekvencijalno.
- **Upangaji wa relative**: Ovo vam omogućava da koristite svojstva `top`, `left`, `right` i `bottom` kako biste postavili element unutar standardnog toka dokumenta. Takođe možete koristiti relativni raspored da elementi zalepite za druge elemente na stranici.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="relative">Relative Element</div>
```

```css
.relative {
    position: relative;
    top: 30px;
    left: 30px;
}
```

:::

- **Apsolutno pozicioniranje**: Omogućava vam da izvučete element iz normalnog toka dokumenta, i da ga učinite samostalnim bez uticaja drugih elemenata.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="parent">
  Parent
  <div class="positioned">Absolute</div>
</div>
```

```css
.parent {
  position: relative;
  height: 120px;
  background: #eee;
}

.positioned {
  position: absolute;
  top: 30px;
  left: 30px;
  background-color: coral;
}
```

:::

## Pozicioniranje fixed i sticky

- **Fiksno pozicioniranje**: Element koji je postavljen na `position: fixed`, uklanja se iz normalnog toka dokumenta i postavlja prema vidljivom području (viewport), što znači da ostaje na mestu čak i kada korisnik skroluje dole ili gore. Ovo se često koristi za elemente kao što su zaglavlja stranice ili navigacione trake koji moraju biti vidljivi uvek.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="navbar">Fixed Navbar</div>
<div style="height: 600px;"></div>
```

```css
.navbar {
  position: fixed; 
  top: 0; 
  width: 100%;
  background: #0077ff;
  color: white;
  padding: 8px; 
}
```

:::

- **Sticky pozicioniranje**: Ovaj tip pozicioniranja je kombinacija relativnog i fiksiranog. Inicialno, element radi kao da je relativno pozicioniran, ostajući unutar toka dokumenta. Međutim, kada korisnik skroluje i element prođe određenu tačku, on se pričvršćuje za prikazni prozor (obično na vrhu) i funkcioniše kao fiksirani. Ovo je idealno za kreiranje elemenata poput *sticky* navigacionih traka, koje postaju fiksne tek kada korisnik stigne do određenog dela stranice.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Scroll down</p>
<div class="positioned">Sticky Element</div>
<div style="height: 500px;"></div>
```

```css
.positioned {
  position: sticky;
  top: 30px;
  left: 30px;
  background: #4caf50;
  color: white;
  padding: 6px;
}
```

:::

## Rad sa svojstvom `z-index`

- **Definicija**: Svojstvo `z-index` unutar CSS koristi se za kontrolisanje vertikalnog rasporeda elemenata sa posebnim pozicioniranjem koji su vezani za stranicu.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box box1">1</div>
  <div class="box box2">2</div>
</div>
```

```css
.container {
  position: relative;
  width: 300px;
  height: 300px;
  border: 1px solid black;
}

.box {
  position: absolute;
  width: 100px;
  height: 100px;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

.box1 {
  background: lightcoral;
  top: 20px;
  left: 20px;
  z-index: 1;
}

.box2 {
  background: steelblue;
  top: 40px;
  left: 40px;
  z-index: 2;
}
```
