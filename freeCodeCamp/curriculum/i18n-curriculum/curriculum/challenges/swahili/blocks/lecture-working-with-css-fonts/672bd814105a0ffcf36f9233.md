# --interactive--

Porodična grupa fontova je skup fontova koji dele zajednički dizajn. Svi fontovi koji pripadaju jednoj porodici potiču iz istog osnovnog tipa fonta, ali takođe imaju razlike u stilu, težini i širini. Možete ih zamisliti kao rođake koji dele slične karakteristike, ali takođe imaju određene razlike. Na primer, Arial je porodična grupa fontova koja uključuje varijacije kao što su Arial Bold i Arial Italic.

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/what-are-font-families-and-how-do-they-work-1.png" alt="Diagram showing 'Arial' at the top with two blue arrows pointing down to 'Arial Bold' on the left and 'Arial Italic' on the right, illustrating font family variations" />

U CSS, možete postaviti porodičnu tipografiju koristeći svojstvo `font-family`. Evo primera postavljanja fonta za paragraf na `Arial`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="arial-font">Example paragraph using Arial font.</p>
<p>Paragraph not using Arial font.</p>
```

```css
#arial-font {
  font-family: Arial;
}
```

:::

Ali šta ako željena porodica fontova nije dostupna? Možeš da navedeš više porodica fontova po redosledu prioriteta, razdvojenih zarezima. One će služiti kao rezervni fontovi. U ovom primeru, Arial je primarni font, a Lato rezervni:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="specified-font">Example paragraph using specified fonts.</p>
<p>Paragraph not using specified fonts.</p>
```

```css
#specified-font {
  font-family: Arial, Lato;
}
```

:::

Pregledač će prikazati prvi font (Arial) ako je dostupan na uređaju. Ako nije, prikazaće drugi font (Lato).

Takođe treba da znaš kako ovaj proces funkcioniše u pozadini. Pretraga se ne završava kada se pronađe prvi font. Fontovi se biraju za svaki karakter pojedinačno, pa ako prvi font ne sadrži određeni znak, pregledač će pokušati da ga pronađe u sledećem fontu sa liste prioriteta. Zanimljivo, zar ne?

U veb razvoju ćeš se susretati i sa generičkim porodicama fontova. One su veoma važne. Generička porodica fontova predstavlja rezervnu kategoriju koju pregledači koriste kada fontovi višeg prioriteta nisu dostupni. Da bi sadržaj ostao čitljiv, pregledač bira najprikladniji dostupni font iz navedene generičke porodice. Neke od najčešće korišćenih generičkih porodica su:

- serif
- sans-serif
- monospace
- cursive
- fantasy

Međutim, postoji još više opcija. U sledećem kodu imamo tri rezervne porodice fontova:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="specified-font">Example paragraph using specified fonts.</p>
<p>Paragraph not using specified fonts.</p>
```

```css
#specified-font {
  font-family: Arial, Lato, sans-serif;
}
```

:::

Arial ima najveći prioritet. Ako nije dostupan, pregledač će pokušati da prikaže Lato. Ako nijedan od ta dva fonta nije dostupan, koristiće generičku porodicu `sans-serif` i izabrati odgovarajući font instaliran na korisnikovom sistemu.

Uvek treba da navedeš generičku porodicu fontova na kraju liste. Imaj na umu da generički font može izgledati drugačije od fonta koji si prvobitno planirao, zato je dobro proveriti kako rezervni fontovi izgledaju u različitim pregledačima.

Da bi korisničko iskustvo bilo što stabilnije, savetuje se korišćenje sigurnih web fontova. Ove porodice fontova su obično instalirane na mnogim uređajima, pa postoji velika verovatnoća da će biti dostupne i ispravno prikazane za većinu korisnika. Vi ćete saznati više o njima u narednim lekcijama.

Font-familije su važne u svetu web dizajna. Korišćenjem svojstva `font-family`, možete odmah promeniti izgled i osećaj vašeg teksta.
