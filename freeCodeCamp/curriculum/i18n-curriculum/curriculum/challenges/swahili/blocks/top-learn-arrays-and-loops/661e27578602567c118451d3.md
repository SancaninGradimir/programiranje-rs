# --description--

Najčešće korišćena metoda za dodavanje novog elementa u niz podataka je korišćenje metode `push()`. Metoda `push()` dodaje jedan ili više elemenata na kraju niza podataka i vraća novu dužinu tog niza.

Na primer, da dodate novi element u red podataka za `pet`, možete koristiti sledeći kod:

```javascript
const pet = ['cat', 'dog', 'bunny'];
pet.push('parrot');
console.log(pet); // Output: ['cat', 'dog', 'bunny', 'parrot']
```

Da biste uklonili poslednji element iz niza podataka, možete koristiti metodu ``pop()``. Metoda ``pop()`` uklanja poslednji element iz niza podataka i vraća taj element.

Na primer, da biste uklonili poslednji element iz niza podataka za `pet`, možete koristiti sledeći kod:

```javascript
const pet = ['cat', 'dog', 'tiger'];
pet.pop();
console.log(pet); // Output: ['cat', 'dog']
```
