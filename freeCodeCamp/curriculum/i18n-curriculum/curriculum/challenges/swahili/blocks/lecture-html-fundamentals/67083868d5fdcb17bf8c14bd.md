# --interactive--

Element HTML, ili referenca karaktera, je skup znakova koji se koriste za predstavljanje karaktera sačuvanog u HTML.

Pretpostavimo da želite prikazati tekst `This is an <img/> element` na ekranu. Ako koristite kod koji se trenutno nalazi u editoru, neće prikazati rezultat koji želite. Čak i ako dodate atribute `src` i `alt` na primer, prikaže će sliku usred pasusa. To nije rezultat koji želite. Da biste interagovali sa ovim primerom, moraćete da omogućite interaktivni editor.

:::interactive_editor

```html
<p>This is an <img /> element</p>
```

:::

Kada HTML parser naiđe na znak manje od (`<`) praćen imenom HTML oznake, tumači ga kao HTML element. Zato ne možete dobiti željeni rezultat za `This is an <img/> element` na ekranu.

Da biste rešili ovaj problem, možete koristiti entitete HTML. Evo ažuriranog primera koji koristi tačne entitete HTML za simbole ispod (`<`) i iznad (`>`). Sada bi trebalo da vidite `This is an <img/> element` na ekranu.

Omogući interaktivni uređivač i pokušaj dodavanja `&lt;p&gt;learning is fun&lt;/p&gt;` ispod elementa paragrafa. Trebalo bi da vidite `<p>learning is fun</p>` na ekranu.

:::interactive_editor

```html
<p>This is an &lt;img /&gt; element</p>

```

:::

Ove vrste referenci karaktera poznate su kao imenovane reference karaktera. Imenovane reference počinju sa simbolom ampersanda (`&`) i završavaju se tačkom (`;`). Korišćenjem imenovane reference karaktera, parser za HTML neće zbuniti ovo sa stvarnim elementom od HTML.

Drugi tip referenciranja karaktera je referenca decimalnog broja. Ova referenca počinje znakom ampersanda i znakom hashtag-a (`#`), praćenim jednim ili više decimalnih cifara, i završava tačkom.

Ovo je primer korišćenja reference decimalnog broja za znak donjeg podkrešivanja.

Omogući editoru da interaguje i menja kod kako bi video različite simbole. Možete koristiti `&#169;` za simbol autorskih prava, a `&#174;` za registrovani simbol trgovačke marke.

:::interactive_editor

```html
&#60;
```

:::

Poslednja vrsta referenci na znakove je heksadecimalna numerička referenca. Ona počinje znakom ampersand (`&`), zatim znakom tarabe (`#`) i slovom `x`. Nakon toga sledi jedna ili više heksadecimalnih ASCII cifara, a završava se tačkom-zarezom.

Ovo je primer upotrebe heksadecimalne referencije za znak podvučnika.

Omogući editoru da se interakcioniše i promeni kod kako bi video različite znakove. Možete koristiti `&#x20AC;` za evro znak i `&#x03A9;` za veliko grčko slovo Omega.

:::interactive_editor

```html
&#x3C;
```

:::
