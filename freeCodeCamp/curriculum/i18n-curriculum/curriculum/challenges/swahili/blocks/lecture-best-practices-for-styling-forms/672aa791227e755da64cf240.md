# --interactive--

Kao što je slučaj sa svim tekstualnim elementima, morate osigurati da su stilovi koje koristite za unos teksta lako dostupni. To znači da font mora imati odgovarajuću veličinu, a boja mora imati dovoljno kontrasta sa pozadinom.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<form class="accessible-form">
  <label for="username">Username</label>
  <input type="text" id="username" name="username" placeholder="Enter your username">
  <button type="submit">Submit</button>
</form>

```

```css
body {
  background-color: #f9fafb;
  color: #222;
  padding: 2rem;
}

.accessible-form {
  max-width: 320px;
  margin: 0 auto;
}

label {
  display: block;
  font-weight: 600;
  margin-bottom: 0.5rem;
}

input[type="text"] {
  width: 100%;
  padding: 0.6rem 0.8rem;
  font-size: 1rem;
  border: 2px solid #555;
  border-radius: 4px;
  background-color: #fff;
  color: #111;
}

input[type="text"]:focus {
  outline: 3px solid #1e90ff;
  border-color: #1e90ff;
}

button {
  margin-top: 1rem;
  padding: 0.6rem 1rem;
  font-size: 1rem;
  background-color: #1e90ff;
  color: #fff;
  border: none;
  border-radius: 4px;
  cursor: pointer;
}

button:hover,
button:focus {
  background-color: #187bcd;
}

```

:::

Još jedna stvar na koju treba obratiti pažnju je da treba dozvoliti korisniku da izmenjuje unos. Na primer, ako je tekstualno polje, ne bi trebalo ukloniti mogućnost menjanja njegove veličine. Unos takođe treba da se prilagođava adekvatno kada korisnik povećava stranicu/prikaz.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<form class="accessible-form">
  <label for="message">Your message</label>
  <textarea id="message" name="message" placeholder="Type your message here..."></textarea>
  <button type="submit">Send</button>
</form>

```

  margin: 0 auto;

  display: block;

Elementi za unos takođe mogu dobiti fokus. Kada prilagođavate svoje stilove, trebalo bi da osigurate da postavite jasno vidljiv indikator kada element ima fokus, kao što je snažna granica.

Štaviše, postoji stanje greške koje treba uzeti u obzir. Kada tekst korisnika ne zadovoljava validaciju unosa, postoji vidljiv indikator koji ukazuje na problem.

Da bi se poruka o grešci ažurirala u realnom vremenu, morate koristiti JavaScript, o čemu ćete naučiti na narednim predavanjima.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<form class="accessible-form">
  <label for="email">Email address</label>
  <input 
    type="email" 
    id="email" 
    name="email" 
    placeholder="you@example.com" 
    aria-describedby="email-error"
  >
  <p id="email-error" class="error-message">
    Please enter a valid email address.
  </p>

  <button type="submit">Submit</button>
</form>

<script src="index.js"></script>
```
