# --interactive--

Dobra programska praksa je pisanje koda koji je čist, predvidiv i lak za testiranje i pronalaženje grešaka. Dve važne koncepte koje ćemo danas istražiti su čiste funkcije i izbegavanje bočnih efekata.

Počinjemo sa jednostavnim zadacima.

Čista funkcija je funkcija koja, kada dobije isti ulaz, uvek vraća isti rezultat i ne menja ništa van sebe.

Evo primera čiste funkcije:

```js
function add(a, b) {
  return a + b;
}
```

Ova funkcija je čista jer uvek vraća isti rezultat za isti ulaz i ništa ne menja izvan svog opsega. Bez obzira koliko puta ga koristite `add(2, 3)`, uvek će vratiti `5` i neće uticati na nijedan drugi deo vašeg programa.

Sada, ako uporedimo sa nečistom funkcijom:

```js
let total = 0;
function addToTotal(value) {
  total += value;
  return total;
}
```

Ova funkcija nije čista jer menja parametar izvan svog opsega (`total`). Rezultati `addToTotal(5)` će se razlikovati u zavisnosti od toga šta je bilo `total` pre pozivanja funkcije. Nepredvidivost ovoga može učiniti naš kod teškim za razumevanje i otklanjanje grešaka.

Ovo nas vodi do našeg sledećeg važnog koncepta, koji su sporedni efekti.

Sporedni efekat je bilo kakva promena koja se dešava u stanju programa kada se pozove funkcija. To može uključivati menjanje globalne varijable, pisanje u fajl ili pozivanje API.

Čisti enzimi nemaju sporedne efekte, ali nečisti enzimi imaju ove efekte.

Izbegavanje sporednih efekata je ključni princip funkcionalnog programiranja. Kada smanjimo sporedne efekte, naš kod postaje predvidljiviji i lakši za testiranje.

Kada pišemo kod, naša glavna namera bi trebalo da bude da funkcije rade samo jednu stvar i to veoma dobro, bez menjanja drugih delova našeg programa.

Ovde je primer funkcije sa sporednim efektom:

:::interactive_editor

```js
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet("Alice");
```

:::

Ova funkcija ima nuspojav jer interakcioniše sa spoljnim svetom tako što piše na konzolu.

U programu performansi, možemo prepisati ovako:

:::interactive_editor

```js
function greet(name) {
  return `Hello, ${name}!`;
}
console.log(greet("Alice"));
```

:::

Sada `greet` je čista funkcija koja samo vraća sekvencu karaktera, a mi sporedni efekat (pisanje u konzolu) obrađujemo odvojeno.

Performansni pristup nas podstiče da kreiramo naše aplikacije korišćenjem čistih funkcija, razdvajanjem našeg koda na male, predvidive komponente. Time, možemo izgraditi programe koji su laki za razumevanje, testiranje i održavanje.
