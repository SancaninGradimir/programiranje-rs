# --description--

Neka je $S_m = (x_1, x_2, \ldots, x_m)$ uređena m-torka pozitivnih realnih brojeva za koju važi $x_1 + x_2 + \cdots + x_m = m$, pri čemu je $P_m = x_1 \times {x_2}^2 \times \cdots \times {x_m}^m$ maksimalan.

Na primer, može se potvrditi da $[P_{10}] = 4112$ ( [ ] je funkcija dela kompletnog broja).

Pronađi $\sum {[P_m]}$ za $2 ≤ m ≤ 15$.
