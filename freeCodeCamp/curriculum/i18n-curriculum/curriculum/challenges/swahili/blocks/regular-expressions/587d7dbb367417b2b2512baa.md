# --description--

Reci da želiš/želite da uporediš reč koja se ponavlja više puta, kao što je prikazano ispod.

```js
let repeatStr = "row row row your boat";
```

Možete koristiti `/row row row/`, ali šta ako ne znate specifičnu ponavljajuću reč? <dfn>Grupe za hvatanje</dfn> mogu se koristiti za pronalaženje delova ponavljajuće sekvence slova.

Grupe za hvatanje se kreiraju postavljanjem obrasca regulare izraze koji je potrebno uhvatiti unutar običnih zagrada. U ovom slučaju, cilj je uhvatiti reč koja sadrži slova i brojeve, pa će grupa za hvatanje biti `\w+` okružena običnim zagradama: `/(\w+)/`.

Dio sekvence slova koji odgovara toj grupi pohranjen je u privremeni „parametar“, pristupačan unutar istog regulacionog izraza korišćenjem povratnog kosog crte i broja uhvatne grupe (primer `\1`). Uhvatne grupe se numerišu direktno prema poziciji njihovih otvarajućih zagrada (od leve ka desnoj), počevši od broja 1.

Primer ispod poredi reč koja se ponavlja tri puta, razdvojena razmakom:

```js
let repeatRegex = /(\w+) \1 \1/;
repeatRegex.test(repeatStr); // Returns true
repeatStr.match(repeatRegex); // Returns ["row row row", "row"]
```

Koristeći način `.match()` u nizu znakova za vraćanje grupe podataka sa odgovarajućim delovima i njihovim grupama za hvatanje.

Koristite grupe hvatanja u `reRegex` za poređenje sekvence slova koja sadrži isti broj koji se ponavlja tri puta uz razmak od jednog mesta.
