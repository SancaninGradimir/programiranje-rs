# --description--

Subleq je primer računara sa jednim skupom instrukcija (OISC).

Dobio je svoje ime zbog njegovih instrukcija koje su samo: **SU**bstract i **B**ranch ako je manji od ili jednak nuli.

Tvoj zadatak je da napraviš prevod koji liči na takvu mašinu.

Memorija mašine predstavlja niz celih brojeva sa znakom. Bilo koja odgovarajuća veličina reči je prihvatljiva, ali memorija mora moći da čuva i negativne i pozitivne brojeve.

Izvršavanje počinje tako što pokazivač instrukcije pokazuje na prvu reč, odnosno adresu 0. Dalji postupak je sledeći:

<ol>
<li>Neka su A, B i C vrednosti smeštene u tri uzastopne memorijske reči počev od adrese na koju pokazuje pokazivač instrukcije.</li>
<li>Pomeri pokazivač instrukcije unapred za 3 memorijske reči tako da pokazuje na adresu odmah posle C.</li>
<li>Ako je A jednako -1, pročitaj znak sa standardnog ulaza i sačuvaj njegov kod na adresi zadatoj vrednošću B. C se u tom slučaju ignoriše.</li>
<li>Ako je B jednako -1, vrednost sa adrese zadate vrednošću A tumači se kao kod znaka i odgovarajući znak se ispisuje na izlaz. C se ponovo ignoriše.</li>
<li>U suprotnom, i A i B se koriste kao adrese memorijskih lokacija. Broj sa adrese koju određuje A oduzima se od broja sa adrese koju određuje B (a rezultat se ponovo upisuje na adresu B). Ako je rezultat nula ili negativan, vrednost C postaje novi pokazivač instrukcije.</li>
  <li>Kama kiashiria cha maelekezo kinakuwa hasi, utekelezaji unasimama.</li>
</ol>

Ostale negativne adrese osim -1 mogu se tretirati isto kao -1 ili izazvati grešku, u zavisnosti od implementacije.

Tvoje rešenje treba da prihvati program koji će se izvršavati na mašini, odvojeno od ulaza koji se prosleđuje samom programu.

Taj program treba da bude izvorni Subleq mašinski kod – decimalni brojevi razdvojeni razmacima, bez simboličkih imena ili drugih dodataka asemblerskog nivoa, koji će biti učitani u memoriju počevši od adrese 0. Prikaži rezultat svog rešenja izvršavanjem ovog programa „Hello, world!“ bez ulaza. (Primer pretpostavlja ASCII ili njegovu varijantu, kao i bilo koji Latin-N ili Unicode skup znakova. Možeš ga prilagoditi drugom skupu znakova ako tvoja implementacija nije kompatibilna sa ASCII-jem.)

<pre>15 17 -1 17 -1 -1 16 1 -1 16 3 -1 15 15 0 0 -1 72 101 108 108 111 44 32 119 111 114 108 100 33 10 0</pre>

Koje odgovara nečemu kao ovo u jeziku uzorka:

<pre>start:
    zero, message, -1
    message, -1, -1
    neg1, start+1, -1
    neg1, start+3, -1
    zero, zero, start
zero: 0
neg1: -1
message: "Hello, world!\n\0"
</pre>

function Subleq(mem) {
  var out = '';
  var instructionPointer = 0;
  do {
    var a = mem[instructionPointer];
    var b = mem[instructionPointer + 1];
    if (a === -1) {
    } else if (b === -1) {
      out += String.fromCharCode(mem[a]);
    } else {
      mem[b] -= mem[a];
      if (mem[b] < 1) {
        instructionPointer = mem[instructionPointer + 2];
        continue;
      }
    }
    instructionPointer += 3;
  } while (instructionPointer >= 0);

  return out;
}
