# --description--

Express nudi različite ugrađene middleware funkcije koje pomažu u obavljanju uobičajenih zadataka, kao što su posluživanje statičkih datoteka, parsiranje JSON tela zahteva i drugo. Takođe možeš koristiti middleware trećih strana za proširenje funkcionalnosti.

**Ovde su neki primeri ugrađenog *middleware*-a**:**

1. `express.json()`: Centralni interni program koji analizira opterećenje podataka sa dolaznog JSON.

```js
app.use(express.json())
```

1. `express.static()`: Centralni program za obradu datoteka static kao slike, CSS, i datoteka JavaScript.

```js
app.use(express.static('public'))
```

1. `express.urlencoded()`: Centralna aplikacija koja analizira dolazne zahteve sa opterećenjem podataka URL-encoded, strukturu koja je obično poslana putem formulara od HTML. Omogućava da analizirani podaci budu dostupni na `req.body`.

```js
app.use(express.urlencoded({ extended: true }))
```

**Sada, evo nekoliko primera middleware programa trećih strana:**

1. `cors`: Popularan middleware treće strane za omogućavanje Cross-Origin Resource Sharing (CORS).

```bash
npm install cors
```

```js
const cors = require('cors')
app.use(cors())
```

1. `morgan`: Programu ya kati ya kurekodi maombi ya HTTP.

```bash
npm install morgan
```

```js
const morgan = require('morgan')
app.use(morgan('tiny'))
```

Sada, hajde da istražimo primer korišćenja middleware-a koji je ugrađen i razvijen od strane trećih strana zajedno:

```js
const express = require('express')
const cors = require('cors')
const morgan = require('morgan')
const app = express()

app.use(express.json()) // Built-in middleware to parse JSON
app.use(cors())       // Third-party middleware for CORS
app.use(morgan('tiny')) // Third-party middleware for logging

app.get('/', (req, res) => {
  res.send('Hello, World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* Ugrađeni middleware `express.json()` parsira dolazni JSON sadržaj zahteva.
* Middleware trećih strana `cors` omogućava zahteve sa drugih izvora, a `morgan` evidentira dolazne HTTP zahteve.

Važno je napomenuti da se middleware funkcije dodaju redosledom kojim se pozivaju pomoću `app.use()`.

Express pruža važan ugrađeni middleware za obradu uobičajenih zadataka kao što je skeniranje JSON i serviranje fajlova static. Takođe možete povećati funkcionalnost korišćenjem middleware-a treće strane za dodatne karakteristike.
