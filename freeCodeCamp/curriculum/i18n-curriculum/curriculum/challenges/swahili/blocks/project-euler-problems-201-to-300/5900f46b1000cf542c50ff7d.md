# --description--

Tafsiri $f(n)$ kama jumla ya faktoria za tarakimu za $n$. Kwa mfano, $f(342) = 3! + 4! + 2! = 32$.

Tafsiri $sf(n)$ kama jumla ya tarakimu za $f(n)$. Hivyo $sf(342) = 3 + 2 = 5$.

Definisati $g(i)$ kao pozitivan ceo broj $n$ takav da $sf(n) = i$. Iako je $sf(342)$ jednako 5, $sf(25)$ takođe je 5, a može se dokazati da je $g(5) = 25$.

Definišite $sg(i)$ kao zbir cifara od $g(i)$. Dakle, $sg(5) = 2 + 5 = 7$.

Osim toga, može se dokazati da je $g(20)$ jednako 267 i da je $\sum sg(i)$ za $1 ≤ i ≤ 20$ jednako 156.

Koja je vrednost $\sum sg(i)$ za $1 ≤ i ≤ 150?
