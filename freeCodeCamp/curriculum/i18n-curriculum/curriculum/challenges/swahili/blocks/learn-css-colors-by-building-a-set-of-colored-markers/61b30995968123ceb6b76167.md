# --description--

Prepoznajte da su granice senke oštre. Ovo je zato što postoji opcionalna vrednost za `blurRadius` za svojstvo `box-shadow`:

```css
box-shadow: offsetX offsetY blurRadius color;
```

Ako vrednost `blurRadius` nije uključena, koristi se vrednost `0` i stvara oštre granice. Kako vrednost `blurRadius` raste, tako raste i efekat zagađenja.

Prema zakonu `.green` od CSS, postavite svojstvo `box-shadow` sa vrednošću `5px` za `offsetX`, `5px` za `offsetY`, `5px` za `blurRadius`, i `green` za `color`.

# --hints--

Tvoja pravila za `.green` koja su CSS treba da imaju karakteristiku skraćivanja `box-shadow` i vrednost `5px 5px 5px green`.

```js
assert(new __helpers.CSSHelp(document).getStyle('.green')?.boxShadow === 'green 5px 5px 5px');
```
