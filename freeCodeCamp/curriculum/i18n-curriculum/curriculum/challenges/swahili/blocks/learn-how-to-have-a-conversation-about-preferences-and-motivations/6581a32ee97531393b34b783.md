# --description--

`Encourage` znači dati nekome pomoć ili samopouzdanje da nešto uradi. Na primer, kada tvoj prijatelj kaže, `You can solve this coding problem!` on te ohrabruje.

`Discourage` je suprotno tome; znači da se neko čini nesigurnim ili da ne voli nešto raditi. Ako taj prijatelj kaže, `That programming language is too hard for beginners`, može vas obeshrabriti za učenje toga.

`Discouraging` opisuje nešto što vas čini očajnim. Kao što je, `Finding a bug in your code after hours of work can be discouraging.`

`Encouraging` je mesto gde nešto daje hrabrost ili nadu. Na primer, `Getting your first program to run is very encouraging.`
