## Uvod u program transformacije energije

- **Definicija**: Program transformacije snage je algoritamska metoda koja rešava složan problem razbijanjem na manje, jednostavnije probleme i skladištenjem rezultata kako bi se izbeglo ponavljanje proračuna.
- **Ponavljajući manji problemi**: Isti mali problemi često se javljaju prilikom rešavanja velikih problema. Umesto da ponovo izračunavamo ove male probleme, skladištimo njihovo rešenje.
- **Optimalna struktura manjih problema**: Optimalno rešenje jednog problema poseduje optimalna rešenja njegovih manjih podproblema. To znači da možemo izgraditi optimalno rešenje kombinovanjem optimalnih rešenja manjih delova.

## Rešenje za program pretvaranja snage

- **Memoization (Top-down pristup)**: Memoizacija čuva rezultate skupih poziva funkcija i vraća sačuvani rezultat kada se ponovi isti ulaz.

```js
function climbStairsMemo(n, memo = {}) {
  // If already calculated, return cached value
  if (memo[n] !== undefined) {
    return memo[n];
  }

  // Base cases: 1 way for 1 step, 2 ways for 2 steps
  if (n <= 2) {
    return n;
  }

  // Calculate once and store in memo for future use
  memo[n] =
    climbStairsMemo(n - 1, memo) +
    climbStairsMemo(n - 2, memo);

  return memo[n];
}
```

- **Tabulacija (Metoda od dna ka vrhu)**: Tabulacija gradi rešenje od dna ka vrhu, popunjavajući tabelu sa rešenjima za manje izazove.

```js
function climbStairsTabulation(n) {
  if (n <= 2) {
    return n;
  }

  // Create an array to store solutions from 0 to n
  const dp = new Array(n + 1).fill(0);

  // Base cases
  dp[1] = 1; // 1 way to reach step 1
  dp[2] = 2; // 2 ways to reach step 2

  // Build up the solution iteratively
  for (let i = 3; i <= n; i++) {
    // Ways to reach step i = ways to reach (i-1) + ways to reach (i-2)
    dp[i] = dp[i - 1] + dp[i - 2];
  }

  return dp[n];
}
```

## Stvarna primena programa za konverziju snage

- **Optimizacija putanje podataka**: GPS sistemi koriste algoritme optimizacije da bi pronašli najkraći put između lokacija.
- **Obrada teksta**: Funkcije za proveru pravopisa i dovršavanje teksta često se oslanjaju na algoritme optimizacije kako bi izračunale rastojanje (edit distance) između reči.
- **Kreiranje finansijskih modela**: Investicione strategije i optimizacija portfolija često koriste metode algoritama optimizacije.
- **Alokacija resursa**: Knapsak problem i njegove varijante pojavljuju se u planiranju rasporeda, budžetiranju i upravljanju resursima.

## Kada koristiti program za transformaciju snage

Trebalo bi vas obratiti pažnju na korišćenje programa za transformaciju snage u sledećim uslovima:

- Izazov se može razbiti na ponavljajuće manje izazove.
- Izazov ima strukturu manjih izazova.
- Standardno rešenje bi zahtevalo rekurzivne proračune.
- Potrebno je poboljšati u smislu vremenske složenosti na štetu prostorne složenosti.
