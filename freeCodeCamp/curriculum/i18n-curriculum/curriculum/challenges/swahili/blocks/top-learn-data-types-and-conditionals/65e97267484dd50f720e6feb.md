# --description--

Znate da JavaScript ima različite tipove podataka, uključujući nizove karaktera, brojeve i vrednosti istina ili laž. Možete koristiti kriterijume za poređenje da uporedite ove tipove podataka i donesete odluke na osnovu rezultata.

Sledeći kriterijumi za poređenje su dostupni u JavaScript:

| Operator | Opis | Primer |
|----------|-----------------------------------|--------------|
| ==       | Jednako (poređenje vrednosti)     | x == y       |
| !=       | Nejednako                         | x != y       |
| >        | Veće od                           | x > y        |
| <        | Manje od                          | x < y        |
| >=       | Veće ili jednako                  | x >= y       |
| <=       | Manje ili jednako                 | x <= y       |

Da biste koristili poređenje u opsegu vrednosti varijable, možete koristiti kriterijum `==`. Primer:

```javascript
let x = 5;
let y = 5;

let result = x == y;

console.log(result); // true
```
