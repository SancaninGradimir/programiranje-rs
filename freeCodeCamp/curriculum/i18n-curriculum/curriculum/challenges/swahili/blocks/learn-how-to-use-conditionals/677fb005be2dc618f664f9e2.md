# --explanation--

`Data backup` Radi se o kreiranju kopija važnih podataka kako bi se osiguralo da mogu biti vraćeni u slučaju neispravnosti sistema, gubitka ili korumpiranja podataka:

- `data` je digitalni podaci sačuvani na uređajima.

- `backup` je proces kopiranja ovih podataka radi sigurnosti.

Zajednički, `data backup` je važan za zaštitu važnih fajlova i smanjenje rizika u IT okruženju. Na primer:

`The IT team performs a data backup every night.` - Tim skladišti sve podatke svakodnevno kako bi osigurao njegovu sigurnost.
