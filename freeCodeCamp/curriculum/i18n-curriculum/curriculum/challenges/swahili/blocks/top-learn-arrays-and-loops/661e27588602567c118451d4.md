# --description--

Najčešće metode korišćene sa nizovima podataka su metode `splice()` i `slice()`. Metoda `splice()` modifikuje sadržaj u nizu podataka uklanjanjem ili promenom elementa u tom nizu. Metoda `slice()` vraća kopiju sekcije niza podataka u novi objekat niza podataka, izabran od `begin` do `end` (`end` nije uključeno). Originalni niz neće biti promenjen.

Na primer, da uklonite drugi element iz niza `characters`, možete koristiti sledeći kod:

```javascript
const characters = ['Harry', 'Ron', 'Hermione'];
characters.splice(1, 1);
console.log(characters); // Output: ['Harry', 'Hermione']
```

Funkcija gore uklanja drugi element iz niza `characters`. Metoda `splice()` prima dva argumenta: referencu elementa za brisanje i broj elemenata za brisanje.

Da biste kreirali novi red sa drugim elementom iz reda `character`, možete koristiti sledeći kod:

```javascript
const characters = ['Harry', 'Ron', 'Hermione'];
const newCharacters = characters.slice(1, 2);
console.log(newCharacters); // Output: ['Ron']
```

Taj kod iznad stvara novi niz ``newCharacters`` koji ima drugi element iz niza ``characters``. Funkcija ``slice()`` prima dva argumenta: model početnog elementa rezanja i model završnog elementa rezanja (isključivo).
