# --interactive--

Kada radiš sa CSS-om za stilizovanje veb stranice, jedan od najčešćih načina za definisanje boja jeste korišćenje heksadecimalnih vrednosti. Heksadecimalne vrednosti boja, ili hex kodovi, predstavljaju skraćen način za predstavljanje boja u RGB modelu. U ovoj lekciji naučićeš šta su hex kodovi, kako funkcionišu u CSS-u i kada ih treba koristiti.

Hex kod (skraćeno za heksadecimalni kod) je sekvenca od šest slova koja se koristi za predstavljanje boja u formatu RGB. "Hex" označava sistem broja sa bazom 16, koji koristi cifre `0` do `9` i slova `A` do `F`.

U kontekstu boja, heksadecimalni kodovi definišu količinu crvenog, zelenog i plavog (RGB) koja čini određenu boju. Svaki heksadecimalni kod počinje sa znakom haša (`#`), praćenih šest slovom koja određuju intenzitet crvenog, zelenog i plavog u toj boji.

Na primer, hex kod za crnu boju je `#000000`, a za belu `#FFFFFF`. Prva dva znaka predstavljaju vrednost crvene boje, sledeća dva zelene, a poslednja dva plave.

U CSS, možete koristiti boje za elemente koristeći heksadecimalne kodove. Ovo je osnovna sintaksa:

```css
element {
  color: #RRGGBB;
}
```

Ovde, `#RRGGBB` je reprezentacija stvarnog heksadecimalnog koda. Podešavanja za `RR`, `GG` i `BB` svaki predstavljaju crvenu, zelenu i plavu vrednost, respektivno.

Ove postavke mogu biti između `00` (najniža snaga) do `FF` (najviša snaga). Kako broj raste, boja će biti dominantnija u konačnoj mešavini. Na primer, uzmimo heksadecimalne boje sa nekoliko elemenata:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h1>This is a heading.</h1>
<p>This is a paragraph.</p>
```

```css
h1 {
  color: #FF5733; /* A reddish-orange color */
}
p {
  background-color: #4CAF50; /* A shade of green */
}
```

:::

U prvom primeru, `#FF5733` predstavlja narandžastu nijansu sa izraženom crvenom komponentom. Evo objašnjenja:

- `FF` ni thamani ya juu kabisa ya nyekundu (imejaa kabisa).
- `57` ni kiasi cha wastani cha kijani.
- `33` ni kiasi kidogo cha buluu.

U drugom primeru, `#4CAF50` predstavlja nijansu zelene boje:

- `4C` je minimalna crvena vrednost.
- `AF` je maksimalna zelena vrednost.
- `50` je srednja plava vrednost.

Hex kodovi su popularni jer su precizni i jednostavni za upotrebu. Omogućavaju vam da prilagodite boje sa preciznošću, pa je odličan za dizajne gde je konzistentnost važna.

Mnogi dizajn programi, kao što su Adobe Photoshop ili Figma, pružaju heksadecimalne kodove za boje koje odaberete, što olakšava kopiranje i lijepljenje ovih vrijednosti direktno u vaše fajlove CSS.

U web razvoju, heks kodovi se često koriste za definisanje boja teksta, pozadine, ivica i drugih vizuelnih elemenata. Primer:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example</button>
```

```css
body {
  background-color: #F0F0F0; /* Light gray background */
}

button {
  background-color: #007BFF; /* A shade of blue */
  color: #FFFFFF; /* White text */
}
```

:::

Ovo osigurava da vaše boje ostanu konzistentne na celom vašem veb-sajtu ili aplikaciji.

U nekim slučajevima, možete videti heks kodove napisane u skraćenem formatu, koristeći tri karaktera umesto šest. To je moguće kada su oba karaktera u svakoj pari boje jednaka. Na primer, `#FF5733` ne može se skratiti, ali `#FFFFFF` (belo) se može napisati kao `#FFF`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<h2>This is a heading.</h2>
```

```css
body {
  background-color: #000;
}

h2 {
  color: #FFF; /* White in shorthand */
}
```

:::

Kratki kod `#FFF` je jednak `#FFFFFF`. Slično tome, `#000` će predstavljati crnu koja je `#000000` za kompletan format.

Hex kodovi su poznati po svojoj preciznosti i konzistentnosti, omogućavajući preko 16 miliona boja u svim pregledačima. Najbolje je kada vam trebaju specifične boje brenda, kada želite da kontrolišete tačan nijansu koji se ne može lako definisati imenovanom bojom ili HSL, ili kada kopirate boje iz programa za dizajn koji podrazumevano pružaju heks kodove.
