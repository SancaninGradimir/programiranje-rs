# --description--

U Hispaniji, broj 20 je `veinte`.

Kada generišete brojeve od 21 do 29, reči se malo menjaju. `Veinte` menja se u `veinti-` (uklanja se poslednji deo `e`), zatim se spaja sa sledećim brojem. Primer:

- `Veinti` + `dos` (mbili) = `veintidós` (22)

- `Veinti` + `cinco` (pet) = `veinticinco` (25)

- `Veinti` + `ocho` (nane) = `veintiocho` (28)
