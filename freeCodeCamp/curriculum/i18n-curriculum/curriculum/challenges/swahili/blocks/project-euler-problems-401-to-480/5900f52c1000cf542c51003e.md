# --description--

Za svaki ceo broj $n > 1$, porodica funkcija $f_{n, a, b}$ je definisana na sledeći način:

$f_{n, a, b}(x) ≡ ax + b\bmod n$ za $a, b, x$ celobrojne i $0 \lt a \lt n$, $0 \le b \lt n$, $0 \le x \lt n$.

Definišemo $f_{n, a, b}$ kao ako je $f_{n, a, b}(f_{n, a, b}(x)) \equiv f_{n, a, b}(x)\bmod n$ za svako $0 \le x < n$.

Neka $R(n)$ bude broj povratnih vrednosti za $n$.

$F(N) = \displaystyle\sum_{n = 2}^N R(n)$.

$F({10}^7) ≡ 638\\,042\\,271\bmod 1\\,000\\,000\\,007$.

Pronađi $F({10}^{14})$. Daj svoj odgovor modulo $1\\,000\\,000\\,007$.
