# --description--

Neka je $n$ pozitivan ceo broj. Razmotrite pozicije gde:

- Postoji $n$ ne-praznih elemenata.
- Svaki element ima veličinu manju od $2^n$.
- Ne postoji dva elementa iste veličine.

Neka $W(n)$ bude broj pobedničkih pozicija nima koje zadovoljavaju navedene uslove (pozicija je pobednička ako prvi igrač ima pobednički strategija).

Na primer, $W(1) = 1$, $W(2) = 6$, $W(3) = 168$, $W(5) = 19\\,764\\,360$ na $W(100)\bmod 1\\,000\\,000\\,007 = 384\\,777\\,056$.

Pronađi $W(10\\,000\\,000)\bmod 1\\,000\\,000\\,007$.
