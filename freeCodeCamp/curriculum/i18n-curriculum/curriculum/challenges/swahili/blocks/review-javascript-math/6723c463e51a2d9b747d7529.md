## Rad sa numeričkim tipom podataka

- **Opis**: Tip od `Number` od JavaScript uključuje pune brojeve, zaokružene brojeve, `Infinity` i `NaN`. Zaokruženi brojevi su brojevi sa decimalnom tačkom. `Infinity` je broj veći od bilo kog drugog broja, dok `-Infinity` je broj manji od bilo kog drugog broja. `NaN` (`Not a Number`) predstavlja nevažeću numeričku vrednost kao niz slova `"Jessica"`.

## Osnovne matematičke operacije

- **Operator za sabiranje**: Ovaj operator (`+`) koristi se za izračunavanje zbira dve ili više brojeva.
- **Operator za oduzimanje**: Ovaj operator (`-`) koristi se za izračunavanje razlike između dva broja.
- **Operator za množenje**: Ovaj operator (`*`) koristi se za izračunavanje proizvoda dve ili više brojeva.
- **Operator za deljenje**: Ovaj operator (`/`) koristi se za izračunavanje količnika između dva broja.
- **Deljenje sa nulom**: Ako pokušate deliti sa nulom, JavaScript će vratiti `Infinity`.
- **Operator za ostatak**: Ovaj operator (`%`) vraća ostatak deljenja.
- **Operator za potencu**: Ovaj operator (`**`) podiže jedan broj na stepen drugog.

## Matematika i sekvenca slova

- **Opis**: Kada koristite operator `+` sa brojevima i niskom, JavaScript će broj pretvoriti u nisku i spojiti te dve vrednosti. Kada koristite operatore `-`, `*` ili `/` sa niskom i brojevima, JavaScript će pokušati da nisku pretvori u broj, a rezultat će biti broj. Za `null` i `undefined`, JavaScript tretira `null` kao `0`, a `undefined` kao `NaN` u aritmetičkim operacijama.

:::interactive_editor

```js
const result = 5 + '10';

console.log(result); // "510"
console.log(typeof result); // string

const subtractionResult = '10' - 5;
console.log(subtractionResult); // 5
console.log(typeof subtractionResult); // number

const multiplicationResult = '10' * 2;
console.log(multiplicationResult); // 20
console.log(typeof multiplicationResult); // number

const divisionResult = '20' / 2;
console.log(divisionResult); // 10
console.log(typeof divisionResult); // number

const result1 = null + 5;
console.log(result1); // 5
console.log(typeof result1); // number

const result2 = undefined + 5;
console.log(result2); // NaN
console.log(typeof result2); // number
```

:::

## Prioritet operete

- **Maelezo**: Prioritet operatora određuje red izvršavanja operacija u matematičkom izrazu. Operatori višeg prioriteta obrađuju se pre onih nižeg prioriteta. Vrednosti unutar zagrada obrađuju se prvo, a množenje/deljenje ima viši prioritet od sabiranja/oduzimanja. Ako operateri imaju isti prioritet, onda JavaScript će koristiti pravilo asocijativnosti.

:::interactive_editor

```js
const result = (2 + 3) * 4;

console.log(result); // 20

const result2 = 10 - 2 + 3;

console.log(result2); // 11

const result3 = 2 ** 3 ** 2;

console.log(result3); // 512
```

:::

- **Opis**: Direktorija izbora govori nam smer izvršavanja izraza kada postoji više operatora istog tipa. Ovo određuje da li će izraz biti parsiran od leve ka desnoj strani (`left-associative`) ili od desne ka levoj strani (`right-associative`). Na primer, operator snage takođe je u smeru od desne ka levoj strani:

:::interactive_editor

```js
const result4 = 5 ** 4 ** 1; 

console.log(result4); // 625
```

:::

## Sabiranje i oduzimanje

- **Operator za dodavanje**: Ovaj operator se koristi za povećanje vrednosti za jedan. Napomena o prefiksu `++num` prvo povećava vrednost parametra, a zatim vraća novu vrednost. Napomena o sufiksu `num++` prvo vraća trenutnu vrednost parametra, a zatim je povećava.

:::interactive_editor

```js
let x = 5;

console.log(++x); // 6
console.log(x); // 6


let y = 5;

console.log(y++); // 5
console.log(y); // 6
```

:::

- **Smanjujuća operacija**: Ova operacija se koristi za smanjenje vrednosti za jedinicu. Napomena o prefiksu i napomena o sufiksu funkcionišu isto kao kod povećavajuće operacije.

:::interactive_editor

```js
let num = 5;

console.log(--num); // 4
console.log(num--); // 4
console.log(num); // 3
```

:::

## Operacija mešavine vrednosnih delova

- **Opereta ya mgawo wa kuongeza (`+=`)**: Ovaj operator vrši dodavanje na vrednost i postavlja rezultat u varijablu.
- **Opereta ya mgawo wa kutoa (`-=`)**: Ovaj operator vrši oduzimanje od vrednosti i postavlja rezultat u varijablu.
- **Opereta ya mgawo wa kuzidisha (`*=`)**: Ovaj operator vrši množenje vrednosti i postavlja rezultat u varijablu.
- **Opereta ya mgawo wa kugawanya (`/=`)**: Ovaj operator vrši deljenje vrednosti i postavlja rezultat u varijablu.
- **Opereta ya mgawo wa mabaki (`%=`)**: Ovaj operator deli varijablu sa navedenim brojem i postavlja ostatak u varijablu.
- **Opereta ya mgawo wa nguvu (`**=`)**: Ovaj operator podiže varijablu na stepen navedenog broja i postavlja rezultat u varijablu.

## Vrednost istine ili laži i ravnoteže

- **Opis vrednosti tačno ili netačno**: Vrednost tačno ili netačno je tip podataka koji može imati samo dve vrednosti: `true` ili `false`.
- **Operater jednakosti (`==`)**: Ovaj operater se koristi za prisiljavanje tipa pre proveravanja da li je vrednost jednaka.

:::interactive_editor

```js
console.log(5 == '5'); // true
```

:::

- **Operater stroge jednakosti (`===`)**: Ovaj operater ne prisiljava tip i proverava da li su i tip i vrednost jednaki.

:::interactive_editor

```js
console.log(5 === '5'); // false
```

:::

- **Operater nejednakosti (`!=`)**: Ovaj operator se koristi za prisiljavanje tipa pre nego što proveri da li je vrednost različita.
- **Operater stroge nejednakosti (`!==`)**: Ovaj operator ne koristi prisiljavanje tipa i proverava da li su i tip i vrednost različiti.

## Opereta za poređenje

- **Operater veći od (`>`)**: Ovaj operator proverava da li je vrednost sa leve strane veća od vrednosti sa desne strane.
- **Operater veći ili jednak (`>=`)**: Ovaj operator proverava da li je vrednost sa leve strane veća ili jednaka vrednosti sa desne strane.
- **Operater manji od (`<`)**: Ovaj operator proverava da li je vrednost sa leve strane manja od vrednosti sa desne strane.
- **Operater manji ili jednak (`<=`)**: Ovaj operator proverava da li je vrednost sa leve strane manja ili jednaka vrednosti sa desne strane.

## Opereta o jednom pandi

- **Operacija za dodavanje sa jedne strane**: Ova operacija menja svoj parametar u broj. Ako je parametar već broj, ostaje nepromenjen.

:::interactive_editor

```js
const str = '42';
const num = +str;

console.log(num); // 42
console.log(typeof num); // number
```

:::

- **Unarni operator negacije (`-`)**: Ovaj operator negira parametar.

:::interactive_editor

```js
const num = 4;
console.log(-num); // -4
```

:::

- **Negacioni logički operator (`!`)**: Ovaj operator menja vrednost tačno ili netačno svog parametra. Dakle, ako je parametar `true`, postaje `false`, a ako je `false`, postaje `true`.

## Operacije za bitwise

- **Bitovni AND operater (`&`)**: Ovaj operater vraća 1 u svakoj poziciji bita gde su odgovarajući biti svih operanata 1.
- **Bitovni AND operater dodeljivanja (`&=`)**: Ovaj operater izvršava operaciju `bitwise AND` sa navedenim brojem i skladišti rezultat u varijablu.
- **Bitovni OR operater (`|`)**: Ovaj operater vraća 1 u svakoj poziciji bita gde su odgovarajući biti barem jednog od operanata 1.
- **Bitovni OR operater dodeljivanja (`|=`)**: Ovaj operater izvršava operaciju `bitwise OR` sa navedenim brojem i skladišti rezultat u varijablu.
- **Bitovni XOR operater (`^`)**: Ovaj operater vraća 1 u svakoj poziciji bita gde su odgovarajući biti barem jedan, ali ne i oba, postavljeni na 1.
- **Bitovni NOT operater (`~`)**: Ovaj operater menja binarno predstavljanje broja.
- **Operater pomeranja ulevo (`<<`)**: Ovaj operater pomera sve bite ulevo za navedeni broj pozicija.
- **Operater pomeranja udesno (`>>`)**: Ovaj operater pomera sve bite udesno.

## Uslovna izjava, istinita vrednost, lažna vrednost i ternarni operator

- **`if/else if/else`**: Funkcija `if` uzima uslov i izvršava deo koda ako je taj uslov `truthy`. Ako je uslov `false`, onda ukazuje na deo `else if`. Ako nema uslova među njima sa `true`, onda rešava klauzulu `else`. Vrednosti `Truthy` su bilo koja vrednost koja rezultuje `true` kada se evaluira u kontekstu tačne ili netačne vrednosti kao klauzula/funkcija `if`. Vrednosti `Falsy` su vrednosti koje se evaluiraju kao `false` u kontekstu tačne ili netačne vrednosti.

:::interactive_editor

```js
const score = 87;

if (score >= 90) {
 console.log('You got an A'); 
} else if (score >= 80) {
 console.log('You got a B'); // You got a B
} else if (score >= 70) {
 console.log('You got a C');
} else {
 console.log('You failed! You need to study more!');
}
```

:::

- **Ternarni operater**: Ovaj operater se često koristi kao kratak način za pisanje delova `if else`.

:::interactive_editor

```js
const temperature = 30;
const weather = temperature > 25 ? 'sunny' : 'cool';

console.log(`It's a ${weather} day!`); // It's a sunny day!
```

:::

## Opereta za Kimantiku za dve strane

- **Logički operator AND (`&&`)**: Ovaj operator proverava da li su svi kriterijumi `true`. Ako su sve `true`, onda vraća drugu vrednost. Ako je bilo koji kriterijum `falsy`, onda vraća vrednost `falsy`. Ako su svi kriterijumi `falsy`, vraća prvu vrednost od `falsy`.

:::interactive_editor

```js
const result = true && 'hello';

console.log(result); // hello
```

:::

- **Logički OR operator (`||`)**: Ovaj operator proverava da li je barem jedna od parametara istinite vrednost.
- **Operator za spajanje beznačajnih vrednosti (`??`)**: Ovaj operator će vratiti vrednost samo ako je prva `null` ili `undefined`.

:::interactive_editor

```js
const userSettings = {
 theme: null,
 volume: 0,
 notifications: false,
};

let theme = userSettings.theme ?? 'light';
console.log(theme); // light
```

:::

## Stvar od `Math`

- **Njia ya `Math.random()`**: Ova funkcija generiše decimalan broj u opsegu između 0 (uključeno) i 1 (isključeno). To znači da mogući rezultati mogu biti 0, ali nikada neće dostići 1.
- **Njia ya `Math.max()`**: Ova funkcija uzima skup brojeva i vraća najveću vrednost.
- **Njia ya `Math.min()`**: Ova funkcija uzima skup brojeva i vraća najmanju vrednost.
- **Njia ya `Math.ceil()`**: Ova funkcija zaokružuje vrednost na sledeći ceo broj.
- **Njia ya `Math.floor()`**: Ova funkcija zaokružuje vrednost na prethodni ceo broj.
- **Njia ya `Math.round()`**: Ova funkcija zaokružuje vrednost na najbliži ceo broj.

:::interactive_editor

```js
console.log(Math.round(2.3)); // 2
console.log(Math.round(4.5)); // 5
console.log(Math.round(4.8)); // 5
```

:::

- **Njia ya `Math.trunc()`**: Ova funkcija uklanja decimalni deo broja, vraćajući samo ceo deo broja, bez zaokruživanja.
- **Njia ya `Math.sqrt()`**: Ova funkcija će vratiti kvadratni koren broja.
- **Njia ya `Math.cbrt()`**: Ova funkcija će vratiti kubični koren broja.
- **Njia ya `Math.abs()`**: Ova funkcija će vratiti apsolutnu vrednost broja.
- **Njia ya `Math.pow()`**: Ova funkcija uzima dva broja i podiže prvi na stepen drugog.

## Standardni načini za brojeve

- **`isNaN()`**: `NaN` označava "Nije broj". To je posebna vrednost koja predstavlja nepredstavljivi ili nedefinisani rezultat broja. Funkcijsko svojstvo `isNaN()` se koristi za određivanje da li je vrednost `NaN` ili ne. `Number.isNaN()` pruža pouzdaniji način proveravanja vrednosti `NaN`, posebno u slučajevima kada konverzija tipa može izazvati nepouzdane rezultate i globalna funkcija `isNaN()`.

:::interactive_editor

```js
console.log(isNaN(NaN));       // true
console.log(isNaN(undefined)); // true
console.log(isNaN({}));        // true

console.log(isNaN(true));      // false
console.log(isNaN(null));      // false
console.log(isNaN(37));        // false


console.log(Number.isNaN(NaN));        // true
console.log(Number.isNaN(Number.NaN)); // true
console.log(Number.isNaN(0 / 0));      // true

console.log(Number.isNaN("NaN"));      // false
console.log(Number.isNaN(undefined));  // false
```

:::

- **Njia ya `parseFloat()`**: Ova metoda analizira argument niz znakova i vraća kružni broj. Kreirana je da izvuče broj sa početka niza znakova, čak i ako niz sadrži ne-numeričke karaktere kasnije.
- **Njia ya `parseInt()`**: Ova metoda analizira argument niz znakova i vraća pun broj. `parseInt()` prestaje sa parsiranjem kada naiđe na ne-numerički karakter. Za kružne brojeve, vraća samo deo punog broja. Ako ne može pronaći validan pun broj na početku niza, vraća `NaN`.
- **Njia ya `toFixed()`**: Ova metoda se poziva na broju i uzima jedan opcionalni argument, koji je broj cifara koje treba videti posle decimalne tačke. Vraća reprezentaciju broja u formatu niza znakova sa navedenim brojem decimalnih mesta.
