# --description--

`There is` i `there are` se koriste za pokazivanje postojanja ili prisutnosti nečega. `There is` se koristi sa jedinstvenim ili nebrojivim imenicama, dok `there are` se koristi sa množinovnim imenicama.

Primeri:

- `There is a book on the table.` (Jednina imenice)

- `There is some water in the glass.` (Abstraktna imenica)

- `There are three apples in the basket.` (Naziv krila)
