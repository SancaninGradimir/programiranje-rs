# --description--

Neka $a_k$, $b_k$, i $c_k$ predstavljaju tri rešenja (realni ili kompleksni brojevi) za matematički izraz $\frac{1}{x} = {\left(\frac{k}{x} \right)}^2 (k + x^2) - kx$.

Na primer, za $k = 5$, primećujemo da $\\{a_5, b_5, c_5\\}$ je približno $\\{5.727244, -0.363622 + 2.057397i, -0.363622 - 2.057397i\\}$.

Za $S(n) = \displaystyle\sum_{p = 1}^n \sum_{k = 1}^n {(a_k + b_k)}^p {(b_k + c_k)}^p {(c_k + a_k)}^p$ za sve cele brojeve $p$, $k$ takve da $1 ≤ p, k ≤ n$.

Što je zanimljivo je da $S(n)$ uvijek je cijeli broj. Na primjer, $S(4) = 51\\,160$.

Pronađi $S({10}^6) \text{ modulo } 1\\,000\\,000\\,007$.
