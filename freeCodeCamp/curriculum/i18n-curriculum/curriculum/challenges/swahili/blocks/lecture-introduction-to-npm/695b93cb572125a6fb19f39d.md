# --description--

Pravilno upravljanje zavisnostima – i znati kako instalirati specifičnu verziju – ključno je za održavanje stabilnosti vašeg projekta. Ovo vam pomaže da izbegnete neočekivane greške i daje vam veću kontrolu kada se izdaju nove verzije.

Da biste instalirali specifičnu verziju paketa, koristite ovaj format:

```bash
npm install package@version-number
```

Na primer, ako je tvom projektu potrebna starija verzija Express-a, recimo `4.21.2`, možeš je instalirati ovako:

```bash
npm install express@4.21.2
```

Ako ne navedeš broj verzije, npm će automatski instalirati najnoviju dostupnu verziju.

Ako nisi siguran koje su verzije dostupne, možeš ih proveriti ovom komandom:

```bash
npm view package-name versions
```

Na primer:

```bash
npm view express versions
```

Da bi ažurirao pakete, možeš ih ažurirati pojedinačno ili sve odjednom. Pre toga je dobro da prvo vidiš koji su zastareli:

```bash
npm outdated
```

Ovo će prikazati koje zavisnosti imaju dostupne novije verzije, a rezultat će izgledati otprilike ovako:

```bash
user@Kolade ~/Desktop/fCC/script-code/node/npm/install-spec-version  % npm outdated
Package  Current  Wanted  Latest  Location              Depended by
chalk      5.5.0   5.5.0   5.6.2  node_modules/chalk    install-spec-version
cleave     1.0.1   1.0.1   1.0.2  node_modules/cleave   install-spec-version
express   4.21.2  4.21.2   5.2.1  node_modules/express  install-spec-version
```

Nakon što proveriš zastarele zavisnosti, možeš početi sa njihovim ažuriranjem.

Da bi ažurirao jednu zavisnost, koristi sledeću komandu:

```bash
npm update package-name
```

Ovo ažurira paket u okviru opsega verzija dozvoljenog u tvojoj datoteci `package.json`. Na primer, za ažuriranje Express-a možeš koristiti sledeću komandu:

```bash
npm update express
```

Ako želiš da zanemariš opseg verzija i instaliraš najnoviju verziju paketa, koristi sledeću komandu:

```bash
npm install package-name@latest
```

Da biste ažurirali sve zavisnosti koje spadaju u raspon dozvoljenih verzija, izvršite:

```bash
npm update
```

Međutim, zapamtite da `npm update` neće objaviti paket dok nova glavna verzija kao `package.json` ne dozvoli. Glavne verzije često imaju *breaking changes*, pa npm neće koristiti direktno.

Ako želite najnoviju glavnu verziju, morate pokrenuti:

```bash
npm install package-name@latest
```

Ovo prisiljava npm da instalira potpuno novu verziju, bez obzira na opseg verzija u `package.json`.
