# --description--

Kako biste rešili ako imate dve grupe elemenata koje dele neke deklaracije stila?

```css
.read {
  color: white;
  background-color: black;
  /* several unique declarations */
}

.unread {
  color: white;
  background-color: black;
  /* several unique declarations */
}
```

Naš selektor za `.read` i `.unread` deli deklaracije za `color: white;` i `background-color: black;`, ali inače imaju sopstvene jedinstvene deklaracije. Da bismo smanjili ponavljanje, možete spojiti ova dva selektora kao listu razdvojenu zarezima:

```css
.read,
.unread {
  color: white;
  background-color: black;
}

.read {
  /* several unique declarations */
}

.unread {
  /* several unique declarations */
}
```

Oba primera iznad (bilo da su povezani ili ne) će dati isti rezultat, ali drugi primer smanjuje ponavljanje deklaracije i olakšava uređivanje `color` ili `background-color` za obe klase odjednom.
