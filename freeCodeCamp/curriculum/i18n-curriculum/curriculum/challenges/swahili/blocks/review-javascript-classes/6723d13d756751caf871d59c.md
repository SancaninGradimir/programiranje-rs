## Osnove rada sa razredima

- **Definicija**: Klase u JavaScript se koriste za definisanje šema za kreiranje objekata i povezivanje podataka. Klase uključuju komponentu koja je posebna metoda nazvana konstruktor kada se kreira novi objekat iz klase. Komponenta se koristi za inicijalizovanje svojstava tog objekta. Glavna reč `this` se ovde koristi za referenciranje instance klase. Ispod komponente, možete imati nešto što se zove metoda. Metoda je funkcija definisana unutar klase koja izvodi akcije ili operacije na podacima ili stanju klase. One se koriste za definisanje ponašanja koja klase instance mogu da izvrše.

:::interactive_editor

```js
class Dog {
  constructor(name) {
    this.name = name;
  }

  bark() {
    console.log(`${this.name} says woof!`);
  }
}

const dog = new Dog("Gino");
console.log(dog.name);  // Gino
```

:::

Da biste kreirali novi primer klase, koristićete glavni termin `new` praćen imenom klase:

```js
const dog = new Dog("Gino");
```

Takođe možete kreirati klase kao izraz klase. Ovo je slučaj kada klasa nema ime i dobije vrednost za određeni parametar.

:::interactive_editor

```js
const Dog = class {
  constructor(name) {
    this.name = name;
  }

  bark() {
    console.log(`${this.name} says woof!`);
  }
};

const dog = new Dog("Gino");
console.log(dog.name);  // Gino
dog.bark();  // Gino says woof!
```

:::

## Nasleđe razreda

- **Definicija**: U programiranju, nasleđivanje vam omogućava da definišete klase koje nasleđuju atribute i metode od roditeljske klase. Ovo povećava ponovna upotreba koda i uspostavlja hijerarhijski odnos između klasa. Roditeljska klasa je klasa koja se koristi kao šablon za druge klase. Ona definiše atribute i metode koje nasleđuju druge klase. Deca klasa je klasa koja nasleđuje atribute i metode neke druge klase. Deca klase takođe mogu proširiti funkcionalnost svoje roditeljske klase dodavanjem novih atributa i metoda. U JavaScript, koristimo ključnu reč `extends` za implementaciju nasleđivanja. Ova ključna reč ukazuje da je klasa podklasa neke druge klase.

:::interactive_editor

```js
class Vehicle {
  constructor(brand, year) {
    this.brand = brand;
    this.year = year;
  }
}

class Car extends Vehicle {
  honk() {
    console.log("Honk! Honk!");
  }
}

const myCar = new Car("freeCodeCamp Motors", 2019);
console.log(myCar.brand);  // freeCodeCamp Motors
console.log(myCar.year);  // 2019
myCar.honk();  // Honk! Honk!
```

:::

Ključna reč za `super` se koristi za pristup putevima, komponentama i delovima roditeljske klase.

:::interactive_editor

```js
class Vehicle {
  constructor(brand, year) {
    this.brand = brand;
    this.year = year;
  }
}

class Car extends Vehicle {
  constructor(brand, year, numDoors) {
    super(brand, year);
    this.numDoors = numDoors;
  }
}

const myCar = new Car("freeCodeCamp Motors", 2019, 4);
console.log(myCar.brand);  // freeCodeCamp Motors
console.log(myCar.year);  // 2019
console.log(myCar.numDoors);  // 4
```

:::

## Rad sa statičkim metodama i statičkim svojstvima

- **Njia za static**: Ove metode se često koriste za servisne funkcije koje ne zahtevaju pristup specifičnom stanju objekta. Definisan/Definisane su unutar klasa radi enkapsulacije povezanih funkcionalnosti.

:::interactive_editor

```js
class Movie {
  constructor(title, rating) {
    this.title = title;
    this.rating = rating;
  }

  static compareMovies(movieA, movieB) {
    if (movieA.rating > movieB.rating) {
      console.log(`${movieA.title} has a higher rating.`);
    } else if (movieA.rating < movieB.rating) {
      console.log(`${movieB.title} has a higher rating.`);
    } else {
      console.log("These movies have the same rating.");
    }
  }
}

let movieA = new Movie("Movie A", 80);
let movieB = new Movie("Movie B", 45);

Movie.compareMovies(movieA, movieB);  // Movie A has a higher rating.
```

:::

Putanje za static takođe pomažu u implementaciji „fabričkih“ putanja. Fabrička putanja je putanja koju definišete zajedno sa komponentom kako biste kreirali stvari prema specifičnim kriterijumima.

```js
class Pizza {
  constructor(type, price) {
    this.type = type;
    this.price = price;
  }

  static createMargherita() {
    return new this("Margherita", 6.99);
  }
}

let myPizza = Pizza.createMargherita();
console.log(myPizza);  // Pizza { type: "Margherita", price: 6.99 }
console.log(myPizza.type);  // Margherita
```

- **Sifa za static**: Ove osobine se koriste za definisanje vrednosti ili svojstava povezanih sa samom klasom, a ne sa instancama klase. Svojstva static su podeljena među svim instancama klase i mogu se pristupiti bez kreiranja instance klase.

:::interactive_editor

```ts
class Car {
  // Static property
  static numberOfWheels = 4;

  constructor(make, model) {
    this.make = make;
    this.model = model;
  }

  // Instance method
  getCarInfo() {
    return `${this.make} ${this.model}`;
  }

  // Static method
  static getNumberOfWheels() {
    return Car.numberOfWheels;
  }
}

// Accessing static property directly from the class
console.log(Car.numberOfWheels);  // 4
```

:::
