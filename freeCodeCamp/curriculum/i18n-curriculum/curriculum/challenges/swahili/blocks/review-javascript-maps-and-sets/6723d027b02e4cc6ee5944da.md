## Sets u JavaScript

- `Set` je ugrađena opcija za upravljanje kolekcijom podataka.
- Set osigurava da se svaka vrednost unutar njega pojavljuje samo jednom, što ga čini neophodnim za uklanjanje duplikata iz niza podataka ili za rukovanje kolekcijama različitih vrednosti.
- Možete kreirati `Set` koristeći komponentu `Set()`:

:::interactive_editor

```js
const set = new Set([1, 2, 3, 4, 5]);
console.log(set); // Set { 1, 2, 3, 4, 5 }
```

:::

- Setovi se mogu kontrolisati koristeći ove metode:

- `add()`: Dodaje novi element u `Set`.
  - `delete()`: Uklanja element iz `Set`.
  - `has()`: Proverava da li je element unutar `Set`.
  - `clear()`: Uklanja sve elemente iz `Set`.
  - `keys()` i `values()`: Oba vraćaju `SetIterator` koji sadrži vrednosti od `Set`. To je isto jer `keys()` je drugi naziv za `values()`.
  - `forEach()`: oko vrednosti od `Set`.

## Weaksets u JavaScript

- `WeakSet` je kolekcija stavki koja vam omogućava skladištenje osetljivih podataka.

## Sets protiv WeakSets

- Za razliku od Sets-a, `WeakSet` ne zavisi od osnovnih tipova kao što su brojevi ili niz znakova.
- `WeakSet` čuva samo objekte, a reference na te objekte je "slaba," što znači da ako objekat nije korišćen nigde drugde u vašem kodu, on se automatski uklanja kako bi se oslobodila memorija.

## Karte u JavaScript

- `Map` je ugrađeni objekat koji čuva parove ključ-vrednost, slično običnom objektu.
- Mapovi se razlikuju od običnih objekata JavaScript po sposobnosti da dozvoljavaju ključeve bilo kog tipa, uključujući objekte i funkcije.
- `Map` pruža bolju performansu od običnog objekta kada je reč o često dodavanju i uklanjanju parova ključ-vrednost.
- Možete kreirati `Map` koristeći komponentu `Map()`:

:::interactive_editor

```js
const map = new Map([
  ['flower', 'rose'],
  ['fruit', 'apple'],
  ['vegetable', 'carrot']
]);
console.log(map); // Map(3) { 'flower' => 'rose', 'fruit' => 'apple', 'vegetable' => 'carrot' }
```

:::

- Karte se mogu upravljati korišćenjem ovih metoda:

- `set()`: Dodaje novi par ključ-vrednost na `Map`.
  - `get()`: Dohvata vrednost ključa iz `Map`.
  - `delete()`: Uklanja par ključ-vrednost iz `Map`.
  - `has()`: Proverava da li postoji ključ unutar `Map`.
  - `clear()`: Uklanja sve parove ključ-vrednost iz `Map`.

Zapamtite da sve mape i setovi imaju karakteristiku `size` koja vraća broj jedinstvenih elemenata unutar njih.

## WeakMaps u JavaScript

- `WeakMap` je kolekcija parova ključ-vrednost, kao `Map`, ali sa slabim referencama na ključeve. Ključevi moraju biti objekti, a vrednosti mogu biti bilo šta što želite.

## Maps protiv WeakMaps

- WeakMaps su jednaki WeakSets jer samo čuvaju objekte i referenca na te objekte je "slaba".
