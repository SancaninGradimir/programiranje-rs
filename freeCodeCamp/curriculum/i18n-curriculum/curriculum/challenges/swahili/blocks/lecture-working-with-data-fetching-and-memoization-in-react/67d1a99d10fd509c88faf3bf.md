# --description--

React aplikacije se često oslanjaju na spoljne API-je i baze podataka za dinamički sadržaj. Da biste preuzeli podatke iz tih API-ja i baza podataka, potrebno je koristiti određene tehnike za pribavljanje podataka.

Istražite kako funkcioniše dostupnost podataka u React i različite opcije koje imate za pristup podacima.

React ne ima specifičnih zahteva za način dohvaćanja vaših podataka, što znači da na osnovnom nivou možete koristiti ugrađeni Fetch API, koji podržavaju svi savremeni pretravači.

Takođe možete koristiti Axios i SWR. Axios je biblioteka zahteva HTTP koja se oslanja na obećanja izgrađena na objektu XMLHttpRequest, a SWR je hook React za pristup podacima kreiranim od strane tima Vercel.

Počnimo sa primerom. Prvo treba da uvezete hook-ove `useState` i `useEffect`:

```js
import { useState, useEffect } from "react";
```

Zatim ćete morati da kreirate tri stanja parametra, nazvana `loading`, `data`, i `error`:

```js
const [data, setData] = useState(null);
const [loading, setLoading] = useState(true);
const [error, setError] = useState(null);
```

Parametar `loading` će pratiti da li su podaci još uvek dostupni. Parametar `data` predstavlja same podatke, a parametar `error` će uhvatiti sve greške koje se mogu pojaviti tokom procesa dohvaćanja podataka.

Pošto je pribavljanje podataka sporedni efekat, najbolje je koristiti Fetch API unutar hook-a `useEffect`.

Evo primera toga:

```js
useEffect(() => {
  fetch("https://jsonplaceholder.typicode.com/posts")
    .then((res) => res.json())
    .then((data) => {
      setData(data);
      setLoading(false);
    })
    .catch((err) => {
      setError(err);
      setLoading(false);
    });
}, []);
```

Hook za `useEffect` dobija podatke koristeći API sa Fetch-om i postavlja sve stanja.

Možete poboljšati stvari koristeći `async`/`await` umesto sintakse od `.then()`. To znači da morate imati različitu funkciju unutar `useEffect` jer ne možete staviti prefiks `useEffect` i glavnu reč od `async`:

```js
useEffect(() => {
  const fetchData = async () => {
    try {
      const res = await fetch("https://jsonplaceholder.typicode.com/posts");
       
      if (!res.ok) {
        throw new Error("Network response was not ok");
      }

      const data = await res.json();
      setData(data);
    } catch (err) {
      setError(err);
    } finally {
      setLoading(false);
    }
  };

  fetchData();
}, []);
```

Zatim možete koristiti sva ta stanja za prikaz podataka sa API.

Evo kompletnog koda:

```jsx
import { useState, useEffect } from "react";

const FetchPosts = () => {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    const fetchData = async () => {
      try {
        const res = await fetch("https://jsonplaceholder.typicode.com/posts");
       
        if (!res.ok) {
          throw new Error("Network response was not ok");
        }

        const data = await res.json();
        setData(data);
      } catch (err) {
        setError(err);
      } finally {
        setLoading(false);
      }
    };

    fetchData();
  }, []);

  if (loading) {
    return <p>Loading...</p>;
  }

  if (error) {
    return <p>{error.message}</p>;
  }

  return (
    <ul>
      {data.map((post) => (
        <li key={post.id}>{post.title}</li>
      ))}
    </ul>
  );
};

export default FetchPosts;
```

Na UI-u, ćete videti `Loading...` na ekranu kada su podaci dostupni, zatim će se prikazati podaci ili greška, zavisno od toga da li je dohvaćanje podataka bilo uspešno.

Zapamtite da smo razgovarali i o pristupu podacima koristeći Axios i SWR. Pogledajmo primer korišćenja Axios-a.

Prvo ćeš morati da instaliraš Axios sa komandne linije kao što je ovo:

```sh
npm i axios
```

Zatim ćete morati da uključite Axios ovako:

```js
import axios from "axios";
```

Zatim možete koristiti prethodno pomenute status parametre i dobiti podatke od API koristeći `axios.get`:

```js
const [data, setData] = useState(null);
const [loading, setLoading] = useState(true);
const [error, setError] = useState(null);

useEffect(() => {
  const fetchData = async () => {
    try {
      const res = await axios.get(
        "https://jsonplaceholder.typicode.com/users"
      );
      setData(res.data);
    } catch (err) {
      setError(err);
    } finally {
      setLoading(false);
    }
  };

  fetchData();
}, []);
```

Možda ste primetili da nema linije `await res.json()` u ovom primeru. To je zato što Axios direktno obrađuje JSON, pa nema potrebe za tom linijom.

Poslednji primer koji ćemo pogledati jeste korišćenje hook-a `useSWR` za preuzimanje podataka.

Kao sa Axios-om, moraćete da instalirate SWR ovako:

```sh
npm install swr
```

Zatim ćete morati da dodate hook `useSWR` unutar fajla kao što je ovaj:

```js
import useSWR from "swr";
```

U poređenju sa prethodnim primerima, sintaksa SWR je kraća. Potrebno je kreirati *fetcher* funkciju i proslediti joj hook ``useSWR`` kao njen drugi parametar (poslednji je prvi parametar).

Takođe, pošto možete pristupiti stanjima podataka i grešaka iz hook-a `useSWR`, stoga ne zahteva hook `useState`.

Ovo je sintaksa:

```js
const fetcher = (url) => fetch(url).then((res) => res.json());
const { data, error } = useSWR(endpoint, fetcher);
```

Zapamti da je naziv "fetcher" ovde samo ugovor (konvencija), tako da ga možete nazvati bilo kojim parametrom koji želite.

Ovo je deo koji dobija podatke od API iz JSON Placeholder:

```jsx
import useSWR from "swr";

const fetcher = (url) => fetch(url).then((res) => res.json());

const FetchTodos = () => {
  const { data, error } = useSWR(
    "https://jsonplaceholder.typicode.com/todos",
    fetcher
  );

  if (!data) {
    return <h2>Loading...</h2>;
  }
  if (error) {
    return <h2>Error: {error.message}</h2>;
  }

  return (
    <>
      <h2>Todos</h2>
      <div>
        {data.map((todo) => (
          <h3 key={todo.id}>{todo.title}</h3>
        ))}
      </div>
    </>
  );
};

export default FetchTodos;
```

Kao što ste naučeni u početnoj obuci o specifičnim hook-ovima, pristup podacima je logika koju možete pružiti unutar određenog hook-a. Stoga, ako dobijate podatke u mnogo sekcija i stranica, bolje je kreirati hook za `useFetch`.

Ovo je hook `useFetch` koji koristi SWR za pristup podacima:

```jsx
import useSWR from "swr";

const fetcher = (url) => fetch(url).then((res) => res.json());

const useFetch = (url) => {
  const { data, error } = useSWR(url, fetcher);

  return {
    data,
    loading: !data && !error,
    error,
  };
};

export default useFetch;
```

Evo kako da se koristi hook `useFetch` za preuređivanje prvog primera koji dobija objave od API iz JSON Placeholder:

```jsx
import useFetch from "./useFetch";

const FetchPosts = () => {
  const { data, loading, error } = useFetch(
    "https://jsonplaceholder.typicode.com/posts"
  );

  if (loading) {
    return <h2>Loading...</h2>;
  }

  if (error) {
    return <h2>{error.message}</h2>;
  }

  return (
    <>
      <h2>Posts</h2>
      <ul>
        {data.map((post) => (
          <li key={post.id}>{post.title}</li>
        ))}
      </ul>
    </>
  );
};

export default FetchPosts;
```
