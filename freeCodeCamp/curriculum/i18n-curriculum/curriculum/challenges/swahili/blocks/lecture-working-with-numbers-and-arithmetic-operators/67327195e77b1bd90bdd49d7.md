# --interactive--

JavaScript je jezik gde stvari ponekad rade na iznenađujuće, pa čak i čudne načine. Jedna od tih iznenađenja se dešava kada pomešate brojeve i nizove slova u matematičkim operacijama. Prva stvar koju ćete pokušati je sabiranje brojeva i nizova slova. U JavaScript, operater `+` obavlja dve funkcije. On se bavi sabiranjem i spajanjem nizova slova, što je način spajanja dva niza slova zajedno.

Kada koristite `+` sa brojevima i sekvencijom slova, JavaScript bira da ih tretira sve kao sekvencu slova i spoji. Ako proverite tip rezultata za operator `typeof`, videćete da je to zapravo sekvenca slova:

:::interactive_editor

```js
const result = 5 + '10';

console.log(result); // 510
console.log(typeof result); // string
```

:::

Šta misliš da će se desiti ako promeniš redosled od `5` do `'10'`?

:::interactive_editor

```js
const result = '10' + 5;

console.log(result); // 105
console.log(typeof result); // string
```

:::

Možete videti da se nešto dogodi. `JavaScript` vidi niz slova u ``'10'`` i brojeve u ``5``, te pretvara brojeve u niz slova i spaja ih. Ovo je poznato kao konverzija tipa. Konverzija tipa je proces kada se vrednost iz jednog tipa podataka menja u drugi tip.

Stvari postaju zanimljivije kada pokušate da izvršite druge matematičke operacije kao što su oduzimanje, množenje ili deljenje koristeći niz slova i brojeva. U ovim situacijama, JavaScript pokušava da konvertuje niz slova u brojeve pre izvođenja kalkulacija – prisiljavajući ponovo tip! Evo šta se dešava:

:::interactive_editor

```js
const subtractionResult = '10' - 5;
console.log(subtractionResult); // 5
console.log(typeof subtractionResult); // number

const multiplicationResult = '10' * 2;
console.log(multiplicationResult); // 20
console.log(typeof multiplicationResult); // number

const divisionResult = '20' / 2;
console.log(divisionResult); // 10
console.log(typeof divisionResult); // number
```

:::

U gornjim primerima, JavaScript uspešno je konvertovao niz slova `'10'` ili `'20'` u brojeve i izvršio proračune. Rezultat je da `'10' - 5` izdaje `5`, `'10' * 2` izdaje `20`, i `'20' / 2` izdaje `10`.

Ali šta ako sekvenca slova nije vidljiva kao brojevi? Pogledajte šta se dešava u tom slučaju:

:::interactive_editor

```js
const subtractionResult = 'abc' - 5;
console.log(subtractionResult); // NaN
console.log(typeof subtractionResult); // number

const multiplicationResult = 'abc' * 2;
console.log(multiplicationResult); // NaN
console.log(typeof multiplicationResult); // number

const divisionResult = 'abc' / 2;
console.log(divisionResult); // NaN
console.log(typeof divisionResult); // number
```

:::

U gornjim primerima, niz slova `'abc'` ne predstavlja validnu numeričku vrednost, stoga JavaScript ne može da ga pretvori u smislen broj. Kada takva konverzija ne uspe, JavaScript vraća `NaN`, što znači "Nije Broj". `NaN` je posebna vrednost tipa `Number` koja predstavlja nevažeći ili nepredstavljivi broj.

Šta se dešava ako izvršavate aritmetičke operacije sa tačnim ili netačnim vrednostima (`true` au `false`)? Pogledajte šta se dešava. JavaScript tretira tačne ili netačne vrednosti kao brojeve u aritmetičkim operacijama: `true` se pretvara u `1`, i `false` se pretvara u `0`.

:::interactive_editor

```js
const result1 = true + 1;
console.log(result1); // 2
console.log(typeof result1); // number

const result2 = false + 1;
console.log(result2); // 1
console.log(typeof result2); // number

const result3 = 'Hello' + true;
console.log(result3); // "Hellotrue"
console.log(typeof result3); // string
```

:::

U prvih dva primera, `true + 1` kreirao je `2`, a na `false + 1` kreirao je `1`. U trećem primeru, `'Hello' + true`, JavaScript je konvertovao `true` u niz znakova i kombinovao ga sa `'Hello'`, generišući `'Hellotrue'`, koji je niz znakova.

Za `null` i `undefined`, JavaScript se ponaša `null` kao `0` i `undefined` kao `NaN` u matematičkim operacijama:

:::interactive_editor

```js
const result1 = null + 5;
console.log(result1); // 5
console.log(typeof result1); // number

const result2 = undefined + 5;
console.log(result2); // NaN
console.log(typeof result2); // number
```

:::

JavaScript često prisiljava tipizaciju, menjajući tipove podataka kao što su brojevi, nizovi karaktera i vrednosti tačno/netačno na načine koji su ponekad iznenađujući. Razumevanje ovih promena je važno kako bi se izbegle greške i napisao robusan kod u vašim projektima.
