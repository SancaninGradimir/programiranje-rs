## Uvod u osnovne module za Node.js

- **Osnovni moduli za Node.js**: Ugrađeni moduli koji dolaze gotovi sa Node.js, pružajući ključnu funkcionalnost za operacije sa fajlovima, šifrovanje, interakciju sa operativnim sistemom i više.
- **Standardni moduli**: Nije potrebna dodatna instalacija; dostupni su čim je Node.js instaliran.
- **Sintaksa za import**: Koristite `require()` da uvezete osnovni modul u vašu aplikaciju.

```javascript
const fs = require("fs");
const crypto = require("crypto");
const os = require("os");
```

## Modul sistema fajlova (fs)

- **Svrha**: Omogućava metode za rad sa datotekama i folderima, uključujući otvaranje, zatvaranje, čitanje, pisanje i brisanje.
- **Import**: `const fs = require("fs");`
- **Varijabilnost**: Dostupne su za tri tipa: povratni poziv, obećanje (promise) i asinhrono.

### Sistemi za korišćenje ruta

- **Asinhroni poziv povratne funkcije**: Standardno ponašanje, neblokirajući proces.

```javascript
fs.writeFile("filePath", "content", "utf8", (err) => {
  if (err) throw err;
  console.log("File written to!");
});
```

- **Korišćenje obećanja**: Preporučuje se za moderne aplikacije, koristi `async/await`.

```javascript
const fs = require("fs/promises");

async function writeToFile() {
  try {
    await fs.writeFile("article.md", "## Node fs Module");
    console.log("File written to!");
  } catch (err) {
    console.error(err);
  }
}
```

- **Sinkhrono**: Akcija koja blokira, pogodno za jednostavne skripte.

```javascript
try {
  fs.writeFileSync("article.md", "## Node fs Module", "utf8");
  console.log("File written to!");
} catch (err) {
  console.error(err);
}
```

### Glavne operacije fajla

- **Putanja za `writeFile()`**: Kreira ili prepisuje fajl sa dostavljenim sadržajem.

```javascript
await fs.writeFile("article.md", "## Node fs Module: The Complete Guide");
```

- **Put `appendFile()`**: Dodaje sadržaj na kraj postojeće datoteke.

```javascript
await fs.appendFile("article.md", "\n\nIn this article, you will learn...");
```

- **Putanja do `readFile()`**: Čita sadržaj fajla.

```javascript
const content = await fs.readFile("article.md", "utf8");
console.log("File content:", content);
```

- **Putanja za `unlink()`**: Briše fajl iz sistema fajlova.

```javascript
await fs.unlink("article.md");
console.log("File deleted successfully");
```

### Kodiranje znakova

- **UTF-8 Kodiranje**: Objasni `"utf8"` čitanje fajlova kao sekvencu znakova.
- **Buffer Struktura**: Bez specifikacije kodiranja, fajlovi se čitaju kao privremeni binarni skladišni objekti.

```javascript
// Returns string
const textContent = await fs.readFile("file.txt", "utf8");

// Returns Buffer
const bufferContent = await fs.readFile("file.txt");
```

## Buffer moduli

- **Svrha**: Обраđuje binarni podatke direktno u memoriji za fajlove, slike i mrežne tokove.
- **Import**: `const { Buffer } = require("buffer");`
- **Upotreba**: Ulaz/Izlaz fajlova, TCP tokovi, obrada slika i upravljanje binarnim podacima.

### Kreiranje buffera

- **Metoda za `Buffer.from()`**: Kreira *buffer* iz sekvence karaktera, redova ili drugih podataka.

```javascript
// From string
const myStrBuffer = Buffer.from("freeCodeCamp");
console.log(myStrBuffer); // <Buffer 66 72 65 65 43 6f 64 65 43 61 6d 70>

// From array of numbers
const myNumBuffer = Buffer.from([70, 82, 69, 69, 67, 79, 68, 69, 67, 65, 77, 80]);
```

- **Njia ya `Buffer.alloc()`**: Kreira *buffer* veličine koja je zadata, popunjen nulama.

```javascript
const someBuffer = Buffer.alloc(10);
console.log(someBuffer); // <Buffer 00 00 00 00 00 00 00 00 00 00>
```

### Aktivnosti za keš/buffer

- **Pristup kao niz**: Dohvatiti elemente bafera jedan po jedan korišćenjem indeksa.

```javascript
console.log(myStrBuffer[0]); // 102 (byte value for 'f')
```

- **Put za `toString()`**: Konvertuje buffer u čitljiv niz znakova.

```javascript
console.log(myStrBuffer.toString()); // "freeCodeCamp"
```

- **Njia ya `Buffer.write()`**: Piše podatke u dodeljeni keš/buffer.

```javascript
someBuffer.write("Hello fCC");
console.log(someBuffer.toString()); // "Hello fCC"
```

- **Njia ya `Buffer.byteLength()`**: Vraća broj bajtova potrebnih za skladištenje sekvence karaktera.

```javascript
console.log(Buffer.byteLength("Hello freeCodeCamp")); // 18
```

### Putevi za dodatni keš/buffer

- **`Buffer.isBuffer()`**: Proverava da li je nešto *buffer*.
- **`Buffer.compare()`**: Upoređuje dva *buffer*-a i vraća redosled sortiranja.
- **`Buffer.concat()`**: Spaja više *buffer*-a u jedan.

## Kripto moduli

- **Svrha**: Pruža funkcionalnost šifrovanja, uključujući heširanje (hashing), enkripciju, maskiranje i digitalne potpise.
- **Import**: `const crypto = require("crypto");`
- **Napomena o bezbednosti**: Koristite testirane biblioteke kao što su `bcrypt` ili `jsonwebtoken` za verifikaciju identiteta u produkciji.

### Metode za heširanje

- **Njia ya `createHash()`**: Generiše hash jedne putanje koristeći algoritme kao što su SHA256, SHA512 ili MD5.

```javascript
const hashedPassword = crypto
  .createHash("sha256")
  .update("myPassword123")
  .digest("hex");

console.log("createHash result:", hashedPassword);
```

- **Metoda `createHmac()`**: Kreira hash koristeći tajni ključ za autentifikaciju i integritet podataka.

```javascript
const hashedMessage = crypto
  .createHmac("sha256", "secret-key")
  .update("Hello World")
  .digest("hex");
```

### Šifrovanje i maskiranje

- **Njia za `createCipheriv()` na `createDecipheriv()`**: Dekriptovanje i otkrivanje podataka korišćenjem algoritama, ključeva i vektora inicijalizacije.

```javascript
const key = Buffer.from("12345678901234567890123456789012"); // 32 bytes for AES-256
const iv = Buffer.from("1234567890123456"); // 16 bytes for AES

// Encryption
const cipher = crypto.createCipheriv("aes-256-cbc", key, iv);
let encrypted = cipher.update("Hello campers!", "utf8", "hex");
encrypted += cipher.final("hex");

// Decryption
const decipher = crypto.createDecipheriv("aes-256-cbc", key, iv);
let decrypted = decipher.update(encrypted, "hex", "utf8");
decrypted += decipher.final("utf8");
```

### Generisanje nasumičnih vrednosti

- **Njia ya `randomBytes()`**: Generiše kriptografski sigurne nasumične bajtove.

```javascript
console.log("Random Bytes:", crypto.randomBytes(16).toString("hex"));
// Output: a6154ef5a296fa176ad0f332bd94d712
```

- **Metoda `randomInt()`**: Generiše potpune sigurnosne slučajne brojeve unutar definisanog opsega.

```javascript
console.log("Random Int:", crypto.randomInt(0, 100)); // Random number 0-99
```

### Upravljanje ključevima

- **Njia ya `createSecretKey()`**: Generiše objekte kriptografskih ključeva iz sirovih bajtova.

```javascript
const secret = crypto.createSecretKey(crypto.randomBytes(32));
console.log(secret.export().toString('hex'));
```

### Digitalni potpisi

- **Njia za `sign()` na `verify()`**: Kreiranje i verifikacija digitalnih potpisa korišćenjem parova privatnih/javnih ključeva.
- **Upotreba**: Prikazuje validnost i integritet podataka.

### Načini za dodatni kripto

- **`createPublicKey()` na `createPrivateKey()`**: Radi sa spolja generisanim ključevima.
- **`createDiffieHellman()`**: Omogućava kreiranje sigurne zajedničke tajne.
- **`Certificate()`**: Upravlja operacijama sertifikata za HTTPS.

## Moduli operativnog sistema

- **Svrha**: Pruža alate povezane sa operativnim sistemom, sistemskim informacijama i podrškom za različite platforme.
- **Import**: `const os = require("os");`
- **Upotreba**: Monitoring sistema, pisanje skripti za različite platforme, informacije o hardveru.

### Putanje za sistemske informacije

- **Njia ya `platform()`**: Restauracija platforme operativnog sistema.

```javascript
console.log(os.platform()); // 'darwin', 'win32', 'linux', etc.

// Cross-platform usage
if (os.platform() === 'win32') {
  // Windows-specific code
} else {
  // Unix-like systems
}
```

- **Njia ya `arch()`**: Obnavlja arhitekturu procesora (CPU).

```javascript
console.log(os.arch()); // 'arm64', 'x64', 'arm', 'ia32', etc.
```

- **Njia ya `type()`**: Vraćanje zvaničnog imena operativnog sistema.

```javascript
console.log(os.type()); // 'Darwin', 'Linux', 'Windows_NT'
```

### Napomene o verziji

- **Putanja za `release()`**: Pokazuje verziju kernela operativnog sistema.

```javascript
console.log(os.release()); // '25.0.0'
```

- **Putanja za `version()`**: Dohvata detaljne informacije o verziji OS-a.

```javascript
console.log(os.version());
// Darwin Kernel Version 25.0.0: Wed Sep 17 21:41:39 PDT 2025
```

### Informacije o opremi

- **Putanja za `cpus()`**: Obnavlja red podataka o resursima CPU jezgara.

```javascript
console.log(os.cpus()); // Array of CPU core objects with model, speed, times
```

- **Njia ya `uptime()`**: Resetuje trajanje rada sistema u sekundama.

```javascript
console.log(os.uptime()); // 23047 (seconds since system boot)
```

### Napomene

- **Njia ya `totalmem()`**: Dohvata ukupnu sistemsku memoriju po bajtu.
- **Njia ya `freemem()`**: Dohvata dostupnu sistemsku memoriju po bajtu.
