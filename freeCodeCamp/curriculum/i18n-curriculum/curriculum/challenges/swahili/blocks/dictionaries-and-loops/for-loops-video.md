# --questions--

## --text--
Koja od sljedećih metoda je ispravna za iteriranje preko svakog karaktera u nizu znakova `"Giraffe Academy"`?

## --answers--

```python
for (letter) in "Giraffe Academy":
    print(letter)
```

---

```python
for key, letter in "Giraffe Academy":
    print(letter)
```

---

```python
for letter in "Giraffe Academy":
    print(letter)
```

---

```python
for letter, value in "Giraffe Academy":
    print(letter)
```
