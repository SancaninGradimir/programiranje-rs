# --description--

Neka $S_n$ bude niz celih brojeva generisan sledećim generatorom pseudo-slučajnih brojeva:

$$\begin{align}
        S_0 & = 290\\,797 \\\\
  S_{n + 1} & = {S_n}^2\bmod 50\\,515\\,093
\end{align}$$

Neka $A(i, j)$ bude najmanji broj u nizu $S_i, S_{i + 1}, \ldots, S_j$ za $i ≤ j$. Neka $M(N) = \sum A(i, j)$ za $1 ≤ i ≤ j ≤ N$.

Možemo potvrditi da je $M(10) = 432\\,256\\,955$ i $M(10\\,000) = 3\\,264\\,567\\,774\\,119$.

Pronađi $M(2\,000\,000\,000)$.
