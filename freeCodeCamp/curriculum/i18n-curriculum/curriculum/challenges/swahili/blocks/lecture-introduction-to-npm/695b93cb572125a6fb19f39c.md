# --description--

Tokom inicijalnog kursa, naučili ste koje zavisnosti i kako izabrati odgovarajuće zavisnosti za vaš projekat.

U ovom tutorijalu, naučićete kako da instalirate i uklonite zavisnosti koristeći npm.

Da biste instalirali zavisnost, koristite komandu ``npm install`` kao što sledi:

```bash
npm install dependencyName
```

Možeš skratiti `install` u `i`, ovako:

```bash
npm i dependencyName
```

Nakon instaliranja paketa, npm dodaje novi deo `dependencies` u vaš fajl `package.json` (ako već ne postoji) i navodi instalirani paket. Na primer, ovako izgleda kada dodate paket `chalk`:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "chalk": "^5.6.2"
  }
}
```

Kada dodate zavisnosti, dve nove stvari se pojavljuju u glavnom direktorijumu vašeg projekta:

* `node_modules/`: fascikla koja sadrži stvarni kod instaliranih paketa.
    
* `package-lock.json`: datoteka koja čuva detaljne informacije o tačnim verzijama tih paketa, zajedno sa njihovim podzavisnostima. (Više o ovoj datoteci naučićeš u sledećoj lekciji.)
    

<img src="https://cdn.freecodecamp.org/curriculum/lecture-transcripts/how-can-you-install-and-remove-dependencies-1.png" alt="">

Mnogi projekti zahtevaju više od jedne zavisnosti. Umesto da ih instaliraš jednu po jednu, npm ti omogućava da instaliraš više paketa odjednom tako što navedeš njihove nazive odvojene razmacima:

```bash
npm install express lodash bcrypt
```

Kao i ranije, kada instaliraš više zavisnosti odjednom, npm ažurira `package.json`, `package-lock.json` i `node_modules` kako bi ih uključio. Ovako će izgledati `package.json`:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "bcrypt": "^6.0.0",
    "chalk": "^5.6.2",
    "express": "^5.1.0",
    "lodash": "^4.17.21"
  }
}
```

Da bi instalirao `devDependency`, dodaj zastavicu `--save-dev` komandi `install`. Možeš koristiti i skraćenu verziju `-D`:

```bash
npm i nodemon -D
```

Nakon izvršavanja ove komande, npm ažurira vaš fajl `package.json` i dodaje paket ispod sekcije `devDependencies`. Evo primera kako to izgleda:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "dependencies": {
    "bcrypt": "^6.0.0",
    "chalk": "^5.6.2",
    "express": "^5.1.0",
    "lodash": "^4.17.21"
  },
  "devDependencies": {
    "nodemon": "^3.1.10"
  }
}
```

Kada izvršite komandu `npm install`, npm će prikazati poruku koja vas obaveštava da li je instalacija uspešna ili ne. Poruka o uspehu obično izgleda ovako:

Da biste izbrisali ili uklonili paket, koristite komandu `npm uninstall`:

```bash
npm uninstall dependencyName
```

Na primer, da biste izbrisali `chalk` iz vašeg projekta, izvršićete ovu komandu:

```bash
npm uninstall chalk
```

Takođe možete obrisati više paketa odjednom navođenjem njihovih imena i razmaka između svakog imena paketa:

```bash
npm uninstall express lodash bcrypt
```

Takođe ćete dobiti poruku koja prikazuje da li je komanda za brisanje uspela ili nije:


Na vašoj datoteci za `package.json` će biti ažurirano automatski:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC",
  "devDependencies": {
    "nodemon": "^3.1.10"
  }
}
```

Da biste obrisali `devDependency`, možete samo pokrenuti:

```bash
npm uninstall nodemon
```

Zapamtite da vam nije potreban zastavica za `-D` prilikom brisanja. npm će obrisati paket direktno iz ispravne lokacije, tj. `devDependencies` u ovom slučaju.

Nakon brisanja vašeg `devDependency`, `package.json`, ažuriraće se ponovo:

```json
{
  "name": "install-uninstall-deps",
  "version": "1.0.0",
  "description": "a simple demo on how to install and uninstall dependencies",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": ["npm", "deps"],
  "author": "Kolade Chris",
  "license": "ISC"
}
```
