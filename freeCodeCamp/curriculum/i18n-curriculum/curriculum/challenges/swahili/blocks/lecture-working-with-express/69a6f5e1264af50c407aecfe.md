# --description--

U prošloj lekciji, kreirao si projektnu mapu i instalirao Express:

```bash
mkdir my-app
cd my-app
npm init -y
npm install express
```

Kao podsetnik, ove komande kreiraju novu projektnu mapu, `cd` unutar projektne mape, inicijalizuju projekat i kreiraju osnovnu datoteku za `package.json`, te instaliraju Express.

Sledeći, otvori editor koda, kreiraj fajl pod nazivom `index.js`, i napiši svoj prvi Express:

```javascript
const express = require("express");
const app = express();
const port = 3000;

app.get("/", (req, res) => {
  res.send("Hello World!");
});

app.listen(port, () => {
  console.log(`Example app listening on port ${port}`);
});
```

Možete pokrenuti svoj kod korišćenjem sledeće komande u terminalu:

```bash
node index.js
```

Otvori svoju garažnu kapiju i videćeš poruku [http://localhost:3000](http://localhost:3000) **"Hello World!"**

Tako brzo možete početi sa Express! Sada hajde da pogledamo kod korak po korak.

U prvoj liniji koda, učitavate biblioteku Express i postavljate vrednost parametru nazvanom `express`:

```javascript
const express = require("express");
```

Parametar `express` je ugrađena funkcija. Kada ga pozovete, kreira i vraća novu instancu programa za Express:

```javascript
const app = express();
```

Sada `app` ima pristup svim metodama koje pruža Express, kao što su `.get()`, `.post()`, `.listen()`, itd. Ovo je vaš glavni serverski centar.

Sljedeći red objašnjava parametar `port`. Port je mrežna tačka slušanja koju vaša aplikacija prati, a `3000` je port koja se često koristi za lokalno razvijanje:

```javascript
const port = 3000;
```

Sada, pogledajmo sledeći deo ovde:

```javascript
app.get("/", (req, res) => {
  res.send("Hello World!");
});
```

Ovaj deo koda objašnjava putanju podataka i govori serveru kako da odgovori na zahteve. `app.get()` registruje rukovatelja za zahteve tipa HTTP GET. GET zahtev je standardan način HTTP za dobijanje podataka.

Lista za `"/"` unutar `app.get()` predstavlja putanju podataka. U ovom slučaju, ukazuje na URL korena programa.

Deo `(req, res) => { ... }` je funkcija odgovora koja se izvršava kada zahtev odgovara ovoj putanji podataka. Servis `req` sadrži informacije o dolaznom zahtevu, a servis `res` se koristi za slanje odgovora nazad klijentu.

Na kraju, `res.send("Hello World!")` šalje odgovor klijentu. U ovom primeru, odgovor je običan tekst `"Hello World!"`.

Sad, hajde da pregledamo poslednji deo koda ovde:

```javascript
app.listen(port, () => {
  console.log(`Example app listening on port ${port}`);
});
```

Putanja za `app.listen()` inicijalizuje aplikaciju Express i govori Node-u da sluša zahteve HTTP koji dolaze na pomenuti port, u ovom slučaju, `3000`.

Kada počne, dešava se rukovanje događajem i poruka `Example app listening on port 3000` piše se u konzolu, potvrđujući da je aplikacija Express uspešno pokrenuta i spremna za primanje zahteva.
