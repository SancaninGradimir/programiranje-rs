# --interactive--

Floats u CSS je tehnika prvobitno kreirana da bi omogućila tekstu da se zaokruži oko elementa, kao što su slike. Međutim, sa vremenom, programeri su otkrili nove načine korišćenja floats-a, koristeći ih kreativno u dizajnu rasporeda. Iako se moderne metode kao što su Flexbox i Grid sada koriste više, razumevanje floats-a je i dalje važno, posebno kada koristite stari kod ili vam je potrebno postići specifične efekte rasporeda.

Kada učinite da je element "float" (plutajući), uklanja se iz normalnog toka dokumenta i pomiče se levo ili desno unutar svog kontejnera. Sadržaj koji sledi će obaviti plutajući element, popunjavajući preostali prostor. Uobičajena upotreba je obavijanje teksta pored slike, gde se slika postavlja na jednu stranu, a tekst se oko nje obavija. Ova tehnika i dalje je široko korišćena, posebno u člancima i blogovima gde su slike potrebne pored teksta. Evo primjera koda:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <img src="https://placehold.co/150x150" alt="Placeholder Image">
</div>
```

```css
.container {
  border: 1px solid black;
  padding: 10px;
}

img {
  float: left;
  margin-right: 20px;
}
```

:::

U ovom primeru, kontejner ne obuhvata/omotava plutajuću sliku. Slika je van normalnog toka dokumenta, i kontejner se srušio na visinu od nule jer ne detektuje plutajući element unutar sebe.

Floats su takođe bili popularni za kreiranje višerednih rasporeda pre nego što su Flexbox i Grid postali standard. Postavljanjem elemenata jedan pored drugog, programeri su mogli da kreiraju redove sa jednakim poravnanjem. Međutim, kada koristite floats, važno je rešiti problem kolapsa roditeljskog elementa kada su elementi deteta floatovani. Rešenje metode clearfix primenjuje se na element sa klasom `container` da bi popravilo ovaj problem.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <img src="https://placehold.co/150x150" alt="Placeholder Image">
  <p> This is an example of text flowing around a floated image.</p>
</div>
```

```css
.container {
  border: 1px solid black;
}

/* Clearfix CSS */
.container::after {
  content: "";  
  display: block;
  clear: both;
}

img {
  float: left;
  margin-right: 20px;
}
```

:::

- `::after` je zamenski element koji dodaje nevidljivi segment nakon sadržaja kontejnera.
- `content: ""` osigurava da postoji zamenski element, ali ne prikazuje nikakav sadržaj.
- `display: block` čini zamenski element blok nivo/elementom.
- `clear: both` osigurava da zamenski element uklanja obe strane bilo kojeg float elementa koji je iznad njega.

U ovom primeru, dodali smo novi paragrafski element kako bismo jasnije demonstrirali kolaps kontejnera. Pošto je paragraf još uvek u normalnom toku dokumenta, kontejner se proširi dovoljno da ga obuhvati. Zatim koristimo tehniku 'clearfix' da ispravimo kolaps i osiguramo da granica kontejnera bude pravilno prikazana. Možete pokušati da uklonite pravilo 'clearfix' kako biste videli kako se kontejner ponovo kolapsira.

Metoda clearfix osigurava da roditeljski element pravilno obuhvati svoje floatovane elemente. Clearfix prisiljava roditeljski kontejner da "vidi" floatovane elemente dodavanjem svojstva clear nakon floatovanog sadržaja.

Iako float više nije uobičajen način za složene rasporede zbog modernih tehnika kao što su Flexbox i Grid, oni i dalje imaju važnu ulogu u određenim situacijama. Bilo da obavijaš tekst oko slike ili radiš na starijim projektima, razumevanje kako float funkcioniše i kako ga pravilno ukloniti važno je za održavanje dobro organizovanih i prilagodljivih rasporeda.
