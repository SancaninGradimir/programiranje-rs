# --description--

`Anyone who` koristi se za referisanje na bilo koju osobu koja ispunjava određeni uslov ili potrebu.

U tehnologiji, često se koristi za opisivanje ljudi sa posebnim veštinama ili znanjem. Na primer, `Anyone who understands JavaScript can modify this script.`

# --questions--

## --text--
