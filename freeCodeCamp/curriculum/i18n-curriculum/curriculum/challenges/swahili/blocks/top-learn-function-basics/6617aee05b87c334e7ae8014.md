# --description--

Varijabla deklarisana unutar funkcije vidljiva je samo unutar te funkcije. Ovo se naziva opseg varijable, a ovaj specifičan opseg nazivaju lokalni opseg. Varijable deklarisane izvan funkcije nazivaju se globalne varijable i vidljive su u celom programu. Na primer, sledeći kod će proizvesti grešku:

```js
function myFunction() {
  let x = 10;
}

console.log(x);
```

Parametar `x` je deklarisan unutar funkcije `myFunction` i nije vidljiv van te funkcije. Kod će vratiti `ReferenceError` jer `x` nije deklarisan.

Ako deklarišete parametar van funkcije, on se naziva globalni parametar i vidljiv je u celom programu. Na primer, sledeći kod će raditi:

```js
let x = 10;

function myFunction() {
  console.log(x);
}

myFunction();
```

Ako se parametar sa istim imenom deklarisan unutar funkcije, onda će sakriti eksterni parametar.
