## Formatiranje liste

- **Svojstvo `line-height`**: Ovo svojstvo se koristi za kreiranje razmaka između redova teksta. Dozvoljene vrednosti za `line-height` uključuju glavne reči `normal`, brojeve, procenate i jedinice dužine kao što je jedinica `em`.
- **Svojstvo `list-style-type`**: Ovo svojstvo se koristi za određivanje oznake stavke u listi. Dozvoljene vrednosti mogu uključivati krug, disk ili decimalne brojeve.

Ovo je primer koji koristi `list-style-type` za menjanje stila znaka:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
ul {
  list-style-type: square;
}
```

:::

- **Atribut za `list-style-position`**: Ovaj atribut se koristi za određivanje položaja oznake liste. Prihvatljive su samo dve vrednosti: unutra i spolja.

Ovo je primer koji pokazuje razliku između `inside` i `outside`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="inside-list">
  <li>Item with inside position</li>
  <li>Item with inside position</li>
</ul>
<ul class="outside-list">
  <li>Item with outside position</li>
  <li>Item with outside position</li>
</ul>
```

```css
.inside-list {
  list-style-position: inside;
}

.outside-list {
  list-style-position: outside;
}
```

:::

- **Sifa ya `list-style-image`**: Ovo svojstvo se koristi za korišćenje slike kao markera stavke u listi. Standardna upotreba je korištenje funkcije `url` sa vrednošću postavljenu za validnu lokaciju slike.

## Dodavanje razmaka između stavki liste korišćenjem `margin`

- Osim `line-height`, margine se takođe mogu koristiti u CSS za poboljšanje razmaka i čitljivosti stavki liste.
- Margine kreiraju prostor izvan svake stavke od `li`, omogućavajući kontrolu razmaka između stavki liste.
- `margin-bottom` se koristi za kreiranje prostora ispod svake stavke liste. Na primer, `margin-bottom: 10px;` će stvoriti razmak od 10 piksela ispod svake stavke liste.

Evo primera koji koristi `margin-bottom` da doda razmak između stavki liste:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ul class="list">
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>
```

```css
.list li {
  margin-bottom: 20px;
}
```

:::

## Začinjavanje sastojaka

- **`pseudo-class`**: Ovo je glavni termin koji se dodaje u selektor/padajući meni koji vam omogućava da izaberete elemente na osnovu određenog stanja. Uobičajena stanja uključuju stanja za `:hover`, `:visited` i `:focus`.
- **`:link pseudo-class`**: Ova pseudo-klasa se koristi za stilizovanje linkova koje korisnik još nije posetio.

Ovo je primer koji koristi lažnu klasu za `:link`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:link {
  color: red;
}
```

:::

- **`:visited pseudo-class`**: Ovaj pseudo-klasa se koristi za stilizovanje linkova koje je korisnik već posetio.
- **`:hover pseudo-class`**: Ovaj pseudo-klasa se koristi za stilizovanje elemenata koje korisnik trenutno pregleda tokom navigacije.

Evo primera koji koristi dummi klasu `:hover`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Hover over me!</a>
```

```css
a:hover {
  color: green;
  text-decoration: underline;
}
```

:::

- **`:focus pseudo-class`**: Ova lažna klasa se koristi za dekorisanje elementa kada primi pažnju. Primeri uključuju elemente od `input` ili `select` gde kliknete ili dodirnete element da mu date pažnju.

Ovo je primer koji koristi sintetičku klasu za `:focus`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Example link</a>
```

```css
a:focus {
  outline: 2px solid orange;
}
```

:::

- **`:active pseudo-class`**: Ova pseudo-klasa se koristi za stilizovanje elementa koji je interaktivan sa strane korisnika. Uobičajen primer je kada korisnik klikne na dugme.

Ovo je primer koji koristi mock klasu `:active`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="/">Click me!</a>
```

```css
a:active {
  color: pink;
}
```

:::

## Rad sa pejzažima i granicama

- **Svojstvo `background-size`**: Ovo svojstvo se koristi za postavljanje veličine pozadine elementu. Uobičajene vrednosti uključuju `cover` za pozadinsku sliku koja pokriva ceo element i `contain` za pozadinsku sliku koja odgovara unutar elementa.

Ovde je primer koji koristi `background-size: cover`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

Ovo je primer koji koristi `background-size: contain`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **Atribut `background-repeat`**: Ovaj atribut se koristi za određivanje kako pejzažne slike ponavljaju duž horizontalne i vertikalne ose. Podrazumevana vrednost `background-repeat` je `repeat`, što znači da će se slika ponavljati horizontalno i vertikalno. Takođe možete specificirati neponavljanje slike korišćenjem atributa `no-repeat`.

Ovo je primer koji koristi `background-repeat: no-repeat`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

- **Svojstvo `background-position`**: Ovo svojstvo se koristi za određivanje pozicije pozadinske slike. Može se postaviti sa specifičnom dužinom, procentom ili rečima kao što su `top`, `bottom`, `left`, `right`, i `center`.

Ovo je primer koji koristi `background-position`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

- **Svojstvo `background-attachment`**: Ovo svojstvo se koristi za određivanje da li će pozadinska slika pratiti sadržaj ili ostati na mestu. Glavna vrednost je `scroll` (podrazumevano), gde pozadinska slika prati sadržaj, i `fixed`, gde pozadinska slika ostaje na mestu na ekranu.
- **Svojstvo `background-image`**: Ovo svojstvo se koristi za postavljanje pozadinske slike elementa. Možete postaviti više pozadinskih slika odjednom i koristiti funkciju `url`, `radial-gradient` ili `linear-gradient` kao vrednosti.

Ovo je primer koji koristi `background-image`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **Sifa ya `background`**: Ovo je skraćeni atribut za postavljanje svih atributa teme u jednoj deklaraciji. Evo primera kako postaviti pozadinsku sliku i sprečiti njeno ponavljanje: `background: no-repeat url("example-url-goes-here");`

Ovde je primer koji koristi funkciju skraćivanja za `background`:

:::interactive_editor

```html
<style>
  body {
    background: center top no-repeat url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    min-height: 100px;
  }
</style>
```

:::

- **Dobar kontrast za pozadinu i prednji plan**: Važno je osigurati da boje pozadine i teksta imaju dobar odnos kontrasta kako bi tekst bio čitljiv. Web Content Accessibility Guidelines (WCAG) preporučuju minimalni odnos kontrasta od 4,5:1 za običan tekst i 3:1 za veliki tekst.

## Granice

- **Sifa za `border-top`**: Ova svojstvo se koristi za postavljanje stilova gornje ivice elementa. `border-top: 3px solid blue;` postavlja čvrstu plavu ivicu širine 3 piksela na gornjoj strani elementa.
- **Sifa za `border-right`**: Ova svojstvo se koristi za postavljanje stilova desne ivice elementa. `border-right: 2px solid red;` postavlja čvrstu crvenu ivicu širine 2 piksela na desnoj strani elementa.
- **Sifa za `border-bottom`**: Ova svojstvo se koristi za postavljanje stilova donje ivice elementa. `border-bottom: 1px dashed green;` postavlja zelenu ivicu sa kružnim linijama širine 1 piksel na donjoj strani elementa.
- **Sifa za `border-left`**: Ova svojstvo se koristi za postavljanje stilova leve ivice elementa. `border-left: 4px dotted orange;` postavlja narandžastu ivicu sa tačkama širine 4 piksela na levoj strani elementa.

Evo primera koji koristi privatna svojstva:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="bordered-box">Box with different borders</div>
```

```css
.bordered-box {
  border-top: 3px solid blue;
  border-right: 2px solid red;
  border-bottom: 1px dashed green;
  border-left: 4px dotted orange;
  padding: 20px;
}
```

:::

- **Svojstvo `border`**: Ovo je skraćeno svojstvo za postavljanje širine, stila i boje granice elementa. `border: 1px solid black;` postavlja čvrstu crnu granicu sa širinom od 1 piksel.

Ovo je primer koji koristi značaj skraćivanja `border`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

- **Svojstvo `border-radius`**: Ovo svojstvo se koristi za kreiranje zaobljenosti uglova za granicu elementa.

Ovo je primer koji koristi `border-radius`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

- **Sifa za `border-style`**: Ova svojstvo se koristi za postavljanje stila ivice elementa. Prihvatljive vrednosti uključuju `solid`, `dashed`, `dotted`, na `double`.

Ovde je primer koji koristi različite vrednosti za `border-style`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="solid-border">Solid border</div>
<div class="dashed-border">Dashed border</div>
<div class="dotted-border">Dotted border</div>
<div class="double-border">Double border</div>
```

```css
.solid-border {
  border: 3px solid blue;
  margin-bottom: 10px;
  padding: 10px;
}

.dashed-border {
  border: 3px dashed red;
  margin-bottom: 10px;
  padding: 10px;
}

.dotted-border {
  border: 3px dotted green;
  padding: 10px;
}

.double-border {
  border: 3px double blueviolet;
  padding: 10px;
}
```

:::

## Gradijent boje

- **Funkcija za `linear-gradient()`**: Ova funkcija, koja je deo CSS, koristi se za kreiranje prelaza između više boja duž prave linije.

Ovde je primer koji koristi `linear-gradient()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

- **Funkcija `radial-gradient()`**: Ova funkcija CSS stvara sliku koja dolazi iz određenog mesta, kao krug ili prsten, i polako prolazi kroz mnogo boja.

Ovo je primer koji koristi `radial-gradient()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```
