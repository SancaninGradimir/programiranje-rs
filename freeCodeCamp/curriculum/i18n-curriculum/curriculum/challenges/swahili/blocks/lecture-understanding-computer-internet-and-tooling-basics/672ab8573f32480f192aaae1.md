# --description--

Profesionalni programeri se oslanjaju na različite alate da bi povećali efikasnost i kvalitet koda. Naučićemo se o tim alatima, uključujući one koji su potpuno vidljivi.

Prvo među tim alatima je računar. Računar je glavno razvojno okruženje. Može biti teški desktop ili prenosivi laptop sa operativnim sistemom Windows, macOS ili Linux.

Profesionalni programeri često biraju računare sa brzim procesorskim moći i velikom količinom RAM-a za obradu zadataka koji zahtevaju mnogo resursa.

Kompijuter je program za pisanje i uređivanje koda direktno na tom kompijuteru. To je uređivač koda ili integrisano razvojno okruženje (IDE).

IDE kao Visual Studio, IntelliJ IDEA, JetBrains i PyCharm nude moćne funkcije kao što su kompletiranje koda, debagovanje grešaka i podrška integrisanom terminalu.

Visual Studio Code (VS Code) osnovno je editor koda, ali takođe pruža ove usluge kroz svoju bogatu biblioteku ekstenzija.

Mahariri wengine wa msimbo ni Sublime Text, Atom, Notepad++, na Brackets.

Kada pišete kod koristeći editori koda i IDE, morate pratiti promene koje vršite. Alat koji vam omogućava da pratite te promene je sistem za kontrolu verzija.

Git je sistem za kontrolu verzija koji se široko koristi u zajednici razvoja.

Platforme kao što su GitHub, GitLab i Bitbucket nude usluge hostovanja Git repozitorijuma u oblaku. Ove platforme omogućavaju saradnju sa drugim programerima, rad na različitim granama i spajanje izmena u kodu.

Upravljači paketima su još jedan važan alat. Oni programerima olakšavaju dodavanje, ažuriranje i uklanjanje biblioteka i zavisnosti projekta.

Primjeri popularnih menadžera paketa su:

* NPM, Yarn, i PNPM za JavaScript
* PIP za Python
* Composer za PHP
* Maven za Java

Nakon što napišu kod koristeći različite alate, programeri testiraju taj kod kako bi osigurali da radi onako kako je očekivano.

U vezi s tim, programeri koriste testne okvire kao što su Cypress, Playwright, Selenium i slične. Postoje takođe specijalizovani testni okviri za određene jezike. Primeri su:

* Jest za JavaScript
* pytest za Python
* JUnit za Java
* PHPUnit za PHP

Morate da testirate kod kako biste osigurali da radi onako kako je očekivano. Takođe morate da testirate izgled koda za krajnje korisnike. Programeri koriste veb pretraživače za ovo.

Savremeni pretraživači kao što su Chrome, Firefox, Edge i Safari nude razvojne alate za inspekciju koda HTML, CSS, i JavaScript. Postoje takođe alati za otklanjanje grešaka (debugging) i testiranje performansi.

Ovi alati pomažu programerima da testiraju i poboljšaju svoje veb aplikacije u različitim pregledačima za krajnje korisnike.
