# --description--

**Cilj:** Popunite korisničke priče ispod i prođite sve testove kako biste završili ovaj laboratorijski rad.

**Priče korisnika:**

1. Trebalo bi da postoji funkcija pod nazivom `pyramid` koja prima tri argumenta.
1. Prvi argument treba da bude niz znakova koji predstavlja ponavljajući uzorak u vašoj piramidi.
1. Drugi argument treba da bude ceo broj koji predstavlja broj redova u piramidi.
1. Treći argument treba da bude booleanski tip vrednosti, ili tačno ili netačno.
1. Funkcija `pyramid` treba da vrati niz znakova gde se uzorak ponavlja i aranžira kako bi stvorio piramidu sa vrhom usmerenim nagore kada je treći argument `false`.
1. Kada je treći argument `true`, piramida treba da ima vrh usmeren ka dole.
1. Red vrha treba da ima jedan uzorak, a svaki sledeći red mora imati dva više uzorka od prethodnog.
1. Svaki red treba da počinje sa dovoljno razmaka kako bi centralni uzorak svakog reda bio u jednom staklenom redu, i ne sme imati razmake na kraju svakog reda.
1. Piramida treba da počinje i završava sa znakom novog reda (newline).

Na primer, kuitisha `pyramid("o", 4, false)` trebalo bi da vrati ove rezultate:

```js

   o
  ooo
 ooooo
ooooooo

```

function pyramid(char, count, isInverted) {
    const rows = []
    for (let i = 1; i <= count; i++) {
        if (isInverted) {
            rows.unshift(" ".repeat(count - i) + char.repeat(2 * i - 1))
        } else {
            rows.push(" ".repeat(count - i) + char.repeat(2 * i - 1))
        }        
    }
    return "\n" + rows.join("\n") + "\n";
}

console.log(pyramid("#", 10, false))
