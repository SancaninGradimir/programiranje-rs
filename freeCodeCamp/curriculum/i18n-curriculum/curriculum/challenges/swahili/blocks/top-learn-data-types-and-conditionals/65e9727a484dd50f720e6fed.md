# --description--

U bilo kom programskom jeziku, morate uporediti vrednosti kako biste doneli odluke. Na primer, ako je vreme sunčano, idete napolje; inače ostajete kod kuće. Drugi primer je proveravanje da li korisnik ima dovoljno poena za savladavanje nivoa u igri.

Informacija `if-else` se koristi za donošenje odluka u JavaScript. Često se koristi zajedno sa kriterijumima poređenja radi upoređivanja vrednosti i donošenja odluka na osnovu rezultata.

Primer izveštaja za if je:

```javascript
let x = 5;

if (x > 3) {
  console.log("x is greater than 3");
} else {
  console.log("x is less than or equal to 3");
}
```

U ovom primeru, razlika od `x` se poredi sa brojem `3` koristeći parametar `>`. Ako je `x` veće od `3`, izvršiće se prvi deo koda. Inače, izvršiće se drugi deo koda.
