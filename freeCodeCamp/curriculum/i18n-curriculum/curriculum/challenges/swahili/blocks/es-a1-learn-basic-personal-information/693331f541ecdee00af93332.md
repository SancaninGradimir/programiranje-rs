# --description--

Kiswahili često koristi reči `cuál` i `cómo` da bi stvorio uobičajena pitanja.

`¿Cuál ... ?` se koristi za postavljanje pitanja **specifičnih informacija**, posebno pre glagola `ser`. Primer:

- `¿Cuál es tu nombre?` – Kako se zoveš?

- `¿Cuál es tu número de teléfono?` – Koji je vaš broj telefona?

`¿Cómo ... ?` koristi se za pitanje **kako** nešto radi, piše se ili izgovara. Primer:

- `¿Cómo te llamas?` – Kako se zoveš?

- `¿Cómo se escribe?` – Kako se piše?

# --explanation--

`Cuál` se koristi za traženje specifičnih informacija, posebno pre glagola `ser`.

`Es` potiče od glagola `ser` i odgovara `nacionalidad`.

`Tu` to pokazuje da govornik pita jednu osobu.

Ovo pitanje prati uobičajenu strukturu `¿Cuál es tu` + imenica? Na primer:

- `¿Cuál es tu nombre?` - Kako se zoveš?

- `¿Cuál es tu nacionalidad?` - Kakvo je vaše državljanstvo?
