# --description--

Naučimo više o metodi `querySelectorAll()`. Ovu metodu možete koristiti za pronalaženje liste svih DOM elemenata koji odgovaraju određenom CSS selektoru.

Ovo je kako možete nazvati putanju za `querySelectorAll()`:

```js
document.querySelectorAll(selectors);
```

Kliknite na objekat `document` i unesite niz slova sa opcijama CSS kao argument. Argument mora biti važeći niz selektora CSS. Inače, će biti izbačena greška `SyntaxError`.

`querySelectorAll()` vraća element `NodeList`, skup čvorova koji odgovara selektoru CSS koji je naveden.

Lista će sadržati jedan `Element` objekat za svaki element koji odgovara CSS selektoru. Ako nema podudaranja, lista će biti prazna.

Elementi će biti poređani onim redosledom kojim se pojavljuju u HTML dokumentu. Na primer, možete pronaći sve elemente određenog tipa:

```js
document.querySelectorAll("div");
```

Svi elementi sa određenom klasom:

```js
document.querySelectorAll(".rounded");
```

Svi elementi sa određenim identifikatorom:

```js
document.querySelectorAll("#logo");
```

Ili svi elementi sa određenim atributom, kao što su svi linkovi koji vode korisnike na određeni URL:

```js
document.querySelectorAll("a[href='https://www.freecodecamp.org/']");
```

Takođe možete koristiti složenije CSS selektore, kao što je ovaj, koji pronalazi sve stavke neuređene liste unutar neuređene liste sa klasom `ingredients`:

```js
document.querySelectorAll("ul.ingredients li");
```

Pošto ova metoda vraća kolekciju čvorova, možete sačuvati tu kolekciju u promenljivoj i koristiti je u aplikacijama za rad sa pojedinačnim elementima, kao što se vidi u ovom primeru sa opštom sintaksom:

```js
const matches = document.querySelectorAll(selectors); 
```

Na primer, ako imate neuređeni spisak sa tri stavke:

```html
<ul class="ingredients">
  <li>Flour</li>
  <li>Cheese</li>
  <li>Water</li>
</ul>
```

Možete pozvati metodu ``querySelectorAll()`` upoređujući sve elemente liste unutar neuređenog spiska sa klasom ``ingredients`` i postavljanjem vrednosti `return` na parametar ``matches``:

```js
const matches = document.querySelectorAll("ul.ingredients li");
```

Ovo vraća kolekciju od tri čvora. Svaki čvor predstavlja element liste:

```js
// NodeList(3)
{
  0: `<li>Flour</li>`,
  1: `<li>Cheese</li>`,
  2: `<li>Water</li>`,
  length: 3,
}
```

Možete raditi sa ovom kolekcijom kao što radite sa bilo kojim redom JavaScript.

Na primer, možete ga prikazati u konzoli za `console.log()`:

```js
console.log(matches);
```

Takođe možete proveriti dužinu kolekcije za svojstvo `length`:

```js
console.log(matches.length);
```

Rezultat će biti celobrojni broj koji predstavlja broj čvorova u kolekciji:

```md
3
```

Takođe možete pronaći pojedinačne komponente kolekcije sa njihovim odgovarajućim modelom. Prva komponenta će biti u šablonu `0`:

```js
console.log(matches[0]);
console.log(matches[1]);
console.log(matches[2]);
```

Ovo su rezultati:

```html
<li>Flour</li>
<li>Cheese</li>
<li>Water</li>
```

Elementi se štampaju/prikazuju jedan po jedan na konzoli.

Takođe možete ponoviti na delovima koristeći petlju `for`:

```js
for (let i = 0; i < matches.length; i++) {
  console.log(matches[i]);    
}
```

Rezultati će biti potpuno isti u ovom slučaju. Međutim, ovo može biti previše snažno za rad sa svim komponentama sklopova/montaže.

Način za `querySelectorAll()` u JavaScript je moćan alat za odabir više elemenata na osnovu njihovih opcija/izbora CSS. Na ovaj način, možete izabrati specifične elemente unutar vaših veb stranica i koristiti ih po potrebi.
