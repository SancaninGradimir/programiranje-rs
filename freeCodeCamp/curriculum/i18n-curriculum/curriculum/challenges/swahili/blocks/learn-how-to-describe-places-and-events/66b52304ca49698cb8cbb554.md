# --explanation--

U vremenu `Present Simple`, forma trećeg lica množine glagola obično se završava sa `-s` ili `-es`. Na primer:

- `He codes every day.` - Глагол `codes` završava se sa `-s` zato što se odnosi na `he`.

- `She fixes bugs in the system.` - Glagol `fixes` ističe na `-es` zato što se odnosi na `she`.

`Concern` znači imati anksioznost ili učiniti da neko oseća zabrinutost zbog nečega. Na primer:

- `The security of the network concerns the IT team.` - IT tim je zabrinut za bezbednost mreže.

- `It concerns me that the software isn't updating correctly.` - Zabrinati ste zbog programa koji nije ažuriran kako je potrebno.
