# --description--

Drugi način za postavljanje prozirnosti elementa je korišćenje <dfn>alpha channel</dfn>. Kao svojstvo `opacity`, alpha kanal kontroliše kako boja može biti vidljiva ili zaklonjena.

Postavili ste neprozirnost rukava koristeći boju sa imenom i svojstvom `opacity`, ali možete dodati alfa kanal za druga svojstva boja CSS.

U skladu sa pravilom `.sleeve`, uklonite atribut sa vrednošću `opacity`.

# --hints--

Tvoja pravila za `.sleeve` od CSS ne bi trebalo da imaju atribut i vrednost `opacity`.

```js
assert(!new __helpers.CSSHelp(document).getStyle('.sleeve')?.opacity);
```
