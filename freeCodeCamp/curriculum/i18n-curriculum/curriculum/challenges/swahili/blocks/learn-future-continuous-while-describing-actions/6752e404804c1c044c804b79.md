# --explanation--

`Future Continuous` opisuje akciju koja će se dešavati u nekom trenutku u budućnosti. U ovom slučaju, radi se o održavanju koje će se sprovesti u petak uveče/noć, obaveštavajući korisnike o tome unapred.
