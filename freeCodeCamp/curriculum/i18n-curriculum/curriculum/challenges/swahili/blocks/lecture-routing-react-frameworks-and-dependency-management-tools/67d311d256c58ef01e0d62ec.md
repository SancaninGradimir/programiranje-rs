# --description--

Sada, koristio si React da bi napravio korisnički interfejs. Ako ti treba više funkcija kao što je planiranje ruta, onda si bio primoran da dodaš biblioteku treće strane poput React Router kako bi mogao prebacivati između različitih prikaza.

Ali šta se dešava kada želite da napravite kompletnu web aplikaciju? Možete koristiti React za front-end i koristiti Node i Express za back-end logiku ako želite da ostanete samo sa JavaScript. Ili možete koristiti drugi jezik kao što su Go, Python, ili Java za back-end.

Iako su sve ovo moguće opcije, postoje situacije kada biste možda želeli da koristite funkcionalni sistem React umesto toga. Funkcionalni sistemi React nude funkcije kao što su planiranje ruta, poboljšanje slika, prijem podataka, verifikacija identiteta i još mnogo toga. To znači da možda nećete morati da inicijalizujete zasebne *front-end* i *back-end* aplikacije za određene namene.

Detaljno ćemo istražiti Next.js, koji je popularan framework za React. Jedna od glavnih karakteristika Next.js-a je rutiranje bazirano na sistemu datoteka (file system). Ovaj sistem rutiranja uključuje podršku za dinamičke rute podataka, paralelne rute podataka, rukovače (handlers) za rute podataka, preusmeravanje, internacionalizaciju jezika i više.

Evo primera kreiranja obradnika specifičnog zahteva:

```js
export async function GET() {
  const res = await fetch("https://example-api.com");
  const data = await res.json();

  return Response.json({ data });
}
```

Možete definisati handler-e za rute podataka kao što su GET ili POST zahtevi u fajlu pod nazivom `route.js` unutar direktorijuma `app/api`.

Još jedna funkcija Next.js-a je automatsko poboljšanje slika i fontova.

Ovo je primer kako raditi sa delom `Image` unutar fajla `page.js`:

```jsx
import Image from "next/image";

export default function Page() {
  return (
    <Image src="link-to-image-goes-here" alt="descriptive-title-goes-here" />
  );
}
```

Sekcija ``Image`` dodaje nativnu karakteristiku `HTML `img``, omogućavajući brže učitavanje stranica i optimizaciju veličine. To znači da će slike biti učitane samo kada uđu u vidokrug, a sekcija ``Image`` automatski će servirati slike odgovarajuće veličine za svaki uređaj.

Iako su ovo samo neke funkcionalnosti Next.js-a, postoji mnogo više funkcija koje možete koristiti za izgradnju potpunih i robusnih veb aplikacija. Postoje i drugi radni okviri kao što je React poput Remixa koji će pružiti jednaku sposobnost za izgradnju potpunih modernih veb aplikacija.

Iako biblioteke i sistemi za rada JavaScript se često menjaju, važno je da imate razumevanje opcija koje su vam dostupne i naučite o prednostima i manama svakog sistema.
