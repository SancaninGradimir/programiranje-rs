# --description--

Na ovom radionici, naučićete kako da počnete sa osnovama Tailwind CSS-a gradeći komponentu poruke o grešci.

Za prve korake, fokusiraćete se na kreiranje početnog HTML markupa.

Unutar elementa `body`, dodajte element `div`. Zatim unutar tog elementa `div`, dodajte element `p`.

# --hints--

Trebalo bi da imate element `div`.

```js
assert.exists(document.querySelector("div"));
```

Trebalo bi da imate element `p`.

```js
assert.exists(document.querySelector("p"));
```

Vaš `p` element bi trebalo da bude unutar `div` elementa.

```js
assert.exists(document.querySelector("div p"));
```
