# --description--

```css
/* rule 1 */
.class.second-class {
  font-size: 12px;
}

/* rule 2 */
.class .second-class {
  font-size: 24px;
}
```

Ovde, Pravilo 1 i Pravilo 2 imaju jednaku validnost. Pravilo 1 koristi sekvencijalni spajanje mehanizam (bez razmaka), a Pravilo 2 koristi kombinaciju nasleđivanja (prazan prostor). Međutim, oba pravila imaju dve klase, a sama oznaka kombinacije ne dodaje nikakvu validnost.

```css
/* rule 1 */
.class.second-class {
  font-size: 12px;
}
/* rule 2 */
.class > .second-class {
  font-size: 24px;
}
```

Ovaj primer pokazuje isto. Iako Pravilo 2 koristi kombinaciju deteta (`>`), ovo ne menja vrednost veštine. Oba pravila i dalje imaju dve klase, pa imaju jednaku vrednost veštine.

*Napomena:* Nije sve što povećava veštine

Kada se porede kandidati, možete naići na posebne oznake svetskog kandidata (`*`) zajedno sa kombinacijom (`+`, `~`, `>`, i prazno mesto). Ove oznake same po sebi ne dodaju nikakvu veštinu.
