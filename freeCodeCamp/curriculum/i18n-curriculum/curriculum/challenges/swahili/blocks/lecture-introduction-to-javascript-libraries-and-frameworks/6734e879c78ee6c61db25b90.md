# --description--

Komponente su osnovni gradivni elementi React aplikacija koji omogućavaju programerima da složen korisnički interfejs podele na manje, lakše upravljive celine, što olakšava razvoj i održavanje velikih aplikacija.

Dva tipa komponenti u React su funkcionalne komponente i komponente klase. U modernom React, programeri će koristiti funkcionalne komponente, a svi primeri koje istražujemo danas biće funkcionalne komponente.

Sa višeg gledišta, možete zamisliti delove kao funkcije za JavaScript koje vraćaju komponente koje opisuju korisnički interfejs.

Ovaj korisnički interfejs je definisan korišćenjem JSX, to je sintaktičko proširenje za JavaScript slično HTML, ali vam omogućava da pišete komponente korisničkog interfejsa na opisniji način.

Istražite primer dela React:

```jsx
function Greeting() {
  const name = "John"
  {/* The result will be Hello John*/}
  return <h1 className="title">Hello {name}</h1>;
}
```

U ovom primeru napravili smo komponentu pod nazivom `Greeting`. Vitičaste zagrade `{}` unutar oznake `h1` omogućavaju nam da ubacimo JavaScript izraz i tako pristupimo vrednosti promenljive `name` unutar elementa `h1`.

Pia tunatumia `className` iitwayo `title` kwenye kipengele cha `h1`.

Ali zašto koristimo `className` umesto `class`, kao što je slučaj sa standardnim komponentama od HTML?

To je zato što je u JavaScript-u `class` rezervisana reč. Zato umesto nje moramo koristiti `className`.

Takođe koristimo komentare u JSX-u da bismo prikazali očekivani rezultat. Važno je razumeti da možete koristiti uobičajenu sintaksu komentara, ali ona mora biti obuhvaćena vitičastim zagradama da bi radila.

```jsx
{/* Block Comments */}
```

Još jedna stvar koju ste možda primetili jeste da koristimo veliko početno slovo u nazivu komponente. Ali zašto ne možemo koristiti samo mala slova?

Ovo je zato što React prepoznaje delove sa velikim slovima kao posebne delove, dok se elementi sa malim slovima smatraju običnim elementima za HTML.

Kada React naiđe na oznaku malog slova, kao što su `<div>` ili `<span>`, smatra ga standardnom funkcijom od HTML. Međutim, ako ime komponente počinje velikim slovom, React će ga prepoznati kao korisnički generisan deo i prikazati ga shodno tome. Ova razlika pomaže React da razlikuje između ugrađenih oznaka za HTML i komponenti koje kreirate.

Da bismo koristili ovaj deo od `Greeting` u našem programu, napisali bismo nešto slično ovome:

```jsx
<Greeting />
```

Ovo će prikazati element `h1` sa tekstom `Hello John` na stranici. Ali, pažljivo pogledajte sintaksu ovde. Kada koristimo deo, završava se povratnom kosom crtom i zatim znakiem veće od.

Kada radite sa JSX-om, sve oznake i upotrebe komponenti moraju biti eksplicitno zatvorene. Dakle, ako komponenta ili oznaka nema detlji, onda je morate eksplicitno zatvoriti kao što je prikazano ovde:

```jsx
<Greeting /> {/* /> is required */}
```

Do sada smo gledali kako prikazati jedan element od `h1`. Ali možete prikazati više elemenata.

Istražite sledeći primer koda ovde:

```jsx
function Greeting() {
  const name = "John";
  {/* This will throw an error */}
  return <h1>Hello {name}</h1> 
  <p>Nice to meet you.</p>
}
```

Pokušavamo da dodamo još jednu rečenicu sa `Nice to meet you`, ali se ne prikazuje dobro na stranici.

Izgleda da umesto toga postoji poruka o grešci. Poruka o grešci glasi `Adjacent JSX elements must be wrapped in an enclosing tag.`

Razlog za dobijanje te poruke o grešci je da mnogo susednih elemenata mora biti pokriveno roditeljskim elementom. Iako je lako pokriti elemente `h1` i `p` unutar `div`, postoji drugi način da se ta greška ukloni.

Delovi React se koriste za povezivanje komponenti. Ovo je modifikovani primer, izgledaće ovako:

```jsx
function Greeting() {
  const name = "John";
  return (
    <Fragment>
      <h1>Hello {name}</h1>
      <p>Nice to meet you.</p>
    </Fragment>
  );
}
```

Takođe možete izabrati da koristite prazne JSX tagove koje se mogu koristiti kao skraćeni oblik za komponente:

```jsx
function Greeting() {
  const name = "John";
  return (
    <>
      <h1>Hello {name}</h1>
      <p>Nice to meet you.</p>
    </>
  );
}
```

U narednim lekcijama, nastavićemo da učimo više o tome kako raditi sa komponentama i JSX-om. Ali za sada, naučili ste čvrst uvod u izgradnju korisničkog interfejsa korišćenjem komponenti, postavljajući čvrste temelje za ono što sledi.
