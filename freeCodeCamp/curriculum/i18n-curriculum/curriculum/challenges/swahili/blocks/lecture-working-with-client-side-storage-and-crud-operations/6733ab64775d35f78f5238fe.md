# --description--

CRUD je akronim koji predstavlja Create, Read, Update i Delete. Ovo su četiri operacije trajnog skladištenja.

Trajno skladištenje se odnosi na čuvanje podataka na način koji omogućava pristup čak i nakon gašenja struje ili ponovnog pokretanja uređaja.

Razumevanje načina na koji funkcionišu osnovne CRUD operacije od suštinskog je značaja za razvoj veb aplikacija, jer predstavljaju osnovu za rad sa bazama podataka i izradu aplikacija u kojima korisnici mogu da dodaju, pregledaju, menjaju i brišu podatke.

Sada, hajde da proučimo svaki deo CRUD u većoj detaljnosti

- Create se odnosi na kreiranje novih podataka. Na primer, u veb aplikaciji to može biti trenutak kada korisnik doda novu objavu na blog.

- Read je operacija u kojoj se podaci preuzimaju iz baze podataka. Na primer, kada posetite objavu bloga ili pregledate svoj profil na veb-sajtu, izvršavate operaciju čitanja kako biste dobili i prikazali podatke sačuvanije u bazi podataka.

- Ažuriranje podrazumeva menjanje postojećih podataka u bazi podataka. Primer je uređivanje objave na blogu ili ispravljanje informacija sa vašeg profila.

- Brisanje je operacija koja briše podatke iz baze podataka. Na primer, kada tražite blog objavu ili nalog, izvršavate brisanje operaciju.

CRUD se koristi pri radu sa bazama podataka, korisničkim interfejsima i RESTful API-jima. RESTful API predstavlja skup pravila za izgradnju veb servisa koji omogućavaju klijentu da komunicira sa bazom podataka ili pozadinskim sistemom izvršavanjem CRUD operacija putem standardnih HTTP metoda.

HTTP je skraćenica za Hypertext Transfer Protocol i predstavlja osnovu prenosa podataka na internetu. HTTP definiše metode koje određuju koje se radnje mogu izvršavati nad resursima. Najčešće metode su `GET`, `POST`, `PUT`, `PATCH` i `DELETE`.

Naučićete više o RESTful API-jima i HTTP-u u narednim lekcijama, ali ovde je kratak pregled kako se CRUD uklapa sa različitim HTTP metodama.

- `POST` koristi se za kreiranje novih resursa.

- `GET` Koristi se za dobijanje ili čitanje podataka.

- `PUT` Koristi se za korekciju resursa ili za njihovu kompletnu zamenu.

- `PATCH` Koristi se za ispravljanje dela resursa.

- `DELETE` Koristi se za uklanjanje resursa.

Ovde je primer kako se CRUD operacije mogu predstaviti u kodu koristeći jednostavan niz podataka u JavaScript:

```js
let items = [];

// Create
function createItem(item) {
  items.push(item);
}

// Read
function readItems() {
  return items;
}

// Update
function updateItem(index, newItem) {
  items[index] = newItem;
}

// Delete
function deleteItem(index) {
  items.splice(index, 1);
}

// Example Usage
createItem('Book');
console.log(readItems()); // ['Book']
updateItem(0, 'Magazine');
console.log(readItems()); // ['Magazine']
deleteItem(0);
console.log(readItems()); // []
```

U ovom primeru:

- Kreiramo element dodavanjem u niz podataka.

- Čita elemente za vraćanje reda podataka.

- Korigujemo element menjanjem atributa niza podataka prema određenom obrascu.

Tražimo element uklanjanjem iz niza podataka koristeći `splice()`.
