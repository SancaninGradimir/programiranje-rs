# --description--

Aplikacioni middleware je srednji sloj koji se povezuje sa primerom programa Express. Obično se koristi za obradu zahteva koji zahtevaju obradu kroz više kanala podataka. Može se koristiti opštije ili na specifičan način.

Ovaj middleware je napisan korišćenjem `app.use()` ili specifičnih putanja podataka kao što su `app.get()`, `app.post()`, itd. Radi za svaki dolazni zahtev osim ako nije ograničen specifičnim putanjama podataka.

```js
const express = require('express')
const app = express()

// Application-level middleware that logs all incoming requests
app.use((req, res, next) => {
  console.log('Request received at:', new Date())
  next(); // Passes control to the next middleware
})

app.get('/', (req, res) => {
  res.send('Hello World!')
})

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* `app.use()`: Ovo je middleware na nivou programa. Beleži svaki zahtev poslat serveru, bez obzira na putanju podataka.
* Middleware radi za svaki dolazni zahtev, pružajući serveru detalje kao što su trenutni datum i vreme.

Aplikacioni middleware je važan za funkcije koje zahtevaju izvršavanje na više ili svim putevima podataka u aplikaciji. Ovo može uključivati logovanje, autentifikaciju identiteta i rukovanje greškama.
