# --interactive--

Nedefinisani element opisa, ili skraćeno element `u`, koristi se za predstavljanje teksta unutar linije koji sadrži nedefinisane meta-podatke.

Ovde je primer korišćenja elementa `u` za prikaz različitih pravopisnih grešaka:

:::interactive_editor

```html
<p>
  You can use the unarticulated annotation element to highlight
  <u>inccccort</u> <u>spling</u> <u>issses</u>.
</p>
```

:::

U ovom primeru, reči `incorrect`, `spelling`, i `issues` su pogrešno napisane. Podrazumevani stil za element `u` je crna linija ispod teksta.

U HTML4, elementi za `u` je korišćen u svrhu stilizovanja. Ali u HTML5, elementi za `u` bi trebalo koristiti samo da pokaže da tekst sadrži ne-tekstualne informacije.

Ako želite da formatirate deo teksta podvučenjem, trebalo bi koristiti CSS umesto HTML.

Elementi linije koja je prošla, ili skraćeno elementa `s`, treba da se koristi za predstavljanje kada tekst nije tačan ili više nije relevantan. Evo primera korišćenja elementa `s` za prikaz brisanja aktivnosti:

:::interactive_editor

```html
<p><s>Tomorrow's hike will be meeting at noon.</s></p>

<p>Due to unforeseen weather conditions, the hike has been canceled.</p>
```

:::

U ovom primeru, prva rečenica je precrtana jer je putovanje otkazano zbog vremenskih razloga.

Element `s` ne bi trebalo koristiti samo za prikaz promena u dokumentu. Najprikladniji elementi u tom slučaju su element za izbrisani tekst i element za dodati tekst.

Element ``ruby`` predstavlja mali tekst koji se prikazuje iznad ili ispod glavnog teksta. Obično se koristi za prikaz izgovora istočnoazijskih znakova. Evo primera elementa ``ruby`` sa stranice dokumentacije web-a MDN:

:::interactive_editor

```html
<ruby> 明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp> </ruby>
```

:::

Elementi za `rp`, ili elementi rezervnog zagrada/fallback-a za rubij, koriste se kao alternativa za pretraživače koji nemaju podršku za prikaz detalja rubija.

Element ``rt``, ili rubinski tekstualni element, koristi se za prikaz teksta ruby anotacija. Ovaj tekst se obično koristi za izgovor ili objašnjenja prevoda u sistemima pisanja Istočne Azije.

Iako element `ruby` može se koristiti za druge vrste opisa, njegova najčešća upotreba je za dizajn pisma Istočne Azije.
