# --description--

Prvi važan koncept koji treba da razumete da biste bili uspešni u CSS-u jeste model kutije. Nije komplikovan, ali njegovo preskakanje sada će kasnije izazvati mnogo frustracija.

Sve na veb stranici je pravougaona kutija. Te kutije mogu sadržati druge kutije i mogu biti postavljene jedna pored druge. Opštu predstavu o tome kako ovo funkcioniše možete steći tako što ćete dodati ivicu svakom elementu na stranici ovako:

```css
* {
  border: 1px solid red;
}
```

# --assignment--

Dodajte ivicu svakom elementu na stranici i pogledajte kako su kutije raspoređene.
