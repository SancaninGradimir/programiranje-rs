# --explanation--

James koristi model `Second Conditional` (`It'd be great if we had...`) za razgovor o nerealnom ili zamišljenom stanju.

Struktura `Second Conditional` je: `It would` + osnovni glagol... `if` + subjekat + glagol u `Simple Past`. Koristi se za opisivanje nerealnih ili maštovitih situacija u sadašnjosti ili budućnosti. Primer:

`We would finish earlier if we had more team members.` - Ovo znači da nemaju više članova tima, ali ako bi ih imali, mogli bi završiti ranije.
