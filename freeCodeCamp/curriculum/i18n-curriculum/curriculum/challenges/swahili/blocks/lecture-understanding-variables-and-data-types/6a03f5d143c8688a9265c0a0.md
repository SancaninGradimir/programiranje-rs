# --description--

Na prošloj lekciji ste naučili o uobičajenim tipovima podataka u Python, uključujući tipove stringova, cele brojeve i float. Kada pravite svoje programe, moraćete da naučite kako da proverite tip varijable.

Evo primera varijable:

```py
developer = 'Devin'
```

Da biste videli kakav je tip `developer`, možete koristiti funkciju `type()` na sledeći način:

```py
developer = 'Devin'

print(type(developer)) # <class 'str'>
```

Rezultati `<class 'str'>` znače da `developer` je tip sekvence slova.

Ako ne pružite nijedan argument za `type()`, primaćete poruku o grešci kao što je sledeće:

```md
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: type() takes 1 or 3 arguments
```

Ovde su sve vrste podataka koje ste naučili do sada, zajedno sa njihovim tipovima, u terminalu:

```python
my_integer_var = 10
print(type(my_integer_var))  # <class 'int'>

my_float_var = 4.50
print(type(my_float_var))  # <class 'float'>

my_string_var = 'hello'
print(type(my_string_var))  # <class 'str'>

my_boolean_var = True
print(type(my_boolean_var))  # <class 'bool'>

my_set_var = {7, 'hello', 8.5}
print(type(my_set_var))  # <class 'set'>

my_dictionary_var = {'name': 'Alice', 'age': 25}
print(type(my_dictionary_var))  # <class 'dict'>

my_tuple_var = (7, 'hello', 8.5)
print(type(my_tuple_var))  # <class 'tuple'>

my_range_var = range(5)
print(type(my_range_var))  # <class 'range'>

my_list = [22, 'Hello world', 3.14, True]
print(type(my_list)) # <class 'list'>

my_none_var = None
print(type(my_none_var))  # <class 'NoneType'>
```

Postojeće će trenutke u vašoj aplikaciji kada ćete morati da proverite da li je određena varijabla određen tip pre nego što izvršite operaciju na njoj. Ovde funkcija `isinstance()` postaje korisna.

Ovde je primer varijable sa nizom slova kojoj je dodeljena vrednost:

```py
account_balance = '12'
```

Ako pokušate da izvršite matematički izraz poput deljenja koristeći varijablu ``account_balance``, onda ćete dobiti poruku o grešci.

```py
account_balance = '12'

account_balance / 2

# Traceback (most recent call last):
#   File "<stdin>", line 1, in <module>
# TypeError: unsupported operand type(s) for /: 'str' and 'int'
```

Da biste proverili da li je `account_balance` ceo broj, možete koristiti funkciju `isinstance()` na sledeći način:

```py
account_balance = '12'

isinstance(account_balance, int) # False
```

Ugrađena funkcija `isinstance()` omogućava vam da proverite da li varijabla odgovara određenom tipu podataka. Uzima predmet i tip koji želite da proverite, zatim vraća vrednost tačno ili netačno (boolean). U ovom slučaju, pošto je `account_balance` niz znakova (string), on će vratiti `False`.

Funkcija `isinstance()` takođe vam omogućava da pregledate više tipova istovremeno.

Ovo je primer za proveru da li `account_balance` je `int` ili `float`:

```py
account_balance = 12
isinstance(account_balance, (int, float)) # True
```

U ovom primeru, `account_balance` je integer, stoga `isinstance()` vraća `True`. Ako bi `account_balance` bilo `12.0`, `isinstance()` i dalje bi koristio `True` jer proveravate integer ili float.

U nadolazećim radionicama i laboratorijama, koristićeš funkcije `type()` i `isinstance()` da bi osigurao da tvoje varijable imaju ispravne tipove podataka pre obavljanja operacija na njima.
