# --description--

Permutacija je raspored elemenata određenim redosledom. Na primer, 3124 je jedna moguća permutacija brojeva 1, 2, 3 i 4. Ako se sve permutacije poređaju numeričkim ili abecednim redom, taj redosled nazivamo leksikografskim.

<div style='text-align: center;'>012 021 102 120 201 210</div>

Koja je `n`-ta leksikografska permutacija cifara 0, 1, 2, 3, 4, 5, 6, 7, 8 i 9?
