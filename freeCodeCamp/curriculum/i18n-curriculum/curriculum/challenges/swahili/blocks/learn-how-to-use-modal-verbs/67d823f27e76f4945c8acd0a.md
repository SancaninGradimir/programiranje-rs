# --explanation--

`Should` je modalni glagol koji izražava savet, predloge ili najbolji način za nešto — ali nije obaveza.

- `You should update your password regularly.` - Dobra je ideja da promenite svoju lozinku redovno radi sigurnosti.

- `You should check for errors in the code before running it.` - Dobra praksa je pronaći greške pre izvršavanja koda.

Pošto Jake kaže `we should focus on the critical systems first`, on daje ozbiljan savet, ali tim nema strogu obavezu da to uradi.
