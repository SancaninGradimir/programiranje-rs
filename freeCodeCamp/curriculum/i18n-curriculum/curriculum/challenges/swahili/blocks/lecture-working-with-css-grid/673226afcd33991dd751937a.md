# --interactive--

Svojstva za `grid-column` na `grid-row` omogućavaju vam da definišete horizontalne i vertikalne pozicije grid segmenta unutar rasporeda grida.

Drugim rečima, obe vam omogućavaju da kontrolišete gde deo mreže počinje i gde se završava, referenciranjem linija mreže. Ove linije mreže su granice koje razdvajaju redove i kolone koje ste već pomenuli korišćenjem atributa `grid-template-rows` i `grid-template-columns`.

Pogledajmo sintaksu svojstava `grid-row` i `grid-column`, zajedno sa praktičnim primerom.

Hii ndiyo sintaksia ya sifa ya `grid-row`:

```css
grid-row: <start-line> / <end-line>;
```

И иа `grid-column`:

```css
grid-column: <start-line> / <end-line>;
```

`<start-line>` je linija mreže na kojoj stavka počinje, a `<end-line>` je linija na kojoj se završava. Obe se broje počev od 1, a ne od 0.

Napomena, linije mrežice za redove generišu se prema broju redova definisanih u atributu `grid-template-rows`. Isto važi i za kolone za atribut `grid-template-columns`.

Kao praktičan primer, počnimo sa linijama koje kreira pretraživač kada koristi atribute za `grid-template-rows` i `grid-template-columns`.

Ovo je primer mreže sa 4 kolone i 3 reda:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}
```

:::

Provera alata za mrežu (klasa `grid`) pokazuje da su svaki red i kolona okruženi sa dve linije – početna linija na početku reda ili kolone, i završna linija na kraju reda ili kolone.

Možeš ciljati ove linije pomoću svojstava `grid-row` i `grid-column` da odrediš gde treba postaviti stavku.

Ovo je kako možete učiniti da prvi deo mreže zauzme dve prve kolone:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid">
  <div class="item1">1</div>
  <div class="item2">2</div>
  <div class="item3">3</div>
  <div class="item4">4</div>
  <div class="item5">5</div>
  <div class="item6">6</div>
  <div class="item7">7</div>
  <div class="item8">8</div>
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;
}

.grid > div {
  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;
}

.item1 {
  grid-column: 1 / 3;
}
```

:::

U vezi ovoga, kažeš da prvi deo mrežice treba da počne na koloni 1 i završi se pre kolone 3.

Prvi deo će sada uzeti dve kolone, a četvrti deo će biti pomeren u drugi red. Super!

```html
  <div class="item9">9</div>
  <div class="item10">10</div>
  <div class="item11">11</div>
  <div class="item12">12</div>
</div>
```

```css
.grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 equal columns */
  grid-template-rows: repeat(3, 100px); /* 3 equal rows */
  gap: 10px;

  display: grid;
  place-items: center;
  background: crimson;
  color: white;
  font-size: 4rem;

  grid-column: 1 / 3;
  grid-row: 1 / 3;

```

:::


:::

Možete nastaviti da koristite ovu metodu za bilo koji komad na mreži i postaviti ih gde god želite.
