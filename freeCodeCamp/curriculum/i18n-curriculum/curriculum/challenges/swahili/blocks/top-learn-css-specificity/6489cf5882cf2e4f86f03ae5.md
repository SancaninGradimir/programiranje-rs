# --description--

Sad, promenimo stvari malo:

```html
<!-- index.html -->

<div class="main">
  <div class="list" id="subsection"></div>
</div>
```

```css
/* rule 1 */
#subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

U gornjem primeru, iako pravilo 2 ima više determinatora klase predmeta nego ID determinatore, pravilo 1 je specifičnije jer ID nadjačava klasu predmeta.
