# --description--

Trenutno, sve što imaš je u tvom `server.js` fajlu. Ovo može dovesti do koda koji je težak za održavanje i proširivanje. Kreiraj 2 nova fajla: `routes.js` i `auth.js`

Oba treba da počnu sa sledećim kodom:

```js
module.exports = function (app, myDataBase) {

}
```

Sada, na vrhu tvog server fajla, uvezi ove fajlove ovako: `const routes = require('./routes.js');` Čim uspostaviš dobru konekciju sa bazom podataka, inicijalizuj svaki ovako: `routes(app, myDataBase)`

Konačno, uzmi sve rute sa tvog servera i nalepi ih u nove fajlove, zatim ih ukloni iz server fajla. Takođe uzmi `ensureAuthenticated` funkciju, jer je kreirana specifično za rute. Sada, moraćeš da dodaš ispravno zavisnosti koje se koriste, kao što je `const passport = require('passport');`, na samom vrhu, iznad linije za izvoz u tvom `routes.js` fajlu.

Nastavi da ih dodaješ dok više nema grešaka, i tvoj server fajl nema više ruta (**osim rute u catch delu**)!

Uradi isto u svom `auth.js` fajlu za sve stvari vezane za autentifikaciju kao što su serijalizacija i konfiguracija local strategije, zatim ih obriši iz server fajla. Uveri se da dodaš zavisnosti i pozoveš `auth(app, myDataBase)` u serveru na istom mestu.

Pošalji svoju stranicu kada vidiš da si uradio dobro. Ako naiđeš na greške, možeš <a href="https://forum.freecodecamp.org/t/advanced-node-and-express/567135#clean-up-your-project-with-modules-2" target="_blank" rel="noopener noreferrer nofollow">pogledati primer završenog projekta</a>.
