# --description--

HTML i CSS su dva jezika koja rade zajedno kako bi kreirala sve što vidite kada pregledate nešto na internetu. HTML su sirovi podaci kojima je napravljena veb stranica. Sve reči, linkovi, kartice, liste i dugmad kreiraju se pomoću HTML. CSS je ono što dodaje stil tim jednostavnim elementima. HTML stavlja informacije na veb stranicu, a CSS te informacije postavlja negde, daje im boju, menja font i čini da izgledaju dobro!

# --questions--
    
## --text--

Koje su glavne funkcije HTML i CSS za kreiranje web stranice?

## --answers--

HTML stvara strukturu sadržaja, dok CSS daje mu elemente stila.

---

HTML kreira vizuelnu strukturu, dok CSS priprema sadržaj.

---

CSS kontroliše performanse, dok HTML kontroliše podešavanja.
