# --description--

Ova vežba se fokusira na proveru upotrebe `can` radi demonstracije sposobnosti i upotrebe glagola `try` u trajenom prošlom vremenu.

`Can` koristi se za razgovor o sposobnosti osobe da nešto uradi, kao `I can solve this problem.`

`Try` u kontinuiranom prošlom vremenu (`was`/`were` + glagol u formi `-ing`) pokazuje radnju koja je trajala u prošlosti, kao što `I was trying to fix the issue.`
