# --description--

Ako su dati tajni broj i pokušaj pogađanja, odredi da li je pokušaj tačan.

Rudisha:

- `"higher"` ako je tajni broj veći od pokušaja.
- `"lower"` ako je tajni broj manji od pokušaja.
- `"you got it!"` ikiwa makisio ni sahihi.
