# --description--

U React, može postojati situacija u kojoj vam je potreban direktan pristup funkciji DOM. Tamo su korisne "refs". Refs takođe mogu čuvati promenljive vrednosti, ali *state* je bolji izbor za to.

U normalnom JavaScript, koristili ste metode `getElementById()` i `querySelector()` da biste došli do karakteristika DOM. Ali u React, koristite refs da biste došli do karakteristika u DOM.

Jedna glavna razlika je u tome što, korišćenjem refs-a, ne treba vam identifikatori poput ID-jeva niti klasa objekata za referenciranje komponenti.

Pa, kako možeš da kreiraš i koristiš refs? React pruža dugme sa `useRef()` koje ti omogućava to.

Prvi korak je da prenesemo taj taster od React:

```js
import { useRef } from "react";
```

Zatim treba da napravite promenljivu koja čuva ref sa početnom vrednošću unutar `useRef` poziva, na primer `sectionRef` postavljen na `null`:

```js
const sectionRef = useRef(null);
```

Poslednji korak je dodati ref parametar na vaš JSX element koristeći atribut `ref`:

```jsx
<section ref={sectionRef}>
  {/* Section content */}
</section>
```

Kada ispišete referencu na konzoli, videćete nešto sa trenutnom vrednošću, u ovom slučaju, `null`:

```js
console.log(sectionRef); // { current: null }
```

Takođe možete ispisati trenutnu vrednost na konzolu koristeći atribut `current` da biste videli vrednost direktno:

```js
console.log(sectionRef.current); // null
```

Sledeće vrednosti za referencu zavise od životnog ciklusa komponente.

Na primer, početna vrednost `sectionRef` će uvek biti `null` jer je tako inicijalizovana. Nakon što se komponenta prikaže, vrednost ref-a će biti `section` element na koji je ref povezan.

Ako se deo ukloni, referentna vrednost se vraća na početnu vrednost od `null`.

Uobičajen način prikazivanje reference je fokusiranje na polje za unos tokom prikaza, ili klikom na dugme.

Ovo je kako to radi kada kliknete na dugme:

```jsx
import { useRef } from "react";

const Focus = () => {
  const inputRef = useRef(null);

  const handleFocus = () => {
    if (inputRef.current) {
      inputRef.current.focus();
    }
  };

  return (
    <div>
      <input ref={inputRef} type="text" placeholder="Enter text" />
      <button onClick={handleFocus}>Focus Input</button>
    </div>
  );
};

export default Focus;
```

U gornjem kodu, `inputRef` je kreiran i vezan za karakteristiku `input`. Takođe postoji dugme sa događajem `onClick` koje poziva funkciju `handleFocus`.

Ono što `handleFocus` funkcija radi jeste pozivanje metode `focus()` nad `input` elementom. Zapamtite, pošto je `input` ugrađena komponenta koja dolazi sa React-om, pravi DOM `input` element se nalazi u `current` svojstvu ref-a. Zato pozivate metodu `focus()` pomoću `input.current.focus()`.

Evo najboljih metoda koje biste trebali znati kada koristite reference:

- Koristi refove posebno za interakciju sa DOM. Takođe ih možeš koristiti i za dinamičke podatke, ali 'state' je najbolji izbor za to.

- Nemojte koristiti ref-ove za standardno upravljanje stanjem – to je odgovornost `useState`.

- Proverite da li `ref.current` postoji pre nego što pristupite njegovim svojstvima. Evo kako to ponovo uraditi:

```js
const handleFocus = () => {
  if (inputRef.current) {
    inputRef.current.focus();
  }
};
```

Ovo sprečava greške ako se referenca pristupi pre nego što je dodata sa DOM ili nakon uklanjanja.
