# --interactive--

HTML, koji je skraćenica za Hypertext Markup Language, je značajni jezik za kreiranje web stranica. Kada posetite veb sajt i vidite sadržaj kao što su pasuși, naslovi sekcija, linkovi, slike i video snimci; to je HTML.

Evo malog primera koji koristi funkcije HTML. Pokušajte da promenite nešto tekstualno u editoru i vidite kako se promene pojavljuju u prozoru za prikaz.

:::interactive_editor

```html
<h1>Main heading element</h1>

<p>I am a paragraph element.</p>
```

:::

HTML predstavlja sadržaj i strukturu web stranice putem korišćenja elemenata. Većina elemenata će imati otvarajući i zatvarajući tag. Ponekad se ti tagovi nazivaju početni i krajnji tagovi. Između ova dva taga biće sadržaj. Ovaj sadržaj može biti tekst ili drugi elementi od HTML.

Ovo je još jedan primer funkcije pasusa. Promenite tekst u editoru da piše `I love coding!` i vidite rezultat u prozoru za prikaz.

:::interactive_editor

```html
<p>I am a paragraph element.</p>
```

:::

Sve oznake za otvaranje i zatvaranje počinju sa levom uglastom zagradom (`<`), a završavaju se desnom uglastom zagradom (`>`), sa imenom oznake koje je između ovih uglastih zagrada. Iako imena oznaka HTML ne razmatraju velika ili mala slova, to je široko prihvaćeno konvencionalno i najbolja praksa da se ova imena pišu malim slovima.

Ovo je bliski pogled na oznake za otvaranje i zatvaranje stihova:

```html
<p>
```

```html
</p>
```

Razlika između oznake otvaranja i oznake zatvaranja je znak prednje linije (`/`) koji se postavlja odmah nakon levog trougla na oznaci zatvaranja. Neki elementi za HTML nemaju oznaku zatvaranja. Ovo se naziva prazni element.

Evo primera elementa slike koji je prazan element:

```html
<img>
```

Napomena: Ovaj element slike nema zatvarajući tag i ne sadrži nikakav sadržaj. Prazni elementi ne mogu imati sadržaj i imaju samo otvarajući tag.

Ponekad ćete videti prazne elemente koji koriste `/` pre od `>`, kao što je ovo:

```html
<img />
```

Iako mnogi programeri, poput Prettier-a, biraju da uključuju `/` u prazne elemente, dokumentacija za HTML kaže da prisustvo `/` "ne ukazuje na oznaku početka kao samostaračnu već je neželjena i nema nikakav uticaj."

U stvarnom razvoju, videćete oba tipa, stoga je važno da razumete oba.

Ako želite prikazati sliku, moraćete uključiti nekoliko atributa unutar vašeg elementa slike. Atribut je specifična vrednost koja se koristi za prilagođavanje ponašanja elementa HTML.

Ovo je primer elementa slike sa atributom `src`. Ažurirajte vrednost atributa `src` na `"https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"` i videćete kako se slika menja u dve mačke koje mirno spavaju.

:::interactive_editor

```html
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/running-cats.jpg" />
```

:::

Atribut `src` koristi se za određivanje lokacije te slike. Za elemente slike, najbolji način je uključivanje drugog atributa koji se zove atribut `alt`. Atribut `alt` koristi se za pružanje kratkog teksta koji opisuje sliku.

Ovo je primer elementa slike sa atributima ``src`` i ``alt``. Pokušajte da razbijete sliku ažuriranjem vrednosti ``src`` na ``"https://.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"``. Videćete da slika nestaje i vidljive su samo tekstualne oznake od ``alt``.

:::interactive_editor

```html
<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" alt="Two tabby kittens sleeping together on a couch." />
```

:::

Pa, možda se pitate da li je samo HTML dovoljno za izradu veb stranice. Odgovor je: zavisi od toga. Ako gradite mali projekat za vežbanje koji prikazuje samo tekst i slike, HTML bi moglo biti dovoljno. Međutim, ako kreirate moderno profesionalno veb sajt, biće vam potrebno imati HTML, CSS i JavaScript.

HTML je za sadržaj i strukturu. CSS je za stil. JavaScript je za povećanje interakcije na vašoj veb stranici. Dobar primer ovoga je poređenje HTML, CSS i JavaScript sa kompletnom zgradom.

HTML predstavlja cigle, cement i čelik koji čine zidove. To je osnova koja čini zgradu stabilnom. CSS predstavlja unutrašnji i spoljašnji dizajn koji čini kuću lepšom. JavaScript predstavlja električni i vodeni sistem koji obezbeđuje neprekidno snabdevanje vodom i elektricitetom.
