# --interactive--

Jedna od najzanimljivijih mogućnosti CSS-a jeste upotreba pseudo-elemenata. U ovom kontekstu, „pseudo“ znači „nije stvaran“, pa pseudo-elementi predstavljaju virtuelne elemente koji ne odgovaraju direktno stvarnim HTML elementima. Oni omogućavaju stilizovanje određenih delova elementa ili umetanje sadržaja bez dodavanja novog HTML-a.

Da bi koristio pseudo-element, dodaj ga selektoru izvornog elementa pomoću dve dvotačke (`::`). Selektor može biti bilo koje vrste, na primer selektor klase ili ID-a. Ovo je osnovna sintaksa pseudo-elementa:

```css
selector::pseudo-element {
  property: value;
}
```

Ove dve tačke su ono što razdvaja pseudo-element od lažne klase, koja koristi jednu tačku.

Pseudo-element omogućava stilizovanje određenih delova sadržaja elementa ili umetanje sadržaja pre ili posle njega, ali ne može postojati samostalno. Element za koji je vezan naziva se izvorni element.

Počnimo sa pregledom pseudo-elemenata za `::before` i `::after`. Kako njihova imena sugerišu, `::before` omogućava unos sadržaja potpuno pre sadržaja elementa, dok `::after` omogućava unos sadržaja posle njega.

Evo primera elementa dugmeta. U CSS-u ćemo koristiti apsolutno pozicioniranje i pseudo-element `::before` kako bismo dodali zvezdicu ispred teksta dugmeta `Learn More`. O apsolutnom pozicioniranju naučićeš više u narednim lekcijama.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: lightseagreen;
  color: white;
  border: none;
  padding: 10px 20px;
  cursor: pointer;
  position: relative;
}

.cta-button::before {
  content: "⭐";
  position: absolute;
  left: 3px;
  top: 8px;
  font-size: 0.75rem;
}
```

:::

Svojstvo `content` koristi se za definisanje sadržaja koji želiš da dodaš pre teksta dugmeta. U ovom primeru dodaje se zvezdica. Osim umetanja sadržaja, možeš ga i stilizovati. Evo primera sa pseudo-elementom `::after` na istom dugmetu:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: '➡️';
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}
```

:::

U pseudo-elementu `::after`, svojstvo `transition` se koristi za pokretanje promena tokom trajanja od 0,3 sekunde sa glatkim efektom, stvarajući postepene promene umesto naglih. Naučićete više o svojstvu `transition` u budućim lekcijama.

Takođe možete dodeliti pseudo-klasu na sadržaj koji ubacujete u drugi sadržaj koristeći pseudo-elemente za `::before` i `::after`. Na primer, stanje prelaska mišem (hover) preko sadržaja. Evo primera:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button class="cta-button">Learn More</button>
```

```css
.cta-button {
  background-color: orange;
  border: none;
  padding: 10px 30px;
  cursor: pointer;
  position: relative;
}

.cta-button::after {
  content: "➡️";
  position: absolute;
  right: 5px;
  bottom: 6px;
  font-size: 1.125rem;
  transition: transform 0.3s ease;
}

.cta-button:hover::after {
  transform: translateX(2px);
}
```

:::

Pomoću `transform: translateX(2px)` u stanju hover, sadržaj se pomiče udesno za `2px` svaki put kada korisnik klikne dugme. Svojstvo transition na samom `::after` osigurava da proces traje `0.3s`.

Ovo je funkcija `transform` – koja vam omogućava da rotirate, savijete, proširite ili transformišete element u određenom smeru. Vi ćete saznati više o tome u budućim lekcijama.

U sledećem primeru, pogledaćemo pseudo-element `::first-letter`. Pseudo-element `::first-letter` cilja prvo slovo sadržaja elementa, omogućavajući vam da ga ukrasite. Evo primera teksta pasusa. Ako želimo da ukrasimo prvo slovo, možemo koristiti pseudo-element `::first-letter` na sledeći način:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>freeCodeCamp lets you learn to code without having to pay.</p>
```

```css
p::first-letter {
  font-size: 4rem;
}
```

:::

U poslednjem primeru, ćemo pogledati pseudo-element `::marker`, koji vam omogućava da izaberete simbole, tačke ili brojeve stavki liste za dekoraciju. Pseudo-element `::marker` pruža način da se poboljša brend identitet vaše web stranice personalizovanjem simbola liste kako bi odgovarali vašem rasporedu boja.

Ovde je primer neuređene liste i uređene liste. Da biste promenili boju i veličinu markera stavki liste, možete koristiti pseudoelement `::marker` na sledeći način:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<ul>
  <li>Unordered list item 1</li>
  <li>Unordered list item 2</li>
  <li>Unordered list item 3</li>
  <li>Unordered list item 4</li>
</ul>

<ol>
  <li>Ordered list item 1</li>
  <li>Ordered list item 2</li>
  <li>Ordered list item 3</li>
  <li>Ordered list item 4</li>
</ol>
```

```css
li::marker {
  color: crimson;
  font-size: 1.5em;
  font-weight: bold;
}
```

:::

U ovom kursu, dotakli smo samo nekoliko pseudo-elemenata. Ali postoji mnogo više kao što su `::placeholder`, `::spelling-error` i `::selection`, koje vas podstičem da istražite sami.
