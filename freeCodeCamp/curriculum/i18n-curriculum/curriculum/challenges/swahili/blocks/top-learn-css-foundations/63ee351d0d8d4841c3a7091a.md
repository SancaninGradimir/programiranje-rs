# --description--

Selektor tipa (ili selektor elementa) će izabrati sve elemente određenog tipa elementa, a sintaksa je samo naziv elementa:

```html
<!-- index.html -->

<div>Hello, World!</div>
<div>Hello again!</div>
<p>Hi...</p>
<div>Okay, bye.</div>
```

```css
/* styles.css */

div {
  color: white;
}
```

Ovde, sva tri elementa za `<div>` će biti selektovana, dok element `<p>` neće biti selektovan.

# --questions--    

## --text--

Koji od sljedećih opisuje kod CSS naveden iznad?

## --answers--

Kod koristi boju `white` za sve elemente u fajlu HTML.

---

Msimbo koristi boju `white` za sve elemente `div` u fajlu HTML.

---

Kod koristi boju `white` za sve elemente od `p` u fajlu HTML.
