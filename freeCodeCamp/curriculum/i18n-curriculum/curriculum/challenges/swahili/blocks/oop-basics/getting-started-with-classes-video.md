## --text--

Koji je način među sledećim za dodavanje metode u klasu objekata?

## --answers--

```py
class Dog:
    set bark(self):
        print("Woof!")
```

---

```py
class Dog:
    def bark(self):
        print("Woof!")
```

---

```py
class Dog:
    method bark(self):
        print("Woof!")
```

---

```py
class Dog:
    def bark:
        print("Woof!")
```
