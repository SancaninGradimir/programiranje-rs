# --description--

Gramatički izraz ``thanks for`` koristi se za izražavanje zahvalnosti za određene akcije ili primljenu pomoć. Ovaj model je obično praćen vrstom glagola ``-ing`` (kontinuirani glagol) ili imenicom kako bi se odredilo zašto se zahvalnost izražava.

Primeri:

`Thanks for helping me with my homework.`

`Thanks for the advice.`
