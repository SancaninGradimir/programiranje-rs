# --description--

Sada hajde da proverimo brojeve od 90 do 99.

Na španski, broj 90 se zove `noventa`.

Ovi brojevi se formiraju korišćenjem reči trideset `noventa` (90), praćen spojnikom `y` ("na"), zatim brojem jedinica (1-9).

| Broj | Kihispania |
| :--- | :--- |
| 90 | `Noventa` |
| 91 | `Noventa y uno` |
| 92 | `Noventa y dos` |
| 93 | `Noventa y tres` |
| 94 | `Noventa y cuatro` |
| 95 | `Noventa y cinco` |
| 96 | `Noventa y seis` |
| 97 | `Noventa y siete` |
| 98 | `Noventa y ocho` |
| 99 | `Noventa y nueve` |

Broj `noventa y uno` (91) je jedinstven u ovoj seriji jer se menja u `noventa y un` pre muškog imena. Na primer:

`Noventa y un años` - Devedeset i jedna godina.
