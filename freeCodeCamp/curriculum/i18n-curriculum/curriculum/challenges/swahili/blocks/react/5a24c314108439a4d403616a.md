# --description--

Prethodni izazov pokazao je kako se informacije prosleđuju iz roditeljske komponente u podređenu preko `props`. Ovaj izazov pokazuje kako se nizovi podataka mogu proslediti kao `props`. Da bi prosledio niz podataka JSX elementu, moraš ga tretirati kao JavaScript i staviti u vitičaste zagrade.

```jsx
<ParentComponent>
  <ChildComponent colors={["green", "blue", "red"]} />
</ParentComponent>
```

Detilji deo zatim ima pristup svojstvu niza podataka `colors`. Metode nizova kao što su `join()` mogu se koristiti prilikom dobijanja tog svojstva.

```jsx
const ChildComponent = (props) => <p>{props.colors.join(', ')}</p>
```

Ovo će spojiti sve elemente niza `colors` u tekstualni niz razdvojen zarezima i prikazati: `<p>green, blue, red</p>`. Kasnije ćeš naučiti i druge uobičajene načine prikazivanja nizova u React-u.

# --instructions--

Postoje sekcije `List` i `ToDo` u editoru koda. Kada prikažete svaki `List` iz sekcije `ToDo`, prođe svojstvo `tasks` koje je postavljeno na vrednost niza vežbi za rad, na primer `["walk dog", "workout"]`. Zatim pronađite ovaj niz `tasks` u sekciji `List` i prikažite njegovu vrednost unutar elementa `p`. Koristite `join(", ")` da prikažete niz `props.tasks` u elementu `p` kao listu od vrednosti razdvojenih zarezima. Današnja lista treba da ima najmanje 2 vežbe, a ona za sutra treba da ima najmanje 3 vežbe.
