# --interactive--

Kada radiš sa JavaScript, možeš naići na situaciju u kojoj moraš ponoviti sekvencu karaktera određeni broj puta.

Bilo da generišete ponavljajuće obrasce ili samo kopirate tekst, metoda `repeat()` nudi jednostavan i efikasan način za postizanje ovoga.

Putanja za `repeat()` je funkcija implementirana unutar JavaScript koja omogućava ponavljanje sekvence karaktera za navedeni broj puta. Evo osnovne sintakse:

```js
string.repeat(count);
```

`string` je niz karaktera koji želite ponoviti, a `count` je broj puta koliko želite da se taj niz ponovi. Evo primera:

:::interactive_editor

```js
let word = "Hello!";
let repeatedWord = word.repeat(3);
console.log(repeatedWord);  // "Hello!Hello!Hello!"
```

:::

U ovom slučaju, sekvenca znakova `Hello!` se ponavlja tri puta, što dovodi do `Hello!Hello!Hello!`.

Iako je način `repeat()` koristan, postoje određene pristrasnosti i ograničenja koja treba uzeti u obzir.

Parametar `count` mora biti neujem broj. Ako prosledite negativan broj, JavaScript će baciti `RangeError`.

```js
let word = "Test";
console.log(word.repeat(-1));  // Throws RangeError: Invalid count value
```

Parametar za `count` mora biti poslednji broj. Ako pokušate da ponovite sekvencu slova beskonačno ili koristite `Infinity` kao brojač, takođe ćete dobiti `RangeError`.

U JavaScript, `Infinity` je posebna vrednost koja predstavlja beskonačnu količinu. Koristi se za prikaz broja većeg od bilo kog konačnog broja.

```js
let word = "Test";
console.log(word.repeat(Infinity));  // Throws RangeError: Invalid count value
```

Ako rezultat nije ceo broj (kao decimala kao `2.5`), funkcija/način `repeat()` će zaokružiti nadole na najbliži ceo broj.

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(2.5));  // "TestTest"
```

:::

Ako prosledite `0` kao izračun, putanja za `repeat()` će vratiti praznu sekvencu znakova.

:::interactive_editor

```js
let word = "Test";
console.log(word.repeat(0));  // ""
```

:::

Način `repeat()` može pojednostaviti zadatke koji uključuju kopiranje sekvenci karaktera, čineći vaš kod kraćim i lakšim za čitanje.

Ako generišete ponavljajuće tekstualne obrasce ili popunjavate praznine slovima, `repeat()` može vam uštedeti pisanje složenih petlji ili koda.

Omogućeno je samo prosleđivanje broja direktno u putanju na `repeat()`. Takođe možete proslediti parametar koji čuva vrednost broja.

:::interactive_editor

```js
let count = 4;
let word = "Test";
let repeatedWord = word.repeat(count);
console.log(repeatedWord); // TestTestTestTest
```

:::

U ovom primeru, parametar `count` čuva broj ponavljanja. Ovo može biti korisno kada broj ponavljanja zavisi od unosa korisnika ili drugih promenljivih vrednosti u vašoj aplikaciji.
