# --interactive--

U programiranju, razumevanje kako su slova predstavljena kao brojevi je ključna osnova. Ovde ulazi ASCII. ASCII, skraćenica za American Standard Code for Information Interchange, je standard kodiranja znakova koji se koristi u računarima za predstavljanje teksta. On dodeljuje numeričke vrednosti svakom slovu, koje mašine generalno prepoznaju.

U ovom kursu ćemo pogledati šta je ASCII, kako funkcioniše i kako su JavaScript metode kao što su `charCodeAt()` i `fromCharCode()` povezane sa kodiranjem znakova. Iako JavaScript interno koristi Unicode (UTF-16) za niske znakova, ASCII vrednosti odgovaraju prvih 128 Unicode znakova, pa primeri zasnovani na ASCII-u funkcionišu u JavaScript-u.

ASCII je sistem kodiranja koji pretvara znakove, kao što su slova, brojevi i simbole, u numeričke vrednosti. Svaki znak povezan je sa određenim brojem.

Na primer, velika slova `A` je predstavljeno brojem `65` u ASCII-ju, dok su mala slova `a` predstavljena sa `97`. Ovo kodiranje omogućava računaru da skladišti i obrađuje tekst.

Standard ASCII uključuje 128 znakova, uključujući:

- Velika i mala slova engleskog alfabeta (A-Z, a-z).
- Brojevi (0-9).
- Pravopisni znaci i obični simboli (!, @, #, itd.).
- Kontrolni znakovi (poput novog reda i tabulatora).

U JavaScript, možete dobiti numerički kod slova koristeći metod `charCodeAt()`. Ovaj metod vraća jedinicu koda slova u UTF-16 formatu na određeni način. Za prvih 128 slova, ova vrednost odgovara ASCII kodu.

Na primer:

:::interactive_editor

```js
let letter = "A";
console.log(letter.charCodeAt(0));  // 65
```

:::

U ovom primeru, `A` je prvo slovo sekvence slova, i poziv `charCodeAt(0)` vraća svoj numerički kod (koji odgovara njegovoj ASCII vrednosti za osnovna latinska slova), `65`.

Takođe možete koristiti ovu metodu sa drugim slovima da dobijete njihove numeričke kod vrednosti:

:::interactive_editor

```js
let symbol = "!";
console.log(symbol.charCodeAt(0));  // 33
```

:::

Ovde, kod broja znaka uzvika `!` vraća se kao `33` (korespondira sa njegovom ASCII vrednošću).

Kada `charCodeAt()` vam pomaže da dobijete numerički kod slova, način `fromCharCode()` omogućava vam da uradite suprotno: konvertovanje UTF-16 jedinice koda (koja odgovara ASCII za osnovna slova) u odgovarajuće slovo.

Razmotrimo ovaj primer:

:::interactive_editor

```js
let char = String.fromCharCode(65);
console.log(char);  //  A
```

:::

U ovom primeru, `fromCharCode(65)` konvertuje kod broja `65` (koji odgovara ASCII vrednosti `A`) nazad u slovo `A`.

Još jedan primer je promena broja `97` u odgovarajuće mala slovo:

:::interactive_editor

```js
let char = String.fromCharCode(97);
console.log(char);  // a
```

:::

Ove metode su važne posebno kada je potrebno rukovati ili uporediti karaktere prema njihovim numeričkim kodnim vrednostima.

Na primer, možete koristiti `charCodeAt()` da proverite da li je znak veliko slovo, malo slovo ili broj poređenjem njegove ASCII vrednosti.

S druge strane, `fromCharCode()` se može koristiti za generisanje znakova iz njihovih ASCII kodova.
