# --description--

Niz celih brojeva $S = \\{s_i\\}$ naziva se nizom dužine $n$ ako ima $n$ elemenata i svaki element $s_i$ zadovoljava $1 ≤ s_i ≤ n$. Ukupan broj različitih nizova dužine $n$ je $n^n$.

$f_{n, a, b}(x) ≡ ax + b\bmod n$ kwa $a, b, x$ nambari kamili na $0 \lt a \lt n$, $0 \le b \lt n$, $0 \le x \lt n$.

Za bilo koji niz $S$, neka $L(S)$ predstavlja dužinu najdužeg uzastopnog podniza sa istim vrednostima. Na primer, za gore dati niz $S$, važi $L(S) = 3$, zbog tri uzastopne sedmice.

Acha $R(n)$ iwe idadi ya kurudisha kwa $n$.

Umepatiwa kwamba

$$\sum_{k = 1}^{99\\,999} R(\displaystyle\binom{100\\,000}{k}) \equiv 628\\,701\\,600\bmod 1\\,000\\,000\\,007$$

Tafuta $$\sum_{k = 1}^{9\\,999\\,999} R(\displaystyle\binom{10\\,000\\,000}{k})$$ Toa jibu lako modulo $1\\,000\\,000\\,007$.
