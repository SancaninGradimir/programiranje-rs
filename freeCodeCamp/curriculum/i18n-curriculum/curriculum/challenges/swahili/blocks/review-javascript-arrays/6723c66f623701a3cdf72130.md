## Osnovi rednih podataka za JavaScript

- **Opis**: Niz podataka za JavaScript je organizovana kolekcija vrednosti, svaka identifikovana numeričkim indeksom. Vrednosti u nizu podataka za JavaScript mogu biti različitih tipova podataka, uključujući brojeve, nizove karaktera, booleanske vrednosti (tačno ili netačno), objekte i čak druge nizove. Nizovi podataka su neprekidni u memoriji, što znači da su svi elementi sačuvani na jednom kontinualnom delu memorijskih lokacija, omogućavajući brast i efikasan pristup korišćenjem indeksa.

```js
const developers = ["Jessica", "Naomi", "Tom"];
```

- **Pristup elementima iz niza podataka**: Da biste pristupili elementima iz niza podataka, morate se referisati na taj niz praćen brojem indeksa unutar uglastih zagrada. Nizovi podataka JavaScript imaju indeksiranje koje počinje od nule, što znači da je prvi element na indeksu 0, drugi element na indeksu 1, itd. Ako pokušate dohvatiti indeks koji ne postoji u nizu, JavaScript će vratiti `undefined`.

:::interactive_editor

```js
const developers = ["Jessica", "Naomi", "Tom"];
console.log(developers[0]) // "Jessica"
console.log(developers[1]) // "Naomi"

console.log(developers[10]) // undefined
```

:::

- **Atribut za `length`**: Ovaj atribut se koristi za vraćanje broja elemenata u redu podataka.

:::interactive_editor

```js
const developers = ["Jessica", "Naomi", "Tom"];
console.log(developers.length) // 3
```

:::

- **Izmenjivanje elemenata u redovima podataka**: Da biste izmenili element u redu podataka, koristite operator dodeljivanja (`=`) da postavite novu vrednost za element koji se nalazi u određenom modelu.

:::interactive_editor

```js
const fruits = ['apple', 'banana', 'cherry'];
fruits[1] = 'blueberry';

console.log(fruits); // ['apple', 'blueberry', 'cherry']
```

:::

## Red za podatke sa dve dimenzije

- **Opis**: Dvodimenzionalni niz podataka je niz niza podataka. Koristi se za predstavljanje podataka sa mrežastom strukturom, kao što su šahovska tabla, kalkulacioni list ili pikseli u slici. Da biste dobili element iz dvodimenzionalnog niza, potrebni su vam dva indeksa: jedan za red i drugi za kolonu.

:::interactive_editor

```js
const chessboard = [
    ['R', 'N', 'B', 'Q', 'K', 'B', 'N', 'R'],
    ['P', 'P', 'P', 'P', 'P', 'P', 'P', 'P'],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    [' ', ' ', ' ', ' ', ' ', ' ', ' ', ' '],
    ['p', 'p', 'p', 'p', 'p', 'p', 'p', 'p'],
    ['r', 'n', 'b', 'q', 'k', 'b', 'n', 'r']
];

console.log(chessboard[0][3]); // "Q"
```

:::

## Rešenje strukture reda podataka

- **Opis**: Konstrukcija niza podataka je funkcija u JavaScript koja vam omogućava da izvučete vrednosti iz nizova podataka i dodelite ih varijablama na kraći i razumljiviji način. Pruža jednostavnu sintaksu za pristup elementima niza podataka u različite varijable.

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange"];

const [first, second, third] = fruits;

console.log(first); // "apple"
console.log(second); // "banana"
console.log(third); // "orange"
```

:::

- **Sintaksa za REST**: Ovo vam omogućava da uhvatite preostale elemente redova podataka koji nisu razrešeni i da ih stavi u novi red.

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange", "mango", "kiwi"];
const [first, second, ...rest] = fruits;

console.log(first); // "apple"
console.log(second); // "banana"
console.log(rest); // ["orange", "mango", "kiwi"]
```

:::

## Standardne metode za red podataka

- **Metoda za `push()`**: Ova metoda se koristi za dodavanje elemenata na kraj niza podataka i vraća novu dužinu niza.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.push("ice cream");

console.log(desserts); // ["cake", "cookies", "pie", "ice cream"];
```

:::

- **Metoda za `pop()`**: Ova metoda se koristi za uklanjanje poslednjeg elementa iz niza podataka i vraća uklonjeni element. Ako je niz prazan, onda će vrednost return biti `undefined`.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.pop();

console.log(desserts); // ["cake", "cookies"];
```

:::

- **Metoda `shift()`**: Ova metoda se koristi za uklanjanje prvog elementa iz niza podataka i vraćanje uklonjenog elementa. Ako je niz prazan, vrednost od return biće `undefined`.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.shift();

console.log(desserts); // ["cookies", "pie"];
```

:::

- **Njia ya `unshift()`**: Ova metoda se koristi za dodavanje elemenata na početak niza podataka i vraća novu dužinu niza.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
desserts.unshift("ice cream");

console.log(desserts); // ["ice cream", "cake", "cookies", "pie"];
```

:::

- **Putanja za `indexOf()`**: Ova putanja je važna za dobijanje prvog primerka određenog elementa unutar niza podataka. Ako element nije pronađen, onda će vratiti `-1`.

:::interactive_editor

```js
const fruits = ["apple", "banana", "orange", "banana"];
const index = fruits.indexOf("banana");

console.log(index); // 1
console.log(fruits.indexOf("not found")); // -1
```

:::

- **Njia ya `splice()`**: Ova metoda se koristi za dodavanje ili uklanjanje elemenata na bilo kojoj poziciji u nizu podataka. Vrednost return metode `splice()` će biti niz uklonjenih elemenata iz niza. Ako ništa nije uklonjeno, vraća se prazan niz. Ova metoda modifikuje originalni niz, menjajući ga na mestu umesto da kreira novi niz. Prvi argument opisuje početni model za modifikaciju niza. Drugi argument je broj elemenata koje želite da uklonite. Sledeći argumenti su elementi koje želite da dodate.

:::interactive_editor

```js
const colors = ["red", "green", "blue"];
colors.splice(1, 0, "yellow", "purple");

console.log(colors); // ["red", "yellow", "purple", "green", "blue"]
```

:::

- **Njia ya `includes()`**: Ova funkcija se koristi za proveru da li red ima određenu vrednost. Ona vraća `true` ako red sadrži spomenuti element, i `false` inače.

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];

console.log(programmingLanguages.includes("Python")); // true
console.log(programmingLanguages.includes("Perl")); // false
```

:::

- **Njia ya `concat()`**: Ova putanja kreira novi red spajanjem dva ili više redova.

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];
const newList = programmingLanguages.concat("Perl");

console.log(newList); // ["JavaScript", "Python", "C++", "Perl"]
```

:::

- **Metoda `slice()`**: Ova metoda vraća novi niz koji sadrži kopiju sekcije iz originalnog niza, definisane početnim i krajnjim indeksima. Novi niz ima reference na elemente jednake onima iz originalnog niza (nisu kopije elemenata). To znači da ako su elementi primitivni tipovi (poput brojeva ili nizova karaktera), kopiraju se vrednosti; ali ako su elementi objekti ili nizovi, kopiraju se reference, a ne sami objekti.

:::interactive_editor

```js
const programmingLanguages = ["JavaScript", "Python", "C++"];
const newList = programmingLanguages.slice(1);

console.log(newList); // ["Python", "C++"]
```

:::

- **Spread sintaksa**: Spread sintaksa se koristi za kreiranje dubokih kopija niza podataka.

:::interactive_editor

```js
const originalArray = [1, 2, 3];
const shallowCopiedArray = [...originalArray];

shallowCopiedArray.push(4);

console.log(originalArray); // [1, 2, 3]
console.log(shallowCopiedArray); // [1, 2, 3, 4]
```

:::

- **Metoda za `split()`**: Ova metoda razdvaja niz znakova na niz manjih delova i objašnjava gde bi svako razdvajanje trebalo da se desi u zavisnosti od datoog glagola. Ako nije dostavljen glagol, ova metoda vraća niz sa prvobitnim nizom znakova kao jedan element.

:::interactive_editor

```js
const str = "hello";
const charArray = str.split("");

console.log(charArray); // ["h", "e", "l", "l", "o"]
```

:::

- **Putanja za `reverse()`**: Ova putanja konvertuje red podataka na tom mestu.

:::interactive_editor

```js
const desserts = ["cake", "cookies", "pie"];
console.log(desserts.reverse()); // ["pie", "cookies", "cake"]
```

:::

- **Puta za `join()`**: Ova putanja spaja sve elemente niza podataka u jednu sekvencu karaktera, pri čemu je svaki element razdvojen navedenim separatorom. Ako nije naveden separator, ili ako je korišćena prazna sekvenca karaktera (`""`), elementi će biti spojeni bez ikakvog separatora između njih.

:::interactive_editor

```js
const reversedArray = ["o", "l", "l", "e", "h"];
const reversedString = reversedArray.join("");

console.log(reversedString); // "olleh"
```

:::
