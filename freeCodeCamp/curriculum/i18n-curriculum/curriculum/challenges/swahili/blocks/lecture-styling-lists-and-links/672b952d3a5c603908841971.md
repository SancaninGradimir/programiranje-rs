# --interactive--

Podrazumevani stilovi linkova igraju važnu ulogu u poboljšanju upotrebljivosti i online pristupačnosti.

Ovi stilovi su obično plavi za neposete linkove i ljubičasti za posete linkove, što je postalo standard koji korisnici očekuju i na koji se oslanjaju kada posećuju veb stranicu.

Glavni cilj podrazumevanih stilova linkova je pružiti jasne vizuelne indikacije koje pomažu korisnicima da razlikuju između interaktivnih i neinteraktivnih elemenata na veb stranici.

Ova razlika je osnova za kreiranje korisničkog iskustva koje je razumljivo i jednostavno za upotrebu.

Razmotrimo osnovne podrazumevane stilove za linkove:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
a:link {
  color: blue;
  text-decoration: underline;
}

a:visited {
  color: purple;
}
```

:::

Ovi stilovi pružaju nekoliko važnih funkcija.

Prvo, plava boja za neposete linkove dobro izgleda u odnosu na mnoge pozadinske boje i tekstove, što čini linkove lakim za prepoznavanje. Ova razlika je važna za korisnike koji brzo pretražuju stranice i pronalaze navigacione elemente ili važne informacije.

Podvučena linija naglašava da je tekst klikabilan, pružajući dodatni vizuelni indikator. Ovo je posebno korisno za korisnike koji imaju daltonizam ili im je teško razlikovati boje.

Promene boje za posećene linkove (obično u ljubičastu) pomažu korisnicima da prate gde su već bili. Ova funkcija je veoma korisna prilikom pregledanja velikih veb lokacija ili istraživanja, jer sprečava korisnike da slučajno posećuju iste stranice više puta.

Pogledajte ovaj primer za HTML:

:::interactive_editor

```html
<p>Learn more about <a href="https://www.example.com/cats">cats</a> and <a href="https://www.example.com/dogs">dogs</a>.</p>
```

:::

Bez posebnog CSS-a, većina pregledača će prikazati ove linkove plavom bojom i sa podvučenim tekstom. Nakon klika na jedan link, njegova boja će se promeniti u ljubičastu, pružajući korisniku trenutnu povratnu informaciju o istoriji pregledanja.

Iako je uobičajeno da dizajneri menjaju ove podrazumevane stilove kako bi odgovarali izgledu veb-sajta, važno je sačuvati osnovne principe koji stoje iza njih.

Ako odlučite da promenite podrazumevane stilove, uverite se da linkovi i dalje izgledaju jasno drugačije od običnog teksta, postoji vidljiva razlika između posetjenih i neposetjenih linkova, a da izabrane boje imaju dovoljno kontrasta sa bojom pozadine.

Na primer, možete koristiti prilagođeni model kao ovaj:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
a:link {
  color: blue;
  text-decoration: none;
  border-bottom: 1px solid blue;
}

a:visited {
  color: purple;
  border-bottom: 1px solid purple;
}
```

:::

Ovo zadržava raspored plave i ljubičaste boje, istovremeno menjajući donju liniju i donju ivicu za moderniji izgled.

Takođe je važno uzeti u obzir različita stanja elemenata/linkova. Pored stanja podrazumevano (default) i poseta (visited), linkovi obično imaju stanja hover i active:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
a:hover {
  color: red;
}

a:active {
  color: darkorange;
}
```

:::

Ovi elementi pružaju trenutnu povratnu informaciju korisnicima kada interaguju sa linkovima, poboljšavajući ukupnu upotrebljivost veb-sajta.

Konačno, iako je moguće prilagoditi stili elemenata, principi koji stoje iza podrazumevanih stilova treba da se sačuvaju.

Imaju važnu ulogu u kreiranju upotrebljivog i lako dostupnog online iskustva, pomažući korisnicima da efikasno pretražuju i interaguju sa sadržajem.

Uvek prioritet daj jasnoći i korisničkom iskustvu kada dizajniraš linkove za svoje veb-sajtove.
