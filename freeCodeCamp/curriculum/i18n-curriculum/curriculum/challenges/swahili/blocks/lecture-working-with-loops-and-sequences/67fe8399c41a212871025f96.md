# --description--

U narednim nekoliko lekcija ćemo naučiti o listama, tuplovima i opsezima, koje su tri osnovna tipa sekvenci korišćene u Python.

Tip podataka lista predstavlja uređeni niz elemenata koji mogu biti niske karaktera, brojevi ili čak druge liste. Liste su promenljive i koriste indeksiranje koje počinje od nule, što znači da se prvi element liste nalazi na indeksu nula.

Ovde je osnovna sintaksa liste:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
```

Da biste dobili element iz liste ``cities``, možete se referirati na njegov indeks u sekvenci. Evo primera kako dobiti prvi element iz liste ``cities``:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # 'Los Angeles'
```

Negativni indeksi koriste se za pristup elementima počev od kraja liste umesto od početka na indeksu `0`. Da pristupiš poslednjem elementu bilo koje liste, možeš da koristiš `-1` ovako:

```py
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # 'Tokyo'
```

Drugi način za kreiranje liste je korišćenjem konstruktora `list()`. Konstruktor `list()` služi za pretvaranje iterabilnog objekta u listu, kao u sledećem primeru:

```py
developer = 'Jessica'
list(developer) # ['J', 'e', 's', 's', 'i', 'c', 'a']
```

Ponovljivi objekat je poseban tip objekta koji se može ponoviti po jedan element u isto vreme. Naučićete više o petljama u Python kasnije.

Da dobiješ ukupan broj elemenata u listi, možeš da koristiš funkciju `len()` ovako:

```py
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

Ako želiš da izmeniš vrednost na određenom indeksu, možeš da uradiš nešto poput ovoga:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

Pošto su liste promenljive, možeš da izmeniš bilo koji element liste sve dok navedeš važeći indeks. Ako navedeš indeks (pozitivan ili negativan) koji je van granica liste, dobićeš `IndexError`:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list assignment index out of range
"""
```

Ako želiš da ukloniš element iz liste, možeš da koristiš ključnu reč `del` ovako:

```py
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

Ponekad je korisno proveriti da li se element nalazi u listi. Da biste to uradili, možete koristiti ključnu reč `in` kao što sledi:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

Ponekad je uobičajeno imati liste organizovane unutar druge liste, kao što sledi:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

U ovom primeru, imamo jedan ugnježdeni niz sa tri popularna programerska jezika. Da biste dobili ugnježdeni niz, moraćete da ga dobijete koristeći šablon `2` jer niz koristi šablon koji počinje od nule:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

Da biste dobili drugi jezik iz tog organizovanog spiska, moraćete da ga dobijete koristeći šablon `1` kao što sledi:

```py
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # 'Rust'
```

Drugi uobičajeni metod koji se koristi sa listom je pristup vrednosti.

Raspakivanje vrednosti iz liste je tehnika kojom se vrednosti iz liste dodeljuju novim promenljivama. Evo primera raspakivanja liste `developer` u nove promenljive `name`, `age` i `job`.

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

Ovde, `name` ima vrednost `'Alice'`, `age` ima vrednost `34`, na `job` ima vrednost `'Rust Developer'`.

Ako treba da prikupiš sve preostale elemente iz liste, možeš da koristiš operator zvezdice (`*`) ovako:

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

U ovom primeru, `name` će i dalje imati vrednost `'Alice'`, a `rest` je lista od dva elementa: broj `34` i niz slova `'Rust Developer'`.

Ako broj parametara na levoj strani operacije raspodele ne odgovara ukupnom broju stavki u listi, dobićete `ValueError`:

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job, city = developer

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
ValueError: not enough values to unpack (expected 4, got 3)
"""
```

Poslednji koncept o kojem ćemo govoriti je operater za *slice* (`:`). Kao i kod sekvence karaktera, možete dobiti delove niza koristeći operater za *slice*, kao što sledi:

```py
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie', 'Brownies']
desserts[1:4] # ['Cookies', 'Ice Cream', 'Pie']
```

U ovom primeru, početni indeks je `1` jer odgovara drugom elementu u listi. Zatim koristimo operator isečka sa završnim indeksom `4`, koji uključuje sve do (ali ne i) element na tom indeksu.

Još jedna stvar koju možete uraditi sa *slice* operatorom `:` je navođenje koraka koji određuje koliko se indeks povećava. Recimo da imate listu brojeva kao što je sledeće:

```py
numbers = [1, 2, 3, 4, 5, 6]
```

Ako želiš da izdvojiš samo listu pozitivnih brojeva, možeš da koristiš operator isečka (slice) ovako:

```py
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

Prvi pozitivan broj je na šablonu `1`, pa će to biti početni šablon. Pošto želimo da prođemo do kraja liste, zanemarujemo poslednji šablon. Na kraju, definišemo `2` za privremenu koraku kako bi se podigao na `2` umesto standardnog `1`.

Liste su koristan i fleksibilan tip podataka koji ćeš često koristiti u svojim Python programima. U sledećoj lekciji naučićeš o uobičajenim metodama koje možeš da koristiš sa listama.
