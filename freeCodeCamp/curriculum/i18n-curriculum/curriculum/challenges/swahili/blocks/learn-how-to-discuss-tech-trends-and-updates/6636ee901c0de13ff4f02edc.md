# --description--

Term `run code` se odnosi na čin izvršavanja ili pokretanja programa ili skripte na računaru. Izvršavanje `run` u ovom kontekstu znači pokretanje procesa i nastavak sve dok program ne obavi ono za šta je dizajniran, bilo da je reč o izračunavanju brojeva, organizovanju podataka ili bilo čemu drugom.

Usemi `run code without managing servers` Odnosi se na pokretanje računarskih programa ili aplikacija bez potrebe za upravljanjem fizičkim ili mrežnim serverima na kojima se kod izvršava. Ovo je važan deo serverless računarstva, gde pružalac usluga u oblaku upravlja serverima, omogućavajući programerima da se fokusiraju više na kod umesto na upravljanje serverima.
