# --interactive--

Kada radiš sa CSS-om, često ćeš se susretati sa tri različita načina prikaza elemenata: `inline`, `block` i `inline-block`.

Svaka od ovih vrednosti utiče na način na koji se elementi raspoređuju i kako međusobno deluju na stranici.

U ovoj lekciji ćemo se fokusirati na to kako funkcioniše svojstvo `inline-block` i po čemu se razlikuje od `inline` i `block` elemenata.

Block elementi se ponašaju kao veliki kontejneri ili "blokovi" koji zauzimaju celu širinu svog roditeljskog kontejnera. Uvek počinju u novom redu, a njihova širina i visina mogu da se menjaju.

Inline elementi zauzimaju samo onoliko prostora koliko je potrebno. Nalaze se u toku teksta i ne započinju novi red.

Svojstvo `inline-block` predstavlja kombinaciju ova dva ponašanja. Kao i inline elementi, elementi sa `inline-block` ostaju u toku teksta bez započinjanja novog reda.

Međutim, za razliku od inline elemenata, kod `inline-block` elemenata možeš da menjaš širinu i visinu, baš kao kod block elemenata.

Ukratko, glavna razlika između `inline` i `inline-block` elemenata je u tome što se veličina inline elemenata ne može menjati, dok `inline-block` omogućava potpunu kontrolu nad dimenzijama, a element i dalje ostaje u istom redu sa ostalim sadržajem.

Tuchunguze mfano mmoja.

:::interactive_editor

```html
<link href="styles.css" rel="stylesheet">

<div class="container">
  <span class="inline-block-element element1">Inline-Block</span>
  <span class="inline-block-element element2">Inline-Block</span>
</div>
```

```css
.inline-block-element {
  display: inline-block;
  width: 150px;
  height: 100px;
}

.element1 {
  background-color: lightblue;
}

.element2 {
  background-color: lightgreen;
}
```

:::

U ovom primeru imamo `div` sa klasom `container`. Unutar tog `div` elementa nalaze se dva `span` elementa.

Svaki `span` element ima postavljeno `display: inline-block` i definisanu širinu i visinu.

Inline-blok elementi će se pojaviti jedan pored drugog jer funkcionišu kao inline elementi, ali takođe imaju definisanu širinu i visinu, što im daje karakteristike blok elementa.

Ali, uklanjanjem karakteristike `display: inline-block`, niti `height` niti `width` ne mogu biti izvršeni, iako ste to jasno naveli.

Evo CSS: ispravljeno

:::interactive_editor

```html
<link href="styles.css" rel="stylesheet">

<div class="container">
  <span class="inline-block-element element1">Span element</span>
  <span class="inline-block-element element2">Span element</span>
</div>
```

```css
.inline-block-element {
  width: 150px;
  height: 100px;
}

.element1 {
  background-color: lightblue;
}

.element2 {
  background-color: lightgreen;
}
```

:::

U ovom primeru uklonili smo svojstvo `display: inline-block;`, dok je sve ostalo ostalo isto. Zbog toga se `span` elementi vraćaju svom podrazumevanom ponašanju kao inline elementi.

Kao rezultat toga, definisana širina i visina se ne primenjuju, pa elementi zauzimaju samo onoliko prostora koliko je potrebno za njihov sadržaj.

Razumevanje načina na koji funkcioniše `inline-block` važno je zato što ga možeš koristiti za pravljenje rasporeda koji zahtevaju poravnanje i kontrolu dimenzija unutar neprekinutog toka teksta.

## --text--

Koja je glavna osobina `inline-block` elementa koja ga razlikuje od `inline` elementa?

## --answers--

Inline-block elementi ne mogu da se postavljaju unutar block elemenata.
