# --description--

Kada radiš sa nizovima slova, spajanje različitih delova teksta zajedno je uobičajena operacija koju često obavljaš.

U Python, možete spojiti nekoliko nizova znakova koristeći plus operator (`+`). Ovaj proces se zove **konkatenacija niza znakova**. Evo kako spojiti dva niza znakova pomoću plus operatora:

```python
my_str_1 = 'Hello'
my_str_2 = "World"

str_plus_str = my_str_1 + ' ' + my_str_2
print(str_plus_str) # Hello World
```

Ali zapamti da ovo radi samo sa nizom slova. Ako pokušaš da spojiš niz slova i brojeve, dobićeš `TypeError`:

```python
name = 'John Doe'
age = 26

name_and_age = name + age
print(name_and_age) # TypeError: can only concatenate str (not "int") to str
```

Ovo se dešava zato što Python ne konvertuje automatski druge tipove podataka, kao što su cele brojeve, u niz znakova kada ih spajate. Python zahteva da sve komponente budu niza znakova pre nego što može da ih spoji. Da biste ovo popravili, možete konvertovati broj u niz znakova koristeći ugrađenu funkciju `str()`, koja vraća reprezentaciju niza znakova datog objekta bez menjanja originalnog objekta:

```python
name = 'John Doe'
age = 26

name_and_age = name + str(age)
print(name_and_age) # John Doe26
```

Unaweza pia kutumia opereta ya mgawo ulioboreshwa kwa ajili ya kuunganisha. Hii inawakilishwa na alama ya pamoja na sawa (`+=`), na hufanya uunganishaji na mgawo kwa hatua moja. Hapa kuna mfano wake:

```py
name = 'John Doe'
age = 26

name_and_age = name  # Start with the name
name_and_age += str(age)  # Append the age as string

print(name_and_age)  # John Doe26
```

Proces umetanja promenljivih i izraza u tekstualni string naziva se **interpolacija stringova**. Python ima tip stringa koji se zove **f-string** (skraćeno od formatted string literal), koji omogućava jednostavnu i preglednu interpolaciju vrednosti.

F-stringovi počinju sa `f` (bilo da je mala ili velika slova) pre navodnika, i ne dozvoljava vam da unesete varijable ili matematičke izraze unutar delova umetanje koji su identifikovani uglastim zagradama `{}`. Evo primera:

```python
name = 'John Doe'
age = 26
name_and_age = f'My name is {name} and I am {age} years old'
print(name_and_age) # My name is John Doe and I am 26 years old

num1 = 5
num2 = 10
print(f'The sum of {num1} and {num2} is {num1 + num2}') # The sum of 5 and 10 is 15
```

Pogledajte kako ne morate da konvertujete nije sekvencijalne tipove koristeći funkciju ``str()``. U navedenom primeru, vrednosti parametara ``age``, ``num1`` i ``num2`` se automatski konvertuju u niz znakova tokom procesa unosa vrednosti.
