# --description--

U ovoj radionici, vežbaće se rada sa različitim pseudo-klasama i pseudo-elementima dizajniranjem čestitke. HTML šablon je dostavljen za vas.

Počnite radionicu povezivanjem fajla `styles.css`.

# --hints--

Vaš `link` element treba da bude unutar vašeg `head` elementa.

```js
const link = document.querySelector('head > link');
assert.isNotNull(link);
```

Tvoj element `link` treba da ima atribut `rel` sa vrednošću `stylesheet`.

```js
const linkRelValue = document.querySelector('link')?.getAttribute('rel');
assert.strictEqual(linkRelValue, 'stylesheet');
```

Element vašeg `link` treba da ima atribut `href` sa vrednošću `styles.css`.

```js
const linkHrefValue = document.querySelector('link')?.dataset?.href;
assert.match(linkHrefValue, /^(\.\/)?styles\.css$/);
```
