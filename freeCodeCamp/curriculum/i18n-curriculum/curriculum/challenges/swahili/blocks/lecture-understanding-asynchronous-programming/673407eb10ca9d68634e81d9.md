# --description--

API geolokacija pruža način da sajt zatraži pristup lokaciji korisnika. Važno je razumeti da, iz razloga privatnosti, korisnik mora dati dozvolu pre nego što se njegova lokacija može pristupiti putem sajta.

Glavna metoda na koju ćemo danas fokusirati se je `getCurrentPosition`. Ova metoda se koristi za prikupljanje geografske lokacije uređaja. Evo primera kako možete koristiti `getCurrentPosition`:

```js
navigator.geolocation.getCurrentPosition(
  (position) => {
    console.log("Latitude: " + position.coords.latitude);
    console.log("Longitude: " + position.coords.longitude);
  },
  (error) => {
    console.log("Error: " + error.message);
  }
);
```

U ovom kodu pozivamo `getCurrentPosition` i prosleđujemo mu funkciju koja će biti pozvana kada se lokacija uspešno dobije. Objekat položaja sadrži različite informacije, ali nas zanimaju samo `latitude` i `longitude`.

Ako dođe do greške pri dobijanju lokacije, ona će biti ispisana u konzoli.

Metoda `getCurrentPosition` koristi GPS, Wi-Fi mreže ili pristup lokaciji putem IP adrese, zavisno od uređaja i njegovih podešavanja. Kada je lokacija dostupna, callback funkcija za uspeh poziva sa objektom lokacije.

Objekat položaja sadrži različita svojstva. Najčešće se koriste `latitude` i `longitude`, ali može sadržati i `altitude`, `accuracy`, `speed`, `heading` i druga.

Jedna važna stvar koju treba imati na umu prilikom korišćenja geolokacije jeste privatnost korisnika. Obavezno objasnite svojim korisnicima zašto vam je potrebna njihova lokacija i kako ćete koristiti te podatke.
