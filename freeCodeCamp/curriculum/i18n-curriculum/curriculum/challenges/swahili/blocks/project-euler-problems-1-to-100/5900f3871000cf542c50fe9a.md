# --description--

Euler je otkrio zadivljujuću matematičku kvadratnu formulu:

<div style='margin-left: 4em;'>$n^2 + n + 41$</div>

Dokazano je da ovo tvrdnje daje osnovni broj 40 za uzastopne vrednosti celih brojeva $0 \\le n \\le 39$. Međutim, kada je $n = 40, 40^2 + 40 + 41 = 40(40 + 1) + 41$ deljivo sa 41, i naravno kada je $n = 41, 41^2 + 41 + 41$ očigledno deljivo sa 41.

Otkriven je neverovatan izraz $n^2 - 79n + 1601$, koji generiše brojeve u bazi 80 za opseg uzastopnih vrednosti $0 \\le n \\le 79$. Zbir koeficijenata, −79 i 1601, je −126479.

Razmotrenje kvadratnih funkcija tipa:

<div style='margin-left: 4em;'>
  $n^2 + an + b$, gde je $|a| < range$ i $|b| \le range$<br>
  gde je $|n|$ modul/apsolutna vrednost $n$<br>
  primer $|11| = 11$ i $|-4| = 4$<br>
</div>

Pronađite koeficijente, $a$ i $b$, za kvadratni izraz koji daje najveće vrednosti za uzastopne vrednosti $n$, počevši od $n = 0$.
