# --description--

U Express, element `res` (odgovor) koristi se za slanje odgovora nazad klijentu nakon obrade dolaznih zahteva. Express nudi nekoliko ugrađenih načina za slanje različitih tipova odgovora. Na primer, možete poslati običan tekst, podatke JSON, uputiti korisnika na drugi URL, ili prikazati šablon.

Najčešći načini odgovora su `res.send()`:

```javascript
app.get("/text", (req, res) => {
  res.send("Hello, World!");
});
```

Metod `res.send()` šalje odgovor nazad klijentu. Može da pošalje običan tekst, HTML, JSON ili čak i binarne podatke. U ovom primeru, kada klijent poseti `/text`, server će odgovoriti porukom `Hello, World!`.

Drugi često korišćeni način rešenja je `res.json()`.

Ova metoda šalje JSON odgovor klijentu. Najvažnija je prilikom izgradnje API-ja, gde često trebate da vratite podatke kao što su objekti ili nizovi. Express automatski konvertuje JavaScript objekat u JSON string:

```javascript
app.get("/user", (req, res) => {
  res.json({ name: "John Doe", age: 30 });
});
```

Ako klijent pošalje zahtev na `/user`, server će odgovoriti na sledeći način:

```json
{
  "name": "John Doe",
  "age": 30
}
```

Kada treba da kontrolišete HTTP statusni kod odgovora, možete koristiti `res.status()`. Ova metoda postavlja statusni kod i obično se koristi zajedno sa drugom metodom za odgovor, kao što su `res.send()` ili `res.json()`:

```javascript
app.get("/notfound", (req, res) => {
  res.status(404).send("Not Found");
});
```

U ovom slučaju, server postavlja statusni kod na `404` i šalje poruku `Not Found` nazad klijentu.

Ako želite da preusmerite korisnika na drugi URL, možete koristiti `res.redirect()`:

```javascript
app.get("/oldpage", (req, res) => {
  res.redirect("/newpage");
});
```

Kada klijent poseti `/oldpage`, onemogućen je odmah redirektovan na `/newpage`.

Na kraju, `res.render()` se koristi kada radite sa šablonom poput Pug-a, EJS-a ili Handlebars-a. Ovaj metod prikazuje šablon i šalje HTML klijentu:

```javascript
app.get("/profile", (req, res) => {
  res.render("profile", { username: "john_doe" });
});
```

U ovom primeru, server prikazuje pogled `profile` i prenosi promenljivu `username` koja se može prikazati u HTML.

Ove metode odgovora omogućavaju da vratite različite tipove podataka u zavisnosti od prirode zahteva. Bez obzira da li koristite tekst, JSON podatke ili usmeravate klijenta, Express ima jednostavan set alata za rukovanje tim.
