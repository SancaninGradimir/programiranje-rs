# --description--

Za konstante cele brojeve $a$, $b$, $c$, definisaćemo funkciju $F(n)$ na sledeći način:

$$\begin{align}
  & F(n) = n - c \\;\text{ kwa wote } n > b \\\\
  & F(n) = F(a + F(a + F(a + F(a + n)))) \\;\text{ kwa wote } n ≤ b.
\end{align}$$

Pia, definiši $S(a, b, c) = \displaystyle\sum_{n = 0}^b F(n)$.

Primer, ako su $a = 50$, $b = 2000$ i $c = 40$, onda je $F(0) = 3240$ i $F(2000) = 2040$. Takođe, $S(50, 2000, 40) = 5\\,204\\,240$.

Pronađi poslednje 9 cifara za $S({21}^7, 7^{21}, {12}^7)$.
