# --description--

Izjava za `switch` se koristi za izvršavanje različitih radnji u zavisnosti od različitih uslova. Slična je izjavi za `if-else`, ali je lakša za čitanje i razumevanje kada postoji mnogo uslova koje treba proveriti. Izjava za `switch` se koristi za odabir jednog od više kodnih delova/sekcija za izvršenje.

Izjava za `switch` procenjuje matematički izraz poredeći ga sa vrednostima svakog slučaja. Ako postoji jednačina, rešavan je povezani deo koda. Ako ne postoji jednačina, rešavan je deo koda za default.

```javascript
let day = "Monday";

switch (day) {
  case "Monday":
    console.log("Today is Monday");
    break;
  case "Tuesday":
    console.log("Today is Tuesday");
    break;
  case "Wednesday":
    console.log("Today is Wednesday");
    break;
  case "Thursday":
    console.log("Today is Thursday");
    break;
  case "Friday":
    console.log("Today is Friday");
    break;
  case "Saturday":
    console.log("Today is Saturday");
    break;
  case "Sunday":
    console.log("Today is Sunday");
    break;
  default:
    console.log("Invalid day");
}
```

U gornjem primeru, vrednost modifikacije `day` se poredi sa vrednostima svakog slučaja. Ako postoji podudarnost, identifikovan će biti povezani deo koda. Ako ne postoji podudarnost, identifikovan će biti deo koda default.

## --text--

Na osnovu fragmenta koda JavaScript prikazanog ispod, koji menja vrednost `activity` u zavisnosti od dana u nedelji, koja će biti vrednost `activity` ako je vrednost `day` `"Tuesday"`?

```javascript
let day = "Tuesday";
let activity;

switch (day) {
  case "Monday":
    activity = "Go to the gym";
    break;
  case "Tuesday":
    activity = "Attend coding meetup";
    break;
  case "Wednesday":
    activity = "Watch a movie";
    break;
  case "Thursday":
    activity = "Visit a museum";
    break;
  case "Friday":
    activity = "Dinner with friends";
    break;
  case "Saturday":
    activity = "Hiking in the mountains";
    break;
  case "Sunday":
    activity = "Rest at home";
    break;
  default:
    activity = "Undefined day";
}

```

## --answers--

Vrednost promene `activity` će biti `"Attend coding meetup"`.
