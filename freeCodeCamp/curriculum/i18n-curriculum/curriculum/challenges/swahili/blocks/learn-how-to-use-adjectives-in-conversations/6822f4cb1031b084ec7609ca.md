# --explanation--

`Looks better` to znači nešto što izgleda privlačnije ili je prijatno za oko. Na primer:

`The updated login screen looks better with the new icons.` – Dizajn je poboljšan i privlačniji.

`Works better` To znači nešto što radi efikasnije ili je lakše za upotrebu. Na primer:

`The search feature works better after the update.` – Radi brže ili pruža preciznije rezultate.

Struktura `not only ... but also ...` se koristi za prikazivanje dve povezane pozitivne promene. U ovom kontekstu, nova verzija povećava kvalitet u penampilanu i performansama.
