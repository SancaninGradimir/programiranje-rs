# --description--

Middleware na nivou rutera je isto kao middleware na nivou aplikacije, ali je vezan za primer `express.Router()` umesto glavnog primjera programa. Može se koristiti za podelu vašeg programa Express kreiranjem zasebnih modula putanja podataka.

Middleware na nivou rutera radi na isti način kao middleware na nivou aplikacije, ali se koristi za specifične obrasce rutiranja. To čini lakšim korišćenje middleware-a za određeni skup putanja podataka umesto opšte upotrebe.

```js
const express = require('express')
const app = express()
const router = express.Router()

// Router-level middleware to log requests to '/menu' routes
router.use((req, res, next) => {
  console.log('Request made to /menu route')
  next() // Pass control to the next handler
})

// Define routes
router.get('/drinks', (req, res) => {
  res.send('Welcome to the drinks menu!')
})

app.use('/menu', router) // Apply the router

app.listen(3000, () => {
  console.log('Server running on http://localhost:3000')
})
```

* `router.use()`: Ovo je middleware na nivou rutera, koji se koristi samo za putanje podataka koje obrađuje ruter.
* `app.use('/menu', router)`: Middleware se koristi samo za putanje podataka koje prethode `/menu` (npr., `/menu/drinks`).

Middleware na nivou rutera omogućava vam da razdelite vašu aplikaciju na modularne delove i namerno koristite middleware za grupu putanja podataka. Ovo pomaže u organizaciji i čitljivosti koda.
