## Alati za Python

- **Uvod**: U Python, tip podataka liste je organizovan red elemenata koji mogu biti niz slova, brojeva ili čak drugih lista. Lista je dinamična i ima indeksiranje koje počinje od nule.

```python
cities = ['Los Angeles', 'London', 'Tokyo']
```

- **Pristup elementima u listi**: Da biste dobili element iz liste `cities`, možete se osvrnuti na njegov indeks u nizu:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[0] # Los Angeles
```

- **Dohvatanje elemenata korišćenjem negativnog selektora**: Da biste dobili poslednji element bilo koje liste, možete koristiti `-1` kao identifikator selektora:

```python
cities = ['Los Angeles', 'London', 'Tokyo']
cities[-1] # Tokyo
```

- Negativni obrazac se koristi za dobijanje elemenata počevši od kraja liste, a ne od početka, za obrazac `0`.

- **Kreiranje liste korišćenjem komponente `list()`**: Liste se mogu kreirati takođe i koristeći komponentu `list()`. Komponenta `list()` koristi se za konverziju ponovljivog elementa u listu:

```python
developer = 'Jessica'

print(list(developer)) 
# Result: ['J', 'e', 's', 's', 'i', 'c', 'a']
```

- **Dobijanje dužine liste**: Možete koristiti funkciju `len()` da dobijete dužinu liste:

```python
numbers = [1, 2, 3, 4, 5]
len(numbers) # 5
```

- **Izmena liste**: Lista je dinamička, što znači da možete izmeniti bilo koji element unutar liste samo ako dostavite validan ključni broj. Da biste izmenili listu za određeni ključ, možete postaviti novu vrednost za taj ključ:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[0] = 'JavaScript'
print(programming_languages) # ['JavaScript', 'Java', 'C++', 'Rust']
```

- **Greška modela izvan opsega**: Ako unesete parametar (pozitivan ili negativan) koji je van granica liste, dobićete `IndexError`:

```python
programming_languages = ['Python', 'Java', 'C++', 'Rust']
programming_languages[10] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list assignment index out of range
"""
```

- **Uklanjanje stavki iz liste**: Stavke se mogu ukloniti sa liste korišćenjem ključne reči `del`:

```python
developer = ['Jane Doe', 23, 'Python Developer']
del developer[1]
print(developer) # ['Jane Doe', 'Python Developer']
```

- **Provera da li je element u listi**: Glavna funkcija `in` može se koristiti za proveru da li je element u listi:

```py
programming_languages = ['Python', 'Java', 'C++', 'Rust']

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **Ugnježđivanje lista unutar drugog lista**: Liste se mogu organizovati/porediti unutar drugog lista:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
```

- Da biste dobili interno sortirani spisak, moraćete da ga dobijete koristeći `2` jer liste imaju indeks koji počinje od nule.

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2] # ['Python', 'Rust', 'C++']
```

- Da biste dobili drugi jezik iz liste koja je unutra, moraćete ga dobiti koristeći šablon `1`:

```python
developer = ['Alice', 25, ['Python', 'Rust', 'C++']]
developer[2][1] # Rust
```

- **Izvlačenje vrednosti iz liste:** Izvlačenje vrednosti iz liste je metoda koja se koristi za postavljanje vrednosti iz liste za nove parametre. Evo primera izvlačenja liste za `developer` za nove parametre nazvane `name`, `age` i `job`, kao što sledi:

```py
developer = ['Alice', 34, 'Rust Developer']
name, age, job = developer
```

- Ako broj parametara na levi strani operatera dodeljivanja ne odgovara ukupnom broju elemenata u spisku, dobićete `ValueError`.

- **Sakupljanje preostalih stavki sa liste**: Da biste sakupljali bilo koje preostale stavke sa liste, možete koristiti zvezdastu operaciju (`*`) kao što sledi:

```py
developer = ['Alice', 34, 'Rust Developer']
name, *rest = developer
```

- **Delimična obrada liste**: Procesiranje je koncepcija dobijanja dela liste korišćenjem operatora za procesiranje `:`. Da biste obradili listu počevši od elementa `1` i završili pre elementa `3`, možete koristiti sledeću sintaksu:

```python
desserts = ['Cake', 'Cookies', 'Ice Cream', 'Pie']
desserts[1:3] # ['Cookies', 'Ice Cream']
```

- **Inkrementalni korak**: Takođe je moguće specificirati inkrementalni korak koji određuje koliko se vrednost povećava. Evo primera ako želite da izbacite samo listu pozitivnih celih brojeva koristeći obradu:

```python
numbers = [1, 2, 3, 4, 5, 6]
numbers[1::2] # [2, 4, 6]
```

## Načini za listu

- **append()**: Koristi se za dodavanje elementa na kraj liste. Evo primera korišćenja metode `append()` za dodavanje broja `6` u listu `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **Dodavanje druge liste**: Metoda `append()` se takođe može koristiti za dodavanje jedne liste na kraj druge:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

- **extend()**: Koristi se za dodavanje više stavki na kraj liste. Evo primera dodavanja brojeva `6`, `8`, i `10` na kraj liste `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

- **insert()**: Koristi se za umetanje elementa na određenoj poziciji unutar liste pomoću specifičnog primera/sintakse. Evo primera korišćenja metode `insert()`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

- **remove():** Koristi se za uklanjanje elementa iz liste. Metoda `remove()` će ukloniti samo prvo pojavljivanje elementa unutar liste:

```py
numbers = [1, 2, 3, 4, 5, 5, 5]
numbers.remove(5)

print(numbers) # [1, 2, 3, 4, 5, 5]

```

- **pop()**: Koristi se za uklanjanje određenog elementa iz niza i vraćanje njegove vrednosti:

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

- Ako ne navedete element na način `pop`, onda se briše poslednji element.

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

- **clear()**: Koristi se za uklanjanje svih elemenata sa liste:

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

- **sort()**: Metoda `sort()` se koristi za sortiranje elemenata na mestu. Evo primera sortiranja nasumičnog niza `numbers` na mestu:

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

- **sorted()**: Unutrašnja funkcija koja vraća novu sortiranu listu umesto da modifikuje originalnu listu:

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
print(numbers) # [19, 2, 35, 1, 67, 41]
```

- **reverse()**: Koristi se za menjanje redosleda elemenata u nizu:

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

- **index()**: Koristi se za dobijanje indeksa prvog elementa koji se može pronaći u nizu:

```python
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

- Ako element ne može biti dostupan putem `index()`, onda će rezultat biti `ValueError`.

## Tuple u Python

- **Opis**: Tuple je tip podataka Python koji se koristi za kreiranje usrednjenog niza vrednosti. Tuple mogu sadržati mešavinu različitih tipova podataka:

```py
developer = ('Alice', 34, 'Rust Developer')
```

- Tuple je nepromenljiv (immutable), jer se elementi unutar tuple-a ne mogu promeniti nakon kreiranja. Ako pokušate da modifikujete element tuple-a, dobićete `TypeError`:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object does not support item assignment
"""
```

- **Dobijanje elemenata iz tupaola**: Da biste dobili element iz tupaola, koristite notaciju zagrada i indeks:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

- Negativni indeks se može koristiti za dobijanje elemenata počevši od kraja tupela:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

- Ako pokušate da unesete indeks broj koji prelazi ili je jednak dužini tupola, dobićete `IndexError`:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: tuple index out of range
"""
```

- Tuple se takođe može kreirati koristeći komponentu `tuple()`. Unutar komponente, možete uneti ponovljive elemente kao niz karaktera, listu i čak drugi tuple.

```python
developer = 'Jessica'

print(tuple(developer)) 
# Result: ('J', 'e', 's', 's', 'i', 'c', 'a')
```

- **Provera elemenata u tupelu**: Da biste proverili da li je element u tupelu, možete koristiti ključnu reč `in` kao što sledi:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

- **Izvlačenje elemenata iz tupola**: Elementi se mogu izvlačiti iz tupola na sledeći način:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer
```

- Ako trebate sakupiti preostale elemente iz tupaola, možete koristiti zvezdasti operator (`*`) kao što sledi:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer
```

- **Procesiranje tupa**: Procesiranje se može koristiti za izvlačenje dela tupa. Na primer, stavke `pie` i `cookies` mogu se procesirati u različite tuple:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

- **Uklanjanje elemenata iz tuple-a**: Uklanjanje elementa iz tuple-a će izazvati `TypeError` jer su tuple-i nepromenljivi:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

- **Kada koristiti tuple umesto liste?**: Ako vam je potreban promenljiv skup stavki gde možete dodavati, uklanjati i menjati elemente, koristite listu. Ako znate da radite sa fiksnom i nepromenljivom kolekcijom podataka, koristite tuple.

## Uobičajene metode za tuple

- **`count()`**: Koristi se za određivanje koliko puta je element prisutan u tuplu. Na primer, možete proveriti koliko puta se jezik `'Rust'` pojavljuje u tuplu:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('Rust') # 2
```

- Ako element pomenut u putanji `count()` potpuno nije prisutan u tuple, onda vrednost return će biti `0`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.count('JavaScript') # 0
```

- Ako nisu navedeni argumenti putem `count()`, onda Python vraća `TypeError`.

- **index()**: Koristi se za pronalaženje indeksa gde se određeni element nalazi u tuplu. Evo primera korišćenja metode `index()` za dobijanje indeksa jezika `'Java'`:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust')
programming_languages.index('Java') # 1
```

- Ako navedeni element ne može biti pronađen, onda će Python vratiti `ValueError`.

- Takođe možete uneti početni obrazac pomoću `index()` da specificirate gde ćete početi pretraživati element u tuplu:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
programming_languages.index('Python', 3) # 5
```

- Takođe možete umetnuti poslednji primer koristeći `index()` da biste specificirali gde ćete prestati pretraživati element u tupelu:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python', 'JavaScript', 'Python')
programming_languages.index('Python', 2, 5) # 2
```

- **`sorted()`**: Koristi se za aranžiranje elemenata u bilo kom ponovljivom objektu i vraćanje nove, organizovane liste. Evo primera kreiranja nove numerisane liste korišćenjem funkcije `sorted()`:

```py
numbers = (13, 2, 78, 3, 45, 67, 18, 7)
sorted(numbers) # [2, 3, 7, 13, 18, 45, 67, 78]
```

- **Измена ponašanja sortiranja**: Možete prilagoditi ponašanje sortiranja za ponovljivi element koristeći opcionalne argumente ``reverse`` i ``key``. Evo primera korišćenja argumenta ``key`` za sortiranje elemenata u tupel prema dužini:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')
sorted(programming_languages, key=len)

# Result
# ['C++', 'Rust', 'Java', 'Rust', 'Python', 'Python']
```

- Možete kreirati novu listu vrednosti u obrnutom redosledu, koristeći argument `reverse`, kao što sledi:

```py
programming_languages = ('Rust', 'Java', 'Python', 'C++', 'Rust', 'Python')

print(sorted(programming_languages, reverse=True))

# Result
# ['Rust', 'Rust', 'Python', 'Python', 'Java', 'C++']
```

## Krugovi u Python

- **Opis**: Petlje se koriste za ponavljanje dela koda određeni broj puta.

- **Petlja za `for`**: Koristi se za iteriranje preko sekvence (poput liste, tupa ili niza karaktera) i izvršavanje bloka koda za svaki element u toj sekvenci. Evo primera korišćenja petlje `for` za iteriranje preko liste i štampanje svakog elementa na konzolu:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']

for language in programming_languages:
    print(language)

"""
Result 

Rust
Java
Python
C++
"""
```

- Ovo je primer korišćenja petlje `for` za ponavljanje sekvence slova `code` i štampanje svakog slova:

```py
for char in 'code':
    print(char)

"""
Result 

c
o
d
e
"""
```

- Krugovi `for` mogu se ugraditi unutar. Evo primera korišćenja kruga `for` koji je ugrađen:

```py
categories = ['Fruit', 'Vegetable']
foods = ['Apple', 'Carrot', 'Banana']

for category in categories:
    for food in foods:
        print(category, food)

"""
Result

Fruit Apple
Fruit Carrot
Fruit Banana
Vegetable Apple
Vegetable Carrot
Vegetable Banana
"""

```

- **Petlja za `while`**: Ponovi deo koda dok uslov nije `False`. Evo primera korišćenja petlje `while` za igru na sreću:

```py
secret_number = 3
guess = 0

while guess != secret_number:
    guess = int(input('Guess the number (1-5): '))
    if guess != secret_number:
        print('Wrong! Try again.')

print('You got it!')

"""
Result

Guess the number (1-5): 2
Wrong! Try again.
Guess the number (1-5): 1
Wrong! Try again.
Guess the number (1-5): 3
You got it!
"""

```

- **Kifungu cha `break` na `continue`**: Koristi se u krugovima za promenu izvršenja kruga.

- Klauzula `break` se koristi jednom po ciklusu kada se ispuni određeni uslov. Evo primera korišćenja klauzule `break` za listu `developer_names`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        break
    print(developer)
```

- Deo `continue` se koristi za preskakanje te iteracije i nastavak sa sledećom iteracijom ciklusa. Evo primera korišćenja deoa `continue` umesto deoa `break`:

```py
developer_names = ['Jess', 'Naomi', 'Tom']

for developer in developer_names:
    if developer == 'Naomi':
        continue
    print(developer)
```

- Ciklusi `for` i `while` mogu biti povezani sa odjeljkom `else`, koji se izvršava samo kada ciklus ne završi sa odjeljkom `break`:

```python
words = ['sky', 'apple', 'rhythm', 'fly', 'orange']

for word in words:
    for letter in word:
        if letter.lower() in 'aeiou':
            print(f"'{word}' contains the vowel '{letter}'")
            break
    else:
        print(f"'{word}' has no vowels")
```

## Ranges i njihova primena u krugovima

- **Funkcija `range()`**: Koristi se za generisanje sekvence celobrojnih brojeva.

```py
range(start, stop, step)
```

- Važan argument za `stop` je kompletan (neinkluzivan) broj koji predstavlja kraj generisane sekvence brojeva. Evo primera korišćenja funkcije `range()`:

```py
for num in range(3):
    print(num)
```

- Ako argument ``start`` nije dostavljen, podrazumevano će biti ``0``. Za podrazumevanu sekvencu celih brojeva, inkrement će biti ``1``. Možete koristiti opcioni argument ``step`` da promenite vrednost podrazumevanog inkrementa. Evo primera generisanja sekvence pozitivnih celih brojeva od 2 do 10 (uključujući 10).

```py
for num in range(2, 11, 2):
    print(num)
```

- Ako ne pružite nijedan argument za funkciju `range()`, onda ćete dobiti `TypeError`.

- Funkcija `range()` prihvata samo cele brojeve kao argumente, a ne decimalne brojeve. Korišćenjem decimalnih brojeva će doći do `TypeError`:

```py
ERROR!
Traceback (most recent call last):
  File "<main.py>", line 1, in <module>
TypeError: 'float' object cannot be interpreted as an integer
```

- Možete koristiti negativan ceo broj za argument `step` da biste generisali niz celih brojeva u opadajućem redosledu:

```py
for num in range(40, 0, -10):
    print(num)
```

- Funkcija `range()` takođe se može koristiti za kreiranje liste celobrojnih brojeva korišćenjem zajedno sa komponentom `list`. Komponenta `list` koristi se za konverziju iterabilnog objekta u listu. Evo primera generisanja liste pozitivnih celobrojnih brojeva između 2 i 10, uključujući:

```py
numbers = list(range(2, 11, 2))
print(numbers) # [2, 4, 6, 8, 10]
```

## Funkcija `enumerate()` i `zip()` u Python

- **`enumerate()`**: Koristi se za iteraciju kroz niz i praćenje indeksa svakog elementa u tom nizu. Funkcija `enumerate()`, koja uzima iterabilni objekat kao argument i vraća objekat `enumerate` koji uključuje indeks i vrednost svakog elementa u iterabilnom objektu.

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

for index, language in enumerate(languages):
    print(f'Index {index} and language {language}')

# Result
# Index 0 and language Spanish
# Index 1 and language English
# Index 2 and language Russian
# Index 3 and language Chinese

```

- Funkcija `enumerate()` takođe se može koristiti izvan opsega `for`:

```python
languages = ['Spanish', 'English', 'Russian', 'Chinese']

print(list(enumerate(languages)))
# [(0, 'Spanish'), (1, 'English'), (2, 'Russian'), (3, 'Chinese')]
```

- Funkcija `enumerate()` takođe prima opcionu argumentu `start` koja određuje početnu vrednost za brojanje. Ako ovaj argument nije dostavljen, onda će brojanje početi od `0`.

- **`zip()`** : Koristi se za ponavljanje preko više stavki koje mogu biti ponovljene paralelno. Evo primera korišćenja funkcije `zip()` za ponavljanje preko `developers` i `ids`:

```python
developers = ['Naomi', 'Dario', 'Jessica', 'Tom']
ids = [1, 2, 3, 4]

for name, id in zip(developers, ids):
    print(f'Name: {name}')
    print(f'ID: {id}')


"""
Result

Name: Naomi
ID: 1
Name: Dario
ID: 2
Name: Jessica
ID: 3
Name: Tom
ID: 4
"""

```

## Razumevanje liste u Python

- **Opis**: List komprehensija vam omogućava da kreirate novi list u jednoj liniji kombinujući petlju i uslov direktno unutar uglastih zagrada. To čini kod kraćim i često lakšim za čitanje.

```py
even_numbers = [num for num in range(21) if num % 2 == 0]
print(even_numbers)
```

## Načini za ponavljajuće stavke

- **`filter()`**: Koristi se za filtriranje elemenata iz iterabilnog objekta prema određenom uslovu. Vraća generator sa elementima koji zadovoljavaju taj uslov. Evo primera kreiranja nove liste reči koje imaju više od četiri slova:

```py
words = ['tree', 'sky', 'mountain', 'river', 'cloud', 'sun']

def is_long_word(word):
    return len(word) > 4

long_words = list(filter(is_long_word, words))
print(long_words) # ['mountain', 'river', 'cloud']
```

- **`map()`**: Koristi se za korišćenje funkcije za svaki element u iterativnom objektu i vraćanje novog objekta sa rezultatima. Evo primera korišćenja funkcije `map()` za konverziju liste temperatura iz Celzija u Farenhajta:

```py
celsius = [0, 10, 20, 30, 40]

def to_fahrenheit(temp):
    return (temp * 9/5) + 32

fahrenheit = list(map(to_fahrenheit, celsius))
print(fahrenheit) # [32.0, 50.0, 68.0, 86.0, 104.0]
```

- **`sum()`**: Koristi se za dobijanje zbira iz iterabilnog objekta kao što su lista ili tupel. Evo primera korišćenja funkcije `sum()`:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers)
print(total) # Result: 50
```

- Takođe možete uključiti opcionalni argument `start` koji postavlja početnu vrednost za ukupno. Evo ažuriranog primera korišćenja argumenta `start` kao pozicionog parametra:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, 10) # positional argument
print(total) # 60
```

- Takođe možete izabrati da koristite argument `start` kao glavni argument reči, kako sledi:

```py
numbers = [5, 10, 15, 20]
total = sum(numbers, start=10) # keyword argument
print(total) # 60
```

## Lambda funkcija

- **Opis**: Lambda funkcija u Python je kratak način za kreiranje anonimne funkcije (anonimna funkcija).
- Lambda funkcija se često koristi kao argument za drugu funkciju. Evo primera lambda funkcije:

```py
numbers = [1, 2, 3, 4, 5]

even_numbers = list(filter(lambda x: x % 2 == 0, numbers))
print(even_numbers)  # [2, 4]
```

- Najbolji načini korišćenja lambda funkcija su izostavljanje njene vrednosti u parametru, čime se čini jednostavnim i razumljivim, i korišćenje za kratke, jednokratne funkcije.
