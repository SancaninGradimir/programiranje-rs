# --description--

Lako je dokazati da ne postoji trougao čije su sve stranice i površina cele brojeve. Međutim, gotovo izoscelesni trougao sa stranicama 5-5-6 ima površinu od 12 kvadratnih jedinica.

Definisaćemo <dfn>gotovo jednakostranični trougao</dfn> kao trougao kod kojeg su dve stranice jednake, a treća se razlikuje za najviše jednu jedinicu.

Pronađi zbir obima svih gotovo jednakostraničnih trouglova sa celobrojnim dužinama stranica i celobrojnom površinom čiji obim ne prelazi `limit`.

```js
function almostEquilateralTriangles(limit) {
  // Based on https://blog.dreamshire.com/project-euler-94-solution/
  let perimetersSum = 0;

  let sidesAB = 1;
  let sideC = 1;
  let perimeter = 0;
  let perimeterOffset = 1;

  while (perimeter <= limit) {
    [sidesAB, sideC] = [4 * sidesAB - sideC + 2 * perimeterOffset, sidesAB];
    perimeterOffset = -perimeterOffset;

    perimetersSum += perimeter;
    perimeter = 3 * sidesAB - perimeterOffset;
  }

  return perimetersSum;
}
```
