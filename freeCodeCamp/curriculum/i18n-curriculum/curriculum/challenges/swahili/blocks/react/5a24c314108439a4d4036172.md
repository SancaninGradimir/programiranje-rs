# --description--

Postoji još jedan način pristupa `state` u komponenti. U metodi `render()`, pre naredbe `return`, možeš direktno pisati JavaScript. Na primer, možeš definisati funkcije, dohvatiti podatke iz `state` ili `props`, izvršiti proračune i slično. Zatim možeš koristiti bilo koju od tih vrednosti unutar izraza `return`.

# --instructions--

U toku rendera `MyComponent`, definišite `const` nazvano `name` i postavite ga jednak vrednosti imena u `state` sekcije. Budući da možete direktno napisati JavaScript u ovu kod sekciju, ne morate obuhvatati ovu referencu uglastim zagradama.

Zatim u izrazu `return` prikaži ovu vrednost u elementu `h1` koristeći promenljivu `name`. Zapamti da moraš koristiti JSX sintaksu (vitičaste zagrade za JavaScript) u izrazu `return`.

# --hints--

`MyComponent` treba imati `name` sa vrednošću `freeCodeCamp` sačuvanu u svom stanju.

```js
assert(
  Enzyme.mount(React.createElement(MyComponent)).state('name') ===
    'freeCodeCamp'
);
```

`MyComponent` treba da prikaže naslov sekcije za `h1`, okružen jednim `div`.

```js
assert(
  /<div><h1>.*<\/h1><\/div>/.test(
    Enzyme.mount(React.createElement(MyComponent)).html()
  )
);
```

Etiketa za `h1` koja je nacrtana mora da uključuje referencu na `{name}`.

```js
assert(/<h1>\n*\s*\{\s*name\s*\}\s*\n*<\/h1>/.test(code));
```

Zaglavlje dela `h1` koje je prikazano treba da ima tekst koji se prikazuje iz stanja dela.

```js
  const waitForIt = (fn) =>
    new Promise((resolve, reject) => setTimeout(() => resolve(fn()), 250));
  const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
  const first = () => {
    mockedComponent.setState({ name: 'TestName' });
    return waitForIt(() => mockedComponent.html());
  };
  const firstValue = await first();
  assert(firstValue === '<div><h1>TestName</h1></div>');
```
