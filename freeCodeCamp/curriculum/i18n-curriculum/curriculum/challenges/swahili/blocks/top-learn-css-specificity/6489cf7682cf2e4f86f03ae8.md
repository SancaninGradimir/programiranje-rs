# --description--

Nasleđivanje se odnosi na određena svojstva za CSS koja, kada se primene na element, nasleđuju se i potomci tog elementa, čak i ako ne pišemo pravila direktno za te potomke. Svojstva povezana sa tipom fonta (`color`, `font-size`, `font-family`, itd.) obično se nasleđuju, dok mnogi drugi svojstva nisu nasleđivi.

Razlika je kada ciljate direktno na element, jer ovo uvek pobjeđuje nasljeđe:

```html
<!-- index.html -->

<div id="parent">
  <div class="child"></div>
</div>
```

```css
/* styles.css */

#parent {
  color: red;
}

.child {
  color: blue;
}
```

Iako element `parent` ima veću specifičnost korišćenjem ID-a, element `child` će dobiti stil `color: blue` jer ta deklaracija direktno cilja, dok je `color: red` od roditelja samo nasleđivanje.
