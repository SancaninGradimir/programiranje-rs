`We uploaded the new website features to the staging server for testing.` – U ovoj rečenici `staging server` je testno okruženje koje se koristi kako bi se proverilo da li nove funkcije veb sajta rade ispravno pre nego što budu objavljene.

`We can finish the task today, can't we?` - U ovoj rečenici, pozitivna izjava (`We can finish the task today`) je praćena negativnim pitanjem (`can't we?`) radi proveravanja saglasnosti ili potvrde.
