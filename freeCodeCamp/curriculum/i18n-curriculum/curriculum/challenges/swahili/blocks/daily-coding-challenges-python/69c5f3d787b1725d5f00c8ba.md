# --description--
Razmatrajući dva trenutka, odluči da li možeš stići od prvog do drugog brže tako što ćeš se pomeriti napred ili nazad.

- Vremenski podaci su prikazani u 24-satnom formatu (`"HH:MM"`)
- Sat se vraća (23:59 prelazi u 00:00 kada se kreće napred, a 00:00 prelazi u 23:59 kada se kreće unazad)

Return:

- `"forward"` ako je kretanje napred kraće
- `"backward"` ako je kretanje unazad kraće
- `"equal"` ako oba pravca traju isto vreme

```py
def get_direction(time1, time2):
    def to_minutes(t):
        h, m = map(int, t.split(':'))
        return h * 60 + m

    start = to_minutes(time1)
    end = to_minutes(time2)
    forward = (end - start + 1440) % 1440
    backward = (start - end + 1440) % 1440

    if forward < backward:
        return 'forward'
    if backward < forward:
        return 'backward'
    return 'equal'
```
