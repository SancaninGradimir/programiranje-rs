# --explanation--

`Next to` znači pored ili vrlo blizu nečega u prostoru ili mestu.

U pravilnoj rečenici, štampač je smešten pored fizičkog stola.

Neke rečenice loše koriste `next to`, bilo u pogrešnom vremenu ili kontekstu.
