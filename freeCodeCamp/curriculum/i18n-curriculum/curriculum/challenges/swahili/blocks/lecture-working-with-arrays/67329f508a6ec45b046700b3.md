# --interactive--

Niz podataka u JavaScript je organizovana kolekcija vrednosti, svaka identifikovana numeričkim indeksom. Vrednosti u nizu podataka za JavaScript mogu imati različite tipove podataka, uključujući brojeve, sekvence slova, logičke vrednosti (tačno/netačno), objekte i čak druge nizove podataka.

Da biste kreirali red podataka u JavaScript, možete koristiti uglaste zagrade, ``[]``, i razdvojiti vrednosti zarezom. Evo primera:

```js
let fruits = ["apple", "banana", "orange"];
```

U ovom primeru, deklarišemo parametar `fruits` i postavljamo mu vrednost na niz podataka sa sekvencom od tri karaktera: `apple`, `banana`, i `orange`.

Jedna od glavnih karakteristika nizova je da koriste indeksiranje koje počinje od nule, što znači da prvi element ima indeks `0`, drugi indeks `1` i tako dalje. Pojedinačnim elementima možeš da pristupiš pomoću njihovog indeksa. Na primer:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits[0]); // "apple"
console.log(fruits[2]); // "orange"
```

:::

U ovom primeru, koristimo model `0` da bismo dobili prvi element (`apple`) i model `2` da bismo dobili treći element (`orange`).

Redovi podataka u JavaScript imaju posebnu karakteristiku `length` koja vraća broj elemenata sadržanih u redu. Možete dobiti ovu karakteristiku korišćenjem karakteristike `length`. Primer:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
console.log(fruits.length); // 3
```

:::

Još jedna glavna karakteristika nizova podataka u JavaScript je što su dinamički (ili: promenljivi), što znači da njihova veličina može da se promeni nakon kreiranja. Možete dodati ili ukloniti elemente iz niza koristeći različite metode nizova, kao što su `push()`, `pop()`, `shift()`, `unshift()`, `splice()`, i druge. Ove metode će biti objašnjene u budućim lekcijama.

JavaScript nizovi su veoma fleksibilni i izuzetno korisni za čuvanje podataka u aplikacijama. U ovom modulu ćeš detaljnije naučiti kako rad sa nizovima pomaže u efikasnom upravljanju i izmeni kolekcija podataka.
