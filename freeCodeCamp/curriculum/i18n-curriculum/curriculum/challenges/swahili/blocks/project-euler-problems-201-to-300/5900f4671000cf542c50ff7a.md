# --description--

Trijplet pozitivnih celih brojeva ($a$,$b$,$c$) naziva se Trijplet Kardana ako zadovoljava uslov:

$$\sqrt[3]{a + b \sqrt{c}} + \sqrt[3]{a - b \sqrt{c}} = 1$$

Na primer, (2,1,5) je Cardano Triplet.

Postoje Triplets 149 za Cardano koje $a + b + c ≤ 1000$.

Pronađi koliko Cardano Triplets-a postoji uslovljenim da $a + b + c ≤ 110\\,000\\,000$.
