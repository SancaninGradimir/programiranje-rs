# --interactive--

Kada radite sa pozadinskim slikama u CSS, imate nekoliko funkcija za kontrolisanje načina na koji su ove slike prikazane.

Glavne karakteristike koje ćemo uzeti u obzir su `background-size`, `background-repeat`, `background-position`, na `background-attachment`.

Prvo, razmotrimo karakteristiku od `background-image`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

CSS Iznad postavlja sliku mačke kao pozadinu za element `body`.

Ako želite postaviti veličinu pozadinske slike, možete koristiti svojstvo `background-size`.

Možete koristiti `contain` da proširite sliku da bude što veća moguće bez sečenja ili skaliranja.

Evo primera koji ima `background-size: contain`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    min-height: 100px;
  }
</style>
```

:::

Postavljamo `min-height` na `100px` kako bi pozadinska slika bila vidljiva i kako bi raspored zadržao minimalnu visinu, osiguravajući da izgleda lepo čak i kada ima malo sadržaja.

Ako promenimo svojstvo `background-size` koristeći vrednost `cover`, onda će pozadinska slika se proširiti da pokrije ceo element od `body` uz zadržavanje svog aspektnog ommera.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    min-height: 100px;
  }
</style>
```

:::

U prethodnim primerima, možda ste primetili da se pozadinska slika često ponavlja.

Po podrazumevanju, pozadinske slike se ponavljaju vertikalno i horizontalno kako bi popunile ceo element. Međutim, možete kontrolisati ovo ponašanje.

Možete koristiti atribut `background-repeat` i njegovu vrednost postavljenu na `no-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

:::

Pošto je `background-size` postavljeno na `contain` i `background-repeat` je postavljeno na `no-repeat`, slika više neće biti prikazana na ekranu.

Ako želite da duplirate pozadinsku sliku ravnomerno, možete koristiti vrednost `repeat-x` za svojstvo `background-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-x;
    min-height: 100px;
  }
</style>
```

:::

Ako želite da ponovite pozadinsku sliku vertikalno, možete koristiti vrednost `repeat-y` za svojstvo `background-repeat`.

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-y;
    min-height: 100px;
  }
</style>
```

:::

Da biste postavili pozadinsku sliku negde na ekranu, možete koristiti svojstvo `background-position`.

Svojstvo `background-position` vam omogućava da postavite lokaciju unutar elementa gde će se pojaviti pozadinska slika. Možete koristiti ključne reči kao `top`, `bottom`, `left`, `right`, i `center`, ili specifične vrednosti u pikselima ili procentima.

Ovo je primer koji koristi `center` i `top` za `background-position`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

:::

Ovaj kod za CSS postavlja pozadinsku sliku centrirano horizontalno i vertikalno.

Na kraju, `background-attachment` određuje da li će pozadinska slika pomerati sa sadržajem ili će ostati fiksna kada se stranica skroluje.

Glavna vrednost je `scroll` (po podrazumevanju), gde se pozadinska slika kreće sa sadržajem, i `fixed`, gde pozadinska slika ostaje na mestu na ekranu.

Ovde je primer koji koristi vrednost od `fixed` za svojstvo `background-attachment`:

:::interactive_editor

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-position: center top;
    background-attachment: fixed;
  }
</style>
```

:::

Ovaj kod od CSS postavlja pozadinsku sliku koja ostaje fiksirana na mestu čak i kada se stranica skroluje.

Ako želite da kombinujete nekoliko karakteristika u jednoj liniji, možete to učiniti koristeći kratku karakteristiku ``background``.

Evo primera:

:::interactive_editor

```html
<style>
  body {
    background: center top fixed
      url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

:::

Primer iznad povezuje `background-image`, `background-position`, i `background-attachment` u jednoj liniji.

Sa ovim funkcijama, imaćete odličnu kontrolu nad načinom prikazivanja pozadinskih slika u vašem web dizajnu, omogućavajući estetski privlačne i funkcionalne postavke.
