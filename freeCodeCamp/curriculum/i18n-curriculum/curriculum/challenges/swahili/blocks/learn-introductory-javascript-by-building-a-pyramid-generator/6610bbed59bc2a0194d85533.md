# --description--

Pogledajte kako je vrednost unutar reda podataka `rows` automatski promenjena? Ovo se zove <dfn>mabadiliko</dfn>. Kada naučite više o redovima podataka, naučićete kada menjati red podatke, i kada ne biste trebali.

Pre nego što nastavite, ovo je sjajna prilika za učenje uobičajenih upotreba nizova podataka. Trenutno, vaš kod dobija poslednji element iz niza podataka korišćenjem `rows[2]`. Međutim, možda ne znate koliko elemenata je u nizu podataka kada želite poslednji element.

Možete koristiti svojstvo `.length` reda podataka – ovo vraća broj elemenata u redu podataka. Da biste dobili poslednji element bilo kog reda podataka, možete koristiti sledeću sintaksu:

```js
array[array.length - 1]
```

`array.length` vraća broj elemenata u nizu podataka. Pruživanjem `1`, dobijate indeks poslednjeg elementa u nizu podataka. Možete koristiti ovaj koncept za niz podataka `rows`.

Ispravite vašu `rows[2]` kako biste prisilno dobili poslednji element u redu podataka `rows`. Pogledajte primer iznad radi pomoći.

Trebalo bi da vidiš bilo kakve promene u svojoj konzoli.
