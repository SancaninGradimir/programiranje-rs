# --description--

Izračunaj sve tri <a href="https://en.wikipedia.org/wiki/Pythagorean_means" target="_blank" rel="noopener noreferrer nofollow">Pitagorine sredine</a> za skup celih brojeva od $1$ do $10$ (uključujući oba broja).

Pokaži da za ovaj skup pozitivnih celih brojeva važi $A(x_1,\ldots,x_n) \geq G(x_1,\ldots,x_n) \geq H(x_1,\ldots,x_n)$.

<ul>
<li>Najčešće korišćena od ove tri sredine, <a class='rosetta__link--rosetta' href='https://rosettacode.org/wiki/Averages/Arithmetic_mean' title='Averages/Arithmetic mean' target='_blank'>aritmetička sredina</a>, jednaka je zbiru elemenata liste podeljenom njenom dužinom:<br>
  <big>$ A(x_1, \ldots, x_n) = \frac{x_1 + \cdots + x_n}{n}$</big></li>
<li><a class='rosetta__link--wiki' href='https://en.wikipedia.org/wiki/Geometric_mean' title='wp: Geometric mean' target='_blank'>Geometrijska sredina</a> je n-ti koren proizvoda svih elemenata liste:<br>
  <big>$ G(x_1, \ldots, x_n) = \sqrt[n]{x_1 \cdots x_n} $</big></li>
<li><a class='rosetta__link--wiki' href='https://en.wikipedia.org/wiki/Harmonic_mean' title='wp: Harmonic mean' target='_blank'>Harmonijska sredina</a> jednaka je broju elemenata podeljenom zbiru recipročnih vrednosti svih elemenata liste:<br>
  <big>$ H(x_1, \ldots, x_n) = \frac{n}{\frac{1}{x_1} + \cdots + \frac{1}{x_n}} $</big></li>
</ul>

# --instructions--

Kada pišeš funkciju, pretpostavi da je ulaz sortiran niz koji sadrži samo brojeve.

U odgovoru navedite rezultat u sledećem formatu:

```js
    Geometric: 4.528728688116765,

  },

}
```
