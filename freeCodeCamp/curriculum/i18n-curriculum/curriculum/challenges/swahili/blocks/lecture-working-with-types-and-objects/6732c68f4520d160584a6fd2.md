# --interactive--

Metoda `toString()` je osnovna karakteristika u JavaScript koja konvertuje vrednost u niz karaktera koji predstavlja tu vrednost. To je metoda koju možete koristiti za brojeve, vrednosti tačno ili netačno, nizove podataka i objekte. Jedna od upotreba `toString()` je konvertovanje broja u niz karaktera koji predstavlja taj broj. Evo primera:

:::interactive_editor

```js
const num = 10;
console.log(num.toString()); // "10"
```

:::

Ova putanja prima opcionalnu osnovu broja koja je broj od `2` do `36`. Ova osnovna vrednost broja predstavlja bazu, kao što je baza 2 za binarni ili baza 8 za oktalni. Ako nije navedena baza, podrazumevano je baza 10, odnosno decimalni. Evo primera prosleđivanja `2` kao argumenta putem `toString()`:

:::interactive_editor

```js
const num = 10;
console.log(num.toString(2)); // "1010"
```

:::

Rezultat će biti `1010`, što je binarno predstavljanje za decimalni broj `10`.

Takođe možete koristiti metodu `toString()` da pretvorite redove podataka i elemente u niz znakova. Redovi podataka imaju specijalnu implementaciju `toString()` koja pretvara svaki element u niz znakova i spaja ih zarezima. Evo primera:

:::interactive_editor

```js
const arr = [1, 2, 3];
console.log(arr.toString()); // "1,2,3"
```

:::

U ovom primeru, svi elementi podataka u redu su spojeni zajedno kako bi stvorili niz znakova `1,2,3`.

Kada se metoda `toString()` koristi za objekte, rezultat neće biti string reprezentacija svojstava tog objekta.

:::interactive_editor

```js
const person = {
  name: "John",
  age: 30,
  isStudent: true
};

console.log(person.toString()); // "[object Object]"
```

:::

U ovom primeru, rezultat će biti reprezentacija podrazumevanog niza karaktera za objekat koji je `[object Object]`. Da biste dobili vrednost niza karaktera svojstava objekta `person`, moraćete da koristite `JSON.stringify()`, što ćete više naučiti u narednim lekcijama.

Zaključno, metoda `toString()` se koristi za konverziju vrednosti u niz znakova. Razumevanje kako funkcioniše sa različitim tipovima podataka i kako ga konvertovati za vaše sopstvene stvari može značajno povećati vašu sposobnost rukovanja i prikazivanja podataka u vašim aplikacijama za JavaScript.
