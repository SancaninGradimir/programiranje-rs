# --description--

Engine JavaScript ima sposobnost da čita, razume i izvrši vaš kod. Radi kao konverzija koja uzima vaš kod i pretvara ga u instrukcije koje računar može razumeti i izvršiti.

Jedan od najpoznatijih JavaScript motora je V8, koji je razvio Google i koji se koristi u Chrome-u i Node.js-u. JavaScript motor radi kroz nekoliko koraka. Prvo analizira tvoj kod, čitajući ga red po red kako bi proverio da li postoje greške. Zatim ga prevodi u bajtkod (bytecode), što je jednostavnija međuverzija koda koju računar lakše razume i izvršava. Na kraju izvršava taj bajtkod kako bi sproveo instrukcije tvog programa. Evo primera JavaScript koda:

```js
const greeting = "Hello, World!";
console.log(greeting);
```

Kada izvršite ovaj kod, motor JavaScript prvo ga analizira da bi proverio da li postoje sintaktičke greške. Analizirati znači da motor čita kod i razlaže ga u razumljivu strukturu, proveravajući greške tokom procesa.

Zatim, on ga kompajlira u međuformat (često je to *bytecode* ili mašinski kod, zavisno od engine-a). Kompilacija je proces pretvaranja koda koji je čovek razumljiv u optimizovaniji format koji računar može izvršiti brže.

Mwishowe, injini inatekeleza msimbo, ikichapisha `Hello, World!` kwenye konsoli.

Sada, hajde da pričamo o radnom okruženju za JavaScript. Radno okruženje JavaScript je okruženje u kojem se vaš kod JavaScript izvršava. Uključuje motor JavaScript (kao V8 u Chromu ili SpiderMonkey u Firefoxu), koji obrađuje i izvršava kod, zajedno sa dodatnim funkcijama koje pruža okruženje (kao što je veb pregledač ili Node.js, o čemu ćete saznati više u budućim lekcijama).

Dok sam JavaScript jezik upravlja stvarima poput promenljivih, petlji i funkcija, runtime obezbeđuje dodatne alate koji omogućavaju JavaScript-u da komunicira sa stvarima izvan samog jezika, kao što su DOM (za veb stranice) ili Fetch API (za mrežne zahteve).

Ukratko, runtime omogućava JavaScript-u da radi više od osnovnog programiranja — kao što je interakcija sa veb stranicama ili upravljanje vremenski zavisnim događajima — obezbeđujući ove dodatne mogućnosti izvan samog jezika.

Iako nije potrebno da znaš svaki detalj o JavaScript motoru i runtime-u da bi pisao JavaScript, osnovno razumevanje može ti pomoći da pišeš efikasniji kod i lakše pronalaziš greške.
