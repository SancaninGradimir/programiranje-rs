# --description--

Kada pišete kod Python, uobičajeno je naići na greške. Razumeti ove greške je važno za brzo i efikasno otklanjanje grešaka u vašem kodu. Ova poruka vam govori tačno koja je greška nastala, ako znate kako da je pročitate.

Uobičajene greške za Python su `SyntaxError`, `NameError`, `TypeError`, `IndexError`, i na `AttributeError`. Ove se javljaju kada Python ne razume vaš kod, ili kada vaša logika ne odgovara podacima koje koristite.

Ovo je primer za `SyntaxError`:

```py
print("Hello, world!"
# SyntaxError: unexpected EOF while parsing
```

Ova linija nedostaje skraćenica za zatvaranje zagrade. Python izaziva `SyntaxError` jer kod ne prati pravilne sintaksičke pravila.

Ovo je primer za `NameError`:

```py
print(name)
# NameError: name 'name' is not defined
```

Pokušavaš da ispišeš promenljivu koja još nije definisana. Python podiže `NameError` kada ne može da pronađe promenljivu sa tim imenom.

Ovo je primer za `TypeError`:

```py
5 + "5"
# TypeError: unsupported operand type(s) for +: 'int' and 'str'
```

Ne možete dodati pun broj i niz slova zajedno. Python generiše `TypeError` kada pokušate da izvršite operaciju na nekompatibilnim tipovima podataka.

Ovo je primer za `IndexError`:

```py
my_list = [1, 2, 3]
print(my_list[5])
# IndexError: list index out of range
```

Pokušavate pronaći model koji nije na listi. Python izbacuje `IndexError` kada izađete van granica.

Evo primera od `AttributeError`:

```py
num = 42
num.append(5)
# AttributeError: 'int' object has no attribute 'append'
```

Tip `int` nema metodu `append()`. Python podiže `AttributeError` kada pokušaš da koristiš metodu ili atribut koji ne postoji za taj tip podataka.
