# --interactive--

U CSS, postoji nekoliko načina za dodavanje granica oko slike, pri čemu svaka nudi različite opcije stila i kontrole.

Istražite neke od najpopularnijih i najdinamičnijih ruta.

Najlakši način za dodavanje ivice slici je korišćenje atributa `border`. Ovaj atribut je skraćenica koja vam omogućava da postavite širinu, stil i boju ivice odjednom.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border: 2px solid red;
}
```

:::

Ova pravila za CSS dodaju čvrstu crvenu ivicu širine od 2 piksela oko svih elemenata `img`. Možete promeniti širinu, stil (kao što su `dashed`, `dotted`, ili `double`) i boju kako bi odgovaralo potrebama vašeg dizajna.

Ako vam je potreban više kontrole nad privatnim stranama granice, možete koristiti specifične granične atribute za svaku stranu:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cat lying on its back.">
```

```css
img {
  border-top: 10px solid red;
  border-right: 10px dashed green;
  border-bottom: 10px dotted blue;
  border-left: 10px double purple;
}
```

:::

Ovo vam omogućava da kreirate jedinstvene stilove ivice/okvira za svaku stranu slike.

Drug način za kreiranje efekta ivice je korišćenje svojstva `outline`. Iako je sličan 'borderu', 'outline' ne menja dimenzije niti raspored elementa:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  outline: 3px solid gold;
}
```

:::

Ovo stvara zlatnu liniju pored slike bez menjanja njene veličine ili pozicije.

Ako želite da kreirate zaobnjene uglove na vašoj okvisi, možete koristiti svojstvo `border-radius` zajedno sa border:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="A cute cate lying on its back.">
```

```css
img {
  border: 2px solid black;
  border-radius: 10px;
}
```

:::

Zapamti, ove tehnike mogu se kombinovati i prilagođavati kako bi se napravili jedinstveni efekti ivica. Izbor metode zavisi od tvojih potreba dizajna i nivoa složenosti koji želiš.
