# --interactive--

Ako radiš sa brojevima i moraš da povećaš ili smanjiš vrednost za jedinicu, inkrementalni i dekrementalni operater čini taj posao lakim. Hajde da to analiziramo na jednostavan način.

Operater za povećanje i smanjenje je predstavljen sa `++` i `--`, respektivno. Oba vam omogućavaju da prilagodite vrednost parametra za `1`.

Umesto da napišete nešto kao `x = x + 1` ili `x = x - 1`, samo možete koristiti `x++` za povećanje `1`, ili `x--` za smanjenje `1`. Brzo, jasno i lako za čitanje.

Postoji mala promena u načinu rada inkrementacionog i dekrementacionog operatora: oni dolaze u dve vrste, prefiks i sufiks, što zavisi od toga kada se vrednost ažurira. Za inkrementacioni operator, prefiks je `++x` i sufiks je `x++`.

Predpona (`++x`) povećava vrednost parametra prvo, a zatim vraća novu vrednost. Sufiksa (`x++`) vraća trenutnu vrednost parametra prvo, a zatim je povećava:

:::interactive_editor

```js
let x = 5;

console.log(++x); // 6
console.log(x); // 6
```

:::

U gornjem kodu, `++x` znači "prvo dodaj `x`, pa ga koristi". Dakle, kada napišete `++x`, odmah dobijate dodatu vrednost, koja je `6`.

Sada, hajde da uzmemo primer koristeći sufiks:

:::interactive_editor

```js
let y = 5;

console.log(y++); // 5
console.log(y); // 6
```

:::

U ovom primeru, `y++` znači "prvo koristi `y`, a zatim dodaj". Kada napišete `y++`, dobijate `5`, ali `y` postaje `6` nakon te linije koda.

Dekrement operacija radi isto kao inkrement operacija, osim što smanjuje vrednost za `1`. Takođe, postoje dve vrste: prefiksni operator (`--x`) prvo smanjuje vrednost varijable, a zatim vraća novu vrednost. I sufiksni operator (`x--`) prvo vraća trenutnu vrednost, a zatim smanjuje:

:::interactive_editor

```js
let x = 5;
console.log(--x); // 4
console.log(x); // 4

let y = 5;
console.log(y--); // 5
console.log(y); // 4
```

:::

Dakle, šta bi trebalo da koristiš: prefiks ili sufiks? U mnogim slučajevima, nije važno koji koristiš. Oba rade. Međutim, ako koristiš vrednost jednom u matematičkom izrazu, ta razlika postaje važna. Uzmimo ovaj primer:

:::interactive_editor

```js
let a = 5;
let b = ++a;
console.log(b); // 6 (a was incremented before assignment)

let c = 5;
let d = c++;
console.log(d); // 5 (c was incremented after assignment)
```

:::

Stoga, ako vam je potrebna odmah ažurirana vrednost, koristite prefiks. Ako želite prvo trenutnu vrednost i niste zabrinuti za inkrementiranje kasnije, koristite sufiks.
