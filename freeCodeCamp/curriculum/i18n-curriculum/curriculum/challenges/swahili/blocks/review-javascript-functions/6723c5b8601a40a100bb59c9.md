## Funkcije za JavaScript

- Funkcija je ponovno upotrebljivi deo koda koji obavlja specifičan zadatak.
- Funkcija se može deklarisati korišćenjem ključne reči `function`, praćene imenom, listom parametara i delom koda koji obavlja tu funkciju.

:::interactive_editor

```js
function addNumbers(x, y, z) {
  return x + y + z;
}

console.log(addNumbers(5, 3, 8)); // Output: 16
```

:::

- Argumenti su vrednosti koje se prosleđuju funkciji kada se pozove.
- Pozivanje funkcije je proces izvršavanja funkcije u programu, navođenjem imena funkcije praćenim standardnim zagradama, opciono sa argumentima unutar tih zagrada.
- Kada funkcija završi izvršavanje, uvek vraća vrednost.
- Podrazumevana vrednost za return funkcije je `undefined`.
- Ključna reč `return` koristi se za određivanje vrednosti koja će biti vraćena iz funkcije i za završetak izvršavanja te funkcije.
- Podrazumevani parametri omogućavaju funkciji da ima unapred postavljene vrednosti koje će se koristiti ako nije dostavljen argument kada se funkcija pozove. Ovo čini funkciju efikasnijom i sprečava greške kada bi određeni argumenti mogli biti izostavljeni.

:::interactive_editor

```js
const calculateTotal = (amount, taxRate = 0.05) => {
  return amount + (amount * taxRate);
};

console.log(calculateTotal(100)); // Output: 105
```

:::

- Anonimna funkcija je funkcija koja nema ime i može primiti vrednosti putem parametara. Postavljanjem u parametre, možete ih ponovo koristiti bilo gde gde je taj parametar dostupan.

:::interactive_editor

```js
const multiplyNumbers = function(firstNumber, secondNumber) {
  return firstNumber * secondNumber;
};

console.log(multiplyNumbers(4, 5)); // Output: 20
```

:::

## Funkcija strele

- Strelična funkcija je najkraći način za pisanje funkcije u JavaScript.

:::interactive_editor

```js
const calculateArea = (length, width) => {
  const area = length * width;
  return `The area of the rectangle is ${area} square units.`;
};

console.log(calculateArea(5, 10)); // Output: "The area of the rectangle is 50 square units."
```

:::

- Kada deklariraš funkcijsku strelicu, ne treba ti glavna reč `function`.
- Ako koristiš samo jedan parametar, možeš da izostaviš obične zagrade koje okružuju listu parametara.

:::interactive_editor

```js
const cube = x => {
  return x * x * x;
};

console.log(cube(3)); // Output: 27
```

:::

- Ako telo funkcije ima samo jedan matematički izraz, možete ukloniti vitičaste zagrade i glavni deo od `return`.

:::interactive_editor

```js
const square = number => number * number;

console.log(square(5)); // Output: 25
```

:::

## Opseg u programu

- **Globalni opseg**: Ovo je potpuno eksterni opseg u JavaScript. Parametri deklarisani u globalnom opsegu dostupni su iz bilo kog dela koda i nazivaju se globalni parametri.
- **Lokalni opseg**: Ovo se odnosi na parametre deklarisane unutar funkcije. Ovi parametri su dostupni samo unutar funkcije u kojoj su deklarisani i nazivaju se lokalni parametri.
- **Blok opseg**: Blok je skup instrukcija zatvorenih u zaokružene zagrade `{}`, kao što su u instrukcijama `if`, ili petlje.
- Sposobnost kontrole pristupa parametrima korišćenjem `let` i `const` pruža još detaljniju kontrolu nad dostupnošću parametara, pomažući u sprečavanju grešaka i čineći vaš kod pouzdanijim.
