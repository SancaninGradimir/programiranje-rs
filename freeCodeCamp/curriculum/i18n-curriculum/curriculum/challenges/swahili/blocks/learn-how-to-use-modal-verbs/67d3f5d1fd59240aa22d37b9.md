# --explanation--

`Must` je modalni glagol, pa prati ova pravila:

- Uvek je praćeno osnovnom formom glagola (bila `to`).

- Upotrijemi `-s` u trećem licu jednine.

- Ne koristim `do/does` u pitanjima ili negiranju.
