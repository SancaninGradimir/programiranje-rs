# --interactive--

JavaScript pruža alate za obavljanje osnovnih matematičkih operacija na brojevima, kao što su sabiranje, oduzimanje, množenje i deljenje. JavaScript takođe ima operacije za složenije matematičke operacije, kao što su ostatak i potencija.

Svi ti alati se nazivaju matematičke operacione opere. Hajde da istražimo ove opere u detalju, kako ih koristiti i kako ih kombinovati.

Operater za dodavanje je znak za dodavanje (`+`). Operater za dodavanje vam omogućava da dobijete zbir dve ili više brojeva. U operacijama dodavanja, redosled brojeva nije važan:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 + num2;
const result2 = num2 + num1;
const result3 = num2 + num1 + num3;

console.log(result1); // 15
console.log(result2); // 15
console.log(result3); // 30
```

:::

Operater oduzimanja je znak za oduzimanje (`-`). Omogućava vam da pronađete razliku između dve brojeve. Koristite znak za oduzimanje kada želite da oduzmete jedan broj od drugog, obično manji od većeg:

:::interactive_editor

```js
const difference = 10 - 5;
console.log(difference); // 5
```

:::

Ako manji broj dođe prvi, dobićete negativne rezultate:

:::interactive_editor

```js
const difference = 5 - 10;
console.log(difference); // -5
```

:::

Takođe možete postaviti brojeve za parametre i izvršiti izlaz koristeći imena parametara:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const result = num1 - num2;

console.log(result); // 5
```

:::

U JavaScript-u, operacija množenja predstavljena je zvezdicom (`*`) i koristi se za dobijanje proizvoda dva ili više brojeva. Redosled brojeva koji se množe nije važan:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 * num2;
const result2 = num2 * num1;
const result3 = num2 * num1 * num3;

console.log(result1); // 50
console.log(result2); // 50
console.log(result3); // 750
```

:::

U JavaScript, operator deljenja je linija greške (`/`), za razliku od simbola deljenja koji se koristi u običnim proračunima (`÷`). Izvršavate operacije deljenja koristeći operator deljenja. Ovde je redosled brojeva koji se dele važan:

:::interactive_editor

```js
const num1 = 10;
const num2 = 5;
const num3 = 15;

const result1 = num1 / num2;
const result2 = num2 / num1;
const result3 = num2 / num1 / num3;

console.log(result1); // 2
console.log(result2); // 0.5
console.log(result3); // 0.03333333333333333
```

:::

Važno je znati da ako pokušate deliti sa nulom, JavaScript će vratiti `Infinity`:

:::interactive_editor

```js
const result = 10 / 0; 

console.log(result); // Infinity
```

:::

Uverite se da izbegavate te vrste kalkulacija kako biste izbegli nepouzdane rezultate u svom kodu.

Opereta preostanka, predstavljena znakom procenata (`%`), vraća ostatak deljenja. Ostatak u računanju je vrednost koja ostaje nakon deljenja:

:::interactive_editor

```js
const num1 = 10;
const num2 = 3;
const remainder = num1 % num2;

console.log(remainder); // 1
```

:::

Operacija potenciranja, predstavljena sa dve zvezdice (`**`), podiže jedan broj na stepen drugog:

:::interactive_editor

```js
const num1 = 2;
const num2 = 3;

const exponent = num1 ** num2;
console.log(exponent); // 8
```

:::

Moguće je kombinovati operetu u operaciji ili izrazu za računanje:

:::interactive_editor

```js
const result = 10 + 5 * 2 - 8 / 4;
console.log(result); // 18
```

:::

Kada kombinujete različite operatore u jednom izrazu, motor JavaScript prati sistem koji se zove prioritet operatera kako bi odredio redosled operacija. Prioritet operatera određuje redosled izvršavanja operacija u matematičkom izrazu. Naučićete više o prioritetu operatera u budućim lekcijama.
