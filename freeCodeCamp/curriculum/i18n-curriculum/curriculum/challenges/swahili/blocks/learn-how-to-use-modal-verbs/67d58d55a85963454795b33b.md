# --explanation--

`Could` na `might` i modalički glagoli koji se koriste za izražavanje mogućnosti.

`Could` to znači da je stvar moguća, ali nije potvrđena. Na primer:

`This update could make the app run faster.` - Ažuriranje ima mogućnost poboljšanja brzine programa.

`Might` takođe pokazuje mogućnost, ali često ukazuje na manji prostor od `could`. Na primer:

`The new feature might improve the user experience.` - Moguće je da će funkcija poboljšati iskustvo.
