# --interactive--

U prethodnoj lekciji naučili ste kako da radite sa `"click"` i `"input"` događajima koristeći metod `addEventListener()`. U ovoj lekciji naučićete kako da uklonite osluškivače događaja pomoću metoda `removeEventListener()`.

Metoda `removeEventListener()` koristi se za uklanjanje osluškivača događaja koji je prethodno dodat elementu pomoću metode `addEventListener()`. Ovo je korisno kada želite da prestanete da pratite određeni događaj na elementu.

Ovde je osnovna sintaksa putem `removeEventListener()`:

```js
element.removeEventListener("event", listener);
```

Kao i metoda `addEventListener()`, metoda `removeEventListener()` prima dva argumenta: događaj koji želite da uklonite i funkciju osluškivača koja je prethodno dodata.

Takođe postoji treći opcioni predlog koji se može usvojiti putem `removeEventListener()`. Ovaj predlog može biti `options` ili `useCapture`.

Argument `options` je objekat koji opisuje opcije osluškivača događaja, kao što su da li osluškivač treba da bude pasivan ili da se izvrši samo jednom (`once`).

Argument `useCapture` je logička vrednost koja određuje da li događaj treba da bude uhvaćen tokom faze širenja događaja.

U većini slučajeva, potrebno je proslediti samo argumente događaja i osluškivača metodi `removeEventListener()`.

Ako želimo da promenimo boju pozadine između sive i plave, možemo koristiti obrađivač događaja za tu svrhu.

Prvo nam je potrebno da dobijemo pristup elementu `button` zajedno sa elementom `body`:

```js
const bodyEl = document.querySelector("body");
const btn = document.getElementById("btn");
```

Zatim moramo kreirati funkciju koja se bavi menjanjem između sive i plave boje:

```js
let isBgColorGrey = true;

function toggleBgColor() {
  bodyEl.style.backgroundColor = isBgColorGrey ? "blue" : "grey";
  isBgColorGrey = !isBgColorGrey;
}
```

Koristimo boolsku vrednost `isBgColorGrey` za praćenje trenutne pozadinske boje. Ako je pozadina siva, onda menjamo u plavu, i obrnuto.

Zatim nam je potrebno dodati slušalicu događaja na element `button` kako bismo pozvali funkciju `toggleBgColor` kada se dugme klikne:

```js
btn.addEventListener("click", toggleBgColor);
```

Svaki put kada se dugme klikne, pozadina tela će menjati boju između sive i plave.

Možemo ažurirati HTML da uključimo element paragrafa koji će se koristiti za uklanjanje slušalice događaja kada se pređe mišem:

```html
<link rel="stylesheet" href="styles.css" />
<h1>Event Listener examples</h1>
<p id="para">MouseOver this text to remove the event listener</p>
<button id="btn">Change background color</button>
<script src="index.js"></script>
```

Da bismo uklonili obrađivač događaja kada se pokazivač miša pređe preko pasusa, možemo dodati obrađivač događaja elementu pasusa. Koristimo događaj `mouseover` i prosleđujemo funkciju koja uklanja obrađivač događaja sa elementa `button`.

```js
const para = document.getElementById("para");
const btn = document.getElementById("btn");

para.addEventListener("mouseover", () => {
  btn.removeEventListener("click", toggleBgColor);
});
```

Kada se preko pasusa pređe mišem, osluškivač događaja za klik dugmeta se uklanja i boja pozadine se više neće menjati kada se dugme klikne.
