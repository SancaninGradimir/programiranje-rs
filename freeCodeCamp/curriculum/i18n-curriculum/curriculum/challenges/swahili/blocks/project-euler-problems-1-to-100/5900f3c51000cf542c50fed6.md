# --description--

Prirodan broj, `N`, koji se može zapisati kao zbir i proizvod nekog skupa od najmanje dva prirodna broja, $\\{a_1, a_2, \ldots , a_k\\}$ naziva se broj množenja i sabiranja: $N = a_1 + a_2 + \cdots + a_k = a_1 × a_2 × \cdots × a_k$.

Na primer, 6 = 1 + 2 + 3 = 1 × 2 × 3.

Za određeni skup veličina, `k`, nazivaćemo najmanji N sa ovom svojstvom minimalnog eksponencijalnog broja. Najmanji eksponencijalni brojevi za skupe veličina, `k` = 2, 3, 4, 5, i 6 su sledeći.

<div style='margin-left: 4em;'>
  <var>k</var>=2: 4 = 2 × 2 = 2 + 2<br>
  <var>k</var>=3: 6 = 1 × 2 × 3 = 1 + 2 + 3<br>
  <var>k</var>=4: 8 = 1 × 1 × 2 × 4 = 1 + 1 + 2 + 4<br>
  <var>k</var>=5: 8 = 1 × 1 × 2 × 2 × 2  = 1 + 1 + 2 + 2 + 2<br>
  <var>k</var>=6: 12 = 1 × 1 × 1 × 1 × 2 × 6 = 1 + 1 + 1 + 1 + 2 + 6
</div><br>

Stoga, za 2 ≤ `k` ≤ 6, ukupan je svih malih brojeva dobijenih množenjem 4 + 6 + 8 + 12 = 30; imajte na umu da `8` se računa samo jednom u ukupnom zbiru.

Zapravo, ako je kompletan skup brojeva koje množimo sa 2 od 2 ≤ `k` ≤ 12 iznosi $\\{4, 6, 8, 12, 15, 16\\}$, ukupno je `61`.

Koja je suma svih parnih brojeva za 2 ≤ `k` ≤ `limit`?

function productSumNumbers(limit) {
  function getProductSums(curProduct, curSum, factorsCount, start) {
    const k = curProduct - curSum + factorsCount;
    if (k <= limit) {
      if (curProduct < minimalProductSums[k]) {
        minimalProductSums[k] = curProduct;
      }
      for (let i = start; i < Math.floor((limit / curProduct) * 2) + 1; i++) {
        getProductSums(curProduct * i, curSum + i, factorsCount + 1, i);
      }
    }
  }

  const minimalProductSums = new Array(limit + 1).fill(2 * limit);
  getProductSums(1, 1, 1, 2);

  const uniqueProductSums = [...new Set(minimalProductSums.slice(2))];

  let sum = 0;
  for (let i = 0; i < uniqueProductSums.length; i++) {
    sum += uniqueProductSums[i];
  }

  return sum;
}
