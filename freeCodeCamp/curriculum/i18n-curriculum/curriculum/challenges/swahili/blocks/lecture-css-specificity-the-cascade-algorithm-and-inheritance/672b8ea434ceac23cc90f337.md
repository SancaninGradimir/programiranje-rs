# --interactive--

ID selektori su među najmoćnijim selektorima u CSS, što omogućava programeru korišćenje stilova za specifične elemente sa jedinstvenim identifikatorima.

Ovo povećava njihovu efikasnost ciljanjem na individualne komponente koje zahtevaju jedinstvene stilove.

ID selektori označavaju se znakom tarabe (`#`) praćenim nazivom ID-ja. Oni treba da budu jedinstveni u HTML dokumentu, što znači da nijedna dva elementa ne bi trebalo da dele isti ID.

Ovde je primer korišćenja selektora ID-a:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p id="unique">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#unique {
  color: purple;
}
```

:::

U ovom primeru, element sa ID-om `unique` imaće boju teksta `purple`.

ID selektori imaju veoma visoku specifičnost, veću od selektora tipa i selektora klase, ali manju od inline stilova. Vrednost specifičnosti za ID selektor je `(0, 1, 0, 0)`.

To znači da ID selektori mogu da nadjačaju selektore klase i selektore tipa, ali ih mogu nadjačati inline stilovi.
