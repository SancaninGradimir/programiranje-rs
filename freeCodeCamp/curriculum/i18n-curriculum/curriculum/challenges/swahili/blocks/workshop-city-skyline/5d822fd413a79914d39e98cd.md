# --description--

U CSS, možete ciljati sve koristeći zvezdice. Proširite do svega koristeći selektor `*`, i ako dobijete svojstvo `border` sa vrednošću `1px solid black`. Ovo je metoda koja pomaže da se vidi gde su stvari i njihov veličina. Biće objavljeno kasnije.

# --hints--

Trebalo bi da koristiš alat `*`.

```js
assert.exists(new __helpers.CSSHelp(document).getStyle('*'));
```

Trebalo bi da koristiš atribut `border` za kreiranje stila za sve elemente.

```js
assert.exists(new __helpers.CSSHelp(document).isPropertyUsed('border'));
```

Svi elementi moraju imati granicu od `1px solid black`.

```js
const astStyles = new __helpers.CSSHelp(document).getStyle('*');
assert.equal(astStyles?.border, '1px solid black');
```
