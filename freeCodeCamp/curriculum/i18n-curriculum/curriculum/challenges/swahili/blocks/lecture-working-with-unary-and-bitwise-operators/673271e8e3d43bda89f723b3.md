# --interactive--

Unarni operateri funkcionišu samo sa jednim operandom kako bi izvršili operacije kao što su konverzija tipa, modifikacija vrednosti ili provera određenih uslova. Istražićemo neke poznate unarne operatore i način na koji funkcionišu.

Operater jedinstvenog plusa konvertuje svoj operand u broj. Ako je operand već broj, on se ne konvertuje.

:::interactive_editor

```js
const str = '42';
const strToNum = +str;

console.log(strToNum); // 42
console.log(typeof str); // string
console.log(typeof strToNum); // number
```

:::

Unarni plus je važan kada želite da osigurate da radite sa numeričkom vrednošću. Kao što možete da pogodite, postoji operater unarnog negiranja. On menja vrednost operanda. Radi kao unarni plus, osim što menja znak.

:::interactive_editor

```js
const str = '42';
const strToNegativeNum = -str;

console.log(strToNegativeNum); // -42
console.log(typeof str); // string
console.log(typeof strToNegativeNum); // number
```

:::

Logički NOT operator, koji je predstavljen znakom uzvika (`!`), jeste još jedan unarni operator. Menja boolean vrednost svog operanda. Dakle, ako je operand `true`, menja se u `false`, a ako je `false`, menja se u `true`.

:::interactive_editor

```js
let isOnline = true;
console.log(!isOnline); // false

let isOffline = false;
console.log(!isOffline); // true
```

:::

Operater bitwise NOT je veoma retko korišćen operater jednorazlične prirode. Predstavljen je tildom, `~`, i menja binarno predstavljanje broja. Računari skladište brojeve u binarnom sistemu (jedinice i nule). Operater `~` menja svaki bit, što znači da sve jedinice menja u nule, a sve nule u jedinice. Vi ćete saznati više o binarnom sistemu i bitovima u sledećoj lekciji.

:::interactive_editor

```js
const num = 5; // The binary for 5 is 00000101

console.log(~num); // -6
```

:::

U ovom primeru, `5` postaje `-6` jer korišćenjem operatera `~` za `5`, dobijate `- (5 + 1)`, što je jednako `-6` zbog reprezentacije two's complement. Two's complement je metoda koju računari koriste za predstavljanje negativnih brojeva u binarnom sistemu. Možda nećete često koristiti bitwise NOT operater osim ako ne radite zadatke niskonivošnjeg programiranja, kao što je direktno menjanje bita.

Glavni termin za `void` je jedinarni operater koji analizira matematičke izraze i vraća `undefined`.

:::interactive_editor

```js
const result = void (2 + 2);

console.log(result); // undefined
```

:::

`void` Takođe se često koristi u mrežnim linkovima radi sprečavanja preusmeravanja:

```js
<a href="javascript:void(0);">Click Me</a>
```

Konačno, postoji operator `typeof` koji ste naučili u prethodnim lekcijama. Ovo vraća tip njegovog operanda kao niz znakova.

:::interactive_editor

```js
const value = 'Hello world';

console.log(typeof value); // string
```

:::
