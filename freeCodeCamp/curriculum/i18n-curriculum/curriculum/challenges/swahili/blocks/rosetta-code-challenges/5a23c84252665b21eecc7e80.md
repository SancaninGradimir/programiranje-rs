# --description--

Grački kod je vrsta binarnog kodiranja gde se promena između uzastopnih brojeva razlikuje samo za jedan bit.

Ovo je korisno kodiranje za smanjenje rizika od grešaka u hardverskim podacima i vrednostima koje se brzo menjaju i/ili se prenose preko sporih uređaja za unos.

Takođe je važno za generisanje ulaza za Karnogove mape u redosledu od leve ka desnoj ili odozgo nadole.

# --instructions--

Napravite funkciju za enkodiranje i dekodiranje brojeva iz Gray koda. Funkcija će imati dva parametra.

Prvi parametar će biti vrednost istina ili lažno. Funkcija će kodirati za istinu i dekodirati za laž. Drugi parametar će biti broj za kodiranje/dekodiranje.

Prikaži standardnu binarnu reprezentaciju, Gray kodnu reprezentaciju i ažurirane vrijednosti Gray koda za sve pet-bitne binarne brojeve (uključujući 0-31, nije potrebno prikazivati vodeće nule).

Postoji mnogo mogućih Gray kodova. Sledeće opisuje ono što se naziva "binary reflected Gray code."

Kodiranje (MSB je bit 0, b je binarni, g je Gray kod):

<pre>ikiwa b[i-1] = 1
  g[i] = si b[i]
vengine
  g[i] = b[i]
</pre>

Au:

<pre>g = b xor (b pomereno udesno za 1 bit logički)
</pre>

Kombinacije (MSB je bit 0, b je binary, g je Gray kod):

<pre>b[0] = g[0]<br>
Za sledeći bit:
b[i] = g[i] xor b[i-1]
</pre>
