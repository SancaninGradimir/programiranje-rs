# --explanation--

Negativni tip vremena `Present Perfect Continuous` formira se korišćenjem `hasn't/haven't + been + present participle`, prikazujući tekuću akciju koja nije nastala kako je očekivano tokom vremena. Primer:

`He hasn't been exercising regularly.` - Ovo znači da je njegov raspored vežbanja bio neobičan.
  
Negativni tip vremena za `Present Perfect` koristi `hasn't/haven't + past participle`, opisujući radnju koja nije završena do ovog trenutka. Primer:

`He hasn't exercised this week.` - Ovo pokazuje da nema vežbi napravljenih ove nedelje.
