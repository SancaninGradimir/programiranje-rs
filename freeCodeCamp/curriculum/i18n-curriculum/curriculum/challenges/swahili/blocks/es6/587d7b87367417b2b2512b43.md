# --description--

U JavaScript, često nam nije potrebno davati imena našim funkcijama, posebno kada prosleđujemo funkciju kao argument drugoj funkciji. Umesto toga, kreiramo funkcije u liniji (inline). Ne moramo davati imena ovim funkcijama jer ih nigde drugde nećemo koristiti.

Da bismo ovo postigli, često koristimo sledeću sintaksu:

```js
const myFunc = function() {
  const myVar = "value";
  return myVar;
}
```

ES6 nam daje sintaktičku šećer da ne moramo da pišemo anonimne funkcije na ovaj način. Umesto toga, možete koristiti **sintaksu strelične funkcije**:

```js
const myFunc = () => {
  const myVar = "value";
  return myVar;
}
```

Kada nema funkcionalnog tela, a postoji samo vrijednost return, sintaksa strelične funkcije omogućava vam da ostavite glavnu riječ `return` zajedno sa uglastim zagradama koje okružuju kod. Ovo pomaže u pojednostavljivanju malih funkcija u jednu rečenicu unutar jedne linije:

```js
const myFunc = () => "value";
```

Ovaj kod i dalje vraća sekvencu slova `value` podrazumevano.
