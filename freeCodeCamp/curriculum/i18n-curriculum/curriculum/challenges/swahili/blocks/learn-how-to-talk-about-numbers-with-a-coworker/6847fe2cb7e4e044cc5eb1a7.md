# --explanation--

Pravilan način za pisanje <0xE2><0x85><0x97> rečima je `three-fifths`. Koristite ovaj format:

Pridev (u osnovnom obliku) + glagol (u množini ako je pridev > 1). Primer:

- <0xE2><0x85><0x96> = `two-fifths`

- <0xE2><0x85><0x98> = `four-fifths`

Ovo je alat za objašnjavanje podataka ili deljenje finansijskih informacija tokom usmenih prezentacija.
