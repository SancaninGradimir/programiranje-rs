# --explanation--

Vreme `Present Continuous` koristi se za opisivanje radnji koje se dešavaju u trenutku govora. Gradi se pomoću glagola `to be` (`am`, `is`, `are`) + osnovni oblik glagola + `-ing`. Na primer:

- `She is coding a new feature right now.` - On je/Ona je trenutno u procesu pisanja nove funkcije.

- `They are testing the software.` – Oni upravo testiraju softver.
