# --description--

U prethodnom izazovu napravio si nadtip (`supertype`) pod nazivom `Animal`, koji definiše ponašanja zajednička svim životinjama:

```js
function Animal() { }
Animal.prototype.eat = function() {
  console.log("nom nom nom");
};
```

Ovaj i sledeći izazov pokazaće kako da ponovo koristiš metode objekta `Animal` u objektima `Bird` i `Dog` bez njihovog ponovnog pisanja. To se postiže tehnikom koja se zove nasleđivanje. Ovaj izazov se bavi prvim korakom: kreiranjem instance nadtipa (ili roditelja). Već znaš jedan način za kreiranje instance objekta `Animal` pomoću operatora `new`:

```js
let animal = new Animal();
```

Postoje određeni nedostatci prilikom korišćenja ove sintakse za nasleđivanje, što je veoma teško u opsegu ovog izazova. Umesto toga, ovde je alternativa metoda koja nema te gubitke:

```js
let animal = Object.create(Animal.prototype);
```

`Object.create(obj)` kreira novi objekat, i postavlja `obj` kao `prototype` tog novog objekta. Zapamtite da je `prototype` kao "recept" za kreiranje objekta. Postavljanjem `prototype` od `animal` u `prototype` od `Animal`, dajete primer za `animal` "recept" sličan bilo kom drugom primeru od `Animal`.

```js
animal.eat();
animal instanceof Animal;
```

Put do `instanceof` ovde će vratiti `true`.

# --instructions--

Tumia `Object.create` kutengeneza mifano miwili ya `Animal` iitwayo `duck` na `beagle`.
