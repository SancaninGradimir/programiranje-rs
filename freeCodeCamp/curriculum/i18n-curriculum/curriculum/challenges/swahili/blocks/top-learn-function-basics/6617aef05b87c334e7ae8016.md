# --description--

Kao što si ranije video, funkcije mogu vraćati vrednosti pomoću ključne reči `return`. Ključna reč `return` koristi se za vraćanje vrednosti iz funkcije. Kada se izvrši `return`, funkcija prestaje sa radom i vraća vrednost navedenu iza ključne reči `return`.

```js
function add(a, b) {
  return a + b
}

console.log(add(2, 3)); // Output: 5
```

Ali šta se dešava ako glavna reč za `return` bude korišćena pre kraja funkcije?
Da bismo odgovorili na ovo pitanje, razmislite sledeći primer:

```js
function add(a, b) {
  if(a > 2){
    return b;
  }

  return a + b;
}

console.log(add(3, 7)); // Output: 7
``` 

U gornjem primeru, funkcija ``add`` ima uslovnu izjavu koja proverava da li je vrednost ``a`` veća od ``2``. Ako je uslov zadovoljen, funkcija će vratiti vrednost ``b`` i prestati izvršavanje. Ako uslov nije zadovoljen, funkcija će vratiti zbir ``a`` i ``b``.
