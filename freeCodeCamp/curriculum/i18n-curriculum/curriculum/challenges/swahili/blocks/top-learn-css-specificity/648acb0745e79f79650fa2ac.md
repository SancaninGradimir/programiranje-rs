# --description--

Poslednji element, kraj linije, separator separatora. Recimo da nakon razmatranja svakog drugog elementa, i dalje postoji mnogo protivrečnih pravila koja ciljaju na jedan element. Kako sekvenca određuje koje pravilo treba koristiti?

Vrlo jednostavno, zaista. Pravilo koje je objašnjeno na kraju je pobednik.

```css
/* styles.css */

.alert {
  color: red;
}

.warning {
  color: yellow;
}
```

Za element sa klasom objekata `alert` i `warning`, sekvenca će iterirati kroz svaki drugi element, uključujući nasleđivanje (nije ovde) i specifičnu stvarnost (pravilo ne nadjačava drugo za specifičnu stvarnost). Pošto je pravilo `.warning` definisano poslednje, i nijedan drugi element nije mogao da utvrdi koje pravilo treba koristiti, ono će biti izvršeno za taj element.
