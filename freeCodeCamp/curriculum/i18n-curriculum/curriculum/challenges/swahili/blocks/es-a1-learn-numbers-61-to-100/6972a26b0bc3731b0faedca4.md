# --description--

Sada je vreme da naučimo brojeve od 70 do 79.

U Kihispaniji, broj 70 se zove `setenta`.

Ovi brojevi su formirani korišćenjem reči trideset `setenta` (70), praćeno veznikom `y` ("na"), a zatim brojem jedinica (1-9).

| Broj | Španski |
| :--- | :--- |
| 70 | `Setenta` |
| 71 | `Setenta y uno` |
| 72 | `Setenta y dos` |
| 73 | `Setenta y tres` |
| 74 | `Setenta y cuatro` |
| 75 | `Setenta y cinco` |
| 76 | `Setenta y seis` |
| 77 | `Setenta y siete` |
| 78 | `Setenta y ocho` |
| 79 | `Setenta y nueve` |

Broj `setenta y uno` (71) je jedinstven u ovom redu jer se menja u `setenta y un` pre maskulinog roda imenice. Primer:

`Setenta y un años` - Sedamdeset i jedna.
