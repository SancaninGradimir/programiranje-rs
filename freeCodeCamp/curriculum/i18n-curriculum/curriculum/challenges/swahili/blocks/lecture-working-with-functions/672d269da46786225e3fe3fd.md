# --interactive--

Funkcije su ponovno upotrebljivi delovi koda koji obavljaju određeni zadatak ili izračunavaju vrednost. Zamislite funkciju kao mašinu koja prima određen unos, vrši na njemu neke operacije, a zatim izdaje rezultat. Evo primera deklarisanja funkcije:

```js
function greet() {
  console.log("Hello, Jessica!");
}
```

U ovom primeru, deklarisali smo funkciju pod nazivom `greet`. Unutar te funkcije, imamo `console.log` koja štampa poruku `Hello, Jessica!`. Ako pokrenemo ovaj kod, nećemo videti poruku prikazanu u konzoli. To je zato što moramo pozvati funkciju.

Pozivanje funkcije, ili invocacija, je kada koristimo ili pokrećemo funkciju. Da biste pozvali funkciju, moraćete da navedete ime funkcije praćeno standardnim zagradama:

:::interactive_editor

```js
function greet() {
  console.log("Hello, Jessica!");
}

greet(); // "Hello, Jessica!"
```

:::

Sada će poruka `Hello, Jessica!` biti napisana u konzolu. Ali šta ako želimo da poruka kaže `Hello, Nick!` ili `Hello, Anna!`? Ne želimo da pišemo novu funkciju svaki put kada pozdravljamo različit korisnik. Umesto toga, možemo kreirati ponovno upotrebljivu funkciju koja koristi parametre i argumente funkcije.

Parametri funkcionišu kao predstavnici vrednosti koje se prosleđuju funkciji kada se pozove. Omogućavaju funkcijama da primaju ulaz i rade sa tim ulazom. Argumenti su stvarne vrednosti koje se prosleđuju funkciji kada se pozove. Evo ažurirane verzije funkcije za `greet` koja koristi parametre i argumente:

:::interactive_editor

```js
function greet(name) {
  console.log("Hello, " + name + "!");
}

greet("Alice"); // Hello, Alice!
greet("Nick"); // Hello, Nick!
```

:::

`name` koristi se kao parametar kada sekvenca karaktera `Alice` i `Nick` služe kao argumenti. Sada imamo ponovo upotrebljivu funkciju koja se može koristiti više puta u našem kodu sa različitim argumentima.

Funkcija nakon završetka izvršavanja uvek vraća vrednost. Podrazumevano, vrednost za return će biti `undefined`. Evo primera:

:::interactive_editor

```js
function doSomething() {
  console.log("Doing something...");
}

let result = doSomething();
console.log(result); // undefined
```

:::

Ako vaša funkcija mora da vrati određenu vrednost, moraćete da koristite deklaraciju `return`. Evo primera korišćenja deklaracije `return` za vraćanje zbira dve vrednosti:

:::interactive_editor

```js
function calculateSum(num1, num2) {
  return num1 + num2;
}

console.log(calculateSum(3, 4)); // 7
```

:::

Često ćete koristiti deklaraciju `return`, jer možete kasnije u vašem kodu koristiti tu vrednost koju je funkcija pružila.

Do sada smo radili sa funkcijama sa imenom, ali možete kreirati i ono što se zove anonimna funkcija. Anonimna funkcija je anonimna funkcija koja može dobiti vrednost parametra kao ovo:

:::interactive_editor

```js
const sum = function (num1, num2) {
  return num1 + num2;
};

console.log(sum(3, 4)); // 7
```

:::

U ovom primeru, imamo parametar `const` nazvan `sum` i postavljamo mu vrednost anonimnom funkciji koja vraća zbir od `num1` i `num2`. Zatim možemo pozvati `sum` i proslediti brojeve `3` i `4` kako bismo dobili rezultat `7`.

Funkcije podržavaju zadane parametre, što vam omogućava postavljanje zadnjih vrednosti za parametre. Ove zadane vrednosti se koriste ako funkcija bude pozvana bez argumenta za taj parametar. Evo primera:

:::interactive_editor

```js
function greetings(name = "Guest") {
  console.log("Hello, " + name + "!");
}

greetings(); // Hello, Guest!
greetings("Anna"); // Hello, Anna!
```

:::

U ovom primeru, ako nije dostavljen argument za `name`, imaće podrazumevanu vrednost `Guest`.

Ukratko, funkcije vam omogućavaju da pišete ponovno upotrebljiv i organizovan kod. Mogu uzeti ulaz (parametri), izvršiti operacije i vratiti rezultate.
