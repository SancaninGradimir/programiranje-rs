# --description--

Numerički izraz `a lot of` se može koristiti sa brojivim i nebrojivim imenicama, za pokazivanje velikog broja ili velike količine.

Za brojeve imenice, kao što su `cars`, možete reći `a lot of cars`. Za nebrojive imenice, kao što su `water`, ćete reći `a lot of water`.
