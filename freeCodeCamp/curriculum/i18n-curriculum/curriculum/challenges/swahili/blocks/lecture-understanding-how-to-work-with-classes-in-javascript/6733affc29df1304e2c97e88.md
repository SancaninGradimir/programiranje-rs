# --interactive--

U savremenom JavaScript, klase objekata su kao šeme koje možete napisati u kodu kako biste kreirali mnogo stvari istog tipa.

Na primer, ako kreirate sistem za inventarizaciju proizvoda, biće vam potrebno da kreirate mnogo stavki za svaku vrstu proizvoda.

Svaki pojedinačni proizvod će imati svoje karakteristike i funkcije, ali proizvodi iste vrste mogu se kreirati iz iste "karte". To je klasa korišćenih predmeta.

Klase objekata mogu se definisati kao izjava o klasi objekata ili deklaracija klase objekata.

Ovde je osnovna sintaksa za deklaraciju klase:

```js
class MyClassName {
  // Class Methods
  constructor() { ... }
  method1() { ... }
  method2() { ... }
  ...
}
```

Počinjete tako što pišete glavni termin ``class``, praćen nazivom klase objekata. Kada navedite nazive klasa objekata, trebali biste koristiti konvenciju imenovanja ``PascalCase`` umesto ``camelCase``.

Sadržaj unutar okruglih zagrada naziva se telom klase objekata. Ovde će biti napisani sastav i metode.

Konstruktor je posebna metoda koja se automatski poziva kada se kreira novi objekat iz klase. Obično se koristi za inicijalizovanje atributa tog objekta.

U ovom primeru možeš videti klasu `Dog` sa konstruktorom.

```js
class Dog {
  constructor(name) {
    this.name = name;
  }
}
```

Ključna reč `this` se ovde koristi da se odnosi na trenutnu instancu klase `Dog`, omogućavajući ti da postaviš svojstvo `name` za objekat koji se kreira. Više o ključnoj reči `this` naučićeš u sledećoj lekciji.

Unutar komponenti, možete imati metode poznate kao methods. Metoda je funkcija definisana unutar klase objekata koja vrši akcije ili operacije. One se koriste za definisanje ponašanja koje instancama te klase mogu izvršiti.

Ovo je primer dodavanja rute pod nazivom `bark` za klasu `Dog`.

```js
class Dog {
  constructor(name) {
    this.name = name;
  }

  bark() {
    console.log(`${this.name} says woof!`);
  }
}
```

Kada se pozove metoda `bark`, u konzoli će biti ispisana poruka sa stvarnim imenom koje joj je prosleđeno. Ako kreiraš instancu klase `Dog` sa imenom `"Buddy"`, poziv metode `bark()` na toj instanci ispisuje `"Buddy says woof!"` u konzoli.

Kada definišeš svoju klasu, možeš kreirati njenu instancu pomoću ključne reči `new`, imena klase i odgovarajućih argumenata.

U ovom primeru, kreiramo novi primer od `Dog` sa imenom `Gino`, a zatim postavljamo njegovu vrednost u izmene za `dog`.

```js
const dog = new Dog("Gino");
```

Kada unesete `new Dog("Gino")` u svoj kod, stvara se novi element i komponentu automatski imenuju za zadane argumente.

Tada možete dobiti ovo svojstvo koristeći dot notaciju, kako vidite u sledećoj liniji koda:

```js
console.log(dog.name);
```

Evo kompletnog primera:

:::interactive_editor

```js
class Dog {
  constructor(name) {
    this.name = name;
  }

  bark() {
    console.log(`${this.name} says woof!`);
  }
}

const dog = new Dog("Gino");
console.log(dog.name);
```

:::

Takođe možete pozvati funkciju. U ovom primeru, pozivamo funkciju za `bark()` u primeru `dog`.

```js
dog.bark();
```

Evo kompletnog primera sa uključenim pozivom rute:

:::interactive_editor

```js
class Dog {
  constructor(name) {
    this.name = name;
  } 
  bark() {
    console.log(`${this.name} says woof!`);
  }
}

const dog = new Dog("Gino");
console.log(dog.name);
dog.bark();
```

:::

Kao što je prethodno navedeno, takođe možete definisati klase objekata kao izjavu o klasi objekata. Ovo je gde klasa objekata nema ime i dobija vrednost u promeni.

Ovo je kako izgleda početni primer kao izjava klase objekata:

```js
const Dog = class {
  constructor(name) {
    this.name = name;
  }

  bark() {
    console.log(`${this.name} says woof!`);
  }
};
```

Često ćete koristiti deklaraciju klase objekata, ali razumevanje upotrebe te klase je takođe važno, jer pruža veću fleksibilnost, posebno u situacijama dinamički menja programa.

Korišćenjem klas objekata i razumevanjem njihove osnovne sinakse, možete kreirati ponovno upotrebljiv kod, organizovati ga u logičke jedinice, i razviti samostalnu strukturu koda.
