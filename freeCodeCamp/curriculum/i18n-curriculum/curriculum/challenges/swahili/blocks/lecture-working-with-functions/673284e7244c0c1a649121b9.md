# --interactive--

Opseg u programiranju se odnosi na vidljivost i dostupnost varijabli u različitim delovima vašeg koda. To je određivanje mesta gde varijable mogu biti pristupljene ili modifikovane. U JavaScript, razumevanje opsega je ključno za pisanje čistog, efikasnog i bezgrešnog koda. Postoje tri glavna tipa opsega: globalni opseg, lokalni opseg i blok opseg.

Globalni opseg je potpuno eksterni opseg u programu JavaScript. Parametri deklarisani u globalnom opsegu mogu se pristupiti bilo kom delu vašeg koda, uključujući unutar funkcija i zagrada. Ovi parametri se često nazivaju globalni parametri. Iako su globalni parametri laki za korišćenje, trebalo bi ih koristiti umereno jer mogu uzrokovati kolizije imena i učiniti vaš kod teškim za održavanje. Evo primera globalnog parametra:

:::interactive_editor

```js
let globalVar = "I'm a global variable";

function printGlobalVar() {
    console.log(globalVar);
}

printGlobalVar(); // "I'm a global variable"
```

:::

U ovom primeru, `globalVar` je deklarisano u globalnom opsegu i može se pristupiti unutar funkcije `printGlobalVar`.

Opseg (Scope), s druge strane, odnosi se na parametre koji su dostupni samo unutar funkcije. Evo primera lokalnog opsega:

:::interactive_editor

```js
function greet() {
    let message = "Hello, local scope!";
    console.log(message);
}

greet(); // "Hello, local scope!"
// console.log(message); // This will throw an error
```

:::

U ovom kodu, `message` je lokalni parametar unutar funkcije `greet`. Može se koristiti unutar funkcije, ali pokušaj pristupa izvan funkcije će dovesti do greške.

Opseg bloka je koncept uveden ključnim rečima `let` i `const` u ES6. Blok je bilo koji deo koda unutar vitičastih zagrada, `{}`, kao što su u deklaracijama `if`, petljama `for` ili petljama `while`. Koncept petlji će biti objašnjen u sledećoj lekciji.

Parametri deklarisani za `let` ili `const` unutar bloka dostupni su samo unutar tog bloka. Evo primera opsega bloka:

```js
if (true) {
    let blockVar = "I'm in a block";
    console.log(blockVar); // "I'm in a block"
}
console.log(blockVar); // This will throw an error
```

U ovom primeru, `blockVar` je dostupan samo unutar bloka `if`. Pokušaj pristupa izvan bloka će dovesti do greške. Razumevanje ovih različitih vrsta opsega (scope) je važno za upravljanje dostupnošću promenljivih i izbegavanje neočekivanih efekata u vašem kodu.

Globalne varijable bi trebalo koristiti sa oprezom jer mogu uzrokovati kolizije imena i učiniti vaš kod teškim za održavanje. Lokalne varijable pomažu da se različiti delovi vašeg koda odvoje, što je posebno važno u velikim aplikacijama. Opseg bloka korišćenjem `let` i `const` pruža bolju kontrolu nad dostupnošću varijabli, pomažući u sprečavanju grešaka i čineći vaš kod predvidljivijim. Savladavanje ovih osnovnih koncepata globalnog, lokalnog i blok opsega pružiće čvrst temelj za razumevanje naprednijih tema.
