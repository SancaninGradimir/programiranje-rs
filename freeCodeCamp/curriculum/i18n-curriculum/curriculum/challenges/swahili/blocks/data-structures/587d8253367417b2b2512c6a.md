# --description--
Redovi podataka su stvari od JavaScript koji mogu da sadrže mnogo različitih elemenata.

```js
var complexArr = [1, 5, "2", "Word", {"name": "James"}];
```

U suštini, ono što se dešava u pozadini je da vaš pretraživač direktno dodeljuje odgovarajući memorijski prostor za taj red. Takođe, taj prostor se menja po potrebi ako dodate ili uklonite podatke.

Međutim, u svetu visoke performansi i različitih komponenti, ponekad je potrebno biti precizniji o koliko memorije se dodeljuje redu.

<dfn>Niz specifičnog tipa</dfn> je rešenje za ovaj problem. Sada možete odrediti koliko memorije želite dodeliti nizu. Ispod je pregled različitih vrsta nizova dostupnih i veličina svakog elementa u bajtovima.

<table><tbody><tr><th>Tip</th><th>Veličina elemenata po bajtu</th></tr><tr><td><code>Int8Array</code></td><td>1</td></tr><tr><td><code>Uint8Array</code></td><td>1</td></tr><tr><td><code>Uint8ClampedArray</code></td><td>1</td></tr><tr><td><code>Int16Array</code></td><td>2</td></tr><tr><td><code>Uint16Array</code></td><td>2</td></tr><tr><td><code>Int32Array</code></td><td>4</td></tr><tr><td><code>Uint32Array</code></td><td>4</td></tr><tr><td><code>Float32Array</code></td><td>4</td></tr><tr><td><code>Float64Array</code></td><td>8</td></tr></tbody></table>

Postoje dva načina za kreiranje ovih vrsta redova. Jedan način je direktno kreiranje. Ispod je kako da se kreira `Int16Array` dužine 3.

```js
var i8 = new Int16Array(3);
console.log(i8);
// Returns [0, 0, 0]
```

Takođe možete kreirati <dfn>privremeno skladište</dfn> kako biste odredili količinu podataka (u bajtovima) koju želite da red zauzme. **Napomena**
Da biste kreirali red sa određenim tipom koristeći privremeno skladište, morate postaviti broj bajtova na višestruki broj bajtova navedenih iznad.

```js
// Create same Int16Array array differently
var byteSize = 6; // Needs to be multiple of 2
var buffer = new ArrayBuffer(byteSize);
var i8View = new Int16Array(buffer);
buffer.byteLength; // Returns 6
i8View.byteLength; // Returns 6
console.log(i8View); // Returns [0, 0, 0]
```

<dfn>Privremeni skladište</dfn> su opšti predmeti koji nose samo podatke. Ne možete im pristupiti direktno. Da biste im pristupili, prvo morate kreirati <dfn>pogled</dfn>.

```js
i8View[0] = 42;
console.log(i8View); // Returns [42, 0, 0]
```

**Napomena**
Red specifičnog tipa nema neke metode koje redovi opšteg tipa imaju, kao što su `.pop()` ili `.push()`. Ovi redovi takođe ne mogu da prođu/obrade `Array.isArray()` koji proverava da li je nešto red podataka (data row). Iako je ovo jednostavniji, to može biti prednost za motore JavaScript koji nisu dovoljno sposobni da izvrše ove redove.

# --instructions--

# --hints--

Tvoj `buffer` treba da ima veličinu od 64 bajta.

```js
assert(buffer.byteLength === 64);
```

Tvoj prikaz bafera `i32View` treba da ima veličinu od 64 bajta.

```js
assert(i32View.byteLength === 64);
```

Vaš prikaz `i32View` za privremeni keš mora imati dužinu od 16 elemenata.

```js
assert(i32View.length === 16);
```
