# --description--

$ABCD$ je četvorougaonik sa spoljnim uglovima, čije stranice imaju pune brojeve i $1 ≤ AB &lt; BC &lt; CD &lt; AD$.

$BD$ ima dužinu punog broja. $O$ je u sredini $BD$-a. $AO$ ima dužinu punog broja.

Naziva se $ABCD$ četvorostranični pravougaonik savršeno bicliničan ako $AO = CO ≤ BO = DO$.

Na primer, sljedeći četvorustan je kompletan četvorustan biclinika: $AB = 19$, $BC = 29$, $CD = 37$, $AD = 43$, $BD = 48$ i $AO = CO = 23$.

<img alt="mstatili wa pande nne ABCD, na nukta O, katikati ya BD" src="https://cdn.freecodecamp.org/curriculum/project-euler/biclinic-integral-quadrilaterals.gif" style="background-color: white; padding: 10px; display: block; margin-right: auto; margin-left: auto; margin-bottom: 1.2rem;">

Neka je $B(N)$ broj pravougaonika sa četiri pune stranice biclinike različite $ABCD$ koje zadovoljavaju ${AB}^2 + {BC}^2 + {CD}^2 + {AD}^2 ≤ N$. Možemo dokazati da $B(10\\,000) = 49$ i $B(1\\,000\\,000) = 38239$.

Pronađi $B(10\,,000\,000\,000)$.
