# --interactive--

Kada radiš sa nizom znakova u JavaScript, često je potrebno izvući podsekvencu ili deo iz većeg niza.

Na primer, možda želite da izvučete deo reči, određeni niz slova, ili čak deo rečenice.

JavaScript pruža nekoliko metoda za ovu vežbu, jedna od najčešće korišćenih metoda je `slice()`.

Metoda ``slice()`` omogućava vam da izvučete deo niza znakova i vrati novi niz, bez menjanja originalnog niza. Prihvata dva parametra: početni indeks i krajnji indeks koji se može izvući.

Evo osnovne sintakse:

```js
string.slice(startIndex, endIndex);
```

`startIndex` je mesto gde izlaz počinje. `endIndex` je mesto gde izlaz se završava. Ako nije dostavljeno, `slice()` izbacuje do kraja sekvence karaktera.

Pogledajmo jednostavan primer ekstrakcije dela niza znakova:

:::interactive_editor

```js
let message = "Hello, world!";
let greeting = message.slice(0, 5);

console.log(greeting);  // Hello
```

:::

U ovom primeru, `slice(0, 5)` počinje od elementa `0` do, ali ne uključujući element `5`. Kao rezultat, reč `Hello` se dobija.

Ako zanemarite drugi parametar, `slice()` će vratiti sve od početnog modela do kraja sekvence karaktera:

:::interactive_editor

```js
let message = "Hello, world!";
let world = message.slice(7);

console.log(world);  // world!
```

:::

Ovde, `slice(7)` se izvlači iz sekvence karaktera počevši od obrasca `7` do kraja sekvence, a rezultat je `world!`.

Takođe možete koristiti negativne brojeve kao indekse. Kada koristite negativan broj, on broji unazad od kraja niza karaktera:

:::interactive_editor

```js
let message = "JavaScript is fun!";
let lastWord = message.slice(-4);

console.log(lastWord);  // fun!
```

:::

U ovom slučaju, `slice(-4)` uklanja poslednje četiri karaktera iz sekvence i zamenjuje ga sa `fun!`.

Recimo da želite izvući deo iz sredine niza karaktera. Možete navesti početne i krajnje parametre kako biste tačno kontrolisali koji deo niza vam treba:

:::interactive_editor

```js
let message = "I love JavaScript!";
let language = message.slice(7, 17);

console.log(language);  // JavaScript
```

:::

Ovde, `slice(7, 17)` dolazi mala sekvenca počevši od elementa 7 i završava pre elementa `17`, koji je reč `JavaScript`.

Način za `slice()` je moćan alat za izvlačenje delova sekvence karaktera u JavaScript.

Objašnjava početne i krajnje markere, a ta metoda vraća novu sekvencu koja prikazuje uklonjeni deo.

Za opcije pozitivnih, negativnih i neobjavljenih atributa, možete ga menjati u različitim stanjima bez promene originalne sekvence.
