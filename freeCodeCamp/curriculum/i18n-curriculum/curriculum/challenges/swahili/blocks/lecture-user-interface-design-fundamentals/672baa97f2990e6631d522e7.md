# --interactive--

**KUMBUKA**: Baadhi ya mifano ya mwingiliano inaweza kutumia CSS ambayo bado hujajifunza. Usijali kuhusu kuelewa msimbo wote. Lengo la mifano hii ni kukuonyesha mapitio ya dhana hizi za muundo ili uelewe vizuri jinsi vitu vinavyofanya kazi.

Prvo, možda ćeš/ćete morati da naučiš/naučite o kontrastu. Kontrast je razlika između dve boje — ili koliko je lako da ih razlikuješ/razlikujete.

Rangi zenye uwiano mkubwa wa utofauti wa rangi zitaonekana tofauti zaidi kwa macho, wakati rangi zenye uwiano mdogo wa utofauti zitakuwa karibu zaidi kwa mwonekano. Kwa mfano, rangi nyeusi na nyeupe zina uwiano mkubwa sana wa utofauti. Wakati rangi ya samawati nyepesi na zambarau nyepesi zina uwiano mdogo wa utofauti.

:::interactive_editor

```html
<style>
  .example {
    padding: 20px;
    margin-bottom: 20px;
    font-size: 18px;
    font-family: sans-serif;
  }

  .high-contrast {
    background-color: black;
    color: white;
  }

  .low-contrast {
    background-color: #add8e6; 
    color: #dda0dd; 
  }

  .label {
    font-weight: bold;
    margin-bottom: 5px;
  }
</style>

<div class="example high-contrast">
  <div class="label">High Contrast</div>
  This text has high contrast (black background and white text).
</div>

<div class="example low-contrast">
  <div class="label">Low Contrast</div>
  This text has low contrast (light blue background and light purple text).
</div>
```

:::

Bila shaka, katika mifano hii tofauti inaweza kuonekana wazi zaidi kutokana na mpangilio. Lakini vipi kuhusu kutumia rangi hizi kwenye maandishi? Unaweza kuwa na maandishi meusi yenye utofauti mkubwa kwenye mandhari nyeupe, na maandishi ya zambarau yenye utofauti mdogo kwenye mandhari ya samawati.

:::interactive_editor

```html
<style>
  .text-example {
    padding: 10px;
    margin-bottom: 20px;
    font-size: 18px;
    font-family: sans-serif;
  }

  .text-high-contrast {
    background-color: white;
    color: black;
  }

  .text-low-contrast {
    background-color: #add8e6; 
    color: #dda0dd; 
  }

  .label {
    font-weight: bold;
    margin-bottom: 5px;
    display: block;
  }
</style>

<div class="text-example text-high-contrast">
  <span class="label">High Contrast Text</span>
  This is high contrast text: black text on a white background.
</div>

<div class="text-example text-low-contrast">
  <span class="label">Low Contrast Text</span>
  This is low contrast text: purple text on a light blue background.
</div>
```

:::

Lakini utajuaje ni uwiano gani wa utofauti unaofaa? Huwezi kuamua tu kwa jinsi maandishi yanavyoonekana kwako, kwa sababu kila mtumiaji atakuwa na uzoefu tofauti. Kwa bahati nzuri, Miongozo ya Ufikiaji wa Maudhui ya Mtandao, au WCAG, hutoa kiwango cha kawaida kwa hili.

Tekst sa kontrastnim omjerom od 4.5:1 smatra se AA nivoom, što je minimalni standard koji morate pratiti kako bi bio dostupan širokom spektru korisnika. Tekst sa kontrastnim omjerom od 7:1 smatra se AAA nivoom i osigurava najbolji nivo pristupačnosti.

:::interactive_editor

```html
<style>
  .contrast-example {
    padding: 15px;
    margin-bottom: 20px;
    font-size: 18px;
    font-family: sans-serif;
  }

  .aa-contrast {
    background-color: #ffffff; 
    color: #4b4b4b; 
  }

  .aaa-contrast {
    background-color: #ffffff; 
    color: #1a1a1a; 
  }

  .label {
    font-weight: bold;
    margin-bottom: 8px;
    display: block;
  }
</style>

<div class="contrast-example aa-contrast">
  <span class="label">AA Standard (Contrast Ratio ~4.5:1)</span>
  This text meets the minimum accessibility standard for contrast.
</div>

<div class="contrast-example aaa-contrast">
  <span class="label">AAA Standard (Contrast Ratio ~7:1)</span>
  This text meets the highest accessibility standard for contrast.
</div>
```

:::

Kuna tovuti nyingi zinazoweza kukagua uwiano wa utofauti kati ya rangi mbili, lakini vivinjari vingi vitakuruhusu kufanya hivi moja kwa moja katika zana za msanidi programu kwenye tovuti yako.

Tuchunguze mifano michache zaidi ya uwiano wa utofauti ili kuelewa dhana hii vizuri.

Mfano huu una uwiano wa utofauti wa 1.48:1 ambao hata haukidhi kiwango cha AA.

Možeš li ovo popraviti? Dobro, postoje tri stvari koje utiču na koeficijent različitosti.

Prvo je nijansa (hue), koja predstavlja vrste boja uopšteno, kao što su crvena, plava, narandžasta. Hajde da promenimo nijansu u nešto što je najdalje od plave. U ovom slučaju, koristite crvenu.

:::interactive_editor

```html
<style>
  .contrast-21 {
    background-color: white;
    color: black;
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }

  .label {
    font-weight: bold;
    margin-bottom: 8px;
    display: block;
  }

  .red-on-blue {
    background-color: #0000cc; 
    color: #ff0000; 
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }
</style>

<div class="contrast-21">
  <span class="label">Contrast Ratio 21:1</span>
  This example more than meets the AAA standard with black text on white background.
</div>

<div class="red-on-blue">
  <span class="label">Red on Blue (Higher Contrast Hue Shift)</span>
  This doesn't meet accessibility standards.
</div>
```

:::

Nažalost, ove promene su dovele odnos kontrasta na samo 1.49:1, što znači da morate promeniti saturaciju ili "količinu" postojeće boje. Povećajte količinu crvene u tekstu. Ovo nas približava cilju, sa odnosom kontrasta od 3.54:1.

```html
  }

    color: #ff4d4d;
    padding: 15px;
    font-family: sans-serif;
    font-size: 18px;
    margin-bottom: 20px;
  }
</style>

<div class="contrast-21">
  <span class="label">Contrast Ratio 21:1</span>
  This example more than meets the AAA standard with black text on white background.
</div>

<div class="low-sat-red-on-blue">
  <span class="label">Low Saturation Red on Blue (Contrast ~1.49:1)</span>

</div>
```

Međutim, možete se približiti menjanjem poslednje vrednosti, koja je svetlina. Svetlina predstavlja količinu belog prisutnog u boji.

```html
  .high-sat-red-on-blue {

  }

    background-color: #add8e6; 
    color: #8b0000;
  }
</style>

<div class="contrast-21">
  <span class="label">Contrast Ratio 21:1</span>
  This example more than meets the AAA standard with black text on white background.
</div>

  <span class="label">Low Saturation Red on Medium Blue (Contrast ~1.49:1)</span>

</div>

</div>

  <span class="label">Darker Red on Light Blue (Contrast ~10.34:1)</span>
  Decreasing the lightness of the red increases the contrast ratio significantly.
</div>
```
