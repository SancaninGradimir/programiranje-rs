# --description--

Izjava CSS sa većom pažnjom biće prioritetna u odnosu na one sa manjom pažnjom. Inline obrasci, koje si naučio u prethodnoj lekciji, imaju više pažnje u poređenju sa selektorima, dok svaki tip selektora ima svoj nivo pažnje koji doprinosi ukupnom nivou pažnje izjave. Drugi selektori doprinose pažnjom, ali ti se fokusiraš samo na ono što si naučio do sada:

1. Selektori za ID (najdetaljniji selektor)
2. Selektori za klasu objekata
3. Selektor po tipu

Prioritet se uzima u obzir samo kada element ima više protivrečnih izjava koje ciljaju na taj element, kao što je pravilo za određivanje pobednika. Selektor ID će uvek pobediti bilo koji broj selektora klase objekata, selektor klase objekata će uvek pobediti bilo koji broj selektora tipa, i selektor tipa će uvek pobediti bilo koji broj selektora sa nižim prioritetom od njega. Kada nema izjave sa selektorom višeg prioriteta, veći broj jednog selektora pobediće manji broj istog selektora.
