# --interactive--

U prethodnim lekcijama radili smo sa svojstvom `grid-template-columns` i ručno zadavali vrednosti za nekoliko kolona.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

Iako je sledeći kod potpuno validan, postoji jednostavan način da ponovite sve sekcije ili liste vaših ruta.

Funkcija `repeat()` koristi se za ponavljanje jedne staze ili cele liste staza za redove ili kolone. Ona prima broj ponavljanja i stazu koju želiš da ponoviš.

Ovde je revidirana verzija originalnog primera koristeći funkciju 'repeat':

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  column-gap: 10px;
}

.box {
  width: 100px;
  height: 100px;
  background-color: darkblue;
}
```

:::

U pregledaču neće biti promena u prikazu stilova, ali je ovo kraći način za pisanje ponovljenih vrednosti kolona.

Funkcija `repeat()` prihvata bilo koji ispravan obrazac koji možeš koristiti za redove ili kolone.

Ovde je primer korišćenjem funkcije `repeat()` za postavljanje prve i treće kolone na 20 piksela, a druge i četvrte kolone na jednu frakcionu jedinicu.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
  <div class="box"></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(2, 20px 1fr);
  column-gap: 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

Ponekad možeš izabrati da svaku vrednost napišeš pojedinačno umesto da koristiš funkciju `repeat()`. Međutim, postoje situacije u kojima je ova funkcija veoma korisna, naročito kada želiš da ponoviš određeni obrazac u listi staza.
