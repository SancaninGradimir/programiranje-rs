# --explanation--

`Physical access` To znači dobijanje pristupa lokaciji, kao što je kancelarija ili serverska soba.

`physical access breach` To se dešava kada osoba uđe u siguran prostor bez ovlašćenja. Na primer:

- `To avoid a physical access breach, we should require ID checks at the entrance.` - Kompanija može sprečiti neovlašćeni ulaz proverom identifikacione kartice na vratima.

- `A physical access breach can happen if someone enters the server room without permission.` - Problem sa fizičkom bezbednošću može nastati kada neko uđe u server salu bez ovlašćenja.
