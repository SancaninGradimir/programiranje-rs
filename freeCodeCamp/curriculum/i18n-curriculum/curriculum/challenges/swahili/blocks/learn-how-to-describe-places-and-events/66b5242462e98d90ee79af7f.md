# --explanation--

`Might` Koristi se za prikaz mogućnosti ili neizvesnosti. Ukazuje da je nešto moguće, ali nije sigurno. Na primer:

- `The system might crash if too many users log in at the same time.` - Postoji mogućnost da sistem će da padne, ali to nije sigurno.

- `He might finish the coding task by the end of the day.` - Može završiti vežbu, ali to nije sigurno.
