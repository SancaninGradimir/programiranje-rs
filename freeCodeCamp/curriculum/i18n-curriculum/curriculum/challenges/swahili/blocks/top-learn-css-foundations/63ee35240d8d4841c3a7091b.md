# --description--

Selektori klase će izabrati sve elemente koji imaju `class`, što je svojstvo elementa koji postavljate na element HTML. Evo kako dodati klasu na etiketu HTML i izabrati u CSS:

```html
<!-- index.html -->

<div class="alert-text">
  Please agree to our terms of service.
</div>
```

```css
/* styles.css */

.alert-text {
  color: red;
}
```

Zapamtite sintaksu selektora za `class`: tačka odmah praćena vrednošću klase atributa koja je prepoznata velikim i malim slovima. Ove klase ne mogu biti jedinstvene, pa možete koristiti `class` na više elemenata koliko želite.

Još jedna stvar koju možete uraditi sa svojstvom `class` je dodavanje mnogo klas u jedan element kao lista odvojena razmacima, kao `class="alert-text severe-alert"`. Pošto se razmaci koriste za razdvajanje imena `class` poput ovoga, ne biste trebali koristiti razmake za imena sa više reči i umesto toga koristite crticu (-).

## ID Selektori
ID selektori su isti kao selektori za `class`. Selektuju element koji ima `id`, što je drugi atribut koji postavljate na element HTML:

```html
<!-- index.html -->

<div id="title">My Awesome 90's Page</div>
```

```css
/* styles.css */

#title {
  background-color: red;
}
```

Umesto tačke, koristite znak hashtag praćen odmah vrednošću atributa `id` koji je prepoznatljiv velikim i malim slovima. Uobičajni problem je što ljudi prekomerno koriste atribut `id` kada to nije potrebno, a kada će biti dovoljno koristiti klase elemenata. Iako postoje slučajevi kada je korišćenje `id` tačno ili neophodno, kao što je korišćenje specifične prednosti ili linkovi koji preusmeravaju na sekciju trenutne stranice, trebalo bi da koristite `id` u maloj meri (ili uopšte).

Glavna razlika između klasa i ID-ja je što element može imati samo jedan `id`. `id` se ne može ponoviti na jednoj stranici, a atribut `id` ne sme imati nikakvo razmak.

## --text--

Koja je sintaksa selektora za klasu objekata i ID?

## --answers--

Ako izaberete `class`, koristite `$`, i ako izaberete `id`, koristite `#`

---

Za odabir `class` koristite `.` i ili za odabir `id` koristite `*`

---

Ili birate `class` koristi `.`, i ili birate `id` koristi `#`
