# --description--
Dve poslednje izazova se tiču osnova za prikazivanje dinamičkih podataka korišćenjem D3 za metode `data()` i `enter()`. Ove metode uzimaju set podataka i, zajedno sa metodom `append()`, kreiraju novi element DOM za svaki element u setu podataka.

U prethodnom izazovu, kreirao si novi element `h2` za svaku stavku u redu podataka `dataset`, ali svi su imali isti sadržaj, `New Title`. To je zato što ne koristiš podatke povezane sa svakim elementom od `h2`.

Putanja D3 za `text()` može prihvatiti niz znakova ili funkciju povratnog poziva kao argument:

```js
selection.text(d => d);
```

U gornjem primeru, parametar ``d`` se odnosi na jedan element u skupu podataka kojem je izabranje vezano.

Koristeći ovaj primer kao kontekst, prvi element od `h2` je povezan sa 12, drugi element od `h2` je povezan sa 31, treći element od `h2` je povezan sa 22, i tako dalje.

# --instructions--
Promeni putanju `text()` tako da svaki element `h2` ima odgovarajuću vrednost iz kolone podataka `dataset`, zajedno sa jednim razmakom i nizom slova `USD`. Na primer, prva zaglavlja bi trebalo da budu `12 USD`.
