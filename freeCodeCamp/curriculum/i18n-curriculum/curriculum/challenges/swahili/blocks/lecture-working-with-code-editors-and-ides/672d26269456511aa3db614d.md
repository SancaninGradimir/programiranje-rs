# --description--

Možda mislite da su ovo iste stvari, ali to nije uopšte. Editor koda je bilo koji program koji vam omogućava da uređujete fajlove sa kodom. IDE, ili Integralno razvojno okruženje (Integrated Development Environment), je kompletan program koji vam omogućava da kompajlirate, pokrenete i pronađete greške u vašem kodu dok ga uređujete.

Možda nekoliko primera će pomoći.

Prvo, razmotrimo nekoliko popularnih IDE-ova. Visual Studio je integrisano razvojno okruženje koje pruža kompletne alate za izgradnju, debagovanje i distribuciju aplikacija na različitim platformama.

Drugi IDE je Xcode. Xcode je integrisano razvojno okruženje koje je Apple napravio i namenjeno za kreiranje, pisanje koda i pronalaženje grešaka u aplikacijama za macOS, iOS, watchOS i tvOS.

Još jedan IDE je Android Studio. Android Studio je integrisano razvojno okruženje koje pruža Google i koje je razvijeno posebno za izradu, debagovanje i testiranje Android aplikacija.

Ovo su samo nekoliko primera IDE-a. Sada, hajde da pogledamo neke popularne editore koda.

Visual Studio Code je lagani, open-source editor koda od Microsofta koji podržava mnoge programski jezike i nudi funkcije kao što su pronalaženje grešaka, isticanje sintakse i kontrolu verzija putem ekstenzija.

Još jedan popularan editor je Sublime Text. Sublime Text je brzi i moćni tekstualni editor poznat po svom lepog izgledu, snažnim funkcijama i podršci za mnoge programski jezike putem isticanja sintakse koja se može prilagoditi dodacima (plugins).

Drugi je Notepad++. Ovo je besplatan, open-source uređivač teksta i izvornog koda za Windows koji nudi isticanje sintakse, skupštavanje koda i različite vrste dodataka radi povećanja efikasnosti i fleksibilnosti.

Video si kako uređivači koda uglavnom obraćaju pažnju na sadržaj tekstualnih datoteka, dok IDE-ovi nude razne alate za upravljanje kodom.

Sve ove primere su lokalne aplikacije koje možete pokrenuti na svom računaru, ali postoje i online editori koje možete koristiti.

Onlajn uređivač je onlajn alat koji omogućava korisnicima da pišu, uređuju i upravljaju kodom ili tekstom direktno putem veb pregledača bez potrebe za lokalnom instalacijom softvera.

Neka razmotrimo neke onlajn urednike.

Replit je online platforma koja pruža kolaborativno okruženje za pisanje koda, omogućavajući korisnicima da pišu, pokreću i dele kod u različitim programskim jezicima direktno iz web pregledača.

Još jedan popularni onlajn editor je GitHub Codespaces. Ovo je cloud razvojno okruženje koje pruža trenutni pristup punom editoru koda i alatima za razvoj direktno iz GitHub, omogućavajući neprekidno kodiranje i saradnju.

I drugi je Ona. To je okruženje za web razvoj koje uključuje GitHub i GitLab, pružajući radne prostore koji se mogu trenutno modifikovati za pisanje koda, buildovanje, i otklanjanje grešaka direktno iz vašeg pregledača.

Postoji još mnogo opcija. Neke opcije, kao što je Visual Studio Code, mogu biti veoma proširene i mogu raditi sa različitim tipovima projekata i jezika. Ostale opcije mogu biti specifične za manji skup jezika ili tipove projekata.

Program koji koristite može se razlikovati za specifične projekte. Trebalo bi da istražite opcije kako biste videli koja vam najviše odgovara vašim potrebama.
