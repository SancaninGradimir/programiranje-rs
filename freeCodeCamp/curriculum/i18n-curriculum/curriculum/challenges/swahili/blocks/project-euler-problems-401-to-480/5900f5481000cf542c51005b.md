# --description--

Neka je $R(a, b, c)$ najveća površina pokrivena od strane tri nepreklapajuće kružnice unutar trougla sa dužinama stranica $a$, $b$ i $c$.

Neka je $S(n)$ prosečna vrednost $R(a, b, c)$ za sve skupove od tri celobrojka $(a, b, c)$ koji zadovoljavaju uslov $1 ≤ a ≤ b ≤ c &lt; a + b ≤ n$.

Vi ste primili $S(2) = R(1, 1, 1) ≈ 0.31998$, $S(5) ≈ 1.25899$.

Izračunaj $S(1803)$, zaokruženo na 5 decimalnih mesta.
