# --interactive--

U JavaScript, šablonski znakovi su moćan i jednostavan način za rad sa nizovima karaktera. Za razliku od uobičajenih nizova znakova, koji koriste pojedinačne navodne znakove (`'`) ili dvostruke navodne znakove (`"`), šablonski znakovi se definišu korišćenjem backtick znaka (`` ` ``).

Omogućava veće olakšanje upravljanja nizovima karaktera, uključujući direktno unos parametara unutar niza karaktera, funkcija poznata kao unos vrednosti.

Template literali olakšavaju kreiranje niski koje se prostiru kroz više redova ili sadrže ugrađene izraze (kao što su promenljive ili čak JavaScript kod) direktno unutar niske.

Evo primera slova iz kalupa:

:::interactive_editor

```js
const name = "Alice";
const greeting = `Hello, ${name}!`;

console.log(greeting);
```

:::

Pogledajte upotrebu backtick-ova umesto pojedinačnih ili dvostruki navodnika. Sintaksa `${name}` je primer ubacivanja vrednosti, gde se vrednost parametra `name` direktno ubacuje unutar niza karaktera.

Unos vrednosti omogućio je unos parametara i matematičkih izraza unutar sekvence karaktera. Ovo je veliki napredak u poređenju sa starom metodom, gde ste kombinovali sekvencu karaktera i parametre korišćenjem operatera `+`.

Ovo je primer koji koristi konkatenaciju sekvence slova i operator za sabiranje (`+`):

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = "My name is " + name + " and I am " + age + " years old.";
console.log(message); 
```

:::

I ovde je primer koji koristi šablonske znakove (placeholder-e) i unos vrednosti:

:::interactive_editor

```js
const name = "Alice";
const age = 25;
const message = `My name is ${name} and I am ${age} years old.`;
console.log(message); 
```

:::

Kao što vidite, unos vrednosti za šablonne elemente je čistije i lakše za čitanje, posebno kada radite sa mnogo parametara.

Još jedna korisna osobina template literala jeste to što podržavaju višelinijske niske. Kod običnih niski morao bi da koristiš escape sekvencu (`\n`) za novi red. Sa template literalima dovoljno je da tekst napišeš u više redova i formatiranje će biti sačuvano:

:::interactive_editor

```js
let poem = `Roses are red,
Violets are blue,
JavaScript is fun,
And so are you.`;

console.log(poem);
```

:::

Još jedna karakteristika ovog skupa znakova je da ne dozvoljava direktno unosenje izraza JavaScript u sekvencu znakova, kao što je ovde prikazano:

:::interactive_editor

```js
const song = "Bohemian Rhapsody";
const score = 9.5;
const highestScore = 10;
const output = `One of my favorite songs is "${song}". I rated it ${
  (score / highestScore) * 100
}%.`;
console.log(output); 
```

:::

Digitalni znaci su važni posebno kada je potrebno uključiti parametre ili matematičke izraze u sekvencu znakova, kreirati složene sekvence znakova ili raditi sa višelinijskim tekstom. Kraći i lakši za čitanje su u poređenju sa tradicionalnim spajanjem sekvenci znakova.
