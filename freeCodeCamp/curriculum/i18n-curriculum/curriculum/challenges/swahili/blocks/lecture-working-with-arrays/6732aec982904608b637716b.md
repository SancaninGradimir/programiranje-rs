# --interactive--

Nizovi podataka u JavaScript su mutabilni, što znači da možete lako dodati ili ukloniti elemente iz njih. Postoji četiri glavna načina za dodavanje i uklanjanje elemenata na početku i kraju niza podataka: `push()`, `pop()`, `shift()`, i `unshift()`. Pogledaćemo sve ove metode u detalju.

Metoda `push()` koristi se za dodavanje jednog ili više elemenata na kraj niza. Povratna vrednost metode `push()` je nova dužina niza. Evo primera dodavanja novog voća u postojeći niz `fruits`:

:::interactive_editor

```js
const fruits = ["apple", "banana"];
const newLength = fruits.push("orange");
console.log(newLength); // 3
console.log(fruits); // ["apple", "banana", "orange"]
```

:::

U ovom primeru, počinjemo sa nizom podataka nazvan `fruits` koji ima dva elementa. Zatim koristimo metodu `push()` da dodamo sekvencu karaktera `orange` na kraj niza podataka.

Možda ste primetili da koristimo `const` kada deklarišemo niz podataka `fruits`. Ali zašto je moguće dodati više elemenata u ovaj niz podataka `fruits` kada je `fruits` konstantan? To je moguće jer deklarisanje niza podataka sa glavnom reči `const` stvara referencu na niz podataka. Iako je sam niz podataka promenljiv i može se menjati, ne možete dodeliti novu vrednost konstanti `fruits`, kao što sledi:

```js
const fruits = ["apple", "banana"];
fruits = ["This", "will", "not", "work"];
console.log(fruits); // Uncaught TypeError: Assignment to constant variable. 
```

Sledeći metod koji ćemo istražiti je metoda `pop()`. Metoda `pop()` uklanja poslednji element iz niza podataka i vraća taj element. Takođe modifikuje originalni niz podataka. Ovako funkcioniše:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
let lastFruit = fruits.pop();
console.log(fruits); // ["apple", "banana"]
console.log(lastFruit); // "orange"
```

:::

U ovom primeru, počinjemo sa nizom od tri voća. Metoda `pop()` uklanja poslednji element (`orange`) iz niza podataka i vraća ga. Originalni niz podataka `fruits` je modifikovan i sada ima samo dva elementa.

Metoda `unshift()` dodaje jedan ili više elemenata na početak niza podataka i vraća njegovu novu dužinu. Radi kao `push()`, ali menja početak niza podataka umesto kraja. Evo primera:

:::interactive_editor

```js
let numbers = [2, 3];
let newLength = numbers.unshift(1);
console.log(numbers); // [1, 2, 3]
console.log(newLength); // 3
```

:::

U ovom primeru koristimo `unshift()` da dodamo broj `1` na početak niza `numbers`. Ova metoda vraća novu dužinu niza, koja iznosi `3`.

Na kraju, metoda `shift()` uklanja prvi element iz niza podataka i vraća taj element. Slično je `pop()`, ali radi na početku niza podataka umesto na kraju. Evo kako to funkcioniše:

:::interactive_editor

```js
let colors = ["red", "green", "blue"];
let firstColor = colors.shift();
console.log(colors); // ["green", "blue"]
console.log(firstColor); // "red"
```

:::

U ovom primeru, počinjemo sa redom od tri boje. Metoda `shift()` uklanja prvi element (`red`) iz niza podataka i vraća ga. Originalni niz podataka `colors` je modifikovan i sada ima samo dva elementa.

Zapamti da, iako `push()` i `unshift()` mogu da dodaju više elemenata odjednom, `pop()` i `shift()` uklanjaju samo jedan element pri svakom pozivu.
