# --explanation--

Marija kaže, `at the end of each day`, što jasno pokazuje da su rezervne kopije zakazane svakog dana. To znači da je vežba planirana da se održi jednom svakog dana. Na primer:

- `At the end of each day, the system runs a data backup.` - Sistem skladišti sve podatke svake večeri radi sprečavanja gubitka.

- `We ensure the servers are updated at the end of each day.` - Ažuriranja se vrše na serveru svake noći kako bi se izbeglo poremećaj tokom radnog vremena.
