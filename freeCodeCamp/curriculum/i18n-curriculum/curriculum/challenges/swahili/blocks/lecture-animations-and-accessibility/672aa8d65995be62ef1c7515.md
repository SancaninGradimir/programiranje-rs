# --interactive--

Dizajni za CSS vam omogućavaju da lako kreirate dinamičke vizuelne efekte na web stranicama bez potrebe za JavaScript ili složenim programima. Oni pružaju način za glatko prebacivanje elemenata između različitih stilova tokom definisanog perioda.

U osnovi, diagram CSS ima dva glavna dela: princip `@keyframes` i funkcija animacije.

Kanuni `@keyframes` opisuje korake i stilove crteža. Opisuje stilove koje element treba da ima na različitim delovima crteža.

Evo primera:

```css
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(0);
  }
}
```

Ovo pravilo za `@keyframes`, sa imenom `slide-in`, opisuje crtež koji pomera element iz leve u desnu stranu. Procenti predstavljaju napredak crteža, gde je `0%` početak, a `100%` kraj.

Funkcija `translateX` u vašem crtežu `@keyframes` kontroliše poziciju elementa kada se pojavljuje.

Da biste koristili ovaj dijagram za komponentu, koristite svojstvo `animation`. Ovaj primer takođe ponavlja crtež neograničeno kako biste videli kako funkcioniše (moraćete da omogućite interaktivni uresnik kako biste videli prikaz):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="sliding-element">Hello, I slide in!</div>
```

```css
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
  }
  100% {
    transform: translateX(0);
  }
}

.sliding-element {
  animation: slide-in 2s ease-out infinite;
}
```

:::

Ovo koristi crtež `slide-in` za karakteristiku tokom 2 sekunde i funkciju vremena `ease-out`.

Atribut `animation` je skraćenica za više atributa pojedinačne osobe:

`animation-name` koje opisuje pravilo `@keyframes` koje će se koristiti.

`animation-duration` što određuje koliko vremena je potrebno da se crtež završi.

`animation-timing-function` koji opisuje kako se crtež menja tokom vremena - kao što su `ease`, `linear`, `ease-in-out`.

`animation-delay` što opisuje kašnjenje pre nego što crtež počne.

`animation-iteration-count` koji određuje koliko puta bi uzorak trebalo da se ponovi.

`animation-direction` što određuje da li se uzorak treba izvršiti napred, nazad ili zamijeniti.

`animation-fill-mode` koji objašnjava kako element treba da bude stilizovan pre i posle grafike.

`animation-play-state` što vam omogućava da pauzirate i nastavite crtež.

Možete koristiti ove funkcije jednu po jednu za detaljniju kontrolu:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="complex-animation">Watch my colors change!</div>
```

```css
.complex-animation {
  animation-name: color-change;
  animation-duration: 3s;
  animation-timing-function: linear;
  animation-delay: 1s;
  animation-iteration-count: infinite;
  animation-direction: alternate;
}

@keyframes color-change {
  0% {
    background-color: red;
  }
  50% {
    background-color: blue;
  }
  100% {
    background-color: green;
  }
}
```

:::

Ovo kreira uzorak koji stalno menja pozadinsku boju elementa između crvenog, plavog i zelenog.

Crteži za CSS mogu biti pokrenuti različitim događajima, kao što je prikazivanje elementa prelaskom miša preko njega:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<button class="button">Hover over me!</button>
```

```css
.button {
  background-color: blue;
  transition: background-color 0.3s;
}

.button:hover {
  background-color: red;
}
```

:::

Iako ovaj primer koristi atribut `transition`, koji je jednostavan za osnovne efekte, on pokazuje kako CSS može kreirati interaktivne komponente sa crtežima.

Važno je znati da, iako su grafike za CSS moćne, treba ih koristiti umereno. Prekomerno korišćenje grafike može uzrokovati loše performanse i može predstavljati prepreku ili problem za korisnike sa posebnim potrebama pristupačnosti. Uvek razmislite o pružanju opcija smanjenja ili onemogućavanja grafike korisnicima koji preferiraju minimalno kretanje.

Dizajni CSS pružaju način za kreiranje privlačnog i interaktivnog mrežnog iskustva bez zavisnosti od JavaScript. Razumevanjem principa `@keyframes` i karakteristika `animation`, možete dati život svojim mrežnim dizajnom sa odličnim performansama i dostupnošću.
