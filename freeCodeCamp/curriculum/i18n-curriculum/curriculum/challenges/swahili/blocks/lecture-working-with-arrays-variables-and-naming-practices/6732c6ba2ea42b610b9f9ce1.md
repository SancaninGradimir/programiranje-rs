# --description--

U prethodnoj lekciji, prvi put je uveden atribut ``length``, koji vraća broj elemenata u redu podataka. Evo kratkog podsjetnika kako funkcioniše:

```js
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits.length); // 3
```

Moguće je imati nizove podataka sa praznim mestima. Prazni prostori su definisani kao prostori koji ništa ne sadrže. Ovo je različito od niza podataka sa vrednošću `undefined`. Ove vrste nizova poznate su kao razređeni nizovi podataka (sparse arrays). Evo primera:

```js
const sparseArray = [1, , , 4];
console.log(sparseArray.length); // 4
```

U ovom slučaju, iako imamo samo dva deklarisana elementa, `1` i `4`, visina je `4` jer nam najgornji element (`3`) zajedno sa `1` daje visinu od `4`.

Sada analizirajmo kako kreirati prazan red fiksne visine. Postoji nekoliko načina da se ovo učini u JavaScript, ali jedan od najčešćih je korišćenje komponente `Array()` i argumenta broja. Komponenta `Array()` može se koristiti sa ključnom reči `new` za kreiranje novog reda. Evo primera:

```js
const emptyArray = new Array(5);
console.log(emptyArray.length); // 5
console.log(emptyArray); // [ , , , , ]
```

U ovom primeru, kreiramo novi red koristeći `Array(5)`. Ovo creira niz podataka sa prazninama dužine `5` gde su sve lokacije puste.

Drugi način za kreiranje praznog reda fiksne dužine je korišćenje metode `Array.from()` sa argumentom dužine. Za razliku od `new Array(n)`, ova metoda kreira red navedene dužine gde svi elementi postoje i imaju vrednost `undefined`:

```js
const fixedLengthArray = Array.from({ length: 5 });
console.log(fixedLengthArray.length); // 5
console.log(fixedLengthArray); // [undefined, undefined, undefined, undefined, undefined]
```

Ako želite kreirati niz određene dužine i popuniti ga podrazumevanom vrednošću, možete koristiti metodu `Array.fill()`:

```js
const filledArray = new Array(3).fill(0);
console.log(filledArray); // [0, 0, 0]
```

Ovo kreira red sa dužinom od tri i popunjava sve elemente sa vrednošću `0`. Napomena: kada ga popunite više stvari, sve lokacije se odnose na jedan objekat; ako vam treba nezavisna kopija, koristite povratni poziv ili `Array.from()` umesto toga.

Razumeti kako dobiti dužinu niza podataka i kreirati nizove fiksne dužine je važno za mnoge programerske zadatke, posebno kada inicijalizujete niz za specifične algoritme ili strukture podataka.
