## Svojstvo preljeva (overflow) za CSS

- **Značenje**: Preliv se odnosi na način na koji elementi rukuju sa sadržajem koji prelazi, ili "preliva", veličinu elemenata u kojima je zatvoren. Preliv je u dvodimenzionalnom smeru.
- **`overflow-x`**: X osa određuje horizontalni preliv.
- **`overflow-y`**: Y osa određuje vertikalni preliv.
- **`overflow`**: Svojstvo skraćivanja za `overflow-x` i `overflow-y`. Ako mu je dato jedna vrednost, oba prelivanja će koristiti tu. Ako su date dve vrednosti, `overflow-x` će koristiti prvu, a `overflow-y` će koristiti drugu.

:::interactive_editor

```html
<div class="overflow-box">

  <br />
  This is very long content that will overflow horizontally and vertically.

  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
  <br />
  This is very long content that will overflow horizontally and vertically.
</div>
```

```css
.overflow-box {
  width: 140px;
  height: 70px;
  border: 2px solid black;
  overflow-x: auto;
  overflow-y: auto;
  white-space:nowrap;
}
```

## Karakteristika transformacije CSS

- **Značenje**: Ova funkcija vam omogućava da koristite različite transformacije za elemente, kao što su rotiranje, skaliranje, savijanje ili translacija u 2D ili 3D prostoru.
- **Funkcija `translate()`**: Ova funkcija se koristi za pomeranje elementa sa trenutne lokacije.
- **Funkcija `scale()`**: Ova funkcija vam omogućava da promenite veličinu elementa.
- **Funkcija `rotate()`**: Ova funkcija vam omogućava da rotirate element.
- **Funkcija `skew()`**: Ova funkcija vam omogućava da savijete element.
- **Transformacije i pristupačnost**: Ako koristite transform za skrivanje ili prikaz sadržaja, osigurajte da je sadržaj i dalje dostupan za čitače ekrana i navigaciju tastaturom. Skriveni sadržaj treba da bude potpuno sakriven, kao što je korišćenje `display: none` ili `visibility: hidden`, umesto samo pomeranja van ekrana vizuelno.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="transform-box translate">Translate</div>
  <div class="transform-box scale">Scale</div>
  <div class="transform-box rotate">Rotate</div>
  <div class="transform-box skew">Skew</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: column;
  align-items: center;
  gap: 16px;
  flex-wrap: wrap;
}

.transform-box {
  width: 100px;
  height: 100px;
  background: coral;
  color: white;
  display: flex;
  align-items: center;
  justify-content: center;
}

/* translate() */
.translate {
  transform: translateX(20px);
}

/* scale() */
.scale {
  transform: scale(1.2);
}

/* rotate() */
.rotate {
  transform: rotate(15deg);
}

/* skew() */
.skew {
  transform: skew(10deg, 5deg);
}
```

## Primer kutije (box model)

- **Definicija**: U primeru okvira CSS, svaki element je okružen okvirima. Ovaj okvir ima četiri dela: područje sadržaja, `padding`, `border`, `margin`.
- **Područje sadržaja**: Područje sadržaja je najunutrašnji deo okvira. To je prostor koji drži stvarni sadržaj elementa, kao što su tekst ili slike.
- **`padding`**: Unutrašnji razmak je prostor koji sledi odmah nakon područja sadržaja. To je prostor između područja sadržaja i granice elementa.
- **`border`**: Granica je spoljni ivica ili linija elementa u primeru okvira CSS. To je vidljiva granica elementa.
- **`margin`**: Margina je prostor izvan granice elementa. Ona određuje udaljenost između elementa i drugih elemenata koji ga okružuju.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="box-model">Content</div>
```

```css
.box-model {
  width: 120px;
  background: lightblue;
  padding: 16px;
  border: 4px solid teal;
  margin: 20px;
  text-align: center;
}
```

## Kolaps margina (margin collapse)

- **Značenje**: Ova karakteristika se javlja kada vertikalne ivice susednih elemenata interaguju, stvarajući jednu ivicu koja je kombinacija obe. Ova karakteristika se odnosi samo na vertikalne ivice (gore i dole), a ne na horizontalne (levo i desno).

## Vrednosti atributa za `content-box` i `border-box`

- **Svojstvo `box-sizing`**: Ovo svojstvo se koristi za određivanje kako su poslednji `width` i `height` izračunati za element sa HTML.
- **Vrednost `content-box`**: U primeru `content-box`, postavljanje `width` i `height` za element određuje dimenzije sadržarnog područja, ali ne uključuje `padding`, `border` ili `margin`.
- **Vrednost `border-box`**: Za `border-box`, `width` i `height` elementa uključuju sadržarno područje, `padding` i `border`, ali ne uključuju `margin`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="sizing-box content-box">Content-box</div>
<div class="sizing-box border-box">Border-box</div>
```

```css
.sizing-box {
  width: 150px;
  padding: 10px;
  border: 4px solid teal;
  margin-bottom: 10px;
  background: peachpuff;
  text-align: center;
}

.content-box {
  box-sizing: content-box;  
}

.border-box {
  box-sizing: border-box;
  background: lightgreen;
}
```

## Obnova CSS (CSS reset)

- **Značenje**: Resetovanje CSS je stilski list koji se pruža radi uklanjanja bilo kakvog ili nekog podrazumevanog stila koji web pretraživači primenjuju na elemente HTML. Treće strane opcije za resetovanje CSS uključuju `sanitize.css` i `normalize.css`.

## Svojstvo filtera (filter) za CSS

- **Značenje**: Ovo svojstvo može se koristiti za kreiranje različitih efekata, kao što su zamućenje, promena boje i podešavanje osvetljenosti.
- **Funkcija `blur()`**: Ova funkcija primenjuje Gausovo zamućenje na element. Vrednost se zadaje u pikselima i predstavlja poluprečnik zamućenja.
- **Funkcija `brightness()`**: Ova funkcija podešava osvetljenost elementa. Vrednost od 0% učiniće element potpuno tamnim, dok vrednost veća od 100% povećava osvetljenost.
- **Funkcija `contrast()`**: Ova funkcija podešava kontrast elementa. Vrednost od 0% učiniće element potpuno sivim, 100% daje normalan izgled, a vrednosti veće od 100% povećavaju kontrast.
- **Funkcija `grayscale()`**: Ova funkcija pretvara element u nijanse sive. Vrednost se izražava u procentima, gde 100% znači potpuno sivu sliku, a 0% ostavlja sliku neizmenjenom.
- **Funkcija `sepia()`**: Ova funkcija primenjuje sepija efekat na element. Kao i `grayscale()`, koristi procentualnu vrednost.
- **Funkcija `hue-rotate()`**: Ova funkcija rotira nijansu boje elementa. Vrednost se zadaje u stepenima i predstavlja ugao rotacije na krugu boja.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <img class="filter blur" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Blur" />
  <img class="filter brightness" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Brightness" />
  <img class="filter contrast" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Contrast" />
  <img class="filter grayscale" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Grayscale" />
  <img class="filter sepia" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Sepia" />
  <img class="filter hue" src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Hue Rotate" />
</div>
```

```css
.container {
  display: flex;
  gap: 12px;
  flex-wrap: wrap;
}

.filter {
  width: 120px;
}

/* blur() */
.blur {
  filter: blur(3px);
}

/* brightness() */
.brightness {
  filter: brightness(130%);
}

/* contrast() */
.contrast {
  filter: contrast(150%);
}

/* grayscale() */
.grayscale {
  filter: grayscale(100%);
}

/* sepia() */
.sepia {
  filter: sepia(100%);
}

/* hue-rotate() */
.hue {
  filter: hue-rotate(90deg);
}
```
