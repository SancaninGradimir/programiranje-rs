# --description--

Uzmi sve tri sa:

- Svi vrhovi leže na opupu.
- Centar kruga (circumcenter) je u koordinatnom početku O.
- Centar ortogonala (orthocenter) je u tački H(5, 0).

Postoje devet trouglova kao onih sa $\text{ukingo} ≤ 50$.
