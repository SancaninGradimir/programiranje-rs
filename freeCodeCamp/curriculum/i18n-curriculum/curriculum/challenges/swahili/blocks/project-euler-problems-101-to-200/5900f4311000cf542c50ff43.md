# --description--

Nazvaćemo ga trougao sa stranicama celih brojeva i samo jednim uglom od 60°, odnosno 60° trougao.

Neka je $r$ krug rotacije nacrtan unutar trougla od 60°.

Postoje trouglovi 1234 za 60° koji $r ≤ 100$.

Neka je $T(n)$ broj trouglova sa uglovima od 60° koji zadovoljavaju uslov $r ≤ n$, pa stoga $T(100) = 1234$, $T(1000) = 22767$, i $T(10000) = 359912$.

Pronađi $T(1053779)$.
