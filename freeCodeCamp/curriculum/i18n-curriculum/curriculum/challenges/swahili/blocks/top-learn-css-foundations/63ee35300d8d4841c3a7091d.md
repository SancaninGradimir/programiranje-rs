# --description--

Drugi način korišćenja selektora je da ih povežete kao listu bez razdvajanja bilo čime. Recimo da imate HTML sledeći:

```html
<div>
  <div class="subsection header">Latest Posts</div>
  <p class="subsection preview">This is where a preview for a post might go.</p>
</div>
```

Imate dva elementa sa klasom `subsection` koji imaju jedinstvene stilove, ali šta ako želite da koristite drugačilo pravilo za element koji takođe ima `header` kao drugu klasu? Da, možete kombinovati sve selektore od `class` zajedno u vašem CSS na sledeći način:

```css
.subsection.header {
  color: red;
}
```

Ono što radi ``.subsection.header`` je odabir bilo kog elementa koji ima klase ``subsection`` i ``header`` zajedno. Pogledajte kako nema razmaka između selektora ``.subsection`` i ``.header`` ``class``. Ova sintaksa radi kombinovanjem bilo koje kombinacije selektora, osim spajanja više od jednog tipa selektorâ.

Ovo se takođe može koristiti za povezivanje klase objekata i identifikatora (ID), kao što je prikazano ispod:

```html
<div>
  <div class="subsection header">Latest Posts</div>
  <p class="subsection" id="preview">This is where a preview for a post might go.</p>
</div>
```

Možete kombinovati dva navedena elementa sa sledećim:

```css
.subsection.header {
  color: red;
}

.subsection#preview {
  color: blue;
}
```

Generalno, ne možete kombinovati više od selektora jednog tipa jer element ne može biti dva različita tipa istovremeno. Na primer, kombinovanje dva selektora tipa kao što su `div` i `p` stvorilo bi selektor `divp`, koji ne bi radio jer bi selektor tražio element tipa `<divp>`, a on ne postoji.
