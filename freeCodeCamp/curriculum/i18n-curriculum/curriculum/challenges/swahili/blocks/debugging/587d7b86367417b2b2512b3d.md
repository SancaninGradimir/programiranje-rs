# --description--
Posledovna tema je zastrašujuća beskonačna petlja. Petlje su koristan alat kada vam treba da vaša programa izvrši određeni kod više puta ili dok ne bude ispunjen određeni uslov, ali zahtevaju pravi uslov za izlazak koji prekida tu petlju. Beskonačne petlje mogu uzrokovati zamrzavanje ili pad pregledača, izazivajući opšti haos u izvršavanju programa, što nikome ne odgovara.

Postojao je primer beskonačnog kruga u uvodu ovog dela - nije imao validan način da izađe iz kruga `while` unutar `loopy()`. NE KORISTI ovu funkciju!

```js
function loopy() {
  while(true) {
    console.log("Hello, world!");
  }
}
```

Odgovornost programa je da osigura da se na kraju postigne uslov za izlazak, koji programu govori kada napustiti kod petlje. Jedna greška je pogrešno povećanje ili smanjenje vrednosti brojačke varijable u odnosu na uslov za izlazak. Druga je slučajno resetovanje vrednosti brojačke varijable ili bilo koje varijable unutar koda petlje, umesto da se ona poveća ili smanji.

# --instructions--
Funkcija `myFunc()` ima beskonačan ciklus jer validno stanje završetka `i != 4` nikada neće biti `false` (i prekida petlju) - `i` raste za 2 svaki ciklus, i direktno preskače preko 4 jer `i` nije ispravan broj od početka. Ispravite komparator u validnom stanju završetka da ciklus radi samo za `i` manji ili jednak 4.

# --hints--
Vaš kod bi trebalo da promeni uporedni pokazivač u validnom stanju završetka (srednji deo) ciklusa `for`.

```js
assert(__helpers.removeJSComments(code).match(/i\s*?<=\s*?4;/g).length == 1);
```

Tvoj kod mora da ispravi upoređivač poređenja u validnom stanju završetka ciklusa.

```js
assert(!__helpers.removeJSComments(code).match(/i\s*?!=\s*?4;/g));
```
