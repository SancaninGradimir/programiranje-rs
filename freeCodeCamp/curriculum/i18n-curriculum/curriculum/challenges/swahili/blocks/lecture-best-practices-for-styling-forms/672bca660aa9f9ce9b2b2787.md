# --interactive--

Naučite se o nekim uobičajenim problemima prilikom pokušaja dekorisanja specifičnih elemenata unosa, kao što su unos `datetime-local` i `color`.

Ovi specijalni tipovi ulaznih polja zavise od složenih elemenata koji su teški za kreiranje stvari poput birača datuma i boja. Ovo stvara veliki izazov u stilovanju ovih ulaznih polja. Jedan izazov je što, pošto osnovni stil potpuno zavisi od pretraživača, CSS koji čini da izbor izgleda isto u jednom pretraživaču može dati potpuno različite rezultate u drugom pretraživaču.

Ovo je primer unosa boje:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<form>
  <label for="favorite-color">Pick your favorite color:</label>
  <input type="color" id="favorite-color" name="favorite-color">
</form>
```

```css
input {
  padding: 8px 12px;
  margin: 8px 0;
  border-radius: 6px;
  border: 1px solid #ccc;
}

input[type="color"] {
  width: 60px;
  height: 40px;
  padding: 0;
  border: 2px solid #555;
  border-radius: 4px;
  cursor: pointer;
}

```

:::

Drugi izazov može biti kompleksnost umjetnog elementa. Razmislite o biraču datuma; ovdje ima mnogo pokretnih dijelova, a složena struktura umjetnog elementa može stvoriti veliki izazov pri primjeni stilizovanja na odgovarajuće lokacije.

Ovde je primer unosa datuma:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<form>
  <label for="birthdate">Select your birthdate:</label>
  <input type="date" id="birthdate" name="birthdate">
</form>
```

```css
input {
  padding: 8px 12px;
  margin: 8px 0;
  border-radius: 6px;
  border: 1px solid #ccc;
}

input[type="date"] {
  padding: 6px 10px;
  border: 2px solid #555;
  border-radius: 4px;
  font-size: 14px;
  cursor: pointer;
}

input[type="date"]::-webkit-calendar-picker-indicator {
  background-color: #4CAF50;
  color: white;
  border-radius: 4px;
  cursor: pointer;
}

```

:::

Bez sumnje, sa ovim teškim komponentama, takođe ste izloženi riziku da slučajno izgubite važnu funkciju dok ih ručno montirate/dekorujete. Ne samo da možete izgubiti važne indikatore, kao što je stanje pažnje ili odabrana komponenta, već možete potpuno razbiti selektor.

Zato što mnogi programeri zavise od biblioteka JavaScript ili specifičnih komponenti umesto korišćenja ugrađenih komponenata pregledača.
