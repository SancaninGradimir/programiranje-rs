# --description--

U prethodoj lekciji naučio si o tipu podataka lista i kako da pristupiš elementima liste i isečeš listu. U ovoj lekciji nastavićeš da učiš o listama i nekim uobičajenim metodama koje se koriste sa njima, kao što su `append()`, `pop()` i `sort()`.

Prva metoda o kojoj ćemo govoriti je `append()`. Ona se koristi za dodavanje elementa na kraj liste. Evo primera korišćenja metode `append()` za dodavanje broja `6` u listu brojeva:

```py
numbers = [1, 2, 3, 4, 5]
numbers.append(6)
print(numbers) # [1, 2, 3, 4, 5, 6]
```

Ako želiš da dodaš jednu listu na kraj druge, možeš takođe da koristiš metodu `append()` na sledeći način:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.append(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, [6, 8, 10]]
```

Obrati pažnju da je cela lista `even_numbers` ugnježdena unutar liste `numbers`.

Ali ako želiš da dodaš sve lične brojeve iz liste `even_numbers` na kraju liste `numbers`, onda možeš koristiti metodu `extend()`.

Putanja za `extend()` je ista kao putanja za `append()`, ali sa `extend()` možete dodati mnogo elemenata iz jedne liste u drugu. Evo primera dodavanja brojeva `6`, `8` i `10` iz jedne liste na kraju liste za `numbers`:

```py
numbers = [1, 2, 3, 4, 5]
even_numbers = [6, 8, 10]

numbers.extend(even_numbers)
print(numbers) # [1, 2, 3, 4, 5, 6, 8, 10]
```

Kao što vidiš, ugnježdena lista je uklonjena i sada je to samo lista brojeva.

Ako želiš da umetneš element na određeno mesto u listi, možeš da koristiš metodu `insert()`. Ova metoda prihvata dva argumenta: indeks na koji želiš da umetneš novi element i element koji želiš da umetneš.

Evo primera korišćenja metode `insert()`:

```py
numbers = [1, 2, 3, 4, 5]
numbers.insert(2, 2.5)

print(numbers) # [1, 2, 2.5, 3, 4, 5]
```

Sledeći kod će umetnuti broj `2.5` na indeks `2` u listi `numbers`.

Ako želiš da ukloniš element iz liste, možeš da koristiš metodu `remove()`. Metoda `remove()` kao argument prima vrednost elementa koji želiš da ukloniš:

```py
numbers = [10, 20, 30, 40, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50]
```

Važno je znati da ovaj metod će ukloniti samo prvo pojavljivanje elementa. Ne sve:

```py
numbers = [10, 20, 30, 40, 50, 50, 50]
numbers.remove(50)

print(numbers) # [10, 20, 30, 40, 50, 50]
```

Da biste uklonili element koji se nalazi u određenom polju u listi, možete koristiti metodu `pop()`, kao što sledi:

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop(1) # The number 2 is returned
```

Ako ne navedete element putem `pop`, vraća se poslednji element.

```py
numbers = [1, 2, 3, 4, 5]
numbers.pop() # The number 5 is returned
```

Ako želite da uklonite ceo spisak, onda možete koristiti metodu `clear()` kao što sledi:

```py
numbers = [1, 2, 3, 4, 5]
numbers.clear()

print(numbers) # []
```

Sledeći metod koji ćemo razgovarati je metoda `sort()`. Ovaj metod se koristi za raspoređivanje elemenata na njihovo mesto. Evo primera raspoređivanja liste brojeva po redosuku na njihovo mesto:

```py
numbers = [19, 2, 35, 1, 67, 41]
numbers.sort()

print(numbers) # [1, 2, 19, 35, 41, 67]
```

Za razliku od metode `sort()`, postoji funkcija `sorted()` koja radi za sve što se može iterirati i vraća novu sortiranu listu umesto da modifikuje originalnu listu. Primer:

```py
numbers = [19, 2, 35, 1, 67, 41]
sorted_numbers = sorted(numbers)

print(numbers) # [19, 2, 35, 1, 67, 41]
print(sorted_numbers) # [1, 2, 19, 35, 41, 67]
```

Kao podsjetnik, nešto što se može ponoviti je posebna vrsta stvari koje možete ponoviti, omogućavajući vam da dobijete svaki element direktno. Naučićete više o tome kako rade kola u Python u sledećoj lekciji.

Metoda `sort()` i funkcija `sorted()` prihvataju opcione parametre `key` i `reverse`. Saznaćete više o ovim opcionalnim parametrima u sledećoj lekciji kada počnete da učite o tuplovima. Takođe ćete saznati više o drugim ugrađenim funkcijama kao što je `sorted()` u narednim lekcijama.

Sledeća metoda koju ćemo razmotriti je metoda `reverse()`. Ova metoda će promeniti spisak komponenti na mestu, kao što sledi:

```py
numbers = [6, 5, 4, 3, 2, 1]
numbers.reverse()

print(numbers) # [1, 2, 3, 4, 5, 6]
```

Poslednja metoda koju ćemo pomenuti je `index`. Ovo se koristi za dobijanje prvog primera gde je funkcija dostupna u listi. Evo primera korišćenja metode `index` za pretraživanje jezika `'Java'` u listi `programming_languages`:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('Java') # 1
```

Ako element ne može biti pronađen, onda Python vraća `ValueError`:

```py
programming_languages = ['Rust', 'Java', 'Python', 'C++']
programming_languages.index('JavaScript')

"""
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ValueError: 'JavaScript' is not in list
"""
```

Postoji nekoliko drugih lista ruta od Python, ali ova početna lista ruta je dobro mesto za početak.
