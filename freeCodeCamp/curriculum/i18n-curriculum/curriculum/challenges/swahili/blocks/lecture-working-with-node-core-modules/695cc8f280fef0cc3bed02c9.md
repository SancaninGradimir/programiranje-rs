# --description--

Modul za OS je još jedan uobičajeni modul koji je uključen unutar Node.js.

Omogućava vam pristup trenutno pokrenutom operativnom sistemu gde je Node u funkciji ili dobijanje ključnih informacija kao što su tip OS-a, detalji procesora (CPU), dostupna memorija, ukupna memorija, mrežne konekcije i više.

Da bi koristio modul `os`, uvezi ga na sledeći način:

```js
const os = require("os");
```

Modul `os` pruža više metoda koje omogućavaju da tvoj program bude bolje prilagođen sistemu na kojem se izvršava. Pogledajmo neke od najčešće korišćenih metoda.

Metod `platform()` vraća operativni sistem na kojem se Node trenutno izvršava:

```js
const os = require("os");
console.log(os.platform()); // darwin
```

Ovo može biti korisno za pisanje skripti koje rade na različitim operativnim sistemima (cross-platform).

```js
if (os.platform() === 'win32') {
 // Windows specific code
} else {
 // Non-Windows specific code
}
```

Metod `arch()` vraća tekstualni niz koji predstavlja arhitekturu procesora (CPU) za koju je Node.js napravljen:

```js
const os = require("os");
console.log(os.arch()); // arm64
```

Ovo može pomoći ako želite da korisnici preuzmu binare i odgovarajuće zavisnosti za određenu arhitekturu.

Thamani nyingine zinazowezekana ni `'x64'`, `'arm'`, `'arm64'`, `'ia32'`, `'mips'`, `'ppc'`, na zingine.

`type()` dobija zvanično ime OS-a, ili može da detektuje operativne sisteme programski:

```js
const os = require("os");
console.log(os.type()); // Darwin (the core OS for macOS, iOS, and other Apple products)
```

`release()` prikazuje verziju kernela operativnog sistema, osnovnog dela operativnog sistema koji upravlja sistemskim resursima i komunikacijom između hardvera i softverskih komponenti. Ova metoda može pomoći u praćenju kompatibilnosti između verzija kernela operativnog sistema i zahteva servera.

Vraćam ovo kao niz slova, na primer `'20.6.0'`:

```js
const os = require("os");
console.log(os.release()); // 25.0.0
```

Metod `version()` vraća tačnu verziju operativnog sistema, sa više detalja nego metod `release()`.

```js
const os = require("os");
console.log(os.version());
// Darwin Kernel Version 25.0.0: Wed Sep 17 21:41:39 PDT 2025;
// root:xnu-12377.1.9~141/RELEASE_ARM64_T8103
```

`cpus()` Vraća niz objekata sa informacijama o svakom logičkom jezgru procesora (CPU). Ovo može pomoći u praćenju opterećenja procesora:

`uptime()` prikazuje vreme od kada je sistem pokrenut. Može pomoći u određivanju vremena koliko su serveri radili:

```js
const os = require("os");
console.log(os.uptime()); // 23047
```

`totalmem()` na `freemem()` pokazuju ukupnu sistemsku memoriju u bajtovima i slobodnu sistemsku memoriju u bajtovima, respektivno:

```js
const os = require("os");
console.log(os.totalmem()); // 8589934592 (8 GB)
console.log(os.freemem()); // 93585408 (87 MB)
```

`userInfo()` Vraća objekat sa informacijama o korišćenju trenutnog sistema:

```js
const os = require("os");
console.log(os.userInfo());

/*

 gid: 20,

 homedir: '/Users/user',
 shell: '/bin/zsh'
}
*/
```
