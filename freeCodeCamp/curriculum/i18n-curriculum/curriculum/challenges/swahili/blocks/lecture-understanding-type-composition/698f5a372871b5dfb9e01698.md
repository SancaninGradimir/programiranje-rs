# --description--

U prethodnim lekcijama, naučili ste kako da radite sa osnovnim tipovima kao što su `string`, i `boolean`. Ali TypeScript vam omogućava da kreirate nove tipove iz postojećih. Jedan primer ovoga je tip unije.

Vrsta kombinacije je vrsta koja spaja dve ili više vrsta.

Evo osnovne sintakse:

```ts
type1 | type2 | type3
```

Istražite nekoliko primera ili bolje razumete ovaj koncept.

Ovo je promenljiva pod nazivom `id` koja ima unioni tip `string` ili `number`:

```ts
let id: string | number = "user_783";
```

Vrednost može biti promenjena u broj, što je i dalje važeće za unioni tip:

```ts
let id: string | number = 1024;
```

Ako pokušate da postavite drugu vrednost koja nije `string` ili `number`, onda TypeScript će prikazati poruku o grešci:

```ts
// Type 'string[]' is not assignable to type 'string | number'.
let id: string | number = ["Java", "Python"];
```

Kada radite sa vrstama spajanja, važno je koristiti metode koje funkcionišu za sve vrste, inače TypeScript će nam dati grešku kao ova:

```ts
function getId(id: string | number) {
  // Property 'toUpperCase' does not exist on type 'string | number'.
  return id.toUpperCase();
}
```

Postoji rešenje za ovo koje uključuje smanjenje tipa veze. Ali naučićete više o tome, zajedno sa načinom na koji različiti tipovi veza funkcionišu i nazivima tih tipova u budućim časovima.

Vrste povezivanja takođe su korisne kada radite sa vrednostima `null` i `undefined`.

Evo primera gde promenljiva može imati vrednost tipa `string`, ili može biti `null` ako joj vrednost još nije dodeljena:

```ts
let username: string | null;

// Type 'number' is not assignable to type 'string | null'.
username = 42;
```

U ovom slučaju, `username` može biti `string` ili `null`, ali ništa drugo. `null` možeš koristiti kada namerno želiš da predstaviš odsustvo vrednosti.

Evo još jednog primera koji koristi `undefined`:

```ts
let score: number | undefined;

score = undefined; 

// Type 'string' is not assignable to type 'number | undefined'.
score = "high";
```

U ovom primeru, `score` može biti `number` ili `undefined`. Možete koristiti `undefined` kada vrednost nije definisana.

Upotreba tipova spajanja sa `null` i `undefined` čini jasno kada vrednost može biti nedostupna, ali i dalje omogućava TypeScript da detektuje distribuciju nevažećih vrednosti.
