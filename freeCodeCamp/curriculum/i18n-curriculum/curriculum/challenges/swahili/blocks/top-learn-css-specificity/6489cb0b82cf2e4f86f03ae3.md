# --description--

Hajde da pogledamo nekoliko brzi primera kako funkcioniše stvarnost. Pogledajte kod HTML i CSS u nastavku:

```html
<!-- index.html -->

<div class="main">
  <div class="list subsection"></div>
</div>
```

```css
/* rule 1 */
.subsection {
  color: blue;
}

/* rule 2 */
.main .list {
  color: red;
}
```

U ovom gornjem primeru, oba pravila koriste samo selektore klase elemenata, ali je pravilo 2 specifičnije jer koristi više selektora klase elemenata.
