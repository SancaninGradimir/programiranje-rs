# --description--

Termin `potential` se odnosi na nešto što bi moglo da se dogodi ili postoji u budućnosti, iako nije sigurno. U programiranju, pominjanje `potential issue` znači da može postojati problem u kodu koji bi mogao nastupiti pod određenim uslovima. To nije siguran problem, ali je nešto što može biti problem.

Na primer, `There is a potential issue if the user enters a very large number.` Ovo znači da se problem trenutno nije pojavio, ali može nastati ako se ta situacija dogodi.
