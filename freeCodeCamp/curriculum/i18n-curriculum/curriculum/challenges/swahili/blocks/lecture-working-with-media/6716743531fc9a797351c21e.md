# --interactive--

Izmenjeni element je element čije sadržaj određuje eksterni resurs umesto samog CSS. CSS, ili cascading stylesheets, koriste se za dodavanje stilova na web stranicu. Uobičajeni primeri izmenjenih elemenata uključuju elemente slike (image elements), iframe i video.

Za prilagodljive elemente, možete kontrolisati poziciju ili raspored elementa. Ali CSS ne može direktno da promeni sadržaj tog elementa. Ovo se može lakše razumeti korišćenjem primera. Razmislite o elementu za slike, koji ubacuje sliku na vaš veb sajt:

```html
<img src="example-img-url" alt="Descriptive text goes here">
```

Sam element se menja nečim spoljnim: slikom. CSS možete kontrolisati stvari kao što je veličina slike ili primeniti filter na nju, ali ne možete promeniti samu sliku. Stabilniji primer može biti element `iframe`, koji ugrađuje eksternu veb stranicu na vašu web stranicu.

Ovo je primer korišćenja funkcije `iframe` za popularni YouTube video sa kanala freeCodeCamp. Ako želite da vidite drugi video u prozoru za prikaz, promenite vrednost atributa `src` na željeni video. Da biste videli prikaz, moraćete da omogućite interfejsni editor.

**NAPOMENA**: Ne brinite o dodatnim karakteristikama pomenutim u interfejsu kao `referrerpolicy` i `allowfullscreen`. Naučićete više o radu sa komponentama `iframe` na sledejem radionici.

:::interactive_editor

```html
<iframe width="400" height="200" src="https://www.youtube.com/embed/u43gJJrVa1I?si=BoDW_puFsy8OEr_Z" title="Professional Cloud Architect Certification Course – Pass the Exam! (YouTube video)" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen></iframe>
```

:::

Drugi uobičajeni primeri upotrebe elementa ``iframe`` su ugradnja mape na stranicu. Ovde je primer ugrađene mape.

Omogući interaktivni editor i pokušaj da radiš sa samom mapom zumiranjem unutra/uvlačenjem.

:::interactive_editor

```html
<iframe
  title="Map of the Royal Observatory, Greenwich, London"
  width="300"
  height="200"
  src="https://www.openstreetmap.org/export/embed.html?bbox=-0.004017949104309083%2C51.47612752641776%2C0.00030577182769775396%2C51.478569861898606&amp;layer=mapnik">
</iframe>
```

:::

Sam element je zamenjen nečim spoljašnjim: web stranicom. Vaš CSS može da promeni raspored ugrađene web stranice, ali ne možete menjati njen sadržaj. Da biste više razumeli, ako ugrađena web stranica ima funkciju `h1`, vaš CSS ne može da joj doda tu stilsku funkciju `h1`. Ne možete menjati veličinu, boju fonta i slično.

Postoje drugi modifikovani elementi, kao `video` i `embed`. I neki elementi funkcionišu kao modifikovani elementi pod posebnim uslovima. Evo primera komponente `input` sa svojstvom `type` postavljeno na `image`:

```html
<input type="image" alt="Descriptive text goes here" src="example-img-url">
```

Ovaj tip `input` smatra se modifikovanim elementom, ali drugi tipovi `input` kao što su `text` ili `email` nisu modifikovani elementi.
