# --description--

Da biste pristupili elementima niza podataka, možete koristiti indeks broj. Indeksi počinju od 0, pa je prvi element niza podataka na indeksu 0, drugi element na indeksu 1, i tako dalje.

Na primer, da biste dobili prvi element reda `fruits`, možete koristiti sledeći kod:

```javascript
const fruits = ['apple', 'banana', 'orange'];
console.log(fruits[0]); // Output: apple
console.log(fruits[2]); // Output: orange
```

Ako je model van granica, JavaScript će vratiti `undefined`. To znači da ako pokušate da dobijete `fruits[3]` u navedenom primeru, će vratiti `undefined`.
