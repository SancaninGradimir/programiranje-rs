# --description--

Zapamtite da korišćenje oznaka `h1`, `h2` i sličnih određuje semantičku strukturu vašeg HTML. Međutim, možete prilagoditi CSS ovih elemenata da kontrolišete vizuelni tok i nivo strukture.

Napravi pravilo za `h1` i postavi svojstvo `font-weight` na `800`. Ovo će učiniti vaš tekst `h1` podebljanijim slovima.

# --hints--

Treba da napraviš selektor za `h1`.

```js
assert(new __helpers.CSSHelp(document).getStyle('h1'));
```

Vaš selektor za `h1` mora imati svojstvo `font-weight` postavljeno na `800`.

```js
assert(new __helpers.CSSHelp(document).getStyle('h1')?.fontWeight === '800');
```
