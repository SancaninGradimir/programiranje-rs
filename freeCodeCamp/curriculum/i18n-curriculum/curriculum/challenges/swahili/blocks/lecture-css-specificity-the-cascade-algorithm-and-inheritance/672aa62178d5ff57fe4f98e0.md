# --interactive--

Prioritet za CSS je osnovni koncept koji određuje koje stilove koristiti za element kada se može primeniti više pravila.

Razumevanje prioriteta pomaže programerima da reše konflikte između različitih pravila za CSS i osiguraju da se zahtevani stil koristi tačno u svakom trenutku.

Prioritet za CSS se računa u zavisnosti od tipa birača koji se koriste.

Najviši prioritet se dodeljuje stilovima unutar linije, koji se direktno primenjuju na element putem atributa `style`.

U ovom primeru, prvi pasus će biti crven dok će ostali elementi za `p` biti plavi. To je zato što inline stilovi imaju viši prioritet od selektora tipa kao što je selektor `p` prikazan u fajlu `styles.css`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p style="color: red;">Red paragraph</p>
<p>Other paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p {
  color: blue;
}
```

:::

Baada ya hapo, vichaguzi vya ID vina kiwango cha juu cha kipaumbele.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Red paragraph</p>
<p>Other paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#para {
  color: red; 
}

p {
  color: blue;
}
```

:::

Zatim dolaze selektori klase, atributski selektori i pseudoklase.

Primeri uključuju selektore klase kao što su `.container` i `.button`, atributske selektore kao što je `[type="text"]` i pseudoklase kao što je `:hover`. Oni imaju srednji nivo specifičnosti.

**NAPOMENA**: O pseudoklasama ćete naučiti više u narednim lekcijama.

U ovom primeru prvi pasus će biti crven zato što ID selektor ima veću specifičnost od selektora klase i selektora tipa. Elementi sa klasom `.example-para` biće zeleni, dok će ostali elementi pasusa biti plavi.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Example paragraph</p>
<p class="example-para">Other paragraph</p>
<p class="example-para">Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
#para {
  color: red; 
}

.example-para {
  color: green;
}

p {
  color: blue;
}
```

:::

Selektori tipa, kao što su `div` i `h1`, kao i pseudoelementi poput `::before` i `::after`, imaju nižu specifičnost u odnosu na prethodne grupe.

**PODSETNIK**: Naučićete više o veštačkim komponentama u nadolazećim časovima.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Example paragraph</p>
<p>Other paragraph</p>
<p>Yet another paragraph</p>
```

```css
#para {
  color: red; 
}

p {
  color: blue;
}
```

:::

Na kraju, svjetski šampion, predstavljen zvezdom `*`, ima najniži prioritet od svih.

Ovde je primer postavljanja boje za sve elemente na crveno korišćenjem selektora `*`. Međutim, nećete videti nikakve elemente u crvenoj boji jer postoje selektori za ID i tip koji nadjačavaju te stilove, što pokazuje nešto viši prioritet selektora `*`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p id="para">Example paragraph</p>
<p>Other paragraph</p>
<p>Yet another paragraph</p>
```

```css
* {
  color: red;
}

#para {
  color: green; 
}

p {
  color: blue;
}
```

:::

Vrednost medalje se računa kao vrednost četvrtine `(a, b, c, d)`:

- `a`: Inline stilovi (`1` ili `0`).
- `b`: Broj selektora za ID.
- `c`: Broj selektora klasa objekata, atributnih selektora i pseudo-klasa.
- `d`: Broj selektora tipova, pseudo-elemenata i globalnih selektora.

Svaka komponenta sa prioritetom ima drugačiju težinu:

- **Inline stilovi** (`a`) imaju najvišu težinu, doprinose vrednosti `1` za prvi deo prioritetne vrednosti.
    
- **Selektori ID** (`b`) imaju sledeću težinu, pri čemu svaki ID doprinosi `1` za drugi deo prioritetne vrednosti.
    
- **Selektori elemenata po klasi, selektori atributa i pseudoklasa** (`c`) imaju umerenu težinu, pri čemu svaki doprinosi `1` za treći deo prioritetne vrednosti.
    
- **Selektori tipa i pseudoelementi** (`d`) imaju najnižu težinu, pri čemu svaki doprinosi `1` za četvrti deo prioritetne vrednosti.
    
- **Universal selektor** (`*`): Universal selektor doprinosi `0` u proračun prioriteta i ne menja prioritet. Njegovo uključivanje u selektor ne menja vrednost prioriteta.

CSS Inline ima najviši prioritet jer se direktno primenjuje na element. Prekriva interni ili eksterni CSS. Vrednost prioriteta za inline stilove je `(1, 0, 0, 0)`.

Evo još jednog primera stila unutar reda:

:::interactive_editor

```html
<p style="color: red;">This text is red.</p>
```

:::

CSS interni je pisan unutar elementa `style` u sekciji `head` dokumenta HTML. Ima nešto viši prioritet od stilova unutar linije. CSS interni i eksterni imaju isti nivo prioriteta — prioritet se određuje korišćenim selektorima, a ne mestom gde CSS je napisano. Kada dva pravila imaju isti prioritet, pobeđuje ono koje se pojavi kasnije u dokumentu. Stoga, ako je eksterni fajl stila povezan nakon internog elementa `style`, eksterne pravila dobijaju prioritet. Ako je povezan pre, pobediuju interni pravila.

Prioritetna vrednost za interne stilove određena je korišćenim selektorima. Na primer, selektor ID unutar CSS ima prioritetnu vrednost od `(0, 1, 0, 0)`.

:::interactive_editor

```html
<head>
  <style>
    #text {
      color: blue;
    }
  </style>
</head>
<body>
  <div id="text">This text is blue.</div>
</body>
```

:::

U ovom primeru, tekst će biti plav osim ako se ne koristi inline stil ili selektor sa višim prioritetom.

CSS eksterni se povezuje putem funkcije `link` u sekciji `head` i zapisivan je u različitim fajlovima `.css`. Kao CSS interni, prioritet eksternih stilova određuje korišćeni selektor. CSS eksterni pruža bolju održivost za velike projekte.

Prioritetna vrednost za eksterne stilove određuje i korišćenje selektora. Na primer, selektor klase elemenata unutar CSS eksterni ima prioritetnu vrednost od `(0, 0, 1, 0)`.

:::interactive_editor

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <div class="text">This text's color is defined in an external CSS file.</div>
</body>
```

```css
.text {
  color: purple;
}
```

:::

U ovom primeru, boja teksta je definisana u fajlu `styles.css` i koristiće se osim ako nije nadjačana selektorom višeg prioriteta ili inline stilom.
