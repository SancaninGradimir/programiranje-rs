# --interactive--

Petlja `for...of` se koristi kada je potrebno iterirati nad vrednostima iz nečega što je iterabilno. Primeri iterabilnih objekata su nizovi podataka i sekvence karaktera.

Evo osnovne sintakse `for...of` petlje:

```js
for (variable of iterable) {
  // code block to be executed
}
```

Parametar u primeru predstavlja trenutnu vrednost ponavljajućeg objekta.

Ukoliko imate niz brojeva, parametar će biti trenutni broj u nizu. Ukoliko imate niz slova, parametar će biti trenutno slovo u nizu.

Pogledajmo nekoliko primera kako bismo bolje razumeli kako funkcioniše `for...of` petlja.

U ovom prvom primeru, imamo niz brojeva i želimo iterirati kroz svaki broj i ispisati ga na konzolu.

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
}
```

:::

Napravili smo parametar nazvan `num` koji će predstavljati trenutni broj u redu. Za ponavljanje za iteraciju 1, `num` će biti `1`, za ponavljanje za iteraciju 2, `num` će biti `2`, i tako dalje.

U petlji, pišemo trenutnu brojku na konzoli.

Ovo je još jedan primer gde imamo niz slova i želimo da iteriramo kroz svako slovo i ispišemo ga na konzolu.

:::interactive_editor

```js
const str = 'freeCodeCamp';

for (let char of str) {
  console.log(char);
}
```

:::

U ovom primeru, kreirali smo parametar koji se zove `char` koji će predstavljati trenutno slovo u sekvenci.

U svakoj iteraciji petlja će ispisati trenutno slovo u konzolu.

Važno je znati da možete koristiti `let`, ili `const` prilikom deklarisanja parametra u opsegu `for...of`.

Međutim, ako koristiš `const`, osiguraj da se vrednost parametra ne menja unutar petlje. Ako se promeni, dobićeš grešku.

Ovo je primer korišćenja `const` što uzrokuje grešku:

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
  num = num + 1; // This will cause an error
}
```

U ovom primeru, pokušavamo da promenimo vrednost `num` unutar opsega. Pošto smo deklarisali `num` za `const`, dobićemo grešku. Stoga, ako morate da promenite vrednost parametra unutar opsega, koristite `let` umesto toga.

Pogledajmo poslednji primer koji je povezan sa nizom stavki.

:::interactive_editor

```js
const people = [
  { name: 'John', age: 30 },
  { name: 'Jane', age: 25 },
  { name: 'Jim', age: 40 }
];

for (const person of people) {
  console.log(`${person.name} is ${person.age} years old`);
}
```

:::

U ovom primeru, imamo niz stavki nazvan `people`. Svaka stavka ima atribute od `name` i `age`.

Kada iteriramo kroz red, kreira se promenljiva pod nazivom `person` koja će predstavljati trenutni element u redu.

Unutar petlje, ispisujemo poruku na konzoli.

Prvi deo će biti `John is 30 years old`, drugi deo će biti `Jane is 25 years old`, a treći deo će biti `Jim is 40 years old`.

Petlje za `for...of` su veoma važne kada vam je potrebno iterirati preko vrednosti iz nekog iterabilnog objekta, kao što su niz ili sekvenca karaktera. Takođe je lako za čitanje i može učiniti vaš kod kraćim.
