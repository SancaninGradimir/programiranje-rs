# --interactive--

CSS se može primeniti na veb stranicu na tri glavna načina: inline, interni i eksterni.

Svaki put ima svoju upotrebu, prednosti i ograničenja, pa je važno znati kada koristiti svaki za pisanje čistog, efikasnog i održivog koda.

Analizirajmo tri tipa za CSS i kada bi trebalo da ih koristiš.

Inline CSS se piše direktno unutar HTML elementa pomoću atributa `style`. Koristi se za stilizovanje jednog određenog elementa.

Evo primera korišćenja inline CSS-a:

:::interactive_editor

```html
<p style="color: green;">This is an inline-styled paragraph.</p>
```

:::

U ovom primeru koristimo atribut `style` da postavimo boju teksta pasusa na `green`.

Inline CSS se obično koristi za brze, jednokratne stilove ili za prepisivanje drugih stilova na određenom elementu.

Međutim, u većini slučajeva treba ga izbegavati jer može zatrpati HTML i otežati održavanje koda.

U većini slučajeva bolje je koristiti interni ili eksterni CSS kako bi stilovi bili organizovani i lakši za održavanje.

Interni CSS se piše unutar oznaka `style` u delu `head` HTML dokumenta. Koristi se za stilizovanje cele stranice i koristan je kada želiš da stilizuješ samo jedan dokument.

Evo primera internog CSS-a:

:::interactive_editor

```html
<head>
  <style>
    p {
      color: blue;
    }
  </style>
</head>
<body>
  <p>This paragraph is styled using internal CSS.</p>
</body>
```

:::

U gornjem primeru, CSS interni postavlja plavi tekst za sve elemente od `p` na stranici.

CSS je najbolje koristiti kada želite definisati stilove za specifičnu stranicu umesto da ih primenite na više stranica. Važno je za jednostranične sajtove ili kada stilovi ne bi smeli biti korišćeni nigde drugde.

Međutim, postoji određeni nedostatak, kao što je olakšanje ponovne upotrebe na mnogo stranica. Štaviše, CSS unutar linije meša HTML sa CSS, čineći kod teškim za održavanje u velikim projektima.

Eksterni CSS se piše u posebnoj `.css` datoteci i povezuje sa HTML dokumentom pomoću elementa `link` u delu `head`.

Omogućava vam da stilizujete mnoge stranice sa koherentnim stilo i predstavlja preporučeni način u profesionalnom web razvoju.

Ovo je primer za stilizovanje svih elemenata paragrafa:

:::interactive_editor

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <p>This paragraph is styled using external CSS.</p>
</body>
```

```css
p {
  color: red;
}
```

:::

U prethodnoj lekciji naučio si da element `link` ima atribut `rel` koji određuje odnos između trenutnog dokumenta i povezanog resursa, kao što je povezivanje sa stilskim listom ili drugim spoljnim resursom.

Sa druge strane, atribut `href` navodi URL povezanog resursa, odnosno mesto gde se taj resurs nalazi.

Eksterni CSS je najbolji izbor za veće projekte u kojima želiš da održavaš dosledan stil na više stranica.

On podstiče razdvajanje odgovornosti tako što HTML određuje strukturu, a CSS izgled, čime kod postaje lakši za održavanje i proširivanje.

Razumevanje kada treba koristiti svaki tip CSS-a važno je za efikasan i kvalitetan razvoj veb stranica.

U većini slučajeva eksterni CSS treba da bude tvoj podrazumevani izbor, naročito za veće i složenije projekte.
