# --interactive--

Funkcija višegrade je moćan koncept u JavaScript koji može značajno povećati vašu sposobnost pisanja koda i učiniti da je vaš kod lak za modifikaciju i ponovnu upotrebu.

Ukratko rečeno, visoko-redosledna funkcija je funkcija koja prima jednu ili više funkcija kao argumente i vraća funkciju, ili obe stvari.

Da bismo razumeli funkcije višegrade, prvo moramo smatrati funkcije kao građane prvog ranga u JavaScript. To znači da funkcije mogu biti tretirane kao bilo koja druga vrednost – mogu se dodeliti parametrima, proslediti kao argumenti drugim funkcijama i vraćati iz funkcija. Ova jednostavnost omogućava kreiranje i korišćenje funkcija višegrade.

Uobičajena upotreba funkcija višeg nivoa je skrivanje ili uklanjanje složenih operacija.

Na primer, možeš imati funkciju koja izvršava određenu operaciju nad svakim elementom niza. Umesto da pišeš posebnu funkciju za svaku operaciju, možeš napraviti funkciju višeg reda koja prihvata operaciju kao argument. To ti omogućava da koristiš istu strukturu funkcije za različita ponašanja.

Evo primera kako da se pokaže ovaj koncept:

:::interactive_editor

```js
function operateOnArray(arr, operation) {
  let result = [];
  for (let i = 0; i < arr.length; i++) {
    result.push(operation(arr[i]));
}
  return result;
}

function double(x) {
    return x * 2;
}

let numbers = [1, 2, 3, 4, 5];
let doubledNumbers = operateOnArray(numbers, double);
console.log(doubledNumbers); // [2, 4, 6, 8, 10]
```

:::

U ovom primeru, `operateOnArray` je funkcija višeg reda. Ona prima niz i funkciju (`operation`) kao argumente. Zatim primenjuje tu operaciju na svaki element niza. Funkcija `double` prosleđuje se kao argument funkciji `operateOnArray`, pokazujući kako funkcije mogu biti korišćene kao vrednosti.

Komponente višeg ranga takođe mogu da ponovovore komponente. Ovo je posebno važno za kreiranje specifičnih komponenti koje potiču od opštih komponenti. Ovaj koncept se često naziva industrija komponenti. Evo primera:

:::interactive_editor

```js
function multiplyBy(factor) {
  return function(number) {
    return number * factor;
  }
}

let double = multiplyBy(2);
let triple = multiplyBy(3);

console.log(double(5)); // 10
console.log(triple(5)); // 15
```

:::

U ovom slučaju, `multiplyBy` je funkcija višeg reda koja vraća novu funkciju. Ta nova funkcija prilagođena je faktoru prosleđenom funkciji `multiplyBy`. To nam omogućava da lako kreiramo specijalizovane funkcije za množenje.

Funkcije višeg reda nisu samo teorijski koncept – veoma se često koriste u JavaScript-u.

Mnoge ugrađene metode za nizove u JavaScript-u, kao što su `map()`, `filter()` i `reduce()`, predstavljaju funkcije višeg reda. Ove metode primaju funkciju kao argument i primenjuju je na elemente niza na različite načine. O njima ćeš naučiti više u narednim lekcijama.

Korišćenje funkcija višeg nivoa može dovesti do koda koji je opisniji i lakši za razumevanje.

Umesto da opisujete korak po korak kako izvršiti zadatak (imperativno programiranje), funkcije višeg nivoa omogućavaju vam da opišete ono što želite postići (deklarativno programiranje). Ovo može učiniti vaš kod lakšim za čitanje i održavanje.

Kada nastavite da radite sa JavaScript, često ćete koristiti funkcije visokog nivoa. To je važan deo funkcionalnosti u JavaScript i neophodno je za pisanje čistog, efikasnog i lako održivog koda.

Razumevanje i korišćenje naprednih funkcija će vam povećati sposobnost pisanja programskih aplikacija JavaScript sa veštinom i privlačnošću.
