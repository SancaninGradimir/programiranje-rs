# --description--

Funkcija $lcm(a, b)$ predstavlja najmanji zajednički višestrukostvornik brojeva $a$ i $b$.

Neka je $A(n)$ srednja vrednost izraza $lcm(n, i)$ za $1 ≤ i ≤ n$.

Primer: $A(2) = \frac{2 + 2}{2} = 2$ na $A(10) = \frac{10 + 10 + 30 + 20 + 10 + 30 + 70 + 40 + 90 + 10}{10} = 32$.

Dakle $S(n) = \sum A(k)$ za $1 ≤ k ≤ n$.

$S(100) = 122\\,726$.

Pronađi $S(99\\,999\\,999\\,019)\bmod 999\\,999\\,017$.
