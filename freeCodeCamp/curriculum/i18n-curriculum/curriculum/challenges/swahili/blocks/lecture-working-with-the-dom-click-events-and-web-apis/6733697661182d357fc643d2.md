# --interactive--

API od `Canvas` je moćan alat koji vam omogućava da upravljate crtežima unutar vašeg fajla JavaScript. Sve počinje sa funkcijom `canvas` u HTML. Ova funkcija pruža uređivačku površinu za crtež koja koristi metode i svojstva primjera API od `Canvas`.

API za `Canvas` pruža sve što vam je potrebno da kreirate impresivne slike, uključujući oblike, tekst, zanimljive crteže i čak složene igre. Ima interfejse kao što su `HTMLCanvasElement`, `CanvasRenderingContext2D`, `CanvasGradient`, `CanvasPattern`, `TextMetrics` koji pružaju načine i funkcije koje možete koristiti za kreiranje crteža u vašoj datoteci JavaScript.

Pogledajte kako API od `Canvas` radi, kako biste počeli crtati koristeći JavaScript.

Prvo, morate kreirati komponentu za `canvas` u vašoj datoteci od HTML:

```html
<canvas id="my-canvas"></canvas>
```

Funkcija `canvas` predstavljena je interfejsom `HTMLCanvasElement` koji pruža metode i atribute za kontrolisanje. Pored toga, možete koristiti metode i atribute iz drugih interfejsa u API od `Canvas`.

Možete mu dati vaš `canvas` i `width` sa `height` unutar HTML:

```html
<canvas id="my-canvas" width="400" height="400"></canvas>
```

Ili možete koristiti svojstva za `width` i `height` za interfejs `HTMLCanvasElement`:

```js
const canvas = document.getElementById("my-canvas");
canvas.width = 400;
canvas.height = 400;
```

Trenutno, ne možete ništa videti na ekranu. Nakon kreiranja vašeg elementa `canvas`, sledeći korak je dobijanje pristupa kontekstu za crtanje platna korišćenjem metode `getContext()` interfejsa `HTMLCanvasElement`.

Najčešći kontekst je `2d`, što vam omogućava crtanje u dve dimenzije:

```js
const canvas = document.getElementById("my-canvas");
const ctx = canvas.getContext('2d');
```

Ako ispišete vrednost promenljive `ctx` u konzoli, videćete metode i osobine objekta `CanvasRenderingContext2D` koje možete koristiti za kreiranje oblika, boja, linija i još mnogo toga, zajedno sa njihovim podrazumevanim vrednostima:

```js
console.log(ctx);
```

Kada imate 2D kontekst, možete početi da crtate na platnu za crtanje.

API od `Canvas` pruža nekoliko metoda i svojstava za crtanje oblika, linija i teksta. Jedno od njih je svojstvo `fillStyle` koje možete kombinovati sa metodom `fillRect()` za crtanje pravougaonika ili kvadrata:

:::interactive_editor

```html
<html>
  <head>
  </head>
  <body>
    <canvas id="my-canvas" width="400" height="400"></canvas>
    <script src="index.js"></script>
  </body>
</html>
```

```js
const canvas = document.getElementById("my-canvas");

const ctx = canvas.getContext("2d");

// Set the background color
ctx.fillStyle = "crimson";

// Draw a rectangle
ctx.fillRect(1, 1, 150, 100);
```

:::

`fillRect` uzima četiri vrednosti za brojeve koji predstavljaju X-osu, Y-osu, širinu i dužinu, respektivno.

Sada je nešto na ekranu. Takođe možete crtati tekst ili čak kreirati zanimljive grafike. Evo platna za crtež koje predstavlja tekst:

```html
<canvas id="my-text-canvas" width="300" height="70"></canvas>
```

Da biste konačno nacrtali tekst, proslijedite taj tekst putem `fillText()` kao prvi argument, praćen vrijednostima koordinata X i Y:

:::interactive_editor

```html
<html>
  <head>
  </head>
  <body>
    <canvas id="my-text-canvas" width="300" height="70"></canvas>
    <script src="index.js"></script>
  </body>
</html>
```

```js
const textCanvas = document.getElementById("my-text-canvas");

const textCanvasCtx = textCanvas.getContext("2d");

// Set font family and size
textCanvasCtx.font = "30px Arial";

// Set text color
textCanvasCtx.fillStyle = "crimson";

// Draw the text
textCanvasCtx.fillText("Hello HTML Canvas!", 1, 50);
```

:::

Rezultati u pretraživaču će biti crveni tekst `Hello HTML Canvas!`.

Postoji još mnogo toga što možete uraditi sa API od `Canvas`. Na primer, možete ga kombinovati sa `requestAnimationFrame` da kreirate uzbudljive crteže, prikaze, igre i još mnogo toga.
