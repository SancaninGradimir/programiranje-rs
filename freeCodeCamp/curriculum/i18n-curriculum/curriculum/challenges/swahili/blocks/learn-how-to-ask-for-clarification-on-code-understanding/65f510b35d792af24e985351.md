# --description--

U programiranju/kodiranju, reč `persistent` se često odnosi na nešto što je stalno prisutno ili nastaje, posebno problem ili grešku koju je teško rešiti.

`Persistence` je imenica, što znači stanje održivosti.

Na primer, `persistent error` u kodu je greška koja se često ponavlja i teško je da se popravi.
