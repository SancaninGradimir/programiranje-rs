# --explanation--

Ova rečenica koristi `Second Conditional`, koja je kreirana na sledeći način:

`If` + `Past Simple` ili `Past Continuous` kada, `might / would / could` + osnovni glagol. U ovom primeru:

`If they closed` je neophodno (razmišljanje o zatvaranju).

`It would affect` je rezultat (što će se pojaviti kao rezultat).

Ovaj tip uslova se koristi za razgovor o nerealnim ili hipotetičkim situacijama.
