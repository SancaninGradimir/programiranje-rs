# --description--

Acha $x_1, x_2, \ldots, x_n$ iwe mfululizo wa urefu $n$ unaokidhi:

- $x_1 = 2$
- kwa kila $1 &lt; i ≤ n : x_{i - 1} &lt; x_i$
- kwa kila $i$ na $j$ zenye $1 ≤ i, j ≤ n : {(x_i)}^j &lt; {(x_j + 1)}^i$

Postoji samo pet nizova dužine 2, odnosno: {2,4}, {2,5}, {2,6}, {2,7} i {2,8}. Postoje 293 niza dužine 5; tri primera su: {2,5,11,25,55}, {2,6,14,36,88}, {2,8,22,64,181}.

Neka $t(n)$ bude broj takvih nizova dužine $n$. Dobijeno je da je $t(10) = 86195$ i $t(20) = 5227991891$.

Tafuta $t({10}^{10})$ na toa jibu lako modulo $10^9$.
