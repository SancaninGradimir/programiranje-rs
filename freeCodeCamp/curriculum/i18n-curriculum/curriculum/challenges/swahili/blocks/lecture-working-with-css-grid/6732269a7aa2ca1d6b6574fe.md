# --interactive--

Nedefinisana mreža odnosi se na redove i kolone koje pregledač automatski stvara prilikom postavljanja elemenata u mrežni raspored – one koje nisu eksplicitno definisane korišćenjem `grid-template-rows` ili `grid-template-columns`.

Slike koje kontrolišu kolone i redove kreirane direktno u pregledaču su `grid-auto-columns` i `grid-auto-rows`.

Implicitna mreža (implicit grid) takođe obuhvata dodatne redove i kolone koje pregledač automatski kreira kada postavi element izvan eksplicitno definisanih redova i kolona.

Na primer, pretpostavimo da si eksplicitno definisao samo dve kolone u rasporedu mreže na ovaj način:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
  <div class="grid-item">Item 4</div>
  <div class="grid-item">Item 5</div>
  <div class="grid-item">Item 6</div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 100px 100px; /* Only 2 explicit columns */
}

.grid-item {
  background-color: burlywood;
  border: 1px solid orangered;
  padding: 0.5rem;
  margin: 0.5rem;
}
```

:::

Dve stavke su popunile prvi red koristeći dve eksplicitno definisane kolone – `Item 1` u prvoj koloni i `Item 2` u drugoj koloni.

Sledeći elementi započinju novi red, a `Item 3` ide u prvu kolonu drugog reda, i `Item 4` u drugu kolonu drugog reda, itd.

Kao što si već video, eksplicitna mreža (explicit grid) sastoji se od oblasti mreže koje si namerno definisao. To su redovi i kolone koje eksplicitno određuješ pomoću svojstava `grid-template-rows` i `grid-template-columns`.

Evo tabele koja sažima razlike između implicitne i eksplicitne mreže:

| **Kipengele**          | **Grid iliyosemwa wazi**                                                                  | **Grid isiyoelezwa**                                                                          |
|----------------------|------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| **Kontrola veličine** | Potpuno se kontroliše pomoću `grid-template-rows` i `grid-template-columns`. | Kontroliše se pomoću `grid-auto-rows` i `grid-auto-columns`, ili koristi podrazumevanu vrednost `auto`. |
| **Podrazumevano ponašanje** | Ne menja se osim ako nije eksplicitno definisana. | Automatski se proširuje za elemente postavljene izvan eksplicitne mreže. |
| **Složenost** | Zahteva više planiranja rasporeda. | Jednostavnija je za sadržaj koji nije unapred definisan ili se menja. |
| **Fleksibilnost** | Fiksna struktura sa unapred definisanim redovima i kolonama. | Fleksibilna i prilagođava se dinamički dodatom sadržaju. |
| **Performanse** | Može imati bolje performanse jer je raspored unapred definisan. | Može zahtevati dodatne proračune pregledača za kreiranje implicitne mreže. |
| **Upotreba** | Pogodna kada je raspored mreže unapred poznat i definisan. | Pogodna za dinamičke rasporede u kojima sadržaj nije unapred poznat ili se često menja. |
