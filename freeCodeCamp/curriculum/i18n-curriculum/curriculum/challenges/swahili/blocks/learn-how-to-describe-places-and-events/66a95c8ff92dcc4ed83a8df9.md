# --explanation--

`How many` koristi se za pitanje broja nečega. `Usually` pokazuje nešto što se često dešava.

Pitanja u kontekstu `Present Simple` se koriste za postavljanje pitanja o ponašanju, činjenicama ili opštom istinom. Na primer:

- `How many books do you read in a month?` - Ovo je pitanje o prosečnom broju knjiga koje osoba čita.

- `How many people usually attend the meeting?` - Ovo je upit o prosečnom broju učesnika na redovnom sastanku.
