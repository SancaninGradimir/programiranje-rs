# --description--

Broj sreće je definisan sledećim procesom:

Počevši od bilo kog pozitivnog celog broja, zamenite taj broj zbirom kvadrata njegovih cifara, zatim ponavljajte ovaj postupak dok broj ne postane `1` (i ostane takav) ili dok ne uđe u beskonačnu petlju koja ne uključuje `1`. Brojevi za koje se ovaj postupak završava sa `1` nazivaju se srećni brojevi, dok se oni koji se ne završavaju sa `1` nazivaju nesrećni brojevi.

# --instructions--

Implementiraj funkciju koja vraća true ako je broj srećan, ili false ako nije.

```js
function happy (number) {
  let m;
  let digit;
  const cycle = [];

  while (number !== 1 && cycle[number] !== true) {
    cycle[number] = true;
    m = 0;
    while (number > 0) {
      digit = number % 10;
      m += Math.pow(digit, 2);
      number = (number - digit) / 10;
    }
    number = m;
  }
  return (number === 1);
}
```
