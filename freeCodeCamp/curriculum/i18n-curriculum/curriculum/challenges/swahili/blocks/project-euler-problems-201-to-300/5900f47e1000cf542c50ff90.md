# --description--

Istražite jednačinu tipa: $a^2 + b^2 = N$, $0 $\le a \le b$, gde su $a$, $b$ i $N$ celobrojne brojeve.

Za $N = 65$ postoje dva rešenja:

$a = 1, b = 8$ do $a = 4, b = 7$.

Nazivamo $S(N)$ broj vrednosti za $a$ za sve rešenja $a^2 + b^2 = N$, $0 ≤ a ≤ b$, gde su $a$ i $b$ ceo brojevi.

Dakle $S(65) = 1 + 4 = 5$.

Pronađi $\sum S(N)$, za sve kvadratno slobodne $N$ koje se dele samo sa prim brojevima tipa $4k + 1$ koji imaju $4k + 1 < 150$.
