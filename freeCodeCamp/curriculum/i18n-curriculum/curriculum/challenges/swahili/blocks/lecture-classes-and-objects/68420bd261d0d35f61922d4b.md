# --description--

U Python, klase objekata rade zajedno da organizuju i upravljaju podacima. Kreirate klasu kako biste definisali zajedničke karakteristike, a zatim kreirate objekte koji koriste te karakteristike.

Drugim rečima, klasa je kao mapa ili šablon koji koristiš da stvaraš stvari.

Istražimo kakve klase/kurseve postoje, i kako ih koristiti za kreiranje stvari.

Da biste kreirali klasu, koristite ključnu reč `class` praćenu imenom klase i tačkom. Zatim unutar klase, možete dodati atribute/svojstva, zajedno sa svojstvima polja i metodama.

Atribut je kao promenljiva unutar klase i koristi se za skladištenje podataka. Metoda je funkcija definisana unutar klase, i predstavlja akcije koje objekti kreirani iz te klase mogu da izvrše.

Ovde je osnovna sintaksa klase:

```python
class ClassName:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def sample_method(self):               
        print(self.name.upper())
```

* `class ClassName` kreira se pomoću ključne reči `class` za kreiranje klase, praćeno imenom klase, ovde nazvanim `ClassName`. Uobičajeno je u Python koristiti konvenciju **PascalCase** prilikom imenovanja klasa.
    
* `def __init__(self, name, age)` je posebna metoda koja se automatski poziva kada se kreira novi objekat. Inicijalizuje atribute objekata koji će biti kreirani u klasi.
    
    Pored toga, prvi parametar `__init__` je uvek referenca na specifični objekat koji se kreira ili koristi. Konvencionalno, ovaj parametar se zove `self`, ali tehnički, možete koristiti bilo koje ime. `self` vam omogućava pristup atributima i metodama tog objekta.
    
* `self.name = name` i `self.age = age` su atributi koje će objekat imati.
    
* `def sample_method(self):` je metoda koju svaki kreirani objekat može pozvati.
    
* `print(self.name.upper())` je ono što radi metoda `sample_method`; u ovom slučaju, štampa ime velikim slovima.

Ako sve ovo izgleda previše, ne brini. Pogledajmo sličan primer klase `Dog`, i kako možeš kreirati stvari iz nje:

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name.upper()} says woof woof!")
```

Za ovu klasu `Dog`, možete kreirati nešto. Evo osnovne sintakse za kreiranje objekata iz klase:

```python
object_1 = ClassName(attribute_1, attribute_2)
object_2 = ClassName(attribute_1, attribute_2)
```

Takođe možete pozvati bilo koju metodu definisanu unutar klase iz bilo kog mesta:

```python
object_1.method_name()
object_2.method_name()
```

Sada kreirajmo dva psa koristeći klasu `Dog` kao šablon:

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name.upper()} says woof woof! I'm {self.age} years old!")

dog_1 = Dog("Jack", 3)
dog_2 = Dog("Thatcher", 5)

# Call the bark method
dog_1.bark()  # JACK says woof woof! I'm 3 years old!
dog_2.bark()  # THATCHER says woof woof! I'm 5 years old!
```

Kao što vidiš, kreiramo dva objekta psa koristeći klasu `Dog`. Prilikom kreiranja `dog_1`, prosleđuju se niz znakova `Jack` i broj `3`, koji postavljaju osobine `name` i `age` za tu instancu. `dog_2` se inicijalizuje sa nizom znakova `Thatcher` i brojem `5` kao svojim osobinama `name` i `age`.

Zatim, kada pozoveš metodu `.bark()` na `dog_1` i `dog_2`, možeš videti kako se oba rezultata razlikuju koristeći jedinstvene osobine `name` i `age` koje su prosleđene prilikom kreiranja svakog objekta.

Ukratko, razlika između klase i objekta je u tome što je klasa šablon ili mapa, a objekat je ono što se kreira korišćenjem tog šablona.

Pia, klasa definiše podatke i karakteristike koje nešto mora imati, a objekat čuva stvarne podatke i koristi te karakteristike. Pišete klasu jednom, pa možete kreirati mnogo objekata iz nje, svaki sa različitim podacima.
