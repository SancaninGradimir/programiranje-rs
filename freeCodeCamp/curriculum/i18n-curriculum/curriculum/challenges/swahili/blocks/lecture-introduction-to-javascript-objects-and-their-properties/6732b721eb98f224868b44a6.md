# --interactive--

Postoje nekoliko načina za uklanjanje atributa sa objekta, gde je operater `delete` najlakši i najčešće korišćen način.

Koristiš `delete`, briše odabrati atribut sa objekta. Evo primera kako da koristiš operater `delete`:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  job: "Engineer"
};

delete person.job;

console.log(person.job); // undefined
```

:::

U ovom primeru, počinjemo sa objektom `person` koji ima tri svojstva: `name`, `age`, i `job`. Zatim, koristimo operaciju `delete` da obrišemo svojstvo `job`. Nakon brisanja, objekat `person` više nema svojstvo `job`.

Drugi način za uklanjanje karakteristika je korišćenje distribucije rešenja strukture zajedno sa dodatnim parametrima. Ovaj način zapravo ne uklanja karakteristike, već kreira nešto novo koje nema spomenute karakteristike:

:::interactive_editor

```js
const person = {
  name: "Bob",
  age: 25,
  job: "Designer",
  city: "New York"
};

const { job, city, ...remainingProperties } = person;

// { name: "Bob", age: 25 }
console.log(remainingProperties);
```

:::

U ovom primeru, koristimo strukturalno rešenje da izvučemo `job` i `city` iz objekta `person`, i prikupimo preostale atribute u novi objekat nazvan `remainingProperties`. Ovo stvara novi objekat bez atributa `job` i `city`.

Razumeti kako da se uklone atributi sa objekata je ključna veština u programima JavaScript. Omogućava vam da kontrolišete objekte putem promena, i čistite nepotrebne podatke.
