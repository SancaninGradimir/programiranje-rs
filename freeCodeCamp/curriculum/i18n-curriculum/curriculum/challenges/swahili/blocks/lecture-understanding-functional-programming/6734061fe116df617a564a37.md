# --interactive--

Kuringa je metoda u kojoj transformišemo funkciju koja prima više argumenata u niz funkcija, pri čemu svaka funkcija prima samo jedan argument.

Počnimo sa osnovnim primerom. Zamislite da imamo funkciju koja sabira dva broja:

:::interactive_editor

```js
function add(a, b) {
  return a + b;
}

console.log(add(3, 4)); // 7
```

:::

Ovo je funkcija koja prima dva argumenta i vraća njihov zbir. Sada, hajde da istražimo kako možemo implementirati ovu funkciju:

:::interactive_editor

```js
function curriedAdd(a) {
  return function(b) {
    return a + b;
  }
}

console.log(curriedAdd(3)(4)); // 7
```

:::

U ovom kodu koji je modifikovan da radi petlju/iteraciju, umesto da uzima dva argumenta odjednom, imamo funkciju koja prima prvi argument i vraća drugu funkciju. Ova vraćena funkcija zatim uzima drugi argument i izvršava inkrement. Pozivamo ga kao `curriedAdd(3)(4)`, gde par običnih zagrada predstavlja poziv funkcije.

Ali zašto bismo hteli da radimo ovo?

Kuringa nam omogućava da lako kreiramo neke posebne funkcije. Na primer, možemo kreirati funkciju koja svaki put dodaje pet bilo kom broju:

:::interactive_editor

```js
function curriedAdd(a) {
  return function(b) {
    return a + b;
  }
}

const addFive = curriedAdd(5);
console.log(addFive(10)); // 15
console.log(addFive(20)); // 25
```

:::

Hapa, `addFive` je funkcija koja je uvijek spremna da doda pet bilo kojem broju koji joj damo. Ovo je jednostavan primjer upotrebe modula, gdje vezujemo određeni broj argumenata za jednu funkciju i kreiramo drugu funkciju koja uzima malo argumenata.

Iako su naši modeli uzeli u obzir funkcije sa dva argumenta, kuringa se može koristiti za funkcije sa bilo kojim brojem argumenata.

Kada nastavljate sa svojim radom u JavaScript, često ćete otkriti gde refaktoring može učiniti vaš kod čistijim i sa boljom čitljivošću.

## --text--

Koja od sljedećih je primjer funkcije obuhvaćene u JavaScript?

## --answers--

```js
function add(a, b, c) {
  return a + b + c;
}
```

### --feedback--

Ovo je standardna funkcija koja uzima sve parametre istovremeno.

---

```js
function add(a) {
  return function (b) {
    return function (c) {
      return a + b + c;
    };
  };
}
```

---

```js
function add(a, b, c) {
  return a() + b() + c();
}
```

### --feedback--

Ova funkcija prima sve parametre istovremeno i tretira ih kao funkcije, što ne odgovara definiciji obuhvaćene funkcije.

---

```js
function add(a) {
  function inner(b) {
    return a + b;
  }
}
```

### --feedback--

Unutrašnja funkcija se nikada ne vraća, stoga `add(2)` će vratiti `undefined` i `add(2)(3)` će baciti grešku.
