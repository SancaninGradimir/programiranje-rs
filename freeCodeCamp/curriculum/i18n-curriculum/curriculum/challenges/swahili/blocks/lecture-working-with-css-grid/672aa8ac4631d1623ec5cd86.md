# --interactive--

CSS Grid je moćan sistem za organizaciju koji omogućava mrežnim programerima da lako kreiraju složene i prilagodljive postavke uređaju.

Zamislite kada raspoređujete nameštaj u sobi – CSS Mreža je kao da imate nevidljivu mrežu na podu koja vam pomaže da sve postavite tačno na svoje mesto.

Kada pravimo veb-sajtove, često moramo da rasporedimo različite elemente na stranici.

Pre CSS Grid-a, ovo je ponekad predstavljalo izazov, naročito za složene rasporede. CSS Grid pojednostavljuje ovaj proces tako što deli veb stranicu na redove i kolone, stvarajući mrežni raspored.

Zamisli da radiš sa `div` kontejnerom koji sadrži više elemenata raspoređenih ovako:

:::interactive_editor

```html
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

:::

Ako želiš da te elemente rasporediš u mrežu, možeš da postaviš `display` na `grid` i definišeš kolone ovako:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-gap: 20px;
}

.item {
  background-color: lightgray;
  padding: 20px;
  text-align: center;
  border: 1px solid #ccc;
}
```

:::

U ovom kodu govorimo pregledaču da napravi mrežu sa tri kolone jednake širine. To je značenje `1fr 1fr 1fr`, a zatim dodajemo razmak od 20 piksela između svake stavke mreže.

Sada se možda pitaš: „A šta je sa Flexbox-om? Zar se i on ne koristi za rasporede?“

U pravu si! Flexbox je još jedan CSS sistem za raspoređivanje i takođe je veoma koristan. Međutim, postoji važna razlika.

Flexbox je jednodimenzionalan, dok je Grid dvodimenzionalan. To znači da je Flexbox najbolji za raspoređivanje elemenata u jednom redu ili jednoj koloni, dok je Grid namenjen rasporedima koji imaju i redove i kolone.

Flexbox polazi od sadržaja i prilagođava raspored sadržaju. Grid, s druge strane, polazi od rasporeda, omogućavajući ti da prvo napraviš strukturu, a zatim u nju smestiš elemente. Grid ti daje precizniju kontrolu nad položajem elemenata. Možeš tačno odrediti u kom redu i koloni će se određeni element nalaziti.

Evo primera Flexbox-a za poređenje:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="item">Item 1</div>
  <div class="item">Item 2</div>
  <div class="item">Item 3</div>
  <div class="item">Item 4</div>
  <div class="item">Item 5</div>
  <div class="item">Item 6</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
}
```

:::

Ovo stvara prilagodljivi uređaj gde su komponente raspoređene ravnomerno duž glavne ose.

I Grid i Flexbox imaju svoje snage, a često najbolje konfiguracije koriste kombinaciju oba. Možete koristiti Grid za opšti raspored stranice, pa zatim Flexbox za aranžiranje elemenata unutar svake ćelije mreže.

Ukratko, CSS Grid je moćan alat koji omogućava precizno podešavanje u dva pravca. Iako može delovati teško na početku, uz vežbanje, postaje važan alat za kreiranje složenih i prilagodljivih podešavanja u zavisnosti od uređaja.
