# --description--

Albert bira pozitivan ceo broj $k$, zatim dva realna broja $a$, $b$ slučajno se biraju u intervalu [0,1] sa normalnom distribucijom.

Koren iz zbira kvadrata ${(ka + 1)}^2 + {(kb + 1)}^2$ se izračunava, a zatim zaokružuje do najbližeg celog broja. Ako je rezultat jednak $k$, dobija bod $k$; inače ne dobija nijedan bod.

Na primer, ako $k = 6$, $a = 0.2$ i $b = 0.85$, onda ${(ka + 1)}^2 + {(kb + 1)}^2 = 42.05$. Kvadratni koren 42.05 je 6.484... i zaokruženo do najbližeg celog broja, postaje 6. Ovo je jednako $k$, pa dobija ocenu 6.

Može se pokazati da ako igra 10 kola i $k = 1, k = 2, \ldots, k = 10$, očekivana vrednost njegovog ukupnog rezultata, zaokružena na pet decimala, je 10.20914.

Ako igra runde ${10}^5$ i za $k = 1, k = 2, k = 3, \ldots, k = {10}^5$, koja je očekivana vrednost njegovog ukupnog rezultata, zaokružena na pet decimala?
