# --explanation--

`I'd` je skraćenica za `I would`, koja se koristi za razgovor o ličnoj pristranosti. Na primer:

`I'd like to finish the project by the end of the week.` - Prikazuje vaše zahteve ili preferenciju završetka projekta.

`Rather` se koristi za prikaz preferencije između dve opcije. Često je praćen sa `would` u frazama kao što su `I would rather`. Primer:

`I would rather use Python for data analysis than JavaScript.` - Prikazuje vaš izbor između jedne stvari i druge.
