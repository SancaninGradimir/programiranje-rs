# --interactive--

Atribut elementa `aria-controls` se koristi za kreiranje programskog odnosa između elementa koji kontroliše postojanje drugog elementa na stranici i kontrolovisanog elementa. Ovaj odnos može pomoći korisnicima čitača ekrana da bolje razumeju kako kontroler funkcioniše. Uobičajene primere su skup kartica koji kontrolišu vidljivost različitih delova sadržaja, ili dugme koje menja vidljivost menija.

Pogledajmo primer da vidimo kako ovo funkcioniše. U ovom primeru interfejsa sa karticama, imamo element `div` koji ima set od tri dugmeta:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div role="tablist">
  <button role="tab" id="tab1" aria-controls="section1" aria-selected="true">
    Tab 1
  </button>
  <button role="tab" id="tab2" aria-controls="section2" aria-selected="false">
    Tab 2
  </button>
  <button role="tab" id="tab3" aria-controls="section3" aria-selected="false">
    Tab 3
  </button>
</div>
```

```css
[role="tablist"] {
  display: flex;
  border-bottom: 2px solid #ccc;
  margin-bottom: 1em;
  gap: 4px;
}

[role="tab"] {
  background: #f5f5f5;
  border: 1px solid #ccc;
  border-bottom: none;
  border-radius: 4px 4px 0 0;
  padding: 8px 16px;
  cursor: pointer;
  font-size: 14px;
  color: #333;
  transition: background 0.2s, color 0.2s;
}

[role="tab"]:hover,
[role="tab"]:focus {
  background: #e8f0fe;
  outline: none;
}

[role="tab"][aria-selected="true"] {
  background: #fff;
  border-color: #ccc;
  border-bottom: 2px solid #fff;
  color: #000;
  font-weight: 600;
  position: relative;
  top: 1px; 
}

[role="tab"]:focus-visible {
  outline: 2px solid #0078d4;
  outline-offset: 2px;
}

```

:::

`div` koristi `role="tablist"` da pokaže da je kontejnerski element za grupu tabova.

Svaki dugme predstavlja karticu i ima atribut elementa `role` podešen na `tab`. U interfejsima sa više kartica, prvi panel kartice je vidljiv po podrazumevanju, pa dugme prve kartice ima atribut elementa `aria-selected` podešen na `true` da pokaže da je njegov sadržaj trenutno vidljiv. Atribut elementa `aria-controls` koristi se za povezivanje svakog dugmeta sa sadržajnim delom koji kontroliše.

Ovde postoje tri sekcije sadržaja koje odgovaraju karticama:

:::interactive_editor

```html

    Tab 2
  </button>
  <button role="tab" id="tab3" aria-controls="section3" aria-selected="false">
    Tab 3

```

```js
const tabs = document.querySelectorAll('[role="tab"]');
const tabList = document.querySelector('[role="tablist"]');

tabs.forEach(tab => {
  tab.addEventListener('click', () => {
    activateTab(tab);
  });

  tab.addEventListener('keydown', (e) => {
    const key = e.key;
    const currentIndex = Array.from(tabs).indexOf(tab);
    let newIndex = null;

    if (key === 'ArrowRight') {
      newIndex = (currentIndex + 1) % tabs.length;
    } else if (key === 'ArrowLeft') {
      newIndex = (currentIndex - 1 + tabs.length) % tabs.length;
    } else if (key === 'Home') {
      newIndex = 0;
    } else if (key === 'End') {
      newIndex = tabs.length - 1;
    }

    if (newIndex !== null) {
      tabs[newIndex].focus();
      activateTab(tabs[newIndex]);
    }
  });
});

function activateTab(tab) {
  const tabPanels = document.querySelectorAll('[role="tabpanel"]');

  tabs.forEach(t => {
    t.setAttribute('aria-selected', 'false');
    t.setAttribute('tabindex', '-1');
  });

  tabPanels.forEach(panel => panel.hidden = true);

  tab.setAttribute('aria-selected', 'true');
  tab.removeAttribute('tabindex');

  const panelId = tab.getAttribute('aria-controls');
  const panel = document.getElementById(panelId);
  panel.hidden = false;
  tab.focus();
}

```

:::

Svaka sekcija sadržaja ima atribut elementa `role` postavljen na `tabpanel` i atribut elementa `aria-labelledby` koji se odnosi na odgovarajuću karticu radi pružanja dostupnog imena za panel. Atribut elementa `hidden` koristi se za skrivanje sekcija sadržaja koje trenutno nisu odabrane. Svaki ID sekcije odgovara vrednosti atributa elementa `aria-controls` na odgovarajućem dugmetu. ID `section1` odgovara atributu elementa `aria-controls` na prvom dugmetu, `section2` odgovara `aria-controls` na drugom dugmetu, i `section3` odgovara `aria-controls` na trećem dugmetu. Na ovakav način se uspostavlja veza između kartica i njihovih sekcija sadržaja.

Da biste prebacili između različitih sekcija, ćete morati da koristite JavaScript da modifikujete atribut elementa `hidden` na sekciji i atribut elementa `aria-selected` na dugmetu, u zavisnosti od toga koja je sekcija trenutno prikazana. Samo jedna sekcija može biti vidljiva u isto vreme i ne bi trebalo da ima atribut elementa `hidden` i `aria-selected`, već treba da bude postavljena na `true` na svom dugmetu. Sve skrivene sekcije treba da imaju atribut elementa `hidden` i `aria-selected`, te treba da se postave na `false` na svojim dugmetima.

Tabovi su česta primena atributa `aria-controls`, ali on se može koristiti i kada jedan element upravlja prikazom ili ponašanjem drugog, na primer kod harmonika, padajućih menija i modalnih prozora.

Sledeći put kada praviš korisnički interfejs u kojem jedan element upravlja drugim, razmisli o korišćenju atributa `aria-controls` za uspostavljanje te veze.
