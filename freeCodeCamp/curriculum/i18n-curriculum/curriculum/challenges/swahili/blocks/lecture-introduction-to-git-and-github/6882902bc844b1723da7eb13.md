# --description--

Pre nego što počnete stvarno da koristite kontrolu verzija, i sajt poput GitHub, morate imati Git instaliran i spreman za upotrebu.

Da biste proverili da li Git je već instalirano na vašem računaru, možete pokrenuti sledeću komandu u terminalu:

```sh
git --version
```

Ako vidite broj verzije, to znači da je Git instalirano. Inače, moraćete ga instalirati.

Na Linux sistemima, Git često dolazi već instalirano sa mnogih distribucija. Ako nemate Git već instalirano, trebalo bi da ga možete instalirate pomoću komandi menadžera paketa kao što su `sudo apt-get install git` ili `sudo pacman -S git`.

Za korisnike Mac-a, možete instalirati Git putem Homebrewa za `brew install git`, ili možete preuzeti instalacioni paket sa veb-stranice Git.

Na Windows-u, možete preuzeti instalacioni paket sa veb lokacije Git. Ili, ako ste instalirali Chocolatey, možete pokrenuti `choco install git.install` u PowerShell-u. Zapamtite da na Windows-u možda želite da preuzmete GitBash kako biste imali okruženje shella slično Unix-u.

Da potvrdiš da je instalacija uspešna, ponovo pokreni komandu `git --version` u terminalu.

Kada imate `Git` instaliran, moraćete napraviti nekoliko promena u konfiguraciji. Trebalo bi da izvršite ove promene samo jednom na vašem računaru i nećete morati da ih ponavljate svaki put kada ažurirate.

`git config` služi za podešavanje konfiguracionih opcija koje određuju kako Git radi na tvom računaru. Da vidiš trenutna podešavanja i gde su sačuvana, pokreni sledeću komandu:

```sh
git config --list --show-origin
```

Trenutno bi trebalo da vidite samo podešavanja na nivou sistema ako prvi put instalirate Git. Ako ste Git instalirali pre ove lekcije, verovatno ćete videti i korisnička podešavanja za korisničko ime i adresu e-pošte.

Da biste postavili svoje korisničko ime, možete pokrenuti sledeću komandu:

```sh
git config --global user.name "Jane Doe"
```

Flag `--global` koristi se ovde za postavljanje korisničkog imena za sve projekte u vašem sistemu koji koristi Git. Ako trebate promeniti korisničko ime za određeni projekat, onda možete pokrenuti tu komandu u direktorijumu odgovarajućeg projekta bez flag-a `--global`.

Da biste postavili e-mail adresu korisnika, možete pokrenuti sljedeću komandu:

```sh
git config --global user.email janedoe@example.com
```

Konfigurisanje vašeg korisničkog imena i e-pošte je važno jer Git koristiće ove informacije za svaki commit koji napravite na projektu. Vi ćete saznati više o commitima u sledećoj lekciji.

Druga konfiguracija koju možete postaviti je omiljeni uređivač Git, koristite ga. Evo primera kako postaviti omiljeni uređivač na Emacs:

```sh
git config --global core.editor emacs
```

Ostali izbori uključuju Vim, Nano, VS Code i slično. Na primer, ako koristite VS Code kao svoj editor, onda možete konfigurisati podešavanja Git na sledeći način:

```sh
git config --global core.editor "code --wait"
```

Ako ne želiš da postaviš omiljeni editor, onda Git će koristiti podrazumevani editor tvog sistema.

Ako koristite Windows računar i želite da promenite vaš editor, moraćete da navedete pun put do izvršne datoteke na sledeći način:

```sh
git config --global core.editor "'C:/Program Files/Notepad++/notepad++.exe' -multiInst -notabbar -nosession -noPlugin"
```

Da biste videli vašu listu konfiguracije, možete pokrenuti sledeću komandu:

```sh
git config --list
```

Sada ste instalirali i konfigurisali Git, spremni ste da počnete dodavati upravljanje verzijama na vaše projekte.
