# --explanation--

U `by next month`, `by` pokazuje poslednji datum/rok, što znači da se nešto mora uraditi pre ili najkasnije do tog vremena.

Rečenica `We need to finish the report by Friday` koristi `by` na isti način, postavljajući jasan vremenski limit.
