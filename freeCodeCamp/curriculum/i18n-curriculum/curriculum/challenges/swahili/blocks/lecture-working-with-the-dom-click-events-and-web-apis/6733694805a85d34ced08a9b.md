# --interactive--

Kada gradiš svoje programe, često ćeš želeti da nešto se dogodi nakon kratkog kašnjenja ili ponovi u redovnim intervalima. Tu su metode `setTimeout()` i `setInterval()` korisne. Oba metoda primaju dva parametra: funkciju i kašnjenje.

Istražite svaki način detaljnije kako biste počeli da ih koristite u svojim projekatima na mreži.

`setTimeout()` Omogućava vam da odložite akciju na određeno vrijeme. Evo osnovne sintakse za metodu `setTimeout()`:

```js
setTimeout(functionToRun, delay);
```

`functionToRun` je kod koji želi da se izvrši nakon definisanog kašnjenja, i `delay` je vreme (u milisekundama) čekanja pre pokretanja funkcije. Evo primera:

:::interactive_editor

```js
setTimeout(function () {
 console.log("This runs after 3 seconds");
}, 3000);
```

:::

Takođe možete koristiti funkciju sa strelicom za poziv povratka, kao što je ovo:

:::interactive_editor

```js
setTimeout(() => {
 console.log("This runs after 3 seconds"); 
}, 3000);
```

:::

`setInterval()` nastavlja da ponavlja funkciju u zadatom intervalu. To je kao da imate zvono koje zvoni svakih nekoliko sekundi. `setInterval()` je savršeno kada želite da nešto radi periodično, kao što je osvežavanje podataka ili kreiranje neprekidne animacije.

Kao `setTimeout()`, koristi funkciju sa kašnjenjem:

```js
setInterval(functionToRun, delay);
```

`functionToRun` je kod koji želi da se izvršava periodično, dok `delay` je vremenski interval u milisekundama između svakog izvršavanja te funkcije. Evo primera:

:::interactive_editor

```js
setInterval(() => {
 console.log("This runs every 2 seconds");
}, 2000);

// Result:
// This runs every 2 seconds
// This runs every 2 seconds
// This runs every 2 seconds
// ...
```

:::

Pošto `setInterval()` nastavlja sa izvršavanjem funkcije koja je pokrenuta za određeni period, možda želite da ga zaustavite. Da biste to učinili, morate koristiti metodu `clearInterval()`.

Da bi se izvršio taj zadatak, `clearInterval()` uzima identifikator `setInterval()` koji želite da zaustavite. Ovo može biti parametar koji ste postavili za taj period. Jedan način za zaustavljanje perioda je unutar `setTimeout()`, jer će to zaustaviti period nakon određenog vremena:

:::interactive_editor

```js
const intervalID = setInterval(() => {
 console.log("This will stop after 5 seconds");
}, 1000);

setTimeout(() => {
 clearInterval(intervalID);
}, 5000);
```

:::

Takođe možete sprečiti timeout putem `clearTimeout()`:

:::interactive_editor

```js
let timeoutID = setTimeout(() => {
 console.log("This will not run");
}, 5000);

clearTimeout(timeoutID);
```

:::

Sa ovim kodom, `setTimeout()` se nikada neće izvršiti. Zato je bolji način da se ovo uradi preko DOM-a, dodavanjem osluškivača događaja klika na dugme kako bi se sprečio timeout.

:::interactive_editor

```html
<h1>Cancel Timeout Example</h1>
<button id="cancelButton">Cancel Timeout</button>
<script src="index.js"></script>
```

```js
let timeoutID = setTimeout(() => {
  console.log("This will run if not canceled");
}, 5000);

document
.querySelector("#cancelButton")
.addEventListener("click", () => {
  clearTimeout(timeoutID);
  console.log("Timeout canceled!");
});
```

:::
