# --description--

Uzimajući u obzir stavke sa menija i cene, postoji velika razlika između svake stavke.

Koristite postojeći selektor koji cilja sve elemente `p` unutar elemenata sa klasom `item` i postavite gornju i donju marginu na `5px`.

# --hints--

Trebalo bi da postaviš vrednost za `margin-top` na `5px`.

```js
const hasMarginTop = new __helpers.CSSHelp(document).getCSSRules().some(x => x.style['margin-top'] === '5px');
assert(hasMarginTop);
```

Trebalo bi da postaviš vrednost `margin-bottom` na `5px`.

```js
const hasMarginBottom = new __helpers.CSSHelp(document).getCSSRules().some(x => x.style['margin-bottom'] === '5px');
assert(hasMarginBottom);
```

Trebalo bi da koristiš postojeću prečicu `.item p`.

```js
const hasOneSelector = new __helpers.CSSHelp(document).getStyleDeclarations('.item p');
assert(hasOneSelector.length === 1);
```

Vaši elementi ``p``, raspoređeni unutar vaših elemenata ``.item``, bi trebalo da imaju granicu/ivicu ``margin-top`` od ``5px``.

```js
const itemPMarginTop = new __helpers.CSSHelp(document).getStyle('.item p')?.getPropertyValue('margin-top');
assert(itemPMarginTop === '5px');
```

Vaši elementi za `p`, raspoređeni unutar vaših elemenata za `.item`, treba da imaju granicu od `margin-bottom` od `5px`.

```js
const itemPMarginBottom = new __helpers.CSSHelp(document).getStyle('.item p')?.getPropertyValue('margin-bottom');
assert(itemPMarginBottom === '5px');
```

Vaši elementi sa `p` postavljeni unutar vaših elemenata sa `.item` ne bi trebalo da imaju atribut `margin-left` koji je postavljen.

```js
const itemPMarginLeft = new __helpers.CSSHelp(document).getStyle('.item p')?.getPropertyValue('margin-left');
assert(!itemPMarginLeft);
```

Vaši elementi `p` koji su postavljeni unutar vaših elemenata `.item` ne bi trebalo da imaju svojstvo `margin-right` koje je definisano.

```js
const itemPMarginRight = new __helpers.CSSHelp(document).getStyle('.item p')?.getPropertyValue('margin-right');
assert(!itemPMarginRight);
```
