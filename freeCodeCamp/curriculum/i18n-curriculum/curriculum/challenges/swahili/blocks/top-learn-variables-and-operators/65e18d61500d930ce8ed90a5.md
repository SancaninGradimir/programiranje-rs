# --description--

Možete razmotriti parametre kao podatke "kontejnera za skladištenje" u vašem kodu.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/variables-and-operators/top-learn-variables-and-operators.png" alt='the concept of variables in coding as boxes labeled "birthday" that store and provide access to data, specifically a date "16-09-2003"'>

Kriterijum je "imenovano skladište" podataka. Možete koristiti kriterijume za skladištenje resursa, gostiju i drugih podataka.

Da biste kreirali parametar u JavaScript, koristite ključnu reč `let`.

Sledeće kreira (drugim rečima: objavljuje) parametar sa imenom "message":

```js
let message;
```

Sada, možete uneti podatke u njega koristeći operater raspršivanja `=`:

```js
let message;

// store the string 'Hello' in the variable named message
message = 'Hello'; 
```

Radi sažetosti, možete kombinovati deklaraciju parametra i dodelu vrednosti u jednoj liniji:

```js
let message = 'Hello';
```
