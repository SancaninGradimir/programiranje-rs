# --explanation--

`It` koristi se za promenu imenice koja je već pomenuta, pomažući da se izbegne ponavljanje u rečenici. Primer:

- `I bought a new laptop. It works really well.` - Ovde, `it` zamenjuje `laptop` ili kako bi izbegao ponavljanje te reči.

U `The Live Preview isn't showing in Visual Studio Code. Isn't it supposed to support that feature?`, `it` se odnosi na `Visual Studio Code` zato što Tom govori o programu i njegovoj sposobnosti da podrži funkciju.

`It` menja `Visual Studio Code` ili izbegavanje ponovnog pominjanja imena programa.

S obzirom na očekivanu podršku za funkciju, možete utvrditi da ``it`` se odnosi na sam program.
