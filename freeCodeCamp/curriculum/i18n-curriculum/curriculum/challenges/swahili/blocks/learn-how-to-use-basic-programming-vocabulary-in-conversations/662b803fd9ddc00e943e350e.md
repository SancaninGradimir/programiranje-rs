# --description--

`assignment operator` u programu se koristi za dodeljivanje vrednosti promenljivoj. U mnogim programskim jezicima, uključujući Python, `assignment operator` je predstavljen znakom `=`. Ovo instruira računar da sačuva vrednost koja se nalazi na desnoj strani `=` unutar promenljive navedene na levoj strani. Na primer, `number = 10` dodeljuje vrednost `10` promenljivoj `number`.
