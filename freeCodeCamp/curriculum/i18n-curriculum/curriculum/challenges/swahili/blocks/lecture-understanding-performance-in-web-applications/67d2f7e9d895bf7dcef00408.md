# --description--

Svaki projekat je jedinstven, pa treba da pažljivo razmotrite koja od ovih metoda će najbolje pomoći vašoj web aplikaciji.

Prvo, pošto se svi stilovi obrađuju, trebalo bi da uklonite nepotrebne stilove iz vaših tabela stilova za CSS.

Trebalo bi da razdvojite svoje stilove CSS u više fajlova. Na ovaj način, ako fajl ne zavisi od prikazivanja određene veb stranice, neće biti učitan i neće blokirati proces prikaza stranice.

Podrazumevano, pretraživač pretpostavlja da sve stil tabele treba da spreče proces prikazivanja stranice. Međutim, možete reći pretraživaču kada bi stil tabela trebalo da se učitaju koristeći atribut elementa `media` i media query.

U ovom primeru, tabela stilova `print.css` biće primenjena samo kada se dokument štampa, zato što ima atribut `media` sa vrednošću `print`:

```html
<link rel="stylesheet" href="print.css" media="print" />
```

Takođe možete direktno smanjiti veličinu vaših fajlova CSS kako biste smanjili njihovu veličinu tokom procesa izgradnje i kompresovanja ovih fajlova na serveru gde hostujete svoju veb aplikaciju.

Pokušajte da učinite svoje izbore za CSS što je moguće jednostavnije i nemojte birati više elemenata od onih koji su potrebni.

Koristi prirodnu prednost kontinuiteta za CSS gde je to moguće. Neki stilovi će biti nasledjeni.

Dohvatanje resursa za `rel="preload"` je još jedna važna tehnika. Važni resursi treba učitati prvo.

Zapamtite da crteži CSS mogu uticati na performanse.

Promena nekih svojstava za CSS, kao što su dimenzije, raspored i pozicioniranje, uzrokuje proces koji se zove "reflow," gde pregledač ponovo prilagođava poziciju i geometriju određenih elemenata na stranici. Ovo zahteva ponovno crtanje (redrawing), što zahteva veliku računarsku snagu.

Stoga, savetuje se smanjiti broj crteža CSS koliko je moguće ili barem dati korisniku opciju da isključi ili uključi te crteže.

Iako fontovi mogu poboljšati izgled vaše web aplikacije, mogu uticati i na performanse. Neki fontovi imaju velike fajlove, do nekoliko megabajta. Pokušajte da koristite samo dva ili tri fonta i koristite sigurne web fontove u mogućnosti.

Takođe, možete razmotriti da preuzmete najvažnije fontove unapred kako bi bili spremni za brzu upotrebu.

Ovo su neke važne i često korišćene tehnike za poboljšanje CSS tokom razvijanja mreže. Korišćenjem njih, možete unaprediti performanse vaše veb stranice kako biste kreirali glatko korisničko iskustvo.
