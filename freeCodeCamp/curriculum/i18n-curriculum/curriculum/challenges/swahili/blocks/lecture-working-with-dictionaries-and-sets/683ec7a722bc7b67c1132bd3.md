# --description--

U Python, rečnik je interni struktur podataka koji skladišti kolekcije parova ključ-vrednost. Oni funkcionišu na način veoma sličan stvarnom rečniku, gde tražite reč da biste pronašli njeno odgovarajuće značenje.

U Python rečnicima koristiš ključ da pristupiš odgovarajućoj vrednosti. Rečnik treba da koristiš kada želiš da povežeš vrednosti sa jedinstvenim ključevima. Ovo je korisno kada treba brzo da pronađeš vrednost na osnovu ključa ili da predstaviš strukturirane podatke.

Ovo je sintaksa rečenice iz rečnika Python:

```python
dictionary = {
    key1: value1,
    key2: value2
}
```

Prvo, tražimo parametar koji sadrži rječnik. Nije neophodno da staviš rječnik u parametar, ali je veoma uobičajeno to učiniti kako bi ga sačuvao u memoriji i koristio kasnije u kodu.

Zatim postoje vitičaste zagrade, koje se ponekad nazivaju vitičastim zagradama. I unutar vitičastih zagrada, nalaze se parovi ključ-vrednost.

Svakom ključu je pridružena određena vrednost, pa pomoću ključa možeš da pristupiš toj vrednosti. Posle svake vrednosti, osim poslednje, nalazi se zarez koji razdvaja različite parove ključ–vrednost. Ključevi u rečniku moraju biti jedinstveni i moraju biti nepromenljivog tipa podataka. Međutim, vrednosti mogu biti duplirane i mogu biti bilo kog tipa podataka.

Ovde imamo primer rečnika koji čuva informacije o receptu za Margheritu pizzu:

```python
pizza = {
    'name': 'Margherita Pizza',
    'price': 8.9,
    'calories_per_slice': 250,
    'toppings': ['mozzarella', 'basil']
}
```

Rečnik je dodeljen vrednost u parametru `pizza`. Ima četiri para ključ-vrednost: `name`, `price`, `calories_per_slice`, i `toppings`.

Drug način je korišćenje komponente `dict()`, koja gradi rečnik iz sekvence parova ključ-vrednost.

Ovo će biti ista sintaksa za naš primer sa pizzom. Prosleđujemo listu tupola kao argumente komponenti `dict()`. Ovi tupi imaju ključ kao prvi element i vrednost kao drugi element.

```python
pizza = dict([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250), ('toppings', ['mozzarella', 'basil'])])
```

Da bi pristupio vrednosti para ključ–vrednost, možeš da koristiš sledeću sintaksu, poznatu kao notacija sa uglastim zagradama. To je naziv promenljive koja sadrži rečnik, nakon kojeg slede uglaste zagrade sa ključem kojem želiš da pristupiš:

```python
dictionary[key]
```

U našem primeru sa pizzom, ako želite da dobijete vrednost za `name`, ćete napisati naziv parametra, `pizza`, praćeno uglastim zagradama i ključem, `name`, unutar navodnika:

```python
pizza['name']
```

Ovo će biti procenjeno kao:

```python
'Margherita Pizza'
```

Za ispravljanje vrednosti, potrebno je samo dodati distributivni operator, praćen novom vrednošću.

Ako ključ ne postoji u rječniku, kreira će se nova par ključ-vrijednost. U novijim verzijama Python, rječnik čuva redoslijed unosa. Ovo je korisno kada trebate iterirati preko rječnika:

```python
pizza['name'] = 'Margherita'
```

Sada vrednost ključa `name` je `'Margherita'`:

```python
print(pizza['name']) # 'Margherita'
```

Rečnici takođe imaju načine pomoći za obavljanje uobičajenih aktivnosti.

Put `.get()` dobija vrednost povezanu sa ključem. Slično je onom statičkom zapisivanju koje smo koristili, ali prednost je u tome što možete postaviti podrazumevanu vrednost, tako da nećete dobiti grešku ako ključ ne postoji:

```python
dictionary.get(key, default)
```

U ovom primeru, ako ključ `toppings` ne postoji, vraća praznu listu, što je podrazumevana vrednost koju ovde prosleđujemo kao drugi argument. Ali ako `toppings` postoji, vraća tu vrednost:

```python
pizza.get('toppings', []) # ['mozzarella', 'basil']
```

Metode za `.keys()` i `.values()` vraćaju objekat pogleda sa svim ključevima i svim vrednostima u rečnik, respektivno:

```python
pizza.keys()
# dict_keys(['name', 'price', 'calories_per_slice'])

pizza.values()
# dict_values(['Margherita Pizza', 8.9, 250])
```

Prikaz je samo način za pregledavanje sadržaja rječnika bez kreiranja zasebne kopije podataka.

Metoda ``.items()`` vraća objekat pogleda koji sadrži sve parove ključ-vrednost u rečniku, uključujući ključeve i vrednosti:

```python
pizza.items()
# dict_items([('name', 'Margherita Pizza'), ('price', 8.9), ('calories_per_slice', 250)])
```

Metoda `.clear()` uklanja sve parove ključ-vrednost iz rečnika:

```python
pizza.clear()
```

Metoda ``.pop()`` uklanja par ključ-vrednost čiji je ključ definisan kao prvi argument i vraća njegovu vrednost. Ako ključ ne postoji, vraća podrazumevanu vrednost definisanu kao drugi argument. Ako ključ ne postoji i ne prolazi podrazumevanu vrednost, baca se ``KeyError``:

```python
pizza.pop('price', 10)
pizza.pop('total_price') # KeyError
```

U Python 3.7 u najnovijoj verziji, metoda `.popitem()` uklanja poslednji dodati element:

```python
pizza.popitem()
```

Konačno, metoda `.update()` ažurira parove ključ-vrednost i parove ključ-vrednost drugog rečnika. Ako postoje podudarni ključevi, njihove vrednosti se prepisuju.

U ovom primeru, ažuriramo rječnik za `pizza`. Ključ `price` postoji u oba, pa će njegova vrijednost biti promijenjena na `15`.

Ali `total_time` je nov, pa će biti dodato u rječnik `pizza` kao novi par ključ-vrijednost:

```python
pizza.update({ 'price': 15, 'total_time': 25 })
```

Ovo je novi rečnik sa `price`, ažuriran novim `total_time`. Pogledajte kako je cena sada `15` i `total_time` novi par ključ-vrednost:

```python
{
    'name': 'Margherita Pizza', 
    'price': 15, 
    'calories_per_slice': 250, 
    'toppings': ['mozzarella', 'basil'], 
    'total_time': 25
}
```

Ovo su neke od najčešće korišćenih metoda za rečnike, ali postoji mnogo više. Izbor pravog načina može pomoći da se obavezi teških zadataka efikasno.
