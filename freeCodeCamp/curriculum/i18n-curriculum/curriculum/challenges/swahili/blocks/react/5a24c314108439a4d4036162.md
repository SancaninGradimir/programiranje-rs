# --description--

Komponente su osnova za React. Sve u React je komponenta, i ovde ćete naučiti kako da napravite jednu.

Postoje dva načina za kreiranje komponente `React`. Prvi način je korišćenje funkcije `JavaScript`. Opisivanje komponente na ovaj način stvara *komponentu bez stanja funkcije*. Koncept stanja u aplikaciji će biti diskutovan u narednim izazovima. Trenutno, zamislite komponentu bez stanja kao nešto što može da prima i prikazuje podatke, ali ne kontroliše niti prati promene tih podataka. (Razgovaraćemo o drugom načinu kreiranja komponente `React` u sledećem izazovu.)

Da biste kreirali komponentu sa funkcijom, samo napišite funkciju JavaScript koja vraća ili JSX ili `null`. Važno je napomenuti da React zahteva da ime vaše funkcije počinje velikim slovom. Evo primera komponente bez funkcije koja dodaje klasu HTML u JSX:

```jsx
const DemoComponent = function() {
  return (
    <div className='customClass' />
  );
};
```

Nakon izmene, `<div>` će imati CSS klasu `customClass`.

Budući da JSX komponenta predstavlja HTML, možete spojiti nekoliko komponenti kako biste kreirali složeniju stranicu za HTML. Ovo je jedna od glavnih prednosti arhitekture komponenti koju pruža React. Omogućava vam da kreirate svoj UI iz mnogo različitih, razdvojenih komponenti. To olakšava izgradnju i održavanje složenih korisničkih interfejsa.

# --instructions--

Uređivač koda sadrži funkciju pod nazivom `MyComponent`. Dovrši ovu funkciju tako da vrati jedan `div` element koji sadrži tekstualni niz.

**Napomena:** Tekst se smatra potomkom elementa `div`, stoga neće moći da koristi samozatvarajuću oznaku.

# --hints--

`MyComponent` Trebalo bi da ponovo učita JSX.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
    return mockedComponent.length === 1;
  })()
);
```

`MyComponent` treba da vrati `div` element.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
    return mockedComponent.children().type() === 'div';
  })()
);
```

Element `div` treba da sadrži tekstualni niz.

```js
assert(
  (function () {
    const mockedComponent = Enzyme.mount(React.createElement(MyComponent));
    return mockedComponent.find('div').text() !== '';
  })()
);
```
