# --interactive--

Kada radiš sa bojama u CSS, razumevanje RGB modela boja je veoma važno. RGB stoji za Crvena (Red), Zelena (Green) i Plava (Blue) — osnovne boje svetlosti. Ove tri boje se kombinuju različitim intenzitetima kako bi se stvorile mnoge vrste boja.

RGB model boja je aditivni model, što znači da se boje stvaraju kombinovanjem svetlosti različitih intenziteta.

Snaga svake boje može biti od `0` (što znači nema svetlosti) do `255` (što znači puna svetlost). Kombinovanjem različitih nivoa crvene, zelene i plave boje, možete proizvesti bilo koju boju koju vidite na svom ekranu. Na primer:

- Crna je predstavljena sa `rgb(0, 0, 0)`, što znači nulta svetlosna snaga za sve tri boje.
- Bela je predstavljena sa `rgb(255, 255, 255)`, što znači puna jačina svetlosti za sve tri boje.
- Crvena je predstavljena sa `rgb(255, 0, 0)`, sa punom jačinom za crvenu i nultom za zelenu ili plavu.

U CSS-u funkcija `rgb()` omogućava definisanje boje pomoću RGB modela. Ova funkcija prihvata tri vrednosti koje predstavljaju intenzitet crvene, zelene i plave svetlosti. Evo osnovne sintakse:

```css
element {
  color: rgb(red, green, blue);
}
```

Vrednosti za crveno, zeleno i plavo mogu biti od `0` do `255`, gde `0` predstavlja odsustvo te boje, a `255` predstavlja potpunu čistoću. Evo primera kako možete koristiti funkciju `rgb()` u CSS:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p>This is a paragraph.</p>
```

```css
p {
  color: rgb(255, 0, 0);
}
```

:::

Ovaj kod će promeniti boju teksta pasusa u crvenu zato što je vrednost crvene postavljena na `255`, dok su vrednosti zelene i plave postavljene na `0`.

CSS takođe nudi funkciju `rgba()`, koja dodaje četvrtu vrednost — alfa kanal — za kontrolu providnosti boje. Vrednost alfa kanala kreće se od `0` (potpuno providno) do `1` (potpuno neprovidno). Evo primera korišćenja funkcije `rgba()`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div>This is a div.</div>
```

```css
div {
  background-color: rgba(0, 0, 255, 0.5);
}
```

:::

Ovaj kod će postaviti poluprovidnu plavu pozadinu elementu `div` zato što su vrednosti crvene i zelene `0`, vrednost plave `255`, a alfa kanal `0.5` (50% providnosti).

RGB model boja je izuzetno važan za digitalne medije jer odgovara načinu na koji ekrani prikazuju boje. Monitori i ekrani koriste sitne crvene, zelene i plave piksele za stvaranje boja koje vidiš. Kontrolisanjem intenziteta tih piksela pomoću RGB vrednosti možeš da dobiješ veliki broj različitih boja za svoj veb dizajn.

Pored toga, RGB model je odličan kada radiš sa dizajnima koji zahtevaju promene u realnom vremenu. Na primer, ako želiš da napraviš animacije ili efekte boja koji se dinamički menjaju, fleksibilnost funkcije `rgb()` omogućava ti da programski menjaš boje.

Razumevanjem RGB modela boja i načina korišćenja funkcija `rgb()` i `rgba()` u CSS-u imaćeš veću kontrolu nad načinom na koji se boje prikazuju na tvojim veb stranicama. Moći ćeš da kreiraš živopisne dizajne, koristiš providnost i čak programski menjaš boje radi dinamičkih efekata.
