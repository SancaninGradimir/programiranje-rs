# --interactive--

CSS kombinatori koriste se za definisanje odnosa između selektora u CSS-u. Oni omogućavaju izbor elemenata na osnovu njihovog odnosa sa drugim elementima, što omogućava preciznije i efikasnije stilizovanje.

Razradićemo neke glavne mešavine CSS i njihove primene, počevši sa mešavinom semena.

Kombinacija potomaka koristi se za odabir elemenata koji odgovaraju drugom selektoru ako su postavljeni unutar roditeljskog elementa koji odgovara prvom selektoru. Roditelj može biti roditeljski element ili preporoditeljski element.

Da bismo bolje razumeli kako ovo funkcioniše, pogledajmo primer.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<figure>
  <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Relaxing Cat">
  <figcaption>A relaxing cat with a border</figcaption>
</figure>
```

```css
figure img {
  border: 4px solid red;
}
```

:::

U prethodnom primeru koristimo kombinator potomka da izaberemo sve elemente `img` unutar elemenata `figure` i postavimo punu crvenu ivicu sa sve četiri strane.

Obrati pažnju da se između selektora roditelja i potomka koristi razmak.

Katika kesi hii, `figure` itakuwa mzazi na `img` itakuwa mtoto.

Ako bi imao više slika unutar elementa `figure`, korišćenje kombinatora potomka bio bi dobar način da svakoj slici dodaš punu tamnu ivicu.

Druga vrsta mešavine je dečija mešavina.

Kombinator direktnog deteta (`>`) u CSS-u koristi se za izbor elemenata koji su direktna deca navedenog roditeljskog elementa.

Ovaj kombinator cilja samo elemente koji imaju određenog roditelja, čineći CSS pravila preciznijim i sprečavajući neželjeno stilizovanje dublje ugnježdenih elemenata.

Uzmi primer od HTML sledeći:

:::interactive_editor

```html
<div class="container">
  <p>First</p>
  <div>
    <p>Second</p>
  </div>
  <div>
    <p>Third</p>
  </div>
</div>
```

:::

U strukturi HTML gore, klasa `container` je postavljena na elementu/atributu `div`.

U ovom kontejneru, postoji direktni element `p` ("Prvi"), praćen dva dodatna elementa `div`, svaki sa elementom `p` ("Drugi" i "Treći").

Prvi element `p` je direktno dete elementa `.container`, dok su druga dva elementa `p` ugnježdena unutar drugih `div` elemenata i predstavljaju dublje potomke.

Da bi primenio stilove na direktno dete klase `container`, možeš koristiti kombinator direktnog deteta na sledeći način:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <p>First</p>
  <div>
    <p>Second</p>
  </div>
  <div>
    <p>Third</p>
  </div>
</div>
```

```css
.container > p {
  color: blue;
}
```

:::

U prethodnom primeru ciljamo samo direktno dete klase `container`. Ono će dobiti plavu boju teksta.

Pošto su druga dva elementa pasusa ugnježdena unutar `div` elemenata, oni nisu direktna deca klase `container` i zato neće dobiti plavu boju teksta.

Još jedan često korišćen kombinator je kombinator susednog elementa.

Kombinator susednog elementa (`+`) u CSS-u bira element koji se nalazi odmah iza navedenog susednog elementa. Koristan je kada želiš da primeniš stil na element koji neposredno sledi drugi element.

Uzmi primer od HTML sledeći:

:::interactive_editor

```html
<figure>
  <img
    src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg"
    alt="A cute orange cat lying on its back."
  />
  <figcaption>A cute orange cat lying on its back.</figcaption>
</figure>
```

:::

Ovde imamo element `figure` koji sadrži element `img`, a odmah iza njega nalazi se element `figcaption`. Element `figcaption` je neposredni sused elementa `img`.

Ako želiš da dodaš crnu ivicu oko elementa `figcaption`, možeš koristiti kombinator susednog elementa na sledeći način:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<figure>
  <img
    src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg"
    alt="A cute orange cat lying on its back."
  />
  <figcaption>A cute orange cat lying on its back.</figcaption>
</figure>
```

```css
img + figcaption {
  border: 4px solid black;
}
```

:::

U ovom primeru kombinator susednog elementa (`+`) bira element `figcaption` koji se nalazi odmah iza elementa `img`. Primenjeno CSS pravilo dodaje ivicu `4px solid black` oko elementa `figcaption`.

Mchanganyiko mwingine wa kawaida ni mchanganyiko wa ndugu wa baadaye.

Kombinacija sledećih braće (`~`) u CSS bira sve braće navedenog elementa koje ga slede.

Za razliku od prethodnog brava mixa, koji cilja samo neposredno sledećeg brata/sestre, kombinacija naknadnih brava (`~`) može da cilja bilo kog brata/sestre koji sledi navedeni element, čime pruža veću fleksibilnost u dizajnu.

Uzmi primer od HTML sledeći:

:::interactive_editor

```html
<div class="container">
  <h2>Subheading</h2>
  <p>First paragraph.</p>
  <p>Second paragraph.</p>
  <p>Third paragraph.</p>
  <p>Another paragraph element</p>
</div>
```

:::

U ovoj strukturi od HTML, imamo element `h2` praćen sa četiri paragrafa elementa. Paragrafski elementi su braća/sestre elemenata `h2`.

Ako želite stilizovati sve elemente pasusa koji sledeći element od `h2`, možete koristiti kombinaciju budućih braće kao što sledi:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <h2>Subheading</h2>
  <p>First paragraph.</p>
  <p>Second paragraph.</p>
  <p>Third paragraph.</p>
  <p>Another paragraph element</p>
</div>
```

```css
h2 ~ p {
  color: green;
}
```

:::

U ovom primeru, svi elementi paragrafa koji prate element `h2` će imati zelenu boju teksta.

Kombinacija sledećih braće (`~`) cilja na sve braće paragrafa koji se pojavljuju nakon elementa `h2`, bez obzira da li su direktni braća ili ne.

Zaključno, razumevanje i korišćenje različitih kombinacija CSS omogućava vam da koristite odgovarajuće stilove za vaše HTML elemente, povećavajući kontrolu i slobodu vašeg dizajna.

Sa ovim selektorima, možete kreirati složenije i ciljane pravila stila koja poboljšavaju izgled i performanse vaših web stranica.
