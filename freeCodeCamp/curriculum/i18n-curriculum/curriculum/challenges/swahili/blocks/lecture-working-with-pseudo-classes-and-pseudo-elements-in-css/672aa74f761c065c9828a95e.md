# --interactive--

Pseudo-class je specifična ključna reč za CSS koja omogućava odabir elementa na osnovu njegovog specifičnog stanja ili položaja. Stanje ili položaj elementa može uključivati:

- Linapokuwa hai (active).
- Linapobofyozwa na panya.
- Linapokuwa mtoto wa kwanza wa mzazi.
- Linapokuwa mtoto wa mwisho wa mzazi.
- Linapokuwa kiungo kilichotembelewa.
- Linapokuwa limezimwa.

Da biste koristili pseudo-class, dodajete ga u selektor koristeći dve tačke (`:`), praćene nazivom pseudo-class:

```css
selector:pseudo-class {
  /* CSS properties */
}
```

Istražite kako možete koristiti pseudo-class da predstavite svaku od stanja i lokacija koje smo pomenuli.

Pseudo-class od `:active` vam omogućava da izaberete stanje funkcionalne komponente, kao što je klik na dugme:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button>Example Button</button>
```

```css
button:active {
  background: greenyellow;
}
```

:::

Pseudo-class za `:hover` opisuje stanje prelaska mišem elementa. Primer je kada kliknete na dugme ili link:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="#">Hover over me!</a>  
```

```css
a:hover {
  text-decoration: none;
  color: white;
  background: crimson;
}
```

:::

Pseudo-class od `:first-child` bira element koji je prvi potomak svog roditelja. Evo primera ciljanja prvog pasusa unutar kontejnera `div`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:first-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

Prvi pasus unutar elementa `div` dobiće pozadinsku boju `lightcoral` i `padding` od `0.4rem` sa svih strana.

Pseudo-class za `:last-child` cilja na poslednji element koji je dete svog roditelja. Evo primera ciljanja poslednjeg pasusa unutar kontejnera `div`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <p>first child</p>
  <p>second child</p>
  <p>third child</p>
  <p>last child</p>
</div>
```

```css
.container p:last-child {
  background: lightcoral;
  padding: 0.4rem;
}
```

:::

Pseudo-class ya `:visited` omogućava vam da uredite link koji je korisnik već posetio:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<a href="https://www.example.com" target="_blank">Visit Example.com</a>
```

```css
a:visited {
  color: purple;
}
```

:::

Pseudo-class ya `:disabled` inakuwezesha kupangilia kipengele kinachoshirikiana kilichozimwa:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button disabled>Disabled Button</button>
```

```css
button:disabled {
  background-color: lightgray;  
}
```

:::

Kao što vidite, pseudo-class vam omogućavaju da organizujete elemente na osnovu interakcije korisnika, kao što je *hover* ili poseta linku. Ovo čini vaš sajt vizuelno interaktivnijim.

Osim pseudo-class spomenutih, postoje i drugi slični:

- `:focus`
- `:first-of-type`
- `:last-of-type`
- `:nth-of-type`
- `:modal`
- `:enabled`
- `:checked`
- `:required`, više.
