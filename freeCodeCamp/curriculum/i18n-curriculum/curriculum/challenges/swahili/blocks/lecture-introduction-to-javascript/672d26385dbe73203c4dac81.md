# --interactive--

JavaScript je programski jezik koji omogućava interakciju i upravljanje promenama na veb stranici.

Dok su HTML i CSS jezičke oznake koje se koriste za kreiranje strukture sadržaja i dekorisanje elemenata na stranici, JavaScript ide dalje od njih omogućavanjem naprednijih funkcija, kao što je obrada unosa korisnika, animacija elemenata, pa čak i izgradnja kompletnih veb aplikacija.

Na primer, kada kliknete na dugme, podnošenje forme ili prelazak prstom nad menijem, JavaScript određuje kako stranica funkcioniše.

Evo primera kako ova tri funkcionišu zajedno:

:::interactive_editor

```html
<!DOCTYPE html>
<html>
<head>
  <style>
    h1 {
      color: green;
    }
  </style>
</head>
<body>
  <h1>Hello, World!</h1>
  <button onclick="alert('Button clicked!')">Click me</button>
</body>
</html>
```

:::

U ovom primeru, HTML se koristi za definisanje sadržaja: naslov sekcije (element sa `h1`) i dugme (element sa `button`).

CSS se koristi za stilizovanje naslova, na primer za postavljanje zelene boje teksta. JavaScript se koristi za prikazivanje upozorenja kada korisnik klikne na dugme.

Ukratko, HTML pruža strukturu, CSS dodaje stilove, a JavaScript omogućava funkcionalnost, stvarajući kompletnu upotrebu sa mrežnom interakcijom.
