# --interactive--

**NAPOMENA**: Neki primeri interakcije mogu koristiti CSS koji još niste naučili. Ne brinite zbog razumevanja celog koda. Svrha primera je da vam pokaže pregled ovih dizajnerskih koncepata kako biste bolje razumeli kako stvari funkcionišu. Da biste videli pregled, moraćete da omogućite interaktivni editor.

Kada je u pitanju dizajniranje veb-sajta, postoji mnogo vrsta pomoćnih elemenata za navigaciju koje možete koristiti. Primjeri uključuju gornje trake za navigaciju, bočne trake i krušne krumke (breadcrumbs). Međutim, ako je vaš sajt složeniji i ima duboke nivoe navigacije, možda ćete želeti da razmislite o korišćenju krušnih krumki.

Breadcrumbs su navigacioni alat koji korisniku pokazuje gde se nalazi u kontekstu strukture sajta. Evo primera kako izgledaju breadcrumbs na primeru veb-sajta za elektronsku opremu:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  Home / Electronics / Phones / Smartphone XYZ
</div>
```

```css
.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

Na mnogim sajtovima, krušne krumpeže (breadcrumbs) se prikazuju na vrhu stranice, pokazujući korisniku put kojim je stigao do trenutne stranice. Krećući se od `Homepage`, korisnik je navigirao do kategorije `Electronics`, zatim do kategorije `Phones`, i na kraju do proizvoda `Smartphone XYZ`. Verovatno ste naišli na 'breadcrumb' trake na nekom sajtu kada ste tražili određeni proizvod ili deo informacija.

Breadcrumb navigacija je korisna jer pomaže korisnicima da razumeju gde se nalaze u strukturi veb sajta i kako da se vrate na prethodne stranice. Ovo je posebno važno kada korisnik dođe sa rezultata pretrage ili spoljne veze i treba da razume kontekst stranice na kojoj se nalazi.

Kada je reč o dizajniranju breadcrumb navigacije, treba uzeti u obzir nekoliko stvari. Prva je izbor razdvajača. Razdvajač je simbol koji odvaja različite nivoe strukture. Najčešće se koriste znak veće od (`>`), dvostruki ugaoni navodnici (`»`) i kosa crta (`/`).

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ
</div>
```

```css
.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

Druga stvar je gde postaviti breadcrumbs. Breadcrumbs se obično nalaze na vrhu stranice, iznad ili ispod glavne navigacione trake. Korisnici ne bi trebalo da imaju poteškoća da ih pronađu, zato se pobrinite da budu vidljivi i lako dostupni.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<nav class="main-nav">
  <a href="#">Home</a>
  <a href="#">Electronics</a>
  <a href="#">Phones</a>
</nav>
<div class="breadcrumbs">
  Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ
</div>
```

```css
.main-nav {
  background-color: #333;
  padding: 1rem;
}

.main-nav a {
  color: white;
  margin-right: 1rem;
  text-decoration: none;
}

.breadcrumbs {
  font-size: 0.9rem;
  color: #555;
  margin: 1rem 0;
}
```

:::

Drugostvar je veličina "breadcrumbs". Želite da osigurate da su oni dovoljno veliki da se lako čitaju, ali da nisu preveliki pa zauzimaju previše prostora na stranici. Zapamtite, "breadcrumbs" nisu namenjeni za korišćenje kao glavni navigacioni alat, već kao sekundarna navigaciona pomoć. Na sajtovima koji imaju mnogo informacija na stranici, korisnici mogu lako da vide gde se nalaze u strukturi i da se vrate na prethodne stranice korišćenjem "breadcrumbs".

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="breadcrumbs">
  Home &gt; Electronics &gt; Phones &gt; Smartphone XYZ
</div>
<p>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.
</p>
```

```css
.breadcrumbs {
  font-size: 1rem;
  color: #555;
  margin: 1rem 0;
}   

p {
  font-size: 1.2rem;
  line-height: 1.6;
}
```

:::

Kada treba da koristiš breadcrumb navigaciju na svom veb sajtu? Ako je tvoj sajt jednostavan i ima plitku hijerarhijsku strukturu, breadcrumb navigacija verovatno nije potrebna. Plitka struktura znači da sajt ima malo nivoa navigacije, pa breadcrumb navigacija ne donosi veliku korist. Najčešće ćeš je videti na internet prodavnicama, sajtovima sa vestima i drugim sajtovima sa dubokom hijerarhijskom strukturom, poput sajtova sa tehničkom dokumentacijom.
