# --description--

Hiper-eksponencijacija ili tetracija broja $a$ sa pozitivnim celim brojem $b$, prikazana kao $a↑↑b$ ili ${}^ba$, definisana je serijom na sledeći način:

$a↑↑1 = a$,

$a↑↑(k+1) = a^{(a↑↑k)}$.

Dakle imamo primer $3↑↑2 = 3^3 = 27$, i stoga $3↑↑3 = 3^{27} = 7625597484987$ i $3↑↑4$ je približno ${10}^{3.6383346400240996 \times {10}^{12}}$. Pronađite poslednjih 8 cifara od $1777↑↑1855$.

# --hints--

`hyperexponentation()` treba da vrati `95962097`.

```js
assert.strictEqual(hyperexponentation(), 95962097);
```
