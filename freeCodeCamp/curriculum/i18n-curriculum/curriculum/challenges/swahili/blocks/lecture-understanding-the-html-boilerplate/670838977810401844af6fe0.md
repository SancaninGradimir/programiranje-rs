# --description--

Naučite se o funkciji `link` i kako je koristiti za povezivanje sa eksternim stilovima.

Element `link` se koristi za povezivanje sa eksternim resursima kao što su stilovi i oznake sajta. Evo osnovne sintakse korišćenja elementa `link` u eksternoj datoteci CSS:

```html
<link rel="stylesheet" href="./styles.css" />
```

Atribut `rel` koristi se za određivanje odnosa između povezanog resursa i dokumenta HTML. U ovom slučaju, moramo da odredimo da je ovaj povezani resurs `stylesheet`.

Smatra se da je najbolji način razdvajanje vaših HTML i CSS u različite fajlove. Programeri će koristiti karakteristiku `link` za svoj eksterni fajl CSS umesto da sve pišu unutar dokumenta HTML.

Atribut od `href` se koristi za određivanje lokacije/položaja URL za eksterni resurs.

`dot` praćen sa naprednim kosim crtama u ovom primeru govori računaru da traži u trenutnom direktorijumu, ili fascikli, fajl `styles.css`.

Element ``link`` treba da bude postavljen unutar elementa ``head``, kao što je prikazano u sledećem primeru:

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Examples of the link element</title>
  <link rel="stylesheet" href="./styles.css" />
</head>
```

Često ćete videti mnogo elemenata `link` unutar profesionalnog koda koji se povezuju sa različitim stilovima, fontovima i simbolima. Evo primera korišćenja elementa `link` za povezivanje sa eksternim Google fontom pod nazivom *Playwright Cuba*:

```html
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link
  href="https://fonts.googleapis.com/css2?family=Playwrite+CU:wght@100..400&display=swap"
  rel="stylesheet"
/>
```

Google Fonts je set besplatnih i slobodno dostupnih fontova koje možete koristiti u bilo kojem projektu. Možete odabrati fontove koje želite da koristite, a Google će vam dati potreban kod HTML na CSS. U ovom primeru, vrednost `preconnect` za atribut `rel` govori pretraživaču da uspostavi rano povezivanje sa vrednošću definisanom u atributu `href`. Ovo se radi kako bi se ubrzilo učitavanje ovih eksternih resursa.

Još jedna uobičajena upotreba elementa `link` je povezivanje sa ikonom/simbolom. Evo primera kako se povezati sa favicon-om:

```html
<link rel="icon" href="favicon.ico" />
```

Favicon, koji je skraćenica od 'favorite icon', je mala ikona koja se obično nalazi na kartici pretraživača pored naslova sajta. Mnogi sajtovi koriste favicon da prikažu svoj brending/znak.
