# --description--

Neno `so` je veznik koji se koristi za pokazivanje uzroka i posledica ili objašnjavanje razloga nečega. U tehnologiji i programiranju, `so` često povezuje problem ili stanje sa rešenjem ili rezultatom.

# --questions--

## --text--

Izaberite rečenicu u kojoj je `so` pravilno korišćeno za povezivanje uzroka i posledica u kontekstu tehnologije.

## --answers--

`The program crashed, so we must reboot the computer.`

---

`We wrote the code, so but it didn't compile.`

### --feedback--

Reč `so` korišćena je pogrešno ovde. `But` ne bi trebalo da se koristi nakon `so`. Trebalo bi da bude `The code didn't compile, so we rewrote it`, ili `We wrote the code, but it didn't compile.`

---

`The server is online, so therefore we can't connect.`

### --feedback--

Korišćenje `so` i `therefore` je dodatno. Treba biti `The server is online, so we can connect` ili `The server is online; therefore, we can connect.`

---

`We need more data, so we can ignore the results.`

### --feedback--

Ova rečenica ne uspostavlja logičku vezu između uzroka i posledice. Najbolja upotreba od `so` bi bila, `We need more data, so we will perform more tests.`
