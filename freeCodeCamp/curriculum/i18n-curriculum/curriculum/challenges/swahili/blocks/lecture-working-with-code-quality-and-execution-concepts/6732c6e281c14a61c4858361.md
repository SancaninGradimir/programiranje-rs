# --interactive--

Scope je jedan od najmoćnijih aspekata i često se pogrešno razume u JavaScript. U svojoj suštini, scope je funkcija koja ima pristup promenljivama u svom spoljašnjem gramatičkom opsegu, čak i nakon što vanjska funkcija vrati. Ovo može delovati komplikovano, ali je fundamentalni koncept koji omogućava mnogim naprednim sistemima programiranja u JavaScript.

Da biste razumeli instalaciju, počnimo sa primerom:

:::interactive_editor

```js
function outerFunction(x) {
    let y = 10;
    function innerFunction(){
        console.log(x + y);
    }
    return innerFunction;
}

let closure = outerFunction(5);
console.log(closure()); // 15
```

:::

U ovom primeru, `outerFunction` prima parametar `x` i kreira interni parametar `y`. Zatim kreira funkciju `innerFunction` koja koristi `x` i `y`. Na kraju vraća `innerFunction`. Kada pozovemo `outerFunction(5)`, vraćamo `innerFunction` čiju vrednost dodeljujemo parametru `closure`. Kasnije kada pozovemo `closure()`, i dalje ima pristup `x` i `y` iz `outerFunction`, iako je `outerFunction` već završio izvršenje. Ovo je srž paketa.

Unutrašnja funkcija sačuvava referencu na svoje spoljašnje gramatičko okruženje, zadržavajući dostupnost parametara u tom okruženju čak i nakon što spoljašnja funkcija završi.

Instalacija je veoma važna za kreiranje privatnih parametara i funkcija. Pogledajte ovaj primer:

:::interactive_editor

```js
function createCounter() {
    let count = 0;
    return function () {
        count++;
        return count;
    };
}

let counter = createCounter();
console.log(counter()); // 1
console.log(counter()); // 2
```

:::

U ovom slučaju, `createCounter` vraća funkciju koja dodaje i vraća parametar `count`. Parametar `count` nije dostupan direktno izvan `createCounter`, ali funkcija koja se vraća (naša implementacija) ima mu pristup. Svaki put kada pozovemo `counter()`, dodaje i vraća `count`.

Paket takođe može da rukuje sa više parametara iz svog eksternog opsega. Na primer:

:::interactive_editor

```js
function multiply(x) {
    return function (y) {
        return x * y;
    };
}

let double = multiply(2);
console.log(double(5)); // 10
```

:::

Ovde unutrašnja funkcija hvata parametar `x` iz `multiply`. Kada kreiramo `double` pozivanjem `multiply(2)`, vraća funkciju koja povećava svoj argument za `2` svaki put.

Važno je znati o vezivanju da ono zadržava parametre po referenci, a ne po vrednosti. To znači da ako se vrednost vezanog parametra promeni, vezivanje će videti novu vrednost. Na primer:

:::interactive_editor

```js
function createIncrementer() {
    let count = 0;
    return function () {
        count++;
        console.log(count);
    };
}

let increment = createIncrementer();
increment(); // 1
increment(); // 2
```

:::

Svaki put kada pozovemo `increment` koji radi sa jednim parametrom `count`, nije kopija njegove prvobitne vrednosti. Formatiranje je moćan alat u JavaScript. Kada nastavite da radite sa JavaScript, otkrićete da razumevanje i efikasna upotreba formatiranja može značajno povećati vašu sposobnost pisanja čistog, efikasanog i moćnog koda.
