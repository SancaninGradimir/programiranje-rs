# --interactive--

U prošloj lekciji, naučili ste kako da počnete sa radom sa linijama različitih boja za prave i kružne. U ovoj lekciji, detaljnije ćemo ući kako da radimo sa parametrima CSS za kreiranje privlačnijih i složenijih dizajna.

Gradijenti boja u CSS omogućavaju vam da kreirate gladak prelaz između dve ili više definisanih boja. Često se koriste za dodavanje dubine, teksture i vizuelne privlačnosti u web dizajne bez potrebe za fajlovima slika.

CSS podržava dve glavne vrste gradijenata: linearne i radijalne gradijente.

Linearni gradijenti stvaraju postepeni prelaz između boja duž prave linije. Možeš da kontrolišeš smer gradijenta i boje koje se koriste. Evo osnovnog primera linearnog gradijenta:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="linear-gradient"></div>
```

```css
.linear-gradient {
  background: linear-gradient(to right, red, blue);
  height: 40vh;
}
```

:::

Ovo kreira gradijent koji prelazi iz crvene boje sa leve strane u plavu sa desne strane. Parametar `to right` određuje smer gradijenta. Možeš koristiti i druge smerove kao što su `to left`, `to top`, `to bottom` ili navesti ugao, na primer `45deg`.

Takođe možeš da koristiš više tačaka prelaza boja u linearnom gradijentu:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="multi-color-gradient"></div>
```

```css
.multi-color-gradient {
  background: linear-gradient(to right, red, yellow, blue);
  height: 40vh;
}
```

:::

Ovo stvara gradijent boje koji se menja od crvene do žute do plave, sa jednakim rasporedom u komponenti.

Radijalni gradijenti, s druge strane, stvaraju kružne ili elipsaste prelaze boja koji počinju iz centralne tačke. Evo primera:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="radial-gradient"></div>
```

```css
.radial-gradient {
  background: radial-gradient(circle, red, blue);
  height: 40vh;
}
```

:::

Ovo kreira radijalni gradijent koji počinje crvenom bojom u centru i prelazi u plavu na ivicama. Takođe možeš da odrediš oblik i veličinu radijalnog gradijenta:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="custom-radial-gradient"></div>
```

```css
.custom-radial-gradient {
  background: radial-gradient(ellipse at top left, red, blue);
  height: 40vh;
}
```

:::

Ovo kreira elipsasti radijalni gradijent koji počinje iz gornjeg levog ugla elementa. I linearni i radijalni gradijenti omogućavaju da precizno odrediš položaje tačaka prelaza boja. Na primer:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="precise-gradient"></div>
```

```css
.precise-gradient {
  background: linear-gradient(to right, red 20%, yellow 40%, blue 80%);
  height: 40vh;
}
```

:::

Ovo stvara gradijent boje gde crvena uzima prvu `20%`, zatim se menja u žuto na `40%`, i na kraju postaje plavo na `80%`.

Gradijentne linije boja mogu se koristiti bilo gde gde koristite sliku u CSS. Ovo olakšava kreiranje pozadina, dugmadi ili čak tekstualnih efekata kada se kombinuje sa drugim parametrima CSS.

Zapamtite, iako kontrasne linije mogu povećati vizuelnu privlačnost vašeg sajta, važno je osigurati dovoljan kontrast za pristupačnost (Accessibility), posebno kada koristite kontraste boja iza teksta.

## --text--

Šta je razlika između linearnih i kružnih linija različite boje u CSS?

## --answers--

Linije različitih boja za liniju koriste samo dve boje, dok linije različitih boja za krugove koriste mnogo boja.

---
