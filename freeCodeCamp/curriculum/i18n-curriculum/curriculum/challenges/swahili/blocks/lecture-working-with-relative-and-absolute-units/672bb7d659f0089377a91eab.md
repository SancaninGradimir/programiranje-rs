# --interactive--

Procenti u CSS su povezane jedinice koje vam omogućavaju da definišete veličinu, dimenzije i druge karakteristike kao deo merenja njegovog roditeljskog kontejnera. Kada koristite procenatnu vrednost, zapravo mislite na „Neka ovo bude X% njegovog kontejnera.“

Na primer, ako postavite `width: 50%` u element, zauzimaće polovinu širine svog roditeljskog kontejnera. To čini procenat veoma važnim za kreiranje struktura koje se prilagođavaju različitim uređajima.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="container">
  <div class="box"></div>
</div>
```

```css
.container {
  width: 400px;
  height: 200px;
  background-color: lightgray;
}

.box {
  width: 50%;
  height: 100%;
  background-color: red;
}
```

:::

Procenti su najbolji za kreiranje rasporeda koji se lako prilagođavaju različitim veličinama ekrana. Na primer, postavljanje kontejnera na `width: 80%` osigurava da zauzima 80% širine svog roditelja, bez obzira na uređaj koji se koristi.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="parent">
  <div class="child"></div>
</div>
```

```css
.parent {
  width: 100%;
  height: 300px;
  background-color: lightblue;
}

.child {
  width: 80%;
  height: 100%;
  background-color: red;
}
```

:::

Korišćenje procenata za prilagodljive slike je još jedna uobičajena praksa. Korišćenjem `max-width: 100%` za slike, omogućavate im da se smanje na malim ekranima zadržavajući svoj omjer dimenzija.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<img src="https://placehold.co/150x150" alt="Example Product Image" />
```

```css
img {
  max-width: 100%;
  height: auto;
}
```

:::

Iako nije uobičajeno, procenti se mogu koristiti i za veličinu fonta kako bi se kreirao tekst čija je veličina promenljiva. Na primer, `font-size: 120%` će učiniti da je tekst 20% veći od veličine fonta svog roditelja.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="text-container">
  Parent text.
  <p class="text">This is some example text.</p>
</div>
```

```css
.text-container {
  font-size: 16px;
}

.text {
  font-size: 120%;
}
```

:::

Procenti mogu biti veoma korisni za vertikalno centriranje elementa. Evo primera kako možeš koristiti procente i svojstvo `transform` da vertikalno centriraš element.

Ovaj primer postavlja element na 50% od vrha njegovog kontejnera, a zatim koristi `transform` da ga pomeri nagore za polovinu njegove visine, čime ga efektivno vertikalno centrira.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="centered"></div>
```

```css
.centered {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
  width: 300px;
  height: 300px;
  background-color: red;
}
```

:::

Naučićete više o tome kako absolute positioning i svojstva transformacije funkcionišu detaljnije u narednim lekcijama.

Zapamtite, procenat je uvek u odnosu na nešto drugo. Za horizontalne atribute kao `width`, on je u odnosu na širinu roditelja. Za vertikalne atribute kao `height`, obično je u odnosu na visinu roditelja (ako je definisano).

Međutim, budite oprezni prilikom postavljanja elemenata sa procentualnim merama, jer to može izazvati nepouzdane rezultate. Takođe, zapamtite da procenatna visina može biti problematična ako roditelj nema postavljenu visinu.

Ukratko, procenti u CSS-u su moćan alat za pravljenje fleksibilnih i prilagodljivih rasporeda. Koristi ih kada želiš da se elementi prilagođavaju proporcionalno svom kontejneru ili kada želiš raspored koji se lako prilagođava različitim veličinama ekrana.
