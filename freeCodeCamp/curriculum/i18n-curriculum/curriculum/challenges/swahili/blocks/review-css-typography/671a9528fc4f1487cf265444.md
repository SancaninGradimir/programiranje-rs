## Uvod u tipografiju

- **Definicija**: Tipografija je umetnost izbora odgovarajućih fontova i dizajna kako bi tekst izgledao

## Karakteristika CSS `font-family`

- **Definicija**: Porodica fontova je skup fontova koji dele zajednički dizajn. Svi fontovi pripadajući toj porodici potiču iz jednog osnovnog tipografskog dizajna, ali takođe imaju varijacije u stilu, težini i širini. Možete navesti više porodica fontova u prioritetnom redosledu, od gornjeg ka donjem, razdvojene zarezima. Ovi alternativni fontovi će delovati kao zamena. Uvek trebate uključiti opštu porodicu fontova na kraju liste porodica fontova.

```css
font-family: Arial, Lato;
```

- **Često korišćene porodice fontova**: Evo nekih od najčešće korišćenih porodica fontova u CSS: serif, sans-serif, monospace, cursive, fantasy

## Sigurni fontovi za internet

- **Objašnjenje**: Ovi fontovi su mali setovi fontova koji su najverovatnije instalirani na računaru ili uređaju. Kada pretraživač pokušava da prikaže font, on pokušava da pronađe fajl fonta u korisničkom sistemu. Međutim, ako font nije dostupan, obično se vraća na podrazumevani sistemski font.

- **Sigurni fontovi za internet**:
  - Sans-serif fontovi se često koriste u razvoju web stranica jer nemaju male "noge" ili linije na krajevima slova, pa su laki za čitanje na ekranu. Primjeri sigurnih sans-serif fontova za web su: Arial, Verdana i Trebuchet MS.
  - S druge strane, serif fontovi imaju male "noge" na krajevima slova, pa se često koriste za tradicionalno štampanje. Sigurni serif fontovi za internet uključuju: Times New Roman i Georgia.

## Pravila za at na Pravilo od `@font-face`

- **Definicija**: `@` pravila su deklaracije koje daju instrukcije pretraživaču. Možete ih koristiti za definisanje različitih elemenata *stylesheet*-a, kao što su media upiti, *keyframes*, porodični font i još mnogo toga.
- **`@font-face`**: Ovo vam omogućava da definišete specifične fontove navođenjem font fajla, strukture i drugih važnih svojstava, kao što su težina i stil. Da bi pravilo `@font-face` bilo važeće, takođe morate definisati svojstvo `src`. Ovo svojstvo se odnosi na font resurse.

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2"),
       url("path/to/font.woff"),
       url("path/to/font.otf");
}
```

- **Sistemi fontova**: Za svaki resurs fonta, možete takođe specificirati format. Ovo je opcionalno. To je indikacija za pretraživač o formatu fonta. Ako nije navedeno format, resurs će biti preuzet i format detektovan nakon preuzimanja. Ako nije validan format, resurs neće biti preuzet. Mogući sistemi fontova su `collection`, `embedded-opentype`, `opentype`, `svg`, `truetype`, `woff`, i `woff2`.

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
       url("path/to/font.otf") format("opentype"),
       url("path/to/font.woff") format("woff");
}
```

- **`woff` i `woff2`**: `woff` se odnosi na "Web Open Font Format." To je veoma popularan font format razvijen od strane Mozilla u saradnji sa Type Supply, LettError i drugim organizacijama. Razlika između `woff` i `woff2` je algoritam koji se koristi za obradu podataka.
- **OpenType**: Ovo je promenljivi format fonta za računare, razvijen od strane Microsoft-a i Adobe-a, koji korisniku omogućava pristup dodatnim funkcijama unutar fonta. Često se koristi u velikim operativnim sistemima.
- **`tech()`**: Koristi se za određivanje tehnologije resursa fonta. Ovo je opcionalno.

```css
@font-face {
  font-family: "MyCustomFont"; 
  src: url("path/to/font.woff2") format("woff2"),
       url("path/to/font.otf") format("opentype") tech(color-COLRv1),
       url("path/to/font.woff") format("woff");
}
```

## Rad sa eksternim fontovima

- **Definicija**: Eksterni fontovi su fajlovi fontova koji nisu direktno uključeni u fajlove vašeg projekta. Obično se skladište na zasebnom serveru. Da biste uključili ove eksterne fontove u vaš projekat, možete koristiti element `link` ili informaciju `@import` unutar vašeg CSS.
- **Google Fonts**: Ovo je usluga kompanije Google koja pruža kolekciju fontova, mnogi od kojih su posebno kreirani za web razvoj.

Ovo je primer korišćenja funkcije `link`:

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap" rel="stylesheet">
```

```css
.roboto-thin {
  font-family: "Roboto", sans-serif;
  font-weight: 100;
  font-style: normal;
}
```

Ovo je primer korišćenja informacija iz `@import`:

```css
@import url('https://fonts.googleapis.com/css2?family=Roboto:ital,wght@0,100;0,300;0,400;0,500;0,700;0,900;1,100;1,300;1,400;1,500;1,700;1,900&display=swap');
```

- **Font Squirrel**: Ovo je još jedan popularni resurs koji možete koristiti za dodavanje prilagođenih eksternih fontova u vaše projekte.

## Svojstvo `text-shadow`

- **Definicija**: Ovo svojstvo se koristi za postavljanje senki na tekst. Morate specificirati udaljenosti X i Y, koje predstavljaju horizontalnu i vertikalnu udaljenost senke od teksta, respektivno. Ove vrednosti su neophodne. Pozitivne vrednosti za udaljenosti X i Y pomeriće senku udesno i dole, respektivno, dok će negativne vrednosti pomeriti senku ulevo i gore.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Typography Shadow</p>
```

```css
  p {
    text-shadow: 3px 2px;
  }
```

:::

- **Boja senke**: Takođe možete specificirati boju senke navođenjem ove vrednosti pre ili posle `mbali`. Ako boja nije navedena, pregledač će automatski odrediti boju, pa je uobičajeno bolje eksplicitno definisati.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Colored Shadow</p>
```

```css
p {
  text-shadow: 3px 2px #00ffc3;
}
```

:::

- **Radijus zamućenja (Blur Radius)**: Radijus zamućenja je opciono, ali će učiniti da senka izgleda mekše i nežnije. Podrazumevana vrednost radijusa zamućenja je nula. Kako se vrednost povećava, tako raste i zamućenje, što znači da senka postaje sve svetlja/mekša.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Blurred Shadow</p>
```

```css
p {
  text-shadow: 3px 2px 3px #00ffc3;
}
```

:::

- **Korišćenje više senki teksta**: Tekst može imati više od jedne senke. Senke će se primeniti u slojevima, od prednjeg ka zadnjem, a prva senka je na vrhu.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p>Multiple Shadows</p>
```

```css
p {
  text-shadow:
    3px 2px 3px #00ffc3,
    -3px -2px 3px #0077ff,
    5px 4px 3px #dee7e5;
}
```
