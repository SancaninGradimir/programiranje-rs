# --interactive--

Atribut `id` dodeljuje jedinstveni identifikator HTML elementu.

Ovo je primer elementa za `h1` sa `id` od `title`.

Ispod elementa ``h1``, dodajte element ``h2`` koji je podešen na ``id`` i koji je postavljen na ``"subtitle"``. Možete napisati bilo koji tekst za ``h2`` i videćete promene u prozoru za prikaz. Da biste interagovali sa ovim primerom, moraćete da omogućite interaktivni editor.

:::interactive_editor

```html
<h1 id="title">Movie Review Page</h1>

```

:::

Možete se osvrnuti na ime `id` za `title` unutar vašeg JavaScript ili CSS. Ovde je primer za CSS koji se odnosi na `id` za `title`, menjanje boje teksta sa `color` na `red`.

**NAPOMENA**: Neki CSS su vam dati u ovom interaktivnom primeru. Ne brinite o razumevanju koda CSS trenutno, jer ćete više naučiti o tome u budućim lekcijama. Ali ako želite da vidite kako će boja teksta promeniti u plavu, omogućite interaktivni editor, kliknite na karticu `styles.css` i promenite `color: red;` u `color: blue;`.

:::interactive_editor

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>Review page Example</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
    <h1 id="title">Movie Review Page</h1>
  </body>
</html>
```

```css
#title {
  color: red;
}
```

:::

Znak tarabe (`#`) ispred `title` govori računaru da želite da ciljate `id` sa tom vrednošću. Vrednosti atributa `id` ne smeju se koristiti više puta i uvek moraju biti jedinstvene. Još jedna važna stvar je da vrednosti atributa `id` ne mogu sadržati razmake. Evo primera korišćenja reči `main` i `heading` kao vrednosti atributa `id`:

```html
<h1 id="main heading">Main heading</h1>
```

Pretraživači će ovo mesto videti kao deo `id`, što će izazvati neželjene probleme kada dođe pitanje stilizovanja i kodiranja. Vrednosti svojstva elementa `id` treba da sadrže samo slova, brojeve, donje podkreškinge i crtice.

Razlika u atributu `id`, vrednost atributa `class` ne bi trebalo da bude jedinstvena i može imati više instanci.

Ovo je primer korišćenja klase nazvane `box` za element `div`.

```html
<div class="box"></div>
```

Ako želite dodati više imena klasama na elementu, možete to učiniti razdvajanjem imena razmakima. Evo ažuriranog primera korišćenja više klasa za element: `div`.

```html
<div class="box red-box"></div>
```

Ovo je još jedan primer korišćenja mnogo klas za mnogo elemenata od `div`.

**NAPOMENA**: Neki od CSS su vam dati u ovom interaktivnom primeru. Ne brinite o razumevanju koda CSS trenutno, jer ćete više o tome naučiti u budućim lekcijama. Ali ako želite da promenite boju prvog i trećeg kutija, omogućite interaktivni urednik i kliknite na karticu `styles.css` i promenite `background-color: red;` u `background-color: black;`.

:::interactive_editor

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>Colored boxes example</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
    <div class="box red-box"></div>
    <div class="box blue-box"></div>
    <div class="box red-box"></div>
    <div class="box blue-box"></div>
  </body>
</html>
```

```css
.box {
  width: 100px;
  height: 100px;
}

.red-box {
  background-color: red;
}

.blue-box {
  background-color: blue;
}
```

:::

Dakle, sažetno, kada bi trebalo koristiti `id` umesto `class`? Klase su najbolje za korišćenje kada želiš primeniti skup stilova na više elemenata. Ako želiš da targetuješ specifičan element, bolje je koristiti `id` jer te vrednosti treba da budu jedinstvene.
