# --description--
Najjednostavniji način za početak je da napraviš HTML datoteku na svom računaru i u nju upišeš osnovnu HTML strukturu:

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>
  <script>
    // Your JavaScript goes here!
    console.log("Hello, World!")
  </script>
</body>
</html>
```

Sačuvajte i otvorite ovaj fajl u web pregledaču (možete koristiti <a href="https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer" target="_blank" >"Live Server" na Visual Studio Code</a> za to!) pa otvorite konzolu pretraživača desnim klikom na praznu veb stranicu i odabirom "Inspect" ili "Inspect Element". U prozoru 'Developer Tools' potražite i izaberite karticu 'Console', gde bi trebalo da vidite rezultate našeg izraza `console.log`.

> `console.log()` To je komanda za štampanje nečega u konzolu developera u tvom pretraživaču. Možeš ovo da koristiš za štampanje rezultata iz bilo kojih članaka i vežbi ovde. Podstičemo te da pišeš kod uz sve primere u ovom i budućim časovima.

Drugi način za uključivanje JavaScript na web stranicu je putem eksterne skripte. Ovo je veoma slično povezivanju spoljnih fajlova CSS na vašoj veb stranici.

```html
  <script src="javascript.js"></script>
```

Datoteke za JavaScript imaju prilog od `.js`, kao što su `.css` za stilove. Eksterni fajlovi JavaScript se koriste za složeniji skript.

# --questions--

## --text--

Koja je izjava koja tačno opisuje kako uključiti JavaScript u dokument HTML?

## --answers--

JavaScript se može uključiti samo unutar oznake `<script>` unutar tela HTML.

---

Kod za JavaScript mora biti unet u sekciju od `<head>` dokumenta HTML kako bi se pravilno implementirao.

---

JavaScript može biti uključen direktno u HTML datoteku pomoću oznake `<script>` ili povezivanjem spoljne datoteke pomoću oznake `<script src="javascript.js">`.

---

Fajlovi za `JavaScript` eksterni zahtevaju posebne atribute u atributu/tag-u ``<script>`` kako bi ih prepoznali web pregledači.
