# --interactive--

U prethodnoj lekciji, predstavljeno je sa funkcijama i nizovima podataka, kao i kako dobiti različite elemente u nizu. Evo podsetnika o tome kako dobiti drugi element u nizu:

:::interactive_editor

```js
const fruits = ["apple", "banana", "cherry"];

console.log(fruits[1]); // "banana"
```

:::

Pošto nizovi koriste indeksiranje koje počinje od nule, prvi element nalazi se na indeksu `0`, drugi na indeksu `1` itd. Važno je znati da će JavaScript vratiti `undefined` ako pokušaš da pristupiš indeksu koji ne postoji u nizu.

:::interactive_editor

```js
let fruits = ["apple", "banana", "cherry"];
console.log(fruits[3]); // undefined
```

:::

U ovom primeru, nema polja u modelu `3` za red/polje `fruits`. Stoga će zapis vratiti `undefined`. Sada, hajde da istražimo kako da ispravite polja u redu. Možete izmeniti polje postavljanjem nove vrednosti za određeni model.

:::interactive_editor

```js
let fruits = ["apple", "banana", "cherry"];
fruits[1] = "blueberry";
console.log(fruits); // ["apple", "blueberry", "cherry"]
```

:::

U ovom primeru zamenili smo `banana` sa `blueberry` na indeksu `1`. Ovaj način ti omogućava da izmeniš bilo koji element niza ako znaš njegov indeks. Takođe možeš da dodaš nove elemente dodeljivanjem vrednosti indeksu koji još ne postoji:

:::interactive_editor

```js
let fruits = ["apple", "banana", "cherry"];
fruits[3] = "date";
console.log(fruits); // ["apple", "banana", "cherry", "date"]
```

:::

Međutim, budi pažljiv kada ovo radiš. Ako postaviš dimenziju mnogo veću od trenutne dužine reda, stvorićeš neizdefinovane elemente sa postojećim placeholderima u sredini, što može izazvati neočekivano ponašanje. Dok nastaviš da radiš sa JavaScript, shvatićeš da su ove metode pristupa i popravljanja elemenata reda osnova mnogih programskih zadataka. Bilo da gradiš jednostavnu listu funkcija ili radiš sa složenim strukturama podataka, ovo znanje će biti veoma korisno.
