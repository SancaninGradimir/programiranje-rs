# --description--

Na poslednjoj lekciji, naučili ste o klasama objekata i kako se one koriste kao mape za kreiranje stvari.

Ovde ćemo detaljnije osvetliti karakteristike elementa i metode.

Prvo analizirajmo svojstva elementa, a zatim i metod.

Atributi objekta su parametri koje poseduje objekt, pa oni skladište podatke. Postoje dve vrste atributa objekta: atributi instance i atributi klase.

Svojstva instance su jedinstvena za svaki objekat kreiran iz klase, i obično se postavljaju putem `__init__`. Svojstva klase, s druge strane, pripadaju samoj klasi i dele se sa svim instancama te klase.

Da biste dobili svojstvo elementa, koristite tačkastu notaciju.

Ovde su primjeri atributa instanci i atributa klase, i kako ih izvući iz objekata:

```python
class Dog:
    species = "French Bulldog" # Class attribute

    def __init__(self, name):
        self.name = name # Instance attribute

print(Dog.species) # French Bulldog

dog1 = Dog("Jack")
print(dog1.name)    # Jack
print(dog1.species) # French Bulldog

dog2 = Dog("Tom")
print(dog2.name)    # Tom
print(dog2.species) # French Bulldog
```

Zapamtite da možete pristupiti svojstvima klase objekata direktno iz same klase, ali morate prvo kreirati objekt i dati mu podatke pre nego što pristupite svojstvima instance.

Automobili su još jedan dobar primer, jer svi automobili imaju model i boju:

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

car_1 = Car("red", "Toyota Corolla")
car_2 = Car("green", "Lamborghini Revuelto")

print(car_1.model) # Toyota Corolla
print(car_2.model) # Lamborghini Revuelto

print(car_1.color) # red
print(car_2.color) # green
```

Metode su definisane unutar klase objekata. Ovim metodama, bilo koji objekat definisan iz te klase može izvršiti radnje ili promeniti sopstvene podatke. Takođe dobijate metode korišćenjem tačkasto zapisivanja (dot notation).

Na primer, psi mogu da vuku. Dakle, možemo imati način za `bark` u klasi `Dog`, kao što ste videli u prethodnoj lekciji:

```python
class Dog:
   species = "French Bulldog"

   def __init__(self, name):
     self.name = name

   def bark(self):
       return f"{self.name} says woof woof!"

jack = Dog("Jack")
jill = Dog("Jill")

print(jack.bark()) # Jack says woof woof!
print(jill.bark()) # Jill says woof woof!
```

Klasa od `Car` takođe može imati metodu `describe`:

```python
class Car:
    def __init__(self, color, model):
        self.color = color  # Instance attribute
        self.model = model  # Instance attribute

    def describe(self):
        return f"This car is a {self.color} {self.model}"

car_1 = Car("red", "Toyota Corolla")
car_2 = Car("green", "Lamborghini Revuelto")

print(car_1.describe()) # This car is a red Toyota Corolla
print(car_2.describe()) # This car is a green Lamborghini Revuelto
```
