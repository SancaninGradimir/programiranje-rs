# --description--

U prvim dva reda fajla `myApp.js`, možete videti kako je jednostavno kreirati Express aplikacioni objekat. Ovaj objekat ima nekoliko metoda, i naučićete mnoge od njih u ovim zadacima. Jedna važna metoda je `app.listen(port)`. Ona govori vašem serveru da sluša na određenom portu, i postavlja ga u stanje pokretanja. Zbog testiranja, treba nam da aplikacija radi na root nivou, zato smo dodali ovu metodu u `server.js` fajl za vas.

Hajde da obradimo naš prvi string! U Express-u, rute imaju sledeći format: `app.METHOD(PATH, HANDLER)`. METODA je http metoda u malim slovima. PUTANJA RUTE je putanja na serveru (može biti string, ili čak regularni izraz). HANDLER je funkcija koju Express poziva kada se ruta poklapa. Handler obično ima format `function(req, res) {...}`, gde je req objekat zahteva, i res objekat odgovora. Na primer, handler

```js
function(req, res) {
  res.send('Response String');
}
```

će poslati string 'Response String'.
