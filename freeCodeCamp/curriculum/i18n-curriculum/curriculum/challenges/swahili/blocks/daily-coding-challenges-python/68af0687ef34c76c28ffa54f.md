# --description--
Ako vam je dat opseg celih brojeva od 1 do `n`, uključujući, vratite opseg svih celih brojeva koji nedostaju između 1 i `n` (gde je `n` najveći broj u datom opsegu podataka).

- Izvuceni red podataka može biti neukladen i može sadržati duplikate.
- Red podataka koji se vraća mora biti u uzlaznom redosledu.
- Ako nema nedostajućih celobrojnih brojeva, vratite prazan red podataka.

```py
def find_missing_numbers(arr):
    if not arr:
        return []
    
    n = max(arr)
    seen = set(arr)
    result = [i for i in range(1, n + 1) if i not in seen]
    return result
```
