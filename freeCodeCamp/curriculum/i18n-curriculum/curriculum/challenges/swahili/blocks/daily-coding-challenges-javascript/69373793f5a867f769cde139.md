# --description--
Uzimajući u obzir tačno dve brojke, par za rupu gofu i broj crteža koje je igrač gofu uzeo iz te rupe, vratite rezultat igrača koristeći reči gofu.

Rudisha:

- `"Hole in one!"` ikiwa ilichukua mchoro mmoja.
- `"Eagle"` ako je odigrao dve rupe ispod para.
- `"Birdie"` ako je odigrao jednu rupu ispod para.
- `"Par"` ikiwa ilichukua idadi sawa ya michoro na par.
- `"Bogey"` ako je odigrao jednu rupu iznad para.
- `"Double bogey"` ako je odigrao dve rupe iznad para.

```js
function golfScore(par, strokes) {
  if (strokes === 1) return "Hole in one!";

  const diff = strokes - par;

  if (diff === -2) return "Eagle";
  if (diff === -1) return "Birdie";
  if (diff === 0) return "Par";
  if (diff === 1) return "Bogey";
  if (diff === 2) return "Double bogey";
}
```
