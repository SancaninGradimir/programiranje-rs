# --interactive--

Komponenta `Number()` se koristi za kreiranje brojčanog objekta. Brojčani objekat ima nekoliko važnih parametara i metoda kao što su `isNaN` i metoda `toFixed`. Evo primera koji koristi komponentu `Number()` zajedno sa glavnom reči `new`:

:::interactive_editor

```js
const myNum = new Number("34");
console.log(typeof myNum); // "object" 
```

:::

U ovom primeru pregledavamo sekvencu karaktera za komponentu `Number()` i tip return je tip `object` umesto `string`.

Kada komponenta `Number()` bude pozvana kao funkcija bez glavne reči `new`, onda će vrednost return biti osnovni numerički tip. Često ćete koristiti komponentu `Number()` da konvertujete druge tipove podataka u numerički tip. Evo primera za konverziju niza slova u broj:

:::interactive_editor

```js
const myNum = Number("100");
console.log(myNum); // 100

console.log(typeof myNum); // number
```

:::

Ovo je pomoć kada dobijete unos od korisnika i morate da konvertujete taj unos sekvence slova u broj kako biste mogli da izvršite matematičke proračune.

Ako pokušate da pozovete komponentu `Number()` sa praznim nizom karaktera, rezultat će biti broj `0`:

:::interactive_editor

```js
const num = Number("");
console.log(num); // 0
```

:::

To je zato što će JavaScript pokušati da obradi tekstualni niz i, pošto ne sadrži nijednu cifru, rezultat će biti nula.

Ako pokušate da unesete sekvencu slova sa slučajnim slovima, rezultat će biti `NaN` ili "Nije broj".

:::interactive_editor

```js
const num = Number("random");
console.log(num); // NaN
```

:::

Kada radi sa tačnim ili netačnim vrednostima, `true` vraća `1` jer `true` je konvertovano u jedan i `false` vraća `0` jer `false` je konvertovano u nulu.

:::interactive_editor

```js
const boolTrue = Number(true);
const boolFalse = Number(false);

console.log(boolTrue); // 1
console.log(boolFalse); // 0
```

:::

Ako pošaljete `null`, rezultat će biti `0` i ako pošaljete `undefined`, rezultat će biti `NaN`.

:::interactive_editor

```js
const undefinedNum = Number(undefined);
const nullNum = Number(null);

console.log(undefinedNum); // NaN
console.log(nullNum); // 0
```

:::

Kada radiš sa redovima podataka, ima nekoliko stvari koje treba uzeti u obzir.

Prazan red podataka će vratiti `0`. Red podataka sa jednim brojem će vratiti taj broj. Red podataka sa više brojeva će vratiti `NaN`. I red podataka sa sekvencom slova takođe će vratiti `NaN`.

:::interactive_editor

```js
const emptyArr = Number([]);
const arrOneNum = Number([7]);
const arrMultiNum = Number([7, 36, 12]);
const arrStr = Number(["str1"]);
const arrMultiStr = Number(["str1", "str2"]);

console.log(emptyArr); // 0
console.log(arrOneNum); // 7
console.log(arrMultiNum); // NaN
console.log(arrStr); // NaN
console.log(arrMultiStr); // NaN
```

:::

Kada radiš sa stvarima, rezultat je uvek `NaN`.

:::interactive_editor

```js
const obj1 = Number({});
const obj2 = Number({2: 2});
const obj3 = Number({key: "val"});
const obj4 = Number({key: true});

console.log(obj1); // NaN
console.log(obj2); // NaN
console.log(obj3); // NaN
console.log(obj4); // NaN
```

:::

Zaključno, više ćete koristiti komponentu `Number()` za menjanje tipova, nego za kreiranje brojeva ili objekata sa brojevima.
