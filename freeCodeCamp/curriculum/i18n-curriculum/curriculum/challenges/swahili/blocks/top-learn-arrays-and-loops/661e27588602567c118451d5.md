# --description--

Sada, nakon što smo dobili osnovno razumevanje o nizovima podataka, hajde da pričamo o petljama. Petlje se koriste za izvršavanje dela koda više puta. Jedna od tih petlji je petlja `while`. Petlja `while` izvršava deo koda samo ako je uslov tačan. Sintaksa petlje za `while` je sledeća:

```javascript
while (condition) {
  // code block to be executed
}
```

Na primer, deo koda ispod ispisuje brojeve od 1 do 5:

```javascript
let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
```

Kodni segment iznad inicijalizuje parametar `i` sa vrednošću `1`. Petlja `while` izvršava kodni segment uslovom da je vrednost `i` manja ili jednaka `5`. Vrednost `i` se povećava za `1` za svaku iteraciju.
