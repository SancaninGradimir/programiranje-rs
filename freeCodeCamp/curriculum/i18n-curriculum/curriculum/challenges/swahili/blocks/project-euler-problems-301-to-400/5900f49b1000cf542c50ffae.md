# --description--

Za ceo pozitivan broj $n$, definišite $f(n)$ kao najmanji pozitivan višestrukost $n$-a koji, kada je predstavljen u bazi 10, koristi samo cifre $\leq 2$.

Dakle $f(2) = 2$, $f(3) = 12$, $f(7) = 21$, $f(42) = 210$, $f(89) = 1\\,121\\,222$.

Pia, $\displaystyle\sum_{n = 1}^{100} \frac{f(n)}{n} = 11\\,363\\,107$.

Izračunaj $\displaystyle\sum_{n = 1}^{10\\,000} \frac{f(n)}{n}$.
