# --description--

U Python, opseg određuje gde možete pristupiti promenljivoj. On kontroliše životni vek promenljive i kako se ona rešava u različitim delovima koda.

Da bi se tačno utvrdio opseg, Python prati princip **LEGB**, koji stoji za sledeće:

- **Lokalni opseg (L)**: Parametri deklarisani unutar funkcija ili klasa objekata.
    
- **Okružni opseg (E)**: Parametri deklarisani u funkcijama koje su ugnježđene ili još više ugnježđene.
    
- **Globalni opseg (G):** Parametri deklarisani na nivou modula ili fajla.
    
- **Ugrađeni opseg (B)**: Imena sačuvana u Python za prethodno deklarisane funkcije, module, ključne reči i objekte.

Python koristi pravilo LEGB za rešavanje opsega varijabli u vašoj aplikaciji. Analiziraćemo svako od ovih pravila kako biste dobili bolje razumevanje procesa.

**Lokalni opseg** znači da varijabla deklarisana unutar funkcije ili klase objekata može se pristupiti samo unutar te funkcije ili te klase.

Evo primera:

```python
def my_func():
    my_var = 10
    print(my_var)
```

U ovom slučaju, funkcija `my_func` ima svoj opseg koji ne može biti dostupan izvan funkcije. Pozivanje `my_func` će prikazati `10`, ali štampanje `my_var` izvan funkcije će dovesti do `NameError`:

```python
def my_func():
    my_var = 10 # Locally scoped to my_func
    print(my_var)

my_func() # 10

print(my_var) # NameError: name 'my_var' is not defined
```

**Opseg dometa** znači da funkcija definisana unutar druge funkcije može pristupiti parametrima funkcije koja je definisana unutar nje.

Na primer:

```python
def outer_func():
    msg = 'Hello there!'

    def inner_func():
        print(msg)

    inner_func()

outer_func() # Hello there!
```

U ovom primeru, unutrašnja funkcija, `inner_func`, može slobodno pristupiti parametru `msg` deklarisanom u spoljašnjoj funkciji, `outer_func`. Međutim, zapamtite da spoljne funkcije ne mogu pristupiti parametrima deklarisanim unutar ugnježdene funkcije:

```python
def outer_func():
    msg = 'Hello there!'
    print(res)

    def inner_func():
        res = 'How are you?'
        print(msg)

    inner_func()

outer_func() # NameError: name 'res' is not defined
```

Ovo je zato što `res` je lokalni opseg za `inner_func`. Takođe, proverite da `outer_func` pokušava da objavi `res` pre nego što `inner_func` bude pozvan.

Jedno rešenje je da se kreira `res` kao prazan niz znakova u obuhvatu, koji je unutar `outer_func`. Zatim unutar `inner_func`, učinite `res` opcionalnim parametrom i ključnom rečju `nonlocal`:

```python
def outer_func():
    msg = 'Hello there!'
    res = ""  # Declare res in the enclosing scope

    def inner_func():
        nonlocal res  # Allow modification of an enclosing variable
        res = 'How are you?'
        print(msg)  # Accessing msg from outer_func()

    inner_func()
    print(res)  # Now res is accessible and modified

outer_func()

# Output:
# Hello there!
# How are you?
```

**Globalni opseg** odnosi se na varijable deklarisane van funkcija ili klasa objekata koje se mogu pristupiti bilo gde u programu. Ovde, `my_var` može se pristupiti bilo gde, čak ni unutar funkcije koja ga ne deklarše:

```python
my_var = 100

def show_var():
    print(my_var)

show_var() # 100
print(my_var) # 100
```

I ako želite da deklarisani parametar unutar funkcije ima globalnu dostupnost, možete koristiti ključnu reč `global`:

```python
my_var_1 = 7

def show_vars():
    global my_var_2
    my_var_2 = 10
    print(my_var_1)
    print(my_var_2)

show_vars() # 7 10

# my_var_2 is now a global variable and can be accessed anywhere in the program
print(my_var_2) # 10
```

Takođe možete koristiti ključnu reč za `global` da promenite globalni parametar:

```python
my_var = 10  # A global variable

def change_var():
    global my_var  # Allows modification of a global variable
    my_var = 20

change_var()

print(my_var)  # my_var is now modified globally to 20
```

Konačno, **ugrađeni opseg** obuhvata sve prethodno deklarisane funkcije za Python, module i ključne riječi, dostupne bilo gdje u vašem programu:

```python
print(str(45)) # '45'
print(type(3.14)) # <class 'float'>
print(isinstance(3, str)) # False
```
