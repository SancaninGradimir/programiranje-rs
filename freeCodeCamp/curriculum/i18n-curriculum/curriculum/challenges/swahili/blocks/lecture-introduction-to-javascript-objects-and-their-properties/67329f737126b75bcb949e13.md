# --interactive--

U JavaScript, to je osnovna struktura podataka koja omogućava skladištenje i organizovanje podataka i povezane funkcionalnosti.

Možete zamisliti nešto kao kontejner koji drži razne komadiće informacija, poput fišnog ormana koji čuva različite fascikle i dokumente.

Ovi komadi informacija nazivaju se svojstva, i imaju ime (ili ključ) i vrednost.

```js
const exampleObject = {
  propertyName: value,
}
```

Objekti su veoma različitih vrsta i oni su osnova za JavaScript. Zapravo, skoro sve u JavaScript je objekat ili se može tretirati kao objekat. Ovo uključuje strukture podataka, funkcije, pa čak i osnovne tipove podataka kao što su nizovi slova i brojeva kada se koriste na neki način.

Ova priroda JavaScript koja se fokusira na stvari je jedan od razloga zašto je jezik promenljiv i moćan. Pogledajte kako možete nešto kreirati:

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};
```

U ovom primeru, kreirali smo nešto nazvano `person` sa tri svojstva: `name`, `age`, i `city`. Svako svojstvo ima ime i vrednost, razdvojeno je dve tačke.

Sada, hajde da istražimo kako možete dobiti ova svojstva/atributi. Postoje dva glavna načina za dobijanje atributa objekta u JavaScript: korišćenje tačaka i korišćenje zagrada.

Notacija sa tačkom je najčešći i najlakši način za dobijanje svojstava objekta. Ovo je osnovna sintaksa notacije sa tačkom:

```js
objectName.propertyName
```

Ovo je kako koristiš dot notaciju i naš element za `person`:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};

console.log(person.name);  // Alice
console.log(person.age);   // 30
```

:::

Zapis pomoću tačke je kraći i lakši za čitanje i preporučuje se kada znate tačno ime svojstva kojem želite da pristupite i kada je to ime važeći JavaScript identifikator (odnosno ne počinje brojem i nema posebne znakove ili razmake).

S druge strane, zapis pomoću uglastih zagrada omogućava pristup svojstvima objekta koristeći tekstualni niz unutar uglastih zagrada. Ovako se koristi zapis pomoću uglastih zagrada:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};

console.log(person["name"]); // Alice
console.log(person["age"]); //  30
```

:::

Korišćenje uglastih zagrada je lakše od korišćenja tačaka jer vam omogućava da koristite imena atributa koja nisu važeći identifikatori za JavaScript. Na primer, ako imate ime atributa sa razmakom ili crticama i brojevima, morate koristiti zapis u uglastim zagradama:

:::interactive_editor

```js
const oddObject = {
  "1stProperty": "Hello",
  "property with spaces": "World"
};

console.log(oddObject["1stProperty"]);  // Hello
console.log(oddObject["property with spaces"]);  // World
```

:::

Još jedna prednost korišćenja zagrada je što vam omogućava da koristite parametre za dobijanje atributa putem transformacije:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  city: "Wonderland"
};

let propertyName = "city";
console.log(person[propertyName]); // Wonderland
```

:::

Ova jednostavnost čini neophodnim korišćenje zagrada posebno kada ne znate pravi naziv varijable dok pišete kod, ili kada koristite nazive varijabli koji dolaze iz unosa korisnika ili drugog izvora promenljivih podataka.

Važno je znati da su objekti u JavaScript moćni i višestruki. Ne mogu samo da skladište jednostavne vrednosti kao niz slova i brojeva, već i nizove podataka ili druge stvari.

Razumevanje stvari i kako da se s njima radi veoma je važno u JavaScript jer se često koristi u jeziku i u bibliotekama i sistemima za rad JavaScript.

Kada nastavite da učite i radite sa JavaScript, otkrićete da posedovanje stvari otvara mnogo mogućnosti za kreiranje složenih i moćnih programa.
