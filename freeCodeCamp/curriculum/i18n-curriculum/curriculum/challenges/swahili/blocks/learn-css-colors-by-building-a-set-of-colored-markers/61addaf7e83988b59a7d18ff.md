# --description--

Funkcija skraćivanja za `border-left` vam omogućava da istovremeno postavite širinu, stil i boju leve ivice.

Ovde je sintaksa:

```css
border-left: width style color;
```

U pravilu `.sleeve` CSS, promeni svojstva za `border-left-width`, `border-left-style`, i `border-left-color` na skraćeno svojstvo od `border-left`. Vrednosti za širinu, stil i boju leve ivice moraju biti iste.

# --hints--

```js
assert(!/border-left-width/g.test(code));
```

Tvoj zakon od `.sleeve` CSS ne bi trebalo da ima karakteristike i vrednost `border-left-style`.


Vaš pravilnik za `.sleeve` CSS ne bi trebalo da ima kvalitet i vrednost `border-left-color`.

```js

```
