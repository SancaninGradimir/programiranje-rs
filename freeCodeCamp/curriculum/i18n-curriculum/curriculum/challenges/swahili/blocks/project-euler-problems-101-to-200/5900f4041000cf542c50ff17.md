# --description--

Postoje više načina da se napiše broj $\frac{1}{2}$ kao zbir kvadrata različitih celih brojeva.

Na primer, brojevi {2,3,4,5,7,12,15,20,28,35} mogu se koristiti:

$$\frac{1}{2} = \frac{1}{2^2} + \frac{1}{3^2} + \frac{1}{4^2} + \frac{1}{5^2} + \frac{1}{7^2} + \frac{1}{{12}^2} + \frac{1}{{15}^2} + \frac{1}{{20}^2} + \frac{1}{{28}^2} + \frac{1}{{35}^2}$$

Zaista, koristeći samo potpune brojeve između 2 i 45 zajedno, postoji samo tri načina da se to učini; dva preostala načina su: {2,3,4,6,7,9,10,20,28,35,36,45} i {2,3,4,6,7,9,12,15,28,30,35,36,45}.

Koliko načina postoji da se broj $\frac{1}{2}$ napiše kao zbir kvadrata koristeći različite cele brojeve između 2 i 80?
