# --description--

Dobrodošli u projekat CSS Variables Skyline! Počnite dodavanjem deklaracije za `!DOCTYPE html` iznad dokumenta kako bi pretraživač znao koji je tip dokumenta koji čita.

# --hints--

Vaš kod treba da ima referencu na `DOCTYPE`.

```js
assert(code.match(/<!DOCTYPE/gi));
```

Treba da staviš razmak nakon reference `DOCTYPE`.

```js
assert(code.match(/<!DOCTYPE\s+/gi));
```

Trebalo bi da definišete tip dokumenta kao `html`.

```js
assert(code.match(/html/gi));
```

Morate zatvoriti deklaraciju za `DOCTYPE` za `>` nakon tipa.

```js
assert(code.match(/html\s*>/gi));
```

Tvoje saopštenje za `DOCTYPE` trebalo bi biti na početku tvog HTML.

```js
assert(__helpers.removeHtmlComments(code).match(/^\s*<!DOCTYPE\s+html\s*>/i));
```
