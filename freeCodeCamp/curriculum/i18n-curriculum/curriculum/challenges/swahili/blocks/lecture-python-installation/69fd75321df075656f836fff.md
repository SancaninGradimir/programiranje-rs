# --description--

U prethodnoj lekciji, naučili ste kako da pokrećete skripte Python unutar vašeg računara. Ali postoje situacije kada ne želite da kreirate pune programe Python i samo vam je potrebno testirati određeni kod Python. Ovde dolazi interaktivna šelja (shell) Python.

Ali prvo, hajde da vidimo šta je terminal.

Terminal je tekstualni interfejs koji vam omogućava da komunicirate sa svojim računarom pisanjem komandi. Svaki operativni sistem dolazi sa osnovnim terminalnim programom. Na macOS-u, možete koristiti aplikaciju Terminal. Na Windowsu, možete koristiti Command Prompt ili PowerShell. Na Linuksu, svako desktop okruženje ima svoj osnovni terminalni program, kao što su GNOME Terminal ili Konsole.

Otvori terminal aplikaciju i ukucaj `python`, zatim pritisni `Enter`. Ovo će pokrenuti Python interactive shell. Interactive shell je program koji vam omogućava da direktno ukucate komande i vidite njihove rezultate.

Kada započnete novu sesiju, možete videti ovakve rezultate u početku:

```bash
Python 3.12.2 (main, Mar 21 2024, 22:48:26) [Clang 14.0.3 (clang-1403.0.22.14.1)] on darwin
Type "help", "copyright", "credits" or "license" for more information.
>>> 
```

Znak `>>>` znači Python i čeka da uneseš komandu. Pokušaj da zalepiš `"Hello, world!"` u terminal:

```bash
print("Hello, world!")
```

Trebalo bi da vidite tekst koji izgleda ovako:

```bash
>>> print("Hello, world!")
Hello, world!
```

Nakon što se tekst objavi, Python se vraća na sledeće:

```bash
>>>
```

Ovo znači da možete napisati drugu komandu.

Interpreter Python prati ciklus poznat kao Čitaj, Procjeni, Ispiši, Ponovi (Read, Evaluate, Print, Loop). Ili kraće, REPL. Svaki put kada napišete komandu, interpreter čita unos, proverava ga, ispisuje rezultate i ponavlja kako bi prikazao `>>>` ili da napišete više komandi.

Šta se dešava ako pokušate da unesete nalog kao što je ovaj?

```bash
>>> something random
```

Dakle, interpreter za Python i dalje će pratiti taj REPL proces. U ovom slučaju, dobićete poruku o grešci:

```bash
>>> something random
  File "<stdin>", line 1
    something random
              ^^^^^^
SyntaxError: invalid syntax
```

Ako želite da napustite interaktivnu šelju, možete ukucati `exit()` ili kliknuti `Ctrl + D` za (Mac/Linux) ili `Ctrl + Z + Enter` za (Windows).

Korišćenje Python interaktivne šelje je dobro za male testove koda ili osnovno istraživanje. Kada nastavite sa preostalim lekcijama u ovom kursu, možete napisati neke nove metode koje ste naučili unutar interaktivne šelje.

Ako planirate da radite sa dugim programima ili zadacima koji uključuju mnogo fajlova, onda je bolje raditi u editoru koda ili IDE-u.
