# --explanation--

U **indirektnom govoru**, izraz `said that` koristi se za prenošenje onoga što je neko rekao bez navođenja njegovih tačnih reči. Reč `that` nije obavezna i često se izostavlja u svakodnevnom govoru. Na primer:

- Usemi wa moja kwa moja: `We need to update the software.`

- Izveštani sadržaj je: `He said that we need to update the software.` ili `He said we need to update the software.`

Oba izdanja su tačna i imaju isto značenje.
