# --interactive--

API za `Web Animations` (WAAPI) omogućava vam da kreirate i kontrolišete animacije direktno u JavaScript-u. Pomoću WAAPI-ja možete raditi sa animacijama na dinamičniji način, što olakšava njihovu izmenu.

Istražite API detaljnije, kako biste počeli da radite sa animacijom direktno unutar vašeg koda JavaScript. Osnovni deo API je komponenta `Animation`, koja pruža nekoliko metoda i svojstava koja vam omogućavaju da animirate elemente dinamički.

Glavna metoda u objektu `Animation` je `animate()`. Ona omogućava kreiranje animacije definisanjem ključnih kadrova i opcija kao što su trajanje, smer, način ublažavanja prelaza i broj ponavljanja.

Ovo je osnovna sintaksa metode `animate()`:

```js
element.animate(keyframes, options);
```

Tuchukulie mfano.

```html

<div class="square" id="square"></div>
```

```css
body {

  background: #1b1b32;
```

```js
const square = document.querySelector("#square");

const animation = square.animate(
  [{ transform: "translateX(0px)" }, { transform: "translateX(100px)" }],
  {
    duration: 2000, // makes animation lasts 2 seconds
    iterations: Infinity, // loops indefinitely
    direction: "alternate", // moves back and forth
    easing: "ease-in-out" // smooth easing
  }
);
```

Izmenimo ovaj primer koristeći metode `play()`, `pause()` i osobinu `onfinish`. Evo nove verzije:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<div class="square" id="square"></div>

<button id="playBtn">Play</button>
<button id="pauseBtn">Pause</button>
<script src="index.js"></script>
```

```css
body {
  background: #f1f1f1;
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  height: 100vh;
}
.square {
  background: #1b1b32;
  width: 10rem;
  aspect-ratio: 1/1;
  margin-bottom: 20px;
}
button {
  margin: 10px;
  padding: 10px 20px;
  font-size: 16px;
}
```

```js
const square = document.querySelector("#square");
const playBtn = document.querySelector("#playBtn");
const pauseBtn = document.querySelector("#pauseBtn");

const animation = square.animate(
  [{ transform: "translateX(0px)" }, { transform: "translateX(200px)" }],
  {
    duration: 5000, // Animation lasts 5 seconds
    // iterations: Infinity, // Loops indefinitely
    direction: "alternate", // Moves back and forth
    easing: "ease-in-out" // Smooth easing function
  }
);

// Set the onfinish property to log a message when the animation ends
animation.onfinish = () => {
  console.log("Animation finished!");
};

// Play the animation when the "Play" button is clicked
playBtn.addEventListener("click", () => {
  animation.play();
  console.log("You start the animation");
});

// Pause the animation when the "Pause" button is clicked
pauseBtn.addEventListener("click", () => {
  animation.pause();
  console.log("You pause the animation");
});
```

:::

Rezultat u pregledaču će prikazati plavu kutiju koja se pomera sleva nadesno kada se klikne dugme `play`.

WAAPI proširuje mogućnosti CSS animacija pružajući veću dinamičku kontrolu nad animacijama unutar JavaScript-a.

Za animaciju CSS, opisuje animaciju deklarativnom metodom koristeći svojstva kao što su `animation-name`, `animation-duration`, i `animation-timing-function`. Takođe ima mogućnost da uradi isto koristeći WAAPI metod putem `animate()`.

Razlika je u tome što možete direktno i dinamičnije kontrolisati animacije koje kreirate pomoću `animate()`, dok kod CSS animacija morate uraditi više posla kroz kreiranje posebnih stilova i njihovo pokretanje iz JavaScript fajla.

CSS animacije su odlične za jednostavne i deklarativne animacije koje se izvršavaju direktno. To uključuje efekte pojavljivanja, promene stanja ili animacije koje ne zahtevaju mnogo interakcije nakon pokretanja. Ako vaša animacija treba da reaguje na interakciju korisnika kao što su klikovi, pomeranje stranice ili želite da korisnik može da je pauzira, vrati unazad ili dinamički promeni brzinu, WAAPI je bolji izbor.

Takođe možete razmotriti kombinovanje WAAPI-ja i CSS animacija kada vam je potrebna jednostavnost CSS-a za osnovnu postavku, ali i kontrola animacije tokom izvršavanja.
