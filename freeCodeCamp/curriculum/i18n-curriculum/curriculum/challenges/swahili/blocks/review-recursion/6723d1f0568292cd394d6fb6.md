# --description--

- Rekurzija je koncept programa koji omogućava ponovno pozivanje funkcije sve dok ne stigne do baznog slučaja.

Ovde je primer rekurentne funkcije koja računa faktorijel broja:

```js
function findFactorial(n) {
  if (n === 0) {
    return 1;
  }
  return n * findFactorial(n - 1);
}
```

U ovom gornjem primeru, funkcija `findFactorial` je pozvana rekurzivno dok `n` ne dosegne `0`. Kada je `n` jednako `0`, postiže se bazni slučaj i funkcija vraća `1`. Zatim funkcija vraća proizvod `n` i rezultat rekurzivnog poziva za `findFactorial(n - 1)`.

- Rekurzija vam omogućava da obrađujete nešto sa nepoznatom dubinom, kao što su ugnježdeni elementi ili sistem fajlova.
- Stack poziva koristi se za praćenje poziva funkcija tokom rekurzije. Svaki put kada je funkcija pozvana, dodaje se na stack poziva. Kada se dostigne bazni slučaj, pozivi funkcija uklanjaju se sa steka.
- Treba pažljivo definisati bazni slučaj jer neograničeni pozivi mogu uzrokovati pad vašeg koda. To je zato što rekurzija nastavlja dodavati sve više poziva funkcija dok sistem ne potroši memoriju.
- Rekurzija se koristi za rešavanje matematičkih problema kao što su faktorijel i Fibonaccijev niz, za prolazak kroz drveće i grafove, generisanje kombinacija i mnogo više.
