# --description--

U programiranju, posebno kada koristite sisteme za kontrolu verzija kao što su Git, `push`, to znači učitavanje promena iz lokalne baze podataka u udaljenu.

Stari status `pushed` pokazuje da je ovaj korak već izvršen.

Na primer, `He pushed his code to the GitHub repository` znači da je poslao svoje najnovije izmene koda u GitHub repozitorijum.
