# --description--

JavaScript takođe ima mogućnost poređivanja tipova i vrednosti korišćenjem operatora stroge jednakosti `===`. Ovaj operator proverava da li su dve vrednosti jednake i imaju isti tip. Primer:

```javascript
let x = 5;
let y = "5";

let result = x === y;

console.log(result); // false
```

U ovom primeru, varijabla `result` će imati `false` jer je `x` broj, a `y` je niz znakova. Međutim, kada se koristi operator `==`, rezultat će biti `true` jer će JavaScript konvertovati niz znakova u broj i uporediti vrednosti.

Ovaj operator se zove operater stroge jednakosti jer proverava jednakost vrednosti i tipa. Često se savetuje korišćenje operatera `===` kako bi se izbegli nepouzdani rezultati prilikom poređenja vrednosti.

Postoji i operator za značajnu razliku `!==` koji proverava da li dve vrednosti nisu jednake, ali imaju isti tip.

Postoji mnogo načina da se koristi alat za strogo uravnoteživanje (leveling), i važno je razumeti kako funkcioniše.
