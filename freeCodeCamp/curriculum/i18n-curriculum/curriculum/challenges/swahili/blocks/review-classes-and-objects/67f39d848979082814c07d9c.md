## Klase i predmete za Python

- **Definicija klase objekata**: Klasa je šema za kreiranje objekata. Definiše karakteristike koje će objekat imati putem svojih svojstava i metoda. Evo osnovnog primera definicije klase u Python:

```python
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f'{self.name.upper()} says woof woof!')
```

- **Kreiranje objekata**: Objekti su instancije klase. Kreiraju se pozivanjem klase i potrebnih argumenata.

```python
dog1 = Dog('Jack', 3)
dog2 = Dog('Thatcher', 5)

dog1.bark()  # JACK says woof woof!
dog2.bark()  # THATCHER says woof woof!
```

- **Pozivanje metoda na objektima**: Možete pozvati metode na objektima kako biste izvršili akcije ili dobili informacije.

```python
object_name1.method_name()
object_name2.method_name()
```

- **Razlika između klase i objekta**: Klasa je ponovno upotrebljiv šablon, dok je objekat specifična instanca te klase sa stvarnim podacima.

## Atributi elementa

- **Svojstva instance**: Definisana su u `__init__()` koristeći `self`, i jedinstvene su za svaki objekat.
- **Svojstva klase**: Definisana su direktno unutar klase i dele se sa svim objektima.

```python
class Dog:
    species = 'French Bulldog'  # Class attribute

    def __init__(self, name):
        self.name = name  # Instance attribute

print(Dog.species) # French Bulldog

jack = Dog('Jack')
print(jack.name)     # Jack
print(jack.species)  # French Bulldog
```

## Put

- **Njia**: Metode definisane unutar klase koje operišu na svojstvima atributa objekta.

```python
class Car:
    def __init__(self, color, model):
        self.color = color
        self.model = model

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
print(my_car_1.describe())  # This car is a red Tesla Model S
```

- **Pozivanje metode**: Pozivate metode na objektima pomoću tačka-zarez (dot notation). Evo primera pozivanja metode `describe` na dva različita automobila:

```python
class Car:
    def __init__(self, color, model):
        self.color = color  
        self.model = model  

    def describe(self):
        return f'This car is a {self.color} {self.model}'

my_car_1 = Car('red', 'Tesla Model S')
my_car_2 = Car('green', 'Lamborghini Revuelto')

print(my_car_1.describe()) # Calling method using the dot notation

print(my_car_2.describe()) # Calling method using the dot notation
```

## Putevi za dunder (za magiju)

- **Objašnjenje**: Specijalne putanje koje počinju i završavaju sa dvostrukim donjim podvučnicama (npr., `__init__`, `__len__`, `__str__`, `__eq__`). Python ih koristi unutar za ugrađene operacije.

```python
class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __len__(self):
        return self.pages

    def __str__(self):
        return f"'{self.title}' has {self.pages} pages"

    def __eq__(self, other):
        return self.pages == other.pages

book1 = Book('Built Wealth Like a Boss', 420)
print(len(book1))        # 420
print(str(book1))        # 'Built Wealth Like a Boss' has 420 pages
```

- **Pozivanje dunder metoda indirektno**: Nije potrebno pozivati dunder metode direktno. Umesto toga, Python ih poziva automatski kada se izvrše određene akcije. Ove operacije uključuju:

- **Matematičke operacije kao što su sabiranje, oduzimanje, množenje, deljenje i slično**. Pored toga, `__add__()` se zove, `__sub__()` za oduzimanje, `__mul__()` za množenje i `__truediv__()` za deljenje.
        
  - **Operacije sa nizovima karaktera kao što su konkatenacija, ponavljanje, kreiranje šablona i konverzija u tekst**. `__add__()` se zove za spajanje, `__mul__()` za ponavljanje, `__format__()` za kreiranje šablona, `__str__()` i `__repr__()` za konverziju u tekst, itd.
        
  - **Operacije poređenja kao što su jednako, manje od, veće od i slično**. `__eq__()` se zove za proveru jednakosti, `__lt__()` za manje od, `__gt__()` za veće od, itd.
        
  - **Operacije iteracije kao što je činjenje nečega ponovljivim i pomeranje kroz elemente**. `__iter__()` se zove vraćanjem iteratora, a `__next__()` dobijanjem sledećeg elementa.

## Stvarni svetski primer: Kupovni automobil

- **Klasa vozila sa dunder metodama**: Omogućava dodavanje, uklanjanje, iteriranje i proveru sadržaja sa ugrađenim ponašanjem.

```python
class Cart:
    def __init__(self):
        self.items = []

    def add(self, item):
        self.items.append(item)

    def remove(self, item):
        if item in self.items:
            self.items.remove(item)
        else:
            print(f'{item} is not in cart')

    def list_items(self):
        return self.items

    def __len__(self):
        return len(self.items)

    def __getitem__(self, index):
        return self.items[index]

    def __contains__(self, item):
        return item in self.items

    def __iter__(self):
        return iter(self.items)

cart = Cart()
cart.add('Laptop')
print(len(cart))        # 1
print('Laptop' in cart) # True
```
