# --description--

Petlja `for` je drugi tip petlje koji se koristi za izvršavanje bloka koda više puta. Petlja `for` se koristi kada je poznat broj iteracija. Sintaksa petlje `for` je sledeća:

```javascript
for (initialization; condition; increment/decrement) {
  // code block to be executed
}
```

Na primer, deo koda ispod ispisuje brojeve od 1 do 5:

```javascript
for (let i = 1; i <= 5; i++) {
  console.log(i);
}
```

Da biste iterirali preko niza podataka koristeći petlju ``for``, možete koristiti svojstvo dužine tog niza. Na primer, kodni isječak ispod štampa elemente niza podataka:

```javascript
const arr = [1, 2, 3, 4, 5];
for (let i = 0; i < arr.length; i++) {
  console.log(arr[i]);
}
```
