# --interactive--

U prethodnim lekcijama ukratko smo govorili o tome kako da napraviš razmake između stavki mreže. U ovoj lekciji ćemo detaljnije objasniti kako se koriste svojstva `row-gap`, `column-gap` i `gap` u CSS Grid rasporedu.

Ali prvo moramo proveriti šta znači linija u gridu CSS. Linija je prostor između dve susedne linije grida. Ove linije se automatski kreiraju kada koristite CSS Grid. U ovom kontekstu, linije obično se odnose na redove i kolone koje čine raspored grida.

Da bi napravio razmake između kolona u CSS Grid-u, možeš koristiti svojstvo `column-gap`. Prihvatljive vrednosti uključuju piksele, jedinicu `em`, procente ili ključnu reč `normal`.

Ako za svojstvo `column-gap` koristiš vrednost `normal`, rezultat će biti `0` za Grid rasporede i `1em` za višekolonske rasporede.

Evo primera oznaka za raspored mreže sa četiri kolone:

```html
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

Za CSS, postavili smo display na mrežu i svojstvo `column-gap` na 10 piksela:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  height: 100px;
  grid-template-columns: 1fr 1fr 1fr 1fr;
  column-gap: 10px;
}

.grid-container div {
  background-color: darkblue;
}
```

:::

Ako bismo želeli da izmenimo ovaj primer tako da ima dva reda plavih okvira i napravimo više prostora između tih redova, mogli bismo da koristimo svojstvo `row-gap`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  column-gap: 10px;
  row-gap: 30px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

U ovom izmenjenom primeru postavljamo svojstvo `row-gap` na 30 piksela i menjamo `grid-template-columns` da koristi dve jedinice `1fr` umesto četiri, kako bismo napravili dva reda okvira.

Kao i kod svojstva `column-gap`, prihvatljive vrednosti za `row-gap` uključuju procente, `em` i piksele.

Ako želite da koristite svojstvo skraćivanja kako biste kreirali praznine između redova i kolona, možete koristiti svojstvo `gap`. Ovo je osnovna sintaksa:

```css
gap: row-value optional-column-value;
```

Ako navedeš jednu vrednost za svojstvo `gap`, ona će se primeniti i na redove i na kolone. Ako navedeš dve vrednosti, prva će se primeniti na redove, a druga na kolone:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div></div>
  <div></div>
  <div></div>
  <div></div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 30px 10px;
}

.grid-container div {
  height: 100px;
  background-color: darkblue;
}
```

:::

Prihvatljive vrednosti za skraćeno svojstvo `gap` uključuju procente, piksele, `em`, pa čak i vrednosti `calc()`. Međutim, ovde ne možeš koristiti jedinice `fr`.

Svojstva za `row-gap`, `column-gap`, na `gap` pružaju fleksibilne načine kontrole prostora između elemenata u rasporedu CSS Grid. Korišćenjem ovih svojstava, lako možete kreirati mreže sa vizuelnom privlačnošću i prilagodljivim razmakom između redova i kolona.
