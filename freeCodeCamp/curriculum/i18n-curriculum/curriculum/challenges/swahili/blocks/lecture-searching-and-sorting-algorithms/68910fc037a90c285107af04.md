# --description--

Metoda "podeljivanja i osvajanja" u računarskoj nauci je tehnika razbijanja problema na manje probleme korišćenjem rekurzivnog procesa. Jedna od ključnih komponenti ove metode je rekurzija, koja nastaje kada funkcija poziva samu sebe više puta dok ne dođe do osnovnog slučaja (base case). U ovom lekciji, pregledaćemo Merge Sort algoritam kako bismo bolje razumeli kako metoda "podeljivanja i osvajanja" funkcioniše.

Recimo da imamo ovu listu brojeva:

```md
42 37 53 17
```

Cilj je da se sortira ta lista od manjeg ka većem koristeći Merge Sort algoritam. Prvi korak je podeliti tu listu na pola:

```md
42 37 | 53 17
```

Zatim moramo pogledati levu stranu liste:

```md
42 37
```

Uzimamo taj manji spisak i delimo ga na pola dok svaki manji spisak ne sadrži samo jedan element unutar sebe:

```md
42 | 37
```

Lista koja ima samo jedan element je podešena po podrazumevanju. Sledeći korak je da spojimo svaku malu listu sa jednim elementom u organizovanu listu:

```md
37 42
```

Zatim sledimo isti proces sa desne strane početne liste:

```py
# right side of original list
53 17

# divide the list in half
53 | 17

# merge the lists in sorted order
17 53
```

Budući da su oba dela inicijalne liste obrađene, spajamo ta dva dela i sortiramo elemente:

```markdown
17 37 42 53
```

Ovo je kako algoritam izgleda u kodu:

```py
def merge_sort(arr):
    if len(arr) <= 1:
        return arr

    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])

    sorted_list = []
    i = 0
    j = 0

    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            sorted_list.append(left[i])
            i += 1
        else:
            sorted_list.append(right[j])
            j += 1

    sorted_list.extend(left[i:])
    sorted_list.extend(right[j:])

    return sorted_list
```

Vremenska složenost za algoritma merge sort će biti `O(n log n)` jer se lista stalno deli na pola (`log n`) a zatim spaja (`O(n)`). Za razliku od drugih algoritama za sortiranje kao što je bubble sort, merge sort ne radi na mestu i ima prostornu složenost `O(n)`.
