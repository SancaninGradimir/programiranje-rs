# --interactive--

## Značaj semantičkog HTML

- **Redosled strukture za elemente naslova sekcije**: Važno je koristiti odgovarajući element naslova sekcije kako bi se održao kontinuitet strukture sadržaja. Element `h1` predstavlja najviši nivo naslova sekcije, dok element `h6` predstavlja najniži nivo naslova sekcije.
- **Elementi HTML za prikaz izgleda**: Elementi koji opisuju izgled sadržaja. Primer: elementi `center`, `big` i `font` koji više nisu korišćeni.
- **Semantički elementi HTML**: Elementi koji nose značenje i strukturu. Primer: `header`, `nav`, `figure`.

## Semantički elementi HTML

- **Zaglavni element (header)**: Koristi se za definisanje naslova dokumenta ili sekcije.

:::interactive_editor

  ```html
  <header>
    <h1>CatPhotoApp</h1>
    <p>Karibu kwenye maktaba yetu ya picha za paka.</p>
  </header>
  ```

:::

- **Glavni element (main)**: koristi se za uključivanje glavnog sadržaja veb stranice.

:::interactive_editor

  ```html
  <main>
    <section>
      <h2>Picha za paka</h2>
      <p>Vinakagua picha za paka warembo.</p>
    </section>
  </main>
  ```

:::

- **Sekciono polje (section)**: Koristi se za podelu sadržaja na manje sekcije/delove.

:::interactive_editor

  ```html
  <section>
    <h2>Kuhusu mimi</h2>
    <p>Habari, mimi ni Jane Doe na mimi ni msanidi programu wa mtandao.</p>
  </section>
  ```

:::

- **Navigacioni deo (`nav`)**: predstavlja sekciju sa navigacionim linkovima.

:::interactive_editor

  ```html
  <nav>
    <ul>
      <li><a href="#photos">Picha</a></li>
      <li><a href="#videos">Video</a></li>
    </ul>
  </nav>
  ```

:::

- **Element figure (slika)**: koristi se za povezivanje crteža i ilustracija.

:::interactive_editor

  ```html
  <figure>
    <img
      src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg"
      alt="Paka wawili wa tabby wamelala pamoja kwenye kochi."
    />
    <figcaption>Paka <strong>hawapendi</strong> paka wengine.</figcaption>
  </figure>
  ```

:::

- **Atribut naglaska (`em`)**: identifikuje tekst sa naglašavanjem stresa.

:::interactive_editor

  ```html
  <p>
    Usikate tamaa kamwe juu ya ndoto <em>zako</em>.
  </p>
  ```

:::

- **Ideometrički element teksta (`i`)**: Koristi se za prikaz alternativnih zvukova ili stanja, ideomatskih reči iz drugog jezika, tehničkih termina i ideja.

:::interactive_editor

  ```html
  <p>
    Kuna aina fulani ya <i lang="fr">je ne sais quoi</i> hewani.
  </p>
  ```

:::

Atribut `lang` unutar otvarajuće etikete `i` koristi se za određivanje jezika sadržaja. U ovom slučaju, jezik će biti francuski. Element `i` ne pokazuje da li je tekst važan ili ne; on samo pokazuje da je na neki način drugačiji od okružujućeg teksta.

- **Značajna karakteristika (`strong`)**: identifikuje tekstove sa velikom važnosti.

:::interactive_editor

  ```html
  <p>
    <strong>Onyo:</strong> Bidhaa hii inaweza kusababisha athari za mzio.
  </p>
  ```

:::

- **Funkcija za naglašavanje (`b`)**: Koristi se za isticanje teksta koji nije ključan za značenje sadržaja. Često se koristi za prikazivanje važnih reči u sažetku ili naziva proizvoda u recenziji.

:::interactive_editor

  ```html
  <p>
    Tulijaribu bidhaa kadhaa, ikiwa ni pamoja na <b>SuperSound 3000</b> kwa ubora wa sauti, <b>QuickCharge Pro</b> kwa kuchaji haraka, na <b>Ecoclean Vacuum</b> kwa usafi. Mbili za kwanza zilifanya vizuri, lakini <b>Ecoclean Vacuum</b> haikukidhi matarajio.
  </p>
  ```

:::

- **Element liste opisa (`dl`)**: koristi se za predstavljanje liste grupa reči-opisa.
- **Element definicionog reči (`dt`)**: koristi se za predstavljanje definisane reči.
- **Element opisa opisa (`dd`)**: koristi se za predstavljanje opisa reči.

:::interactive_editor

  ```html
  <dl>
    <dt>HTML</dt>
    <dd>Lugha ya alama ya HyperText</dd>
    <dt>CSS</dt>
    <dd>Majedwali ya mtindo yanayopangwa</dd>
  </dl>
  ```

:::

- **Element bloka citata (`blockquote`)**: Koristi se za predstavljanje sekcije preuzete iz drugog izvora. Ovaj element ima atribut `cite`. Vrednost atributa `cite` je URL izvora.

:::interactive_editor

  ```html
  <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
    "Je, unaweza kufikiria itakuwaje kuwa msanidi programu mwenye mafanikio? Kuwa umejenga mifumo ya programu ambayo watu wanategemea?"
  </blockquote>
  ```

:::

- **Referentni element (`cite`)**: Koristi se za prikaz izvora citiranog rada na vizuelan način. Postavlja oznaku naslova referenca.

:::interactive_editor

  ```html
  <div>
    <blockquote cite="https://www.freecodecamp.org/news/learn-to-code-book/">
      "Je, unaweza kufikiria itakuwaje kuwa msanidi programu mwenye mafanikio? Kuwa umejenga mifumo ya programu ambayo watu wanategemea?"
    </blockquote>
    <p>
      -Quincy Larson, <cite>Jinsi ya kujifunza kuandika msimbo na kupata kazi ya msanidi programu [Kitabu kamili].</cite>
    </p>
  </div>
  ```

:::

- **Element za uvodne citate unutar linije (`q`)**: koristi se za predstavljanje kratkog cita unutar linije.

:::interactive_editor

  ```html
  <p>
    Kama Quincy Larson alivyosema,
    <q cite="https://www.freecodecamp.org/news/learn-to-code-book/">
      Mwelekeo ni kila kitu.
    </q>
  </p>
  ```

:::

- **Funkcija skraćenice (`abbr`)**: Koristi se za predstavljanje skraćenica ili akronima. Da bi se korisnicima pomoglo da razumeju skraćenicu ili akronim, možete prikazati punu formu, opis čitljiv ljudima, koristeći atribut `title`.

:::interactive_editor

  ```html
  <p>
    <abbr title="HyperText Markup Language">HTML</abbr> ni msingi wa mtandao.
  </p>
  ```

:::

- **Polje za kontakt adresu (`address`)**: Koristi se za predstavljanje kontaktnih informacija.
- **Polje za datum i/ili vreme (`time`)**: Koristi se za predstavljanje datuma i/ili vremena. Atribut `datetime` koristi se za interpretaciju datuma i vremena u mašinski čitljiv format.

:::interactive_editor

  ```html
  <p>
    Hifadhi zimepangwa kwa saa <time datetime="20:00">20:00 </time>
  </p>
  ```

:::

- **Atribut datuma ISO 8601 (`datetime`)**: koristi se za predstavljanje datuma i vremena u mašinski čitljivom formatu. Standardni format je `YYYY-MM-DDThh:mm:ss`, gde je veliki `T` separator između datuma i vremena.
- **Element teksta sa nadizpisanim indeksima (superscript) (`sup`)**: koristi se za predstavljanje teksta sa nadizpisanim indeksima. Uobičajene upotrebe elemenata `sup` uključuju eksponente, velika slova i sekvencijalne brojeve.

:::interactive_editor

  ```html
  <p>
    2<sup>2</sup> (2 kwa mraba) ni 4.
  </p>
  ```

:::

- **Funkcija podskripte (subscript) (`sub`)**: koristi se za predstavljanje teksta sa manjim indeksima/podskriptom. Uobičajene upotrebe ove karakteristike uključuju hemijske formule, niže podskripte i parametričke oznake.

:::interactive_editor

  ```html
  <p>
    CO<sub>2</sub>
  </p>
  ```

:::

- **Element ugrađenog koda na liniji (`code`)**: koristi se za predstavljanje dela računarskog koda.
- **Element za predformatirani tekst (`pre`)**: predstavlja predformatirani tekst

:::interactive_editor

  ```html
  <pre>
    <code>
      body {
        color: red;
      }
    </code>
  </pre>
  ```

:::

- **Atribut nejasnog opisa (`u`)**: koristi se za predstavljanje dela teksta unutar linije koji treba prikazati na način koji ukazuje da sadrži vantekstualne informacije.

:::interactive_editor

  ```html
  <p>
    Unaweza kutumia kipengele cha maelezo yasiyoeleweka kuonyesha
    <u>makosa</u> <u>ya uandishi</u> <u>ya maneno</u>.
  </p>
  ```

:::

- **Kipengele cha maelezo ya ruby (`ruby`)**: koristi se za detalje izgovora ili objašnjenje značenja teksta. Primer upotrebe je za tipove istočnjačkih znakova Azije.
- **Element alternativnog prikaza ruby zagrada (`rp`)**: koristi se kao alternativa u pregledačima koji ne podržavaju prikaz ruby anotacija.
- **Kipengele cha maandishi ya ruby (`rt`)**: koristi se za prikaz znakova radi ruby detalja. Često se koristi za izgovor ili objašnjenje prevoda u tipovima istočnjačkih znakova Azije.

:::interactive_editor

  ```html
  <ruby>
    明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp>
  </ruby>
  ```

:::

- **Funkcija prečicanja linijom (strikethrough) (`s`)**: Koristi se za predstavljanje sadržaja koji više nije tačan ili je nevažan.

:::interactive_editor

  ```html
  <p>
    <s>Matembezi ya kesho yatakutana saa sita mchana.</s>
  </p>
  <p>
    Kutokana na hali ya hewa isiyotabirika, matembezi yamefutwa.
  </p>
  ```

:::

- **Interni linkovi**: Koriste se za povezivanje drugog dela stranice korišćenjem `href="#id"` na elementu `a` i davanju mu ciljnog elementa `id`. Ovo se često koristi za anchor linkove, sadržajne spiskove ili duge stranice sa više sekcija.

:::interactive_editor

```html
<a href="#about-section">Go to About Section</a>

<section id="about-section">
  <h2>About</h2>
  <p>This is the about section of the page.</p>
</section>
```

:::
