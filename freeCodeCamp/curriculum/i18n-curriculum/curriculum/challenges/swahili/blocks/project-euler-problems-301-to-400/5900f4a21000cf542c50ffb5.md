# --description--

Ajlisa i Bob igraju igru Nim-a sa kvadratnim brojevima.

Nim sa kvadratnim brojevima je kao standardna igra Nima sa tri gomile i uobičajenim pravilima, ali igrači mogu da uklone kamenje iz grmlja samo brojem koji je kvadratni broj.

Broj kamenja u tri stoga je predstavljen nizom ($a$, $b$, $c$).

Ako $0 ≤ a ≤ b ≤ c ≤ 29$ onda broj izgubljenih pozicija za sledećeg igrača je 1160.

Izračunajte broj izgubljenih pozicija za sledećeg igrača ako je $0 ≤ a ≤ b ≤ c ≤ 100\,000$.
