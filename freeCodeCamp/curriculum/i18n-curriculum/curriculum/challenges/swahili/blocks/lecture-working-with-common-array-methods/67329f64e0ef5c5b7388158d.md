# --interactive--

U JavaScript, putanja putem `indexOf()` je važna za dobijanje prvog primerka specifičnog elementa unutar niza podataka. Ako element nije pronađen, onda će vratiti `-1`. Ovo je osnovna sintaksa:

```js
array.indexOf(element, fromIndex)
```

`element` predstavlja vrednost koju želite da pretražujete u nizu podataka, a parametar za `fromIndex` je lokacija od koje ćete početi pretraživanju. Parametar za `fromIndex` je opcionalan. Ako nije navedeno `fromIndex`, pretraživanje počinje od početka niza podataka. Uzmemo primer:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange", "banana"];
let index = fruits.indexOf("banana");
console.log(index); // 1
```

:::

U ovom primeru, imamo podatkovnu listu `fruits` sa različitim imenima voća. Koristimo metodu `indexOf()` da bismo dobili obrazac sekvence karaktera `banana` unutar podatkovne liste `fruits`. Pošto `banana` je u obrascu `1`, ta metoda vraća `1`, koja se čuva u promenljivoj `index` i piše na konzolu.

Ako element koji tražite nije pronađen u redovima podataka, `indexOf()` vraća `-1`. Primer:

:::interactive_editor

```js
let fruits = ["apple", "banana", "orange"];
let index = fruits.indexOf("grape");
console.log(index); // -1
```

:::

Ovde tražimo niz znakova `grape` u nizu voća koristeći `indexOf()`. Pošto `grape` nije u nizu podataka, ta metoda vraća `-1`, koji je sačuvan u varijanti `index` i zapisuje se u konzolu.

Ako želite da započnete pretraživanje elementa nakon specifičnog broja indeksa, onda možete proslediti drugi argument kao u ovom primeru:

:::interactive_editor

```js
let colors = ["red", "green", "blue", "yellow", "green"];
let index = colors.indexOf("green", 3);
console.log(index); // 4
```

:::

U ovom primeru, pretraga ne počinje od početka redova podataka; umesto toga počinje od modela broja `3` koji je `yellow` i dobija rezultat od `4`.
