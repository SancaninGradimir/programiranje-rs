# --interactive--

Tip `Number` je jedan od tipova podataka koji se često koristi u JavaScript-u i drugim programskim jezicima. Brojevi mogu delovati jednostavno, ali postoji mnogo toga za istraživanje kada je reč o radu sa brojevima u JavaScript-u. Zato hajde da ih detaljnije upoznamo. U JavaScript-u tip podataka `Number` predstavlja numeričku vrednost.

Za razliku od mnogih programskih jezika koji razlikuju cele brojeve i decimalne brojeve u različite tipove, JavaScript koristi jedan tip od `Number` za predstavljanje brojeva. To znači da možete raditi sa celim brojevima, decimalnim brojevima i čak posebnim vrednostima svih brojeva unutar jednog tipa podataka od `Number`.

Ovo je osnovni primer koji pokazuje da su svi cijeli brojevi, decimalni brojevi i negativni brojevi tipa broj:

:::interactive_editor

```js
const wholeNumber = 50;
const decimalNumber = 4.5;
const negativeNumber = -7;

console.log(typeof wholeNumber); // number
console.log(typeof decimalNumber); // number
console.log(typeof negativeNumber); // number
```

:::

Tip/Vrsta `Number` od JavaScript obuhvata različite tipove numeričkih vrednosti, počevši od jednostavnih celih brojeva i decimalnih brojeva sve do posebnih slučajeva kao što su `Infinity` i `NaN`, ili "Not a Number". Analizirajmo glavne tipove koje ćete naići. Celi brojevi su brojevi bez decimalnih ili frakcionih delova. Mogu biti pozitivni, negativni ili nuli. Evo primera:

:::interactive_editor

```js
const positiveInteger = 100;
const negativeInteger = -25;
const zero = 0;

console.log(typeof positiveInteger); // number
console.log(typeof negativeInteger); // number
console.log(typeof zero); // number
```

:::

Decimalni brojevi su brojevi koji imaju decimalnu tačku. Često su poznati kao "floats" kod programera JavaScript. Floati su važni kada vam je potrebna veća preciznost, kao što je kod merenja ili valute. Evo primera:

:::interactive_editor

```js
const floatingPointNumber = 4.5;
const anotherFloat = 89.56;
const oneMoreFloat = 16.462;

console.log(typeof floatingPointNumber); // number
console.log(typeof anotherFloat); // number
console.log(typeof oneMoreFloat); // number
```

:::

JavaScript može predstavljati brojeve koji su izvan gornjeg limita korišćenjem `Infinity`. Ovo ćete naići kada pokušavate da podelite broj sa nulom ili u retkim slučajevima, preći gornji limit tipa `Number`. Evo primera:

:::interactive_editor

```js
const infiniteNumber = 1 / 0;
console.log(infiniteNumber); // Infinity
console.log(typeof infiniteNumber); // number
```

:::

Ponekad u JavaScript, neke aritmetičke operacije ne proizvode validan broj. Na primer, ako pokušate da izvršite aritmetičku operaciju na nečemu što nije broj, dobićete `NaN`, što označava "Not a Number":

:::interactive_editor

```js
const notANumber = 'hello world' / 2;
console.log(notANumber); // NaN
```

:::

Iznenadno, vrsta od `NaN` takođe je `Number`:

:::interactive_editor

```js
const notANumber = 'hello world' / 2;
console.log(typeof notANumber); // number
```

:::

Pored standardnog decimalnog sistema (base 10), JavaScript takođe podržava brojeve u različitim bazama kao što su binarni, oktalni i heksadecimalni. Binarni je sistem baze-2 koji koristi samo cifre 1 i 0. Oktalni je sistem baze-8 koji koristi samo cifre od 0 do 7. Heksaadecimalni je sistem baze-16 koji koristi cifre od 0 do 9 i slova od a do f, kao što vidite u heksadecimalnim bojama za CSS.
