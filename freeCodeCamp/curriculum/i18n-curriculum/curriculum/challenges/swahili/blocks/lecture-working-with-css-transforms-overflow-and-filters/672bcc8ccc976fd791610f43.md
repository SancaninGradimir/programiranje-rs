# --interactive--

Funkcija CSS `transform` je moćan alat koji vam omogućava da promenite izgled elemenata na vašoj veb stranici bez uticaja na raspored ostalih elemenata. Omogućava vam da koristite različite transformacije za elemente kao što su rotiranje, skaliranje, savijanje ili pomeranje (translatacija) u 2D ili 3D prostoru.

Svojstvo `transform` funkcioniše korišćenjem matematičkih transformacija na koordinatnom sistemu objekta. To znači da možete kontrolisati oblik i poziciju objekta, istovremeno čuvajući njegovo originalno mesto i tok dokumenta.

Hajde da pogledamo neke od najčešće korišćenih funkcija `transform`. Evo primera elementa u obliku kutije:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
body {
  border: 2px solid black;
}

.box {
  width: 200px;
  height: 200px;
  background-color: red;
}
```

:::

Postavili smo `body` sa čvrstom crnom ivicom kako biste videli element `.box` koji je postavljen unutar elementa `body`.

Funkcija `translate` pomera element sa njegove trenutne pozicije. Evo ažuriranog primera koji koristi funkciju `translate`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
body {
  border: 2px solid black;
}

.box {
  width: 200px;
  height: 200px;
  background-color: red;
  transform: translate(50px, 100px);
}
```

:::

Ovo CSS pravilo će pomeriti element sa klasom `box` za 50 piksela udesno i 100 piksela nadole u odnosu na njegovu početnu poziciju.

Funkcija `rotate` rotira element oko određene tačke. Evo primera korišćenja funkcije `rotate` na elementu `.box` iz prethodnog primera:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: rotate(45deg);
}
```

:::

Ovo će rotirati element za četrdeset pet stepeni u smeru kazaljke na satu.

Funkcija `scale` omogućava promenu veličine elementa. Evo primera:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: scale(1.5, 2);
}
```

:::

Ovo će učiniti element jedan i po put širim i dva puta višim od njegove originalne veličine.

Možete spojiti mnogo promena u jednu izjavu:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="box"></div>
```

```css
.box {
  margin: 100px;
  width: 200px;
  height: 200px;
  background-color: red;
  transform: translate(50px, 50px) rotate(45deg) scale(1.5);
}
```

:::

Ovo će premestiti nešto za 50 piksela udesno i dole, rotirajući ga za 45 stepeni, i skalirati ga na jednač pol veličine svog originalnog razmera.

Iako je funkcija `transform` moćan alat za kreiranje vizuelno privlačnih dizajna, važno je razmotriti pristupačnost kada ga koristite. Evo važnih stvari o pristupačnosti koje treba uzeti u obzir.

Čitači ekrana možda neće moći tačno da opišu izmenjen sadržaj. Na primer, kada koristite `transform` za vizuelno preuređivanje rasporeda stavki, čitači ekrana će i dalje čitati sadržaj u prvobitnom redosledu DOM. Ovo može izazvati konfuziju kod korisnika koji zavise od čitača ekrana.

Kada koristiš `scale` za menjanje veličine teksta, budi pašu da ga ne učini previše malim ili previše velikim. Previše mali tekst može biti težak za čitanje, dok preveliki tekst može da izađe iz kontejnera i da bude težak za čitanje. Generalno, bolje je koristiti pravilne metode fonta prilikom menjanja veličine teksta.

Kada koristiš `transform` za animacije, obrati pažnju na korisnike koji pate od bolesti pokreta. Previše ili prebrzo animacije mogu izazvati nelagodu, pa čak i šok kod nekih ljudi. Razmisli o pružanju korisnicima mogućnosti da smanje ili isključe animacije. Kada koristiš 3D promene, zapamti da ne svi korisnici vide dubinu na isti način. Osiguraj da je važna informacija koja se prenosi putem 3D efekata dostupna i u 2D formatu ili putem teksta.

Kada koristite `transform` za skrivanje ili prikazivanje sadržaja, uverite se da je sadržaj i dalje dostupan za čitače ekrana i navigaciju pomoću tastature. Skriveni sadržaj treba da bude zaista sakriven, kao što je korišćenje `display: none;` ili `visibility: hidden;`, umesto samo pomeranja sa ekrana vizuelno.

Kada koristite `transform` za stvari koje se mogu kliknuti kao dugmad ili linkovi, osigurajte da oblast za klikanje ostane razumljiva i laka za ciljanje. Previše modifikovana dugmad mogu biti vizuelno zbunjujuća ili teška za klikanje, posebno za korisnike sa motoričkim poteškoćama.

Zaključno, iako je funkcija CSS `transform` moćan alat za kreiranje vizuelno privlačnih mrežnih dizajna, važno je koristiti je odgovorno i uzeti u obzir Pristupačnost. Uvek testirajte svoje modifikovane stvari sa različitim asistivnim tehnologijama i razmislite o pružanju alternativnih načina pristupa informacijama ili funkcionalnosti koji mogu biti pogođeni promenama.
