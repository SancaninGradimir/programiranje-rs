# --instructions--
Napravi funkciju nazvanu `calculate()` u `mean_var_std.py` koja koristi Numpy za izračunavanje proseka, varijacije, standardne devijacije, maksimalne vrednosti, minimalne vrednosti i sume redova, kolona i elemenata u 3x3 matrici.

Ulaz funkcije treba da bude lista sa 9 brojeva. Funkcija bi trebalo da konvertuje tu listu u NumPy niz podataka dimenzija 3x3, a zatim da vrati rečnik koji sadrži prosečnu vrednost, varijansu, standardnu devijaciju, maksimalnu vrednost, minimalnu vrednost i zbir za obe strane i za transponovanu matricu.

Vraćeni rečnik treba da ima sledeću strukturu:

```py
{
  'mean': [axis1, axis2, flattened],
  'variance': [axis1, axis2, flattened],
  'standard deviation': [axis1, axis2, flattened],
  'max': [axis1, axis2, flattened],
  'min': [axis1, axis2, flattened],
  'sum': [axis1, axis2, flattened]
}
```

Ako se list sa manje od 9 elemenata prosledi funkciji, treba da izbaci grešku `ValueError` sa porukom: "List must contain nine numbers." Vrednost u vraćenom sözleru treba da bude lista, a ne Numpy niz podataka.

Na primer, `calculate([0,1,2,3,4,5,6,7,8])` treba da vrati:

```py
{
  'mean': [[3.0, 4.0, 5.0], [1.0, 4.0, 7.0], 4.0],
  'variance': [[6.0, 6.0, 6.0], [0.6666666666666666, 0.6666666666666666, 0.6666666666666666], 6.666666666666667],
  'standard deviation': [[2.449489742783178, 2.449489742783178, 2.449489742783178], [0.816496580927726, 0.816496580927726, 0.816496580927726], 2.581988897471611],
  'max': [[6, 7, 8], [2, 5, 8], 8],
  'min': [[0, 1, 2], [0, 3, 6], 0],
  'sum': [[9, 12, 15], [3, 12, 21], 36]
}
```
