# --interactive--

Uslovne izjave vam omogućavaju donošenje odluka u vašem kodu od JavaScript. One omogućavaju vašoj aplikaciji da nastavi na određeni način u zavisnosti od specifičnih uslova. Pogledajmo kako `if`, `else if`, `else` i ternarni operator rade kako bi vam omogućili kontrolu toka vašeg koda.

Izjava `if` uzima određeno stanje i izvršava blok koda ako je to stanje tačno. Logičke vrednosti su bilo koja vrednost koja rezultira `true` kada se proceni u kontekstu istinite ili nistinite vrednosti, kao što je u izjavi `if`. Evo primera istinitih vrednosti:

- nepristopna sekvenca znakova, na primer, `hello`

- bilo koji broj osim `0` na `-0`, na primer, `4`, `-5`, i slično

- redovi podataka

- stvari

- Istina ili laž `true`

S druge strane, lažne vrednosti su vrednosti koje se smatraju `false` u kontekstu istinite ili nistinite vrednosti. JavaScript ima malo lažnih vrednosti, što ih čini lakim za pamćenje. Evo nekoliko lažnih vrednosti:

- Vrednost istine ili laži `false`

- `0` (nula)

- `""` (niz praznih karaktera)

- `null`

- `undefined`

- `NaN` (To nije broj)

Sada, imajući osnovno razumevanje istinitih i lažnih vrednosti, istražimo kako funkcioniše sa podacima iz `if`. U ovom prvom primeru, koristimo nekoliko podataka iz `if` za proveru naspram istinitih i lažnih vrednosti:

:::interactive_editor

```js
if (null) {
  console.log("This will not run.");
}

if ("freeCodeCamp") {
  console.log("This will run.");
}
```

:::

Pošto `null` je lažna vrednost, poruka unutar bloka neće biti ispisana na konzolu. Ali za drugu izjavu od `if`, niz znakova `freeCodeCamp` je istinita vrednost, i smatraće se `true` u ovom kontekstu istinite vrednosti izjave od `if`. Posledično, poruka `This will run.` će biti ispisana na konzolu.

Hajde da istražimo još nekoliko primera kako podaci iz `if` funkcionišu sa različitim operaterima poređenja. Evo primera korišćenja podataka iz `if` za proveru da li je korisnik kvalifikovan za glasanje:

:::interactive_editor

```js
const age = 22;

if (age >= 18) {
 console.log("You're eligible to vote"); // You're eligible to vote
}
```

:::

U ovom primeru, pošto `age` je trenutno `22`, to znači da će uslov biti evaluiran na `true` jer `22` je veće ili jednako od `18`. Stoga će poruka `You're eligible to vote` biti ispisana u konzolu. Ako promenimo primer tako da `age` sada bude `15`, onda će uslov biti evaluiran na `false` i poruka neće biti ispisana u konzolu:

:::interactive_editor

```js
const age = 15;

if (age >= 18) {
 console.log("You're eligible to vote"); // Code not running because age is less than 18
}
```

:::

Kada je stanje `false`, onda možete koristiti klauzulu `else`:

:::interactive_editor

```js
const age = 15;

if (age >= 18) {
 console.log("You're eligible to vote");
} else {
 console.log("You're not eligible to vote"); // You're not eligible to vote
}
```

:::

U ovom primeru, `15` nije veće od ili je jednako `18`, stoga će uslov biti `false`. Kod lociran unutar sekcije `else` biće izvršen u ovom slučaju.

Ako želite da pregledate mnogo uslova, možete koristiti sekciju za `else if`. Ovo omogućava vašem programu da bira između više od dve metode.

:::interactive_editor

```js
const score = 87;

if (score >= 90) {
 console.log('You got an A'); 
} else if (score >= 80) {
 console.log('You got a B'); // You got a B
} else if (score >= 70) {
 console.log('You got a C');
} else {
 console.log('You failed! You need to study more!');
}
```

:::

Pošto `score` je trenutno `87`, onda poruka za `You got a B` biće napisana na konzoli.

Ternarni operator je kratak način za pisanje jednostavnih izjava za `if/else`. Sastoji se od tri dela: uslov, rezultat ako je uslov tačan, i rezultat ako nije istinit. Evo osnovne sintakse:

```js
condition ? expressionIfTrue : expressionIfFalse;
```

Evo primera koji se odnosi na temperaturu vazduha u Celzijusima:

:::interactive_editor

```js
const temperature = 30;
const weather = temperature > 25 ? 'sunny' : 'cool';

console.log(`It's a ${weather} day!`);
```

:::

Ako je `temperature` veće od `25`, gornji kod će napisati `It's a sunny day!`. Ako je `temperature` manje ili jednako `25`, napisati će `It's a cool day!`.

Koji treba da koristiš između deklaracije if i ternarnog operatora? Koristi ternarni operator kada koristiš jednu uslovnu granku ili jedan izraz, ili kada želiš kratku sintaksu za jednostavnu logiku. Koristi deklaracije `if/else` kada koristiš složene uslove i više deklaracija, jer stvari postaju teške za čitanje ako ih raspoređuješ unutar ternarnih operatora.
