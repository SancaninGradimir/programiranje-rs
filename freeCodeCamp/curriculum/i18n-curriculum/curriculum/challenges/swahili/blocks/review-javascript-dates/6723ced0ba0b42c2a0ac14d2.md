## Uređaj za `date()` na Standardnoj ruti

- **Definicija**: Objekat `date()` se koristi za kreiranje, kontrolisanje i formatiranje datuma i vremena u JavaScript. U sledećem primeru, glavna reč `new` koristi se za kreiranje nove instance objekta `Date`, a objekat `Date` je zatim dodeljen vrednost promenljive `now`. Ako unesete vrednost `now` u konzolu, videćete trenutni datum i vreme prema sistemovom satu računara koji izvršava taj kod.

:::interactive_editor

```js
const now = new Date();
console.log(now); // prints the current date and time
```

:::

- **Funkcija `Date.now()`**: Ova funkcija se koristi za dobijanje trenutnog datuma i vremena. `Date.now()` vraća broj milisekundi od 1. januara 1970, 00:00:00 UTC. Ovo je poznato kao Unix epoha vreme. Unix epoha vreme je standardan način za predstavljanje datuma i vremena u računarskim sistemima jer je celobrojni broj koji se lako može sačuvati i kontrolisati. UTC stoji za Universal Coordinated Time, što je globalni standard vremena koji se koristi širom sveta za upravljanje satima i vremenom.
- **Funkcija `getDate()`**: Ova funkcija se koristi za dobijanje dana meseca na osnovu trenutnog datuma. `getDate()` će vratiti vrednost celog broja između 1 i 31, u zavisnosti od dana meseca. Ako datum nije validan, vraća `NaN` (Nije broj).

:::interactive_editor

```js
const now = new Date("2014-10-15");
const date = now.getDate();
console.log(date); // 15
```

:::

- **Metoda za `getMonth()`**: Ova metoda se koristi za dobijanje meseca. Meseci počinju od nule, tako da je januar 0, februar 1, i tako dalje. U ovom primeru, rezultat je 9, što odgovara oktobru. Ako mesec nije validan, vratiće `NaN`.

:::interactive_editor

```js
const now = new Date("2014-10-15");
const month = now.getMonth();
console.log(month); // 9
```

:::

- **Putanja za `getFullYear()`**: Ova putanja se koristi za dobijanje pune godine. Ako godina nije validna, vraćaće `NaN`.

:::interactive_editor

```js
const now = new Date("2014-10-15");
const year = now.getFullYear();
console.log(year); // 2014
```

:::

## Različiti načini za kreiranje strukture datuma

- **Put za `toISOString()`**: Ovaj put se koristi za kreiranje formata datuma u proširenom formatu `ISO` (ISO 8601). ISO 8601 je međunarodni standard za predstavljanje datuma i vremena. Format je `YYYY-MM-DDTHH:mm:ss.sssZ`.

:::interactive_editor

```js
const date = new Date("2014-10-15");
console.log(date.toISOString()); // "2014-10-15T00:00:00.000Z"
```

:::

- **Njia ya `toLocaleDateString()`**: Ovaj put se koristi za kreiranje formata datuma u skladu sa lokacijom korisnika.

:::interactive_editor

```js
const date = new Date("2014-10-15");
console.log(date.toLocaleDateString("en-US"));  // "10/15/2014"
```

:::

Metoda `toLocaleDateString()` prihvata dva opciona parametra: locales i options.

Parametar za lokacije (`locales`) je niz znakova koji predstavlja željenu regiju/lokaciju. Na primer, možete uneti ``"en-US"`` za engleski jezik (SAD) ili ``"fr-FR"`` za francuski jezik (Francuska). Ako ne navedete parametar za lokacije, koristiće se podrazumevana regija. Drugi opcioni parametar je `options`. Ovaj parametar vam omogućava da definišete format datuma.

:::interactive_editor

```js
const date = new Date("2014-10-15");
const options = {
  weekday: "long",
  year: "numeric",
  month: "long",
  day: "numeric",
};
console.log(date.toLocaleDateString("en-GB", options)); // "Wednesday, 15 October 2014"
```

:::
