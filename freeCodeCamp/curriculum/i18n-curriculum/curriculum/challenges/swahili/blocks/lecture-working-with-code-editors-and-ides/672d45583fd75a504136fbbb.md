# --description--

Da biste instalirali VS Code, moraćete da posetite njihovu veb stranicu kako biste pronašli odgovarajući instalater za vaš operativni sistem. Proverite da posetićete VS Code sajt, a ne Visual Studio sajt.

Za Windows, možete preuzeti EXE instalater. Nakon što ga preuzmete, pokrenite taj fajl i čarobnjak za podešavanje će vas voditi kroz proces instalacije.

Na Mac-u, možete preuzeti zip fajl koji ima `.app` i koji ga možete pokrenuti. Takođe možete instalirati VS Code preko homebrew-a.

Otvori Terminal aplikaciju na svom Macu i izvrši sledeću komandu:

```sh
brew install --cask visual-studio-code
```

Na Linuxu, možete preuzeti fajl `.deb` ili `.rpm` direktno. Nakon instalacije, možete koristiti odgovarajući izvršni fajl (executable) ili komandu interfejsa linije komandi (CLI) (obično `code`), da pokrenete program. I sada ste spremni za pisanje koda.
