# --description--

`Uncertain` znači nesigurnost o nečemu, kao što je osećaj nesigurnosti oko odluke. Suprotno je od toga `certain`, što znači imati sigurnost. Na primer, `I am certain this is the right way` znači imati sigurnost u smeru.

`Achieve` to znači postizanje cilja nakon napora, kao što je `achieve a good result in a test`. U programiranju, `achieve` često se odnosi na cilje koje programer želi postići svojim kodom.
