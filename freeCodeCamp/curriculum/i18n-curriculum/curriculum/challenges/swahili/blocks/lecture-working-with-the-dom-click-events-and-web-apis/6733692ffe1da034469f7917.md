# --interactive--

Postojaće trenuci kada ćete morati da promenite stil HTML elementa unutar JavaScript fajla. Primer za to je kada treba da sakrijete ili prikažete meni kada korisnik klikne na dugme.

U ovoj lekciji naučićete kako da menjate stilove koristeći osobine `Element.style` i `Element.classList`.

Osobina `Element.style` predstavlja osobinu koja omogućava pristup inline stilovima elementa. Možete je koristiti za dobijanje ili postavljanje stilova elementa.

Evo primera elementa pasusa sa ID-em `para`:

```html
<p id="para">This is a paragraph</p>
```

Možete koristiti osobinu `style` da promenite boju elementa pasusa u crvenu:

:::interactive_editor

```html
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```js
const paraEl = document.getElementById("para");
paraEl.style.color = "red";
```

:::

Osobina `style` može se koristiti za postavljanje mnogih CSS osobina, kao što su `color`, `background-color`, `font-size`, `font-weight` i druge.

Drugi način menjanja stilova jeste korišćenjem osobine `Element.classList`. Osobina `classList` omogućava dodavanje, uklanjanje ili menjanje klasa elementa. Pogledajmo nekoliko primera.

U ovom primeru dodaćete klasu pod nazivom `highlight` elementu pasusa sa ID-em `para`. U CSS-u ćemo podesiti klasu `highlight` tako da promeni boju pozadine elementa pasusa u žutu. U JavaScript-u ćemo dodati klasu `highlight` elementu pasusa pomoću metode `classList.add()`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<p id="para">This is a paragraph</p>
<script src="index.js"></script>
```

```css
.highlight {
  background-color: yellow;
}
```

```js
const paraEl = document.getElementById("para");
paraEl.classList.add("highlight");
```

:::

Takođe možete dodati više klasa elementu tako što ih prosledite kao argumente metodi `classList.add()`.

```js
classList.add("class1", "class2", "class3");
```

Ako treba da uklonite klasu sa elementa, možete koristiti metodu `classList.remove()` na sledeći način:

```js
classList.remove("highlight");
```

Da biste promenili klasu na elementu, možete koristiti metodu `classList.toggle()`. U ovom primeru imamo `button` element sa ID-em `toggle-btn` i `nav` element sa ID-em `menu`. U CSS-u smo dodali stilove za meni i klasu `show` koja menja osobinu `display` elementa `.menu` u `block`. U JavaScript-u pronalazimo element menija i dugme pomoću metode `getElementById()`. Zatim dodajemo osluškivač događaja na dugme koji će menjati klasu `show` na elementu `.menu` kada se dugme klikne:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<button id="toggle-btn">Toggle Menu</button>

<nav id="menu" class="menu">
  <ul>
    <li>Home</li>
    <li>About</li>
    <li>Products</li>
  </ul>
</nav>
<script src="index.js"></script>
```

```css
.menu {
  display: none;
  background-color: lightgray;
  width: 50%;
  padding: 10px;
}

.menu.show {
  display: block;
}
```

```js
const menu = document.getElementById("menu");
const toggleBtn = document.getElementById("toggle-btn");

toggleBtn.addEventListener("click", () => menu.classList.toggle("show"));
```

:::

Kada korisnik klikne dugme, klasa `show` biće dodata elementu `.menu` i meni će biti prikazan. Ako korisnik ponovo klikne dugme, klasa `show` biće uklonjena sa elementa `.menu` i meni će biti sakriven.

Korišćenjem metoda i osobina kao što su `style`, `add()`, `remove()` i `toggle()`, možete lako menjati stilove elemenata pomoću JavaScript-a i kreirati dinamične i interaktivne veb stranice.
