# --description--

Niz znakova je sekvenca karaktera okružena jednostrukim ili dvostrukim navodnicima. U nekim programskim jezicima, karakteri okruženi jednostrukim navodnicima tretiraju se drugačije od karakterima okruženim dvostrukim navodnicima, ali u Python, tretiraju se isto. Stoga, možete koristiti jedan od njih kada koristite niz znakova. Evo primera nizova znakova:

```python
my_str_1 = 'Hello'
my_str_2 = "World"
```

Ukoliko je potreban višelinijski niz znakova, možete koristiti trostruke dvostruke navodne navodnike ili trostruke jednostavne navodne navodnike:

```python
my_str_3 = """Multiline
string"""
my_str_4 = '''Another
multiline
string'''
```

Ako vaš niz znakova sadrži pojedinačni navodnik ili dvostruke navodnike, onda imate dve opcije:

- Koristite različite vrste navodnika. To jest, ako vaša sekvenca ima jednodelne navodnike, koristite dvonodelne navodnike oko sekvence, i obrnuto:

```python
msg = "It's a sunny day"
quote = 'She said, "Hello World!"'
```

- Koristite povratnik (`\`) kako biste sprečili jednopostavke ili dvonapostavke navodnika unutar niza karaktera. Na ovaj način, možete koristiti jednastavke ili dvonastavke navodnike oko tog niza:

```python
msg = 'It\'s a sunny day'
quote = "She said, \"Hello!\""
```

Ponekad, možda ćete morati da proverite da li niz ima jedan ili više karaktera. Za to, Python pruža operator `in` koji vraća vrednost tačno ili netačno koja pokazuje da li karakter ili karakteri postoje unutar niza ili ne.

Hapa kuna mifano:

```python
my_str = 'Hello world'

print('Hello' in my_str)  # True
print('hey' in my_str)    # False
print('hi' in my_str)    # False
print('e' in my_str)  # True
print('f' in my_str)  # False
```

Sada, hajde da istražimo kako možete dobiti dužinu niza karaktera i raditi sa pojedinačnim karakterima unutar niza, proces koji se zove **pristup po primeru**. Da biste dobili dužinu niza karaktera, možete koristiti ugrađenu funkciju `len()`. Evo primera:

```python
my_str = 'Hello world'
print(len(my_str))  # 11
```

Kila herufi ndani ya mfuatano wa herufi ina nafasi inayoitwa kielezo. Kielezo huanza kutoka sifuri, maana yake kielezo cha herufi ya kwanza ni `0`, cha pili ni `1`, na kadhalika. Ili kupata herufi kwa kutumia kielezo chake, unatumia mabano ya mraba (`[]`) na kielezo cha herufi unayotaka kupata ndani yake. Hapa kuna mifano:

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
```

Pristup negativnim obrascima takođe je dozvoljen, stoga možete dobiti poslednje slovo bilo koje sekvence pomoću `-1`, drugo slovo od kraja za `-2`, itd.:

```python
my_str = 'Hello world'
print(my_str[-1])  # d
print(my_str[-2]) # l
```

Mnogi programski jezici generalno klasifikuju tipove podataka u osnovne ili referentne tipove. Osnovni tipovi su jednostavni i nepromenljivi, što znači da se ne mogu promeniti nakon deklarisanja. Referentni tipovi mogu čuvati više vrednosti, a mogu biti promenljivi ili nepromenljivi. Međutim, Python ne pravi striktnu razliku između te dve grupe. Umesto toga, svi podaci se tretiraju kao objekti, a neki objekti su nepromenljivi, dok drugi mogu biti promenljivi.

Nepromenljivi tipovi podataka ne mogu se menjati niti uništiti nakon objavljivanja. Možete napraviti promene njegovih parametara ka novoj stvari, nazvano ponovno raspoređivanje, ali ne možete menjati stvarni objekat dodavanjem, uklanjanjem ili menjanjem njegovih komponenti.

Sekvenca karaktera je nepromenljivi tip podataka u Python. To znači da možete postaviti drugačiju sekvencu karaktera na parametar:

```python
greeting = 'hi'
greeting = 'hello'
print(greeting) # hello
```

Ali direktne promene sekvence slova nisu dozvoljene:

```python
greeting = 'hi'
greeting[0] = 'H' # TypeError: 'str' object does not support item assignment
```

Primeri drugih nepromenljivih tipova u Python su celi brojevi, brojevi sa zaetlenjem, boolean vrednosti (tačno ili netačno), tuple i range. Naučićete o ovim tipovima u narednim lekcijama.
