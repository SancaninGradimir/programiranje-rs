# --interactive--

Postoje različiti stanja veze, uključujući `link`, `visited`, `hover`, `focus`, i na `active`. Ova stanja su važna za pomoć korisnicima u identifikovanju linkova i pružanju jasnih povratnih informacija nakon interakcije, što poboljšava upotrebljivost i Pristupačnost.

Prikaz ovih različitih stanja zglobova je važan za upotrebljivost i pristupačnost, jer pruža vizuelne indikatore o trenutnom stanju zgloba. Ovo pomaže korisnicima da razumeju koji su zglobove posetili, sa kojim zglobom interaguju, i šta se dešava kada kliknu.

Za korisnike sa smanjenim vidom ili intelektualnim teškoćama, ove različite stilove mogu učiniti navigaciju lakšom i razumljivijom.

Štaviše, jasno stanje veza poboljšava korisničko iskustvo pružanjem trenutne povratne informacije tokom interakcije korisnika, smanjujući zbunjenost i poboljšavajući lakoću korišćenja sajta.

Ovi elementi mogu biti stilizovani koristeći nešto nazvano `pseudo-classes` unutar CSS.

Pseudo-klasa je tehnički termin koji se dodaje selektoru i označava posebno stanje izabranog elementa.

Na primer, `:hover` može da promeni boju dugmeta kada kursor korisnika pređe preko njega, dok `:visited` može da promeni boju linka koji je već posetađen.

Pseudo-klasa omogućava vam stilizovanje elemenata u zavisnosti od njihovog stanja ili interakcije korisnika bez potrebe za dodatnim oznakama u HTML vašem.

Sintaksa za `pseudo-class` izgleda ovako, gde je `A` selektor i `:B` je `pseudo-class`:

```css
A:B {
  property: value;
}
```

Da biste bolje razumeli kako dekorisati različite vrste sastojaka, istražimo nekoliko primera.

Lažni element za `:link` ukrašava veze koje nisu linkovi, pokazujući da se mogu kliknuti.

Evo primera ciljanog sidrenog elementa i korišćenja pseudo-klase `:link`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Normal state (unvisited link) */
a:link { 
  color: red;
}
```

:::

Ovaj primer iznad će promeniti boju elementa sa uobičajene plave na `red` kada nije kliknut.

`:visited` dekorira linkove koji su već posetili ili kliknuli, pomažući korisnicima da prate linkove koje su ranije kliknuli. Evo primera korišćenja pseudoklasa za `:visited`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="https://www.freecodecamp.org/learn/" target="_blank">freeCodeCamp</a>
```

```css
/* Visited link */
a:visited {
  color: green;
}
```

:::

Ovaj kod će promeniti boju veze na `green` kada bude kliknuta.

`:hover` menja stil elementa kada korisnik pređe mišem preko njega, pružajući vizuelnu indikaciju da je element interaktivan. Evo primera korišćenja pseudo-klase `:hover`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Hover state */
a:hover {
  color: green;
}
```

:::

Ovaj kod će promeniti boje elemenata na `green` kada mišem pređe preko njega.

`:focus` Dodaje stilove oko elementa kada je fokusiran, kao što je slučaj kada koristite tastaturu za navigaciju ili poboljšanje pristupačnosti.

Ovo je primer korišćenja atributa `outline` da prikaže čvrstu narandžastu liniju kada se pređe mišem preko veze.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Focus state */
a:focus {
  outline: 2px solid orange;
}
```

:::

`:active` menja stil linka kada se klikne na njega, pružajući korisniku trenutnu povratnu informaciju da je njegova akcija snimljena. Evo primera korišćenja lažne klase `:active`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<a href="/">Example link</a>
```

```css
/* Active state */
a:active {
  color: pink;
}
```

:::

Ovaj primer koda će promeniti boju veze u `pink` čim se klikne na vezu.
