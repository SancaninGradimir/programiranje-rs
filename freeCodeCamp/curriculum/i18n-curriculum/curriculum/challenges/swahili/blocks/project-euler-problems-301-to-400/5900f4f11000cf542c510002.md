# --description--

Razmotri sve triple $(a$, $b$, $c$) koje zadovoljavaju $0 ≤ a, b, c ≤ N$.

Iz porekla $O(0, 0, 0)$, sve linije su povučene ka drugim tačkama u ravni. Neka je $D(N)$ broj takvih različitih linija.

Saznalo se da $D(1\\,000\\,000) = 831\\,909\\,254\\,469\\,114\\,121$.

Pronađi $D({10}^{10})$. Daj svoj odgovor tako što ćeš prikazati devet prvih cifara praćeno sa devet poslednjih.
