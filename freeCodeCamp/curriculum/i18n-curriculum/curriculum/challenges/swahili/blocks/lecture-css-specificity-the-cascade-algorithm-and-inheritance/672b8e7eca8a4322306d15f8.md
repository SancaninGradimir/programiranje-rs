# --interactive--

Univerzalni selektor (`*`) je poseban tip selektora za CSS koji odgovara bilo kom elementu u dokumentu.

Često se koristi za postavljanje stila svih elemenata na stranici, što može pomoći u ispravljanju ili osiguravanju konzistentnog izgleda/stila u različitim pretraživačima.

Univerzalni selektor se može koristiti za odabir svih elemenata unutar specifičnog konteksta ili opštije u celom dokumentu.

Ovo je primer korišćenja univerzalnog selektora za postavljanje `margin` i `padding` za ceo dokument HTML:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<h1>Heading element</h1>
<p>example paragraph element</p>
```

```css
* {
  margin: 0;
  padding: 0;
}
```

:::

U ovom kodnom primeru, selektor `*` smanjuje granicu i unutrašnji razmak svih elemenata na nulu, što je uobičajena tehnika korišćena za ispravljanje stilova CSS.

Univerzalni selektor ima minimalnu vrednost definisanja među svim selektorima. Doprinosi 0 za sve delove vrednosti definisanja `(0, 0, 0, 0)`.

Ovo znači da bilo koji drugi selektor, uključujući tip selektore, klasu elemenata, ID, i inline stilove, će nadjačati stilove postavljene univerzalnim selektorom.

Pogledajmo sledeći HTML i CSS primer:

:::interactive_editor

```html
<head>
  <style>
    * {
      color: blue;
    }
    p {
      color: red;
    }
    .highlight {
      color: green;
    }
    #unique {
      color: purple;
    }
  </style>
</head>
<body>
  <p id="unique" class="highlight">This text has multiple styles applied.</p>
</body>
```

:::

Univerzalni selektor `*` postavlja boju teksta na `blue` za sve elemente.

Međutim, selektor tipa `p` nadjačava ovo tako što postavlja boju teksta na `red` posebno za elemente `p`.

Ako element ima klasu `highlight`, selektor klase dobija prioritet i menja boju teksta u `green`.

Konačno, selektor identiteta za `#unique`, koji ima najviši prioritet, će poništiti sve ostalo i postaviti boju teksta na `purple`.
