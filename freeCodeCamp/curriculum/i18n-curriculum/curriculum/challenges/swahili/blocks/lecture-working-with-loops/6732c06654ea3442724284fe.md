# --interactive--

U prethodnim lekcijama, naučili ste kako da radite sa krugovima `for`, krugovima `for...in` i krugovima `for...of`. U ovom treninžu, naučićete o krugu `while` i krugu `do...while`.

Petlja za `while` će izvršiti deo koda samo ako je uslov tačan. Evo osnovne sintakse petlje za `while`:

```js
while (condition) {
  // code block to be executed
}
```

Stanje se proverava pre nego što se deo koda izvrši. Ako je stanje nevažeće, deo koda neće biti izvršen.

Petlje za `while` su korisne kada ne znate koliko puta treba da pokrenete deo koda. Evo primera korišćenja petlje `while`:

:::interactive_editor

```js
let counter = 0;
while(counter < 5) {
  console.log(counter);
  counter++;
}
```

:::

U ovom primeru, imamo parametar nazvan `counter` koji je inicijalizovan sa `0`. Petlja `while` će se izvršavati samo ako vrednost `counter` je manja od `5`. Unutar petlje, zapisujemo vrednost `counter` u konzolu, zatim dodajemo `counter` na `1`.

Još jedna petlja slična `while` petlji je `do...while`. Evo njene osnovne sintakse:

```js
do {
  // code block to be executed
} while (condition);
```

Glavna razlika između `do...while` i `while` petlje jeste to što će `do...while` izvršiti blok koda najmanje jednom pre nego što proveri uslov.

Ako je uslov tačan, deo koda će nastaviti da se izvršava. Ako je uslov netačan, deo koda će biti zaustavljen.

Ovde je primer korišćenja mzunguko `do...while`:

:::interactive_editor

```js
let counter = 0;
do {
  console.log(counter);
  counter++;
} while (counter < 5);  
```

:::

U ovom primeru imamo promenljivu `counter` inicijalizovanu na `0`. Petlja `do...while` će ispisati vrednost promenljive `counter` u konzolu, zatim povećati `counter` za `1`. Nakon izvršavanja bloka koda proverava se da li je `counter` manji od `5`. Ako jeste, petlja nastavlja da se izvršava. Ako nije, petlja se zaustavlja.

U mnogim slučajevima, možda ćete koristiti ciklus `while` češće nego ciklus `do...while`. Međutim, dobro je znati oba tipa ciklusa i kada ih koristiti.

## --text--

Koja je glavna razlika između kruga `while` i kruga `do...while`?

## --answers--

Ciklus `while` izvršava izvođenje bloka koda bar jednom, dok ciklus `do...while` ne radi.
