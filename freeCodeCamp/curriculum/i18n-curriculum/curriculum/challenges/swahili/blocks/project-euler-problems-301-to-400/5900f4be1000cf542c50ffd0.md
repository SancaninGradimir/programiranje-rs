# --description--

Neka je $\\{a_1, a_2, \ldots, a_n\\}$ niz celih brojeva sa dužinom $n$, na sledeći način:

- $a_1 = 6$
- za svako $1 ≤ i &lt; n$ : $φ(a_i) &lt; φ(a_{i + 1}) &lt; a_i &lt; a_{i + 1}$

$φ$ označava funkciju totijenta po Euleru.

Neka $S(N)$ bude broj sekvence poput ove, gde je $a_n \leq N$.

Na primer, $S(10) = 4$: {6}, {6, 8}, {6, 8, 9} na {6, 10}.

Može se potvrditi da $S(100) = 482\\,073\\,668$ i $S(10\\,000)\bmod {10}^8 = 73\\,808\\,307$.

Pronađi $S(20\\,000\\,000)\bmod {10}^8$.
