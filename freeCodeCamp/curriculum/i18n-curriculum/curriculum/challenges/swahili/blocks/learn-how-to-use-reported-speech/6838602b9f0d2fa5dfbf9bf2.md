# --explanation--

`Unusually` je komparativni pridev koji se koristi za opisivanje nečega što se dešava više ili manje od uobičajenog ili očekivanog. S druge strane, `unusual` je pridev koji opisuje nešto što nije uobičajeno ili normalno. Na primer:

- `The server experienced an unusually high load today.` – U ovoj rečenici, `unusually` se odnosi na način kako je teret bio veliki (mnogo više nego obično).

- `It was unusual for the server to slow down like that.` – U ovoj rečenici, `unusual` se odnosi na tu situaciju. Ne utiče ni na glagol ni na pridev.

`query` je upit ili zahtev poslat bazi podataka radi dobijanja informacija ili izvršavanja akcije/operacije. Na primer:

`The application sent a query to the database to fetch user details.` – To znači da je program tražio specifične podatke iz baze podataka.
