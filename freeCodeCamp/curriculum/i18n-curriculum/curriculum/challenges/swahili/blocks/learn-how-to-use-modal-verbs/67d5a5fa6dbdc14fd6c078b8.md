# --explanation--

Ako `wh-` pitanje (`when`, `who`, `why`, `how`) koristi modalni glagol, modalni glagol dolazi ispred subjekta. Glavni glagol ostaje u osnovnom obliku. Na primer:

- `What can you do to fix this?` - Kuuliza kuhusu uwezekano.

- `Why should we start early?` — Traženje saveta.

- `How must we prepare?` - Kuuliza kuhusu mahitaji.

- `When will you finish?` - Kuuliza kuhusu matendo ya baadaye.

- `Where could I find help?` - Pitajte o opcijama.

Na ovaj način, pitaš o onome što je moguće, potrebno ili očekivano.
