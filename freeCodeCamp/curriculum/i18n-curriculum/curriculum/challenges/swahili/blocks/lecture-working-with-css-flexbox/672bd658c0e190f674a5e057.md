# --interactive--

Svojstva flex-a su svojstva koja možete koristiti na fleksibilnim kontejnerima kako biste odredili raspodelu malih elemenata. Razgovaraćemo o nekim često korišćenim svojstvima: `flex-wrap`, `justify-content`, i `align-items`.

Počinjemo sa `flex-wrap`. Ovo svojstvo određuje kako se fleksibilni delovi pakuju unutar fleksibilnog kontejnera da bi odgovarali dostupnom prostoru. `flex-wrap` može uzeti tri moguća vrednosti: `nowrap`, `wrap` i `wrap-reverse`. `nowrap` je podrazumevana vrednost — fleksibilni delovi neće biti pakovani na novu liniju, čak i ako njihova širina premašuje širinu kontejnera.

U kodu ispod, imamo nas tri komada `div`. Obratite pažnju na `width`. Kontejner za `main` napunjen do crvene linije ima `width` od `200px`, dok tri komada `div` unutar njega imaju `width` od `240px` (`80px` svako):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Širina delova `div` prelazi širinu njihovog kontejnera, ali će podrazumevano biti smanjeni da odgovaraju dostupnom prostoru. Ako želite da se delovi prenesu na novu liniju kada premaše širinu svog kontejnera, možete postaviti `flex-wrap: wrap` u dinamički element:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  flex-wrap: wrap;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Komadi `div` će se ponovo rasporediti u redove kada premaše širinu svog kontejnera. Možete učitati fleksibilne komadiće u obrnutom redosledu koristeći `flex-wrap: wrap-reverse`. Svojstvo `flex-flow` je svojstvo skraćivanja za `flex-direction` i `flex-wrap`. U ovom primeru, postavili smo `flex-direction` na `column` i `flex-wrap` na `wrap-reverse`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  width: 200px;
  display: flex;
  flex-flow: column wrap-reverse;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Dobro. Sada hajde da pričamo o `justify-content`. `justify-content` postavlja male komponente paralelno sa glavnom osom prilagodljivog uređaja. Ako postavite vrednost `flex-start` za `justify-content`, prilagodljivi delovi će biti postavljeni na početku glavne ose. Ovo može biti horizontalno ili vertikalno:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: flex-start;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Za `justify-content: flex-end`, prilagodljivi delovi se postavljaju na kraju glavne ose, horizontalno ili vertikalno.

:::interactive_editor

```html

  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  justify-content: flex-end;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

:::interactive_editor

```css
#third-div {
```

:::

:::interactive_editor

```html

  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>

```

```css

  border: 2px solid red;

  width: 80px;
  height: 50px;

#third-div {
```

Dobro. Sada znate kako da podelite varijabilne komade na glavnoj osi. Ali možda želite da ih postavite na poprečnu osu. Zapamtite da je poprečna osa vertikalna u odnosu na glavnu osu. Ovo možete uraditi koristeći svojstvo `align-items`. Da postavite komade u sredinu poprečne ose, samo morate dodati `align-items: center` na varijabilni kontejner:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  height: 300px;
  display: flex;
  align-items: center;
  border: 2px solid red;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Da biste poravnali zamenske delove na poprečnoj osovini, možete koristiti `align-items: stretch`. Ovo takođe radi sa komponentama koje se automatski mere, kao što su one bez instaliranih `width` ili `height`, ili one merene samo do minimalnog standarda od `width` ili `height`. Zamenski delovi se poravnaju kako bi popunili kontejner u smeru poprečne osovine.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box">Three</div>
</div>

```

```css
.container {
  display: flex;
  align-items: stretch; 
  border: 2px solid #444;
  height: 200px; 
}

.box {
  width: 100px;           
  border: 1px solid #888;
  background-color: lightblue;
}
```

:::

Konačno, možete koristiti svojstvo ``align-self`` da postavite različito poravnanje na poprečnoj osi za samostalni element. Na primer, možete ga izravnati na ``align-self: stretch``.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="box">One</div>
  <div class="box">Two</div>
  <div class="box special">Three</div>
</div>

```

```css
.container {

  height: 200px; 
}

.box {
  width: 100px;
  border: 1px solid #888;
  background-color: lightblue;
  margin: 4px;
}

.special {
  align-self: stretch; 
  background-color: lightcoral;
}
```
