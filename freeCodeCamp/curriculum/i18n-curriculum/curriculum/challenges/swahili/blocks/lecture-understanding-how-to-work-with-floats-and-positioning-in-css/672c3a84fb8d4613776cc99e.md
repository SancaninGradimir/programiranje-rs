# --interactive--

U CSS, pozicioniranje nam omogućava da kontrolišemo kako su elementi raspoređeni na stranici. Dva uobičajena tipa pozicioniranja su static i relativno pozicioniranje. Podrazumevano, elementi su postavljeni na static. To znači da prate standardan tok dokumenta, jedan za drugim, odozgo nadole, od leve ka desnoj strani.

Static pozicioniranje je podrazumevano za sve elemente i ne zahteva posebnu CSS deklaraciju. Nećete videti nikakvu razliku kada koristite static pozicioniranje, jer elemente ostavlja na mestu gde se normalno nalaze u dokumentu. Evo primera pasusa koji koristi podrazumevano static pozicioniranje:

:::interactive_editor

```html
<p>This paragraph is statically positioned.</p>
```

:::

Kada se prikazuje u pretraživaču, ovaj pasus će se pojaviti na svom prirodnom mestu, prateći normalan tok dokumenta. Nije korišćen poseban stil da bi se promenilo njegovo mesto, i ostaje na mestu gde bi normalno trebalo da se pojavi u rasporedu.

Relativno pozicioniranje, s druge strane, omogućava elementu da se premesti iz svoje uobičajene pozicije bez uticaja na tok dokumenta. Zamislite da pomerate element iz njegove podrazumevane pozicije u static tako što mu date nove koordinate. Evo kako možete koristiti relativno pozicioniranje:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="relative">This paragraph is positioned relatively.</p>
```

```css
body {
  border: solid 1px black;
}

.relative {
  position: relative;
  top: 30px;
  left: 30px;
}
```

:::

U ovom primeru, pasus će se pojaviti `30px` dole i `30px` desno od svoje originalne pozicije. Međutim, prostor koji bi zauzeo u normalnom toku ostaje rezervisan. Relativno pozicioniranje je veoma važno kada želite da pomerite element malo bez uticaja na ceo raspored.
