# --description--

Slovo `v` ima dva imena poznata u španskom.

Ime `ve` je najbolje poznato kao Južna Amerika, dok je ime `uve` bolje poznato kao Hispanija.

Njegovo ime je gotovo isto kao slovo `b`. Južna Amerika, kako bi se razlikovala od slova `b`, često se naziva `ve corta` ("kračno v") ili `ve pequeña` ("maloko v").

Primer je `vuelo` ("ptice").
