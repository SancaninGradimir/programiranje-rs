# --description--


JavaScript je programski jezik koji se koristi na vebu. Za razliku od HTML-a i CSS-a, koje si ranije učio, JavaScript se uglavnom koristi za pisanje logike, a ne za označavanje sadržaja.

U ovom projektu, naučićete osnove JavaScript i koristiti te koncepte za izradu piramidalnog generatora.

Generator piramida je program u kojem možete postaviti tip ponašanja, broj piramida i pravac piramide. Program će generisati piramide na osnovu ovih unosa.

Pritisnite dugme „Proverite svoj kod“ ili idite na sledeći korak i započnite projekat.
