# --description--

Neka je $R(M, N)$ broj rešetkastih tačaka $(x, y)$ koji zadovoljavaju $M < x \le N$, $M < y \le N$ i $\left\lfloor\frac{y^2}{x^2}\right\rfloor$ je broj koji nije jednak.

Možemo da potvrdimo da $R(0, 100) = 3\\,019$ i $R(100, 10\\,000) = 29\\,750\\,422$.

Pronađi $R(2 \times {10}^6, {10}^9)$.

**Napomena:** $\lfloor x\rfloor$ predstavlja funkciju sa podom (floor function).
