# --interactive--

Postoje dva moćna načina u JavaScript za obradu podataka o JSON: `JSON.parse()` i `JSON.stringify()`. Ove metode se često koriste za konverziju između sekvence karaktera od JSON i objekata iz JavaScript.

`JSON.stringify()` koristi se za konverziju nečega iz JavaScript u niz znakova JSON. Ovo je važno kada želite da sačuvate ili prenesete podatke u format koji se lako deli ili prenosi između sistema.

Ovo je kako možete koristiti metodu `JSON.stringify()`:

:::interactive_editor

```js
const user = {
  name: "John",
  age: 30,
  isAdmin: true
};

const jsonString = JSON.stringify(user);
console.log(jsonString);
```

:::

Putanja ``JSON.stringify()`` takođe prima opcionalni parametar nazvan 'replacer', koji može biti funkcija ili niz podataka. Evo primera za korišćenje niza podataka za opcionalni parametar 'replacer':

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

// result: {"firstName":"Jessica","country":"USA"}
console.log(JSON.stringify(developerObj, ["firstName", "country"]));
```

:::

U ovom primeru, imamo `developerObj` sa četiri atributa. Kada koristimo metodu `JSON.stringify()`, možemo proslediti red podataka po drugom kriterijumu i specificirati koje atribute želimo da budu u nizu karaktera. Rezultat će biti objekat konvertovan u niz karaktera sa atributima `firstName` i `country` samo.

Još jedan opcioni parametar putem `JSON.stringify()` je spacer parametar. Ovo vam omogućava da kontrolišete razmak u izlazu sekvence karaktera:

:::interactive_editor

```js
const developerObj = {
  firstName: "Jessica",
  isAwesome: true,
  isMusician: true,
  country: "USA",
};

console.log(JSON.stringify(developerObj, null, 2));

/* result
{
  "firstName": "Jessica",
  "isAwesome": true,
  "isMusician": true,
  "country": "USA"
}
*/
```

:::

Često koristite ove opcione parametre putem `JSON.stringify()`, ali je i dalje dobro da ih znate.

Drugi način koji ćete često koristiti u svojim programima je način `JSON.parse()`. `JSON.parse()` menja niz karaktera od JSON i vraća ga u nešto od JavaScript. Ovo je važno kada dobijete podatke o JSON sa mrežnog servera ili iz `localStorage` i kada morate da obradite te podatke u vašem programu. Naučićete više o `localStorage` u sledećoj lekciji.

Ovo je primer kako da se radi sa metodom `JSON.parse()`:

:::interactive_editor

```js
const jsonString = '{"name":"John","age":30,"isAdmin":true}';
const userObject = JSON.parse(jsonString);
console.log(userObject);

// Result:
// { name: 'John', age: 30, isAdmin: true }
```

:::

Ovo vam omogućava da radite sa podacima u vašoj aplikaciji kao standardni element za JavaScript, i olakšava vam menjanje i korišćenje.

U narednim modulima, nastavićemo da učimo više o tome kako raditi sa JSON i metodama za `JSON.parse()` i `JSON.stringify()`.
