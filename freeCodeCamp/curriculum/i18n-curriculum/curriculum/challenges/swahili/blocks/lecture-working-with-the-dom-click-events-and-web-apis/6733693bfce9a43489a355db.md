# --interactive--

Došlo je do događaja `DOMContentLoaded` kada su sve stvari u dokumentu HTML preuzete i analizirane. Ako imate eksterne stilove, ili slike, događaj `DOMContentLoaded` neće čekati da se preuzmu. Čekaće samo preuzimanje HTML.

Ovo se razlikuje od događaja `load`, koji čeka da se sve preuzme, uključujući spoljne datoteke stilova, slike i slično.

Evo primera sintakse za korišćenje događaja `DOMContentLoaded`:

```js
document.addEventListener("DOMContentLoaded", function () {
  console.log("DOM is loaded.");
});
```

Kada `DOM` bude preuzet, funkcija će biti pokrenuta sa porukom da ``DOM is loaded.`` biće napisana na konzoli.

Sada pogledajmo još jedan primer u kojem koristimo događaj `DOMContentLoaded`. U ovom primeru imamo sliku u HTML-u. Da bismo ažurirali sliku, možemo kreirati funkciju koja menja atribute `src` i `alt` slike.

:::interactive_editor

```html
<h1>Image Change on DOM Loaded</h1>
<img
  id="example-img"
  src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg"
  alt="Cat relaxing"
/>

<script src="index.js"></script>
```

```js
function changeImg() {
  const img = document.getElementById("example-img");
  img.src =
    "https://cdn.freecodecamp.org/curriculum/responsive-web-design-principles/FCCStickers-CamperBot200x200.jpg";
  img.alt = "CamperBot sticker";
  console.log("image was just changed");
}

changeImg();
```

:::

Zatim možemo proveriti da li se DOM još uvek učitava i dodati obrađivač događaja `DOMContentLoaded`. Ako se događaj `DOMContentLoaded` već dogodio, možemo direktno pozvati funkciju `changeImg`.

```js
if (document.readyState === "loading") {
  document.addEventListener("DOMContentLoaded", changeImg);
} else {
  console.log("DOMContentLoaded has already fired");
  changeImg();
}
```

Ako pokrenete ovaj kod i osvežite stranicu, videćete da se slika menja u novu sliku nakon što se DOM učita. Ovo se dešava veoma brzo, ali ćete primetiti mali bljesak kada se slika promeni.

Događaj `DOMContentLoaded` je važan kada želite pokrenuti kod JavaScript čim se DOM preuzme, ali pre nego što su sve eksterne resurse kao što su slike i stilovi (ili: CSS fajlovi) preuzeti.
