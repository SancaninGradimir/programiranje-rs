# --description--

Za svaki ceo broj $n > 1$, porodica funkcija $f_{n, a, b}$ je definisana kao:

$f_{n, a, b}(x) ≡ ax + b\bmod n$ za cela broja $a, b, x$ i $0 \lt a \lt n$, $0 \le b \lt n$, $0 \le x \lt n$.

Neka je $f_{n, a, b}$ nazovemo ako $f_{n, a, b}(f_{n, a, b}(x)) \equiv f_{n, a, b}(x)\bmod n$ za svako $0 \le x \lt n$.

$R(n)$ je rekurentni broj za $n$.

$F(N) = \displaystyle\sum_{n = 1}^N R(n^4 + 4)$.

$F(1024) = 77\\,532\\,377\\,300\\,600$.

Izračunaj $F({10}^7)$. Daj svoj odgovor modulo $1\\,000\\,000\\,007$.
