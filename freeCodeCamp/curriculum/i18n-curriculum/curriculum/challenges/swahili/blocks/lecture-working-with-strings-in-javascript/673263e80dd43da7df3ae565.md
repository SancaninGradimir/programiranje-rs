# --interactive--

Kada radite sa nizom znakova u JavaScript, postoje situacije kada je potrebno pronaći poziciju za manji niz znakova unutar većeg niza.

Niz malih slova je sekvenca karaktera koja se pojavljuje unutar većeg niza. Na primer, u nizu `hello world`, `hello` i `world` je niz malih slova.

Da bi pronašao poziciju podniske unutar niske, možeš da koristiš metodu `indexOf()`.
Metoda `indexOf()` u JavaScript-u omogućava ti da pronađeš podnisku unutar veće niske.

Ako se pronađe niz malih slova, `indexOf()` vraća indeks (ili poziciju) prvog pojavljivanja tog niza. Ako se niz malih slova ne pronađe, `indexOf()` vraća `-1`, što ukazuje da pretraga nije uspela.

Funkcija ``indexOf()`` prima dva argumenta: prvi je niz malih slova koji želite pronaći u većem nizu, a drugi je opcija za početnu poziciju pretrage. Ako ne navedete početnu poziciju, pretraga će početi od samog početka niza.

U ovom kontekstu, **argument** je vrednost koju pružate funkciji ili metodi kada je pozivate, što omogućava toj funkciji ili metodi da izvrši svoj zadatak koristeći specifične informacije koje pružate. Vi ćete saznati više o argumentima u budućim lekcijama.

Ovo je primer korišćenja metode `indexOf()` za dobijanje sekvence karaktera `awesome`:

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("awesome!");
console.log(position); // 14
```

:::

U ovom primeru, reč `awesome` počinje na indeksu `14` u nisci `JavaScript is awesome!`, pa metoda `indexOf()` vraća `14`.

Sada pogledaj šta se dešava kada podniska nije pronađena:

:::interactive_editor

```js
let sentence = "JavaScript is awesome!";
let position = sentence.indexOf("fantastic");
console.log(position); // -1
```

:::

Pošto se reč `fantastic` ne pojavljuje u nisci, metoda vraća `-1`.

Takođe možete specificirati mesto od kojeg želite da počnete pretragu unutar sekvence tako što ćete dati drugi argument za `indexOf()`. Evo primera:

:::interactive_editor

```js
let sentence = "JavaScript is awesome, and JavaScript is powerful!";
let position = sentence.indexOf("JavaScript", 10);
console.log(position); // 27
```

:::

U ovom slučaju, pretraga za `JavaScript` počinje nakon desetog slova, i tako se drugi događaj `JavaScript` nalazi u modelu `27`.

Bitno je znati da putanja `indexOf()` uzima u obzir velika i mala slova.

U ovom primeru, sledeće vraća `-1` jer veliko slovo `F` nije pronađeno unutar sekvence `freeCodeCamp`.

:::interactive_editor

```js
console.log("freeCodeCamp".indexOf("F")) // -1
```

:::

Korišćenje `indexOf()` može biti veoma korisno kada je potrebno proveriti da li niz malih slova postoji unutar niza i utvrditi njegovu poziciju radi daljih operacija.
