# --explanation--

Ispravna rečenica koristi strukturu `First Conditional`, koja se gradi ovako:

`if` + `Present Simple`: `If we lose critical data`.

Modalni glagol (`could`) + osnovni oblik glagola: `it could set us back weeks`.

`First Conditional` se koristi za izražavanje mogućeg budućeg događaja i njegove posledice.
