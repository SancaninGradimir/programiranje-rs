# --description--

Ako kutija ima dvadeset i jedan bojenih diskova, sa petnaest plavih i šest crvenih diskova, a ako se slučajno izaberu dva diskova, verovatnoća da će biti odabrana dva plava diskova je:

$${P(BB)} = \frac{15}{21}×\frac{14}{20} = \frac{1}{2}$$

Sledeći aranžman ovakvog tipa, koji ima tačno 50 procenat šanse da se slučajno izvuče dve plave ploče, je kutija sa osamdeset pet plavih ploča i trideset pet crvenih ploča.

Da biste dobili prvu konfiguraciju sa više od `limit` diskova ukupno, odredite broj plavih diskova koje bi ta kutija imala.

```js
function arrangedProbability(limit) {
  // Based on https://www.mathblog.dk/project-euler-100-blue-discs-two-blue/
  let blue = 15;
  let discs = 21;

  while (discs < limit) {
    const nextBlue = 3 * blue + 2 * discs - 2;
    const nextDiscs = 4 * blue + 3 * discs - 3;

    blue = nextBlue;
    discs = nextDiscs;
  }
  return blue;
}
```
