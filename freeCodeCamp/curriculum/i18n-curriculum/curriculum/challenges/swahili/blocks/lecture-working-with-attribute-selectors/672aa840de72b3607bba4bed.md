# --interactive--

Selektor svojstava u CSS je moćan alat koji vam omogućava da ciljate elemente od HTML na osnovu njihovih svojstava. To znači da možete primijeniti stilove na elemente sa specifičnim svojstvima ili čak određene vrijednosti tih svojstava.

Posebno je važno kada želite stilizovati elemente korišćenjem transformacija ili kada samo imena klasa elemenata ne pružaju dovoljno specifičnosti.

Na primer, možete koristiti selektor atributa da biste ciljali sve linkove koji imaju atribut `href`. Ovo je veoma lako kada želite da koristite isti stil za sve linkove na stranici.

Ovo je kako možete koristiti selektor atributa za ciljanje elemenata koji imaju atribut `href`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Example link with an href attribute</a>
<a>Example link without an href attribute</a>
```

```css
a {
  display: block;
}

a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

Ovo pravilo za CSS će koristiti plavu boju i donje podvlačenje za svaki link sa atributom `href`, osiguravajući da su svi klikabilni linkovi stilizovani na isti način.

Ali selektori atributa mogu biti napredniji. Zamislite da želite ciljati elemente koji imaju samo atribut `title`. Atribut `title` često pruža dodatne informacije o elementu, i možda želite da stilizujete ove elemente drugačije kako biste pokazali da imaju dostupne dodatne informacije. Evo kako to možete učiniti:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

Ovo pravilo koristi podebljano slovo i uklanja podvlačenje za bilo koji link sa atributom `title`. To je dobar način da se ovi linkovi vizuelno razlikuju od ostalih na stranici.

Takođe možete kombinovati selektore atributa da biste dobili veću tačnost. Na primer, ako želite da izaberete elemente koji imaju oba atributa `href` i `title`, možete to uraditi:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" title="Example link with a title attribute">Example link with a title attribute</a>
<a>Example link without a title or href attribute</a>
```

```css
a {
  display: block;
}

a[href][title] {
  display:block;
  color: green;
}
```

:::

U ovom slučaju, samo elementi sa atributima `href` i `title` će biti obojeni zelenim tekstom. Ovaj nivo kontrole je ono što čini selektore atributa veoma moćnim u CSS.

Drugi primer selektora karakteristika je poređenje jedne vrednosti unutar liste vrednosti koje su ostavljene za pozicije u karakteristiku.

Ovo je primer elementa jangke sa mnogo klasa:

```html
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

Da biste ciljali ovu specifičnu komponentu sidra, možete koristiti sledeći selektor:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

Sintaksa `[attr~=value]` se ovde koristi za ciljanje svih jangkor elemenata čiji atribut klase sadrži reč `"primary"`.

Ako treba da ciljate element čija vrednost atributa ima određeni prefiks, onda možete koristiti sintaksu `[attr^=value]`. Evo primera:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com" class="btn primary large">Visit Example Site</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

U ovom primeru, selektor `a[href^="https://"]` će ciljati sve anker elemente čija vrednost atributa `href` počinje sa `"https://"`.

Da biste ciljali elemente čija vrednost svojstva se završava sa određenom vrednošću, možete koristiti sintaksu ``[attr$=value]``. Ovo je primer:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://example.com">Visit Example Site</a>
```

```css
a[href$=".com"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

U ovom primeru, selektor za `a[href$=".com"]` će ciljati sve elemente anker čija vrednost atributa `href` se završava sa `.com`.

Korišćenje selektora atributa ne samo da poboljšava stilizovanje vaše web stranice, već poboljšava i pristupačnost tako što čini interaktivne elemente kao što su linkovi razlikovanim na osnovu njihovih atributa.
