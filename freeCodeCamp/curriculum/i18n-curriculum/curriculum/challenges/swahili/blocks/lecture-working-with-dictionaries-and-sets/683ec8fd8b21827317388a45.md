# --description--

Skupovi (sets) su jedna od ugrađenih struktura podataka u Python-u. Jedna od njihovih glavnih osobina je da ne čuvaju duplirane vrednosti. Ako pokušaš da dodaš dupliranu vrednost u skup, biće sačuvana samo jedna kopija te vrednosti.

Skupovi (sets) su promenljivi i neuređeni, što znači da se njihovi elementi ne čuvaju određenim redosledom, pa ne možeš da koristiš indekse ili ključeve za pristup elementima. Skupovi mogu da sadrže nepromenljive tipove podataka, kao što su brojevi, niske karaktera i torke (tuple). Takođe podržavaju skupovne operacije, uključujući uniju, presek, razliku i simetričnu razliku.

Da biste definisali set, potrebno je samo da napišete njegove komponente unutar okruglih zagrada i razdvojite ih zarezima. Ovo je numerički primer za set:

```python
my_set = {1, 2, 3, 4, 5} 
```

Jedna čudna navika pri radu sa setovima je da, ako treba da definišete prazan set, morate koristiti funkciju `set()`. Ako nacrtate samo prazne vitičaste zagrade, kao što je `{}`, Python će automatski kreirati mapu (diktionar).

```python
set() # Set
{}    # Dictionary
```

Možete dodati funkciju na set koristeći putem metode `.add()`, i proslediti novu funkciju kao argument:

```python
my_set.add(6)
```

U našem primeru, set novi će biti:

```python
{1, 2, 3, 4, 5, 6}
```

Ako pokušate dodati element koji već postoji u set, samo će biti sačuvan jedan. U ovom slučaju, već imamo broj 5 u set:

```python
my_set.add(5)
```

To set ne menja se:

```python
{1, 2, 3, 4, 5, 6}
```

Da biste uklonili element iz set, imate dve opcije. Možete koristiti metodu `.remove()` ili metodu `.discard()`, i proslediti element koji želite ukloniti kao argument.

Putanja `.remove()` će vratiti `KeyError` ako element nije pronađen, dok putanja `.discard()` neće vratiti:

```python
my_set.remove(4)
my_set.discard(4)
```

Metoda `.clear()` uklanja sve elemente iz set:

```python
my_set.clear()
```

Setovi za Python takođe imaju moćne metode koje obavljaju standardne matematičke operacije za set.

Putanje za `.issubset()` i `.issuperset()` proveravaju da li je set podskup ili nadskup drugog skupa, po odelima.

Ovde proveravamo da li `your_set` je podskup od `my_set`, što je `False` jer nisu svi elementi od `your_set` prisutni u `my_set`.

Pia proveravamo da li je `my_set` veliki set od `your_set`. Ovo je takođe `False` jer `my_set` nema sve elemente od `your_set`:

```python
my_set = {1, 2, 3, 4, 5}
your_set = {2, 3, 4, 6}

print(your_set.issubset(my_set)) # False
print(my_set.issuperset(your_set)) # False
```

Metoda `.isdisjoint()` proverava da li dva skupa nemaju zajednički element, što znači da nema sličnih elemenata. U ovom slučaju, to je `False` jer `my_set` i `your_set` imaju slične elemente – 2, 3, i 4:

```python
print(my_set.isdisjoint(your_set)) # False
```

Operacija spajanja `|` vraća novi skup sa svim komponentama iz oba skupa:

```python
my_set | your_set # {1, 2, 3, 4, 5, 6}
```

Opereta iz zbirke `&` vraća novi set koji sadrži samo elemente koje sve verzije imaju zajedno:

```python
my_set & your_set # {2, 3, 4}
```

Operator razlike `-` vraća novi skup sa elementima prvog skupa koji nisu u drugom skupu. U ovom primeru, brojevi 1 i 5 su u `my_set`, ali NIJE U `your_set`:

```python
my_set - your_set # {1, 5}
```

Operater za razliku jednačine ``^`` vraća novi skup koji sadrži elemente iz prvog ili drugog skupa, ali ne i oba. U ovom slučaju, 1 i 5 se nalaze u ``my_set``, ali nisu u ``your_set``, pa su uključeni. A broj 6 je u ``your_set``, ali nije u ``my_set``, stoga je i on uključen:

```python
my_set ^ your_set # {1, 5, 6}
```

Svaka od ovih operetica takođe ima svoju operetiku mešovitog porekla ako dodate znak jednakosti pored nje. Ove operetike direktno postavljaju rezultat prvih operacija u matematičkom izrazu:

```python
|= &= -= ^=
```

Na primer, operacija `-=` pronalazi razliku između setova i ispravlja prvi set sa tim rezultatima:

```python
my_set -= your_set
```

Nakon ovoga, `my_set` će biti ažurirano na `{1, 5}`:

```python
print(my_set) # {1, 5}
```

Možete proveriti da li je element u skupu ili nije koristeći operatora `in`. Ovde, proveravamo da li je 5 u `my_set`. Rezultat će biti vrednost tačno ili netačno `True` ili `False`:

```python
print(5 in my_set)
```

Ovo je osnova setova. Vrlo je korisno kada ne želite da skladištite vrednosti u određenom redosledu, i kada vam je potrebno da sačuvate samo jedinstvene vrednosti.
