# --description--

Dodavanje ili oduzimanje brojeva je jedna od uobičajenih aritmetičkih operacija.

Stoga, postoje posebni operateri za njega:

- Povećanjem `++` povećava vrednost parametra za 1:

```js
let counter = 2;

// works the same as counter = counter + 1, but is shorter
counter++;      

console.log(counter); // 3
```

- Smanjuje `--` smanjuje vrednost parametra za 1:

```js
let counter = 2;

// works the same as counter = counter - 1, but is shorter
counter--;

console.log(counter); // 1
```

Operatori `++` i `--` mogu biti postavljeni ispred ili iza promenljive.

- Wakati mwendeshaji anafuata kigezo, huitwa "postfix form": `counter++`.

"Prefix form" je kada operator dolazi pre parametra: `++counter`.

Ova dva kaula rade jednu stvar: dodaju `counter` na `1`.

Da li postoji neka razlika? Da, ali ga možete videti samo koristeći vrednost koju vraća `++/--`.

Hajde da objasnimo. Kao što znaš, svi operatori vraćaju vrednost. Inkrementiranje i dekrementiranje nisu izuzetak. Prefiksni oblik vraća novu vrednost, dok postfiksni oblik vraća staru vrednost (pre inkrementiranja ili dekrementiranja).

Da biste videli razliku, evo primera:

```js
let counter = 1;
let a = ++counter; // (*)

console.log(a); // 2
```

Ukratko:

- Ako rezultati povećanja/smanjenja se ne koriste, nema razlike koji tip da koristite:

```js
let counter = 0;
counter++;
++counter;

console.log( counter ); // 2, the lines above did the same
```

- Ako želite da povećate vrednost i odmah koristite rezultat operacije, potreban vam je tip prefikse:

```js
let counter = 0;
console.log( ++counter ); // 1
```

- Ukoliko želiš da povećaš vrednost, ali koristiš njenu početnu vrednost, potreban ti je tip postfixa:

```js 
let counter = 0;
console.log( counter++ ); // 0
```

# --questions--

## --text--

Šta je rezultat dve izjave od `console.log` u kodu JavaScript ispod?

```js
let counter = 1;
console.log(2 * ++counter); // Statement A
```

```js
let counter = 1;
console.log(2 * counter++); // Statement B
```

## --answers--

Nema razlike; oba izjava od `console.log` će prikazati isti rezultat.

---

Izjava A će prikazati `4` jer `++counter` dodaje vrednost pre množenja. Izjava B će prikazati `2` jer `counter++` dodaje vrednost nakon množenja.

---

Izjava A će prikazati `2` jer je `++counter` sintaktička greška. Izjava B će prikazati `4` jer je `counter++` ispravan način za dodavanje pokazivača.

---

Oba izjava će prikazati `3` zato što se pokazivač dodaje u oba slučaja pre `console.log`.
