# --description--

Ako navedimo sve prirodne brojeve manji od 10 koji su višestruki sa 3 ili 5, dobićemo 3, 5, 6 i 9. Zbir ovih višestrukih brojeva je 23.

Pronađi sve višestruke brojeve po 3 ili 5 ispod vrednosti parametra `number`.

```js
const multiplesOf3Or5 = (number) => {
  var total = 0;

  for(var i = 0; i < number; i++) {
    if(i % 3 == 0 || i % 5 == 0) {
      total += i;
    }
  }
  return total;
};
```
