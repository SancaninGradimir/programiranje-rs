# --description--

Za pozitivan ceo broj $n$, neka je $\sigma(n)$ zbir svih delitelja $n$, na primer $\sigma(6) = 1 + 2 + 3 + 6 = 12$.

Savršeni broj, kako znate, je broj koji ima $σ(n) = 2n$.

Molimo objasnite kombinatorički koeficijent za pozitivne cijele brojeve kao što je $p(n) = \frac{σ(n)}{n}$.

Pronađi zbir svih pozitivnih celih brojeva $n ≤ {10}^{18}$ za koje $p(n)$ ima oblik $k + \frac{1}{2}$, gde je $k$ ceo broj.
