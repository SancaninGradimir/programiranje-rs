# --interactive--

Metoda `.toFixed()` je funkcija ugrađena unutar JavaScript koja formatira brojeve korišćenjem fiksnog zapisa (fixed-point). Posebno je važno kada želite da kontrolišete broj decimalnih cifara u broju, posebno prilikom prikazivanja valuta ili korišćenja preciznih mera.

Funkcija `.toFixed()` se poziva na broj i prima jedan opcioni argument, koji je broj decimalnih cifara koje treba prikazati. Vraća niz znakova koji predstavlja taj broj sa navedenim brojem decimalnih cifara. Evo osnovnog primera kako `.toFixed()` radi:

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed(2)); // "3.14"
```

:::

U ovom slučaju, smanjujemo broj decimalnih mesta na dva. Stoga, `3.14159` postaje `3.14`. Važno je razumeti da `.toFixed()` vraća sekvencu slova, a ne brojeva. Ovo je zato što ova metoda je namenjena više za formatiranje brojeva radi prikaza, nego za dalje proračune.

Njia ya `.toFixed()` zaokružuje broj na najbližu vrednost koja se može predstaviti određenim brojem decimalnih mesta. Ovo svojstvo zaokruživanja je važno razumeti:

:::interactive_editor

```js
console.log((3.14159).toFixed(3));  // "3.142"
console.log((3.14449).toFixed(3));  // "3.144"
console.log((3.14550).toFixed(3));  // "3.146"
```

:::

Kao što vidite, `.toFixed()` inkrementira kada je sledeća cifra `5` ili više, i dekrementira u suprotnom. Ako pozovete `.toFixed()` bez argumenata, koristi cifru `0` kao podrazumevano:

:::interactive_editor

```js
let num = 3.14159;
console.log(num.toFixed()); // "3"
```

:::

Način za `.toFixed()` može biti veoma koristan kada vršite finansijske proračune ili prikazujete cene:

:::interactive_editor

```js
let price = 19.99;
let taxRate = 0.08;
let total = price + (price * taxRate);

console.log("Total: $" + total.toFixed(2)); // "Total: $21.59"
```

:::

U ovom primeru, `.toFixed(2)` osigurava da je ukupan iznos prikazan uvek sa dve decimalne začišlja, što je uobičajeno za valute u mnogim zemljama.

Na kraju, njia ya `.toFixed()` je moćan alat za formatiranje brojeva u JavaScript, posebno kada morate kontrolisati kako se decimalne cifre prikazuju. Iako se koristi više za formatiranje rezultata, zapamtite njegovo ponašanje, posebno kada vam trebaju tačni proračuni.
