# --description--

U prethodnom primeru, imali ste dve opcije `if` ili `else`. Ali šta ako želite više od dve opcije? Možete koristiti izjavu `else if` da dodate više uslova u naš kod. Izjava `else if` omogućava nam da proverimo mnogo uslova i izvršimo deo koda kada je prvi uslov tačan. Ako prvi uslov nije tačan, proveriću sledeći uslov i tako dalje. Ako nijedan uslov nije tačan, izvršiće se deo `else`.

Ovo je primer upotrebe `else if`:

```javascript
let x = 5;

if (x > 10) {
  console.log("x is greater than 10");
} else if (x > 5) {
  console.log("x is greater than 5");
} else {
  console.log("x is less than or equal to 5");
}
```

U ovom primeru, parametar `x` se poredi sa brojem `10` koristeći indikator `>`. Ako je `x` veće od `10`, izvršiće se prvi deo koda. Inače, izjava `else if` će proveriti sledeći uslov. Ako je `x` veće od `5`, izvršiće se drugi deo koda. Inače, izvršiće se deo `else`.
