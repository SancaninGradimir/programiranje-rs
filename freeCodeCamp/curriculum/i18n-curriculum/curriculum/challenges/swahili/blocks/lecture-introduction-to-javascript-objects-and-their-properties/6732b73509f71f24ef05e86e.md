# --interactive--

Kada radite sa JavaScript, često ćete naići na složene strukture podataka koje uključuju ugnježđene objekte i nizove podataka unutar tih objekata. Ove strukture mogu predstavljati bogate podatke sa više nivoa, ali zahtevaju i jasno razumevanje kako pristupiti i upravljati podacima unutar njih. Hajde da istražimo kako efikasno navigirati ovim ugnježdenim strukturama.

Dohvatanje atributa iz ugnježdenih elemenata podrazumijeva korištenje tačke notacije ili zagrade notacije, kao što dobijate atribute od jednostavnih elemenata. Međutim, moraćete da kombinujete ove metode pristupa kako biste došli do unutrašnjosti ugnježdenih struktura.

Na primer, uzmimo neku strukturu unutar koja predstavlja osobu sa kontakt podacima:

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};
```

Da biste dobili poslovni broj telefona za `Alice`, biće potrebno povezati metode pristupa akreditivima/privilejima, kao što je sledeće:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person.contact.phone.work); // "098-765-4321"
```

:::

Takođe možete koristiti zapis sa uglastim zagradama, koji je posebno koristan kada nazivi svojstava sadrže razmake ili specijalne znakove, ili kada koristite promenljive za pristup svojstvima:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  contact: {
    email: "alice@example.com",
    phone: {
      home: "123-456-7890",
      work: "098-765-4321"
    }
  }
};

console.log(person['contact']['phone']['work']); // "098-765-4321"
```

:::

Sada pogledajmo kako možemo pristupiti podacima gde jedno od svojstava objekta ima vrednost niza podataka. Evo primera izmenjenog objekta `person` koji uključuje niz adresa:

```js
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};
```

Evo primera kako dobiti grad adrese posla za `Alice`:

:::interactive_editor

```js
const person = {
  name: "Alice",
  age: 30,
  addresses: [
    { type: "home", street: "123 Main St", city: "Anytown" },
    { type: "work", street: "456 Market St", city: "Workville" }
  ]
};

console.log(person.addresses[1].city); // "Workville"
```

:::

U ovom primeru, `person.addresses` se odnosi na niz adresa. Da bismo dobili drugu adresu u tom nizu, koristimo notaciju uglastih zagrada i izraz `1`. Zatim, koristimo tačkastu notaciju da bismo dobili `city` iz objekta adrese.

Razumevanje kako pristupiti svojstvima u ugnježdenim objektima i nizovima važno je kada radite sa složenim strukturama podataka. U predstojećim radionicama i laboratorijskim vežbama imaćete priliku da vežbate rad sa ovim tipovima struktura podataka.
