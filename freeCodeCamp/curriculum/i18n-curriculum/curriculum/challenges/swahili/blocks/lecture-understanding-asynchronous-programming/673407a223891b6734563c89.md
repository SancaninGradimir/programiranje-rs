# --description--

Fetch API omogućava veb aplikacijama da šalju mrežne zahteve, najčešće radi preuzimanja ili slanja podataka serveru. Ovaj API pruža metodu `fetch()` koju možeš koristiti za slanje tih zahteva. Pogledajmo osnovni primer korišćenja metode `fetch()`:

```js
fetch('https://api.example.com/data')
```

U ovom primeru, vršimo zahtev za `GET` za `https://api.example.com/data`. Ovo će vratiti podatke koji moraju biti konvertovani u format JSON pre upotrebe.

Po podrazumevanju, API Fetch koristi metodu `GET` za dobijanje podataka. Ovo će biti diskutovano u sledećoj lekciji, zajedno sa drugim uobičajenim metodama zahteva za HTTP.

Sada, analizirajmo uobičajene vrste resursa dostupne iz mreže.

U našim veb aplikacijama, potrebni su nam comuni podaci kao što su podaci o vremenu, lista profesija, spisak imena država, kodovi država ili slike zastava država. Korišćenjem ovih podataka možemo učiniti našu aplikaciju informativnijom i interaktivnijom. Zahvaljujući Fetch-u API, možemo dobiti ove resurse iz mreže.

Slike su jedan od najčešće korišćenih resursa. Možeš koristiti `fetch` za tiho ili dinamičko učitavanje slika u zavisnosti od korisničkih radnji i njihovo prikazivanje u svojoj veb aplikaciji.

Tekstualni fajlovi su drugi tip resursa koji su često dostupni. Ovo može uključivati konfiguraционе fajlove, log fajlove, ili čak ceo dokument koji želite prikazati na vašoj veb stranici.

U nekim slučajevima, možete pronaći audio ili video fajlove. API Fetch može da obrađuje i ove vrste resursa, omogućavajući vam da radite sa različitim vrstama podataka.
