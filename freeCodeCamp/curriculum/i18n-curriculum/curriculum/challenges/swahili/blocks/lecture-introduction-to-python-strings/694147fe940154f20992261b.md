# --description--

U prvom kursu, naučili ste kako se svaki karakter u nizu karaktera može identifikovati po svom indeksu (počevši od nule), i pristupiti pomoću oznake zagrada:

```python
my_str = "Hello world"

print(my_str[0])  # H
print(my_str[6])  # w
print(my_str[-1]) # d
```

**Izrezivanje niza znakova** omogućava vam da izvučete deo niza znakova ili radite samo sa određenim delom tog niza. Evo osnovne sintakse:

```python
string[start:stop]
```

Ako želite da izvučete znakove iz jednog obrasca u drugi, ono što morate da uradite je razdvajanje obrasca `start` i `stop` korišćenjem dve tačke:

```python
my_str = 'Hello world'
print(my_str[1:4]) # ell
```

Napomenuti da obrazac `stop` ne uključuje, stoga `[1:4]` izvlači znakove iz obrasca `1`, do, ali bez uključenja, znaka u obrascu `4`.

Takođe možete izostaviti obrazac `start` i `stop`, a Python će koristiti podrazumevanu vrednost (ili opciju) `0` ili kraj sekvence karaktera, po odvojenom. Na primer, ovo se dešava ako izostavite obrazac `start`:

```python
my_str = 'Hello world'
print(my_str[:7])  # Hello w
```

Ovo pokriva sve od šablona `0` do (ali ne uključujući) karaktera koji se nalazi u šablonu `7`. I ovo je ono što se dešava ako izostavite šablon `stop`:

```python
my_str = 'Hello world'
print(my_str[8:])  # rld
```

Hii hutoa kila kitu kuanzia herufi yenye kielezo cha `8` hadi mwisho wa mfuatano wa herufi.

Zapamtite da se sekvenca slova ne menja sa prvobitnom sekvencom slova:

```py
my_str = 'Hello world'
print(my_str[8:])  # rld
print(my_str)  # Hello world
```

Takođe možete ostaviti model `start` i `stop` zajedno, što će dati celokupnu sekvencu slova:

```python
my_str = 'Hello world'
print(my_str[:])  # Hello world
```

Baada ya kielezo cha `start` na `stop`, pia kuna kigezo cha hiari cha `step`, kinachotumika kubainisha ongezeko kati ya kila kielezo katika ukataji.

Hapa kuna sintaksia yake:

```python
string[start:stop:step]
```

U dole navedenom primeru, niz počinje na elementu `0`, stoji pre `11`, i izvlači svaki drugi karakter:

```python
my_str = 'Hello world'
print(my_str[0:11:2])  # Hlowrd
```

Jedan način pomoći koji možete koristiti sa parametrom `step` je da promenite niz karaktera postavljanjem koraka (step) na `-1`, i ostavite `start` i `stop` nepromenjeno:

```python
my_str = 'Hello world'
print(my_str[::-1]) # dlrow olleH
```
