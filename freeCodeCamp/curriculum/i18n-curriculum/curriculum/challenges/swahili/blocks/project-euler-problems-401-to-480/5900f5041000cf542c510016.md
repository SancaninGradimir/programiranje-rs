# --description--

Ako izračunamo $a^2\bmod 6$ za $0 ≤ a ≤ 5$, dobijamo: 0, 1, 4, 3, 4, 1.

Najveći broj $a$ za koji je $a^2 \equiv a \pmod{6}$ je $4$.

Definisaćemo $M(n)$ kao najveću vrednost za $a < n$ takvu da $a^2 \equiv a (\text{mod } n)$. Stoga $M(6) = 4$.

Pronađi $\sum M(n)$ za $1 ≤ n ≤ {10}^7$.
