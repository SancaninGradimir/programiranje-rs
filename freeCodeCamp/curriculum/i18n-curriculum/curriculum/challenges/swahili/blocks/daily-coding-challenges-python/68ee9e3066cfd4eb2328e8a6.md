# --description--
Ako dobijete dve potpune vrednosti (broj redova i broj kolona), vratite matricu (niz podataka redova podataka) ispunjenu nulama (`0`) za zadane dimenzije.

Na primer, ako dobijete `2` i `3`, vratite:

```json
[
  [0, 0, 0],
  [0, 0, 0]
]
```
