# --description--

Acha `r` awe mabaki yanapotengenezwa mgawanyo wa ${(a − 1)}^n + {(a + 1)}^n$ kwa $a^2$.

Na primer, ako je $a = 7$ i $n = 3$, tada je $r = 42: 6^3 + 8^3 = 728 ≡ 42 \\ \text{mod}\\ 49$. Kako se `n` menja, menja se i `r`, ali za $a = 7$ važi da je $r_{max} = 42$.

Za $3 ≤ a ≤ 1000$, izračunaj $\sum{r}_{max}$.
