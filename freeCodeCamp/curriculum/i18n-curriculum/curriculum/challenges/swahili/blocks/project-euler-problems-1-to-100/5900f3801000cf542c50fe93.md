# --description--

`n`! inamaanisha `n` × (`n` − 1) × ... × 3 × 2 × 1

Na primer, 10! = 10 × 9 × ... × 3 × 2 × 1 = 3628800, a zbir svih cifara u broju 10! je 3 + 6 + 2 + 8 + 8 + 0 + 0 = 27.

Pronađi sekvencu cifara `n`!

# --solutions--

```js
let factorial = (n) => n <= 1 ? BigInt(n) : BigInt(n) * BigInt(factorial(--n));

let sumDigits = n => n.toString().split('').map(x => parseInt(x)).reduce((a,b) => a + b);

function sumFactorialDigits(n) {
  return sumDigits(factorial(n));
}
```
