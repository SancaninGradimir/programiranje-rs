# --description--

Naučimo se o terminalu i kako se razlikuje od linije komandi.

Komandna linija je osnovni interfejs za unos teksta koji omogućava korisniku da unese „komandu“, obično niz karaktera, i pošalje ili izvrši, obično pritiskom na taster „Enter“. Često ćete videti interfejse komandne linije unutar terminala.

Terminal je specijalni program koji pruža interfejs komandne linije za izvršavanje sistemskih komandi, iznad standardnih operacija čitanja/pisanja. Takođe možete čuti za „terminal emulatore“, koji su programi koji okružuju osnovni terminalni interfejs kako bi pružili dodatne funkcije ili funkcionalnosti. Ali u opšnim ciljevima, možete ih nazvati i „terminal“.

Na kraju, takođe ćemo morati da razgovaramo o "shells". Shell je program koji radi sa linijom komandi i tumači vaš unos kao komandu, vraćajući rezultate.

Ali kako da pristupiš terminalu na svom sistemu?

U stvari, ako koristite modernu verziju Windows-a, postoji mogućnost da imate dve različite opcije. Windows direktno pruža dve različite shell programe: Powershell i Command Prompt.

Ova dva shela imaju sopstvene programe termina (koji dele ime odgovarajućeg shela), pa možete koristiti meni programa da pokrenete PowerShell terminal ili Command Prompt terminal.

Windows takođe pruža novi i moderan emulator terminala poznat kao Microsoft Terminal.

Možete instalirati ovu aplikaciju iz Windows Store-a, a zatim je pokrenuti kao što to radite sa svojim terminalima za PowerShell ili Command Prompt. Prednost Microsoft Terminal-a je u tome što vam omogućava da dobijete PowerShell i Command Prompt shellove iz jedne aplikacije, zajedno sa vašim Linux shellom ako ste instalirali Windows Subsystems for Linux.

MacOS pruža terminal poznat kao "Terminal". Možete ga pronaći putem pretrage Spotlight-a, ili putem pokretača vaših aplikacija.

MacOS takođe pruža podršku za emulatore terminala drugih ljudi, kao što su iTerm ili Ghostty. Ove aplikacije možete instalirati sa njihovih početnih stranica.

Na Linuksu, standardni terminal (i način na koji ga pokrećete) zavisi mnogo od distribucije i okruženja za radnu površinu koje koristite. Postoji puno pomoći i za druge emulatore terminala. Na primer, možete konfigurisati kitty na Arch Linuxu – i možete ga pokrenuti pomoću menadžera aplikacija kao što je wofi.

Linux ima veoma veliki izbor programa za terminal. Većina njih se može instalirati putem menadžera paketa vaše distribucije (distro).

Razlika između "terminala", "shell-a" i "komandne linije" ponekad može izgledati zanemarivo. I u velikoj meri, ove reči se mogu koristiti kao zamena jedna za drugom. Ali važno je znati da imaju specifična značenja, i šta su ta značenja.
