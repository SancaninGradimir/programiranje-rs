# --description--

Atribut za `flex` je zapravo svojstvo skraćivanja za tri atributa koje možete dodati na promenljivu del. Ova svojstva utiču na način na koji promenljive delove mere unutar svog kontejnera. Videli ste neke svojstva skraćivanja ranije, ali još uvek niste spomenuli zvanično.

Karakteristike skraćenja su karakteristike CSS koje vam omogućavaju postavljanje vrednosti za mnogo drugih karakteristika CSS istovremeno. Koristeći karakteristiku skraćenja, možete napisati stilove tabele koji su kraći (a često i čitljiviji), štedeći vreme i snagu.

U ovom slučaju, `flex` je skraćenica za `flex-grow`, `flex-shrink` i `flex-basis`.

<img src="https://cdn.freecodecamp.org/curriculum/odin-project/flex-box/flexbox-04.png" alt="CSS code setting the flex property to 1 for a div element.">

Katika picha hapo juu, `flex: 1` inalingana na: `flex-grow: 1`, `flex-shrink: 1`, `flex-basis: 0`.

Veoma često ćeš videti da je skraćeno svojstvo `flex` navedeno samo sa jednom vrednošću. U tom slučaju ta vrednost se koristi za `flex-grow`. Dakle, kada si postavio `flex: 1` na naše `div` elemente, zapravo si naveo skraćeno svojstvo `flex: 1 1 0`.
