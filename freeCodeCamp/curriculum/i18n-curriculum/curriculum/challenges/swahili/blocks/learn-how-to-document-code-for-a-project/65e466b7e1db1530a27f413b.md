# --description--

U sistemima datoteka računara, `root directory` se odnosi na glavni ili višesložni direktorijum. To je kao gornja fioka u ormarić za fajlove, koja sadrži sve ostale foldere i fajlove.

`main folder` je glavni direktorijum u kojem se projekat ili set fajlova skladišti.

`subfolder` je kao mala fioka unutar gornje, koja se nalazi unutar druge foldere.

Na primer, u projektu pod nazivom `MyProject`, `root directory` će biti foldera `MyProject`, i bilo koja mapa unutar njega, kao što je `Images`, biće `subfolder`.

### --feedback--

Poddirektorijum se nalazi unutar drugog direktorijuma. `root directory` je glavni direktorijum, nije poddirektorijum.

---

Ime fajla

### --feedback--

`root directory` je folder/fascikla, i vrsta fajla. To je glavno mesto gde se projekat fajlovi i folderi čuvaju.

---

Jezik programiranja

### --feedback--

`root directory` je koncept u sistemima fajlova, nije programski jezik. Odnosi se na glavni direktorijum projekta.
