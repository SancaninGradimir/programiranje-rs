# --interactive--

Možda ste već radili sa osobinama elemenata u svojim HTML i CSS projektima. Ali da li ste znali da možete dodavati osobine elemenata svojim HTML elementima iz JavaScript fajla?

U ovoj lekciji naučićete kako da koristite metodu `setAttribute()` za dodavanje atributa elementima u vašim HTML dokumentima.

Evo osnovne sintakse:

```js
setAttribute(attribute, value);
```

Pogledajmo nekoliko primera kako bismo bolje razumeli korišćenje metode `setAttribute()`.

U ovom prvom primeru imamo `p` element unutar HTML-a:

```html
<p id="para">I am a paragraph</p>
```

Da bismo dodali atribut `class`, prvo moramo dobiti referencu na element `p` pomoću metode `getElementById()`. Zatim možemo koristiti metodu `setAttribute()` da dodamo atribut `class` i postavimo njegovu vrednost na `my-class`.

Koristimo `outerHTML` da ispišemo ceo HTML element zajedno sa njegovim atributima.

:::interactive_editor

```html
<p id="para">I am a paragraph</p>
<script src="index.js"></script>
```

```js
const para = document.getElementById("para");
para.setAttribute("class", "my-class");

console.log(`${para.outerHTML}`);
```

:::

Ako element već ima određeni atribut u HTML-u, možete promeniti njegovu vrednost pomoću metode `setAttribute()`.

U ovom primeru imamo `div` element sa atributom `class` postavljenim na `my-class`:

```html
<div class="my-class"></div>
```

Da bismo promenili vrednost atributa `class` u `example`, možemo ponovo koristiti metodu `setAttribute()`.

:::interactive_editor

```html
<div class="my-class"></div>
<script src="index.js"></script>
```

```js
const divEl = document.querySelector(".my-class");
divEl.setAttribute("class", "example");

console.log(`${divEl.outerHTML}`);
```

:::

Koji su realni primeri korišćenja metode `setAttribute()`?

Ako pravite dinamičku galeriju slika, možda ćete morati da promenite atribut `src` elementa slike kada korisnik klikne na umanjenu sliku.

Drugi primer je kada radiš sa validacijom forme i moraš dodati određeno svojstvo kao `required` ili `minlength` za element/polje `input`.
