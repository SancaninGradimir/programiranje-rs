# --explanation--

`User Interface (UI)` što korisnik vidi (ekrani, dugmad, boje).

`User Experience (UX)` je kako se korisnik oseća (jednostavnost, jasnoća, lakoća korišćenja).

Kvalitetni proizvodi uzimaju u obzir oba: jasnu sliku i zadovoljavajuće iskustvo.
