# --interactive--

Datumi i vremena u JavaScript nisu bile lako za korišćenje svaki put. U profesionalnom kodu, možda ćete koristiti biblioteku koja je rešila mnogo takvih problema za vas.

Međutim, biće trenutaka kada ćete morati da radite sa JavaScript, koji je ugrađen (built-in) od `Date`.

U ovoj lekciji, pregledaćemo osnove koncepta `Date` i neke od najčešćih metoda koje ćete koristiti kada radite sa datumima i vremenima u JavaScript.

Objekat `Date` je ugrađeni (built-in) element u JavaScript koji vam omogućava rad sa datumima i vremenima. Evo osnovne sintakse za kreiranje novog `Date`:

```js
const now = new Date();
```

Funkcija `new` se koristi za kreiranje nove instance objekta `Date`, i objektu `Date` je zatim dodeljena promenljiva vrednost `now`. Ako unesete vrednost `now` u konzolu, videćete trenutni datum i vreme prema sistemskom satu računara koji izvršava kod.

Ovo je primer odgovora koji ćete videti ako unesete vrednost `now` u konzolu:

:::interactive_editor

```js
const now = new Date();
console.log(now);
```

:::

Za vreme, koristi vojni format vremena, stoga `14:30` je `2:30 PM`. `GMT-0700` je odstupanje (offset) vremenske zone, i `Pacific Daylight Time` je naziv vremenske zone.

Takođe možete uneti određeni datum i vreme u objekat `Date` tako što ćete navesti vrednosti za godinu, mesec, dan, sat, minut, sekundu i milisekundu kao argumente.

Ovo je primer kreiranja novog objekta za `Date` sa određenim datumom i vremenom:

:::interactive_editor

```js
const specificDate = new Date("July 4, 1776 12:00:00");

console.log(specificDate);
```

:::

Ovo je važno kada vam treba raditi sa specifičnim datumima i vremenima umesto trenutnog datuma i vremena. Unos mora uvek biti sekvenca karaktera, a njegova struktura bi trebalo da bude prepoznata od strane `Date`.

Da biste dobili trenutni datum i vreme, možete koristiti metodu ya `Date.now()` koja vraća broj milisekundi od `January 1, 1970, 00:00:00 UTC`. Ovo je poznato kao vreme Unix epoha.

Vremenski znak Unix epoha je uobičajen način za predstavljanje datuma i vremena u računarskim sistemima jer je ceo broj koji se lako skladišti i obrađuje. UTC znači Univerzalno koordinisano vreme (Universal Time Coordinated), koje je osnovni vremenski standard koji svet koristi za regulisanje sati i vremena.

Ako vam je potrebno dobiti dan meseca na osnovu trenutnog datuma, možete koristiti metodu `getDate`. Evo primera korišćenja metode `getDate`:

:::interactive_editor

```js
const now = new Date();
const date = now.getDate();
console.log(date); 
```

:::

U ovom primeru, kreiramo novi objekat datuma koristeći objekat `Date` i dodeljujemo mu vrednost modifikacije `now`. Zatim pozivamo metodu `getDate` na objektu `now` da bismo dobili dan meseca i dodeljujemo mu vrednost modifikacije `date`. Na kraju, zapisujemo vrednost `date` u konzolu, što će ispisati trenutni dan meseca.

`getDate` vraća vrednost broja između `1` i `31`, u zavisnosti od dana u mesecu. Ako datum nije validan, vraća `NaN` (Nije broj).

Da biste dobili mesec, možete koristiti metodu `getMonth`, kao što sledi:

:::interactive_editor

```js
const now = new Date();
const month = now.getMonth();
console.log(month);
```

:::

Mesec počinje od nule, pa je zato `January` jednako `0`, `February` jednako `1`, itd. Ako je mesec nevažeći, vratiće `NaN`.

Ako vam je potrebno da dobijete punu godinu, možete koristiti metodu `getFullYear` kao što sledi:

:::interactive_editor

```js
const now = new Date();
const year = now.getFullYear();
console.log(year); 
```

:::

Ako godina nije važeća, vratiće se `NaN`.

Postoji mnogo više načina dostupnih na `Date`, uključujući `getHours`, `getMinutes`, `getSeconds`, itd. Preporučujem vam da istražite dalje koristeći dokumentaciju Mozille ili druge resurse.
