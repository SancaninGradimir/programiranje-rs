# --interactive--

Naučite se o DOM i zašto je veoma važno za razvoj mreže. DOM stoji za Document Object Model. To je programski interfejs koji nam omogućava da interagujemo sa dokumentima HTML.

Korišćenjem DOM-a možete dodavati, menjati ili brisati elemente na veb stranici. Možete čak napraviti svoju stranicu interaktivnom tako što ćete omogućiti elementima da slušaju i reaguju na događaje.

U DOM-u, HTML dokument je predstavljen kao stablo čvorova. Svaki čvor atributa predstavlja HTML atribut iz HTML dokumenta.

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>DOM Example</title>
  </head>
  <body>
    <h1>What is the DOM?</h1>
    <h2>Let's learn about the DOM</h2>
  </body>
</html>
```

Evo dijagrama koji predstavlja osnovnu strukturu DOM-a našeg primera:

```md
Document
========

HTML
-----

- Head
  - Title
- Body
  - H1
  - H2   
```

Naravno, ovo može biti mnogo detaljnije i složenije u zavisnosti od HTML strukture stranice.

HTML dokument je korenski čvor u DOM stablu. Ima jedan čvor dete, element `html`. Ovo je korenski element HTML dokumenta, jer svi drugi čvorovi potiču od njega.

Element `html` ima dva deteta: `head` i `body`.

Element `head` sadrži metapodatke o dokumentu. On pruža važne informacije o veb stranici.

Element `body` sadrži vidljivi sadržaj veb stranice.

Ovim elementima možete pristupiti koristeći JavaScript, raditi sa njima u svom kodu i čak ih dinamički menjati. To je snaga DOM-a i veb API-ja.

Da biste pristupili ovim elementima u JavaScript-u, možete koristiti metode `getElementById()` i `querySelector()`. Ove metode su veb API-ji jer pružaju standardni način interakcije sa DOM-om pomoću JavaScript-a.

Za `getElementById()`, možete dobiti nešto što predstavlja karakteristiku HTML sa `id` navedenom. Zapamtite da `id` mora biti jedinstven u svakom dokumentu od HTML, stoga će ova metoda vratiti samo jedan objekat od `Element`. Ovde možete videti primer:

```js
const container = document.getElementById("container");
```

Ova linija koda za JavaScript dobija atribut sa vrednošću `id` od `container` i postavlja taj objekat u varijablu JavaScript. Morate proslediti `id` unutar navodnika kao argument. Ako ovo ispišete na konzolu, videćete rezultat:

:::interactive_editor

```html
<div id="container">
  <h1>Hello, World!</h1>
  <p>Welcome to learning about the DOM.</p>
</div>
<script src="index.js"></script>
```

```js
const container = document.getElementById("container");
console.log(container); 
```

:::

`querySelector()` je širi od `getElementById()`. Za `querySelector()`, možete pronaći prvi element u dokumentu HTML koji odgovara izboru CSS usvojenom kao argument.

U ovom primeru, pronaći ćete element koji odgovara selektoru `section`:

:::interactive_editor

```html
<section>
  <h2>Section Title</h2>
  <p>This is a section of the webpage.</p>
</section>
<script src="index.js"></script>  
```

```js
const sectionEl = document.querySelector("section");
console.log(sectionEl);
```

:::

Ako želite da selektujete element po nazivu klase objekta koristeći `querySelector()`, morate postaviti tačku prefiks (`.`) pre imena klase. Evo primera:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="highlight">
  <p>This content is highlighted.</p>
</div>
<script src="index.js"></script>
```

```css
body {
  background-color: lightgray;
}

.highlight {
  background-color: yellow;
}
```

```js
const highlightEl = document.querySelector(".highlight");
console.log(highlightEl);
```

:::

Pia pronađi druge načine za poređenje mnogo stvari, kao što su `getElementsByClassName()` i `querySelectorAll()`. Naučićete više o ovome u narednim lekcijama.

Da biste razumeli kako da koristite API za mrežu i efikasno kontrolišete DOM, možete kreirati moćne i interoperabilne mrežne aplikacije.
