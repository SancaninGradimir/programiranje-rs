# --description--

Neka je $n$ pozitivan ceo broj. Tri celobrojka ($a$, $b$, $c$) nazivaju se rasporedak broja $n$ ako je:

- $1 ≤ a ≤ b ≤ c$
- $a \times b \times c = n$.

Odrediti $f(n)$ kao $a + b + c$ za particiju ($a$, $b$, $c$) broja $n$ koja minimizira $\frac{c}{a}$. Može se dokazati da su ova tri dela jedinstvena.

Na primer, $f(165) = 19$, $f(100\\,100) = 142$ i $f(20!) = 4\\,034\\,872$.

Izračunaj $f(43!)$.
