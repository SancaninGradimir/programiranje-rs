# --description--

Ponekad će postojati situacija u kojoj znate više o tipu nego što to zna TypeScript. Ovde je verifikacija tipa korisna.

Evo primera korišćenjem metode `querySelector()` za dobijanje elementa sa `id` od `submit`:

```ts
const submitBtn = document.querySelector("#submit");
```

Trenutno, TypeScript zna samo da `submitBtn` je nekakav tip od `Element` ili možda `null`. TypeScript neće proveriti odgovarajuću HTML kako bi to utvrdio.

Ovo je dobar trenutak da se koristi validacija tipa kako bi TypeScript saznao koji je tip ovog elementa. Evo primera:

```ts
const submitBtn = document.querySelector("#submit") as HTMLButtonElement;
```

Trenutno, TypeScript će ovo tretirati kao komponentu od `button` zbog tog tipa validacije.

Postoje ograničenja pri korišćenju ključne reči `as`. Na primer, ne možeš da radiš stvari poput ove, a da TypeScript ne prijavi grešku:

```ts
// Conversion of type 'string' to type 'number' may be a mistake because neither type sufficiently overlaps with the other. If this was intentional, convert the expression to 'unknown' first.
const age = "awesome" as number;
```

U ovom primeru, nije sasvim jasno pokušati da se potvrdi da niz znakova `"awesome"` treba proveravati kao tip broja. TypeScript dozvoljava samo validaciju tipa između kompatibilnih ili povezanih tipova.

Jedan način da se ukloni ta greška je korišćenje dvofaktorske autentifikacije na ovaj način:

```ts
const age = "awesome" as unknown as number;
```

Iako ovo funkcioniše u TypeScript, i dalje nije preporučljivo jer emotivno nije dobro razumevano.

Drugi način za pisanje vrste potvrde je korištenje sintakse uglastih zagrada, kao što je ovo:

```ts
const submitBtn = <HTMLButtonElement>document.querySelector("#submit");
```

Moraš biti pažljiv sa ovom sintaksom kada koristiš `tsx` datoteke, jer će TypeScript pomisliti da pokušavaš da prikažeš komponentu i prijaviće grešku poput ove:

```ts
// file: index.tsx
// This JSX tag requires 'React' to be in scope, but it could not be found.
// JSX element 'HTMLButtonElement' has no corresponding closing tag.
const submitBtn =  <HTMLButtonElement>document.querySelector("#submit");
```

U mnogim slučajevima, videćete da prva sintaksa koja koristi ključnu reč `as` je korišćena više od druge opcije.
