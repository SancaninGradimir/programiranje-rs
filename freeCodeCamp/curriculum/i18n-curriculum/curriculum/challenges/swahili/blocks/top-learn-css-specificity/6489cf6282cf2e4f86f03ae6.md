# --description--

```css
/* rule 1 */
#subsection .list {
  background-color: yellow;
  color: blue;
}

/* rule 2 */
#subsection .main .list {
  color: red;
}
```

U ovom poslednjem primeru, oba pravila koriste selektore za ID i klasu objekata, tako da nijedno pravilo ne koristi specifičniji selektor od drugog. Niza pa proverava broj svakog tipa selektora. Oba pravila imaju samo jedan ID selektor, ali pravilo 2 ima mnogo više selektora klase objekata, stoga je pravilo 2 specifičnije!

Iako bi deklaracija `color: red` zauzela prioritet, deklaracija `background-color: yellow` i dalje bi bila primenjiva jer ne postoji deklaracija koja joj proturudiše.
