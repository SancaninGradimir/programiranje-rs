# --interactive--

Svojstvo `z-index` u CSS se koristi za kontrolu vertikalnog redosleda složene elemenata sa specifičnim pozicijama koji se preklapaju na stranici. Kada su više elemenata složena jedan preko drugog, vrednost `z-index` određuje koji se element prikazuje na vrhu. Kako vrednost `z-index` raste, element se približava gledaocu, dok niže vrednosti postavljaju element dalje u kontekstu složene pozicije.

Međutim, `z-index` radi samo za elemente sa specifičnom pozicijom, što znači da element mora imati vrednost pozicije različitu od `static`, kao što su `relative`, `absolute`, ili `fixed`. Podrazumevana vrednost za `z-index` je `auto`, koja postavlja element u podrazumevani red sortiranja. Pogledajmo primer sa tri kutije:

```html
<div class="container">
  <div class="box1">Box 1</div>
  <div class="box2">Box 2</div>
  <div class="box3">Box 3</div>
</div>
```

Sada, možemo koristiti određene načine da učinimo da ti kutije sudare kako slijedi:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="container">
  <div class="box1">Box 1</div>
  <div class="box2">Box 2</div>
  <div class="box3">Box 3</div>
</div>
```

```css
.container {
  position: relative;
  width: 300px;
  height: 300px;
  border: 1px solid black;
}

.box1 {
  position: absolute;
  z-index: 1;
  background: lightcoral;
  top: 20px;
  left: 20px;
  width: 100px;
  height: 100px;
}

.box2 {
  position: absolute;
  z-index: 3;
  background: gold;
  top: 40px;
  left: 40px;
  width: 100px;
  height: 100px;
}

.box3 {
  position: absolute;
  z-index: 2;
  background: lightgreen;
  top: 60px;
  left: 60px;
  width: 100px;
  height: 100px;
}
```

:::

Za kontejner, prostor će biti `relative` i sve kutije raspoređene unutra će biti postavljene pomoću prostora `absolute`. Svaka kutija ima različitu vrednost od `z-index` koja uzrokuje da se kutije rasporede jedna na drugu.

Možete smatrati `z-index` kao način za kreiranje slojeva na veb stranici, i elementi sa najvišim vrednostima od `z-index` biće postavljeni preko onih sa nižim vrednostima. Ovo je posebno važno za kontrolisanje kako se ponašaju preklapajući elementi u složenim rasporedima, kao što su modalni prozori, pop-up prozori ili naslovci alata (tooltip).
