# --description--

Python Pruža nekoliko ugrađenih načina koji olakšavaju rad sa nizovima karaktera. Uključuju, ali nisu ograničeni na, sledeće:

- `upper()`: Konvertuje novi niz slova gde su sva slova promenjena u VELIKA SLOVA.

```python
my_str = 'hello world'

uppercase_my_str = my_str.upper()
print(uppercase_my_str)  # HELLO WORLD
```

- `lower()`: Konvertuje novu sekvencu slova gde su sva slova promenjena u mala slova.

```python
my_str = 'Hello World'

lowercase_my_str = my_str.lower()
print(lowercase_my_str)  # hello world
```

- `strip()`: Vraća novi niz slova gde su određena početna i krajnja slova uklonjena. Ako nije dat argument, uklanja početne i krajnje razmake.

```python
my_str = '  hello world  '

trimmed_my_str = my_str.strip()
print(trimmed_my_str)  # "hello world"
```

- `replace(old, new)`: Vrati novu sekvencu karaktera gde su sve instance od `old` zamenjene sa `new`.

```python
my_str = 'hello world'

replaced_my_str = my_str.replace('hello', 'hi')
print(replaced_my_str)  # hi world
```

- `split(separator)`: Razdvaja niz znakova po zadatom separatoru i kreira niz stringova znakova. Ako nije definisan separator, deli po razmaku.

```python
my_str = 'hello world'

split_words = my_str.split()
print(split_words)  # ['hello', 'world']
```

- `join(iterable)`: Spaja iterable elemente u string sekvencu koristeći separator.

```python
my_list = ['hello', 'world']

joined_my_str = ' '.join(my_list)
print(joined_my_str)  # hello world
```

- `startswith(prefix)`: Vraća vrednost istina ili laž koja ukazuje da li niz znakova počinje sa navedenim prefiksom.

```python
my_str = 'hello world'

starts_with_hello = my_str.startswith('hello')
print(starts_with_hello)  # True
```

- `endswith(suffix)`: Vraća booleanski vrednost koja ukazuje da li string završava sa navedenim sufiksom.

```python
my_str = 'hello world'

ends_with_world = my_str.endswith('world')
print(ends_with_world)  # True
```

- `find(substring)`: Dohvata model prvog događaja `substring`, ili `-1` ako nije pronađen.

```python
my_str = 'hello world'

world_index = my_str.find('world')
print(world_index)  # 6
```

- `count(substring)`: Vraća broj puta koliko manja sekvenca pojavljuje unutar sekvence.

```python
my_str = 'hello world'

o_count = my_str.count('o')
print(o_count)  # 2
```

- `capitalize()`: Vraća novi niz slova gde je prvo slovo postavljeno kao veliko, a ostala mala.

```python
my_str = 'hello world'

capitalized_my_str = my_str.capitalize()
print(capitalized_my_str)  # Hello world
```

- `isupper()`: Vrati `True` ako su sva slova u nizu velika i `False` inače.

```python
my_str = 'hello world'

is_all_upper = my_str.isupper()
print(is_all_upper)  # False
```

- `islower()`: Vrati `True` ako su sva slova u nizu mala i `False` ako ne.

```python
my_str = 'hello world'

is_all_lower = my_str.islower()
print(is_all_lower)  # True
```

- `title()`: Vraća novi niz slova u kojem je prvo slovo svake reči kapitalizovano.

```python
my_str = 'hello world'

title_case_my_str = my_str.title()
print(title_case_my_str)  # Hello World
```
