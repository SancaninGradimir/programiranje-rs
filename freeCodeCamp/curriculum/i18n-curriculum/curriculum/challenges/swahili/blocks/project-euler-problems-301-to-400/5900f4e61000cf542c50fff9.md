# --description--

Neka je $T(n)$ $n$-ti triangularni broj, pa $T(n) = \frac{n(n + 1)}{2}$.

Neka je $dT(n)$ broj delitelja $T(n)$. Primer: $T(7) = 28$ i $dT(7) = 6$.

Neka je $Tr(n)$ broj trojki ($i$, $j$, $k$) gde $1 ≤ i &lt; j &lt; k ≤ n$ i $dT(i) > dT(j) > dT(k)$. $Tr(20) = 14$, $Tr(100) = 5\\,772$ i $Tr(1000) = 11\\,174\\,776$.

Izračunaj $Tr(60\\,000\\,000)$. Navedi poslednjih 18 cifara svog odgovora.
