# --description--

Isti li se ikada zapitali šta se dešava iza kulisa kada unesete URL i pritisnete Enter? Kako pretraživač pretvara kod u veb stranice koje vidimo?

Razumevanje kako pretraživač prikazuje web stranicu je važno za programere koji žele poboljšati performanse i povećati korisničko iskustvo.

U ovom tutorijalu, objasnićemo proces prikazivanja stranice, razbijajući ga na jasne i lako razumljive korake.

Prvo, kran analizira HTML za izgradnju DOM.

Proces počinje sa pretraživačem koji dohvaća sadržaj HTML sa stranice. Pretraživač analizira HTML kako bi izgradio Model objekta dokumenta, ili DOM — hijerarhijsku strukturu koja predstavlja sadržaj stranice.

Razmisli o ovom jednostavnom HTML primeru:

```html
<html>
  <body>
    <h1>Hello, World!</h1>
    <p>Welcome to our website.</p>
  </body>
</html>
```

Pregledač obrađuje ovo kako bi napravio DOM stablo sa elementom `html` kao korenom, koje sadrži `body`, a ono zauzvrat sadrži elemente `h1` i `p`.

Zatim pregledač obrađuje CSS i gradi CSS Object Model (CSSOM). To je još jedna struktura u obliku stabla koja određuje kako elementi treba da budu stilizovani.

Kwa CSS hii:

```css
h1 {
  color: blue;
}
p {
  font-size: 16px;
}
```

CSS Object Model određuje da su elementi `h1` plave boje, a elementi `p` imaju veličinu fonta od 16 piksela.

Pregledač zatim spaja DOM i CSS Object Model kako bi napravio stablo prikaza. Ovo stablo sadrži samo čvorove potrebne za prikaz stranice, pri čemu svaki ima odgovarajuće stilove.

Za naš primer, prikazno drvo uključuje elemente `h1` i `p`, pravilno dekorisane.

Za prikazivanje na ekranu, pretraživač računa stvarni prostor i veličinu svakog elementa u procesu koji se naziva raspored (layout) ili reflow. Ovo osigurava da elementi izgledaju na ispravnom mestu i sa tačnim dimenzijama na ekranu.

Konačno, pretraživač renderuje piksele na ekranu, prikazujući svaki element u skladu sa izračunatim stilovima i rasporedom. Na složenim stranicama, ovo može uključivati više slojeva koji su spojeni kako bi se stvorio konačni vidljivi rezultat.

Na kraju, od renderovanja HTML i CSS do crtanja piksela na ekranu, proces prikaza pretraživača je niz složenih koraka.
