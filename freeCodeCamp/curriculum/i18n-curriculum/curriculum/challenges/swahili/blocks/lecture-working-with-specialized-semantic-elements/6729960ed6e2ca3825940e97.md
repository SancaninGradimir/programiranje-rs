# --interactive--

Kodni elementi unutar linije koriste se za predstavljanje malih kodnih fragmenata unutar teksta. Uobičajena upotreba kodnog elementa je u tehničkim člancima i dokumentacionim stranicama.

Ovo je primer korišćenja funkcije `code` za prikazivanje fragmenta koda CSS:

:::interactive_editor

```html
<p>
  To set the text color to blue in CSS, use the following code:
  <code>color: blue;</code>
</p>
```

:::

U ovom primeru, atribut CSS `color` se koristi za postavljanje boje teksta na `blue`. Omotanjem ovog dela koda unutar oznaka `<code>`, prikazuje pregledaču da je tekst deo koda unutar linije.

Pregledač će primeniti osnovne stilove na sadržaj unutar elementa sa `code`. Osnovni stil je monospacen font.

Element ``code`` je namenjen da predstavlja jednu liniju koda. Ako želite da predstavite više linija koda, moraćete da postavite element ``code`` unutar elemenata sa unapred formatiranim tekstom.

Funkcija unapred formatiranog teksta koristi se za predstavljanje unapred formatiranog teksta. Evo primera korišćenja funkcije unapred formatiranog teksta da bi prikazala deklaraciju CSS:

:::interactive_editor

```html
<pre>
  <code>
    body {
      color: red;
    }
  </code>
</pre>
```

:::

Kada koristite funkciju `pre`, morate obratiti pažnju na razmak jer će se prikazati kako je napisano unutar dokumenta HTML.

U pretraživaču ćete videti da je kod pomeren za nekoliko razmaka udesno. Ako promenite indentaciju u primeru koda, videćete razliku u formatiranju na ekranu.

Kada je u pitanju uključivanje kodnih primera unutar vašeg dokumenta za HTML, trebalo bi vam koristiti funkciju `code` za kratke primere u okviru linije.

Ako je potrebno prikazati dugačke delove koda, onda ćete morati da koristite komponente `pre` i `code`.
