# --description--

U sledećim koordinatama x, y, a n je pozitivan ceo broj.

$$\frac{1}{x} + \frac{1}{y} = \frac{1}{n}$$

Može se potvrditi da kada `n` = 1260 postoji 113 različitih rešenja, i ovo je najmanja vrednost od `n` koja ukupan broj različitih rešenja prelazi sto.

Koja je minimalna vrednost `n` gde broj različitih rešenja prelazi četiri miliona?

**Napomena:** Ovaj izazov je teža verzija Izazova 108, i pošto je veoma daleko od granica metode sirove snage, zahteva pametnu implementaciju.
