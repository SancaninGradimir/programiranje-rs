## Uvod u CSS Flexbox i Adaptivan primer

- **Definicija**: CSS flexbox je model rasporeda dimenzije koji omogućava aranžiranje elemenata u redovima i kolonama unutar kontejnera. 
- **Fleksibilni primer**: Ovaj primer objašnjava kako se fleksibilne komponente raspoređuju unutar fleksibilnog kontejnera. Svaki fleksibilni kontejner ima dve ose: glavnu osu i poprečnu osu.

## Karakteristika od `flex-direction`

- **Opis**: Ova karakteristika određuje smer glavne ose. Podrazumevana vrednost za `flex-direction` je `row`, što postavlja sve dinamičke elemente u jedan red, u smeru jezika vašeg pretraživača (od leve ka desne ili od desne ka levoj).
- **`flex-direction: row-reverse;`**: Ovo menja raspored elemenata u redu.
- **`flex-direction: column;`**: Ovo će postaviti dinamičke elemente vertikalno, umesto horizontalno.
- **`flex-direction: column-reverse;`**: Ovo menja raspored vertikalnih dinamičkih elemenata.

:::interactive_editor

```html
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  flex-direction: row;
  gap: 10px;
}

.box {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}
```

:::

## Karakteristika za `flex-wrap`

- **Opis**: Ovo svojstvo određuje kako se fleksibilni segmenti uklapaju unutar dinamičkog kontejnera da bi odgovarali dostupnom prostoru. `flex-wrap` može uzeti jednu od tri moguće vrednosti: `nowrap`, `wrap`, i `wrap-reverse`.
- **`flex-wrap: nowrap;`**: Ovo je podrazumevana vrednost. Fleksibilni segmenti neće se uklapati na novu liniju, čak i ako njihova širina prelazi širinu kontejnera.
- **`flex-wrap: wrap;`**: Ovo svojstvo će uklapati segmente kada pređu širinu svog kontejnera.
- **`flex-wrap: wrap-reverse;`**: Ovo svojstvo će uklapati fleksibilne segmente u suprotnom redosledu/pravcu.
- **Svojstvo `flex-flow`**: Ovo svojstvo je alias za `flex-direction` i `flex-wrap`.

:::interactive_editor

```html
<div class="container">
  <div class="box">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
  <div class="box">4</div>
  <div class="box">5</div>
</div>
```

```css
.container {
  display: flex;
  flex-wrap: wrap;
  width: 150px;
  background: #f0f0f0;
}

.box {
  width: 60px;
  padding: 10px;
  margin: 5px;
  background: skyblue;
  text-align: center;
}
```

:::

## Svojstvo `justify-content`

- **Opis**: Ovaj atribut poravnava male komponente duž glavne ose transformabilnog alata.
- **`justify-content: flex-start;`**: U ovom slučaju, transformabilni delovi će biti poravnati na početku glavne ose. Ovo može biti horizontalno ili vertikalno.
- **`justify-content: flex-end;`**: U ovom slučaju, transformabilni delovi su poravnati na kraju glavne ose, horizontalno ili vertikalno.
- **`justify-content: center;`**: Ovo postavlja transformabilne delove u sredini duž glavne ose.
- **`justify-content: space-between;`**: Ovo će rasporediti elemente ravnomerno duž glavne ose.
- **`justify-content: space-around;`**: Ovo će rasporediti transformabilne delove ravnomerno unutar glavne ose, dodajući razmak pre prvog i posle poslednjeg dela.
- **`justify-content: space-evenly;`**: Ovo će rasporediti delove ravnomerno duž glavne ose.

:::interactive_editor

```html
<div class="container">
  <div class="box">A</div>
  <div class="box">B</div>
  <div class="box">C</div>
</div>
```

```css
.container {
  display: flex;
  justify-content: space-between;
  background: #eee;
}

.box {
  padding: 20px;
  background: salmon;
}
```

:::
  
## Svojstvo `align-items`

- **Opis**: Ova svojstvo se koristi za raspoređivanje segmenata duž poprečne ose. Imajte na umu da je poprečna osa vertikalna u odnosu na glavnu osu.
- **`align-items: center;`**: Ovo se koristi za postavljanje segmenata u sredini duž poprečne ose.
- **`align-items: flex-start;`**: Ovo poravnava segmente na početku poprečne ose.
- **`align-items: stretch;`**: Ovo se koristi za produžavanje podesivih segmenata duž poprečne ose.

:::interactive_editor

```html
<div class="container">
  <div class="box tall">1</div>
  <div class="box">2</div>
  <div class="box">3</div>
</div>
```

```css
.container {
  display: flex;
  align-items: center;
  height: 150px;
  background: #ddd;
}

.box {
  background: lightgreen;
  padding: 10px;
}

.tall {
  height: 100px;
}
```
