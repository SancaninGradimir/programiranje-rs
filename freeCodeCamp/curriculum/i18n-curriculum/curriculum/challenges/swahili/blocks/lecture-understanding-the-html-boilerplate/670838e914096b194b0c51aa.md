# --description--

Naučimo se o osnovnoj strukturi HTML.

Šta je osnovna struktura za HTML, pitate? To je kao gotov predložak za vaše web stranice. Zamislite ga kao temelj kuće. Osnovna struktura uključuje osnovni dizajn i ključne elemente koje svaki dokument za HTML zahteva. Štedi vam vreme i pomaže da se osigura da su vaše stranice pravilno formatirane. Evo primera:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <meta
       name="viewport"
       content="width=device-width, initial-scale=1.0" />
    <title>freeCodeCamp</title>
    <link rel="stylesheet" href="./styles.css" />
  </head>
  <body>
  </body>
</html>
```

Analizirajmo glavne dijelove ove osnovne strukture. Prvo, postoji deklaracija `DOCTYPE`:

```html
<!DOCTYPE html>
```

Kažite mi koja verzija HTML koristite u pretraživaču. Zatim, postoji oznaka `html`:

```html
<!DOCTYPE html>
<html lang="en">
  <!--All other elements go inside here-->
</html>
```

Ovo obuhvata sav vaš sadržaj i može utvrditi jezik vaše stranice. Unutar oznake `html`, pronaći ćete dva glavna dela - `head`, i `body`:

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <!--Important metadata goes here-->
  </head>
  <body>
    <!--Headings, paragraphs, images, etc. go inside here-->
  </body>
</html>
```

Deo `head` sadrži važne informacije iza kulisa:

```html
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>Document Title Goes Here</title>
  <link rel="stylesheet" href="./styles.css" />
</head>
```

Meta-podaci vaše stranice, koji se nalaze u elementima `meta`, sadrže informacije o stvarima kao što su kodiranje znakova i kako sajtovi poput Twitera treba da prikažu link do vaše stranice. Naslov vaše stranice, koji se nalazi u elementu `title`, određuje tekst koji je vidljiv na kartici ili prozoru pretraživača. Na kraju, obično ćete povezati eksterne stilove vaše stranice u sekciji `head` koristeći elemente `link`.

Sekcija za `body` je mesto gde se sav vaš sadržaj postavlja:

```html
<body>
  <h1>I am a main title</h1>
  <p>Example paragraph text</p>
</body>
```

Sada, zašto je osnovna struktura važna? Osigurava da su vaše stranice pravilno raspoređene i dobro funkcionišu u različitim pregledačima. Korišćenje osnovne strukture pomaže vam da izbegnete uobičajene greške i pratite najbolje prakse. To je odličan početak za bilo koji projekat na mreži. Zapamtite, možete prilagoditi svoju osnovnu strukturu kako bi odgovarala vašim potrebama. Kako dobijate iskustvo, možete dodati svoje omiljene elemente ili oznake `meta`. Što više poboljšavate svoju ličnu osnovnu strukturu, to ćete otkriti da štedite vreme kada započinjete nove projekte.

Sledeći put kada kreirate novi fajl sa HTML, razmislite o korišćenju osnovne šablone. Sigurno ćete pronaći čvrst temelj na kojem možete graditi.
