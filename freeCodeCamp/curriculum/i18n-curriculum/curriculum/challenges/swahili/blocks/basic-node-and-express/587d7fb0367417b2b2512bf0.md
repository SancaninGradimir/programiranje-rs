# --description--

HTML server obicno ima jedan ili vise direktorijuma koji su dostupni korisniku. Tu mozete smestiti staticke resurse koji su potrebni vasoj aplikaciji (style fajlove, skripte, slike).

U Express-u, ovu funkcionalnost mozete podesiti pomocu middleware-a `express.static(path)`, gde je argument `path` apsolutna putanja do direktorijuma koji cuva te resurse.

Ako ne znate sta je middleware... ne brinite, kasnije cemo ga detaljno obraditi. U sustini, middleware su funkcije koje presrecu rukovaoca rutom i dodaju neku vrstu informacija. Middleware se postavlja pomocu metode `app.use(path, middlewareFunction)`. Prvi argument `path` je opcioni. Ako ga izostavite, middleware ce se izvrsavati za sve zahteve.

# --instructions--

Postavite middleware `express.static()` na rutu `/public` koristeci `app.use()`. Apsolutna putanja do direktorijuma sa resursima je `__dirname + /public`.

Sada vasa aplikacija treba da moze da posluzi CSS fajl. Zapamtite da je fajl `/public/style.css` referenciran u `/views/index.html` u osnovnoj strukturi projekta. Vasa pocetna stranica sada treba da izgleda malo bolje.

# --hints--

Vasa aplikacija treba da posluzuje staticke fajlove iz direktorijuma `/public` preko rute `/public`

```js
  const response = await fetch(code + '/public/style.css');
  if (!response.ok) {
    throw new Error(await response.text());
  }
  const data = await response.text();
  assert.match(
    data,
    /body\s*\{[^\}]*\}/,
    'Your app does not serve static assets'
  );
```

Vasa aplikacija ne treba da posluzuje fajlove ni iz jednog drugog direktorijuma osim iz `/public`

```js
  const response = await fetch(code + '/server.js');
  if (response.ok) {
    const data = await response.text();
    assert.equal(
      response.status + '',
      404 + '',
      'Your app must serve files only from "public" directory'
    );
  } else {
    assert.equal(
      response.status + '',
      404 + '',
      'Your app must serve files only from "public" directory'
    );
  }
```
