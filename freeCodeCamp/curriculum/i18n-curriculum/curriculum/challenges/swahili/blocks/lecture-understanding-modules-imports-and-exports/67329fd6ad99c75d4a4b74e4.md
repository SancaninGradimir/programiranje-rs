# --description--

U JavaScript, moduli je nezavisna jedinica koda koja uključuje funkcije, klase objekata ili povezane varijable.

Zamislite modul kao građevni blok vašeg programa, poput poglavlja u knjizi. Svaki modul se fokusira na specifičnu funkcionalnost, čineći vaš kod dobro organizovanim, lak za održavanje i ponovno upotrebljiv. Moduli pomažu da se spreče kolizije naziva i omogućavaju vam da izgradite svoj program iz različitih međusobno povezanih delova.

Koncept modula u JavaScript se menjao tokom vremena, ali najčešći i podržani način je ES6 modulni sistem (ECMAScript 2015). Ovaj sistem pruža standardan način za definisanje i korišćenje modula u različitim okruženjima JavaScript.

Da biste kreirali module, pišete svoj kod od JavaScript u zasebnu datoteku. Sve varijable, funkcije ili klase objekata koje želite da učinite dostupnim drugim delovima vašeg programa moraju biti eksplicitno eksportovane. Ovo možete učiniti korišćenjem ključne reči `export`.

Na primer, recimo da imaš fajl nazvan `math.js` sa nekim matematičkim funkcijama:

```js
export function add(a, b) {
    return a + b;
}

export function subtract(a, b) {
    return a - b;
}

const PI = 3.14159;
export { PI };
```

U ovom primeru, eksportujemo dve funkcije (`add` na `subtract`) i konstantu (`PI`). Možete eksportovati mnogo stvari koliko vam treba iz jednog modula.

Da biste koristili ove komponente koje su izbačene u drugi deo vaše aplikacije, morate ih uvesti. To se radi korišćenjem glavne funkcije `import`.

Pretpostavimo da želite koristiti ove funkcije za izračunavanje u fajlu pod nazivom `app.js`:

```js
import { add, subtract, PI } from './math.js';

console.log(add(5, 3));        // Outputs: 8
console.log(subtract(10, 4));  // Outputs: 6
console.log(PI);               // Outputs: 3.14159
```

Ovde uvozimo specifične funkcije i konstante koje nam trebaju iz modula `math.js`. Sekcija `'./math.js'` objašnjava JavaScript gde da se pronađe fajl modula u odnosu na trenutni fajl.

Ponekad, možda želite da uvezete sve što modul eksportuje. Ovo možete učiniti koristeći zvezdastu sintaksu (`*`):

```js
import * as Math from './math.js';

console.log(Math.add(5, 3));        // Outputs: 8
console.log(Math.subtract(10, 4));  // Outputs: 6
console.log(Math.PI);               // Outputs: 3.14159
```

U ovom slučaju, svi elementi koji su eksportovani iz `math.js` uvoze se kao svojstva objekta nazvanog `Math`.

Još jedan uobičajen obrazac je imati export od default u modulima. Ovo se obično koristi kada modul eksportuje samo jednu funkciju. Možete imati samo jedan export od default za jedan modul.

Ovo je kako izgleda:

```js
// In math.js
export default function multiply(a, b) {
    return a * b;
}

// In app.js
import multiply from './math.js';

console.log(multiply(4, 5));  // Outputs: 20
```

Napomena da kada uvezete export od default, ne morate koristiti vitičaste zagrade, i možete se na import osvrnuti bilo kojim imenom koje želite.

Važno je znati da biste koristili ES6 module u pretraživaču, morate definisati `type` kao `module` u vašem tagu `script`:

```html
<script type="module" src="app.js"></script>
```

Moduli pružaju moćan način za organizovanje i kreiranje vašeg koda od JavaScript. Omogućavaju vam da podelite svoju aplikaciju u male, kontrolisane delove, povećate ponovnu upotrebu koda i pomognete u održavanju čistog razdvajanja odgovornosti. Kako nastavite da gradite velike aplikacije, shvatićete da su moduli ključni alat u vašim razvojnim alatima za JavaScript.
