# --description--

React je jedna od najpoznatijih biblioteka za JavaScript za izgradnju korisničkih interfejsa i web aplikacija. Inicijalno je razvijena i održavana od strane Facebook-a, React je stekla veliku popularnost u web razvoju zbog svoje efikasnosti, fleksibilnosti i funkcija.

Osnovni koncept u React je kreiranje ponovno upotrebljivih UI komponenti. Ove komponente, kao što su dugmad, kartice i avatarni delovi, mogu se lako ponovo koristiti u celoj vašoj aplikaciji. Možete ugnježđivati ove komponente unutar drugih kako biste izgradili složenije i dinamičnije interfejse.

Jedna od glavnih prednosti React je što se ovi specifični delovi mogu ažurirati i prikazivati nezavisno kada se podaci menjaju.

Za razliku od običnog JavaScript, koji zahteva direktno upravljanje sa DOM (Document Object Model), React koristi virtuelni DOM, što povećava performanse i efikasnost. Naučićete više o virtuelnom DOM i kako funkcioniše u budućim časovima.

Pored ponovo upotrebljivih komponenti, React takođe nudi moćan način za upravljanje stanjem vaše aplikacije. Stanje se odnosi na podatke koji definišu kako komponenta izgleda i njeno ponašanje.

Kwa React, možete lako pratiti i ažurirati status sekcije, osiguravajući da UI prikazuje najnovije podatke. Naučićete više o radu sa stanjem u budućim lekcijama.

Iako postoji mnogo biblioteka unutar ekosistema JavaScript, freeCodeCamp će se fokusirati na učenje React zbog njegove široke primene i potrebe u sektoru.

U narednih nekoliko lekcija detaljnije ćemo istražiti izradu komponenti, upravljanje stanjem i još mnogo toga.
