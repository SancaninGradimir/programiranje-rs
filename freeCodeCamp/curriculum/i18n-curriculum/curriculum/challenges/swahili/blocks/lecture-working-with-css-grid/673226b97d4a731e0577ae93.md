# --interactive--

Svojstvo `grid-template-areas` omogućava ti da napraviš vizuelni raspored mreže koristeći imenovane oznake.

Zatim ove oznake dodeljujete specifičnim delovima mreže koristeći svojstvo 'grid-area'. Drugim rečima, ove označene oznake takođe se nazivaju „nazivi oblasti mreže“.

Evo osnovne sintakse svojstva `grid-template-areas`:

```css
grid-template-areas:
 'header header header'
 'left-sidebar main right-sidebar'
 'footer footer footer';
```

Evo stvari koje je važno zapamtiti iz osnovne sintakse:

- Vrednosti poput `header` i `main` predstavljaju nazive oblasti mreže.

- Svaka vrednost odvojena razmacima unutar jednog stringa predstavlja jedan red.

- Svaka sekvenca slova predstavlja jedan red u mreži.

Kwa hivyo, katika sintaksia, tuna kontena la gridi la 3 kwa 3.

Nakon definisanja šablona, treba da koristiš svojstvo `grid-area` sa odgovarajućom imenovanom oznakom (oblašću) kao vrednošću kako bi CSS znao da određeni element pripada toj oblasti mreže.

Svojstvo `grid-area` povezuje stavku mreže sa imenovanom oblašću definisanom u `grid-template-areas`.

Jedan od najpoznatijih načina za prikaz mogućnosti svojstva `grid-template-areas` jeste kreiranje klasičnog Holy Grail rasporeda.

Ako ne znate šta je, Holy Grail raspored je web struktura sa zaglavljem, centralnim delom, dve bočne trake i glavnom oblastima sadržaja. Osigurava da glavni sadržaj dobije prioritet, dok se bočne trake i druge sekcije prilagođavaju uređaju unutar tog rasporeda.

Postoji mnogo rešenja za implementaciju "holy grail" konfiguracije, ali korišćenje `grid-template-areas` i svojstva `grid-area` je najlakši način da ga napravite.

Ovo je primer konfiguracije 'holy graila':

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="header">
    <h2>Header</h2>
  </div>
  <div class="sidebar-left">
    <h2>Left Sidebar</h2>
  </div>
  <div class="main"><h2>Main Content</h2></div>
  <div class="sidebar-right">
    <h2>Right Sidebar</h2>
  </div>
  <div class="footer">
    <h2>Footer</h2>
  </div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-areas:
    'header header header'
    'sidebar-left main sidebar-right'
    'footer footer footer';
  grid-gap: 10px;
  background-color: #2196F3;
  padding: 10px;
}

.header {
  grid-area: header;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.sidebar-left {
  grid-area: sidebar-left;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.main {
  grid-area: main;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.sidebar-right {
  grid-area: sidebar-right;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}

.footer {
  grid-area: footer;
  background-color: rgba(255, 255, 255, 0.8);
  padding: 20px;
  text-align: center;
}
```

:::

Molimo vas da napomenete da se karakteristike `grid-template-areas` i `grid-area` mogu koristiti nezavisno, bez direktne veze.

Sifa ya `grid-template-areas` služi za definisanje prikaza/rasporeda pregleda za mapiranje imenovanih mrežnih područja unutar kontejnera grida.

S druge strane, atribut `grid-area` se koristi za određivanje položaja pojedinačnih gridded elemenata, bilo specifikovanjem njihovih rednih i kolumnih pozicija ili referenciranjem imenovanih područja definisanih atributom `grid-template-areas`.
