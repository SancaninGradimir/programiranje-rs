# --description--

Izjava `throw` u JavaScript se koristi za bacanje izuzetka definisanog korisnikom. Izuzetak u programu je neočekivani događaj koji nastaje i prekida normalan tok programa.

Programeri, važno je da obrade ove izuzetke kako bi vaše programe ne padnu naglo kada se pojave greške. Evo osnovne sintakse rečenice za `throw`:

```js
throw expression;
```

`expression` U ovom slučaju biće objekat ili vrednost koja predstavlja izuzetak koji želite baciti. Primeri su ugrađene klase izuzetaka, kao što su klasa `Error`, `TypeError`, ili `RangeError`. Evo primera korišćenja izjave `throw` za bacanje `TypeError`:

```js
function validateNumber(input) {
  if (typeof input !== "number") {
    throw new TypeError("Expected a number, but received " + typeof input);
  }
  return input * 2;
}
```

U ovom primeru, proveravamo da li je tip `input` različit od tipa `number`. Ako nije, onda bacamo `TypeError` sa specifičnom porukom. Inače, funkcija će vratiti rezultat množenja `input` sa `2`.

Ako želite da damo opštije poruke o grešci, možete se osvrnuti na komponentu `Error` kao što je ovo:

```js
function divide(numerator, denominator) {
  if (denominator === 0) {
    throw new Error("Cannot divide by zero");
  }
  return numerator / denominator;
}
```

Ovde je primer funkcije koja proverava da li je denominator `0`. Ako jeste, onda će nam dati specifičnu poruku o grešci koja kaže `Cannot divide by zero`.

U ovom tutorijalu, pokazaćemo kako baciti poruku o grešci unutar konteksta bloka `try`/`catch` koji se koristi za pravilno rukovanje izuzecima u JavaScript.
