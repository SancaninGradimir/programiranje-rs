# --description--

Ako se funkcija pozove, ali nije dostavljen argument, onda će parametar dobiti vrednost `undefined`. Ovo može izazvati neočekivane rezultate ako funkcija nije dizajnirana da rukuje ovom situacijom.

```js
function favoriteAnimal(animal) {
  return animal + " is my favorite animal!"
}

console.log(favoriteAnimal()); // Output: undefined is my favorite animal!
```

U gornjem primeru, funkcija `favoriteAnimal` se poziva bez argumenata. Parametar `animal` dobio je vrednost `undefined`, koji se zatim spaja sa nizom karaktera `" is my favorite animal!"`.

Ako vrednost nije navedena, možete koristiti podrazumevani parametre. Podrazumevani parametri vam omogućavaju da postavite podrazumevanu vrednost za parametar ako ta vrednost nije navedena kada se pozove funkcija.

```js
function favoriteAnimal(animal = 'Giant Panda') {
  return animal + " is my favorite animal!"
}
```

U gornjem primeru, podrazumevana vrednost parametra `animal` je postavljena na `'Giant Panda'`. Ako se funkcija `favoriteAnimal` pozove bez argumenata, parametar `animal` će dobiti vrednost `'Giant Panda'`.
