# --instructions--

Izvršite skeniranje porta koristeći Python.

U fajlu `port_scanner.py`, kreiraj funkciju nazvanu `get_open_ports` koja prima argument sa `target` i argument sa `port_range`. `target` može biti URL ili IP adresa. `port_range` je lista od dve broja koja pokazuju početni i krajnji broj opsega portova za proveru.

Ovdje su primjeri kako se funkcija može nazvati:

```py
get_open_ports("209.216.230.240", [440, 445])
get_open_ports("www.stackoverflow.com", [79, 82])
```

Funkcija treba da vrati listu otvorenih portova u navedenom opsegu.

Funkcija ``get_open_ports`` takođe bi trebalo da primi treći opcionalni argument ``True`` za prikaz stanja "Verbose". Ako je ovo tačno, funkcija bi trebalo da vrati niz karakteristika sa opisima umesto liste portova.

Ovo je uzorak sekvence znakova koji treba vratiti u režimu detaljnog prikaza (tekst unutar `{}` pokazuje informacije koje bi trebalo da se prikažu):

```bash
Open ports for {URL} ({IP address})
PORT     SERVICE
{port}   {service name}
{port}   {service name}
```

Možete koristiti rečnik koji se nalazi u `common_ports.py` da pronađete tačno ime usluge za svaki port.

Na primer, ako je funkcija nazvana ovako:

```py
port_scanner.get_open_ports("scanme.nmap.org", [20, 80], True)
```

Trebalo bi vratiti sledeće:

```bash
Open ports for scanme.nmap.org (45.33.32.156)
PORT     SERVICE
22       ssh
80       http
```

Uverite se da uključite ispravan razmak i znak novog reda.

Ako je URL poslat u funkciju `get_open_ports` i nije validan, funkcija bi trebalo da vrati niz znakova: "Error: Invalid hostname".

Ako je IP adresa poslata u funkciju `get_open_ports` nepravilna, funkcija bi trebalo da vrati niz znakova: "Error: Invalid IP address".
