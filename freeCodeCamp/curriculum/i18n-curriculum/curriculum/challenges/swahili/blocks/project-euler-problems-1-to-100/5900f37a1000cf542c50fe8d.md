# --description--

Sledeća ponavljajuća sekvenca je definisana skupom pozitivnih celih brojeva:

<div style='padding-left: 4em;'><var>n</var> → <var>n</var>/2 (<var>n</var> je ceo pozitivan broj)</div>

<div style='padding-left: 4em;'><var>n</var> → 3<var>n</var> + 1 (<var>n</var> je nevažeći celobrojni broj)</div>

Koristeći pravilo navedeno iznad i počevši od 13, generišemo sledeću sekvencu:

<div style='text-align: center;'>13 → 40 → 20 → 10 → 5 → 16 → 8 → 4 → 2 → 1</div>

Izgleda da ova sekvenca (počevši od 13 i završavajući na 1) ima 10 brojeva. Iako još nije dokazano (Collatzova pretpostavka), veruje se da sve početne brojeve dosegnu 1.

Koji je početni broj, ispod `limit`, koji stvara najduži niz?

**Napomena:** Kada lanac počne, reči su dozvoljene da idu preko `limit`.

```js
function longestCollatzSequence(limit) {
  let longest = 1;
  let maxLength = 1;
  for (let i = Math.floor(limit / 2); i < limit; i++) {
    let len = colLen(i);
    if (len > maxLength) {
      longest = i;
      maxLength = len;
    }
  }
  return longest;
}

const knownSequence = { '1': 1 };

function colLen(n) {
  if (knownSequence[n]) {
    return knownSequence[n];
  } else {
    const len = n % 2 === 0 ? colLen(n / 2) + 1 : colLen((3 * n + 1) / 2) + 2;
    knownSequence[n] = len;
    return len;
  }
}
```
