# --explanation--

Odrični oblik glagola `to be` u `Simple Past` je `was not` ili `wasn't` za jedninu, odnosno `were not` ili `weren't` za množinu. Koristi se za opisivanje nečega što se nije dogodilo ili nije bilo tačno. Na primer:

- Jednina: `The code wasn't running as expected.` - Kod nije radio kako se očekivalo.

- Wingi zaidi: `The changes weren't implemented correctly.` - Hitilafu katika utekelezaji zilisababisha hitilafu.

U rečenici Džejmsa, koristi `wasn't` da objasni da sistem ne može da se poveže sa bazom podataka. Negativne rečenice koje sadrže `wasn't` ili `weren't` pokazuju ono što nije došlo ili nije radilo kako je očekivano.
