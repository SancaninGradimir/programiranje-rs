# --interactive--

Operatori za poređenje vam omogućavaju da uporedite dve vrednosti i vratite rezultat `true` ili `false`. Zatim možete koristiti te rezultate za donošenje odluke ili kontrolisanje toka vašeg programa. Koristite poređenje u `if`, petljama i mnogim drugim uslovima gde je potrebno doneti odluke na osnovu određenih kriterijuma. Hajde da istražimo najčešće korišćene operatore poređenja i vidimo kako funkcionišu.

Opereta koja je veća, predstavljena znakom desne trokuta (`>`), proverava da li je vrednost na levoj strani veća od one na desnoj strani:

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a > b); // false
console.log(b > a); // true
```

:::

Operator veće ili jednakosti, predstavljen simbolom pravougaonika i znakom jednakosti (`>=`), proverava da li je vrednost sa leve strane veća ili jednaka onoj sa desne strane:

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a >= b); // false
console.log(b >= a); // true
console.log(a >= c); // true
```

:::

Operater za manji ili jednako manji od, predstavljen levim trouglastim znakom (`<`), radi kao `>`, ali u suprotnom smeru. Proverava da li je vrednost na levoj strani manja od vrednosti na desnoj strani:

:::interactive_editor

```js
let a = 6;
let b = 9;

console.log(a < b); // true
console.log(b < a); // false
```

:::

Opereta manja ili jednaka, predstavljena simbolom manje nego sa leve strane i znakom jednakosti (`<=`), proverava da li je vrednost na levoj strani manja ili jednaka onoj na desnoj strani:

:::interactive_editor

```js
let a = 6;
let b = 9;
let c = 6;

console.log(a <= b); // true
console.log(b <= a); // false
console.log(a <= c); // true
```

:::
