# --interactive--

Bitwise operateri u JavaScript su specijalni operateri koji rade na binarnom predstavljanju brojeva. Da bismo razumeli bitwise operatore, prvo moramo razumeti koncept bita i binarnih brojeva. U računaru, bit je najosnovnija jedinica informacija. Može imati samo dve vrednosti: `0` ili `1`. Binarni sistem je sistem brojeva koji koristi samo ove dve cifre za predstavljanje svih brojeva.

Primer, binarna reprezentacija decimalnog broja `10` je `1010`. U ovom sistemu, svaka cifra predstavlja stepen `2`, počevši od najdesnije cifre i povećavajući se kako pomeramo ulevo.

U tabeli iznad, prvi red prikazuje binarni broj `1010`, drugi red prikazuje moć `2` predstavljenu svaku binarnoj poziciji, i treći red prikazuje rezultat svakog množenja. Sumirajući sve vrednosti u trećem redu, ukupan je `10`.

Sada, hajde da pogledamo bitwise operatore. Ovi operateri obavljaju operacije na binarnom predstavljanju brojeva. JavaScript nudi nekoliko bitwise operatera, uključujući AND (`&`), OR (`|`), XOR (`^`), NOT (`~`), levakrenje (`<<`), i desakrenje (`>>`).

Operatori bitwise AND-a (`&`) vraćaju `1` u svakoj bitnoj poziciji gde su odgovarajući biti obe strane `1`. Evo primera:

:::interactive_editor

```js
let a = 5;  // Binary: 101
let b = 3;  // Binary: 011
console.log(a & b);  // 1 (Binary: 001)
```

:::

U ovom primeru, izvršavamo operaciju bitwise AND na `5` (`101` u binarnom) i `3` (`011` u binarnom). Rezultat je `1` (`001` u binarnom) jer je samo najdesni bit `1` u oba broja.

Operatori bitwise OR (`|`) izvršavaju `1` na svakoj bit poziciji gde su odgovarajući biti sa obe strane `1`. Primer:

:::interactive_editor

```js
let a = 5;  // Binary: 101
let b = 3;  // Binary: 011
console.log(a | b);  // 7 (Binary: 111)
```

:::

Ovde, rezultat je `7` (`111` u binarnom) zato što najmanje jedan bit je `1` u svakoj poziciji.

Operatori bitwise XOR (`^`) vraćaju `1` na svakoj bitnoj poziciji gdje su odgovarajući biti sa obe strane, ali ne i oba, `1`. Primer:

:::interactive_editor

```js
let a = 5;  // Binary: 101
let b = 3;  // Binary: 011
console.log(a ^ b);  // 6 (Binary: 110)
```

:::

Rezultat je `6` (`110` u binarnom) jer su prvi i drugi bit od desna različiti u ova dva broja.

Operatori bitwise NOT (`~`) menjaju sve bite svog operanda. Primer:

:::interactive_editor

```js
let a = 5;  // Binary: 101
console.log(~a);  // -6
```

:::

Ovo može delovati iznenađujuće, ali je zbog načina na koji se negativni brojevi predstavljaju u binarnom sistemu korišćenjem dopune dva (two's complement).

Operatori za pomeranje ulevo (`<<`) pomeraju sve bite ulevo za određeni broj pozicija. Na primer:

:::interactive_editor

```js
let a = 5;  // Binary: 101
console.log(a << 1);  // 10 (Binary: 1010)
```

:::

Ovdje, svi bitovi se pomeraju za jednu poziciju ulevo, što efikasno povećava broj sa `2`.

Operatori za pomeranje udesno (`>>`) pomeraju sve bita udesno. Primer:

:::interactive_editor

```js
let a = 5;  // Binary: 101
console.log(a >> 1);  // 2 (Binary: 10)
```

:::

Ovde, svi biti se pomeraju za jednu poziciju udesno, radi efikasnog deljenja broja sa `2` i prebacivanja ih nadole.

Bitovni operateri se često koriste u niskonivo programiranju i kriptografiji. Iako se možda ne koriste mnogo u standardnim aplikacijama od JavaScript, razumevanje ovih operatera može biti korisno za neke specifične zadatke i može povećati vaše razumevanje kako računari rade na osnovnom nivou.
