# --description--

Interna putanja linije `CSS` omogućava dodavanje stilova direktno elementima za `HTML`, iako se ovaj metod ne preporučuje:

```html
<body>
  <div style="color: white; background-color: black;">...</div>
</body>
```

Prva stvar na koju treba obratiti pažnju je da nema selektora ovde, jer se stilovi dodaju direktno u oznaku elementa `<div>` samostalno. Zatim, ima atribut `style`, a njegova vrednost unutar navodnika je deklaracija.

Ako trebate dodati jedinstveni stil samo jednom elementu, ova metoda može dobro da radi. Generalno, međutim, ovo nije preporučeni način za dodavanje CSS na HTML zbog nekoliko razloga:

Može brzo postati neuredno kada počnete dodavati previše deklaracija za jedan element, što uzrokuje da je vaša datoteka HTML nepotrebno velika.
Ako želite da više elemenata ima isti stil, moraćete da kopirate i zalepite taj stil za svaki element, što će uzrokovati mnogo nepotrebnih ponavljanja i povećanje veličine.
Bilo koji CSS unutar linije će poništiti druge dve metode, što može dovesti do nepouzdanih rezultata. (Iako ovo ne istražujete ovde, to se može koristiti uz prednost).
