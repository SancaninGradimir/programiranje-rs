# --description--

Model u kutiji CSS je osnovni koncept za web razvoj. Objašnjava kako se elementi HTML raspoređuju i postavljaju. Razumevanje ovog modela omogućit će vam da kontrolišete veličinu, raspored i izgled elemenata na vašoj veb stranici.

U modelu kutije za CSS, svaki element je okružen kutijom. Ova kutija ima četiri dela: oblast sadržaja, unutrašnji prostor, granica i ivica.

Oblast sadržaja je najunutrašnji deo kontejnera. To je prostor koji drži stvarni sadržaj elementa, kao što su tekst ili slika.

Unutrašnji razmak je područje koje sledi odmah nakon područja sadržaja. To je prostor između područja sadržaja i granice elementa. Korišćenjem unutrašnjeg razmaka, možete dodati prostor oko sadržaja kako biste poboljšali njegovu čitljivost. Možete postaviti različite vrednosti za unutrašnji razmak (gore, desno, dole i levo) koristeći svojstvo ``padding``.

Ovo je primer skraćenog svojstva `padding`, gde postavljamo gornji unutrašnji razmak na petnaest piksela, desni na pet piksela, donji na dva piksela, a levi na osam piksela:

```css
padding: 15px 5px 2px 8px;
```

Ivica (`border`) je spoljašnja granica elementa u CSS box modelu. Predstavlja vidljivu granicu elementa. Možeš prilagoditi stil, širinu, boju i druga svojstva ivice pomoću svojstva `border`. Evo primera gde postavljamo ivicu širine pet piksela, punog stila i plave boje:

```css
border: 5px solid blue;
```

Ako izostaviš neku vrednost, koristiće se njena podrazumevana vrednost. To su `medium` za širinu, `none` za stil i trenutna boja za boju.

Možete postaviti ova tri svojstva direktno u svojstvo skraćivanja `border` ako želite da sve strane budu potpuno jednake. Međutim, ako želite da postavite različit stil za svaku stranu, možete koristiti svojstva `border-width`, `border-style` i `border-color`.

```css
border-width: 2px 4px 7px 12px;
border-style: dashed solid solid dashed;
border-color: blue red green black;
```

Možeš navesti do četiri vrednosti za svako od ovih svojstava. One se primenjuju u smeru kazaljke na satu, počevši od vrha. Ako navedeš samo jednu vrednost, ona će biti primenjena na sve četiri strane.

Na kraju, margina je prostor izvan ivice elementa. Ona određuje razmak između elementa i elemenata koji ga okružuju. Možeš postaviti različite vrednosti za gornju, desnu, donju i levu marginu pomoću svojstva `margin`.

Tako u ovom primeru, gornji rub je tri piksela, desni rub je dvanaest piksela, donji rub je devet piksela i levi rub je sedam piksela:

```css
margin: 3px 12px 9px 7px;
```

Ova četiri dela su važna za izračunavanje ukupne širine i dužine elementa.

U narednim lekcijama, naučićete više o tome kako pregledač obrađuje ovo i kako ga možete personalizovati. Primer u kutiji CSS je osnovna koncepcija za web razvoj.

Razumeti kako se ove komponente međusobno nadopunjuju i doprinose karakteristikama je ključno za implementaciju mrežnih arhitektura.
