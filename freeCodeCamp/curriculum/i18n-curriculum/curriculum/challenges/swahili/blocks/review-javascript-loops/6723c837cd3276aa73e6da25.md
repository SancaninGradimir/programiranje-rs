## Rad sa ciklusima

- **`for` petlja**: Ovaj tip petlje koristi se za ponavljanje bloka koda određeni broj puta. Ova petlja je podeljena na tri dela: inicijalizacija, uslov i deklaracija za inkrement/dekrement. Deklaracija za inicijalizaciju izvršava se pre nego što petlja počne. Obično se koristi za inicijalizovanje brojača. Uslov se proverava pre svake iteracije petlje. Iteracija je prolazak kroz petlju jednom. Ako je uslov `true`, blok koda unutar petlje se izvršava. Ako je uslov `false`, petlja se zaustavlja i nastavlja sa drugim blokom koda. Deklaracija za inkrement/dekrement izvršava se nakon svake iteracije petlje. Obično se koristi za povećanje ili smanjenje brojača.

:::interactive_editor

```js
for (let i = 0; i < 5; i++) {
  console.log(i);
}
```

:::

- **`for...of` petlja**: Ovaj tip petlje se koristi kada je potrebno iterirati kroz vrednosti iz iterabilnog objekta. Primeri iterabilnih objekata su nizovi podataka i stringovi (niz znakova).

:::interactive_editor

```js
const numbers = [1, 2, 3, 4, 5];

for (const num of numbers) {
  console.log(num);
}
```

:::

- **`for...in` ciklus**: Ovaj tip ciklusa je najbolji za korišćenje kada vam je potrebno iterirati po svojstvima objekta. Ovaj ciklus će proći kroz sva enumerabilna svojstva objekta, uključujući nasleđene i nenumerička svojstva.

:::interactive_editor

```js
const fruit = {
  name: 'apple',
  color: 'red',
  price: 0.99
};

for (const prop in fruit) {
  console.log(fruit[prop]);
}
```

:::

- **`while` ciklus**: Ovaj tip ciklusa će izvršiti deo koda samo ako je uslov `true`.

:::interactive_editor

```js
let i = 5;

while (i > 0) {
  console.log(i);
  i--;
}
```

:::

- **`do...while` ciklus**: Ovaj tip ciklusa će izvršiti deo koda barem jednom pre proveravanja stanja.

```js
let userInput;

do {
  userInput = prompt("Please enter a number between 1 and 10");
} while (Number(userInput) < 1 || Number(userInput) > 10);

alert("You entered a valid number!");
```

## Deklaracija za `break` za `continue`

- **Definicija**: Deklaracija `break` koristi se iz ranijeg ciklusa, dok se deklaracija `continue` koristi za preskakanje trenutne repeticije i nastavak sa sledećom repeticijom.

:::interactive_editor

```js
// Example of break statement
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    break;
  }
  console.log(i);
}

// Output: 0, 1, 2, 3, and 4

// Example of continue statement 
for (let i = 0; i < 10; i++) {
  if (i === 5) {
    continue;
  }
  console.log(i);
}

// Output: 0, 1, 2, 3, 4, 6, 7, 8, and 9
```
