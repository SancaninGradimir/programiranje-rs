# --description--

U ovoj vežbi, naučićete razliku između `specific` i `specify`. `Specific` je pridev koji opisuje nešto što je jasno ili ima detaljne opise, dok je `specify` glagol koji znači reći nešto detaljno.

Na primer, uputstva za `specific` mogu biti `use port 8080`, dok `specify` može značiti da se te detalje kaže u detalju: `specify the port number.`

`Setup` odnosi se na organizovanje ili pripremu nečega, često se koristi u kontekstu pripreme programa ili uređaja, kao `setup instructions for the software.`
