# --description--

Funkcije su ponovno upotrebljivi delovi koda koji se izvršavaju kada ih pozovete. Mnogi programski jezici imaju ugrađene funkcije koje pojednostavljuju početak. Python nije drugačije, i već smo pomenuli neke ugrađene funkcije kao `print()` u prethodnim lekcijama.

Druga ugrađena funkcija za pomoć je `input()`, koja vam omogućava da zatražite od korisnika unos:

```python
name = input('What is your name?') # User types "Kolade" and presses Enter  
print('Hello', name) # Output: Hello Kolade
```

S druge strane, `int()` konvertuje brojeve, vrednosti istinitosti ili lažnosti, i niz alfanumeričkih karaktera u kompletan broj:

```python
print(int(3.14)) # 3
print(int('42')) # 42
print(int(True)) # 1
print(int(False)) # 0 
```

Takođe možete napisati sopstvene funkcije. Da biste to učinili, koristite glavnu reč `def`, praćenu imenom koje želite dati svojoj funkciji, običnim zagradama i tačkom. Zatim na novoj liniji, napišite kod koji vaša funkcija treba da izvrši. Kod koji je izvršen od strane funkcije takođe se naziva telo funkcije.

Ovde je primer specijalne funkcije nazvane `hello` koja ispisuje sekvencu karaktera `Hello World` na terminalu:

```python
def hello():
    print('Hello World')
```

Da biste pozvali funkciju, morate je zvati po njenom imenu praćeno standardnim zagrade:

```python
hello() # Hello World
```

Pogledajte prekid linije pre `print('Hello World')`. Kao što ste možda zapamtili iz prethodnih lekcija, Python zavisi od prekida linije da bi utvrdio koje su grupe povezanih izjava. Ove grupe izjava nazivaju se blokovi koda.

Evo još jedne jednostavne funkcije koja štampa zbir dve brojeve na terminalu:

```python
def calculate_sum(a, b):
    print(a + b)
```

Možete videti da naša funkcija, `calculate_sum`, ima `a` i `b` unutar zagrada, razdvojene zarezima. Ovo se zove parametri. Razmislite o parametrima kao o argumentima zameni kojima služe kao "prostor" za vrednosti koje prosleđujete funkcijama kada ih pozivate.

Da biste koristili parametre, morate proslediti „argument“. Argument je vrednost koju prosleđujete funkciji kada je pozivate.

Ovo je kako pozivate funkciju `calculate_sum` ili sabirate brojeve `3` i `1`:

```python
calculate_sum(3, 1) # 4
```

Ako pozovete funkciju bez ispravnog broja argumenata, ćete dobiti `TypeError`:

```python
calculate_sum()

# TypeError: calculate_sum() missing 2 required positional arguments: 'a' and 'b'
```

Funkcije takođe koriste specifičnu ključnu reč `return` da bi izašle iz funkcije i vratile vrednost. Ako nije eksplicitno korišćeno `return`, Python vratiće `None` podrazumevano.

Evo primera:

```python
def calculate_sum(a, b):
    print(a + b)

my_sum = calculate_sum(3, 1) # 4
print(my_sum) # None
```

Možete videti da funkcija `calculate_sum` ispisuje zbir `a` i `b`, ali ne vraća ništa eksplicitno. Stoga, kada ga stavimo u `my_sum`, vrednost je `None`. Da biste to ispravili, možete koristiti ključnu reč `return` da vratite rezultat:

```python
def calculate_sum(a, b):
    return a + b

my_sum = calculate_sum(3, 1)
print(my_sum) # 4
```

Sada, `calculate_sum` vraća zbir `a` i `b`, koji je sačuvan u `my_sum`.
