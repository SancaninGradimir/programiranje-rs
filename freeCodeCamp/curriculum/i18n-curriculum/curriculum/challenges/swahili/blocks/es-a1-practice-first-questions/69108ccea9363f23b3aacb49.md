# --explanation--

Razgovor između Matea i Kamile se bavi literaturom/književnošću.

`¿A qué te dedicas?` znači "Čime se baviš?" ili "Kakav posao radiš za život?"

**`Soy` + [taaluma]** to znači "Ja sam [taaluma]".

U hispanskom kontekstu, mnoge profesije se menjaju zavisno od pola. Međutim, neke profesije koje se završavaju sa `-ista` ili `-nte` često ostaju takve za oba pola:

| Profesija | Muški | Ženski |
| :---: | :---: | :---: |
| Analitičar | `analista` | `analista` |
| Novinar | `periodista` | `periodista` |
| Primalac | `recepcionista` | `recepcionista` |
| Pevač | `cantante` | `cantante` |
