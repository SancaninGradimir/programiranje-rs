# --description--

U prethodnom tutorijalu, naučili ste kako da instalirate Python na vašem uređaju. U ovom tutorijalu, naučićete kako da pokrenete kod Python na uređaju.

Pre nego što počneš da gradiš svoje skripte za Python, biće ti potrebno instalirati uređivač koda ili IDE. IDE je skraćenica za Integrated Development Environment i ima funkcije kao što su uređivač koda, alati za testiranje, terminal i još mnogo toga.

Popularni editor koda koji koriste mnogi programeri je VS Code. Posetite `https://code.visualstudio.com/download` da preuzmete VS Code, koji podržava okruženja Mac, Windows i Linux. Ovo je editor koji će biti korišćen u ovom tutorijalu.

Ostale uobičajene opcije za razvoj Python uključuju:

- PyCharm
- Spyder

Nakon što preuzmete vaš editor, moraćete ga otvoriti. Ako ga otvorite, moraćete da otvorite fasciklu. Možete kliknuti na opciju menija `File` u gornjem levom uglu i kliknuti `Open Folder`. Odatle, možete izabrati ili kreirati fasciklu za svoj projekat. Za dodatnu vežbanje van tutorijala/obuke, možda želite fasciklu pod nazivom `python-projects`.

Nakon što kreirate vašu fasciklu, možete otvoriti File Explorer koji je bočna traka na levoj strani u VS Code-u. Kliknite na ikonu `New File` koja izgleda kao komad papira sa znakom plus na njemu. Zatim kreirajte fajl pod nazivom `main.py` i pritisnite Enter. `.py` je prilog fajla koji ukazuje da je ovo fajl za Python.

Otvori svoju datoteku ``main.py`` i napiši sledeći kod:

```py
print("Hello, world!")
```

Da biste pokrenuli ovaj kod, imate nekoliko opcija. Prva opcija je da kliknete na dugme za izvršavanje koje izgleda kao dugme za reprodukciju u gornjem desnom uglu. To će otvoriti terminal i pokrenuti vaš skript Python tamo. Trebalo bi da vidite tekst `"Hello, world!"`.

Druga opcija je otvoriti terminal i pokrenuti program ručno koristeći:

```bash
python main.py
```

Morate pokrenuti ovu komandu iz foldera gde je `main.py` sačuvan. Na primer, ako je `main.py` unutar foldera pod nazivom `python-projects`, prvo koristite sledeće u terminalu:

```bash
cd python-projects
```

Zatim pokrenite:

```bash
python main.py
```

U nekim okruženjima, posebno na sistemima macOS i Linux gde su sve Python 2 i Python 3 instalirane, možda ćete morati da koristite `python3` umesto `python`. Ovo je uobičajeno kada komanda `python` ne postoji ili ukazuje na stariju verziju Python.

```bash
python3 main.py
```

Kada napredujete kroz ovaj kurs, trebalo je da vežbate primere iz lekcija pažljivo i da pratite rezultate. Takođe, trebalo je da kreirate sopstvene primere kako biste testirali ono što ste naučili iz lekcija.
