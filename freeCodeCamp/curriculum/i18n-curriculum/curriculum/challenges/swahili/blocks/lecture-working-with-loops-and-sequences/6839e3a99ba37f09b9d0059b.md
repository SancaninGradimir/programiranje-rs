# --description--

Tuple je Python tip podataka koji se koristi za kreiranje uređenog niza vrednosti. Tuple može sadržati kombinaciju različitih tipova podataka kao što su:

```py
developer = ('Alice', 34, 'Rust Developer')
```

Tuple je sličan listi, ali dok je lista promenljiv tip podataka, tuple je nepromenljiv. To znači da se elementi unutar tuple-a ne mogu menjati nakon što je kreiran.

Kada pokušate da promenite jedan element unutar tupola, dobićete `TypeError`:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')
programming_languages[0] = 'JavaScript'

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: 'tuple' object does not support item assignment
"""
```

Da biste dobili element iz tupaola, možete koristiti notaciju uglastih zagrada i indeks:

```py
developer = ('Alice', 34, 'Rust Developer')
developer[1] # 34
```

Ako želiš da pristupiš elementima počev od kraja tuple-a, možeš da koristiš negativno indeksiranje. Evo primera korišćenja negativnog indeksa za pristup pretposlednjem elementu tuple-a:

```py
numbers = (1, 2, 3, 4, 5)
numbers[-2] # 4
```

Ako pokušaš da pristupiš indeksu koji je veći ili jednak dužini tuple-a, dobićeš `IndexError` poput ovog:

```py
numbers = (1, 2, 3, 4, 5)
numbers[7]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
IndexError: list index out of range
"""
```

Drugi način za kreiranje tuple-a je korišćenjem konstruktora `tuple()` ovako:

```py
developer = 'Jessica'
tuple(developer) # ('J', 'e', 's', 's', 'i', 'c', 'a')
```

Za komponentu `tuple()`, možete uneti različite ponovljive elemente kao niz znakova, nizu i čak druge tuple.

Da proveriš da li se element nalazi u tuple-u, možeš da koristiš ključnu reč `in` ovako:

```py
programming_languages = ('Python', 'Java', 'C++', 'Rust')

'Rust' in programming_languages # True
'JavaScript' in programming_languages # False
```

Takođe možeš da raspakuješ elemente iz tuple-a isto kao što si radio sa listama:

```py
developer = ('Alice', 34, 'Rust Developer')
name, age, job = developer

print(name) # 'Alice'
print(age) # 34
print(job) # 'Rust Developer'
```

U ovom primeru, `name` ima vrednost od `'Alice'`, `age` ima vrednost od `34`, na `job` ima vrednost od `'Rust Developer'`.

Ako trebate prikupiti sve preostale elemente iz tupa, možete koristiti operater zvezdice (`*`) kao što je ovo:

```py
developer = ('Alice', 34, 'Rust Developer')
name, *rest = developer

print(name) # 'Alice'
print(rest) # [34, 'Rust Developer']
```

Ovde, `name` ima vrednost `'Alice'`, a na `rest` je lista koja uključuje broj `34` i niz slova `'Rust Developer'`.

Kao što je sa listom, možete koristiti operater za sekcionisanje (*slice*) na tuple da biste izvukli njegov deo. Evo primera kako izvući elemente `'pie'` i `'cookies'` iz različitog tuple-a:

```py
desserts = ('cake', 'pie', 'cookies', 'ice cream')
desserts[1:3] # ('pie', 'cookies')
```

Zapamtite da prvi broj pokazuje početni model za izbacivanje tog dela, dok drugi broj pokazuje krajnji model. Ali zapamtite da element koji je na krajnjem modelu nije uključen u vrajenu tupel.

Ako treba da obrišeš element iz tupa, nemoguće je jer je tupa nepromenljiva. Stoga ovaj primer će izazvati grešku:

```py
developer = ('Jane Doe', 23, 'Python Developer')
del developer[1]

"""
Traceback (most recent call last):
  File "<stdin>", line 2, in <module>
TypeError: "tuple" object doesn't support item deletion
"""
```

Pa, kada ćeš koristiti tuple umesto liste?

Ako vam je potreban skup promenljivih elemenata gde možete dodati, izbrisati i modifikovati elemente, trebalo bi da koristite listu. Ako znate da radite sa kolekcijom podataka koja je stabilna i nepromenljiva, trebalo bi da koristite tuple.

U sledećoj lekciji, ćemo pogledati neke uobičajene metode koje ćete koristiti prilikom korišćenja tuple-a.
