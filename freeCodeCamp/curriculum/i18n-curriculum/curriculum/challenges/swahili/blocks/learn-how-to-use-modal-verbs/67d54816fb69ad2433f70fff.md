# --explanation--

`Have to follow` Koristi se za postavljanje pitanja o obavezama ili potrebama. Na primer:

`You have to follow the safety guidelines when using this machine.` - Obavezno je pratiti bezbednosne pravila prilikom upravljanja mašinom.

`Have to` radi kao `must` i pokazuje da je nešto neophodno.

`Follow` je ispravan glagol koji se odnosi na smernice, zakone ili uputstva.
