# --description--

Do sada si prikazivao delove React klijentu. Obično je ovo ono što ćeš raditi svaki put. Međutim, postoje određene upotrebe gde je odgovarajuće prikazati deo React na serveru. Pošto je React biblioteka pogleda za JavaScript i možeš pokrenuti JavaScript na serveru koristeći Node, ovo je moguće. Zapravo, React pruža način za `renderToString()` koji možeš koristiti za ovu svrhu.

Postoje dva glavna razloga zašto se renderovanje na serveru koristi u stvarnim aplikacijama. Prvo, bez njega bi React aplikacije pri prvom učitavanju u pregledaču sadržale malo HTML-a i veliku količinu JavaScript-a. To nije idealno za pretraživače koji indeksiraju sadržaj stranica. Ako početni HTML generiše server i pošalje ga klijentu, pretraživači mogu odmah analizirati sadržaj. Drugo, početno učitavanje stranice je brže jer je renderovani HTML manji od kompletnog JavaScript koda. Nakon toga React i dalje preuzima kontrolu nad aplikacijom.

# --instructions--

Putanja `renderToString()` dostupna je na `ReactDOMServer`, koja postoji ovde kao globalni objekat. Ova putanja prima jedan argument, koji je svojstvo React. Koristite ovo da pokažete `App` kao niz znakova.
