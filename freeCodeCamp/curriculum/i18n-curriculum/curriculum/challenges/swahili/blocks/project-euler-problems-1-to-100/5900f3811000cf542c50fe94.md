# --description--

Neka je d(`n`) definisano kao zbir stvarnih delilaca od `n` (brojevi manji od `n` koji dele `n` bez ostatka).

Ako je d(`a`) = `b` i d(`b`) = `a`, pri čemu je `a` ≠ `b`, tada su `a` i `b` prijateljski par, a svaki od njih naziva se prijateljski broj.

Na primer, delitelji 220 su 1, 2, 4, 5, 10, 11, 20, 22, 44, 55 i 110; stoga d(220) = 284. Delitelji 284 su 1, 2, 4, 71 i 142; pa d(284) = 220.

Izračunajte ukupnu sumu svih brojeva za kompatibilnost ispod `n`.

# --solutions--

```js
const sumAmicableNum = (n) => {
  const fsum = (n) => {
    let sum = 1;
    for (let i = 2; i <= Math.floor(Math.sqrt(n)); i++)
      if (Math.floor(n % i) === 0)
        sum += i + Math.floor(n / i);
    return sum;
  };
  let d = [];
  let amicableSum = 0;
  for (let i=2; i<n; i++) d[i] = fsum(i);
  for (let i=2; i<n; i++) {
    let dsum = d[i];
    if (d[dsum]===i && i!==dsum) amicableSum += i+dsum;
  }
  return amicableSum/2;
};
```
