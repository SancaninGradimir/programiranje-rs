# --description--

<ol>
  <li>Uzmite ceo broj <code>n₀</code>, veći od nule.</li>
  <li>Kreirajte sledeći broj u sekvenci <code>n</code> konvertovanjem <code>n₀</code> i dodavanjem ga <code>n₀</code></li>
  <li>Zaustavite kada <code>n</code> postane palindrom - tj., cifre <code>n</code> u obrnutom redosledu su jednake <code>n</code>.</li>
</ol>

Rekurentna relacija navedena iznad, kada se poveže sa mnogo brojeva počevši od `n` = 1, 2, ... vrlo brzo završava palindromom.

Primer ako `n₀` = 12 dobijamo:

```md
12
12 + 21 = 33,  a palindrome!
```

Ako `n₀` = 55 dobijamo:

```md
55
55 + 55 = 110
110 + 011 = 121,  a palindrome!
```

Proverite da se palindromska provera izvrši *nakon* dodavanja.

Neki početni brojevi izgledaju da traju zauvek; relacija ponavljanja 196 je izračunata za milione iteracija, stvarajući brojeve sa milionima cifara, bez formiranja palindroma. Ovi brojevi koji ne završavaju sa palindromom nazivaju se **Lychrel brojevi**.

Za potrebe ove vežbe, Lychrel broj je bilo koji početni broj koji ne stvara palindrom u roku od 500 iteracija (ili više).

**Povezani brojevi semena i Lychrel brojevi:**

Svaki cijeli broj generisan u sekvenci broja Lychrela takođe je broj Lychrel.

Generalno, bilo koja sekvenca od jednog broja Lychrela može da se spoji sa sekvencijom prethodnog kandidata broja Lychrela; na primer, sekvenca brojeva 196 i zatim 689 počinje:

```md
    196
    196 + 691 = 887
    887 + 788 = 1675
    1675 + 5761 = 7436
    7436 + 6347 = 13783
    13783 + 38731 = 52514
    52514 + 41525 = 94039
    ...
    689
    689 + 986 = 1675
    1675 + 5761 = 7436
    ...
```

Dakle vidimo da niz koji počinje sa 689 se spaja i nastavlja sa brojevima sličnim onim od 196.

Iz ovoga možemo dalje podeliti Lychrel brojeve na stvarni kandidatima za Lychrel brojeve od **semena**, i povezanih brojeva koji ne generišu palindrom, ali imaju potpune brojeve u svojoj sekvenci poznate kao deo sekvence koja potiče od nižeg Lychrel broja.

```js
function isLychrel(n) {
  function reverse(num) {
    return parseInt(
      num
        .toString()
        .split('')
        .reverse()
        .join('')
    );
  }

  var i;
  for (i = 0; i < 500; i++) {
    n = n + reverse(n);
    if (n == reverse(n)) break;
  }

  return i == 500;
}
```
