# --description--

Naučili ste nekoliko načina za postavu čvrste boje u CSS, ali možete koristiti i promenu boja ili <dfn>gradient</dfn>, na elementu.

Gradient je tamo gde jedna boja prelazi u drugu. Funkcija CSS `linear-gradient` vam omogućava da kontrolišete smer tih promena duž linije, i koje boje se koriste.

Jedna stvar za pamćenje je da funkcija `linear-gradient` kreira element `image`, i obično je praćena svojstvom `background` koje može primiti sliku kao vrednost.

U pravilu CSS od `.red`, promeni svojstvo `background-color` u `background`.

# --hints--

Pravilo vaše za CSS od `.red` trebalo bi imati svojstvo `background` sa vrednošću `rgb(255, 0, 0)`.

```js
assert.match(__helpers.removeWhiteSpace(code), /\.red\{.*?background:rgb\(255,0,0\)/);
```
