# --description--

U ovoj laboratoriji, kreiraćete aplikaciju koja se prebacuje između različitih tema. Kada se prebacujete na drugu temu, boja pozadine će se menjati i prikazujeće se druga poruka.

**Cilj:** Popunite korisničke priče navedene ispod i prođite sve testove da biste završili laboratoriju.

**Priče korisnika:**

1. Treba da imaš element `button` sa tekstom `Switch Theme`.
2. Tvoj element `button` treba da ima sledeće atribute:
- Atribut `id` postavljen na `"theme-switcher-button"`.
- Atribut `aria-haspopup` postavljen na `"true"`.
- Atribut `aria-expanded` postavljen na `"false"`.
- Atribut `aria-controls` postavljen na `"theme-dropdown"`.
3. Treba da imaš element `ul` sa sledećim atributima:
- Atribut `id` postavljen na `"theme-dropdown"`.
- Atribut `role` postavljen na `"menu"`.
- Atribut `aria-labelledby` postavljen na `"theme-switcher-button"`.
   - Sifa ya `hidden`.
4. Tvoj element `ul` treba da sadrži najmanje dva elementa `li` sa atributom `role` postavljenim na `"menuitem"` i tekstom po tvom izboru koji predstavlja različite teme.
5. Svaki element `li` treba da ima atribut `id` koji počinje sa `theme-` i završava se nazivom teme prikazanim u tekstu tog elementa `li`. Na primer, ako je jedna od tema `Light`, tada `id` treba da bude `theme-light`.
6. Treba da imaš element sa atributom `aria-live` postavljenim na `"polite"`.
7. Treba da imaš niz `themes` sa najmanje dva objekta, od kojih svaki ima svojstva `name` i `message`. Svojstvo `name` predstavlja različitu temu, a `message` će biti prikazana kada se tema promeni. Vrednosti `name` i `message` možeš izabrati po želji, ali vrednosti `name` moraju odgovarati tekstu elemenata `li`.
8. Kada korisnik klikne na `#theme-switcher-button`, element `#theme-dropdown` treba da prikaže dostupne teme.
9. Kada je element `#theme-dropdown` prikazan, atribut `aria-expanded` na elementu `button` treba da bude postavljen na `"true"`.
10. Kada korisnik klikne na `#theme-switcher-button` dok je element `#theme-dropdown` prikazan, meni treba da se zatvori, atribut `hidden` treba da bude dodat, a atribut `aria-expanded` postavljen na `"false"`.
11. Kada korisnik klikne na `#theme-switcher-button` i izabere temu, odgovarajuća CSS klasa `theme-<name>` treba da bude dodata elementu `body`, a odgovarajuća poruka teme iz niza `themes` treba da bude prikazana u elementu sa `aria-live="polite"`.
12. Kada korisnik klikne na `#theme-switcher-button` i izabere drugu temu od trenutno aktivne, klasa teme na elementu `body` i poruka o statusu treba da se ažuriraju kako bi odgovarale novom izboru.

```js
const themes = [
  {
    name: "light",
    message: "Hello sunshine — Light theme is on!"
  },
  {
    name: "dark",
    message: "The night is yours — Dark theme is on!"
  },
  {
    name: "nord",
    message: "The frost has settled - Nord theme is on!"
  },
  {
    name: "ocean",
    message: "Blue skies and high tides — Ocean theme is on!"
  },
];

const setTheme = (theme) => {
  document.body.className = "";
  document.body.classList.add(`theme-${theme.name}`);
};

const updateStatus = (theme) => {
  const status = document.getElementById("status");
  status.textContent = theme.message;
};

document.addEventListener("DOMContentLoaded", () => {
  themes.forEach((theme) => {
    const button = document.getElementById(`theme-${theme.name}`);
    
    if (button) {
      button.addEventListener("click", () => {
        setTheme(theme);
        updateStatus(theme);
        closeMenu();
      });
    }
  });

  const toggleButton = document.getElementById("theme-switcher-button");
  const menu = document.getElementById("theme-dropdown");

  toggleButton.addEventListener("click", () => {
    const isExpanded = toggleButton.getAttribute("aria-expanded") === "true";
    if (isExpanded) {
      closeMenu();
    } else {
      openMenu();
    }
  });

  function closeMenu() {
    menu.setAttribute("hidden", "");
    toggleButton.setAttribute("aria-expanded", "false");
  }

  function openMenu() {
    menu.removeAttribute("hidden");
    toggleButton.setAttribute("aria-expanded", "true");
  }

  // Close the menu when clicking outside
  document.addEventListener("click", (e) => {
    if (!toggleButton.contains(e.target) && !menu.contains(e.target)) {
      closeMenu();
    }
  });
});
```
