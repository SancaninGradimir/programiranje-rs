## Drvo pristupa

Drvo pristupačnosti je struktura koju koriste asistivne tehnologije, kao što su čitači ekrana, kako bi razumele i interagovale sa sadržajem na web stranici.

## Funkcija za `max()`

Funkcija `max()` vraća najveću vrednost među skupom vrednosti odvojenim zarezima:

:::interactive_editor

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <img src="https://cdn.freecodecamp.org/curriculum/css-photo-gallery/9.jpg" alt="A cat sitting on a table">
</body>
```

```css
img {
  width: max(250px, 25vw);
}
```

:::

U gornjem primeru, širina slike će biti 250px ako je širina vidokruga manja od 1000 piksela. Ako je širina vidokruga veća od 1000 piksela, širina slike će biti 25vw. To je zato što je 25vw jednako 25% širine vidokruga.

## Najbolji način korišćenja CSS za pristup

- **`display: none;`**: Korišćenje `display: none;` znači da čitači ekrana i druge pomoćne tehnologije ne mogu pristupiti ovom sadržaju, jer nije uključen u stablo pristupačnosti. Stoga je važno koristiti ovu metodu samo kada želite potpuno ukloniti sadržaj iz vizuelne prezentacije i pristupačnosti.
- **`visibility: hidden;`**: Ovaj atribut i njegova vrednost skrivaju sadržaj od vida, ali ga ostavljaju u toku dokumenta, što znači da i dalje zauzima prostor na stranici. Ove stavke takođe neće biti ponovo pročitane čitačima ekrana jer će biti uklonjene iz stabla pristupačnosti.
- **Darasa la CSS la `.sr-only`**: Ovo je uobičajena tehnika koja se koristi za skrivanje sadržaja od vida, a istovremeno ga čini dostupnim čitačima ekrana.

:::interactive_editor

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <span>This span element can be read</span>
  <span class="sr-only">This span element can only be read by screen readers</span>
  <p style="display: none">
    This text is hidden with display: none and is NOT accessible to screen readers.
  </p>
  <p style="visibility: hidden">
    This text is hidden with visibility: hidden and is NOT read by screen readers,
    but it still takes up space.
  </p>
</body>

```

```css
.sr-only {
  position: absolute;
  width: 1px;
  height: 1px;
  padding: 0;
  margin: -1px;
  overflow: hidden;
  clip: rect(0, 0, 0, 0);
  white-space: nowrap;
  border: 0;
}

```

:::

- **`scroll-behavior: smooth;`**: Ovo svojstvo i njegova vrednost omogućavaju linearno kretanje.

## Skrivanje sadržaja za atribute HTML

- **Svojstvo `aria-hidden`**: Koristi se za skrivanje elemenata od osoba koje koriste asistivne tehnologije, kao što su čitači ekrana. Na primer, ovo se može koristiti za skrivanje dekorativnih slika koje ne pružaju značajan sadržaj.
- **Svojstvo `hidden`**: Ovo svojstvo je podržano od strane mnogih savremenih pregledača i skriva sadržaj kako vizuelno, tako i iz stabla pristupačnosti. Može se lako modifikovati korišćenjem JavaScript.

:::interactive_editor

```html
<p aria-hidden>This paragraph is visible for sighted users, but is hidden from assistive technology.</p>
<p hidden>This paragraph is hidden from both sighted users and assistive technology.</p>
<p>This is a normal paragraph</p>

```

:::

## Izazov pristupa atributu `placeholder`

Korišćenje naslovnog teksta nije dobro za pristupačnost. Često korisnici mešaju naslovni tekst sa stvarnom vrednošću unosa – misle da u unosu već postoji vrednost.
