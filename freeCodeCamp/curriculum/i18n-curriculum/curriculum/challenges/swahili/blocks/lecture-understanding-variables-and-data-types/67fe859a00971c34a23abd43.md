# --description--

Pre nego što radite sa varijablama od `Python`, važno je razumeti tipove podataka. Tip podataka opisuje vrstu vrednosti koju varijabla skladišti. Na primer, broj, niz znakova ili lista stavki. Programski jezici koriste tipove podataka kako bi znali kako da skladište i rade sa različitim vrstama informacija.

Python je jezik sa dinamički tipizovanim tipovima (dynamically-typed), kao JavaScript, što znači da nemate potrebe eksplicitno deklarisati tipove varijabli. Jezik zna tip podataka varijable na osnovu onoga što joj date unutra.

Evo nekoliko primera:

```python
name = 'John Doe' # Python knows this is a string
age = 25 # Python knows this is an integer
```

Ovo je drugačije od nekih statički tipiziranih jezika kao što su C#, Java i C++, gde morate deklarisati tipove promenljivih, kao što sledi:

```csharp
string name = 'John Doe'
int age = 25
```

Dinamička priroda tipova u Pythonu čini pisanje koda bržim i fleksibilnijim, ali može izazvati neočekivane greške jer se greške tipova otkrivaju tek kada se program pokrene, a ne tokom razvoja programa (compile).

Pošto `Python` bira tipove podataka kada je vaš program **pokrenut**, greške vezane za tip se detektuju samo u tom trenutku. Kada program radi, `Python` izvršava vaš kod red po red. Ako dođe do reda gde nešto treba da uradi nešto, ali ne može, `Python` će se zaustaviti i prikazati grešku.

Nasuprot tome, neki jezici **kompiliraju programe pre izvršavanja**. Kompilacija znači da računar vaš kod proverava unapred i priprema ga za pokretanje. U ovoj fazi, ti jezici mogu da otkriju greške tipa čak i pre nego što je program započeo.

Nema potrebe da učiš te jezike još. Važna ideja je samo ovo:

- **U Python tipovi grešaka mogu nastati tokom izvršavanja**, kada program pokreće i koristi vaš kod.

- **Kompajlirani jezici (compiled) otkrivaju greške tipova tokom faze izgradnje**, pre nego što programu bude dozvoljeno da se pokrene.

Zbog toga, možda ne znate za tip grešku u Python dok program dođe do te specifične linije koda tokom izvršavanja.

Ovo su uobičajeni tipovi podataka koje ćete koristiti u Python:

- Integer: Cjelobrojni broj bez decimalnih delova, na primer, `10` ili `-5`.

```python
my_integer_var = 10
print('Integer:', my_integer_var) # Integer: 10
```

- Float: Broj sa decimalnim delom, kao što su `4.41` au `-0.4`.

```python
my_float_var = 4.50
print('Float:', my_float_var) # Float: 4.5
```

- String: Sekvenca slova postavljena unutar pojedinačnih ili dvostrukih navodnika, kao `'Hello world!'`.

```python
my_string_var = 'hello'
print('String:', my_string_var) # String: hello
```

- Boolean: Vrednost istina ili laž, napisana je kao `True` ili `False`.

```python
my_boolean_var = True
print('Boolean:', my_boolean_var) # Boolean: True
```

- Set: Neuređeni skup različitih elemenata, kao što je `{0.5, 4, 'apple'}`.```python
my_set_var = {7, 'hello', 8.5}
print('Set:', my_set_var) # Set: {7, 'hello', 8.5}
```

- Diktionar: Kolekcija parova ključ-vrednost postavljena unutar vitičastih zagrada, kao što je `{'name': 'John Doe', 'age': 28}`.

```python
my_dictionary_var = {'name': 'Alice', 'age': 25}
print('Dictionary:', my_dictionary_var) # Dictionary: {'name': 'Alice', 'age': 25}
```

- Tuple: Kolekcija uređene i nepromenljive vrednosti, koja je smeštena unutar standardnih zagrada, kao što je `('apple', 4.5, 7)`.

```python
my_tuple_var = (7, 'hello', 8.5)
print('Tuple:', my_tuple_var) # Tuple: (7, 'hello', 8.5)
```

- Opseg: Niz brojeva, često se koristi u petljama (ciklusima), na primer, `range(5)`.

```python
my_range_var = range(5)
print('Range:', my_range_var) # Range: range(0, 5)
```

- List: Organizovano sakupljanje stavki koje podržavaju različite vrste podataka.```python
my_list = [22, 'Hello world', 3.14, True]
print(my_list) # [22, 'Hello world', 3.14, True]
```

- None: Posebna vrednost koja predstavlja odsustvo vrednosti.

```python
my_none_var = None
print('None:', my_none_var) # None: None
```

U narednim studijama, naučićeš više o tome kako da radiš sa svim ovim vrstama podataka.
