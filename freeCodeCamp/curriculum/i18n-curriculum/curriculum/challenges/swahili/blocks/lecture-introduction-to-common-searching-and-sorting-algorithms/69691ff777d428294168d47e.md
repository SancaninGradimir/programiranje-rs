# --description--

Metoda razdvajanja i osvajanja u računarstvu je tehnika podizanja problema na manje probleme pomoću rekurzije. Jedna od ključnih komponenti ove metode je rekurzija, koja nastaje kada funkcija poziva samu sebe više puta sve dok ne dođe do baznog slučaja. U ovom kursu ćemo pogledati algoritam Merge Sort kako bismo bolje razumeli kako metoda razdvajanja i osvajanja funkcioniše.

Recimo da imamo ovu listu brojeva:

```md
42 37 53 17
```

Cilj je da se sortira ta lista od manjeg ka većem koristeći Merge Sort algoritam. Prvi korak je podeliti tu listu na pola:

```md
42 37 | 53 17
```

Zatim moramo pogledati levu stranu liste:

```md
42 37
```

Uzimamo taj mali spisak i ponovo ga delimo na pola sve dok svaki podspisak ne sadrži samo jedan element:

```md
42 | 37
```

Lista sa samo jednim elementom je podešena po defaultu. Zatim moramo spojiti svaku manju listu koja ima jedan element u jednu sortiranu listu:

```md
37 42
```

Zatim sledimo isti proces sa desne strane početne liste:

```javascript
// right side of original list
53 17

// divide the list in half
53 | 17

// merge the lists in sorted order
17 53
```

Pošto su obe polovine početne liste poređene, spajamo te dve polovine zajedno i sortiramo elemente:

```markdown
17 37 42 53
```

Ovo je kako algoritam izgleda u kodu:

```javascript
function mergeSort(arr) {
    if (arr.length <= 1) {
        return arr;
    }

    const mid = Math.floor(arr.length / 2);
    const left = mergeSort(arr.slice(0, mid));
    const right = mergeSort(arr.slice(mid));

    const sorted = [];
    let i = 0;
    let j = 0;

    while (i < left.length && j < right.length) {
        if (left[i] <= right[j]) {
            sorted.push(left[i]);
            i += 1;
        } else {
            sorted.push(right[j]);
            j += 1;
        }
    }

    return sorted.concat(left.slice(i)).concat(right.slice(j));
}
```

Vremenska složenost za merge sort će biti `O(n log n)` jer se lista stalno deli na pola (`log n`) a zatim spaja zajedno (`O(n)`). Za razliku od drugih algoritama za sortiranje kao što je bubble sort, merge sort ne radi in-place i ima prostornu složenost `O(n)`.
