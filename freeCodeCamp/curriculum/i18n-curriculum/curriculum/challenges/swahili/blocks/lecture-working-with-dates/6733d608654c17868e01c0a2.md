# --interactive--

Na prošloj lekciji, naučili ste kako da radite sa nečim iz `Date` u JavaScript. Ali postoji nekoliko različitih načina za kreiranje datuma u JavaScript.

U ovom tutorijalu, ćemo pogledati kako da radimo sa metodama `toISOString()`, `toString()`, i `toLocaleDateString()` za kreiranje datuma u JavaScript.

Pre nego što pogledamo različite načine, prvo da pregledamo izgled stvari od `Date`:

:::interactive_editor

```js
const date = new Date();
console.log(date);
```

:::

Kada zalepiš nešto sa `date` u konzolu, videćeš trenutni datum i vreme prema podešavanjima sistema korisnika. Evo primera rezultata koje ćeš videti:

```js
Sun Sep 29 2024 19:45:37 GMT-0700 (Pacific Daylight Time)
```

Ako koristiš metodu ``toString()`` na objektu ``date``, videćeš rezultate slične onima prethodnim.

Da biste kreirali datum u proširenom ISO formatu (ISO 8601), možete koristiti metodu `toISOString()` kao što sledi:

:::interactive_editor

```js
const date = new Date();
console.log(date.toISOString());
```

:::

ISO 8601 je međunarodni standard za predstavljanje datuma i vremena. Njegov format je `YYYY-MM-DDTHH:mm:ss.sssZ`.

Evo primera rezultata koje ćete videti:

```js
2024-09-30T02:47:20.292Z
```

Drugi način za kreiranje datuma je korišćenje metode `toLocaleDateString()`. Ova metoda vam omogućava kreiranje datuma u zavisnosti od jezika i lokacije korisnika. Evo osnovne sintakse za korišćenje metode `toLocaleDateString()`:

:::interactive_editor

```js
const date = new Date();
console.log(date.toLocaleDateString());
```

:::

Evo primera rezultata koje ćete videti:

```js
9/29/2024
```

Putanj za `toLocaleDateString()` prima dva opciona parametra: `locales` i `options`.

Parametar za `locales` je niz znakova koji predstavlja željenu lokaciju jezika. Na primer, možete uneti `en-US` za engleski jezik (SAD) ili `fr-FR` za francuski jezik (Francuska). Ako ne navedete parametar `locales`, koristi se podrazumevana lokacija jezika. Evo primera kako koristiti parametar `locales`:

:::interactive_editor

```js
const date = new Date();
console.log(date.toLocaleDateString("fr-FR"));
```

:::

Drugi opcioni parametar je parametar `options`. Ovaj parametar je nešto što vam omogućava da odredite strukturu sekvence znakova datuma. Evo primera kako koristiti parametar `options`:

:::interactive_editor

```js
const date = new Date();
const options = {
  weekday: "long",
  year: "numeric",
  month: "long",
  day: "numeric",
};
console.log(date.toLocaleDateString("en-GB", options));
```

:::

U prethodnom primeru, identifikovali smo opciju korišćenja engleskog jezika (Velika Britanija) i prikazivanje punog dana u nedelji, godine, meseca i dana.

Evo primera rezultata koje ćete videti:

```js
Sunday, September 29, 2024
```

U ovom tutorijalu, dotakli smo samo neke načine za kreiranje datuma u JavaScript. Postoji mnogo više metoda i biblioteka dostupnih koje mogu pomoći pri kreiranju datuma u JavaScript. Međutim, metode `toISOString()` i `toLocaleDateString()` su dobar početak za kreiranje datuma u JavaScript.
