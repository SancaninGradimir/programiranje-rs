# --description--

Za razliku od rada sa malim vanilom projektima HTML, CSS, na JavaScript, kreiranje novog projekta React uključuje više koraka kako bi se osiguralo da sve radi ispravno. Umesto da pokušavate sami da konfigurišete sva neophodna podešavanja, postoje alati koji će to učiniti za vas.

Jedan od najpopularnijih alata za podešavanje projekata je Vite. Vite, što na francuskom znači „brz“, predstavlja alat za izgradnju koji je usmeren na pružanje brzog razvojnog iskustva za moderne veb projekte. Vite se može koristiti sa React-om, kao i sa drugim bibliotekama i framework-ovima poput Vue-a ili Svelte-a. Možete ga koristiti čak i za obične JavaScript projekte.

Da biste kreirali novi projekat koristeći Vite, biće vam potrebno da koristite komandnu liniju. Ako koristite računar sa operativnim sistemom Windows, možete koristiti Command Prompt ili Windows PowerShell. Ako koristite Mac, možete koristiti aplikaciju Terminal.

Ako još uvek niste instalirali Node.js, što je potrebno da bi npm radio, ovde je link ka zvaničnom sajtu Node.js: https://nodejs.org. Pratite uputstva za instalaciju, a zatim se vratite ovde da nastavite sa instalacijom React.

Kada je komandni red aktivan, možete koristiti sledeću komandu:

```bash
npm create vite@latest my-react-app -- --template react
```

Ova komanda kreira novi projekat za React, nazvan `my-react-app`, koristeći šablon React od Vite.

Zatim možete otvoriti novi projekat i videti osnovni kod React koji vam je Vite dao.

Jedna od sjajnih stvari kod Vite-a jeste to što će generisati samo datoteke koje su zaista neophodne za pokretanje React projekta.

Da biste pokrenuli projekat, potrebno je da instalirate zavisnosti pomoću sledećih komandi u komandnoj liniji.

```bash
cd my-react-app
npm install
```

Komanda `cd my-react-app` osigurava da pređete u odgovarajući direktorijum u kojem se nalazi vaš projekat.

Komanda `npm install` služi za instalaciju potrebnih zavisnosti kako bi vaš React projekat ispravno radio. Zavisnosti su biblioteke ili paketi koji su potrebni projektu da bi pravilno funkcionisao, kao što su sam React, ReactDOM ili paketi trećih strana koji pružaju dodatne funkcionalnosti.

Izvođenje `npm install` čita fajl `package.json` u direktorijum vašeg projekta i instalira sve zavisnosti navedene tamo, što vam omogućava da izgradite i pokrenete svoju aplikaciju za React bez propuštanja bilo kog ključnog dela.

Fail ``package.json`` je važna konfiguraciona datoteka u projektima koja sadrži metadata o vašem projektu, uključujući njegovo ime, verziju i zavisnosti. Takođe opisuje skripte, informacije o licenci i druge postavke koje pomažu u upravljanju projektom i njegovim zavisnostima.

Kada se instaliraju zavisnosti, pronaći ćete novi direktorij u vašem projektu pod nazivom `node_modules`.

Direktorijum `node_modules` je mesto gde su svi paketi i biblioteke potrebne vašem projektu sačuvane. Ovaj direktorijum sadrži listu zavisnosti navedenih u vašoj datoteci `package.json`, uključujući direktne zavisnosti vašeg projekta i takođe zavisnosti tih zavisnosti.

Za pokretanje svog projekta, koristite komandu `npm run dev` zatim otvorite novu tab u pretraživaču za `http://localhost:5173/`.

Moći ćete da vidite početni šablon koji vam je Vite dao.

Ako želite da zaustavite lokalni server, pritisnite `CTRL + C` na komandnoj liniji.

Da biste videli kod ovog početnog šablona, možete ući u svoj projekat u direktorijum ``src`` i pronaći ćete fajl ``App.jsx``.

Odavde možete početi da menjate fajlove, sačuvati svoje izmene i videti nove promene prikazane u pretraživaču.

Dakle, spreman si da započneš sa kreiranjem svoje aplikacije za React!
