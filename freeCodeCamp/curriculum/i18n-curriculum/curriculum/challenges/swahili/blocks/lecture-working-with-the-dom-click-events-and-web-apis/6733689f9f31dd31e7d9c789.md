# --description--

Naučimo se o čvorovima DOM i njihovom odnosu u drvetu DOM.

Slično kako stvarno drvo ima velike i male grane povezane u hijerarhijsku strukturu, čvorovi za DOM takođe imaju direktne i indirektne veze među njima. Koristićemo ovaj primer da pokažemo ove odnose:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>DOM Tree Example</title>
  </head>
  <body>
    <h1>Heading 1</h1>
    <p>Paragraph 1</p>
    <ul>
      <li>List item 1</li>
      <li>List item 2</li>
    </ul>
  </body>
</html>
```

Počnimo od samog vrha. Koren DOM stabla je `html` element. On je kontejner najvišeg nivoa za sav sadržaj HTML dokumenta. Svi ostali čvorovi su potomci ovog korenskog čvora.

Zatim, ispod korenskog čvora, nalazimo druge čvorove u hijerarhijskoj strukturi. Roditeljski čvor je element koji sadrži druge elemente. Dete čvor je element koji se nalazi unutar drugog elementa.

U ovom primeru, element `body` je roditelj elementa pasusa, dok je element pasusa dete elementa `body`. Ovi elementi su predstavljeni kao čvorovi u DOM stablu i imaju isti odnos.

Kao što imamo roditeljske i dečje čvorove, takođe imamo i čvorove braće i sestara. Čvorovi braće i sestara su elementi koji dele istog roditelja. U našem primeru, dva elementa stavke liste dele istog roditelja, neuređenu listu, pa su zato braća i sestre.

Slično tome, elementi `h1` i pasus su braća i sestre jer dele istog roditelja, element `body`.

Takođe imamo direktne odnose na različitim nivoima hijerarhijske strukture. Čvorovi potomci su elementi koji se nalaze unutar drugog elementa, direktno ili indirektno. Element se smatra potomkom drugog elementa ako se do njega može doći prolaskom niz DOM stablo od njegovog pretka.

U našem primeru, `li` elementi liste su potomci elementa `body`, jer se nalaze unutar njega indirektno.

Čvor pretka je element koji je najviši u hijerarhijskoj strukturi drveta DOM od bilo kog drugog elementa. U našem primeru, element `body` je pretak elemenata `li` u elementu liste stavki.

Razumevanje ovih odnosa je važno za upravljanje i kretanje kroz DOM stablo pomoću JavaScript-a.
