# --description--

Razmotrite neograničenu seriju polinoma $A_G(x) = xG_1 + x^2G_2 + x^3G_3 + \cdots$, gde je $G_k$ k-ti element rekurentne sekvence drugog reda, definisane kao $G_k = G_{k − 1} + G_{k − 2}, G_1 = 1$ i $G_2 = 4$; tj., $1, 4, 5, 9, 14, 23, \ldots$.

Za ovaj zadatak, razmotrićemo vrednosti $x$ za koje je $A_G(x)$ pozitivan ceo broj.

Odgovarajuće vrednosti za $x$ za prvih pet prirodnih brojeva prikazane su ispod.

| $x$                         | $A_G(x)$ |
|-----------------------------|----------|
| $\frac{\sqrt{5} − 1}{4}$    | $1$      |
| $\frac{2}{5}$               | $2$      |
| $\frac{\sqrt{22} − 2}{6}$   | $3$      |
| $\frac{\sqrt{137} − 5}{14}$ | $4$      |
| $\frac{1}{2}$               | $5$      |

Nazivaćemo $A_G(x)$ zlatni komad ako je $x$ broj jednačine jer postaju retki; na primer, zlatni komad 20 je 211345365. Pronađite zbir prvih trideset zlatnih komada.
