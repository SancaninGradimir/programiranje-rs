# --description--

`To handle` Nešto znači da ga obrađuješ ili administriraš. U programu, `handling` uključuje upravljanje podacima ili operacijama. Na primer, `to handle user input means to manage what happens when a user enters data`.

`Error handling` radi se o rukovanju greškama koje nastaju tokom izvršavanja programa. Povezano je sa pisanjem koda koji može da upravlja ovim greškama bez padanja. Na primer, `using try-except blocks in Python to handle errors that might occur when a program runs`.
