# --interactive--

Dizajn veb stranice koji se prilagođava uređaju je metoda web razvoja koja teži kreiranju sajtova koji pružaju optimalno iskustvo pregleda i interakcije na različitim tipovima uređaja, od desktop računara do mobilnih telefona.

Osnovni princip responsivnog dizajna je prilagodljivost – sposobnost sajta da prilagodi svoj raspored i sadržaj u zavisnosti od veličine ekrana i kapaciteta uređaja sa kojeg se gleda.

Standardni prilagodljiv dizajn zavisi od tri glavne komponente – lako prilagodljivi gridovi, prilagodljive slike i medijski upiti:

- Adaptivni gridovi koriste relativne jedinice kao procenat umesto fiksnih jedinica poput piksela, omogućavajući sadržaju da menja veličinu i prilagođava se veličini ekrana.

- Dinamične slike su postavljene da se menjaju unutar svojih sadržajnih elemenata, osiguravajući da ne prelaze granice svog kontejnera na manjem ekranu.

- Media upiti omogućavaju programerima da koriste različite stilove u zavisnosti od karakteristika uređaja, posebno širine vidokruga. Naučićete više o media upitima u narednim lekcijama.

Odnos između responsivnog dizajna veb stranice i alata kao što su CSS Grid i Flexbox je sinergijski. Dok je responsivni dizajn koncept ili metoda, CSS Grid i Flexbox su praktični alati koji pojednostavljuju i povećavaju efikasnost implementacije responsivnog dizajna.

U prethodnim časovima ste naučili kako da radite sa Flexbox-om, a u narednim ćete naučiti kako da radite sa CSS Grid. Ali za sada, evo kratkog uvodnika o CSS Grid.

CSS Mreža je dvodimenzionalni sistem za raspored koji omogućava složenije podešavanje. Idealna je za kreiranje ukupnog rasporeda stranice zajedno sa manjim podešavanjima sekcija.

Ovo je primer kako CSS Grid može biti korišćen za adaptivan dizajn:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div class="grid-container">
  <div class="grid-item">Item 1</div>
  <div class="grid-item">Item 2</div>
  <div class="grid-item">Item 3</div>
</div>
```

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr;
  gap: 20px;
}

@media (min-width: 768px) {
  .grid-container {
    grid-template-columns: 1fr 1fr;
  }
}

@media (min-width: 1024px) {
  .grid-container {
    grid-template-columns: 1fr 1fr 1fr;
  }
}
```

:::

U ovom primeru, grid počinje sa jednim redom na malim ekranima, menja se u dva reda na srednjim ekranima, a zatim postaje tri reda na velikim ekranima. Ovo pokazuje kako Grid može da kreira responsivne dizajne koji se prilagođavaju različitim veličinama ekrana.

Sve jezište/koncepti Flexbox i Grid nude velike prednosti u poređenju sa starijim metodama rasporeda kao što su *floats* ili podešavanja tabela. One pružaju veću fleksibilnost i kontrolu, zahtevaju manji kod, i dizajnirane su uzimajući u obzir responsivan dizajn. Omogućavaju programerima da kreiraju složene, lako prilagodljive dizajne, pa time postaju ključni alat u implementaciji responsivnog web dizajna.

Važno je znati da se Flexbox i Grid često koriste zajedno u responsivnom dizajnu. Flexbox se obično koristi za manje sekcije i jednodimenzionalno raspoređivanje, dok se Grid koristi za ukupnu strukturu stranice i dvodimenzionalno raspoređivanje. Izbor između njih često zavisi od specifičnih potreba rasporeda.

Osim od Flexboxa i Grida, drugi elementi za CSS imaju važnu ulogu u responsivnom dizajnu.

Funkcija `calc()`, na primer, omogućava kombinovanje mera i izvođenje proračuna, što može biti veoma korisno pri kreiranju dinamičnih struktura.

Adaptivne slike su još jedna važna karakteristika responsivnog web dizajna. Svojstvo `srcset` i element `picture` u HTML5 omogućavaju dostavljanje različitih fajlova slika u zavisnosti od kapaciteta uređaja, osiguravajući da korisnici ne učitavaju velike fajlove slika bez potrebe na uređajima sa malim ekranom ili niskom rezolucijom.

Zaključno, responsivni dizajn sajta je pristup koji ima za cilj kreiranje sajtova koji funkcionišu dobro na bilo kom uređaju.

Iako je dizajn filozofija umesto specifične tehnologije, oslanja se na značajke CSS kao što su medijski upiti, i veoma je podržavaju moderne alati za raspored kao što su Flexbox i Grid. Ovi alati pružaju fleksibilnost i kontrolu potrebne za kreiranje zaista responsivnih dizajna, omogućavajući veb-sajtu da se lako prilagodi različitim tipovima uređaja koji se koriste za pristup internetu.
