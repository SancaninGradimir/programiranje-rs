# --explanation--

Lisa koristi `Second Conditional` da diskutuje o hipotetičkoj situaciji.

`Second Conditional` koristi se za razgovor o situacijama koje nisu stvarne ili nemoguće sada ili u budućnosti. Prati strukturu `if` + `Present Simple` ili `Past Continuous` (uslovno) i `might / would / could` + osnovni glagol (za rezultate). Primer:

- `If I had more time, I would learn a new programming language.` - Ovo opisuje hipotetičku situaciju u kojoj bi više vremena dovelo do učenja.

- `If the server were faster, it could handle more users.` - Server trenutno nije brz, ali kada bi to bio, mogao bi da podrži mnogo više korisnika.
