# --explanation--

Negativna forma vremena za `Present Perfect continuous` formira se pomoću `hasn't/haven't + been + present participle`. Opisuje radnju koja je počela u prošlosti i koja se nastavlja ili je važna, ali nije dogodila se kako je očekivano.
