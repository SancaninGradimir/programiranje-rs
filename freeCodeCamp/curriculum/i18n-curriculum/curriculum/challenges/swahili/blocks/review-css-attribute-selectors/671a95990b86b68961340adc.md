# --interactive--

## Rad sa različitim selektorima za atribute elemenata i linkove

- **Značenje**: Selector `attribute` omogućava ciljanje elemenata HTML na osnovu njihovih atributa, kao što su atributi `href` ili `title`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Link with href</a>
<a>No href</a>
```

```css
a[href] {
  color: blue;
  text-decoration: underline;
}
```

:::

- **Atribut za `title`**: Ovaj atribut pruža dodatne informacije o elementu. Evo kako možete ciljati elemente sa atributom `title`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Tooltip">Link with title</a>
<a href="#">Normal link</a>
```

```css
a[title] {
  font-weight: bold;
  text-decoration: none;
}
```

:::

- **Kombinujte selektore kako biste ciljali elemente sa atributima za `href` i `title`**: Ovo je kombinacija više selektora za atribute elemenata.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="#" title="Info">Href + Title</a>
<a href="#">Only href</a>
```

```css
a[href][title] {
  color: green;
}
```

:::

- **Cilja jednu reč unutar liste razdvojene razmakom**: Ovo cilja linkove koji sadrže specifično ime klase.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a class="link primary">Primary link</a>
<a class="link secondary">Secondary link</a>
```

```css
a[class~="primary"] {
  color: red;
  font-weight: bold;
}
```

:::

- **Ciljanje vrednosti koje prethode sa određenim prefiksom**: Ovo cilja elemente koji počinju sa `https://`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">HTTPS link</a>
<a href="http://www.freecodecamp.org">HTTP link</a>
```

```css
a[href^="https://"] {
  color: green;
  text-decoration: underline;
}
```

:::

- **Ciljanje vrednosti koje se završavaju sa specifičnim sufiksom**: Ovo cilja elemente koji se završavaju sa `.jpg`.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="photo.jpg">Image link</a>
<a href="index.html">HTML link</a>
```

```css
a[href$=".jpg"] {
  color: darkgreen;
  text-decoration: underline dotted;
}
```

:::

- **Ciljanje vrednosti koje sadrže podstring znakova bilo gde**: Ovo cilja elemente koji sadrže `https` bilo gde unutar vrednosti.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<a href="https://www.freecodecamp.org">Secure link</a>
<a href="page.html">Local link</a>
```

```css
a[href*="https"] {
  color: teal;
}
```

:::

- **Sažetak**: Ovi sistemi olakšavaju kreiranje šema veza uzimanjem u obzir njihove atribute i vrednosti na stabilan način.

## Ciljanje elemenata sa karakteristikama za `lang` i `data-lang`

- **Svojstvo za `lang`**: Ovo svojstvo se koristi u HTML da bi odredilo jezik sadržaja unutar elementa. Možda želite kreirati različite stilove za elemente u zavisnosti od korišćenog jezika, posebno na višejezičnom sajtu.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p lang="en">English text</p>
<p lang="fr">French text</p>
```

```css
p[lang="en"] {
  font-style: italic;
}
```

:::

- **Svojstvo `data-lang`**: Svojstva standardnih podataka, kao što je svojstvo `data-lang`, često se koriste za skladištenje dodatnih informacija unutar elemenata, kao što je određivanje jezika koji se koristi u određenom delu teksta. Evo kako možete kreirati stilove za elemente zasnovane na svojstvu `data-lang`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<div data-lang="fr">French content</div>
<div data-lang="en">English content</div>
```

```css
div[data-lang="fr"] {
  color: blue;
}
```

:::

## Raditi sa selektorom atributa elementa, elementima strukturirane liste i atributom `type`

- **Svojstvo `type`**: Kada radi sa listama sortiranim u HTML, svojstvo `type` omogućilo je identifikovanje tipa broja koji se koristi, kao što su decimalni brojevi, abeceda ili rimski brojevi.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<ol type="A">
  <li>Alpha</li>
  <li>Beta</li>
</ol>

<ol type="i">
  <li>One</li>
  <li>Two</li>
</ol>
```

```css
/*Example targeting uppercase alphabetical numbering*/
ol[type="A"] {
  color: purple;
  font-weight: bold;
}

/*Example targeting lowercase Roman numerals*/
ol[type="i"] {
  color: green;
}
```
