# --description--

Postoje dva ograničenja za nazive parametara u JavaScript:

1. Ime mora da sadrži samo slova, brojeve ili simbole `$` i `_`.
1. Prvo slovo ne sme da bude broj.

Primeri ispravnih imena:

```js
let userName;
let test123;
```

Zanimljivo je – dolar znak `'$'` i podvučani `'_'` takođe mogu se koristiti u imenima. To su obični simboli, kao slova, bez bilo kakvog posebnog značenja.

Ovi nazivi su važeći:

```js
let $ = 1; // declared a variable with the name "$"
let _ = 2; // and now a variable with the name "_"

console.log($ + _); // 3
```

Primeri neispravnih imena parametara:

```js
let 1a; // cannot start with a digit

let my-name; // hyphens '-' aren't allowed in the name
```
