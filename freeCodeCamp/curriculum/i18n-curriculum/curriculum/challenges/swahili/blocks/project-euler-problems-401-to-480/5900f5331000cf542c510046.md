# --description--

Neka je $f(n)$ pozitivni ceo broj $x$ manji od ${10}^9$, gde poslednjih devet cifara $n^x$ obrazuje broj $x$ (uključujući vodeće nule), ili nula ako ne postoji takav broj.

Na primer:

$$\begin{align}
  & f(4) = 411\\,728\\,896 (4^{411\\,728\\,896} = ...490\underline{411728896}) \\\\
  & f(10) = 0 \\\\
  & f(157) = 743\\,757 (157^{743\\,757} = ...567\underline{000743757}) \\\\
  & \Sigma f(n), 2 ≤ n ≤ 103 = 442\\,530\\,011\\,399
\end{align}$$

Izračunajte $\sum f(n)$, $2 ≤ n ≤ {10}^6$.
