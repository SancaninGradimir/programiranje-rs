# --description--

Kao podsjetnik, ovaj projekat je izgrađen na osnovu početnog projekta preuzet iz <a href="https://github.com/freeCodeCamp/boilerplate-infosec/" target="_blank" rel="noopener noreferrer nofollow">GitHub</a>.

Ovaj izazov pokazuje obećavajuću novu zaštitu koja može značajno smanjiti rizike i uticaje raznih vrsta napada u modernim pretraživačima. Postavljanjem i konfiguracijom politike bezbednosti sadržaja možete sprečiti ubacivanje bilo čega neočekivanog na vašoj stranici. Ovo će zaštititi aplikaciju od XSS ranjivosti, neželjenog praćenja, štetnih frame-ova i mnogo više. CSP funkcioniše definisanjem liste dozvoljenih i pouzdanih izvora sadržaja. Možete ga konfigurisati za sve vrste resursa koje veb stranica može zahtevati (skripte, stilski listovi, fontovi, frame-ovi, mediji, itd.). Dostupno je mnogo uputstava, tako da vlasnik sajta može imati detaljnu kontrolu. Pogledajte HTML 5 Rocks, KeyCDN za više informacija. Nažalost, CSP nije podržan od strane starijih pretraživača.

Po podrazumevanju, uputstvo je vrlo jasno, pa je važno postaviti `defaultSrc` kao alternativu. Helmet podržava formate imena za `defaultSrc` i default-src. Alternativa se koristi za mnoge neobjavljene uputstva.

# --instructions--

U ovoj vežbi, koristite ``helmet.contentSecurityPolicy()``. Konfigurišite dodavanjem stavke ``directives``. U toj stavci, postavite ``defaultSrc`` na ``["'self'"]`` (lista dozvoljenih izvora mora biti u redovima podataka), kako biste podrazumevano verovali samo adresi vaše veb stranice. Takođe postavite direktivu ``scriptSrc`` da dozvolite skriptama da se preuzimaju samo sa vaše veb stranice (``'self'``) i sa domena ``'trusted-cdn.com'``.

Napomena: u reči `'self'`, pojedinačni navodnik je deo te reči sam po sebi, pa mora biti okružen dvostrukim navodnicima da bi funkcionisao.

# --hints--

Centralni program, `helmet.contentSecurityPolicy()` mora biti pravilno podešen

```js
const response = await fetch(code + '/_api/app-info');
if (!response.ok) {
  throw Error(await response.text());
}
const data = await response.json();
assert.include(data.appStack, 'csp');
```

Tvoja CSP konfiguracija nije ispravna. defaultSrc treba da bude ["'self'"] i scriptSrc treba da bude ["'self'", 'trusted-cdn.com']

```js
const response = await fetch(code + '/_api/app-info');
if (!response.ok) {
  throw Error(await response.text());
}
const data = await response.json();
var cspHeader = Object.keys(data.headers).filter(function (k) {
  return (
    k === 'content-security-policy' ||
    k === 'x-webkit-csp' ||
    k === 'x-content-security-policy'
  );
})[0];
assert.equal(
  data.headers[cspHeader],
  "default-src 'self'; script-src 'self' trusted-cdn.com"
);
```
