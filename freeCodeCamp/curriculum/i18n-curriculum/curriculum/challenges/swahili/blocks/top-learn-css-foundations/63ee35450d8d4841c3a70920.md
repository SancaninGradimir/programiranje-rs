# --description--

CSS interni (ili uključeni CSS) podrazumeva dodavanje CSS unutar samog fajla HTML umesto kreiranja potpuno novog fajla. Interno, postavljate sve pravila unutar pare za kružne zagrade za otvaranje i zatvaranje `<style>`, koje se zatim postavljaju unutar kružnih zagrada za otvaranje i zatvaranje `<head>` vašeg fajla HTML. Pošto se stilovi postavljaju direktno unutar kružnih zagrada `<head>`, više nećete imati potrebu za elementom/funkcijom `<link>` koji je potreban eksternim metodama.

Osim ovih razlika, sintaksa je potpuno isto kao spoljni način (selektori, vitičaste zagrade, deklaracija):

```html
<head>
  <style>
    div {
      color: white;
      background-color: black;
    }

    p {
      color: red;
    }
  </style>
</head>
<body>...</body>
```

Ova metoda može biti korisna za dodavanje jedinstvenih stilova na jednu stranicu sajta, ali ne čini stvari drugačije kao eksterni način, a u zavisnosti od broja pravila i deklaracija, može uzrokovati da fajl HTML bude prevelik.

# --questions--    

## --text--

Koja je razlika između unutrašnjih i spoljašnjih metoda CSS?

## --answers--

Eksterni put postavlja pravila za CSS u različit fajl, dok interni put postavlja pravila za CSS unutar samog fajla HTML.

---

Unutrašnji put postavlja CSS različito od HTML, dok spoljašnji put unosi CSS direktno u HTML.

---

Interna putanja koristi funkciju `<link>` za povezivanje CSS i HTML, dok eksternu putanja unosi CSS direktno u HTML.
