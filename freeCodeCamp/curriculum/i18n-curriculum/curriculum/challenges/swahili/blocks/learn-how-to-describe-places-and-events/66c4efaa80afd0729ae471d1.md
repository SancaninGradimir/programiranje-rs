# --explanation--

`Might` koristi se za prikaz mogućnosti. Kada Sara kaže, `might not be running`, predlaže da je moguće da direktna server ekstenzija ne radi ili nije trenutno omogućena. Na primer:

- `The server might be down.` - Moguće je da server ne radi.

- `She might not be available today.` - Moguće je da danas nije slobodan.
