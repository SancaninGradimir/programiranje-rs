# --description--

Pig Latin je način za transformisanje engleskih reči prateći specifična pravila promena.

**Cilj:** Popunite korisničke priče ispod i prođite sve testove kako biste završili ovaj laboratorijski rad.

**Priče korisnika:**

1. Trebalo bi da kreirate funkciju `translatePigLatin` koja prima niz znakova kao argument.
1. Ako argument niz znakova počinje sa konsonantom, vaša funkcija bi trebalo da uzme prvu konsonantu ili grupu konsonanti, premesti kraj reči, doda `ay` i vrati rezultat.
1. Ako argument niz znakova počinje sa vokalom, vaša funkcija bi trebalo da doda `way` na kraju i vrati rezultat.
1. Vaša funkcija bi trebalo da obrađuje argumente niz znakova gde se prva vokala nalazi u sredini reči i vrati odgovarajuće modifikovan niz znakova.
1. Ako argument niz znakova nema vokala, vaša funkcija bi trebalo da doda `ay` na kraju i vrati rezultat.

Napomena: U kontekstu ove laboratorije, glaske su `a`, `e`, `i`, `o`, i `u`. Slovo `y` nije smatrano glaskom.

```js
function translatePigLatin(str) {
  if (isVowel(str.charAt(0))) return str + "way";
  let front = [];
  str = str.split('');
  while (str.length && !isVowel(str[0])) {
    front.push(str.shift());
  }
  return [].concat(str, front).join('') + 'ay';
}

function isVowel(c) {
  return ['a', 'e', 'i', 'o', 'u'].indexOf(c.toLowerCase()) !== -1;
}
```
