# --description--

Funkcija Mobiusa, poznata kao $μ(n)$, definisana je na sledeći način:

- $μ(n) = (-1)^{ω(n)}$ ako je $n$ bez kvadratnog faktora (gde je $\omega(n)$ broj različitih primordnih delitelja za $n$)
- $μ(n) = 0$ ako nije $n$ bez kvadratnog faktora.

$P(a, b)$ je broj cijelih brojeva $n$ u intervalu $[a, b]$ gdje je $\mu(n) = 1$.

$N(a, b)$ je broj celih brojeva $n$ u intervalu $[a, b]$ gde je $\mu(n) = -1$.

Na primer, $P(2, 10) = 2$ i $N(2, 10) = 4$.

Neka je $C(n)$ broj parova celih brojeva $(a, b)$ sa svojstvima:

- $1 ≤ a ≤ b ≤ n$,
- $99 \times N(a, b) ≤ 100 \times P(a, b)$, na
- $99 \times P(a, b) ≤ 100 \times N(a, b)$.

Na primer, $C(10) = 13$, $C(500) = 16\,676$ i $C(10\,000) = 20\,155\,319$.

Pronađi $C(20\,000\,000)$.
