# --description--

Nakon deklarisanja `doctype`-a, morate definisati element ``<html>``. Ovo se naziva korenski element dokumenta, što znači da će svaki drugi element u dokumentu biti njegov potomak.

Ovo postaje sve važnije kasnije kada počnete da učite o upravljanju HTML korišćenjem JavaScript. Trenutno, samo znajte da funkcija `html` mora biti uključena u svaki dokument HTML.

U fajlu `index.html`, dodajmo komponentu `<html>` pisanjem njegovih tagova za otvaranje i zatvaranje, kao što sledi:

```html
<!DOCTYPE html>
<html lang="en">
</html>
```

## Šta je atribut lang?
`lang` Onajmišlja jezik sadržaja teksta u tom elementu. Ovaj atribut se koristi prvenstveno za poboljšanje pristupačnosti web stranice. Omogućava pomoćnim tehnologijama, kao što su čitači ekrana (screen readers), da se prilagode jeziku i pruže tačno izgovaranje.
