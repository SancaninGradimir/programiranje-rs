# --description--

Za pozitivan ceo broj $n$ i poslednje cifre $d$, definišemo $F(n, d)$ kao broj delitelja $n$-a čije zadnje cifre su jednake $d$.

Na primer, $F(84, 4) = 3$. Među delilateljima 84 (1, 2, 3, 4, 6, 7, 12, 14, 21, 28, 42, 84), tri od njih (4, 14, 84) imaju broj koji se završava na 4.

Možemo takođe da potvrdimo da $F(12!, 12) = 11$ na $F(50!, 123) = 17\\,888$.

Pronađi $F({10}^6!, 65\\,432) \text{ modulo } ({10}^{16} + 61)$.
