# --description--

JSON Šta je to i kako se dobija vrednost korišćenjem notacije zagrada i tačaka?

JSON je skraćenica za JavaScript Object Notacija. To je lagani, tekstualni format podataka koji se često koristi za razmenu podataka između servera i mrežnih aplikacija.

Jedan od razloga zašto JSON je popularan u web razvoju jeste zato što je lako mašinama da ga čitaju, a takođe i ljudima da ga razumeju.

Pošto JSON ne zavisi od jezika, možete lako poslati podatke iz JSON iz Java aplikacije u Python aplikaciju, ili iz JavaScript aplikacije u C# aplikaciju.

JSON podržava mnogo tipova podataka, uključujući objekte, nizove podataka, stringove karaktera, vrednosti istina ili lažno, null, i brojeve.

Ovo je primer JSON:

```js
{
  "name": "Alice",
  "age": 30,
  "isStudent": false,
  "list of courses": ["Mathematics", "Physics", "Computer Science"]
}
```

Kao što vidite, JSON koristi parove ključ-vrednost za skladištenje informacija i svaki par je razdvojen zarezom. Svaki ključ mora biti okružen dvostrukim navodnicima, inače ćete dobiti grešku.

Da biste dobili podatke od objekta JSON, možete koristiti notaciju tačke ili notaciju zagrade. U ovom primeru, koristimo notaciju tačke da bismo dobili `age` od objekta JSON:

```js
import data from "./example.json" with { type: "json" };

console.log(data.age);
```

Ovaj specifični primer koristi ono što je poznato kao deklaracija `import`, koja dovodi nešto od JSON unutar ovog fajla kako bismo imali pristup tome. Vi ćete saznati više o deklaraciji `import` u sledećoj lekciji.

Takođe možete koristiti notaciju sa zagradama da biste dobili informacije iz stavki JSON. Evo primera kako dobiti niz za `list of courses`:

```js
import data from "./example.json" with { type: "json" };

console.log(data["list of courses"]);
```

Korišćenje crtica je važno posebno ovde jer ključ ima mnogo reči razdvojenih razmakom. Ako bismo pokušali da koristimo tačke, izazvalo bi grešku.

Ukratko, JSON je fleksibilna struktura koja može da skladišti različite tipove podataka, uključujući nizove podataka i organizovane objekte. Korišćenjem notacije tačke ili notacije zagrade/kvastice, možete lako pristupiti vrednostima sačuvanim unutar objekta JSON.
