# --description--

Generišite niz brojeva koristeći generator nasumičnih brojeva „Blum Blum Shub“:

$$
s_0 = 14025256 \\\\
s_{n + 1} = {s_n}^2 \\; mod \\; 20\\,300\\,713
$$

Spojite ove brojeve $s_0s_1s_2\ldots$ da biste stvorili sekvencu karaktera $w$ sa beskonačnom dužinom. Zatim, $w = 14025256741014958470038053646\ldots$

Za pozitivan ceo broj $k$, ako ne postoji podstring $w$-a čiji je zbir cifara jednak $k$, onda se $p(k)$ definiše kao nula. Ako postoji barem jedan podstring $w$-a čiji je zbir cifara jednak $k$, definišemo $p(k) = z$, gde je $z$ početna pozicija prvog takvog podstringa.

Na primer:

Članovi 1, 14, 1402, … čiji je zbir cifara jednak 1, 5, 7, … počinju na poziciji 1, stoga $p(1) = p(5) = p(7) = \ldots = 1$.

Elementi 4, 402, 4025, … čiji je zbir cifara jednak 4, 6, 11, … počinju u poziciji 2, pa stoga $p(4) = p(6) = p(11) = \ldots = 2$.

Brojevi 02, 0252, … čiji je zbir cifara jednak 2, 9, … počinju u poziciji 3, stoga $p(2) = p(9) = \ldots = 3$.

Zapamtite da mali deo 025 koji se nalazi na poziciji 3 ima ukupan broj cifara jednak 7, ali je postojio prethodni mali deo (koji počinje na poziciji 1) sa ukupnim brojem cifara jednak 7, pa je $p(7) = 1$, a ne 3.

Možemo potvrditi da, za $0 < k ≤ {10}^3$, $\sum p(k) = 4742$.

Izračunaj $\sum p(k)$, kwa $0 < k ≤ 2 \times {10}^{15}$.
