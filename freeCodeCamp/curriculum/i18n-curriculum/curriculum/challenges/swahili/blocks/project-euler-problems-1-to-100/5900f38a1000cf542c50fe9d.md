# --description--

Iznenadno, postoji samo tri broja koja mogu biti napisana kao zbir četvrtih stepeni njihovih cifara:

<div style='margin-left: 4em;'>
  1634 = 1<sup>4</sup> + 6<sup>4</sup> + 3<sup>4</sup> + 4<sup>4</sup><br>
  8208 = 8<sup>4</sup> + 2<sup>4</sup> + 0<sup>4</sup> + 8<sup>4</sup><br>
  9474 = 9<sup>4</sup> + 4<sup>4</sup> + 7<sup>4</sup> + 4<sup>4</sup><br>
</div>

Pošto je 1 = 1<sup>4</sup> nije ukupan iznos, pa nije uključen.

Zbir ovih brojeva je 1634 + 8208 + 9474 = 19316.

Pronađite zbir svih brojeva koji se mogu zapisati kao `n` stepeni njihovih cifara.
