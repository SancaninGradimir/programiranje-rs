# --description--

`Loop` je niz instrukcija koji se ponavlja dok se ne ispuni određeni uslov. `Loops` se koriste za efikasno izvršavanje istog koda više puta. Uobičajeni tipovi `loops` su `for` i `while` petlje.

Kwa mfano, `for loop`: `for (int i = 0; i < 5; i++) { /* code to be executed */ }`

U tom primeru, kod koji se nalazi unutar zagrada ``{ }`` se izvršava ponovo i ponovo dok uslov više nije tačan.
