# --explanation--

`Fail` znači da nešto ne funkcioniše kako bi trebalo. Odnosi se na problem koji utiče na normalan rad. Na primer:

- `The server is failing because of high traffic.` – Server otkazuje zbog velikog broja korisnika u isto vreme.

- `The app failed during the beta testing phase.` - Programu haikufanya kazi vizuri wakati ilipojaribiwa na watumizi.
