# --description--

Programeri često koriste `once` da prikažu prilikom izvršavanja vežbi unutar svog radnog procesa. To znači `as soon as` ili `when`.

Na primer, `Once you receive the client's feedback, update the project documentation.` Ovo znači da ažuriranje dokumenata treba da počne odmah nakon primanja povratnih informacija, prikazujući niz koraka u životnom ciklusu projekta.
