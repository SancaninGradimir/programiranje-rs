# --description--

Sada ste spremni da napravite svoje prve korake u svetu Node.js tako što ćete ga instalirati na svoj računar. Međutim, prvo, pre nego što instalirate Node.js, preporučujemo da instalirate alat nazvan NVM (Node Version Manager). Kao što ime kaže, NVM je alat za upravljanje različitim verzijama Node.js na jednom računaru.

Instaliranje ovog alata od samog početka pomoći će vam u budućnosti jer možda ćete morati da prebacujete između različitih verzija Node.js za različite projekte. Sa NVM-om, to možete učiniti veoma lako.

Istražimo kako možete instalirati NVM na macOS/Linux i Windows, i kako ga koristiti za instaliranje Node.js.

## Kako instalirati NVM na macOS/Linux

Na macOS-u i Linuxu, možete instalirati NVM direktno putem shell skripte.

Ali za macOS, zapamtite da zvanična NVM dokumentacija navodi da „morate ručno instalirati alate komandne linije Xcode-a pre pokretanja instalacionog skripta, inače će propasti.”

Na macOS-u, možete instalirati alate linije komandi Xcode-a pokretanjem ove komande u vašem terminalu:

```bash
xcode-select --install
```

Nakon što se instalacija završi, možete proveriti broj verzije i da potvrdite instalaciju ovom komandom:

```bash
xcode-select --version
```

Zatim je vreme da instaliraš NVM. Možeš preuzeti i pokrenuti NVM instalacionu skriptu izvršavanjem jedne od ovih komandi:

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash
```

```bash
wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash
```

Da biste preuzeli najnoviju verziju NVM-a u trenutku kada čitate ovaj članak, pogledajte zvaničnu NVM dokumentaciju: `https://github.com/nvm-sh/nvm?tab=readme-ov-file#install--update-script`. Tamo ćete pronaći ove komande sa brojem najnovije verzije.

Možda ćete morati da ponovo pokrenete vaš terminal nakon što se proces završi, kako biste učitali nove komande i nove konfiguracije.

Zatim, pokrenite ovu komandu da biste potvrdili da je NVM pravilno instaliran i proverili vašu trenutnu verziju:

```bash
nvm -v
```

Trebalo bi da vidiš svoju verziju NVM u rezultatima.

## Kako instalirati Node.js korišćenjem NVM na macOS/Linux

Sada što ste instalirali NVM na macOS/Linux, možete instalirati Node.js.

Da biste instalirali najnoviju verziju Long-Term Support (LTS) za Node.js, pokrenite ovu komandu:

```bash
nvm install --lts
```

Pogledajte zastavu `--lts`. Ova zastava pokazuje da želite instalirati trenutnu verziju Long-Term Support (LTS) za Node.js. Ova verzija se obično preporučuje jer daje prioritet stabilnosti, pouzdanosti i sigurnosti više nego novim ili eksperimentalnim funkcijama. Temeljno je testirana i spremna za produkciju. Takođe je osigurano da ima duge periode podrške pažljivom popravljanju grešaka i sigurnosnih nadogradnji.

Osim toga, možete instalirati specifičnu verziju od Node.js navodeći broj verzije u komandi:

```bash
nvm install <version>
```

Na primer, ovako možete instalirati Node.js v20:

```bash
nvm install 20
```

Nakon instaliranja verzije Node.js, možete preći na tu verziju ovom komandom:

```bash
nvm use <version>
```

Na primer, ovako menjaš/menjate na verziju 20:

```bash
nvm use 20
```

Konačno, možete proveriti trenutnu verziju Node.js ovom komandom:

```bash
node -v
```

## Kako instalirati NVM na Windows

Sada ćemo istražiti kako možete instalirati i koristiti NVM na Windows-u.

Zvanične verzije NVM podržavaju samo Windows u specifičnim slučajevima. Ako ste instalirali Windows Subsystem for Linux (WSL) na svom uređaju, možete pratiti vodič za macOS/Linux gore kako biste instalirali i koristili NVM.

Međutim, za sam Windows, postoji nekoliko alternativa. Moja je `nvm-windows`, alat za upravljanje verzijama Node.js na Windows-u. Ovo je potpuno drugačiji projekat od zvaničnog NVM projekta. Ali svrha ovih alata je ista — pomoći vam da radite sa mnogo verzija Node.js na jednom uređaju.

Pre instaliranja `nvm-windows`, trebalo bi da znate da zvanični repozitorijum preporučuje uklanjanje bilo koje prethodne instalacije Node-a pre instaliranja NVM-a za Windows. Ako je ovo prvi put da instalirate Node.js, biće u redu. Ako već imate druge verzije Node.js instalirane, savetuje se da ih prvo uklonite.

Za instaliranje `nvm-windows`:

- Idite na sekciju Releases zvaničnog repozitorijuma: `https://github.com/coreybutler/nvm-windows/releases`.
- Preuzmite najnoviji instalater.
- Pratite uputstva.

Nakon završetka instalacije, možda ćete morati da ponovo pokrenete svoj terminal ako komanda `nvm` nije odmah dostupna.

Možete proveriti da li je instalacija uspela i videti vašu trenutnu verziju od `nvm-windows` izvršavanjem ove komande:

```powershell
nvm -v
```

## Kako instalirati Node.js koristeći NVM za Windows

Sada ste postavili `nvm-windows` na Windows-u, spremni da instalirate Node.js.

Da biste instalirali najnoviju verziju Long-Term Support (LTS) za Node.js, pokrenite ovu komandu:

```powershell
nvm install lts
```

Verzija za Long-Term Support (LTS) se obično preporučuje, ali možete instalirati različitu verziju navodeći broj verzije u komandi:

```powershell
nvm install <version>
```

Na primer:

```powershell
nvm install 20
```

Kada se instalacija završi, možete preći na tu verziju Node.js ovom komandom:

```powershell
nvm use <version>
```

Na primer:

```powershell
nvm use 20
```

Takođe možete proveriti vaš trenutni deo/verziju `Node.js` ovim komandom:

```bash
node -v
```

## Osnovne komande

Sada kada znaš kako da instaliraš NVM i Node.js, razmotrićemo neke osnovne komande koje će ti pomoći kada koristiš Node.js sa NVM.

Takođe, iako će sve ove komande raditi za NVM instaliran na macOS/Linux, uz WSL, mogu da rade drugačije ili uopšte ne rade za `nvm-windows`. Ostavićemo alternativne komande za `nvm-windows` gde je to moguće.

### Kako promeniti politiku izvršavanja PowerShell-a

Ako koristite Windows i pratite sa PowerShell-om i `nvm-windows`, možda ćete morati da promenite politiku izvršavanja za PowerShell. Podrazumevana politika izvršavanja za PowerShell je `Restricted`, što je sigurnosna mera koja sprečava pokretanje skripti.

Prvo, pokrenite Powershell kao administrator desnim klikom na Powershell, zatim kliknite na “Run as Administrator“.

Zatim, da biste videli trenutnu politiku izvršavanja za trenutnu sesiju PowerShell-a, pokrenite sledeću komandu:

```powershell
Get-ExecutionPolicy
```

Konačno, postavljanje politike izvršavanja PowerShell-a na `RemoteSigned`, koja omogućava lokalnim skriptama i `npm` da rade, koristite sledevu komandu:

```powershell
Set-ExecutionPolicy RemoteSigned
```

Nakon izvršavanja komande gore, možete zatvoriti administratorski prozor PowerShell-a i otvoriti ne-administratorski prozor PowerShell-a.

### Navedi sve verzije

Da biste naveli sve verzije Node.js koje su instalirane i proverili trenutno korišćenu verziju, pokrenite ovu komandu:

```bash
nvm ls
```

### Postavljanje podrazumevane verzije Node.js (NVM samo za macOS/Linux)

Ova komanda postavlja podrazumevanu verziju Node.js koja će se koristiti za nove terminal sesije:

```bash
nvm alias default <version>
```

Na primer:

```bash
nvm alias default 22.20.0
```

Takođe možete kreirati prilagođene nadimke da dodelite specifično ime verziji:

```bash
nvm alias <new_name> <version>
```

Na primer:

```bash
nvm alias new-api 18.20.8
```

Nakon postavljanja nadimka, možete koristiti ovo novo ime da pređete na tu verziju:

```bash
nvm use <new_name>
```

Na primer:

```bash
nvm use new-api
```

### Ukloni verziju od Node.js

Da biste uklonili verziju Node.js, pokrenite ovu komandu:

```bash
nvm uninstall <version>
```

Na primer:

```bash
nvm uninstall 20.19.5
```

Za NVM na macOS/Linux, možete ukloniti verziju Node.js izvršavanjem komande `uninstall` i glavnu verziju poput `22` ili `20`. Primer:

```bash
nvm uninstall 20
```

Ako imate više verzija glavnog izdanja instalirano kao `20.14.0` i `20.12.2`, najnovija verzija će biti uklonjena prva. U ovom slučaju, izvršavanje `nvm uninstall 20` će prvo ukloniti `20.14.0`.

### Pogledajte vašu verziju od Node.js

Da biste proverili vašu trenutnu verziju Node.js, možete koristiti sledeće:

```bash
node -v
```

### Proverite svoju verziju npm-a

`npm` je menadžer paketa za Node.js. Možete ga koristiti za instalaciju, objavljivanje i upravljanje softverskim paketima. Instalira se automatski kada instalirate Node.js.

Da biste videli vašu trenutnu verziju `npm`, pokrenite ovu komandu:

```bash
npm -v
```

### Pokreni fajl JavaScript

Ova komanda pokreće fajl JavaScript, detektovan u trenutnom radnom okruženju/runtime-u Node.js:

```bash
node <file.js>
```

Na primer:

```bash
node app.js
```

### Kreiraj projekat za Node.js

Ovaj deo stvara novi projekat za Node.js kreiranjem fajla `package.json` koji prati detalje i njegove zavisnosti.

```bash
npm init
```

Kada pokrenete komandu `npm init`, vodiće vas kroz čarobnjak interfejsa komandne linije za inicijalizaciju fajla `package.json`. Pitaće vas pitanja o imenu projekta, opisu, verziji itd. Ako želite početi sa podrazumevanim fajlom `package.json`, možete koristiti komandu `npm init -y` ili `npm init --yes`.

Instaliraj paket

Da biste instalirali određeni paket, pokrenite:

```bash
npm install <package>
```

Na primer:

```bash
npm install express
```

Instaliraj zavisnost

Da biste instalirali sve zavisnosti navedene u fajlu `package.json` projekta, pokrenite:

```bash
npm install
```
