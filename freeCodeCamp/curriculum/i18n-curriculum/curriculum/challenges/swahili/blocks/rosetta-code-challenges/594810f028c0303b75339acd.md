# --description--

Ove definišu tri tipa pozitivnih celih brojeva u zavisnosti od njihovih prostih faktora.

Neka je $P(n)$ zbir svih pravilnih delitelja od `n`, gde su svi pravilni delitelji pozitivni ceo brojevi `n`, osim samog `n`.

Kama `P(n) < n` basi `n` huainishwa kama `deficient`

Ako `P(n) === n` onda `n` klasifikuje se kao `perfect`

Ako `P(n) > n`, onda `n` se klasifikuje kao `abundant`

**Primer**: `6` ima prave delioce `1`, `2` i `3`. Pošto je `1 + 2 + 3 = 6`, broj `6` se klasifikuje kao savršen broj.

# --instructions--

Implementiraj funkciju koja broji koliko je pozitivnih celih broja od `1` do `num` (uključujući) nalazi se u svakoj od tri klase. Vrati rezultate kao niz podataka sa formatom `[deficient, perfect, abundant]`.

```js
function getDPA(num) {
  const dpa = [1, 0, 0];
  for (let n = 2; n <= num; n += 1) {
    let ds = 1;
    const e = Math.sqrt(n);
    for (let d = 2; d < e; d += 1) {
      if (n % d === 0) {
        ds += d + (n / d);
      }
    }
    if (n % e === 0) {
      ds += e;
    }
    dpa[ds < n ? 0 : ds === n ? 1 : 2] += 1;
  }
  return dpa;
}
```
