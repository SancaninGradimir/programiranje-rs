# --explanation--

Modalni glagol je uvek praćen osnovnim glagolom. Modalni glagol ne uzima `to`, `-ing`, niti forme prošlog vremena. Evo primera pogrešnih glagolskih oblika:

- `Must to follow` nije tačno zato što `must` koristi `to`.

- `Must following` je validan jer `must` sledi tip `-ing`.

- `Must followed` nije tačno jer `followed` je u prošlosti.
