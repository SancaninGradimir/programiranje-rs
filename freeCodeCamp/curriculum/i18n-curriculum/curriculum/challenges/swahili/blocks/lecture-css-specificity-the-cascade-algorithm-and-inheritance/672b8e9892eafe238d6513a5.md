# --interactive--

Selektori klasa su važan deo CSS, što omogućava programerima da ciljaju više elemenata sa atributom istog elementa klase i koriste odgovarajuće stilove.

Ovo ih čini veoma upotrebljivim i efikasnim korišćenjem stilova širom cele mreže.

Selektori klase označavaju se tačkom (`.`) praćenom nazivom klase. Mogu se koristiti nad bilo kojim elementom u HTML dokumentu.

Ovo je primer korišćenja klase objekata:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<p class="highlight">Example paragraph</p>
```

```css
.highlight {
  color: green;
}
```

:::

U ovom primeru svaki element koji ima klasu `highlight` imaće zelenu boju teksta.

Selektori klase imaju veću specifičnost od selektora tipa, ali manju od ID selektora i inline stilova.

Vrednost specifičnosti selektora klase je `(0, 0, 1, 0)`. To znači da mogu nadjačati selektore tipa, ali ih mogu nadjačati ID selektori i inline stilovi.

Selektori klase mogu se kombinovati sa drugim selektorima kako bi se napravila specifičnija pravila.

Evo primera kombinovanja selektora tipa `p` i selektora klase:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<p class="bold-text">Example paragraph</p>
<p class="bold-text">Example paragraph</p>
<p>Another paragraph</p>
<p>Yet another paragraph</p>
```

```css
p.bold-text {
  font-weight: bold;
}
```

:::

Ovo pravilo se primenjuje samo na elemente `p` koji imaju klasu `bold-text`, čineći njihov tekst podebljanim.

  <p class="highlight">This text</p>
