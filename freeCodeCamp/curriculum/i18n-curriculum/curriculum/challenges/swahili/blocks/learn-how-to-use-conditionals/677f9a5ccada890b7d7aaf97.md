# --explanation--

`Zero Conditional` Koristi se za stvari koje su uvek tačne kada je uslov ispavljen.

Prati strukturu `if` + `Present Simple` (za status) i `Present Simple` (za rezultate). Primer:

- `If you press the power button, the computer starts.` - Pritiskanje dugmeta za napajanje uvek uključuje računar.

- `If the internet is down, the website doesn't load.` - Web stranica se neće otvoriti ako nema internet konekcije.

U ovom razgovoru, `if the BIOS is corrupted` je stanje, a `it prevents the computer from starting` su rezultati. Ovo je opšta istina o tome kako oštećeni BIOS utiče na računar.
