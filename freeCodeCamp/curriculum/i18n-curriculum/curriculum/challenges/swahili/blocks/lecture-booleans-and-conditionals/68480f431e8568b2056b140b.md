# --description--

U prethodnoj lekciji, naučili ste kako da koristite operatore poređenja i uslovne izjave za kontrolisanje toka vaših programa.

Iako su vrlo snažne, često naiđete na situaciju u kojoj je potrebno uporediti mnogo vrednosti istovremeno. Ovo može dovesti do ugnježdjenih uslovnih rečenica, na primer:
**Napomena:** Prvih 13 koraka mora biti završeno unutar datoteke `index.html`.
```python
is_citizen = True
age = 25

if is_citizen:
    if age >= 18:
        print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

Primer iznad će prvo proveriti da li je `is_citizen` jednako `True`. Ako je tačno, onda će ići na ugnježdenu rečenicu `if` i proveriti da li je `age` veće ili jednako od `18`. Pošto je `age` veće ili jednako od `18`, poruka koja će biti ispisana na terminalu biće `You are eligible to vote`. Ako bi `is_citizen` bilo `False`, onda bi poruka ispisana na terminalu bila `You are not eligible to vote`.

Ako radite sa složenijim uslovnim rečenicama, onda možete koristiti operatore Python `and`, `or`, i `not`.

Ali pre nego što uđemo u te opere/performanse, hajde da istražimo koje su stvarne vrednosti i koje nisu.

U Python, svaka vrednost ima inherentnu istinitu ili lažnu vrednost, ili ugrađeno stanje da li treba tretirati kao `True` ili `False` u logičkom kontekstu. Mnoge vrednosti se smatraju **istinitim vrednostima**, to jest, smatraju se kao `True` u logičkom kontekstu. Ostale su **lažne vrednosti**, što znači da se smatraju kao `False`.

Evo nekih lažnih vrednosti:

- `None`
- `False`
- Puni broj `0`
- Sekvencijalni broj `0.0`
- Niz samo slova `""`

Druge vrednosti, kao što su brojevi koji nisu nula i ne-prazan niz slova, su stvarni vrednosti.

Ako želite da proverite da li je vrednost tačna ili netačna, možete koristiti ugrađenu funkciju `bool()`. Ona izvršava eksplicitnu konverziju vrednosti u tačno ili netačno i vraća `True` za tačne vrednosti i `False` za netačne. Evo nekoliko primera:

```py
print(bool(False)) # False
print(bool(0))  # False
print(bool('')) # False

print(bool(True)) # True
print(bool(1)) # True
print(bool('Hello')) # True
```

Sada kada razumete realne i imaginarne vrednosti, možemo pogledati operatore realnih ili imaginarnih vrednosti, koji su takođe poznati kao logički operateri. Ovo su specijalni operateri koji vam omogućavaju da kombinujete više matematičkih izraza kako biste stvorili složeniju logiku za donošenje odluka u vašem kodu.

Postoji tri opere za vrednost tačno ili netačno u Python: `and`, `or`, na `not`.

Prvo hajde da pregledamo operetu `and`.

Opereta `and` uzima dva parametra i vraća prvi parametar ako je vrednost lažna, inače vraća drugi parametar. Svi parametri moraju biti istinite vrednosti da bi izraz dao istinitu vrednost.

Evo primera:

```python
is_citizen = True
age = 25

print(is_citizen and age) # 25
```

U gornjem primeru, broj 25 se ispisuje na terminal jer će operator ``and`` proceniti drugi parametar ako je prvi parametar ``True``. Operator ``and`` poznat je kao operator za prekid kruga. Prekida krug znači da `Python` proverava vrednosti od leve ka desnoj i zaustavlja se čim detektuje konačan rezultat.

Često ćete koristiti `and` unutar rečenica od `if` da biste proverili da li su zadovoljeni mnogi uslovi. Evo kako možete poboljšati prethodni primer koristeći operator `and` umesto strukturiranih rečenica iz `if`:

```python
is_citizen = True
age = 25

if is_citizen and age >= 18:
    print('You are eligible to vote') # You are eligible to vote
else:
    print('You are not eligible to vote')
```

U gornjem primeru, `is_citizen` je `True`, i na `age >= 18` procenjeno je da je `True`. Pošto su oba parametra operacije `and` tačne vrednosti, uslov `is_citizen and age >= 18` je procenjen da je `True`, i poziv `print` u sekciji `if` se izvršava.

Sada hajde da izučimo operator `or`. Ovaj operator vraća prvi argument ako je istinitostna vrednost, inače vraća drugi argument. Izraz `or` vraća istinitostnu vrednost ako je barem jedan argument istinit. Operator `or` je poznat i kao operator kratkog kruga. Evo primera:

```python
age = 19
is_employed = False

print(age or is_employed) # 19
```

Sledeći kod će ispisati broj 19 jer je prvi parametar `age` vrednost istina.

Ako trebate proveriti da li je jedan ili više izraza `True`, možete koristiti operator `or` u uslovu kao što je ovaj:

```python
age = 19
is_student = True

if age < 18 or is_student:
    print('You are eligible for a student discount') # You are eligible for a student discount
else:
    print('You are not eligible for a student discount')
```

U ovom slučaju, `age < 18` je `False`, ali `is_student` je `True`. Pošto je bar jedan uslov tačan, celokupna izjava od `or` se procenjuje kao `True`, i u sekciji `if` se štampa poruka o smanjenju.

Poslednja operacija koju ćemo istražiti je operacija `not`, koja uzima jedan parametar i menja njegovu istinitu ili lažnu vrednost. Menja istinite vrednosti u `False` i lažne vrednosti u `True`. Za razliku od operacija koje smo videli ranije, `not` uvek vraća `True` ili `False`.

Evo nekoliko primera:

```python
print(not '') # True, because empty string is falsy
print(not 'Hello') # False, because non-empty string is truthy
print(not 0) # True, because 0 is falsy
print(not 1) # False, because 1 is truthy
print(not False) # True, because False is falsy
print(not True) # False, because True is truthy
```

Uobičajeno je koristiti operator ``not`` u uslovima da se proveri da li nešto nije ``True`` ili ``False``, kao što je:

```python
is_admin = False

if not is_admin:
    print('Access denied for non-administrators.') # Access denied for non-administrators.
else:
    print('Welcome, Administrator!')
```

Pošto je `is_admin` `False`, onda `not is_admin` kaže da nije `False` koja je `True`. Stoga poruka `Access denied for non-administrators.` će biti odštampana.

Sada kada razumete istinite i nistinite vrednosti, operatore za `and`, `or`, na `not`, i kako radi proces prekida kruga, možete napisati uslovnu logiku koja je fleksibilnija i lakša za čitanje.
