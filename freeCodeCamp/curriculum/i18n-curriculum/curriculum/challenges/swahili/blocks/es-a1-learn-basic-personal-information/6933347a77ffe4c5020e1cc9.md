# --description--

U ovoj vežbi, čutićete reakciju Anglele kada sazna o Estebanovom poreklu. Pažljivo slušajte ovu izjavu:

- `Ah` je zvuk obične reakcije. Pokazuje razumevanje ili prepoznavanje.

- `de` koristi se za razgovor o poreklu.

- `Muy bien` je dobar odgovor. Pokazuje dozvolu ili zadovoljstvo.
