# --description--

U poslednjoj lekciji, naučili ste da server obrađuje zahtev za HTTP i vraća odgovor. Taj odgovor uključuje različite atribute koje vidite na mreži.

Svojstva vraćena odgovorom `HTTP` uključuju tekstualna svojstva, medijska svojstva, fontove i slično. Svako od ovih svojstava je praćeno zaglavljem **Content-Type**, koje pregledaču govori kako da pravilno rukuje i prikaže to svojstvo.

Tekstualni sadržaji su bili važan deo mreže od samog početka. Zapravo, mreža je počela kao čist tekst. Danas, tekstualni sadržaji uključuju:

* **HTML (Hypertext Markup Language)**: Osnovna struktura svake veb stranice. Bez HTML, veb stranica ne može da postoji.
    
* **CSS (Cascading Stylesheet)**: List koja kontroliše izgled veb stranice. Sa CSS, postavljate stilove kao što su boje, fontovi i raspored za HTML.
    
* **JavaScript**: JavaScript dodaje interaktivnost i dinamički sadržaj HTML-u i CSS-u.
    
* **JSON i XML**: Ove dve strukture omogućavaju prenos strukturiranih podataka između servera i pregledača.
    
Medijski resursi uključuju slike, zvuk i video. Oni doprinose da veb bude življiv slikama i zvukovima.

Mnogi veb-sajtovi se oslanjaju na slike za predstavljanje vizuelnih informacija. Slike dolaze u različitim formatima, kao što su `JPG`, `JPEG`, `PNG`, `WEBP`, i `GIF`.

Websajtovi poput YouTube-a i Vimeo koriste video sadržaj u struktura kao `MP4`, `MOV`, `AVI`, `WebM`, i slično.

Zvuk je važan deo mnogih platformi jer omogućava korisnicima da slušaju muziku, podkaste i drugi audio sadržaj. Uobičajeni audio formati su `MP3`, `AAC`, Ogg i `WAV`. Platforme kao što su Spotify i YouTube Music zavise od ovih formata kako bi pružile najbolje audio iskustvo.

Ostali resursi vraćeni u odgovoru od HTTP su dokumenti kao što su `PDF`, `DOCX`, i `PAGES`, komprimovani fajlovi kao što su `ZIP` i `RAR`, font fajlovi kao što su `WOFF`, `WOFF2`, i `TTF`, i izvršni fajlovi kao što su `EXE`.

Ovo je tabela ovih uobičajenih svojstava i njihovih tipova podataka:

| Tip fajla | Zaglavlje Content-Type |
| --- | --- |
| HTML | text/html |
| CSS | text/css |
| JavaScript | application/javascript |
| JPEG | image/jpeg |
| PNG | image/png |
| GIF | image/gif |
| SVG | image/svg+xml |
| WebP | image/webp |
| WOFF | font/woff |
| WOFF2 | font/woff2 |
| TTF | font/ttf |
| OTF | font/otf |
| MP4 | video/mp4 |
| WebM | video/webm |
| Ogg (Video) | video/ogg |
| MP3 | audio/mpeg |
| AAC | audio/aac |
| WAV | audio/wav |
| Ogg (Audio) | audio/ogg |
| JSON | application/json |
| XML | application/xml |
| PDF | application/pdf |
| ZIP | application/zip |
