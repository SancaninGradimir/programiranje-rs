# --description--

Da deklariš nepromenljivi parametar (nepromenljivo), koristi `const` umesto `let`:

```javascript
const myBirthday = '16-09-2003';
```

Parametri definisani korišćenjem `const` nazivaju se "konstante" (nepromenljive). Ne mogu dobiti vrednost ponovo. Pokušaj toga će izazvati grešku:

```javascript
const myBirthday = '16-09-2003';

myBirthday = '01-01-2001'; // error, can't reassign the constant!
```

Kada je program siguran da parametar nikada neće promeniti, može ga deklarisati koristeći `const` kako bi osigurao i predstavio tu činjenicu svima.
