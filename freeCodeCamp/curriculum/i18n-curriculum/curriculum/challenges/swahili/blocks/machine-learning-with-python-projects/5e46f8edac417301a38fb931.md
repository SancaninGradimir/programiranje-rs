# --instructions--

U ovom izazovu, potrebno je kreirati model maшинskog učenja koji će klasifikovati SMS poruku u „ham“ ili „spam“. Poruka „ham“ je normalna poruka poslata od prijatelja. Poruka „spam“ je reklama ili poruka poslata od kompanije.

Morate kreirati funkciju koja se zove `predict_message`, koja prima niz znakova poruke kao argument i vraća listu. Prvi element u listi treba da bude broj između nule i jedinstvene koji ukazuje verovatnoću za "ham" (0) ili "spam" (1). Drugi element u listi treba da bude reč "ham" ili "spam", zavisno od najverovatnijeg.

Za ovaj izazov, koristićeš skup podataka <a href="http://www.dt.fee.unicamp.br/~tiago/smsspamcollection/" target="_blank" rel="noopener noreferrer nofollow">SMS Spam Collection</a>. Skup podataka je već podeljen na podatke za obuku i podatke za testiranje.
