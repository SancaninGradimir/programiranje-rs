## Osnovi mreže CSS

- **Definicija**: Mreža CSS je dvodimenzionalni sistem raspoređivanja koji se koristi za kreiranje složenih postavki na web stranicama. Gridovi imaju redove i kolone sa razmakima između njih. Da biste definisali strukturu mreže, morate postaviti svojstvo `display` na `grid`.

```css
.container {
  display: grid;
}
```

- **Jedinica za `fr` (Dio)**: Ova jedinica predstavlja deo prostora unutar mrežnog kontejnera. Možete koristiti jedinicu `fr` za kreiranje dinamičnih mreža.
- **Kreiranje praznina između mrežnih puteva**: Postoji tri načina za kreiranje praznina između puteva u mreži CSS. Možete koristiti svojstvo `column-gap` za kreiranje praznina između kolona. Možete koristiti svojstvo `row-gap` za kreiranje praznina između redova. Ili možete koristiti svojstvo skraćivanja `gap` za kreiranje praznina između redova i kolona.
- **`grid-template-columns`**: Ovo se koristi za postavljanje imena i veličine kolona mrežnih puteva.

```css
.container {
  display: grid;
  width: 100%;
  grid-template-columns: 30px 1fr;
}
```

- **`grid-template-rows`**: Koristi se za postavljanje imena linija i dimenzija redova mrežnih puteva.
- **`grid-auto-flow`**: Ovo određuje kako se automatski postavljeni elementi poravnaju u mreži.

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-flow: column;
}
```

- **`grid-auto-columns`**: Ovo se koristi za postavljanje dimenzija kolona koje su kreirane indirektnom metodom.

```css
.container {
  display: grid;
  width: 100%;
  grid-auto-columns: auto;
}
```

- **`place-items`**: Ovo se koristi za poravnavanje elemenata u blok pravcu i unutar linije.
- **`align-items`**: Ovo se koristi za postavljanje poravnanja elemenata u mrežni kontejner (grid container).
- **Funkcija za `repeat()`**: Ova funkcija se koristi za ponavljanje sekcije u listi putanja. Umesto pisanja `grid-template-columns: 1fr 1fr 1fr;`, možete koristiti funkciju `repeat()` umesto toga.

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}
```

- **Eksplicitni Grid**: Možete definisati broj redova ili kolona koristeći svojstva `grid-template-columns` ili `grid-template-rows`.
- **Implicitni Grid**: Kada se elementi postave van grida, redovi i kolone automatski se kreiraju za te eksterne elemente.
- **Funkcija za `minmax()`**: Ova funkcija se koristi za postavljanje minimalnih i maksimalnih vrednosti za kolonu.

```css
.container {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-auto-rows: minmax(150px, auto);
}
```

- **Osnovno pozicioniranje linija**: Sve ćelije imaju linije. Da biste odredili gde element počinje na liniji, možete koristiti svojstva `grid-column-start` i `grid-row-start`. Da biste odredili gde element završava na liniji, možete koristiti svojstva `grid-column-end` i `grid-row-end`. Takođe možete izabrati da koristite skraćena svojstva `grid-column` ili `grid-row` umesto toga.

Ovde je primer korišćenja atributa `grid-column` da se element pojavi na svim kolonama.

```css
.element {
  grid-column: 1 / -1;
}
```

- **`grid-template-areas`**: Ovaj atribut se koristi za dodeljivanje imena elementima koje postavljate u mrežu (grid).

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">

<div class="container">
  <div class="header">Header</div>
  <div class="sidebar">Sidebar</div>
  <div class="main">Main Content</div>
  <div class="footer">Footer</div>
</div>

```

```css
.container {
  display: grid;
  grid-template-columns: 200px 1fr; 
  grid-template-rows: auto 1fr auto; 
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer"; 
  gap: 20px; 
}

.header {
  grid-area: header; 
  background-color: #4CAF50;
  padding: 10px;
  color: white;
}

.sidebar {
  grid-area: sidebar;
  background-color: #f4f4f4;
  padding: 10px;
}

.main {
  grid-area: main; 
  background-color: #e0e0e0;
  padding: 10px;
}

.footer {
  grid-area: footer; 
  background-color: #4CAF50;
  padding: 10px;
  color: white;
}
```

## Ispravljanje grešaka sa CSS

- **Alati za developera (DevTools)**: Alati za developera vam omogućavaju da inspekujete i menjate CSS u realnom vremenu. Sekcija Stila prikazuje sve pravila CSS primenjena na izabrani element, uključujući nasleđene stilove. Možete uključiti ili isključiti pojedinačne atribute, editovati vrednosti i čak dodati nova pravila direktno u pretraživaču. Ova trenutna povratna informacija je veoma korisna za testiranje različitih stilova bez menjanja izvornog koda.
- **Validatori za CSS**: Validatori se koriste za proveru vašeg CSS prema zvaničnim specifikacijama i izveštavanje o bilo kakvim greškama ili upozorenjima. Popularni validator koji možete koristiti je W3C CSS Validator.
- **Rešavanje problema sa responsivnim dizajnom**: Alati za developera imaju opciju da simuliraju kako vaša veb stranica izgleda na različitim veličinama i uređajima. Ovo može pomoći u identifikovanju problema sa tačkama prekida (breaking points) ili stilovima koji nisu pravilno skalirani za različite veličine vidljivog područja (viewport).
