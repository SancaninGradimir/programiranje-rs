# --interactive--

Element `script` se koristi za ubacivanje izvršnog koda. Mnogi programeri ga koriste za izvršavanje JavaScript koda. JavaScript se koristi za dodavanje interaktivnosti veb stranicama. Uobičajeni primeri upotrebe JavaScript-a uključuju interaktivne igre, galerije slika (image sliders) i dinamičke forme koje proveravaju korisnički unos u realnom vremenu.

Ovo je primer korišćenja funkcije `script` u dokumentu HTML. Izbrišite `//` koji se nalazi ispred `alert("Welcome to freeCodeCamp");` i trebalo bi da vidite upozorenje. Da biste videli reviziju, moraćete da omogućite interaktivni editor.

:::interactive_editor

```html
<body>
  <script>
    // alert("Welcome to freeCodeCamp");
  </script>
</body>
```

:::

Iako tehnički možete napisati sav JavaScript kod unutar oznaka `script`, smatra se boljom praksom da umesto toga povežete eksternu JavaScript datoteku. Evo primera korišćenja elementa `script` za povezivanje sa eksternom JavaScript datotekom:

```html
<script src="path-to-javascript-file.js"></script>
```

Atribut `src` se ovde koristi za određivanje lokacije eksterne JavaScript datoteke. `src` je skraćenica za "source" (izvor). Razlog zašto se ne preporučuje da sav JavaScript kod bude unutar HTML dokumenta jeste razdvajanje odgovornosti. To je princip dizajna prema kojem se aplikacija deli na odvojene delove, pri čemu svaki deo ima svoju ulogu. U ovom slučaju želimo da odvojimo JavaScript kod od HTML koda.
