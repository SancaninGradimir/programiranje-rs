# --interactive--

Biće vremena kada ćete morati da dodate ili uklonite čvorove sa DOM i postoje neke API mreže koje možete koristiti.

U ovoj lekciji govorićemo o metodama `appendChild()` i `removeChild()`.

Metoda `appendChild()` se koristi za dodavanje čvora na kraj liste potomaka navedenog roditeljskog čvora.

Evo osnovne sintakse za metodu `appendChild()`:

```js
parentNode.appendChild(newNode);
```

Pogledajte primer kako biste bolje razumeli način korišćenja metode `appendChild()`.

U ovom primeru imamo neuređenu listu sa identifikatorom `desserts` unutar HTML-a. Ova lista sadrži dve stavke: `Cake` i `Pie`.

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
```

Da bismo pristupili toj listi u JavaScript datoteci, možemo koristiti metodu `getElementById()`.

```js
const dessertsList = document.getElementById("desserts");
```

Imamo promenljivu koja se zove `dessertsList` i koja čuva referencu na `ul` element sa identifikatorom `desserts`.

Zatim treba da kreiramo novi element stavke liste koristeći metodu `createElement()`:

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");
```

Sledeći kod će kreirati novi element stavke liste i sačuvati ga u promenljivoj koja se zove `listItem`.

Da bismo dodali taj čvor na kraj liste dece elementa `ul`, možemo koristiti metodu `appendChild()`.

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

dessertsList.appendChild(listItem);
```

Kada pokrenete ovaj kod, videćete da je dodato novo polje liste na kraj liste potomaka svojstva za `ul`.

:::interactive_editor

```html

  <li>Cake</li>
  <li>Pie</li>
</ul>
<script src="index.js"></script>
```

```js
const dessertsList = document.getElementById("desserts");

```

:::

Problem je što novo polje elementa liste je prazno. Da biste dodali tekstualni sadržaj u novo polje elementa liste, možete koristiti svojstvo `textContent`:

:::interactive_editor

```html
<ul id="desserts">
  <li>Cake</li>
  <li>Pie</li>
</ul>
<script src="index.js"></script>
```

```js
const dessertsList = document.getElementById("desserts");
const listItem = document.createElement("li");

listItem.textContent = "Cookies";
dessertsList.appendChild(listItem);
```

:::

Sada će lista prikazati `Cake`, `Pie`, i `Cookies`.

Da biste uklonili čvor iz DOM, možete koristiti način `removeChild()`.

Ovo je osnovna sintaksa na način od `removeChild()`:

```js
parentNode.removeChild(childNode);
```

Pogledaćemo primer gde želimo da uklonimo element poslednjeg pasusa iz elementa `section`:

```html
<section id="example-section">
  <h2>Example sub heading</h2>
  <p>first paragraph</p>
  <p>second paragraph</p>
</section>
```

Možemo dobiti element za `section` unutar fajla JavaScript koristeći metodu `getElementById()`:

```js
const sectionEl = document.getElementById("example-section");
```

Kada dobijemo referencu za element `section`, možemo zatim dobiti poslednji pasus koristeći metodu `querySelector()`:

```js
const sectionEl = document.getElementById("example-section");
const lastParagraph = document.querySelector("#example-section p:last-of-type");
```

Koristimo veštačku klasu `:last-of-type` za odabir elementa poslednjeg pasusa unutar elementa `section`.

Sada kada imamo roditeljski i podređeni čvor, možemo ukloniti poslednji element pasusa iz elementa `section` pomoću metode `removeChild()`.

:::interactive_editor

```html
<section id="example-section">
  <h2>Example sub heading</h2>
  <p>first paragraph</p>
  <p>second paragraph</p>
</section>
<script src="index.js"></script>
```

```js
const sectionEl = document.getElementById("example-section");
const lastParagraph = document.querySelector("#example-section p:last-of-type");

sectionEl.removeChild(lastParagraph);
```

:::

Kada biste želeli da dodate ili uklonite čvor iz DOM-a?

Ako radite sa dinamičkim sadržajem, možda ćete morati da dodajete ili uklanjate čvorove iz DOM-a. Realni primeri uključuju korpe za kupovinu, liste zadataka i izvore vesti sa društvenih mreža.
