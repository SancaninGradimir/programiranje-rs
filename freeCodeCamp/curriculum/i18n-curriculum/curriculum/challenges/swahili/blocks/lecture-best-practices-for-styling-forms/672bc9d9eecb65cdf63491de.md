# --interactive--

Naučićemo kako da koristimo `appearance: none` za rešavanje problema sa poravnavanjem polja za pretragu, čekboksova i radio dugmadi.

Pretraživači postavljaju podrazumevani stil za mnoge komponente. U slučaju ulaznih komponenti, vaša sposobnost stilizovanja pomoću CSS može delovati ograničeno. Stoga, možda želite da koristite `appearance: none` da sakrijete podrazumevane elemente komponente i kreirate svoje umesto njih. Na primer, ovo će sakriti podrazumevane kutije selektorskog polja, omogućavajući vam da koristite prilagođene indikatore kao što su zelena kvačica i crveni X za prikaz statusa. Za polje pretrage, pretraživači koji koriste WebKit će prikazati podrazumevanu ikonu pretrage i dugme za brisanje. Sakrivanjem ovoga omogućavate kreiranje sopstvenih indikatora koji će biti vidljivi u svim pretraživačima.

Evo primera padajućeg menija za specifičan izbor:

**NAPOMENA**: Neki CSS u ovom primeru koriste funkcije koje još nisu naučene. Ovo je da vam dam ideju kako napraviti specifičnu izbirnu kutiju. Naučićete kako sve ovo funkcioniše u budućim časovima i radionicama.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css" />
<form>
  <label>
    <input class="checkbox" type="checkbox" /> Agree
  </label>
</form>
```

```css
.checkbox {
  appearance: none;
  width: 18px;
  height: 18px;
  border: 2px solid #ccc;
  border-radius: 4px;
  display: inline-block;
  position: relative;
  cursor: pointer;
  transition: all 0.25s ease;
  vertical-align: middle; 
}

.checkbox:hover {
  border-color: #888;
}

.checkbox:checked {
  background-color: #4caf50;
  border-color: #4caf50;
}

.checkbox:checked::after {
  content: "";
  position: absolute;
  left: 4px;
  top: 0px;
  width: 5px;
  height: 10px;
  border: solid white;
  border-width: 0 2px 2px 0;
  transform: rotate(45deg);
}

.checkbox:focus {
  outline: 2px solid #90caf9;
  outline-offset: 2px;
}
```

:::

WebKit je programski motor koji pomaže web pregledačima da prikazuju veb sajtove. Pregledači kao što je Safari koriste WebKit kako bi osigurali da web stranice izgledaju i funkcionišu ispravno. Ova funkcija `appearance: none` CSS daje vam potpunu kontrolu nad stilom, ali postoje i stvari koje treba uzeti u obzir. Osnovni interaktivni elementi polja za unos uključuju funkcije kao što su indikatori fokusa i greške koje morate osigurati da ne budu izgubljene.

Kreiranje konzistentnog stila na različitim platformama je dobar razlog za korišćenje ove funkcije. Takođe ga možete koristiti da osigurate da su površine za dodir na mobilnom uređaju dovoljno velike ili da boje u okviru izbora imaju dovoljan kontrast.
