# --description--

Pozitivan ceo broj sa $k$ cifara (za decimalne brojeve) naziva se jednak ako je zbir njegovih prvih cifara $⌈\frac{k}{2}<0xE2><0x8C><0x89>$ jednak zbiru njegovih poslednjih cifara $⌈\frac{k}{2}<0xE2><0x8C><0x89>$, gde se $⌈x<0xE2><0x8C><0x89>$, nazivan kao plafon (ili "ceo deo") $x$, predstavlja najmanji ceo broj $≥ x$, stoga je $⌈π<0xE2><0x8C><0x89> = 4$ i $⌈5<0xE2><0x8C><0x89> = 5$.

Na primer, sve palindrome su jednaki, kao što je 13722.

Neka je $T(n)$ suma svih brojeva manjih od $10^n$.

Dakle: $T(1) = 45$, $T(2) = 540$ i $T(5) = 334\\,795\\,890$.

Pronađi $T(47)\\pmod{3^{15}}$
