# --explanation--

Kada sastavljate pitanje koristeći modale glagole, struktura je: modalni glagol + zamjenica/subjekt + glagol u osnovnom obliku + (šta se radi)? Primjer:

- `Can we complete the project by Friday?` - Pitajte da li je moguće završiti projekat do petka.

- `Should we update the software to the latest version?` - Tražiti savet ili predlog za ispravljanje programa.

U tačnom odgovoru, `can` je postavljeno pre zamenice (`you`), a glagol (`explain`) je u svojoj osnovnoj formi.
