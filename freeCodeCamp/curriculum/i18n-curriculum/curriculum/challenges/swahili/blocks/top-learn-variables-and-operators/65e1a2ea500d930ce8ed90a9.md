# --description--

Sastajte se sa funkcijama operatera JavaScript koje dolaze izvan školskih proračuna.

Obično, operateri za dodavanje `+` sumiraju broj.

Međutim, ako se binarni operateri `+` koriste za sekvencu karaktera, spajaju tu sekvencu:

```js
let s = "my" + "string";
console.log(s); // mystring
```

Zapamtite da ako je jedan od primaoca niz slova, onda se drugi menja u niz slova takođe.

Na primer:

```js
console.log(1 + '2'); // '12'
console.log('1' + 2); // '12'
```

Obrati pažnju, nije važno da li je prvi operand string ili drugi.

Evo još komplikovanijeg primera:

```js
console.log(2 + 2 + '1') // "41" and not "221"
```

Ovde se operatori izvršavaju jedan za drugim. Prvi `+` sabira dva broja i vraća `4`, zatim sledeći `+` dodaje string `1`, pa je to isto kao `4 + '1' = '41'`.

```js
console.log('1' + 2 + 2) // "122" and not "14"
```

Ovde, `+` prvi spaja sekvencu slova i brojeva, time vraćajući `'12'`, zatim sledeći `+` dodaje `2` na njega, pa je kao `'12' + 2 = '122'`.

Binarni operateri `+` su jedini operateri koji podržavaju sekvence karaktera na ovaj način. Ostali aritmetički operateri rade samo sa brojevima i uvek konvertuju svoje argumente u brojeve.

Evo primera oduzimanja i deljenja:

```js
console.log( 6 - '2' ); // 4, converts '2' to a number
console.log( '6' / '2' ); // 3, converts both operands to numbers
```
