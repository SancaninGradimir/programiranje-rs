# --interactive--

CSS flexbox je primer jednodimenzionalnog rasporeda koji omogućava organizovanje elemenata u redove i kolone unutar kontejnera. Takođe možete kontrolisati njegov raspored i smer. Web programeri ga koriste za kreiranje veb sajtova i veb aplikacija koje se prilagođavaju različitim veličinama i orijentacijama ekrana. Pozivamo se na flexbox kao primer jednodimenzionalnog rasporeda jer se fokusira na raspoređivanje elemenata duž jedne ose istovremeno. Osa može biti horizontalna ili vertikalna.

Postoje dva glavna koncepta koja morate znati pre nego što počnete da radite sa flexbox-om: fleksibilni kontejner i fleksibilan element.

Fleksibilni kontejner je karakteristika HTML sa fleksibilnim rasporedom (flex layout). Možete raspoređivati i poravnati elemente na različite načine unutar fleksibilnog kontejnera. Da biste učinili element od HTML fleksibilnim kontejnerom, morate dodati `display: flex` u njegove stilove CSS.

Fleksibilne komponente su direktna deca fleksibilnog kontejnera. Ove komponente se mogu rasporediti i konfigurisati unutar fleksibilnog kontejnera u skladu sa njegovim karakteristikama. Takođe, mogu se smanjiti ili proširiti kako bi odgovarali dostupnom prostoru.

Ovo je primer koji sadrži kontener za `main` i tri elementa deteta `div`:

```html
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

Ako postavite samo `width`, `height`, na `background-color` ovih elemenata od `div` na stilskom listu za CSS, svaki element deteta će biti postavljen u svoj red jer kontejner nije flex po defaultu.

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Međutim, ako dodate `display: flex` u kontejner za `main`, elementi od `div` će biti ponovo organizovani da bi pristigli na jedan red i smanjiće se ako je potrebno (da biste videli prikaz, omogućite interaktivni editor):

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Podrazumevano, fleksibilni kontejner će biti blok nivo element, pa će sam kontejner biti postavljen u sopstveni red u poređenju sa drugim elementima i kontejnerima.

Sada znate više o fleksibilnim kontejnerima i fleksibilnim segmentima, trebalo biste da znate i o svojstvima fleksa. Ova svojstva određuju kako će fleksibilni segmenti biti raspoređeni, promenjeni veličine i podeljeni unutar fleksibilnog kontejnera. Neki od najčešće korišćenih svojstava su `flex-direction`, `justify-content`, `align-items`, i `flex-wrap`.

Dobro. Sada hajde da malo pričamo o fleks modelu. Ovaj model objašnjava kako se raznimljivi delovi raspoređuju unutar raznimljivog kontejnera. Svaki raznimljivi kontejner ima dve osovine:

- Glavna osovina.
- Osin za krsta.

Orientacija ovih osa određuje kako različite karakteristike utiču na raspored i distribuciju modularnih delova. Podrazumevano, glavna osa modularnog kontejnera je horizontalna, a poprečna osa je vertikalna. Modularni delovi su raspoređeni po smeru glavne ose. Poprečna osa je vertikalna u odnosu na glavnu osu.

Svojstvo `flex-direction` postavlja smer glavne ose. Podrazumevana vrednost `flex-direction` je `row`, što postavlja sve prilagodljive elemente na jednoj liniji, prema podrazumevanoj jezičnoj smernici vašeg pregledača (od leve ka desnoj ili od desne ka levoj):

```css
flex-direction: row; /* Default */
```

Da biste promenili redosled elemenata u redu, možete koristiti `flex-direction: row-reverse`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: row-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Ovo će promeniti raspored fleksibilnih delova. Ako želite umesto toga da poravnate fleksibilne delove vertikalno, samo morate postaviti `flex-direction` na `column` u fleksibilni kontejner. Ovo će promeniti smer glavne ose:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

Sada delovi `div` će biti poređani vertikalno jer će glavna osa biti uspravna, a križna osa horizontalna.

Takođe možete promeniti raspored vertikalnih prilagodljivih elemenata za `flex-direction: column-reverse`:

:::interactive_editor

```html
<link rel="stylesheet" href="styles.css">
<main>
  <div id="first-div"></div>
  <div id="second-div"></div>
  <div id="third-div"></div>
</main>
```

```css
main {
  display: flex;
  flex-direction: column-reverse; 
}

div {
  width: 80px;
  height: 50px;
}

#first-div {
  background-color: #4d70b2;
}

#second-div {
  background-color: #5c4db2;
}

#third-div {
  background-color: #4da3b2;
}
```

:::

CSS Flexbox je moćan model rasporeda koji pruža fleksibilan i efikasan način za aranžiranje elemenata unutar kontejnera. Razumevanjem glavnih koncepata fleksibilnih kontejnera, fleksibilnih stavki i različitih svojstava flex-a, možete kreirati veb stranice koje se prilagođavaju uređaju i menjaju veličinu i orijentaciju na različitim ekranima.
