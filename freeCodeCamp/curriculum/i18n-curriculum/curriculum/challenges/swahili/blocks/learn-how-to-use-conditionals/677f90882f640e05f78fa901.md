# --explanation--

Tokom `Present Continuous` često ukazuje na privremeno stanje ili problem koji se javlja tokom govora. Primer:

- `The server isn't running diagnostics.` - Pokazuje da proces proveravanja trenutno nije dostupan.

- `Developers aren't debugging the application.` - To pokazuje da programeri trenutno ne obavljaju taj posao.

`Properly` ima značenje uraditi nešto na ispravan ili očekivan način.

U ovoj rečenici, `aren't booting properly` se odnosi na računare koji trenutno nisu ispravno pokrenuti.
