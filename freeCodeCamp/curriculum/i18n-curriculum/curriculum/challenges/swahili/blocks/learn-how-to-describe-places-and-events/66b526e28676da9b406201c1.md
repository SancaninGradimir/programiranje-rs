# --explanation--

`Perhaps` služi za pokazivanje neizvesnosti ili mogućnosti. Ima značenje slično `maybe`. Na primer:

- `Perhaps we will launch the new feature next week.` - Postoji mogućnost lansiranja nove funkcije, ali nije sigurno.

- `They are perhaps the best developer for this project.` - Možda su dobri programeri, ali to nije sigurno.
